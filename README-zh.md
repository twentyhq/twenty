## 为什么选择 Twenty？

传统的 CRM 系统已经过时了。它们昂贵、复杂，并且锁定你的数据。Twenty 是一个现代化的开源替代方案：

- 🏠 **自托管** — 在你自己的基础设施上运行
- 🔓 **数据自由** — 你的数据永远属于你
- 🎨 **可定制** — 根据你的需求调整
- 💰 **成本效益** — 无需昂贵的许可证
- 🚀 **现代化** — 使用最新的技术栈构建

## 功能特性

### 核心功能

- 📇 **联系人管理** — 组织和管理你的联系人
- 🏢 **公司管理** — 跟踪公司和组织
- 💼 **机会管理** — 管理销售管道和交易
- 📋 **任务管理** — 创建和跟踪任务
- 📧 **电子邮件集成** — 连接你的电子邮件
- 📅 **日历集成** — 同步你的日历事件
- 📊 **自定义对象** — 创建自定义数据模型
- 🔍 **高级搜索** — 强大的搜索和过滤功能

### 开发者功能

- 🛠 **REST API** — 完整的 API 访问
- 🔄 **Webhooks** — 实时事件通知
- 🧩 **自定义字段** — 扩展数据模型
- 📱 **移动应用** — iOS 和 Android 应用
- 🔐 **SSO/SAML** — 企业级身份验证

## 快速开始

### 使用 Docker

```bash
git clone https://github.com/twentyhq/twenty.git
cd twenty
docker compose up -d
```

访问 `http://localhost:3000` 开始使用。

### 使用 Twenty Cloud

访问 [twenty.com](https://twenty.com) 使用托管版本。

## 技术栈

- **前端**：React、TypeScript、GraphQL
- **后端**：Node.js、PostgreSQL、Redis
- **基础设施**：Docker、Kubernetes
