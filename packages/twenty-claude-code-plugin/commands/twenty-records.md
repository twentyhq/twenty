Use the `retrieve-and-present-data` skill to retrieve, list, search, inspect, compare, or summarize Twenty MCP, CRM, app, or workspace records and present them as concise, user-friendly Markdown — not raw API output.

User input: $ARGUMENTS

If a Twenty MCP server is configured, follow the discovery flow (`get_tool_catalog -> learn_tools -> execute_tool`). If no Twenty MCP tools are available in this session, switch to the `setup-mcp` skill to configure one before retrieving data.
