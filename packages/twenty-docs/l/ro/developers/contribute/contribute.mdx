## Prezentare generală

Twenty este cu sursă deschisă și salută contribuțiile din partea comunității. Indiferent dacă remediați erori, adăugați funcționalități sau îmbunătățiți documentația, contribuțiile dvs. ajută la îmbunătățirea Twenty pentru toată lumea.

## Modalități de a contribui

* **Raportați erori**: Ajutați la identificarea și documentarea problemelor
* **Propuneți funcționalități**: Propuneți și implementați funcționalități noi
* **Îmbunătățiți documentația**: Faceți documentația noastră mai clară și mai utilă
* **Dezvoltare frontend**: Lucrați la interfața de utilizator bazată pe React
* **Dezvoltare backend**: Contribuiți la serverul NestJS
