## Pachete de Introducere

Obțineți ajutor din partea echipei noastre de bază pentru a vă configura spațiul de lucru Twenty cu pachetele noastre de Onboarding de 4 ore:

* **Proiectare model de date**: Proiectați și creați modelul de date personalizat cu obiecte, câmpuri și relații
* **Migrarea datelor**: Migrați datele existente din CRM-ul actual la Twenty
* **Creare Workflow**: Creați fluxuri de lucru personalizate pentru a susține procesele dvs. de afaceri

## Parteneri de implementare

Colaborați cu partenerii Twenty certificați pentru personalizări și integrații mai avansate. Contactați echipa noastră la [contact@twenty.com](mailto:contact@twenty.com) pentru a fi puși în legătură cu partenerii noștri.
