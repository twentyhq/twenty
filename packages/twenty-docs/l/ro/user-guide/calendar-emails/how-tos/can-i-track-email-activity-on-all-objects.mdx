## Obiecte acceptate

Urmărirea activității e-mailurilor este disponibilă în prezent pe **trei obiecte standard**:

| Obiect           | Ce veți vedea                                                                 |
| ---------------- | ----------------------------------------------------------------------------- |
| **Persoane**     | Toate e-mailurile schimbate cu respectivul contact                            |
| **Companii**     | Toate e-mailurile cu oricine din acea companie (pe baza domeniului de e-mail) |
| **Oportunități** | Toate e-mailurile legate de compania asociată oportunității                   |

## De ce doar aceste obiecte?

Persoane, Companii și Oportunități sunt obiectele esențiale de relație în care contextul e-mailului aduce cea mai mare valoare. Conversațiile prin e-mail sunt asociate automat pe baza:

* **Adresă de e-mail** → asociată cu înregistrările din Persoane
* **Domeniu de e-mail** → asociat cu înregistrările din Companii
* **Relație cu compania** → legată de Oportunități

## Obiecte personalizate

**Urmărirea e-mailurilor nu este disponibilă pe obiectele personalizate** în acest moment.

Dacă aveți nevoie de contextul e-mailului pe un obiect personalizat, luați în considerare:

* Folosirea unui câmp de relație pentru a lega obiectul personalizat de Persoane sau Companii
* Vizualizarea istoricului e-mailurilor în înregistrarea Persoane/Companie asociată

## Planuri viitoare

Se ia în considerare extinderea vizibilității e-mailurilor la obiectele personalizate. Împărtășiți cazul dvs. de utilizare în [discuțiile noastre pe GitHub](https://github.com/twentyhq/twenty/discussions) pentru a ajuta la prioritizarea acestei funcționalități.
