## Starea actuală

Integrarea Twenty pentru calendar este concepută pentru a **sincroniza și afișa** evenimentele existente din calendarul dumneavoastră.

## Programarea întâlnirilor prin fluxuri de lucru

Deși nu este posibil să programați întâlniri manual, le puteți programa în Twenty folosind fluxuri de lucru

1. Creați un nou flux de lucru
2. Selectați orice declanșator, în funcție de nevoile dumneavoastră
3. Adăugați nodul **Create Calendar Event**
4. Configurați toate datele necesare
5. Activați fluxul de lucru

## Ce puteți face

* **Vizualizați istoricul întâlnirilor** pe înregistrările Persoane, Companii și Oportunități
* **Vizualizați întâlnirile viitoare** cu contactele din CRM-ul dumneavoastră
* **Urmăriți contextul întâlnirilor** alături de comunicările prin e-mail
* **Creați automat contacte** din participanții la întâlniri
