## Vizibilitatea vizualizării

Controlați cine poate vedea vizualizările dvs. personalizate.

### Opțiuni de vizibilitate

| Setare              | Cine poate vedea                |
| ------------------- | ------------------------------- |
| **Spațiu de lucru** | Toți membrii spațiului de lucru |
| **Nelistat**        | Doar dvs.                       |

### Schimbarea vizibilității

1. Deschideți vizualizarea
2. Faceți clic pe **Opțiuni → Vizibilitate**
3. Selectați **Spațiu de lucru** sau **Nelistat**

<Note>
  Vizualizările implicite "Toate [Numele obiectului]" nu pot avea vizibilitatea modificată.
</Note>

## Redenumiți o vizualizare

1. Deschideți meniul derulant al vizualizării
2. Faceți clic pe meniul **⋮** de lângă vizualizare
3. Selectați **Editare**
4. Introduceți noul nume

## Schimbați pictograma vizualizării

1. Deschideți meniul derulant al vizualizării
2. Faceți clic pe meniul **⋮** de lângă vizualizare
3. Selectați **Editare**
4. Faceți clic pe pictogramă pentru a o schimba

## Reordonați vizualizările

Modificați ordinea în care vizualizările apar în lista derulantă:

1. Deschideți meniul derulant al vizualizării
2. Trageți vizualizările folosind mânerul lor
3. Plasați în poziția dorită
4. Ordinea se salvează automat

## Favorite

Fixați vizualizările utilizate frecvent pentru acces rapid:

1. Deschideți meniul derulant al vizualizării
2. Faceți clic pe meniul **⋮** de lângă o vizualizare
3. Selectați **Adaugă la favorite**

Vizualizările adăugate la favorite apar într-o secțiune dedicată pentru acces facil.

## Ștergeți o vizualizare

1. Deschideți meniul derulant al vizualizării
2. Faceți clic pe meniul **⋮** de lângă vizualizare
3. Selectați **Ștergere**
4. Confirmați ștergerea

<Warning>
  Vizualizările șterse nu pot fi recuperate.
</Warning>
