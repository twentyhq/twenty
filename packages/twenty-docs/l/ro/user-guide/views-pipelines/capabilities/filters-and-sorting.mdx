## Filtrarea datelor

Filtrele vă ajută să vă concentrați pe înregistrări specifice, afișând doar pe cele care corespund criteriilor.

### Adăugarea unui filtru

1. Faceți clic pe butonul **Filtru** din bara de instrumente
2. Selectați câmpul după care să filtrați
3. Alegeți operatorul (egal, conține etc.)
4. Introduceți valoarea filtrului
5. Faceți clic pe **Aplică**

### Operatori de filtrare

| Tipul câmpului | Operatori disponibili                                        |
| -------------- | ------------------------------------------------------------ |
| Text           | Este egal cu, Conține, Începe cu, Se termină cu, Este gol    |
| Număr          | Este egal cu, Mai mare decât, Mai mic decât, Între, Este gol |
| Dată           | Este egal cu, Înainte de, După, Între, Este gol              |
| Select         | Este egal cu, Este unul dintre, Este gol                     |
| Bifă           | Este adevărat, Este fals                                     |
| Relație        | Este egal cu, Este gol                                       |

### Filtre multiple

Combinați mai multe filtre pentru a restrânge rezultatele:

* Toate filtrele sunt aplicate cu logica AND
* Fiecare filtru suplimentar restrânge și mai mult rezultatele

### Eliminarea filtrelor

* Faceți clic pe **X** de pe fiecare etichetă de filtru
* Faceți clic pe **Șterge tot** pentru a elimina toate filtrele

## Sortarea datelor

Sortarea determină ordinea în care apar înregistrările.

### Adăugarea unei sortări

1. Faceți clic pe butonul **Sortare** din bara de instrumente
2. Selectați câmpul după care să sortați
3. Alegeți ascendent (A-Z, 0-9) sau descendent (Z-A, 9-0)
4. Faceți clic pe **Aplică**

### Sortări multiple

Adăugați mai multe niveluri de sortare:

* Prima sortare este principală
* Sortările ulterioare se aplică în interiorul grupurilor de valori egale

### Sortare rapidă pe coloană

Faceți clic pe orice antet de coloană pentru a sorta:

* Primul clic: Ascendent
* Al doilea clic: Descendent
* Al treilea clic: Elimină sortarea

## Salvarea setărilor de filtrare și sortare

Filtrele și sortările sunt salvate împreună cu vizualizarea:

1. Configurați filtrele și sortările
2. Faceți clic pe **Salvare** pentru a actualiza vizualizarea curentă
3. Sau faceți clic pe **Salvează ca vizualizare nouă** pentru a crea o variantă
