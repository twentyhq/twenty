Utilizați o vizualizare Kanban pentru a afișa proiectele (sau orice obiect cu etape) ca carduri care se deplasează între coloane.

<VimeoEmbed videoId="927888627" title="Demonstrație video" />

## Cerințe

Obiectul are nevoie de un **câmp Select** pentru a fi folosit ca coloane (de ex., Stare, Etapă, Fază).

Dacă nu aveți unul:

1. Accesați **Setări → Model de date**
2. Selectați obiectul
3. Adăugați un câmp Select cu opțiunile pentru etape

## Pași

1. Navigați la obiectul dvs. (de ex., Proiecte, Sarcini)
2. Faceți clic pe meniul derulant al vizualizării → **+ Adaugă vizualizare**
3. Denumiți vizualizarea (de ex., "Tablou proiect")
4. Faceți clic pe **Creare**
5. Faceți clic pe **Opțiuni** și selectați **Kanban** ca tip de afișare
6. Vizualizarea utilizează automat câmpul dvs. Select pentru coloane
7. Faceți clic pe **Salvare**

## Configurați tabloul

### Afișați câmpurile cheie pe carduri

1. Faceți clic pe **Opțiuni → Câmpuri**
2. Găsiți câmpurile în secțiunea "Câmpuri ascunse"
3. Faceți clic pe **pictograma ochi** pentru a le afișa pe carduri
4. Trageți pentru a reordona

<img src="/images/user-guide/kanban-views/filter.png" style={{width:'100%'}} />

### Activați vizualizarea compactă

Pentru o vedere de ansamblu:

1. Faceți clic pe **Opțiuni**
2. Activați comutatorul pentru **Vizualizare compactă**

Cardurile afișează doar numele înregistrării.

<img src="/images/user-guide/kanban-views/compact-view.png" style={{width:'100%'}} />

### Adăugați agregări

Afișați numărul de elemente sau totalurile în partea de sus a fiecărei coloane:

1. Faceți clic pe numărul de lângă numele unei coloane
2. Selectați o agregare (Număr, Sumă etc.)
3. Alegeți un câmp, dacă este necesar

## Mutarea cardurilor

Trageți și plasați cardurile între coloane pentru a le actualiza starea.

<VimeoEmbed videoId="927888627" title="Demonstrație video" />

## Exemplu: Panou de sarcini

| Coloană (Stare) | Carduri                 |
| --------------- | ----------------------- |
| **De făcut**    | Sarcini noi             |
| **În progres**  | Sarcini active          |
| **Revizuire**   | În așteptarea aprobării |
| **Finalizat**   | Completat               |
