Grupați vizualizarea de tabel după un câmp Select pentru a organiza înregistrările în secțiuni pliabile.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

## Pași

1. Navigați la obiect (Persoane, Companii etc.)
2. Faceți clic pe meniul derulant al vizualizării → **+ Adaugă vizualizare**
3. Denumiți vizualizarea (de ex., "Companii după tip")
4. Faceți clic pe **Creare**
5. Faceți clic pe **Opțiuni → Grupare**
6. Alegeți un câmp Select după care să grupați
7. Faceți clic pe **Salvare**

## Configurați ordinea grupurilor

În **Opțiuni → Grupare → Sortare**, alegeți cum sunt ordonate grupurile:

| Opțiunea             | Descriere                                                    |
| -------------------- | ------------------------------------------------------------ |
| **Alfabetic**        | de la A la Z                                                 |
| **Alfabetic invers** | de la Z la A                                                 |
| **Ordine manuală**   | Trageți grupurile pentru a le reordona în "Grupuri vizibile" |

Faceți clic pe **pictograma în formă de ochi** de lângă un grup pentru a-l ascunde din vizualizare.

<Note>
  **Pentru performanțe optime, limitați la 10-15 grupuri vizibile.** Dacă aveți nevoie de mai multe, luați în considerare utilizarea unui tablou de bord.
</Note>

## Exemplu: Companii după industrie

1. Mergeți la **Companii**
2. Creați o vizualizare nouă numită "După industrie"
3. Faceți clic pe **Opțiuni → Grupare**
4. Selectați câmpul **Industrie**
5. Salvează

Acum companiile dvs. sunt organizate după industrie, ceea ce facilitează concentrarea pe câte un segment odată.
