## Despre

Depozitul Twenty de pe GitHub găzduiește o gamă vastă de resurse precum cod sursă, documentație, discuții și urmărirea problemelor. Aici vei putea accesa codul complet în spatele Twenty.

[Vizitează Twenty pe GitHub](https://github.com/twentyhq/twenty)

## Contribuții

Contribuirea la proiectul Twenty pe GitHub este o modalitate satisfăcătoare de a ajuta la îmbunătățirea software-ului pe care îl folosești. Indiferent dacă remediezi erori, sugerezi funcționalități sau îmbunătățești documentația, contribuțiile tale sunt binevenite.

### Raportarea problemelor

Ai întâmpinat o problemă? [Creează o problemă](https://github.com/twentyhq/twenty/issues/new) pe GitHub, oferind cât mai multe detalii posibil.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggesting Features

Ce îmbunătățiri ai dori să vezi la Twenty? Indiferent de nivelul tău tehnic, poți participa la [conversație aici](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Coding a feature

Începe-ți drumul găsind sarcini prietenoase cu începătorii:

1. Navighează la fila **[Issues](https://github.com/twentyhq/twenty/issues)** din depozitul Twenty.
2. Filter by **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** label to find tasks suited for newcomers.
3. Alege o problemă, împarte depozitul și începe să contribui.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Asigură-te că ești atribuit problemei pentru a evita suprapunerea lucrului cu alți contribuitori.

### Codul de Conduită

Adu-ți aminte să respecți [Codul de Conduită](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) al Twenty pe parcursul procesului tău de contribuție.
