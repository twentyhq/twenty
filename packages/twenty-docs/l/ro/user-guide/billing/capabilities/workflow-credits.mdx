## Prezentare generală

Creditele alimentează automatizările fluxurilor de lucru în Twenty. Fiecare acțiune de flux de lucru consumă credite în funcție de complexitatea sa.

## Alocarea creditelor

Creditele se bazează pe ciclul de facturare, nu pe planul dvs.:

| Ciclu de facturare | Credite           |
| ------------------ | ----------------- |
| Lunar              | 5 milioane/lună   |
| Anual              | 50 de milioane/an |

<Note>
  Cele 5 milioane de credite lunare sunt concepute pentru a vă permite să rulați automatizări fără să vă faceți griji pentru costuri. Pentru majoritatea fluxurilor de lucru care folosesc acțiuni standard, acest lucru este mai mult decât suficient. Veți avea nevoie de credite suplimentare doar atunci când rulați noduri de cod avansate sau funcționalități bazate pe AI.
</Note>

## Consumul de credite

Acțiunile diferite consumă cantități diferite de credite:

| Tip de acțiune                                                        | Utilizare credit                  |
| --------------------------------------------------------------------- | --------------------------------- |
| **Operațiuni de bază** (căutare, actualizare, creare de înregistrări) | Minim                             |
| **Operațiuni complexe** (noduri de cod, apeluri API externe)          | Mai multe credite                 |
| **Indicațiile AI** (în curând)                                        | Variabil, în funcție de utilizare |

Creditele sunt retrase în timp real pe măsură ce fluxurile de lucru se execută.

## Monitorizarea utilizării

Urmăriți consumul de credite:

1. Accesați **Setări → Facturare**
2. Vizualizați utilizarea curentă și creditele rămase
3. Monitorizați tendințele pentru a planifica credite suplimentare, dacă este necesar

## Achiziționarea de credite suplimentare

Aveți nevoie de mai multe credite?

1. Accesați **Setări → Facturare**
2. Faceți clic pe opțiunea de a achiziționa pachete de credite suplimentare
3. Selectați cantitatea de care aveți nevoie
