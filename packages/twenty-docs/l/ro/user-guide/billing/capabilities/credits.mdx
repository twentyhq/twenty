## Prezentare generală

Creditele alimentează automatizările fluxurilor de lucru și capabilitățile AI. Acestea sunt consumate atunci când acțiunile din fluxurile de lucru se execută, când agenții AI procesează sarcini în cadrul fluxurilor de lucru și în chatul AI.

## Alocarea creditelor

Creditele se bazează pe ciclul de facturare, nu pe planul dvs.:

| Ciclu de facturare | Credite |
| ------------------ | ------- |
| Lunar              | 5/lună  |
| Anual              | 50/an   |

<Note>
Cele 5 credite lunare sunt concepute pentru a vă permite să rulați automatizări fără să vă faceți griji în privința costurilor. Pentru majoritatea fluxurilor de lucru care folosesc acțiuni standard, acest lucru este mai mult decât suficient. Veți avea nevoie de credite suplimentare doar atunci când rulați noduri de cod avansate sau funcționalități bazate pe AI.
</Note>

## Reportare de credite

Creditele neutilizate la sfârșitul unei perioade de facturare sunt reportate automat în perioada următoare.

* **Limită**: Suma reportată este limitată la alocarea completă a unei perioade, astfel încât nu veți putea reporta mai multe credite decât oferă planul dvs. pe perioadă.
* **Vizibilitate**: Când există credite reportate, acestea apar ca o linie separată **Credite reportate** în **Setări → Facturare**, alături de un sold **Total disponibil**.

## Consumul de credite

Acțiunile diferite consumă cantități diferite de credite:

| Tip de acțiune                                                        | Utilizare credit                  |
| --------------------------------------------------------------------- | --------------------------------- |
| **Operațiuni de bază** (căutare, actualizare, creare de înregistrări) | Minim                             |
| **Operațiuni complexe** (noduri de cod, apeluri API externe)          | Mai multe credite                 |
| **Prompturi AI și chaturi AI**                                        | Variabil, în funcție de utilizare |

Creditele sunt retrase în timp real când se execută fluxurile de lucru.

## Monitorizarea utilizării

Urmăriți consumul de credite:

1. Accesați **Setări → Facturare**
2. Vizualizați utilizarea curentă și creditele rămase
3. Monitorizați tendințele pentru a planifica credite suplimentare, dacă este necesar

## Achiziționarea de credite suplimentare

Aveți nevoie de mai multe credite?

1. Accesați **Setări → Facturare**
2. Faceți clic pe opțiunea de a achiziționa pachete de credite suplimentare
3. Selectați cantitatea de care aveți nevoie
