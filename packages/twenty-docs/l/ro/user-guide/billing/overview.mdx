Twenty oferă planuri de preț flexibile care se potrivesc nevoilor echipei dvs. Gestionați-vă abonamentul, urmăriți creditele pentru fluxuri de lucru și accesați facturile, toate din **Setări → Facturare**.

## Dintr-o privire

| Plan                          | Funcții cheie                                                                                                                                                                                              |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Gratuit (Autogăzduit)**     | Toate funcțiile Pro, asistență din partea comunității                                                                                                                                                      |
| **Pro (Cloud)**               | Totul, cu excepția funcțiilor Premium (SSO și permisiuni la nivel de rând), asistență standard                                                                                                             |
| **Organizație (Cloud)**       | Tot ce include Pro + funcțiile Premium (SSO și permisiuni la nivel de rând), asistență prioritară                                                                                                          |
| **Organizație (Autogăzduit)** | Tot ce include Pro + funcțiile Premium (SSO, permisiuni la nivel de rând), asistență din partea echipei Twenty, nu este necesar să publicați codul dvs. personalizat ca open-source înainte de distribuire |

## Răspunsuri rapide

**Unde gestionez facturarea?**
Accesați **Setări → Facturare** pentru a vă vedea planul, pentru a actualiza metodele de plată și pentru a accesa facturile.

**Pot folosi Twenty gratuit?**
Da! Găzduiți Twenty pe cont propriu și obțineți toate funcțiile Pro fără costuri.

**Cum fac upgrade?**
Mergeți la **Setări → Facturare** și faceți clic pe **Treceți la Organizație** sau **Treceți la Anual**.

Pentru mai multe întrebări, consultați [Întrebările frecvente despre facturare](/l/ro/user-guide/billing/how-tos/billing-faq).
