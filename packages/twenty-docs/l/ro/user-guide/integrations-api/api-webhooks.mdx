## Chei API

Cheile API permit accesul automatizat la datele dumneavoastră CRM, sincronizarea datelor cu alte sisteme și crearea de integrări sau soluții personalizate.

### Creați o cheie API

1. Mergeți la **Setări → API-uri & Webhook-uri**
2. Click **+ Create key** at the top right
3. Configurați cheia API:
   - **Nume**: Oferiți cheii API un nume descriptiv
   - **Data de expirare**: Stabiliți când ar trebui să expire cheia
4. Click **Save** to generate your API key
5. **Important**: Copiați și stocați cheia API imediat, este afișată doar o dată

Once created, your API key provides access to your custom API documentation and playground where you can test endpoints with your actual data model.

<Warning>
Deoarece cheia API oferă acces la informații sensibile, nu ar trebui să o partajați cu servicii în care nu aveți încredere deplină. If leaked, someone can use it maliciously. Dacă securitatea cheii API este compromisă, dezactivați-o imediat și generați una nouă.
</Warning>

### Gestionați cheile API

**Regenerează o cheie API:**

1. Mergeți la **Setări → API-uri & Webhook-uri**
2. Faceți clic pe cheia API pe care doriți să o regenerați
3. Faceți clic pe butonul **Regenerați**
4. Copiați și salvați noua cheie API imediat

**Șterge cheia API:**

1. Găsiți cheia API în lista dvs.
2. Click on the key to open its details
3. Faceți clic pe **Șterge** pentru a o elimina permanent

## Webhooks

Webhooks permit actualizări imediate către URL-ul specificat al dvs. despre schimbări sau evenimente legate de datele clienților dvs.

De exemplu, când o oportunitate trece la "Câștigat", un webhook poate declanșa automat crearea unei facturi în sistemul dvs. de contabilitate. Note that this type of automation can also be achieved using Twenty's in-app [Workflows feature](/l/ro/user-guide/workflows/getting-started-workflows), which offers triggers based on field updates for internal automation.

Webhooks are ideal for integrating with external systems, while Workflows support both internal automation and external tool connections via webhook triggers, code nodes, and HTTP nodes.

### Creează Webhook

1. Mergeți la **Setări → API-uri şi Webhooks → Webhooks**
2. Faceți clic pe **+ Creează webhook**
3. Introduceți URL-ul webhook-ului dvs. (unde doriți să primiți notificări)
4. Faceți clic pe **Salvare**

Webhook-ul dvs. va începe imediat să primească notificări în timp real despre modificările datelor din CRM-ul dvs.

### Administrează Webhooks

**Șterge un Webhook:**

1. Mergeți la **Setări → API-uri şi Webhooks → Webhooks**
2. Găsiți webhook-ul pe care doriți să-l eliminați
3. Faceți clic pe webhook
4. Click **Delete** and confirm in the popup

**Editeaza un Webhook:**

1. Faceți clic pe webhook-ul pe care doriți să-l modificați
2. Actualizați URL-ul sau alte setări
3. Faceți clic pe **Salvare** pentru a aplica modificările
