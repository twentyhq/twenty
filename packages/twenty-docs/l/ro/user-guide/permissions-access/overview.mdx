## Caracteristici cheie

* **Acces bazat pe roluri**: Creați roluri personalizate cu permisiuni specifice
* **Permisiuni pentru obiecte**: Controlați cine poate vizualiza, edita sau șterge înregistrări
* **Permisiuni pentru câmpuri**: Restricționați accesul la câmpuri sensibile
* **Permisiuni pentru setări**: Controlați accesul la configurarea spațiului de lucru
* **Integrare SSO**: Configurați autentificarea unică pentru securitate la nivel de întreprindere (plan Organization)
