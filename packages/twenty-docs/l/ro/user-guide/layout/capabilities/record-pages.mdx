## Prezentare generală

Când deschideți o înregistrare în Twenty, pagina de detalii este alcătuită din **file** și **widgeturi**. Ambele sunt complet personalizabile pentru fiecare tip de obiect.

## Taburi

Fiecare pagină de înregistrare poate avea mai multe file — similare cu filele dintr-un browser. Folosiți-le pentru a organiza diferite aspecte ale unei înregistrări. De exemplu, o înregistrare Companie poate avea file pentru Prezentare generală, Comunicare, Sarcini și Fișiere.

Puteți:

* Adăugați și eliminați file
* Redenumiți filele
* Reordonați filele prin glisare
* Setați care filă este afișată în mod implicit

## Widgeturi

Widgeturile sunt elementele de bază din fiecare filă. Tipuri de widgeturi disponibile:

| Widget                    | Ce afișează                                        |
| ------------------------- | -------------------------------------------------- |
| **Câmpuri**               | Câmpuri ale înregistrării, grupate sau individuale |
| **Înregistrări asociate** | Tabel cu înregistrări legate printr-o relație      |
| **Email-uri**             | Istoricul emailurilor din conturile conectate      |
| **Calendar**              | Evenimente de calendar asociate înregistrării      |
| **Cronologie**            | Istoricul activităților și evenimentelor           |
| **Sarcini**               | Sarcini asociate                                   |
| **Notițe**                | Notițe cu text îmbogățit                           |
| **Fișiere**               | Atașamente de fișiere                              |
| **Grafice**               | Date vizuale din înregistrări asociate             |
| **iFrame**                | Conținut extern încorporat                         |
| **Text îmbogățit**        | Conținut static sau descrieri                      |

## Personalizarea unei pagini de înregistrare

1. Deschideți orice înregistrare
2. Apăsați `Cmd+K` și căutați "Edit record page layout"

sau

1. Accesați Setări > Model de date > obiectul dorit > Aspect

2. Faceți clic pe butonul "Personalizați pagina înregistrării" pentru acel obiect

3. Acum sunteți în modul de personalizare:
   * **Adăugați widgeturi** din selectorul de widgeturi
   * **Glisați widgeturile** pentru a le repoziționa pe grilă
   * **Redimensionați widgeturile** trăgându-le marginile
   * **Configurați câmpurile** afișate în fiecare widget
   * **Gestionați filele** — adăugați, eliminați, redenumiți, reordonați

4. Salvați modificările — se aplică tuturor înregistrărilor din tipul respectiv de obiect

## Vizibilitatea câmpurilor

Într-un widget Câmpuri, puteți controla care câmpuri sunt vizibile și în ce ordine. Acest lucru vă permite să creați dispuneri concentrate — de exemplu, să afișați doar cele mai importante câmpuri în fila Prezentare generală și să plasați câmpurile detaliate într-o filă separată.
