Aspectul Twenty este personalizabil la trei niveluri: modul în care navigați în aplicație, modul în care răsfoiți listele de înregistrări și ceea ce vedeți când deschideți o înregistrare individuală.

## Navigare

Bara laterală stângă este complet personalizabilă. Puteți:

* **Reordonați elementele** prin glisare și plasare
* **Creați dosare** pentru a grupa obiecte și vizualizări înrudite
* **Ascundeți obiectele** pe care nu le folosiți
* **Adăugați linkuri personalizate** către instrumente externe
* **Fixați elemente favorite** pentru acces rapid la vizualizări, înregistrări sau căutări

[Referință de navigare →](/l/ro/user-guide/layout/capabilities/navigation)

## Vizualizări

Vizualizările controlează modul în care sunt afișate listele de înregistrări. Twenty acceptă trei tipuri de vizualizări:

| Vizualizare  | Cel mai potrivit pentru                                                             |
| ------------ | ----------------------------------------------------------------------------------- |
| **Tabel**    | Lucrul cu multe înregistrări deodată — rânduri și coloane în stil foaie de calcul   |
| **Kanban**   | Urmărirea pipeline-ului — carduri organizate pe etape, cu glisare și plasare        |
| **Calendar** | Planificare în funcție de timp — înregistrări afișate în funcție de un câmp de dată |

Fiecare vizualizare își salvează propriile filtre, ordonarea, vizibilitatea câmpurilor și configurația de grupare. Vizualizările pot fi partajate cu spațiul de lucru sau păstrate private.

[Vizualizări de tip tabel →](/l/ro/user-guide/views-pipelines/capabilities/table-views) · [Vizualizări Kanban →](/l/ro/user-guide/views-pipelines/capabilities/kanban-views) · [Vizualizare calendar →](/l/ro/user-guide/views-pipelines/capabilities/calendar-view)

## Pagini de înregistrare

Când deschideți o înregistrare, pagina de detalii este construită din file și widgeturi configurabile. Puteți:

* **Adăugați, eliminați și reordonați filele** pentru orice tip de înregistrare
* **Configurați widgeturi** — câmpuri, înregistrări asociate, e-mailuri, cronologie, calendar, sarcini, note, fișiere, diagrame, iframe-uri și altele
* **Glisați și redimensionați widgeturile** pe o dispunere în grilă
* **Controlați vizibilitatea câmpurilor** pentru fiecare widget

Intrați în modul de personalizare a aspectului din meniul de comenzi (`Cmd+K` → "Editare aspect pagină de înregistrare").
