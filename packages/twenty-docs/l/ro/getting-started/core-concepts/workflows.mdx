Fluxurile de lucru vă permit să automatizați sarcinile repetitive și să conectați Twenty la instrumente externe — fără a scrie cod (deși puteți, dacă doriți).

## Cum funcționează fluxurile de lucru

Fiecare flux de lucru are trei părți:

1. **Declanșator** — Ce pornește fluxul de lucru
2. **Pași** — Ce se întâmplă în continuare (una sau mai multe acțiuni în secvență)
3. **Variabile** — Datele care circulă între pași

## Declanșatoare

| Declanșator                   | Când este declanșat                                                                                            |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Eveniment de înregistrare** | O înregistrare este creată, actualizată, ștearsă sau upsertată                                                 |
| **Manual**                    | Un utilizator face clic pe un buton (pe o singură înregistrare, pe mai multe înregistrări sau la nivel global) |
| **Programare**                | La un interval recurent (sintaxă cron)                                                                         |
| **Webhook**                   | Un sistem extern trimite un HTTP POST                                                                          |

## Acțiuni

Fluxurile de lucru pot lega în lanț orice combinație de:

* **Operațiuni pe înregistrări** — Crearea, actualizarea, căutarea, ștergerea sau upsertarea înregistrărilor
* **Trimitere e-mail** — Trimiteți sau redactați e-mailuri din conturile conectate
* **Solicitare HTTP** — Apelați orice API extern
* **Cod** — Rulați JavaScript personalizat pentru logică complexă
* **Ramificații** — Condiții if/else pentru a diviza traseul fluxului de lucru
* **Iterator** — Parcurgeți în buclă tablouri de date
* **Agent AI** — Lăsați un agent AI să proceseze datele în mod autonom
* **Întârziere** — Așteptați înainte de a continua
* **Formular** — Colectați input-ul utilizatorului la mijlocul fluxului de lucru

## Ce puteți construi

* Trimiteți alerte Slack când un deal ajunge într-o anumită etapă
* Îmbogățiți automat noii contacte cu date din API-uri externe
* Detectați oportunitățile învechite și notificați deținătorul
* Sincronizați datele între Twenty și sistemul dvs. de facturare
* Generați PDF-uri sau facturi din datele înregistrărilor
* Răspundeți automat la e-mailurile primite care corespund anumitor criterii
