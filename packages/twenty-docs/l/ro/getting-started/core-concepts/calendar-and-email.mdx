Twenty se conectează la instrumentele dvs. existente, astfel încât CRM-ul dvs. rămâne actualizat fără introducere manuală de date.

## Sincronizare e-mail

Conectați-vă căsuța poștală **Google Workspace** sau **Microsoft 365**. După conectare:

* E-mailurile sunt asociate automat înregistrărilor corespunzătoare din Companii și Persoane
* Firele complete de e-mail sunt vizibile pe cronologia fiecărei înregistrări
* Puteți trimite e-mailuri direct din Twenty
* Sunt acceptate mai multe căsuțe poștale per utilizator

Controlați ce este importat — filtrați după interval de date sau expeditor pentru a evita importul de e-mailuri nerelevante.

## Sincronizare calendar

Evenimentele din calendar se sincronizează automat din contul dvs. conectat. Evenimentele apar în înregistrările CRM relevante, oferindu-vă o imagine completă a interacțiunilor cu fiecare contact sau companie.

## Integrări

Dincolo de e-mail și calendar, Twenty se conectează la instrumente externe prin:

| Metodă                                   | Caz de utilizare                                                                               |
| ---------------------------------------- | ---------------------------------------------------------------------------------------------- |
| **API**                                  | Creați integrări personalizate cu API-urile GraphQL sau REST                                   |
| **Webhook-uri**                          | Trimiteți notificări în timp real către sisteme externe atunci când înregistrările se modifică |
| **Zapier**                               | Conectați-vă la peste 5.000 de aplicații fără cod                                              |
| **Acțiuni HTTP pentru fluxuri de lucru** | Apelați orice API extern ca parte a unui flux de lucru automatizat                             |

## Aplicații personalizate

Dezvoltatorii pot construi aplicații complete peste Twenty — adăugând UI personalizat, logică de server și integrări profunde. Aplicațiile pot fi publicate pentru comunitate sau păstrate private.
