## Current Integration Options

### Integracions Natives

Twenty actualment ofereix integració nativa amb:

- **Email i Calendari**: Connecta Gmail, Outlook o proveïdors SMTP/CalDAV
- **API Access**: Use our REST and GraphQL APIs to build custom integrations

For email and calendar setup, visit [Email & Calendar Setup](/l/ca/user-guide/settings/email-calendar-setup).

### Fluxos de treball (Recomanat)

La manera principal de connectar Twenty a altres eines és a través de la nostra característica de **Fluxos de Treball** dins de l'aplicació:

- **HTTP Nodes**: Make API calls to external services
- **Nodes de codi**: Escriu lògica personalitzada per a integracions complexes
- **Activadors de Webhooks**: Rep dades de sistemes externs
- **Activadors d'Actualització de Camps**: Automatitza accions basades en canvis de CRM

Aprèn més a la nostra [secció de Fluxos de Treball](/l/ca/user-guide/workflows/getting-started-workflows).

### Zapier Integration (Legacy)

We maintain a Zapier integration for users who prefer no-code automation:

1. Visita [Twenty en Zapier](https://zapier.com/apps/twenty/integrations)
2. Crea un nou Zap amb Twenty com a activador o acció
3. Genera una clau API a Configuració → API i Webhooks
4. Connecta el teu espai de treball de Twenty a Zapier

## Future Vision: Community-Built Connectors (2026)

### Plataforma d'Extensibilitat

Estem desenvolupant una plataforma d'extensibilitat que permetrà als desenvolupadors crear aplicacions com codi. Això permetrà:

- **Connectors Personalitzats**: Crea integracions amb el teu programari favorit
- **Contribucions de la Comunitat**: Comparteix i descobreix connectors creats per altres
- **Arquitectura Flexible**: Amplia les capacitats de Twenty més enllà de les funcions bàsiques

### Construcció de Fluxos Assistida per IA

Arribant en 2026, l'assistència IA ajudarà als usuaris:

- **Genera Automàticament Fluxos**: Descriu les teves necessitats d'integració en llenguatge senzill
- **Suggeriments Intel·ligents**: Rep recomanacions per connectar les teves eines específiques
- **Biblioteca de Plantilles**: Accedeix a fluxos predefinits per a casos d'ús comuns

El futur de les integracions de Twenty és conduït per la comunitat, assistit per IA i infinitament extensible.
