## Claus API

Les claus de l'API permeten l'accés automatitzat a les dades del CRM, sincronitzant dades amb altres sistemes, i crear integracions o solucions personalitzades.

### Crear una clau d'API

1. Go to **Settings → APIs & Webhooks**
2. Fes clic a **+ Crea clau** a la part superior dreta
3. Configura la teva clau d'API:
   - **Nom**: Dona un nom descriptiu a la teva clau d'API
   - **Data de Caducitat**: Defineix quan ha de caducar la clau
4. Click **Save** to generate your API key
5. **Important**: Copia i emmagatzema la teva clau d'API immediatament, només es mostra una vegada

Un cop creada, la teva clau d'API proporciona accés a la documentació i al camp de proves personalitzats de l'API on pots provar punts finals amb el teu model de dades actual.

<Warning>
Com que la teva clau d'API dóna accés a informació sensible, no hauries de compartir-la amb serveis que no et mereixen plena confiança. Si es filtra, algú pot utilitzar-la de manera malintencionada. Si la seguretat de la teva clau d'API queda compromesa, inhabilita-la immediatament i genera'n una de nova.
</Warning>

### Gestiona les claus d'API

**Regenera una clau d'API:**

1. Go to **Settings → APIs & Webhooks**
2. Fes clic a la clau d'API que vols regenerar
3. Fes clic al botó **Regenera**
4. Copia i desa la nova clau d'API immediatament

**Suprimeix la clau d'API:**

1. Troba la clau d'API a la teva llista
2. Fes clic sobre la clau per obrir-ne els detalls
3. Click **Delete** to remove it permanently

## Webhooks

Els Webhooks permeten actualitzacions immediates al URL especificat sobre canvis o esdeveniments relacionats amb les dades dels teus clients.

Per exemple, quan una Oportunitat es mou a "Tancat Guanyat", un webhook pot activar automàticament la creació de factures al teu sistema comptable. Tingues en compte que aquest tipus d'automatització també pot aconseguir-se utilitzant la funcionalitat [de Workflows de l'aplicació de Twenty](/l/ca/user-guide/workflows/getting-started-workflows), que ofereix activadors basats en actualitzacions de camps per a l'automatització interna.

Els Webhooks són ideals per a la integració amb sistemes externs, mentre que els Workflows donen suport a l'automatització interna i connexions amb eines externes mitjançant activadors de webhook, nodes de codi i nodes HTTP.

### Create a Webhook

1. Ves a **Configuració → API i Webhooks → Webhooks**
2. Fes clic a **+ Crea webhook**
3. Introdueix el teu URL de webhook (on desitges rebre notificacions)
4. Fes clic a **Desa**

El teu webhook començarà immediatament a rebre notificacions en temps real sobre canvis a les teves dades de CRM.

### Gestiona Webhooks

**Esborrar webhook:**

1. Ves a **Configuració → API i Webhooks → Webhooks**
2. Troba el webhook que vols eliminar
3. Fes clic sobre el webhook
4. Fes clic a **Esborrar** i confirma a la finestra emergent

**Edita un Webhook:**

1. Fes clic sobre el webhook que vols modificar
2. Actualitza el URL o altres configuracions
3. Fes clic a **Desa** per aplicar els canvis
