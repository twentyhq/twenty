## Capacitats avançades de vendes

Els equips de GTM sovint necessiten capacitats avançades de vendes com la puntuació de leads, l'enriquiment de dades, el round robin, l'assignació de territoris, recordatoris automatitzats i seqüències de correu electrònic. Encara que aquestes no són funcionalitats integrades a Twenty, totes poden configurar-se i personalitzar-se segons les teves necessitats específiques utilitzant el nostre sistema flexible de fluxos de treball.

Visita la nostra [secció de fluxos de treball](/l/ca/user-guide/workflows/getting-started-workflows) per aprendre a construir aquestes automatitzacions pas a pas. Per a exemples detallats, consulta les nostres guies de [Automatitzacions Internes](/l/ca/user-guide/workflows/internal-automations) i [Integració d'Eines Externes](/l/ca/user-guide/workflows/external-tool-integration).
