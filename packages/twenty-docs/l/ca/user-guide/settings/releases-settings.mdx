## Últimes Versions

Segueix el progrés de desenvolupament del Twenty:

- **Notes de Versió**: Vegeu les novetats de cada versió
- **Actualitzacions de Funcions**: Descobriu noves capacitats i millores

## Funcions del Laboratori

Accediu a funcions experimentals abans que es llancin oficialment:

- **Prova Beta**: Proveu noves funcions mentre s'estan desenvolupant
- **Flags de Funcions**: Activa o desactiva funcionalitats experimentals específiques

## Com Accedir

1. Aneu a **Configuració → Versions**
2. Vegeu les notes de les versions i accediu a la pestanya **Laboratori** per a funcions beta

Per a més recursos comunitaris, consulteu la nostra [Secció de recursos](/l/ca/user-guide/resources/glossary).
