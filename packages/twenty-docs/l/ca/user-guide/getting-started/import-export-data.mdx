## Importa Dades

- You can import data for any object using a .csv, .xlsx, or .xls file.
- Cadascun dels arxius que carreguis ha de contenir **només un tipus d'objecte** (per exemple, només registres de Persones).
- You can use the Import to **create or update records**.

### Download a sample file to match the expected formatting

1. Vés a una vista amb l'objecte que estàs a punt d'importar.
2. Fes clic a la icona `⋮` a la part superior dreta i després selecciona `Importa registres`.
3. Click on `Download sample file`.

### Prepara el teu csv

A continuació hi ha alguns elements a comprovar abans de carregar el teu arxiu.

- Limita el nombre de registres a **10,000 per arxiu**.
- **Elimina duplicats** del teu arxiu.
  - La singularitat en `Persones` està establerta per defecte pel `id` i el `correu electrònic`. També tens la capacitat de definir camps personalitzats de `Persones` com a únics quan configures el teu model de dades.
  - La singularitat en `Empreses` està establerta per defecte pel `id` i el `domini`. També tens la capacitat de definir camps personalitzats de `Empreses` com a únics quan configures el teu model de dades.
  - Per a qualsevol altre objecte, incloent objectes personalitzats, tens la capacitat de definir alguns camps com a únics. Assegura't de no incloure duplicats quan preparis els teus arxius.
- Recomanem utilitzar la sintaxi `https://domini.com` quan carregues els teus dominis, ja que és el que s'utilitza pel nostre connector amb la teva bústia i calendari.
- Pots **importar les relacions entre objectes** proporcionant un dels camps únics del registre associat.
  - Exemple: Vols associar una persona a una empresa. Afegeix una columna a l'arxiu que contingui tots els registres de `Persones` que contengui el `id` de l'empresa -- o el seu `domini`. Podràs mapar aquest camp durant la càrrega.
    <Warning>

**Nota important:**

- Les relacions entre objectes a Twenty són "Un a Molts". Això significa que cada registre d'objecte A es pot associar a diversos registres d'objecte B. Però cada registre d'objecte B només pot pertànyer a un registre d'objecte A. Per exemple, una empresa pot estar associada a diverses persones. I una persona només pot pertànyer a una empresa.

- Per carregar relacions via la funció d'Importació, has de proporcionar el `id` (o qualsevol altre camp únic) de l'objecte adjunt en l'arxiu que contingui els registres en el costat "Molts" de la relació.  
  Per exemple, proporciones el `id` o `domini` de l'empresa quan carregues registres de persones. No proporciones el `id` de les persones (o `correu electrònic`) quan carregues l'arxiu amb empreses.
  </Warning>

### Carrega el teu arxiu

1. Vés a una vista amb l'objecte que estàs a punt d'importar.
2. Fes clic a la icona `⋮` a la part superior dreta i després selecciona `Importa registres`.
3. Fes clic a `Selecciona arxiu`.
4. Validate the mapping of the fields.

- You don't have to import all of them, you can choose the "Do not map" option.
- Per a relacions, és recomanable només mapejar un dels camps únics.
- You might need to also map the values of your select type and multi-select type fields.

5. Click on `Next Steps` and `Review the rows with errors`. Les cel·les amb un problema estan ressaltades. **Pots eliminar la fila o actualitzar la cel·la directament des d'allà**.
6. Un cop acabis, fes clic a `Confirmar`

<details><resum>Puc actualitzar registres existents amb la funció d'Importació?</resum>

Sí, pots actualitzar registres existents utilitzant la funció d'Importació. Assegura't de proporcionar el id (o qualsevol altre camp únic) quan tornis a carregar els teus registres.

</details>

<details><resum>Puc migrar el `id` des de les meves altres eines?</resum>

Sí. Cal crear un camp que defineixes com a únic en el teu model de dades que contindrà el `id` de les teves altres eines. Tingues en compte que el nom `id` està protegit ja que s'utilitza per al id de Twenty.  
Si vols crear relacions entre objectes utilitzant aquest camp, consulta la secció **Prepara el teu csv** més amunt en aquest article. Conté una secció sobre la importació de relacions.

</details>

## Exporta Dades

Pots descarregar dades de la majoria dels teus objectes i fins a 20,000 registres per exportació.
Per exportar dades d'un objecte:

1. Visita l'índex de l'objecte.
2. Trieu la vista per exportar dades. Configura les columnes que vols descarregar amagant o afegint columnes i troba els registres que necessites filtrant la teva vista.
3. Accedeix al panell lateral a través de la icona `⋮` a la part superior dreta.
4. Fes clic a `Exporta vista`.
5. Selecciona la ubicació de desament per a les dades CSV. Nota que l'exportació pot tardar temps si hi ha molts registres.
