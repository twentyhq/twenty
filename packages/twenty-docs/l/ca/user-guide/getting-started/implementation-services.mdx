## Serveis d'implementació

Si necessites ajuda per començar o per crear personalitzacions avançades, tenim una solució.

### Packs d'Integració

Get help from our core team to set up your Twenty workspace with our 4-hour [Onboarding packs](https://twenty.com/onboarding-packages):

- **Disseny de model de dades**: Dissenya i crea el teu model de dades personalitzat amb objectes, camps i relacions
- **Migració de dades**: Migra les teves dades existents del teu CRM actual a Twenty
- **Creació de fluxos de treball**: Crea fluxos de treball personalitzats per donar suport als teus processos empresarials

### Socis d'implementació

Treballar amb socis certificats de Twenty per a personalitzacions i integracions més avançades. Poseu-vos en contacte amb el nostre equip a través de contact@twenty.com per ser emparellat amb els nostres [socis](https://twenty.com/implementation-services).
