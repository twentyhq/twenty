## Pas 1: Registre

1. Navega fins a [Inscripció a Twenty](https://app.twenty.com).
2. Selecciona el mètode de registre desitjat:
   - **Continua amb Google** per a la registració amb el compte de Google.
   - **Continua amb Microsoft** per a la registració amb el compte de Microsoft.
   - O, **Continua amb el correu electrònic** per a la registració per correu electrònic.

## Step 2: Choosing a Trial Period

Tria entre dos períodes de prova:

### 30 dies

Amb targeta de crèdit

### 7 dies

Sense targeta de crèdit

Ambdós períodes de prova inclouen:

- Accés complet
- Contactes il·limitats
- Integració de correu electrònic
- Objectes personalitzats
- API i Webhooks

Pots fer clic a "Canviar de pla" per triar un pla o interval de facturació diferent.

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## Pas 3: Confirmació de pagament i configuració del compte

Després de l'aprovació del pagament a través de Stripe, se't dirigirà a crear el teu espai de treball i perfil d'usuari. Recorda que pots cancel·lar la subscripció en qualsevol moment.
