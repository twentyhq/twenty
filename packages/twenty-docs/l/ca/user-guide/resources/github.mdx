## Sobre

El repositori de Twenty a GitHub allotja una àmplia gamma de recursos com codi font, documentació, discussions i seguiment d'incidències. This is where you will be able to access the full code behind Twenty.

[Visita Twenty a GitHub](https://github.com/twentyhq/twenty)

## Contribució

Contribuir al projecte Twenty a GitHub és una manera gratificant d'ajudar a millorar el programari que utilitzes. Ja sigui corregint errors, suggerint funcions o millorant la documentació, les teves contribucions són benvingudes.

### Reportar problemes

Has trobat un problema? [Create an issue](https://github.com/twentyhq/twenty/issues/new) on GitHub, providing as much detail as possible.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggerir funcions

Quines millores t'agradaria veure a Twenty? Independentment dels teus coneixements tècnics, pots unir-te a [la conversa aquí](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Coding a feature

Comença el teu viatge trobant tasques adaptades als principiants:

1. Navega fins a la pestanya **[Issues](https://github.com/twentyhq/twenty/issues)** al repositori de Twenty.
2. Filtra pel label **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** per trobar tasques adequades per a nouvinguts.
3. Pick an issue, fork the repository, and start contributing.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Assegura't d'estar assignat al problema per evitar treballs superposats amb altres col·laboradors.

### Codi de Conducta

Recorda seguir el [Codi de Conducta de Twenty](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) durant tot el procés de contribució.
