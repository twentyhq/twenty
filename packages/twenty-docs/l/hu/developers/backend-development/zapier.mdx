Egyszerűen szinkronizáld a Twenty-t több mint 3000 alkalmazással a [Zapier](https://zapier.com/) segítségével. Automatizálj feladatokat, növeld a termelékenységet, és turbózd fel vevőkapcsolataidat!

## A Zapier-ről

A Zapier egy olyan eszköz, amely lehetővé teszi a munkafolyamatok automatizálását azáltal, hogy összekapcsolja a csapatod által napi szinten használt alkalmazásokat. The fundamental concept of Zapier is automation workflows, called Zaps, and include triggers and actions.

Többet is megtudhatsz a Zapier működéséről [itt](https://zapier.com/how-it-works).

## Beállítás

### 1. lépés: A Zapier csomagok telepítése

```bash
cd packages/twenty-zapier

yarn
```

### 2. lépés: Bejelentkezés a CLI-vel

Használd a Zapier hitelesítő adataidat a CLI-n keresztüli bejelentkezéshez:

```bash
zapier login
```

### 3. lépés: Környezeti változók beállítása

A `packages/twenty-zapier` könyvtárban futtasd:

```bash
cp .env.example .env
```

Futtasd a helyi alkalmazást, menj a [http://localhost:3000/settings/api-webhooks](http://localhost:3000/settings/api-webhooks) címre, és hozz létre egy API kulcsot.

Cseréld ki a **YOUR_API_KEY** értéket a `.env` fájlban az általad generált API kulcsra.

## Fejlesztés

<Warning>

Győződj meg róla, hogy futtattad a `yarn build` parancsot bármely `zapier` parancs előtt.

</Warning>

### Teszt

```bash
yarn test
```

### Lint

```bash
yarn format
```

### Nézd és fordítsd le, ahogy a kódot szerkeszted

```bash
yarn watch
```

### A Zapier alkalmazásod validálása

```bash
yarn validate
```

### A Zapier alkalmazásod telepítése

```bash
yarn deploy
```

### Minden Zapier CLI parancs megtekintése

```bash
zapier
```
