## What's Coming

### Egyéni Jelentések

Hozzon létre testre szabott jelentéseket az Ön üzletének legfontosabb mutatóinak nyomon követéséhez.

### Értékesítési Irányítópultok

Vizualizálja az értékesítési csatorna teljesítményét interaktív irányítópultok segítségével.

### Teljesítményanalitika

Figyelje a csapat teljesítményét, és azonosítsa a trendeket az ügyféladataiban.

## Stay Updated

Kövesse GitHub tárolónkat vagy ellenőrizze a **Beállítások → Kiadások** részt a munkaterületén, hogy értesítést kapjon, amikor a jelentési funkciók elérhetővé válnak.
