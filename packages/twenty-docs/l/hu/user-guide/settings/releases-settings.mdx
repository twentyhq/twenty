## Legújabb kiadások

Kövesd a Twenty fejlesztési haladását:

- **Kiadási jegyzetek**: Lásd, mi újdonság az egyes verziókban
- **Funkció frissítések**: Fedezd fel az új képességeket és fejlesztéseket

## Labor funkciók

Férj hozzá kísérleti funkciókhoz, mielőtt hivatalosan megjelennének:

- **Béta tesztelés**: Próbáld ki az új funkciókat, miközben fejlesztés alatt állnak
- **Feature Flags**: Enable or disable specific experimental functionality

## Hogyan férj hozzá

1. Menj a **Beállítások → Kiadások** részébe
2. View release notes and access the **Lab** tab for beta features

További közösségi forrásokért nézd meg a [Források szekciót](/l/hu/user-guide/resources/glossary).
