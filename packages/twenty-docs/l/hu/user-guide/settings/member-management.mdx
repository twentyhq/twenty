## Új tagok meghívása

### A meghívó link használata

1. Go to **Settings → Members**
2. Másolja a munkaterület meghívó linkjét
3. Ossza meg a linket az új csapattagokkal
4. Amint regisztrálnak, hozzáférést kapnak

### Közvetlen e-mail meghívó

1. Go to **Settings → Members**
2. Adja meg a személy e-mail címét
3. Kattintson az **Meghívás** gombra
4. E-mailben kapnak meghívót

## Tagok eltávolítása

### Tag törlése

1. Go to **Settings → Members**
2. Keresse meg az eltávolítandó tagot
3. Kattintson a törlés/eltávolítás gombra a nevük mellett
4. Erősítse meg az eltávolítást

**Megjegyzés**: Az eltávolított tagok azonnal elveszítik a hozzáférést, de később újra meghívhatók.
