## 1. lépés: Regisztráció

1. Nyissa meg a [Twenty Sign Up](https://app.twenty.com) oldalt.
2. Válassza ki az Ön számára legmegfelelőbb regisztrációs módot:
   - **Google-val folytatás** a Google-fiók regisztrációhoz.
   - **Microsoft-tal folytatás** a Microsoft-fiók regisztrációhoz.
   - Or, **Continue With Email** for email registration.

- Egyéni objektumok
- API & Webhooks

## 3. lépés: Fizetés megerősítése és fiók beállítása

A fizetés Stripe-on keresztül történő jóváhagyása után vezetünk át a munkaterület és a felhasználói profil létrehozásához. Ne feledje, hogy bármikor lemondhatja előfizetését.
