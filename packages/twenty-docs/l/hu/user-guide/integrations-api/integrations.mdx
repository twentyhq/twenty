## Jelenlegi Integrációs Opciók

### Natív Integrációk

A Twenty jelenleg natív integrációt kínál a következőkkel:

- **E-mail & Naptár**: Csatlakoztassa a Gmail, Outlook, vagy SMTP/CalDAV szolgáltatókat
- **API Hozzáférés**: Használja REST és GraphQL API-inkat egyedi integrációk létrehozásához

E-mail és naptár beállításhoz látogasson el az [E-mail & Naptár Beállítás](user-guide/settings/email-calendar-setup) oldalra.

### Munkafolyamatok (Ajánlott)

A Twenty más eszközökhöz való csatlakoztatásának elsődleges módja az alkalmazásban található **Munkafolyamatok** funkció használata:

- **HTTP Csomópontok**: API hívásokat kezdeményezhet külső szolgáltatásokhoz
- **Code Nodes**: Write custom logic for complex integrations
- **Webhook Trigger-ek**: Adatok fogadása külső rendszerekből
- **Mezőfrissítési Trigger-ek**: Automatizálható műveletek CRM módosítások alapján

További információ: [Munkafolyamatok részlegünkben](/l/hu/user-guide/workflows/getting-started-workflows).

### Zapier Integráció (Régies)

Fenntartunk egy Zapier integrációt azoknak a felhasználóknak, akik kód nélküli automatizálást kedvelnek:

1. Látogassa meg a [Twenty a Zapieren](https://zapier.com/apps/twenty/integrations) oldalt
2. Create a new Zap with Twenty as trigger or action
3. Generáljon API kulcsot a Beállítások → API & Webhooks oldalon
4. Csatlakoztassa a Twenty munkaterületet a Zapierhez

## Jövőkép: Közösség által készített Csatlakozók (2026)

### Kiterjeszthetőség Platformja

Kiterjeszthetőség platformot építünk, amely lehetővé teszi, hogy fejlesztők appokat készítsenek kódként. Ez lehetővé teszi:

- **Egyedi Csatlakozók**: Integrációk létrehozása a kedvenc szoftvereivel
- **Közösségi Hozzájárulások**: Ossza meg és fedezze fel, amit mások építettek
- **Rugalmas Architektúra**: A Twenty képességeinek kiterjesztése a főbb funkciókon túl

### MI-által segített Munkafolyamat Építés

2026-ban jön, az MI segítségével a felhasználók:

- **Automatikus Munkafolyamat Generálás**: Írja le integrációs igényeit egyszerű nyelvezettel
- **Intelligens Javaslatok**: Kap ajánlásokat, hogyan kapcsolja össze speciális eszközeit
- **Sablon Könyvtár**: Előre elkészített munkafolyamatok elérése gyakori felhasználási esetekhez

A Twenty integrációk jövője közösség által vezérelt, MI-által segített és végtelenül kiterjeszthető.
