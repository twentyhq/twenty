Elem a tartalom vizuális kategorizálásához vagy címkézéséhez.

<Tabs>

<Tab title="Usage">

```jsx
import { Tag } from "@/ui/display/tag/components/Tag";

export const MyComponent = () => {
  return (
    <Tag
      className
      color="red"
      text="Urgent"
      onClick={() => console.log("click")}
    />
  );
};
```

</Tab>

<Tab title="Props">

| Tulajdonságok | Típus    | Leírás                                                                                                                                                   |
| ------------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| className     | string   | Opcionális név a további stílushoz                                                                                                                       |
| szín          | string   | A címke színe. Options include: `green`, `turquoise`, `sky`, `blue`, `purple`, `pink`, `red`, `orange`, `yellow`, `gray` |
| szöveg        | string   | A címke tartalma                                                                                                                                         |
| kattintás     | függvény | Opcionális függvény, amely meghívódik, amikor a felhasználó a címkére kattint                                                                            |

</Tab>

</Tabs>
