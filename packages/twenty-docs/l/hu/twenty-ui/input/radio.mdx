Akkor használható, ha a felhasználók csak egy opciót választhatnak egy sor lehetőség közül.

<Tabs>
<Tab title="Usage">

```jsx
import { Radio } from "twenty-ui/display";

export const MyComponent = () => {

  const handleRadioChange = (event) => {
    console.log("Radio button changed:", event.target.checked);
  };

  const handleCheckedChange = (checked) => {
    console.log("Checked state changed:", checked);
  };


  return (
    <Radio
      checked={true}
      value="Option 1"
      onChange={handleRadioChange}
      onCheckedChange={handleCheckedChange}
      size="large"
      disabled={false}
      labelPosition="right"
    />
  );
};

```

</Tab>
<Tab title="Props">

| Tulajdonságok   | Típus                     | Leírás                                                                                                                             |
| --------------- | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| stílus          | `React.CSS` tulajdonságok | További inline stílusok a komponenshez                                                                                             |
| className       | string                    | Opcionális CSS osztály további stílushoz                                                                                           |
| checked         | boolean                   | Jelzi, hogy a rádiógomb be van-e jelölve                                                                                           |
| érték           | string                    | A rádiógombhoz társított címke vagy szöveg                                                                                         |
| onChange        | függvény                  | A funkció, amely akkor hívódik meg, amikor a kiválasztott rádiógomb változik                                                       |
| onCheckedChange | függvény                  | A funkció, amely akkor hívódik meg, amikor a rádiógomb `ellenőrzött` állapota változik                                             |
| méret           | string                    | A rádiógomb mérete. Opciók közé tartozik: `nagy` és `kicsi`                                        |
| letiltva        | boolean                   | If `true`, the radio button is disabled and not clickable                                                                          |
| címkePozíció    | string                    | A címke szövegének pozíciója a rádiógombhoz viszonyítva. Két opcióval rendelkezik: `bal` és `jobb` |

</Tab>
</Tabs>

## Rádiócsoport

Kapcsolódó rádiógombokat foglal össze.

<Tabs>
<Tab title="Usage">

```jsx
import React, { useState } from "react";
import { Radio, RadioGroup } from "twenty-ui/display";

export const MyComponent = () => {

  const [selectedValue, setSelectedValue] = useState("Option 1");

  const handleChange = (event) => {
    setSelectedValue(event.target.value);
  };
  
  return (
    <RadioGroup value={selectedValue} onChange={handleChange}>
      <Radio value="Option 1" />
      <Radio value="Option 2" />
      <Radio value="Option 3" />
    </RadioGroup>
  );
};

```

</Tab>
<Tab title="Props">

| Tulajdonságok | Típus             | Leírás                                                                                                                 |
| ------------- | ----------------- | ---------------------------------------------------------------------------------------------------------------------- |
| érték         | string            | Az éppen kiválasztott rádiógomb értéke                                                                                 |
| onChange      | függvény          | A visszahívási funkció, amely kiváltódik, amikor a rádiógomb megváltozik                                               |
| onValueChange | függvény          | A visszahívási funkció, amely kiváltódik, amikor a csoportban kiválasztott érték változik.             |
| gyermekek     | `React.ReactNode` | Lehetővé teszi, hogy React komponenseket (mint például Rádió) adjon át gyermekként a rádiócsoportba |

</Tab>

</Tabs>
