Navigációs sávot renderel, amely több `NavigationBarItem` komponenst tartalmaz.

<Tabs>
<Tab title="Usage">

```jsx
import { IconHome, IconUser, IconSettings } from '@tabler/icons-react';
import { NavigationBar } from "@/ui/navigation/navigation-bar/components/NavigationBar";

export const MyComponent = () => {

   const navigationItems = [
     {
       name: "Home",
       Icon: IconHome,
       onClick: () => console.log("Home clicked"),
     },
     {
       name: "Profile",
       Icon: IconUser,
       onClick: () => console.log("Profile clicked"),
     },
     {
       name: "Settings",
       Icon: IconSettings,
       onClick: () => console.log("Settings clicked"),
     },
   ];

  return <NavigationBar activeItemName="Home" items={navigationItems}/>;
};
```

</Tab>
<Tab title="Props">

| Tulajdonságok | Típus  | Leírás                                                                                                                                                                                                                                       |
| ------------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| aktívElemNeve | string | Az aktuálisan aktív navigációs elem neve                                                                                                                                                                                                     |
| elemek        | tömb   | A navigációs elemeket jelképező objektumok tömbje. Minden objektum tartalmazza az elem `nevét`, a megjelenítendő `Ikon` komponenst, és egy `onClick` függvényt, amely hívódik, amikor az elemre kattintanak. |

</Tab>
