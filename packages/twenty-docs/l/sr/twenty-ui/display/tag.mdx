Компонента за визуелну категоризацију или означавање садржаја.

<Tabs>

<Tab title="Usage">

```jsx
import { Tag } from "@/ui/display/tag/components/Tag";

export const MyComponent = () => {
  return (
    <Tag
      className
      color="red"
      text="Urgent"
      onClick={() => console.log("click")}
    />
  );
};
```

</Tab>

<Tab title="Props">

| Својства  | Тип      | Опис                                                                                                                                                        |
| --------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| className | низ      | Опционо име за додатно обликовање                                                                                                                           |
| боја      | низ      | Color of the tag. Options include: `green`, `turquoise`, `sky`, `blue`, `purple`, `pink`, `red`, `orange`, `yellow`, `gray` |
| текст     | низ      | The content of the tag                                                                                                                                      |
| приКлику  | функција | Optional function called when a user clicks on the tag                                                                                                      |

</Tab>

</Tabs>
