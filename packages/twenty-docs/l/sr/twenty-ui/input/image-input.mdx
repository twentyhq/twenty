Омогућава корисницима да отпреме и уклоне слику.

<Tabs>
<Tab title="Usage">

```jsx
import { ImageInput } from "@/ui/input/components/ImageInput";

export const MyComponent = () => {
  return <ImageInput/>;
};
```

</Tab>
<Tab title="Props">

| Својства     | Тип               | Опис                                                                                                              |
| ------------ | ----------------- | ----------------------------------------------------------------------------------------------------------------- |
| слика        | низ               | Изворна URL адреса слике                                                                                          |
| onUpload     | функција          | Функција која се позива када корисник отпреми нову слику. Прима објекат типа `File` као параметар |
| onRemove     | функција          | Функција која се позива када корисник кликне на дугме за уклањање                                                 |
| onAbort      | функција          | Функција која се позива када корисник кликне на дугме за прекид током отпремања слике                             |
| isUploading  | булевска вредност | Indicates whether an image is currently being uploaded                                                            |
| errorMessage | низ               | Опциона порука о грешци за приказ испод уноса слике                                                               |
| онемогућено  | булевска вредност | Ако је `true`, цео унос је онемогућен и дугмићи се не могу кликнути                                               |

</Tab>
