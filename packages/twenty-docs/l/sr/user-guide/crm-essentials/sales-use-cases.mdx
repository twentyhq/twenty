## Напредне продајне могућности

GTM тимови често захтевају напредне продајне могућности као што су оцењивање потенцијалних клијената, обогаћивање података, ротирање, додељивање по територији, аутоматски подсетници и секвенце емаилова. While these aren't built-in features in Twenty, they can all be configured and tailored to your specific needs using our flexible workflow system.

Visit our [Workflows section](/l/sr/user-guide/workflows/getting-started-workflows) to learn how to build these automations step by step. For detailed examples, see our [Internal Automations](/l/sr/user-guide/workflows/internal-automations) and [External Tool Integration](/l/sr/user-guide/workflows/external-tool-integration) guides.
