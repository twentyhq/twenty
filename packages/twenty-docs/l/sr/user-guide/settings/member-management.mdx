## Invite New Members

### Using the Invite Link

1. Идите на **Подешавања → Чланови**
2. Copy the workspace invite link
3. Делите линк са новим члановима тима
4. Они ће добити приступ када се пријаве

### Директна позивница путем е-поште

1. Идите на **Подешавања → Чланови**
2. Унесите адресу е-поште те особе
3. Кликните на **Позови**
4. Они ће добити позивницу путем е-поште

## Уклоните чланове

### Избришите члана

1. Идите на **Подешавања → Чланови**
2. Find the member you want to remove
3. Кликните на дугме за брисање/уклањање поред њиховог имена
4. Потврдите уклањање

**Напомена**: Уклоњени чланови одмах губе приступ али им се може поново послати позивница касније.

## Потребна вам је помоћ?

За управљање улогама и дозволама, проверите [чланак о дозволама](/l/sr/user-guide/settings/permissions).
