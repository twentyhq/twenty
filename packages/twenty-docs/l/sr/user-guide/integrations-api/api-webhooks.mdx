## API Кључеви

АПИ кључеви омогућавају аутоматизован приступ подацима у вашем ЦРМ-у, синхронизацију података са другим системима и креирање прилагођених интеграција или решења.

### Креирајте АПИ кључ

1. Go to **Settings → APIs & Webhooks**
2. Кликните **+ Креирај кључ** у горњем десном углу
3. Конфигуришите свој АПИ кључ:
   - **Назив**: Дајте свом АПИ кључу описни назив
   - **Датум истека**: Подесите када кључ треба да истекне
4. Кликните **Сачувај** да бисте генерисали свој АПИ кључ
5. **Важно**: Одмах копирајте и сачувајте свој АПИ кључ, показује се само једном

Када је креиран, ваш АПИ кључ омогућава приступ вашој прилагођеној АПИ документацији и радном окружењу где можете тестирати крајње тачке са вашим стварним подацима.

<Warning>
Пошто ваш АПИ кључ омогућава приступ осетљивим информацијама, не би требало да га делите са услугама којима не верујете у потпуности. Ако процури, неко га може злоупотребити. Ако је безбедност вашег АПИ кључа компромитована, одмах га онемогућите и генеришите нови.
</Warning>

### Управљање АПИ кључевима

**Регенеришити АПИ кључ:**

1. Go to **Settings → APIs & Webhooks**
2. Кликните на АПИ кључ који желите да регенеришете
3. Кликните на дугме **Регенериши**
4. Копирајте и одмах сачувајте нови API кључ

**Обриши API кључ:**

1. Пронађите API кључ на вашој листи
2. Кликните на кључ да бисте отворили његове детаље
3. Кликните на **Обриши** да га трајно уклоните

## Вебхукс

Вебхукс омогућавају тренутна ажурирања вашој специфичној URL адреси о променама или догађајима у вези са подацима о вашим клијентима.

For example, when an Opportunity moves to "Closed Won", a webhook can automatically trigger invoice creation in your accounting system. Note that this type of automation can also be achieved using Twenty's in-app [Workflows feature](/l/sr/user-guide/workflows/getting-started-workflows), which offers triggers based on field updates for internal automation.

Webhooks are ideal for integrating with external systems, while Workflows support both internal automation and external tool connections via webhook triggers, code nodes, and HTTP nodes.

### Креирајте Вебхук

1. Go to **Settings → APIs & Webhooks → Webhooks**
2. Кликните на **+ Креирајте вебхук**
3. Унесите ваш вебхук URL (где желите да примате обавештења)
4. Кликните на **Сачувај**

Ваш вебхук ће одмах почети да прима реал-тайм обавештења о променама ваших CRM података.

### Управљање Вебхуксима

**Delete a Webhook:**

1. Go to **Settings → APIs & Webhooks → Webhooks**
2. Пронађите вебхук који желите уклонити
3. Кликните на вебхук
4. Кликните на **Обриши** и потврдите у попапу

**Измените вебхук:**

1. Кликните на вебхук који желите изменити
2. Ажурирајте URL или друга подешавања
3. Кликните на **Сачувај** за примену измена
