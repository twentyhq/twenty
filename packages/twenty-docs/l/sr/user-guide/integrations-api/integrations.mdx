## Текуће опције интеграције

### Native Integrations

Twenty currently offers native integration with:

- **Е-пошта и календар**: Повежите Gmail, Outlook или SMTP/CalDAV провајдере
- **Приступ API-ју**: Користите наше REST и GraphQL API-е за изградњу прилагођених интеграција

За подешавање е-поште и календара посетите [Подешавање е-поште и календара](/l/sr/user-guide/settings/email-calendar-setup).

### Workflows (Recommended)

Примарни начин повезивања Twenty са другим алаткама је кроз нашу унутрашњу функцију **Радни процеси**:

- **HTTP чворови**: Пошаљите API позиве спољашњим сервисима
- **Код чворови**: Напишите прилагођену логику за сложене интеграције
- **Webhook Triggers**: Receive data from external systems
- **Подстицање ажурирања поља**: Аутоматизујте акције на основу промена у CRM-у

Learn more in our [Workflows section](/l/sr/user-guide/workflows/getting-started-workflows).

### Интеграција са Запиером (Наслеђено)

Одржавамо интеграцију са Запиером за кориснике који преферирају аутоматизацију без кода:

1. Посетите [Twenty на Запиеру](https://zapier.com/apps/twenty/integrations)
2. Креирајте нови ЗАП са Twenty као окидачем или акцијом
3. Генеришите API кључ у Подешавањима → API & Webhooks
4. Повежите ваш Twenty простор са Запиером

## Будућа визија: Конектори изграђени од стране заједнице (2026)

### Платформа за проширивост

Градимо платформу за проширивост која ће омогућити развојним радницима да креирају апликације као код. Ово ће омогућити:

- **Прилагођени конектори**: Изградите интеграције са вашим омиљеним софтвером
- **Доприноси заједнице**: Делите и откривајте конекторе које су изградили други
- **Флексибилна архитектура**: Проширите могућности Twenty изван основних функција

### Изградња радних процеса уз помоћ вештачке интелигенције

Coming in 2026, AI assistance will help users:

- **Аутоматско генерисање радних процеса**: Опишите ваше интеграционе потребе у простом језику
- **Паметне препоруке**: Добити препоруке за повезивање специфичних алатки
- **Библиотека шаблона**: Приступите унапред изграђеним радним процесима за уобичајене случајеве коришћења

Будућност Twenty интеграција је вођена заједницом, асистирана вештачком интелигенцијом и бесконачно проширујућа.
