Käytetään, kun käyttäjät voivat valita vain yhden vaihtoehdon useista vaihtoehdoista.

<Tabs>
<Tab title="Usage">

```jsx
import { Radio } from "twenty-ui/display";

export const MyComponent = () => {

  const handleRadioChange = (event) => {
    console.log("Radio button changed:", event.target.checked);
  };

  const handleCheckedChange = (checked) => {
    console.log("Checked state changed:", checked);
  };


  return (
    <Radio
      checked={true}
      value="Option 1"
      onChange={handleRadioChange}
      onCheckedChange={handleCheckedChange}
      size="large"
      disabled={false}
      labelPosition="right"
    />
  );
};

```

</Tab>
<Tab title="Props">

| Ominaisuudet    | Tyyppi                   | Kuvaus                                                                                                                                 |
| --------------- | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| tyyli           | `React.CSS` ominaisuudet | Lisätyylit komponentille                                                                                                               |
| className       | string                   | Valinnainen CSS-luokka lisätyylitykseen                                                                                                |
| valittu         | looginen                 | Indicates whether the radio button is checked                                                                                          |
| arvo            | string                   | Radiopainikkeeseen liittyvä etiketti tai teksti                                                                                        |
| onChange        | funktio                  | Funktio, joka kutsutaan, kun valittu radiopainike vaihtuu                                                                              |
| onCheckedChange | toiminto                 | Funktio, joka kutsutaan, kun radiopainikkeen `valittu`-tila muuttuu                                                                    |
| koko            | merkkijono               | Radiopainikkeen koko. Vaihtoehdot sisältävät: `iso` ja `pieni`                                         |
| pois käytöstä   | looginen                 | Jos `true`, radiopainike on poissa käytöstä eikä ole napsautettavissa                                                                  |
| labelPosition   | string                   | Etiketin tekstin sijainti suhteessa radiopainikkeeseen. Sisältää kaksi vaihtoehtoa: `vasen` ja `oikea` |

</Tab>
</Tabs>

## Radio Group

Groups together related radio buttons.

<Tabs>
<Tab title="Usage">

```jsx
import React, { useState } from "react";
import { Radio, RadioGroup } from "twenty-ui/display";

export const MyComponent = () => {

  const [selectedValue, setSelectedValue] = useState("Option 1");

  const handleChange = (event) => {
    setSelectedValue(event.target.value);
  };
  
  return (
    <RadioGroup value={selectedValue} onChange={handleChange}>
      <Radio value="Option 1" />
      <Radio value="Option 2" />
      <Radio value="Option 3" />
    </RadioGroup>
  );
};

```

</Tab>
<Tab title="Props">

| Rekvisiitat   | Tyyppi            | Kuvaus                                                                                              |
| ------------- | ----------------- | --------------------------------------------------------------------------------------------------- |
| arvo          | string            | Tällä hetkellä valitun radiopainikkeen arvo                                                         |
| onChange      | funktio           | Palautekutsu, joka laukaisee, kun radiopainike vaihtuu                                              |
| onValueChange | toiminto          | Palautekutsu, joka laukaisee, kun ryhmän valittu arvo muuttuu.                      |
| children      | `React.ReactNode` | Mahdollistaa React-komponenttien (kuten Radio) siirtämisen lapsina Radioryhmälle |

</Tab>

</Tabs>
