Käyttää lohkopohjaista rikasta tekstieditoria [BlockNote](https://www.blocknotejs.org/), jonka avulla käyttäjät voivat muokata ja tarkastella sisältölohkoja.

<Tabs>
<Tab title="Usage">

```jsx
import { useBlockNote } from "@blocknote/react";
import { BlockEditor } from "@/ui/input/editor/components/BlockEditor";

export const MyComponent = () => {
  const BlockNoteEditor = useBlockNote();

  return <BlockEditor editor={BlockNoteEditor} />;
};
```

</Tab>
<Tab title="Props">

| Ominaisuudet | Tyyppi            | Kuvaus                                    |
| ------------ | ----------------- | ----------------------------------------- |
| editori      | `BlockNoteEditor` | Lohkoeditorin instanssi tai konfiguraatio |
