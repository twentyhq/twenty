Näyttää navigointipalkin, joka sisältää useita `NavigationBarItem`-komponentteja.

<Tabs>
<Tab title="Usage">

```jsx
import { IconHome, IconUser, IconSettings } from '@tabler/icons-react';
import { NavigationBar } from "@/ui/navigation/navigation-bar/components/NavigationBar";

export const MyComponent = () => {

   const navigationItems = [
     {
       name: "Home",
       Icon: IconHome,
       onClick: () => console.log("Home clicked"),
     },
     {
       name: "Profile",
       Icon: IconUser,
       onClick: () => console.log("Profile clicked"),
     },
     {
       name: "Settings",
       Icon: IconSettings,
       onClick: () => console.log("Settings clicked"),
     },
   ];

  return <NavigationBar activeItemName="Home" items={navigationItems}/>;
};
```

</Tab>
<Tab title="Props">

| Ominaisuudet          | Tyyppi   | Kuvaus                                                                                                                                                                                                                                        |
| --------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| aktiivisenKohteenNimi | string   | Tällä hetkellä aktiivisen navigointikohteen nimi                                                                                                                                                                                              |
| kohteet               | taulukko | Taulukko objekteista, jotka edustavat kutakin navigointikohdetta. Jokainen objekti sisältää kohteen `nimen`, näytettävän `Icon`-komponentin, ja `onClick`-funktion, joka suoritetaan, kun kohdetta klikataan. |

</Tab>
