## Nykyiset integraatiovaihtoehdot

### Native Integrations

Twenty currently offers native integration with:

- **Sähköposti & Kalenteri**: Yhdistä Gmail, Outlook tai SMTP/CalDAV-palveluntarjoajat
- **API Access**: Use our REST and GraphQL APIs to build custom integrations

For email and calendar setup, visit [Email & Calendar Setup](/l/fi/user-guide/settings/email-calendar-setup).

### Työnkulut (Suositeltu)

The primary way to connect Twenty to other tools is through our in-app **Workflows** feature:

- **HTTP Solmut**: Tee API-pyyntöjä ulkoisiin palveluihin
- **Code Nodes**: Write custom logic for complex integrations
- **Webhook Lähdöt**: Vastaanota dataa ulkoisista järjestelmistä
- **Field Update Triggers**: Automate actions based on CRM changes

Lisätietoja [työnkulut-osiossa](/l/fi/user-guide/workflows/getting-started-workflows).

### Zapier Integration (Legacy)

Ylläpidämme Zapier-integraatiota käyttäjille, jotka haluavat koodittoman automatisoinnin:

1. Käy [Twenty Zapierissä](https://zapier.com/apps/twenty/integrations)
2. Create a new Zap with Twenty as trigger or action
3. Luo API-avain kohdassa Asetukset → API & Webhooks
4. Yhdistä Twenty-työtilasi Zapieriin

## Tulevaisuuden visio: Yhteisön luomat liittimet (2026)

### Laajennettavuusalusta

Rakennamme laajennettavuusalustaa, jonka avulla kehittäjät voivat luoda sovelluksia koodina. Tämä mahdollistaa:

- **Mukautetut Liittimet**: Luo integrointeja suosikki ohjelmistoihisi
- **Community Contributions**: Share and discover connectors built by others
- **Flexible Architecture**: Extend Twenty's capabilities beyond core features

### AI-avusteinen työnkulun rakentaminen

Tulossa vuonna 2026, AI auttaa käyttäjiä:

- **Työnkulkujen Automaattinen Luonti**: Kuvaile integraatiotarpeesi yksinkertaisella kielellä
- **Älykkäät Suositukset**: Saat suosituksia erityisten työkalujesi yhdistämiseksi
- **Mallikirjasto**: Pääsy ennalta rakennettuihin työnkulkuihin yleisiin käyttötapauksiin

Twentyn integraatioiden tulevaisuus on yhteisön vetämä, AI-avusteinen ja lukemattoman laajennettavissa.
