## Kehittyneet myyntiominaisuudet

GTM-tiimit tarvitsevat usein kehittyneitä myyntiominaisuuksia kuten liidien pisteytystä, tietojen rikastamista, vuorottelua, aluejakoa, automaattisia muistutuksia ja sähköpostijonoja. While these aren't built-in features in Twenty, they can all be configured and tailored to your specific needs using our flexible workflow system.

Vieraile [Työnkulkujen osiossa](/l/fi/user-guide/workflows/getting-started-workflows) oppiaksesi rakentamaan nämä automaatiot vaihe vaiheelta. For detailed examples, see our [Internal Automations](/l/fi/user-guide/workflows/internal-automations) and [External Tool Integration](/l/fi/user-guide/workflows/external-tool-integration) guides.
