## Hinnoittelu

<AccordionGroup>
<Accordion title="Can I use Twenty for free while self-hosting?">
Kyllä, voit käyttää Twentyä ilmaiseksi itse isännöimällä. Saat pääsyn kaikkiin (pilvi) Pro -tilaukseen sisältyviin ominaisuuksiin, paitsi ydintiimimme tukeen. Tuki on saatavilla Discord-yhteisömme kautta.

If you want to self-host and have access to the features included in the (cloud) Organization subscription, including support from our core-team, you can do so by choosing the paid Organization licences, available in self-hosting. </Accordion>

<Accordion title="Do you offer free seats for view-only users?">
Emme tarjoa ilmaisia paikkoja. Pricing is per user and each user needs a licence to access Twenty.
</Accordion>

<Accordion title="Where can I switch my subscription to the Organization plan?">
Voit tehdä tämän kohdassa `Asetukset → Laskutus`. Napsauta sitten `Vaihda Organisaatioon`.
</Accordion>

<Accordion title="Where can I switch my subscription to the Pro plan?">
Please reach out to our team directly via the Support, there is no easy way to do this using the UI at the moment.
</Accordion>

<Accordion title="Where can I switch my subscription to Yearly?">
Voit tehdä tämän kohdassa `Asetukset → Laskutus`. Napsauta sitten `Vaihda Vuosiksi`.
</Accordion>

<Accordion title="Where can I switch my subscription to Monthly?">
Please reach out to our team directly via the Support, there is no easy way to do this using the UI at the moment.
</Accordion>

<Accordion title="Where can I see my credit usage?">
You will find this under `Settings → Billing`.
</Accordion>

<Accordion title="How many workflow credits do we have?">
Luottojen määrä vaihtelee suunnitelman mukaan. Kokeilusopimus tarjoaa 5 miljoonaa luottoa, Pro-suunnitelma antaa 10 miljoonaa luottoa kuukaudessa, ja Organisaatio-suunnitelma antaa 20 miljoonaa luottoa kuukaudessa.
</Accordion>

<Accordion title="How does workflow credit consumption work?">
Each workflow action consumes credits based on its complexity:
- **Basic internal operations** (such as search, update, create records) consume very few credits
- **More complex operations** like code nodes and requests to external services consume more credits
- **AI prompts** (coming soon!) kuluttavat myös enemmän luottoja käytön perusteella

Luottoja vähennetään reaaliajassa, kun työnkulut suoritetaan. Voit seurata käyttöäsi kohdassa **Asetukset → Laskutus** saadaksesi selville kulutuksen ja jäljellä olevat luotot. </Accordion>

<Accordion title="Can I buy more workflow credits?">
You can buy additional credits under `Settings → Billing`.
</Accordion>
</AccordionGroup>

## Laskutus

<AccordionGroup>
<Accordion title="Where can I manage the Billing details?">
You can do so under `Settings → Billing`.
</Accordion>

<Accordion title="Where can I update my payment method?">
Voit tehdä tämän kohdassa `Asetukset → Laskutus`. Napsauta sitten `Näytä laskutustiedot`. Voit lisätä uuden maksutavan siellä.
</Accordion>

<Accordion title="Where can I update my billing information?">
Voit tehdä tämän kohdassa `Asetukset → Laskutus`. Napsauta sitten `Näytä laskutustiedot`. Voit muokata laskutustietoja siellä.
</Accordion>

<Accordion title="Where can I access my invoices?">
Voit tehdä tämän kohdassa `Asetukset → Laskutus`. Napsauta sitten `Näytä laskutustiedot`. Näet kaikki laskusi näytön alareunassa.
</Accordion>
</AccordionGroup>
