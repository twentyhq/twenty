## Tietoja

Twenty GitHub-repositorio isännöi laajaa valikoimaa resursseja, kuten lähdekoodia, dokumentaatiota, keskusteluja ja ongelmanseurantaa. This is where you will be able to access the full code behind Twenty.

[Käy Twenty-sivustolla GitHubissa](https://github.com/twentyhq/twenty)

## Osallistuminen

Osallistuminen Twenty-projektiin GitHubissa on palkitseva tapa auttaa parantamaan käyttämääsi ohjelmistoa. Riippumatta siitä, korjaatko virheitä, ehdotatko ominaisuuksia tai parannatko dokumentaatiota, panoksesi on tervetullut.

### Virheiden ilmoittaminen

Encounter an issue? [Luo uusi ongelma](https://github.com/twentyhq/twenty/issues/new) GitHubissa ja anna mahdollisimman paljon yksityiskohtia.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Ominaisuuksien ehdottaminen

Mitä parannuksia haluaisit nähdä Twentyssä? Teknillisestä asiantuntemuksesta riippumatta voit liittyä [keskusteluun täällä](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Ominaisuuden koodaaminen

Aloita matkasi etsimällä aloittelijaystävällisiä tehtäviä:

1. Siirry Twenty-repositorion **[Issues](https://github.com/twentyhq/twenty/issues)** -välilehdelle.
2. Suodata käyttämällä **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** -tunnistetta löytääksesi aloittelijoille sopivia tehtäviä.
3. Valitse ongelma, forkaa repositorio ja aloita osallistuminen.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Varmista, että sinut on osoitettu tehtävään, jotta et tee päällekkäistä työtä muiden osallistujien kanssa.

### Säännöt ja ohjeet

Muista noudattaa Twentyn [käyttäytymissääntöjä](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) koko osallistumisprosessin ajan.
