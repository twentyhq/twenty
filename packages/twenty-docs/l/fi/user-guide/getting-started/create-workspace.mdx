## Vaihe 1: Rekisteröinti

1. Siirry [Twenty Sign Up](https://app.twenty.com).
2. Valitse haluamasi rekisteröintitapa:
   - **Jatka Googlella** Google-tilin rekisteröintiin.
   - **Jatka Microsoftilla** Microsoft-tilin rekisteröintiin.
   - Tai, **Jatka sähköpostilla** sähköpostirekisteröintiä varten.

## Step 2: Choosing a Trial Period

Valitse kahden kokeilujakson välillä:

### 30 päivää

Luottokortilla

### 7 päivää

Ilman luottokorttia

Molemmat kokeilut sisältävät:

- Täysi pääsy
- Rajattomat yhteystiedot
- S\u00e4hk\u00f6posti-integraatio
- Mukautetut objektit
- API & Webhooks

Voit napsauttaa "Vaihda pakettia" valitaksesi toisen paketin tai laskutusvälin.

## Vaihe 3: Maksun vahvistus ja tilin asennus

Maksaessasi Stripe-palvelun kautta saat luvan luoda työtilasi ja käyttäjäprofiilisi. Muista, että voit peruuttaa tilauksesi milloin tahansa.
