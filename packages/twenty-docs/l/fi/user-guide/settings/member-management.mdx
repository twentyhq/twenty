## Kutsu uusia jäseniä

### Using the Invite Link

1. Mene **Asetukset → Jäsenet**
2. Kopioi työtilan kutsulinkki
3. Jaa linkki uusille tiimin jäsenille
4. He saavat käyttöoikeuden rekisteröityessään

### Suora sähköpostikutsu

1. Mene **Asetukset → Jäsenet**
2. Syötä henkilön sähköpostiosoite
3. Klikkaa **Kutsu**
4. He saavat kutsun sähköpostiinsa

## Poista jäseniä

### Poista jäsen

1. Mene **Asetukset → Jäsenet**
2. Löydä jäsen, jonka haluat poistaa
3. Napsauta poista/poista-painiketta henkilön nimen vieressä
4. Vahvista poisto

**Huom:** Poistetut jäsenet menettävät käyttöoikeuden välittömästi, mutta heidät voidaan kutsua uudestaan.
