## Henkilötiedot

### Nimi ja sähköposti

- **Näyttönimi**: Päivitä miten nimesi näkyy muille työtilan jäsenille
- **Sähköpostiosoite**: Muuta kirjautumissähköpostiasi (tarvitsee vahvistuksen)
- **Profiilikuva**: Lataa oma avatar tai käytä nimikirjaimia

## Suojausasetukset

### Kaksivaiheinen tunnistautuminen (2FA)

Ota käyttöön 2FA lisätäksesi ylimääräisen suojakerroksen tiliisi:

1. Siirry kohtaan **Asetukset → Profiiliasetukset**
2. Napsauta **Ota 2FA käyttöön**
3. Skannaa QR-koodi autentikointisovelluksellasi
4. Syötä vahvistuskoodi vahvistaaksesi

### Salasananhallinta

- **Vaihda salasana**: Päivitä nykyinen salasanasi
- **Salasanavaatimukset**: Pitää olla vähintään 8 merkkiä pitkä

## Profiilihallinta

### Poista tili

<Warning>
Tilisi poistaminen poistaa pysyvästi pääsyn kaikkiin työtiloihin. Tämä toimenpide on peruuttamaton, menetät pääsyn kaikkiin työtiloihin, joissa olet jäsen, ja sinun tulisi harkita yksittäisistä työtiloista poistumista, jos haluat vain poistua tietyistä ryhmistä.
</Warning>

Tilin poistaminen:

1. Siirry kohtaan **Asetukset → Profiiliasetukset**
2. Selaa kohtaan **Vaaravyöhyke**
3. Napsauta **Poista tili**
4. Vahvista kirjoittamalla sähköpostiosoitteesi
