## Latest Releases

Seuraa Twentyn kehityksen etenemistä:

- **Julkaisutiedotteet**: Katso, mitä uutta on jokaisessa versiossa
- **Ominaisuuspäivitykset**: Tutustu uusiin ominaisuuksiin ja parannuksiin

## Lab Features

Käytä kokeellisia ominaisuuksia ennen niiden virallista julkaisua:

- **Beta-Testaus**: Kokeile uusia ominaisuuksia niiden kehityksen aikana
- **Feature Flags**: Enable or disable specific experimental functionality

## How to Access

1. Go to **Settings → Releases**
2. View release notes and access the **Lab** tab for beta features
