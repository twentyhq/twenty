يتكامل Twenty مع أدواتك الحالية بحيث يبقى نظام إدارة علاقات العملاء (CRM) محدَّثًا دون الحاجة إلى إدخال البيانات يدويًا.

## مزامنة البريد الإلكتروني

قم بتوصيل صندوق بريد **Google Workspace** أو **Microsoft 365** الخاص بك. بمجرد الاتصال:

* يتم ربط رسائل البريد الإلكتروني تلقائيًا بسجلات الشركة والأشخاص المطابقة
* تظهر سلاسل البريد الإلكتروني الكاملة في المخطط الزمني لكل سجل
* يمكنك إرسال رسائل البريد الإلكتروني مباشرةً من Twenty
* يتم دعم صناديق بريد متعددة لكل مستخدم

يمكنك التحكّم فيما يتم استيراده — قم بالتصفية حسب النطاق الزمني أو المرسِل لتجنّب جلب رسائل بريد إلكتروني غير ذات صلة.

## مزامنة التقويم

تتم مزامنة أحداث التقويم تلقائيًا من حسابك المتصل. تظهر الأحداث في سجلات نظام إدارة علاقات العملاء (CRM) ذات الصلة، مما يمنحك صورة كاملة عن تفاعلاتك مع كل جهة اتصال أو شركة.

## التكاملات

إضافةً إلى البريد الإلكتروني والتقويم، يتكامل Twenty مع الأدوات الخارجية من خلال:

| طريقة                         | حالة استخدام                                               |
| ----------------------------- | ---------------------------------------------------------- |
| **API**                       | أنشئ عمليات تكامل مخصّصة باستخدام GraphQL أو REST API      |
| **خطافات الويب**              | ادفع بإشعارات فورية إلى الأنظمة الخارجية عند تغيّر السجلات |
| **Zapier**                    | اتصل بأكثر من 5,000 تطبيق بدون كود                         |
| **إجراءات HTTP في سير العمل** | استدعِ أي واجهة برمجة تطبيقات خارجية كجزء من سير عمل مؤتمت |

## تطبيقات مخصّصة

يمكن للمطورين إنشاء تطبيقات متكاملة الإمكانات فوق Twenty — لإضافة واجهات مستخدم مخصّصة ومنطق على جانب الخادم وعمليات تكامل عميقة. يمكن نشر التطبيقات للمجتمع أو إبقاؤها خاصة.
