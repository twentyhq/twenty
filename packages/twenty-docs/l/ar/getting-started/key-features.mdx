Twenty هي منصة إدارة علاقات العملاء (CRM) متكاملة المزايا. إليك ما يمكنك بناؤه باستخدامه.

<CardGroup cols={2}>
  <Card href="/l/ar/user-guide/data-model/overview" img="/images/user-guide/halftone/data-model.png">
    <CardTitle>نموذج بيانات مخصص</CardTitle>
    عرّف هيكل البيانات الدقيق الذي يحتاجه عملك.
    <br />أنشئ كائنات مخصصة، وأضف حقولًا مخصصة مع أكثر من 20 نوعًا من الحقول، وأنشئ علاقات بين أي كائنات.
  </Card>

  <Card href="/l/ar/user-guide/views-pipelines/overview" img="/images/user-guide/halftone/layout.png">
    <CardTitle>طرق العرض والمسارات</CardTitle>
    بدّل بين طرق عرض الجدول، ولوحات كانبان، وطرق عرض التقويم.
    <br />قم بالتصفية باستخدام منطق AND/OR، ورتّب حسب عدة حقول، واجمع السجلات، واحفظ طرق عرض مخصصة.
  </Card>

  <Card href="/l/ar/user-guide/workflows/overview" img="/images/user-guide/halftone/workflows.png">
    <CardTitle>سير العمل والأتمتة</CardTitle>
    أتمت أي عملية عمل دون كتابة شفرة برمجية.
    <br />قم بتشغيل سير العمل عند تغيّر السجلات، أو وفق الجداول الزمنية، أو الإجراءات اليدوية، أو خطافات الويب الواردة.
  </Card>

  <Card href="/l/ar/user-guide/calendar-emails/overview" img="/images/user-guide/halftone/calendar-emails.png">
    <CardTitle>مزامنة التقويم والبريد الإلكتروني</CardTitle>
    اربط حساب Google Workspace أو Microsoft 365 الخاص بك.
    <br />تظهر رسائل البريد الإلكتروني وأحداث التقويم تلقائيًا على سجلات إدارة علاقات العملاء (CRM) ذات الصلة.
  </Card>

  <Card href="/l/ar/user-guide/ai/overview" img="/images/user-guide/halftone/ai.png">
    <CardTitle>الذكاء الاصطناعي</CardTitle>
    وكلاء ذكاء اصطناعي يعملون بشكل مستقل داخل نظام إدارة علاقات العملاء (CRM) — يجيبون عن الأسئلة، ويُثْرون السجلات، وينفّذون مهام متعددة الخطوات ضمن نموذج الأذونات الخاص بك.
  </Card>

  <Card href="/l/ar/user-guide/dashboards/overview" img="/images/user-guide/halftone/dashboards.png">
    <CardTitle>لوحات المعلومات والتقارير</CardTitle>
    أنشئ لوحات معلومات مخصصة باستخدام عناصر واجهة في الوقت الفعلي.
    <br />تتبّع مقاييس المسار، وأداء الفريق، ومؤشرات الأداء الرئيسية للأعمال باستخدام مخططات وعوامل تصفية قابلة للتهيئة.
  </Card>

  <Card href="/l/ar/user-guide/permissions-access/overview" img="/images/user-guide/halftone/permissions.png">
    <CardTitle>الصلاحيات والتحكم في الوصول</CardTitle>
    تحكم في الوصول قائم على الأدوار على كل المستويات — الكائنات والحقول والسجلات الفردية.
    <br />قم بتهيئة تسجيل الدخول الأحادي (SSO) باستخدام SAML أو OIDC. سجلات التدقيق تتعقّب من فعل ماذا.
  </Card>

  <Card href="/l/ar/developers/extend/api" img="/images/user-guide/halftone/dev-api.png">
    <CardTitle>واجهة برمجة التطبيقات (API) وقابلية التوسّع</CardTitle>
    واجهة برمجة تطبيقات موجّهة للمطورين تتكيّف مع نموذج البيانات المخصص لديك.
    <br />نقاط نهاية GraphQL وREST معًا، مع توثيق يتم إنشاؤه تلقائيًا لكل مساحة عمل.
  </Card>

  <Card href="/l/ar/user-guide/data-migration/overview" img="/images/user-guide/halftone/data-migration.png">
    <CardTitle>استيراد البيانات وتصديرها</CardTitle>
    استورد البيانات من ملفات CSV أو عبر واجهة برمجة التطبيقات (API).
    <br />تعيين الحقول، واكتشاف التكرار، والتعامل مع الأخطاء مدمجة في النظام. صدّر بياناتك في أي وقت — بدون أي قيود ارتباط بمنصة واحدة.
  </Card>

  <Card href="/l/ar/developers/self-host/self-host" img="/images/user-guide/halftone/dev-self-host.png">
    <CardTitle>الاستضافة الذاتية</CardTitle>
    شغّل Twenty على بنيتك التحتية الخاصة باستخدام أمر Docker Compose واحد.
    <br />تحكّم كامل في بياناتك، وتحديثات وفق الجدول الزمني الذي يناسبك.
  </Card>
</CardGroup>
