## ما القادم

### تقارير مخصصة

أنشئ تقارير مخصصة لتعقب أهم المؤشرات لأعمالك.

### لوحات مبيعات

تصور أداء مسار المبيعات الخاص بك مع لوحات تفاعلية.

### تحليلات الأداء

راقب أداء الفريق واكتشف الاتجاهات في بيانات عملائك.

## ابق على إطلاع

تابع [مستودع GitHub الخاص بنا](https://github.com/twentyhq/twenty) أو راجع قسم **الإعدادات → الإصدارات** في مساحة العمل الخاصة بك لتلقي إشعارات متى أصبحت ميزات التقارير متاحة.
