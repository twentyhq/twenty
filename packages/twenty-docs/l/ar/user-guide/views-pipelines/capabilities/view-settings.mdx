## ظهور العرض

تحكّم بمن يمكنه رؤية عروضك المخصصة.

### خيارات الظهور

| الإعداد         | من يمكنه الرؤية        |
| --------------- | ---------------------- |
| **مساحة العمل** | جميع أعضاء مساحة العمل |
| **غير مدرج**    | أنت فقط                |

### تغيير الظهور

1. افتح العرض
2. انقر على **الخيارات → الظهور**
3. اختر **مساحة العمل** أو **غير مدرج**

<Note>
  لا يمكن تغيير ظهور طرق العرض الافتراضية "All [Object Name]".
</Note>

## إعادة تسمية عرض

1. افتح قائمة العرض المنسدلة
2. انقر على قائمة **⋮** بجوار العرض
3. اختر **تحرير**
4. أدخل الاسم الجديد

## تغيير أيقونة العرض

1. افتح قائمة العرض المنسدلة
2. انقر على قائمة **⋮** بجوار العرض
3. اختر **تحرير**
4. انقر على الأيقونة لتغييرها

## إعادة ترتيب طرق العرض

غيّر ترتيب ظهور العروض في القائمة المنسدلة:

1. افتح قائمة العرض المنسدلة
2. اسحب العروض من مقبضها
3. أفلِت في الموضع المطلوب
4. يتم حفظ الترتيب تلقائيًا

## المفضلات

ثبّت العروض المستخدمة بكثرة للوصول السريع:

1. افتح قائمة العرض المنسدلة
2. انقر على قائمة **⋮** بجوار أحد العروض
3. اختر **إضافة إلى المفضلات**

تظهر العروض المضافة إلى المفضلات في قسم مخصص لسهولة الوصول.

## حذف عرض

1. افتح قائمة العرض المنسدلة
2. انقر على قائمة **⋮** بجوار العرض
3. اختر **حذف**
4. أكد الحذف

<Warning>
  لا يمكن استعادة العروض المحذوفة.
</Warning>
