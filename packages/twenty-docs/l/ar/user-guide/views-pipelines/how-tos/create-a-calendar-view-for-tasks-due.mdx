## المتطلبات الأساسية

يحتاج كائن المهام إلى حقل **Due Date** (من نوع Date أو Date & Time).

## الخطوات

1. انتقل إلى **المهام**
2. انقر على القائمة المنسدلة لطريقة العرض → **+ إضافة طريقة عرض**
3. قم بتسمية العرض (مثال: "تقويم المهام")
4. انقر على **إنشاء**
5. انقر على **الخيارات** واختر **التقويم** كتخطيط
6. اختر **Due Date** كحقل التاريخ
7. انقر على **حفظ**

## تهيئة التقويم

### إظهار الحقول على الأحداث

1. انقر على **الخيارات → الحقول**
2. انقر على **أيقونة العين** لإظهار/إخفاء الحقول
3. اسحب لإعادة الترتيب

الحقول الموصى بعرضها:

* **Title** — اسم المهمة
* **Assignee** — الشخص المسؤول
* **Status** — التقدّم الحالي

### تصفية التقويم

أنشئ عروضًا مركّزة:

* **مهامي**: تصفية حسب Assignee = Me
* **هذا الأسبوع**: تصفية حسب Due Date = This week
* **متأخر**: التصفية حسب تاريخ الاستحقاق < اليوم، الحالة ≠ مكتمل

## حالات استخدام أخرى للتقويم

| كائن            | حقل التاريخ    | الغرض                           |
| --------------- | -------------- | ------------------------------- |
| الفرص           | تاريخ الإغلاق  | تتبع عمليات الإغلاق المتوقعة    |
| الأحداث المخصصة | تاريخ الحدث    | التخطيط للأنشطة                 |
| المشاريع        | الموعد النهائي | مراقبة الجداول الزمنية للمشاريع |

## نصائح

* **راجِع أسبوعياً**: ابدأ كل أسبوع بالتحقق من عرض التقويم
* **اجمعه مع عرض الجدول**: استخدم التقويم للعرض العام، والجدول للتفاصيل
* **عيّن مستوى الرؤية**: اجعل تقاويم المهام الشخصية غير مدرجة
