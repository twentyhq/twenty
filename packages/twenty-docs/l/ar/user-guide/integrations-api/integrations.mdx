## خيارات التكامل الحالية

### التكاملات الأصلية

تقدم Twenty حاليًا تكاملًا أصليًا مع:

- **البريد الإلكتروني والتقويم**: ربط Gmail، Outlook، أو موفري SMTP/CalDAV
- **API Access**: Use our REST and GraphQL APIs to build custom integrations

لإعداد البريد الإلكتروني والتقويم، زر [إعداد البريد الإلكتروني والتقويم](/l/ar/user-guide/settings/email-calendar-setup).

### تدفقات العمل (موصى بها)

الطريقة الأساسية لربط Twenty بأدوات أخرى هي عبر ميزة **تدفقات العمل** لدينا:

- **عقد HTTP**: إجراء مكالمات API إلى الخدمات الخارجية
- **عقد التعليمات البرمجية**: كتابة منطق مخصص للتكاملات المعقدة
- **مشغلات Webhook**: استقبال البيانات من الأنظمة الخارجية
- **مشغلات تحديث الحقل**: أتمتة الإجراءات بناءً على تغييرات CRM

تعلم المزيد في قسم [تدفقات العمل](/l/ar/user-guide/workflows/getting-started-workflows).

### تكامل Zapier (قديم)

نقوم بالحفاظ على تكامل Zapier للمستخدمين الذين يفضلون الأتمتة بدون كود:

1. زوروا [Twenty على Zapier](https://zapier.com/apps/twenty/integrations)
2. أنشئ Zap جديد مع Twenty كعنصر مشغل أو تنفيذ
3. Generate an API key in Settings → API & Webhooks
4. ربط مساحة عمل Twenty الخاصة بك مع Zapier

## رؤية المستقبل: موصلات مبنية من المجتمع (2026)

### منصة التوسع

نحن نبني منصة للتوسع تُمكّن المطورين من إنشاء التطبيقات كترميز. سيمكن ذلك:

- **موصلات مخصصة**: بناء التكاملات مع برمجياتك المفضلة
- **مساهمات المجتمع**: مشاركة واكتشاف الموصلات التي يبنيها الآخرون
- **بنية مرنة**: توسيع قدرات Twenty خارج الميزات الأساسية

### بناء تدفقات العمل بمساعدة الذكاء الاصطناعي

قادمة في 2026، ستساعدك مساعدة الذكاء الاصطناعي:

- **توليد تلقائي لتدفقات العمل**: وصف احتياجات التكامل الخاصة بك بلغة بسيطة
- **اقتراحات ذكية**: الحصول على توصيات لربط أدواتك المحددة
- **مكتبة النماذج**: الوصول إلى تدفقات عمل مبنية مسبقًا للاستخدامات الشائعة

مستقبل تكاملات Twenty يعتمد على المجتمع، بمساعدة الذكاء الاصطناعي، وعلى التوسيع اللامحدود.
