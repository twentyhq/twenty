## مفاتيح واجهة برمجة التطبيقات

تسمح مفاتيح API بالوصول الآلي لبيانات CRM الخاصة بك، ومزامنة البيانات مع الأنظمة الأخرى، وإنشاء تكاملات أو حلول مخصصة.

### قم بإنشاء مفتاح API

1. Go to **Settings → APIs & Webhooks**
2. انقر على **+ إنشاء مفتاح** في الزاوية اليمنى العليا
3. قم بتكوين مفتاح API الخاص بك:
   - **الاسم**: أعط مفتاح API الخاص بك اسمًا وصفيًا
   - **تاريخ الانتهاء**: حدد متى يجب أن ينتهي صلاحية المفتاح
4. انقر على **حفظ** لإنشاء مفتاح API الخاص بك
5. **مهم**: انسخ واحفظ مفتاح API الخاص بك على الفور، يتم عرضه مرة واحدة فقط.

بمجرد الإنشاء، يوفر مفتاح API الخاص بك الوصول إلى وثائق API المخصصة وملعب التجربة حيث يمكنك اختبار نقاط النهاية باستخدام نموذج البيانات الفعلية الخاص بك.

<Warning>
نظرًا لأن مفتاح API يتيح الوصول إلى معلومات حساسة، لا ينبغي عليك مشاركته مع الخدمات التي لا تثق بها تمامًا. إذا تم تسريب المفتاح، يمكن لشخص ما استخدامه بشكل ضار. إذا تم اختراق أمان مفتاح API الخاص بك، قم بتعطيله فورًا وقم بإنشاء آخر جديد.
</Warning>

### إدارة مفاتيح API

**إعادة إنشاء مفتاح API:**

1. Go to **Settings → APIs & Webhooks**
2. انقر على مفتاح API الذي ترغب في إعادة إنشائه
3. انقر على زر **إعادة الإنشاء**
4. انسخ وخزن مفتاح API الجديد فورًا.

**حذف مفتاح API:**

1. اعثر على مفتاح API في قائمتك.
2. انقر على المفتاح لفتح تفاصيله.
3. انقر على **حذف** لإزالته بشكل دائم.

## الويب هوكس

تسمح Webhooks بالتحديثات الفورية إلى عنوان URL المحدد الخاص بك بشأن التغييرات أو الأحداث المتعلقة ببيانات العملاء الخاصة بك.

على سبيل المثال، عندما ينتقل الفرصة إلى "مغلقة مُكتملة بنجاح"، يمكن أن تطلق Webhook إنشاء الفاتورة تلقائيًا في نظام المحاسبة الخاص بك. لاحظ أن هذا النوع من الأتمتة يمكن تحقيقه أيضًا باستخدام خاصية [Workflows](/l/ar/user-guide/workflows/getting-started-workflows) الخاصة بـ Twenty، التي تقدم محفزات تستند إلى تحديثات الحقول للأتمتة الداخلية.

تعتبر Webhooks مثالية للتكامل مع الأنظمة الخارجية، بينما تدعم Workflows الأتمتة الداخلية وربط الأدوات الخارجية عبر محفزات Webhooks، عقد التعليمات البرمجية، وعقد HTTP.

### إنشاء ربط ويب

1. Go to **Settings → APIs & Webhooks → Webhooks**
2. انقر على **+ إنشاء ربط ويب**
3. أدخل عنوان URL الخاص بالربط الذي تود تلقي الإشعارات عليه.
4. انقر على **حفظ**

ربط الويب الخاص بك سيبدأ فورًا في تلقي إشعارات فورية حول التغييرات على بيانات إدارة علاقات العملاء الخاصة بك.

### إدارة Webhooks

**حذف Webhook:**

1. Go to **Settings → APIs & Webhooks → Webhooks**
2. ابحث عن ربط الويب الذي تريد إزالته.
3. انقر على ربط الويب.
4. انقر على **حذف** وقم بالتأكيد في النافذة المنبثقة.

**تعديل ربط ويب:**

1. انقر على ربط الويب الذي تريد تعديله.
2. قم بتحديث عنوان URL أو الإعدادات الأخرى.
3. انقر على **حفظ** لتطبيق التغييرات.
