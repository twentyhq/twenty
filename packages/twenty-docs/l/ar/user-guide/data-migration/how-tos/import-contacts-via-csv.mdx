## نظرة عامة

يرشدك هذا الدليل خلال عملية استيراد جهات الاتصال (الأشخاص) إلى Twenty. **استورد الشركات أولًا** إذا كنت تريد ربط الأشخاص بالشركات.

## قبل أن تبدأ

### قائمة المتطلبات المسبقة

<Check>
  يجب أن يكون ملفك بتنسيق CSV أو XLSX أو XLS
</Check>

<Check>
  يحتوي الملف على أقل من 10,000 سجل
</Check>

<Check>
  لا توجد عناوين بريد إلكتروني مكررة في ملفك
</Check>

<Check>
  **استيراد الشركات أولًا** (إذا كنت تربط الأشخاص بالشركات)
</Check>

<Check>
  جميع الحقول المخصصة موجودة في **Settings → Data Model**
</Check>

<Warning>
  **استيراد الشركات قبل الأشخاص**

  إذا كنت تريد ربط الأشخاص بالشركات، فاستورد الشركات أولًا. يجب أن تكون الشركة موجودة قبل أن تتمكن من الإشارة إليها.
</Warning>

## الخطوة 1: جهّز بيانات جهات الاتصال الخاصة بك

### الحقول المطلوبة والمُوصى بها

| الحقل                 | هل هو مطلوب؟ | التنسيق           | الملاحظات                        |
| --------------------- | ------------ | ----------------- | -------------------------------- |
| **البريد الإلكتروني** | مُوصى به     | `name@domain.com` | يجب أن يكون فريدًا               |
| **الاسم الأول**       | مُوصى به     | نص                |                                  |
| **الاسم الأخير**      | مُوصى به     | نص                |                                  |
| **الشركة**            | اختياري      | النطاق أو المعرّف | يرتبط بشركة موجودة               |
| **هاتف**              | اختياري      | أعمدة متعددة      | انظر أدناه                       |
| **المسمى الوظيفي**    | اختياري      | نص                |                                  |
| **الحقول المخصصة**    | اختياري      | يختلف             | يجب أن تكون موجودة في Data Model |

### تنسيق البريد الإلكتروني

* يجب أن يكون بتنسيق بريد إلكتروني صالح: `name@domain.com`
* **يجب أن يكون فريدًا** — لا تكرارات في ملفك أو في Twenty
* للعناوين الإضافية للبريد الإلكتروني، استخدم عمود **Emails / Additional Emails**:

```
["jane@twenty.com","jane.doe@twenty.com"]
```

### تنسيق الهاتف

الهاتف هو **حقل متداخل** يتطلب أعمدة متعددة:

| العمود                                  | مثال         |
| --------------------------------------- | ------------ |
| **Phones / Primary Phone Number**       | `4159095555` |
| **Phones / Primary Phone Country Code** | `US`         |
| **Phones / Primary Phone Calling Code** | `+1`         |

### الربط بالشركات

أضف عمودًا يضم المعرّف الفريد للشركة:

| اسم العمود      | التنسيق         | مثال                                   |
| --------------- | --------------- | -------------------------------------- |
| `companyDomain` | تنسيق عنوان URL | `https://acme.com`                     |
| `companyId`     | UUID            | `c776ee49-f608-4a77-8cc8-6fe96ae1e43f` |

<Note>
  **استخدم تنسيق عنوان URL للنطاق** (`https://acme.com`)، وليس التسمية. هذا يطابق كيفية تخزين الشركات في Twenty.
</Note>

### بنية CSV نموذجية

```csv
firstName,lastName,email,jobTitle,companyDomain,Phones / Primary Phone Number,Phones / Primary Phone Country Code
John,Smith,john@acme.com,CEO,https://acme.com,4159095555,US
Jane,Doe,jane@widgets.co,CTO,https://widgets.co,2125551234,US
```

<Note>
  **نصيحة احترافية:** انقر **Download sample file** أثناء الاستيراد أو صدّر عددًا قليلًا من الأشخاص الموجودين للاطلاع على أسماء الأعمدة الدقيقة التي تتوقعها Twenty.
</Note>

## الخطوة 2: الوصول إلى ميزة الاستيراد

**الخيار 1: من عرض People**

1. انتقل إلى **People** في الشريط الجانبي الأيسر
2. انقر على أيقونة **⋮** في أعلى اليمين
3. اختر **Import records**

**الخيار 2: باستخدام قائمة الأوامر**

1. اضغط `Cmd + K` (Mac) أو `Ctrl + K` (Windows)
2. اكتب "import"
3. اختر **Import records**
4. اختر **People**

## الخطوة 3: حمّل ملفك

1. انقر فوق **Select file**
2. اختر ملف CSV أو XLSX أو XLS الخاص بك
3. انتظر حتى يقوم Twenty بتحليل ملفك

## الخطوة 4: طابق أعمدتك

يحاول Twenty تلقائيًا مطابقة أعمدتك مع الحقول. راجع وعدّل:

1. **تحقق من المطابقات التلقائية** — تأكد من أنها صحيحة
2. **أصلح المطابقات غير الصحيحة** — انقر على القائمة المنسدلة لاختيار الحقل الصحيح
3. **تخطَّ الأعمدة** — اختر **Do not map** للأعمدة التي لا تريد استيرادها

### قواعد مطابقة مهمة

| نوع العمود                  | التعيين إلى                    | الملاحظات                             |
| --------------------------- | ------------------------------ | ------------------------------------- |
| مرجع الشركة                 | حقل علاقة **Company**          | استخدم النطاق أو المعرّف، وليس كليهما |
| البريد الإلكتروني           | **البريد الإلكتروني**          | عنوان البريد الإلكتروني الأساسي       |
| عناوين بريد إلكتروني إضافية | **Emails / Additional Emails** | تنسيق المصفوفة                        |
| هاتف                        | أعمدة منفصلة                   | الرقم، رمز الدولة، رمز الاتصال        |

<img src="/images/user-guide/import-export-data/match-columns.png" style={{width:'100%'}} />

### مطابقة علاقة الشركة

عند مطابقة عمود الشركة:

1. اعثر على عمود مرجع الشركة (مثلًا، `companyDomain`)
2. طابقه مع حقل علاقة **Company**
3. سيقوم Twenty بربط كل شخص بالشركة المطابقة

<Warning>
  **طابق معرّفًا فريدًا واحدًا فقط للعلاقات.**

  لا تقم بتعيين كلٍّ من `companyId` و`companyDomain` معًا. اختر واحدًا — ويفضَّل استخدام النطاق لأنه مقروء للبشر.
</Warning>

## الخطوة 5: تعيين قيم حقل الاختيار

إذا كانت لديك حقول اختيار أو اختيار متعدد (مثل مصدر العميل المحتمل):

1. تعرض Twenty قيمك إلى جانب الخيارات الموجودة
2. طابِق كل قيمة في ملفك مع خيار في Twenty
3. أو أنشئ خيارات جديدة عند الحاجة

<Note>
  تستخدم خيارات الاختيار **أسماء API**، وليس تسميات العرض. تحقّق من **الإعدادات → نموذج البيانات** → فعِّل **الوضع المتقدم** لمشاهدة أسماء API.
</Note>

## الخطوة 6: مراجعة الأخطاء وإصلاحها

قبل إكمال الاستيراد، تقوم Twenty بالتحقق من صحة بياناتك:

1. انقر على **الخطوات التالية**
2. يتم تمييز الصفوف التي تحتوي على أخطاء باللون **الأصفر**
3. **أصلِح الأخطاء مباشرةً** — انقر على خلية وعدّل القيمة
4. **أزِل الصفوف التي بها مشكلات** — انقر على X لتخطي ذلك الصف

### أخطاء شائعة عند استيراد جهات الاتصال

| خطأ                              | السبب                                                | الحل                                 |
| -------------------------------- | ---------------------------------------------------- | ------------------------------------ |
| **بريد إلكتروني مكرر**           | البريد الإلكتروني موجود مسبقًا في Twenty أو في الملف | أزِل التكرار أو حدِّث السجل الموجود  |
| **تنسيق بريد إلكتروني غير صالح** | تنسيق البريد الإلكتروني غير صحيح                     | صحّحه إلى `name@domain.com`          |
| **العلاقة غير موجودة**           | الشركة غير موجودة                                    | استورد الشركات أولًا أو أصلِح المرجع |
| **حقل مطلوب مفقود**              | الحقل المطلوب فارغ                                   | املأ القيمة أو أزِل الصف             |

## الخطوة 7: إكمال الاستيراد

1. راجِع ملخص الاستيراد
2. انقر على **تأكيد** للاستيراد
3. انتظر حتى يكتمل الاستيراد
4. تحقَّق بفحص عدد قليل من السجلات وروابطها بالشركة

## بعد استيراد جهات الاتصال

جهات الاتصال الخاصة بك موجودة الآن في Twenty! الخطوات التالية:

1. **تحقَّق من روابط الشركة** — افتح عددًا قليلًا من سجلات الأشخاص للتأكد من أنها مرتبطة بالشركة الصحيحة
2. **استورد الفرص** — واربطها عند الحاجة بالأشخاص والشركات
3. **اضبط مزامنة البريد الإلكتروني** — وصّل صندوق بريدك لعرض سجل البريد الإلكتروني في سجلات جهات الاتصال

## تحديث جهات الاتصال الموجودة

لتحديث جهات الاتصال بدلًا من إنشاء جهات جديدة:

1. ضمّن عمود `email` أو `id` في ملفك
2. تقوم Twenty بمطابقة السجلات باستخدام هذا المعرّف الفريد
3. يتم تحديث جهات الاتصال الموجودة؛ ويتم إنشاء جهات جديدة

راجع [كيفية تحديث السجلات الموجودة](/l/ar/user-guide/data-migration/how-tos/update-existing-records-via-import) للتفاصيل.
