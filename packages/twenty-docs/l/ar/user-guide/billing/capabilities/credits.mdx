## نظرة عامة

الأرصدة تدعم أتمتة سير العمل وإمكانات الذكاء الاصطناعي. يتم استهلاكها عند تنفيذ إجراءات سير العمل، وعندما يعالج وكلاء الذكاء الاصطناعي المهام داخل سير العمل، وداخل دردشة الذكاء الاصطناعي.

## تخصيص الأرصدة

تستند الأرصدة إلى دورة الفوترة لديك، وليس إلى خطتك:

| دورة الفوترة | الأرصدة |
| ------------ | ------- |
| شهري         | 5/شهر   |
| سنوي         | 50/سنة  |

<Note>
تم تصميم الأرصدة الشهرية الخمسة لتمكينك من تشغيل الأتمتة دون القلق بشأن التكاليف. بالنسبة إلى معظم مسارات العمل التي تستخدم الإجراءات القياسية، فهذا أكثر من كافٍ. ستحتاج إلى أرصدة إضافية فقط عند تشغيل عقد برمجية متقدمة أو ميزات مدعومة بالذكاء الاصطناعي.
</Note>

## ترحيل الأرصدة

تُرحَّل الأرصدة غير المستخدمة في نهاية فترة الفوترة تلقائيًا إلى الفترة التالية.

* **الحد الأقصى**: يتم تحديد مقدار الترحيل بسقف يعادل التخصيص الكامل لفترة واحدة، لذا لا يمكنك أبدًا ترحيل أرصدة أكثر مما توفره خطتك لكل فترة.
* **الظهور**: عند توفّر أرصدة مُرحّلة، تظهر كبند منفصل بعنوان **الأرصدة المُرحّلة** في **الإعدادات → الفوترة**، إلى جانب رصيد **الإجمالي المتاح**.

## استهلاك الأرصدة

تستهلك الإجراءات المختلفة كميات مختلفة من الأرصدة:

| نوع الإجراء                                           | استخدام الأرصدة        |
| ----------------------------------------------------- | ---------------------- |
| **العمليات الأساسية** (البحث، التحديث، إنشاء السجلات) | ضئيل                   |
| **عمليات معقدة** (عُقَد برمجية، استدعاءات API خارجية) | مزيد من الأرصدة        |
| **موجّه الذكاء الاصطناعي ودردشات الذكاء الاصطناعي**   | متغيِّرة حسب الاستخدام |

يتم خصم الأرصدة فورًا عند تنفيذ سير العمل.

## مراقبة الاستخدام

تتبَّع استهلاكك من الأرصدة:

1. اذهب إلى **الإعدادات → الفواتير**
2. اعرض استخدامك الحالي والأرصدة المتبقية
3. راقِب الاتجاهات للتخطيط للحصول على أرصدة إضافية إذا لزم الأمر

## شراء أرصدة إضافية

هل تحتاج إلى المزيد من الأرصدة؟

1. اذهب إلى **الإعدادات → الفواتير**
2. انقر على الخيار لشراء حزم أرصدة إضافية
3. اختر الكمية التي تحتاج إليها
