## نظرة عامة

الأرصدة تدعم عمليات أتمتة سير العمل لديك في Twenty. يستهلك كل إجراء سير عمل أرصدة بناءً على تعقيده.

## تخصيص الأرصدة

تستند الأرصدة إلى دورة الفوترة لديك، وليس إلى خطتك:

| دورة الفوترة | الأرصدة      |
| ------------ | ------------ |
| شهري         | 5 ملايين/شهر |
| سنوي         | 50 مليون/سنة |

<Note>
  تم تصميم 5 ملايين رصيد شهريًا لتمكينك من تشغيل الأتمتة دون القلق بشأن التكاليف. بالنسبة إلى معظم سير العمل التي تستخدم الإجراءات القياسية، فهذا أكثر من كافٍ. ستحتاج إلى أرصدة إضافية فقط عند تشغيل عقد برمجية متقدمة أو ميزات مدعومة بالذكاء الاصطناعي.
</Note>

## استهلاك الأرصدة

تستهلك الإجراءات المختلفة كميات مختلفة من الأرصدة:

| نوع الإجراء                                           | استخدام الاعتمادات     |
| ----------------------------------------------------- | ---------------------- |
| **العمليات الأساسية** (البحث، التحديث، إنشاء السجلات) | ضئيل                   |
| **عمليات معقدة** (عُقَد برمجية، استدعاءات API خارجية) | مزيد من الأرصدة        |
| **طلبات الذكاء الاصطناعي** (قريبًا)                   | متغيِّرة حسب الاستخدام |

يتم خصم الأرصدة فورًا عند تنفيذ سير العمل.

## مراقبة الاستخدام

تتبَّع استهلاكك من الأرصدة:

1. اذهب إلى **الإعدادات → الفواتير**
2. اعرض استخدامك الحالي والأرصدة المتبقية
3. راقِب الاتجاهات للتخطيط للحصول على أرصدة إضافية إذا لزم الأمر

## شراء رصيد إضافي

هل تحتاج إلى المزيد من الأرصدة؟

1. اذهب إلى **الإعدادات → الفواتير**
2. انقر على الخيار لشراء حزم أرصدة إضافية
3. اختر الكمية التي تحتاج إليها
