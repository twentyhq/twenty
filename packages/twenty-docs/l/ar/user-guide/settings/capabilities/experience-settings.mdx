خصّص تجربة Twenty ضمن **الإعدادات → التجربة**.

## المظهر

اختر السمة المرئية:

* **فاتح**: واجهة نظيفة ومشرقة
* **داكن**: مريح للعينين في ظروف الإضاءة المنخفضة
* **إعدادات النظام**: تتطابق تلقائياً مع سمة جهازك

## اللغة

اختر اللغة المفضلة لواجهة Twenty من القائمة المنسدلة.

## تنسيقات

إعداد التاريخ والوقت والرقم والمنطقة الزمنية ويوم بدء التقويم.

| الإعداد             | الوصف                                                           |
| ------------------- | --------------------------------------------------------------- |
| **المنطقة الزمنية** | المنطقة الزمنية المحلية لديك لضمان دقة الطوابع الزمنية والجدولة |
| **تنسيق التاريخ**   | كيفية ظهور التواريخ (مثال: 12 ديسمبر 2025)                      |
| **تنسيق الوقت**     | صيغة 12 ساعة (7:22 م) أو 24 ساعة                                |
| **تنسيق الأرقام**   | فواصل الأعداد العشرية وآلافها (مثال: 1,234.56)                  |
| **يوم بدء التقويم** | اليوم الأول من الأسبوع (الأحد أو الاثنين)                       |

يمكن تعيين كل إعداد إلى **إعدادات النظام** ليتطابق تلقائياً مع تفضيلات جهازك، أو يمكنك اختيار تنسيق محدد.

## كيفية التحديث

1. انتقل إلى **الإعدادات → التجربة**
2. اضبط تفضيلاتك في كل قسم
3. يتم حفظ التغييرات تلقائياً
