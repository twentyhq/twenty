Använder en blockbaserad rich text-editor från [BlockNote](https://www.blocknotejs.org/) för att låta användare redigera och visa innehållsblock.

<Tabs>
<Tab title="Usage">

```jsx
import { useBlockNote } from "@blocknote/react";
import { BlockEditor } from "@/ui/input/editor/components/BlockEditor";

export const MyComponent = () => {
  const BlockNoteEditor = useBlockNote();

  return <BlockEditor editor={BlockNoteEditor} />;
};
```

</Tab>
<Tab title="Props">

| Egenskaper | Typ               | Beskrivning                               |
| ---------- | ----------------- | ----------------------------------------- |
| redigerare | `BlockNoteEditor` | Blockredigerarinstans eller konfiguration |
