Ett visuellt element som du kan använda som en klickbar eller icke-klickbar behållare med en etikett, valfria vänster- och högerkomponenter, och olika stilalternativ för att visa etiketter och taggar.

<Tabs>

<Tab title="Usage">

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Chip
      size="large"
      label="Clickable Chip"
      clickable={true}
      variant="highlighted"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
    />
  );
};

```

</Tab>

<Tab title="Props">

| Egenskaper   | Typ                      | Beskrivning                                                                                                                  |
| ------------ | ------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| linkToEntity | string                   | Länken till entiteten                                                                                                        |
| entityId     | string                   | Det unika identifieraren för entiteten                                                                                       |
| namn         | string                   | Entitetens namn                                                                                                              |
| pictureUrl   | string                   | s bild",                                                                                                                     |
| avatarType   | Avatar Typ               | Typen av avatar du vill visa. Har två alternativ: `rundad` och `fyrkantig`                   |
| variant      | `EntityChipVariant` enum | Variant av entitetschippet du vill visa. Har två alternativ: `regelbunden` och `transparent` |
| LeftIcon     | IconComponent            | En React-komponent som representerar en ikon. Visas på vänster sida av chippet                               |

</Tab>
</Tabs>

## Exempel

### Transparent Disabled Chip

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Transparent Disabled Chip"
      clickable={false}
      variant="rounded"
      accent="text-secondary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};

```

<br/>

### Inaktiverad Chip med Verktygstips

```jsx
import { Chip } from "twenty-ui/components";
  
export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Disabled chip that triggers a tooltip when overflowing."
      clickable={false}
      variant="regular"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};
```

## Entitetschip

Ett Chip-liknande element för att visa information om en entitet.

<Tabs>

<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';
import { IconTwentyStar } from 'twenty-ui/display';
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Router>
      <Chip
        linkToEntity="/entity-link"
        entityId="entityTest"
        name="Entity name"
        pictureUrl=""
        avatarType="rounded"
        variant="regular"
        LeftIcon={IconTwentyStar}
      />
    </Router>
  );
};
```
