Synkronisera Twenty med 3000+ appar enkelt med [Zapier](https://zapier.com/). Automatisera uppgifter, öka produktiviteten och förbättra kundrelationer!

## Om Zapier

Zapier is a tool that allows you to automate workflows by connecting the apps that your team uses every day. The fundamental concept of Zapier is automation workflows, called Zaps, and include triggers and actions.

Du kan läsa mer om hur Zapier fungerar [här](https://zapier.com/how-it-works).

## Installation

### Steg 1: Installera Zapier-paket

```bash
cd packages/twenty-zapier

yarn
```

### Steg 2: Logga in med CLI

Använd dina Zapier-uppgifter för att logga in via CLI:

```bash
zapier login
```

### Steg 3: Ange miljövariabler

Från mappen `packages/twenty-zapier`, kör:

```bash
cp .env.example .env
```

Kör applikationen lokalt, gå till [http://localhost:3000/settings/api-webhooks](http://localhost:3000/settings/api-webhooks), och generera en API-nyckel.

Ersätt värdet **YOUR_API_KEY** i `.env`-filen med den just genererade API-nyckeln.

## Utveckling

<Warning>

Se till att köra `yarn build` innan du använder något `zapier`-kommando.

</Warning>

### Testa

```bash
yarn test
```

### Lint

```bash
yarn format
```

### Övervaka och kompilera medan du redigerar koden

```bash
yarn watch
```

### Validera din Zapier-app

```bash
yarn validate
```

### Distribuera din Zapier-app

```bash
yarn deploy
```

### Lista alla Zapier CLI-kommandon

```bash
zapier
```
