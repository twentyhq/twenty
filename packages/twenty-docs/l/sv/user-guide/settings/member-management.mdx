## Bjud in nya medlemmar

### Using the Invite Link

1. Gå till **Inställningar → Medlemmar**
2. Kopiera arbetsytans inbjudningslänk
3. Dela länken med nya teammedlemmar
4. De får åtkomst när de registrerar sig

### Direkt e-postinbjudan

1. Gå till **Inställningar → Medlemmar**
2. Ange personens e-postadress
3. Klicka på **Bjud in**
4. De får en e-postinbjudan

## Ta bort medlemmar

### Ta bort en medlem

1. Gå till **Inställningar → Medlemmar**
2. Hitta den medlem du vill ta bort
3. Klicka på knappen radera/ta bort bredvid deras namn
4. Bekräfta borttagningen

**Note**: Removed members lose access immediately but can be re-invited later.
