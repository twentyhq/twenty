## Personlig information

### Namn och e-post

- **Visningsnamn**: Uppdatera hur ditt namn visas för andra medlemmar i arbetsytan
- **E-postadress**: Ändra din inloggnings-e-post (kräver verifiering)
- **Profilbild**: Ladda upp en anpassad avatar eller använd dina initialer

## Säkerhetsinställningar

### Tvåfaktorsautentisering (2FA)

Aktivera 2FA för att lägga till ett extra säkerhetslager till ditt konto:

1. Gå till **Inställningar → Profilinställningar**
2. Klicka på **Aktivera 2FA**
3. Skanna QR-koden med din autentiseringsapp
4. Ange verifieringskoden för att bekräfta

### Lösenordshantering

- **Ändra lösenord**: Uppdatera ditt nuvarande lösenord
- **Lösenordskrav**: Måste vara minst 8 tecken långt

## Profilhantering

### Ta bort konto

<Warning>
Att radera ditt konto kommer permanent att ta bort din åtkomst till alla arbetsplatser. Denna åtgärd kan inte ångras, du kommer att förlora tillgången till alla arbetsytor där du är medlem, och du bör överväga att lämna enskilda arbetsytor istället om du bara vill lämna specifika team.
</Warning>

För att ta bort ditt konto:

1. Gå till **Inställningar → Profilinställningar**
2. Bläddra till **Farlig Zon**
3. Klicka på **Ta bort konto**
4. Bekräfta genom att skriva din e-postadress
