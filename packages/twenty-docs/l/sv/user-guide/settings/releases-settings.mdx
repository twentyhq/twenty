## Latest Releases

Följ Twentys utveckling:

- **Versionsanteckningar**: Se vad som är nytt i varje version
- **Funktionsuppdateringar**: Upptäck nya funktioner och förbättringar

## Labfunktioner

Få tillgång till experimentella funktioner innan de släpps officiellt:

- **Beta-testning**: Prova nya funktioner medan de är under utveckling
- **Funktionsflaggor**: Aktivera eller inaktivera specifik experimentell funktionalitet

## Hur man kommer åt

1. Go to **Settings → Releases**
2. Visa versionsanteckningar och gå till fliken **Lab** för beta-funktioner

För fler community-resurser, kolla vår [Resurssektion](/l/sv/user-guide/resources/glossary).
