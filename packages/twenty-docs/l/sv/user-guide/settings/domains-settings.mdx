## Anpassad arbetsytadomän

Ange en personlig webbadress för din arbetsyta:

- Välj din föredragna subdomän (t.ex. dittföretag.twenty.com)
- Konfigurera anpassade domäner för professionell varumärkesprofilering

## Godkända åtkomstdomäner

Tillåt automatisk arbetsplatsåtkomst för specifika e-postdomäner.

### Hur det fungerar

- Lägg till ditt företags e-postdomäner (t.ex. @dittföretag.com)
- Alla med en e-postadress från dessa domäner kan automatiskt gå med i din arbetsyta
- Inga manuella inbjudningar behövs för teammedlemmar

### Fördelar

- **Snabbare introduktion**: Nya teammedlemmar går med automatiskt
- **Bättre säkerhet**: Endast verifierade företagsmejladresser kan få tillgång
- **Mindre administrativt arbete**: Inga behov av att manuellt bjuda in varje teammedlem

## Installationsinstruktioner

1. Gå till **Inställningar → Domäner**
2. Lägg till din anpassade domän eller godkända e-postdomäner
3. Verifiera domänägande om det behövs

**Notera**: Domänändringar kan ta tid att träda i kraft och kan kräva verifiering.
