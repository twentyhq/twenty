Twentys behörighetssystem ger dig möjlighet att kontrollera åtkomsten till tre huvudområden:

- **Objects and Fields**: Control who can view, edit, or delete records and individual fields
- **Inställningar**: Hantera åtkomst till arbetsytans konfiguration och administrativa funktioner
- **Åtgärder**: Kontrollera allmänna arbetsytans åtgärder som import av data eller att skicka e-post

## Create a Role

För att skapa en ny roll:

1. Gå till **Inställningar → Roller**
2. Under **All Roles**, click on **+ Create Role**
3. Ange ett rollnamn
4. I standardfliken **Behörigheter**, konfigurera behörigheter
5. Klicka **Spara** för att avsluta

## Delete a Role

För att ta bort en roll:

1. Gå till **Inställningar → Roller**
2. Klicka på den roll du vill ta bort
3. Öppna fliken **Inställningar**, klicka sedan på **Ta bort Roll**
4. Klicka **Bekräfta** i modalen

Observera: Om en roll raderas kommer alla arbetsyta medlemmar tilldelade den automatiskt omdirigeras till standardrollen. Alla utom **Admin**-rollen kan raderas. Det måste alltid finnas minst en medlem tilldelad **Admin**-rollen.

## Tilldela Roller till Medlemmar

### View Current Assignments

- Gå till **Inställningar → Roller**
- See all roles and how many members are assigned to each
- Se vilka medlemmar som har vilka roller

### Tilldela en Roll till en Medlem

1. Gå till **Inställningar → Roller**
2. Klicka på rollen du vill tilldela
3. Öppna fliken **Tilldelning**
4. Klicka **+ Tilldela till medlem**
5. Välj arbetsytamedlemen från listan
6. Bekräfta tilldelningen

### Ställ in Standardroll

1. Gå till **Inställningar → Roller**
2. I avsnittet **Alternativ**, hitta **Standardroll**
3. Välj vilken roll nya medlemmar automatiskt ska få
4. Nya medlemmar i arbetsytan kommer att tilldelas denna roll när de ansluter

**Obs**: Du kan bara tilldela roller till befintliga arbetsytamedlemmar. För att bjuda in nya medlemmar, använd [Medlemshantering](/l/sv/user-guide/settings/member-management).

## Anpassa Behörigheter

Behörigheter bestämmer vad varje roll kan komma åt eller ändra inom din arbetsyta, inklusive arbetsyta objekt, inställningar och åtgärder.

### Objekt- och Fältbehörigheter

Kontrollera åtkomst till poster och enskilda fält:

#### Objekt-nivå behörigheter

- Under **Alla Objekt**, tillämpa behörigheter som **Visa Post**, **Redigera Poster**, **Radera Poster** eller **Förstöra Poster** för alla objekt
- Under **Objektnivå Behörigheter**, konfigurera undantag för individuella objekt. Dessa åsidosätter inställningarna från **Alla Objekt**

#### Fältnivå Behörigheter

- Konfigurera behörigheter för enskilda fält inom varje objekt
- Kontrollera vem som kan **Se Fält**, **Redigera Fält**, eller ha **Ingen Åtkomst** till specifika fält
- Fältbehörigheter fungerar på samma sätt som objektbehörigheter med arv och åsidosättningar

#### Hantera Behörighetsåsidosättningar

För att åsidosätta överordnade behörigheter och tillämpa strängare regler:

- Klicka på **X** för att ta bort den ärvda regeln
- Välj de specifika behörigheterna för det valda objektet eller fältet
- Klicka på den orange **Ångra**-ikonen (cirkulär pil) för att återställa ändringar

När klart, klicka **Avsluta**, sedan **Spara** när du omdirigeras till rollsidan.

### Arbetsytans Inställnings Behörigheter

Kontrollera åtkomsten till arbetsytans inställningar på två sätt:

- Växla **Inställningar Full Åtkomst** för att ge full åtkomst
- Eller aktivera specifika behörigheter (t.ex. API-nyckelgenerering, arbetsytans preferenser, rolltilldelning, datamodellkonfiguration, säkerhetsinställningar, och arbetsflödeshantering)

### Arbetsytans Åtgärds Behörigheter

Kontrollera åtkomsten till allmänna arbetsytans åtgärder:

- Växla **Applikation Full Åtkomst** för att ge fulla behörigheter
- Eller aktivera individuella åtgärder såsom **Skicka E-post**, **Importera CSV**, and **Exportera CSV**
