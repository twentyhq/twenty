När du loggar in på Twenty för första gången, bör layouten kännas intuitiv. Den är utformad för att hjälpa dig att arbeta snabbt och vara organiserad, utan att störas.

## Huvudlayouten

Mitten av skärmen är **där dina poster bor**: personer, företag, möjligheter, uppgifter, anteckningar, arbetsflöden och alla andra objekt du har skapat. Det är här det dagliga arbetet sker.
Du kan **visa, redigera, ta bort poster** därifrån samt **skapa nya vyer**.

## Navigeringsfält

På vänster sida, från toppen till botten, kan du:

- Växla mellan dina **flera arbetsytor** med hjälp av rullgardinsmenyn eller skapa en ny arbetsyta
- Välj mellan ljust och mörkt läge
- Använd **sökrutan** (tryck på `/` för att fokusera på den direkt)
- Öppna **inställningsavsnittet**

<Warning>
Please note that our API documentation is accessible under the Settings section and not the User Guide.
</Warning>

- Ha direkt åtkomst till dina **favoritvyer**. Favoriter är unika för varje användare.
- Växla mellan olika objekt
- **Skapa automatiseringar** med hjälp av arbetsflöden
- Ta kontakt med support och öppna vår användarguide.

## Kommandomeny & Snabbsökning

Kommandomenyn ger dig **snabb åtkomst till åtgärder och sökning** i Twenty. Du kan få tillgång till den på två sätt:

- **Tangentbordsgenväg**: Tryck på `Cmd + K` (Mac) eller `Ctrl + K` (Windows)
- **Mus**: Klicka på de tre prickarna i övre högra hörnet

Du kommer också att se en sökruta högst upp i din sidopanel för snabba post-sökningar, eller tryck på `/` för att fokusera på den direkt.

Därifrån kan du:

- Skapa nya poster
- **Importera och exportera data via csv**
- Skapa nya vyer
- Kom åt borttagna poster (Twenty stödjer både mjuka och hårda raderingar)
- Se tangentbordsgenvägar för att snabbt få tillgång till objekt i din arbetsyta

## Visningar

Varje objekt (som möjligheter eller personer) stödjer flera vyer. Du är inte begränsad i antalet vyer per objekt.

Använd rullgardinsmenyn uppe till vänster i huvudlayouten för att växla mellan de olika vyerna. Till exempel:

- Använd en Kanban-vy för att spåra möjligheter efter fas
- Använd Group By-vy för att skapa sektioner och öka effektiviteten
- Använd filter för att fokusera på specifika poster (t.ex. leads som skapades förra veckan)
- Spara filtervyer för att återanvända dem senare
- Favoritvyer för snabb åtkomst

Om du är ny på vyer, läs vår artikel [Hantera vyer](/l/sv/user-guide/crm-essentials/view-management) för att lära dig hur du skapar och anpassar dem.

## Inställningar

Öppna dina inställningar från övre vänstra hörnet för att:

- **Koppla din e-post och kalender** för smidig synkronisering av e-post och kalender
- Anpassa din **datamodell**: skapa anpassade objekt, fält och relationer
- **Access the API playground and configure webhooks**
- **Hantera användarbehörigheter** och åtkomstkontroller för arbetsyta
- Invite team members and manage user roles
- Redigera din profil och arbetsytainställningar
- Konfigurera fakturering och övervaka användningen av arbetsflödeskrediter
- Upptäck de senaste versionerna och kommande funktioner (under Releases → Lab-fliken)

Om du inte ser alla dessa avsnitt under inställningar, kontakta din arbetsytas administratör - några av dem har begränsad åtkomst.
