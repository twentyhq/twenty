## Om Tabellvyer

Tabellvyer är visuella representationer av data strukturerade i rader och kolumner.

## Skapa post

Lägg till poster efter behov, utan begränsningar. För att lägga till en post kan du antingen klicka på **+**-knappen högst upp till höger på skärmen eller högst upp i postens **Namn**-kolumn.

Skriv in postens namn och tryck på `Enter` för att spara. För att redigera ett postnamn, klicka på dess namn på dess detaljsida.

<VimeoEmbed videoId="927071691" title="Video demonstration" />

## Ta bort post

**Indexvy:** För att ta bort en post, markera kryssrutan bredvid posten och klicka på knappen för borttagning i det övre högra hörnet.

**Record Page:** Tap the `⋮` icon in the top right corner, then select delete.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## Lägg till ett anpassat fält

För att skapa ett anpassat fält, klicka på **+**-knappen längst till höger i tabellkolumnerna och välj **Anpassa fält**.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

Du kan också göra det genom att navigera till **Inställningar** > **Datamodell** > **Personer**. Klicka på **Lägg till fält**. Välj ett fältnamn och typ. Det nya fältet kommer att vara tillgängligt i appen.
