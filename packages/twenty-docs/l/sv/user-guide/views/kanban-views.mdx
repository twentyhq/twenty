## About Kanban Views

Kanbanvyer kartlägger visuellt processflöden, där varje kolumn representerar en distinkt fas och varje kort representerar en post.

## Flytta kort mellan faser

Du kan flytta varje kort mellan faser när det genomgår ditt arbetsflöde genom att dra och släppa. För att fortsätta, håll ner klicket på ett kort och flytta det till nästa fas.

## Lägg till och ta bort faser

Du kan skräddarsy ditt arbetsflöde så att det passar dina behov genom att använda faser, som representerar ett värde i ett Urvalsfält:

### Lägg till faser

För att lägga till en fas, gå till inställningarna för Urvalsfält genom att navigera till Inställningar > Datamodell, välj ditt objekt och sedan fältet ditt Kanban-bräda är beroende av.

<VimeoEmbed videoId="927890428" title="Video demonstration" />

### Ta bort faser

För att ta bort en fas, sväva över fasens namn eller `⋮`-ikonen, klicka på `Redigera från inställningar` i inställningarna för Urvalsfält, och klicka sedan på <b>Ta bort</b> bredvid den relevanta fasen.

<img src="/images/user-guide/kanban-views/edit-stage.png" style={{width:'100%'}}/>

## Visa fält

Du kan konfigurera din Kanban-tavla för att visa vissa fält och dölja andra. För att dölja ett fält, klicka på <b>Alternativ</b> uppe till höger, sedan på <b>Fält</b> för att ta fram listan över alternativ. Håll muspekaren över fältet du vill dölja för att visa `-`-knappen. Klicka på den för att dölja fältet.

Du kan också ordna om ordningen på fälten genom att hålla ner fältets namn och dra det dit du vill.

<img src="/images/user-guide/kanban-views/filter.png" style={{width:'100%'}}/>

## Kompakt vy

Du kan också dölja alla fält och få en översikt av alla möjligheter på ett ögonblick. För att göra det, klicka på <b>Alternativ</b> uppe till höger och aktivera växeln för <b>Kompakt vy</b> efter att ha valt layout i Kanban-vy.

<img src="/images/user-guide/kanban-views/compact-view.png" style={{width:'100%'}}/>
