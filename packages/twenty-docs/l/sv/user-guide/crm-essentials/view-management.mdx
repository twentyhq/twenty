## Layoutalternativ

Du kan visa dina data i tre olika layouter, var och en anpassad för olika ändamål. Anpassade layouter för att anpassa hur sidan ser ut för varje typ av post kommer att lanseras i december 2025.

### Standardvy

Varje objekt har en ofiltrerad, osorterad och oavlägsningsbar vy känd som standardvyn. It's named after the object's plural name, such as`All Companies`,`All People`, `All Opportunities`.
<img src="/images/user-guide/views/default-view.png" style={{width:'100%'}}/>

### Listlayout

Det vanliga tabellformatet som visar poster i rader och kolumner. Detta är perfekt för att se detaljerad information i ett ögonblick och jämföra poster sida vid sida.

### List Group By Layout

Organiserar dina poster genom att gruppera dem baserat på ett valt fält. Exempel: du kan gruppera möjligheter efter fas, företag efter plats, eller något annat valt fält. Detta hjälper dig att se mönster och organisera relaterade poster tillsammans.

### Kanban Layout

En visuell tavla där varje kolumn representerar ett steg och varje post visas som ett kort. This layout is ideal for managing pipelines and workflows where records move through different stages. För mer information om hur du använder Kanban-vyer för pipelinehantering, se vår [Pipeline](/l/sv/user-guide/crm-essentials/pipeline) artikel.

## Skapa nya vyer

There are three ways to create a new view:

### Genom att använda Kommandomenyn

Använd `Cmd+K` på Mac (eller `Ctrl+K` på Windows) för att öppna kommandomenyn, klicka sedan på `Skapa en ny vy`.

### Genom att använda Visningsmenyn

Klicka på visningsmenyn (vänster överst), klicka sedan på `Lägg till vy` längst ner. Från denna meny kan du:

- Välj en ikon och namn för din vy
- Välj layouttypen (Lista eller Kanban)

<Warning>
Viktigt: Du måste först välja listlayouten och sedan lägga till Group By. Du kan inte skapa en Grupplista direkt därifrån.
</Warning>
- For Kanban views, select which select field to use as column headers

### Skapa vyer från befintliga filter

När du ändrar sorteringen och filtreringen av en befintlig vy visas en knapp `Spara som ny vy`. Detta låter dig skapa en ny vy baserat på dina nuvarande anpassningar.

## Anpassa dina vyer

Alla layouter stöder samma anpassningsalternativ: sortering, filtrering och fältval. Du kan göra snabba engångsändringar genom att klicka direkt på kolumnnamnet, eller använda `Valmöjlighetsmenyn` för mer omfattande redigering.

### Snabba åtgärder mot alternativmeny

**För engångsändringar**: Klicka direkt på kolumnrubriker för att sortera, eller använd `Flytta vänster`, `Flytta höger`-knapparna.

**För flera redigeringar**: Använd `Valmöjlighetsmenyn` (överst till höger) när du vill göra flera ändringar i rad. Denna meny ger dig tillgång till:

- **Layoutval** (Lista, Gruppvisning, Kanban)
- **Grupperingsalternativ** (för gruppvisningslayout)
- **Fälthantering** (visa/dölj och omordna kolumner)

### Filtrera dina data

Du kan tillämpa filter för att bara visa de poster som matchar dina kriterier. Klicka på `Filter` i verktygsfältet, välj ett fält, välj ditt villkor och ställ in värdet. Du kan lägga till flera filter för avancerad filtrering baserat på flera villkor.

### Sortera dina poster

Kontrollera ordningen på dina poster genom att klicka på en kolumnrubrik för att sortera efter det fältet. Klicka igen för att vända sorteringsordningen. Du kan tillämpa flera sorteringar för komplex organisation.

<VimeoEmbed videoId="927885588" title="Video demonstration" />

### Hantera fält och kolumner

Du kan välja vilka fält som ska visas och ordna dem efter önskemål. För snabba ändringar, klicka direkt på ett kolumnnamn. För flera redigeringar, använd `Valmöjlighetsmenyn → Fält` där du kan:

- Visa eller dölja fält med ögonikonen
- Omarrangera fält genom att dra och släppa
- Gör flera förändringar effektivt på ett ställe

## Hantera dina vyer

### Funktioner i visningsmenyn

Visningsmenyn (vänster överst) är din centrala nav för vyhantering. Härifrån kan du:

- **Redigera vynamn och ikoner**: Klicka på de tre punkterna bredvid en vy
- **Ordna om vyer**: Dra och släpp vyer för att organisera dem efter prioritet
- **Spara vyer som favoriter**: Favoriter visas precis under Inställningar för snabb åtkomst

### Redigera och ta bort vyer

To modify or remove views, open the view dropdown menu and hover over the view you want to change. Klicka på de tre punkterna som visas för att komma åt redigerings- och borttagningsalternativ.

### Favoriter och organisation

Vyer som sparas som favoriter visas precis under Inställningar i din navigering och ger dig direkt tillgång till dina viktigaste vyer. Använd visningsmenyn för att organisera dina vyer genom att dra dem till ordningen som fungerar bäst för ditt arbetsflöde.

För mer avancerad dataorganisation, se vår [Datamodellavdelning](/l/sv/user-guide/data-model/customize-your-data-model) för att lära dig om att anpassa fält och objekt.
