## Förstå pipelines

En försäljningspipeline spårar möjligheter från första kontakt till avslutat avtal. Varje steg representerar ett steg i din försäljningsprocess, och möjligheterna går igenom dessa steg när de närmar sig avslut.

Twenty includes standard sales stages like Prospecting, Qualification, Proposal, Negotiation, Closed. Du kan anpassa dessa steg för att matcha din specifika försäljningsprocess.

## Arbeta med Kanban-vyer

Kanban-vyer visualiserar din pipeline där varje kolumn representerar ett steg och varje kort representerar en möjlighet. Varje kort visar viktig information som affärsvärde, avslutsdatum och ansvarig person vid en överblick. For complete details: including notes, tasks, meetings, and email history, click on any card to open the full opportunity record.

### Flytta möjligheter genom din pipeline

Du kan flytta varje möjlighet mellan steg när den fortskrider genom din försäljningsprocess genom att dra och släppa. Håll ned med klick på ett kort och flytta det till nästa steg.

### Anpassa dina pipeline-steg

Du kan skräddarsy din pipeline för att passa din specifika försäljningsprocess. Stages represent values in a Select Field, so you can add, remove, or rename them as needed.

#### Lägga till steg

För att lägga till ett steg, gå till inställningarna för Välj-fält genom att navigera till Inställningar → Datamodell, välja ditt objekt och sedan fältet din Kanban-tavla beror på.

<VimeoEmbed videoId="927890428" title="Video demonstration" />

#### Ta bort steg

För att ta bort ett steg, håll musen över stegets namn eller `⋮`-ikonen, klicka på `Redigera från inställningar` i Välj-fältets inställningar, och klicka sedan på `Ta bort` bredvid det aktuella steget.

<img src="/images/user-guide/kanban-views/edit-stage.png" style={{width:'100%'}}/>

### Anpassa korten

Du kan konfigurera din Kanban-tavla för att visa vissa fält och dölja andra. För att dölja ett fält, klicka på `Alternativ` uppe till höger och sedan `Fält` för att ta fram listan med alternativ. Håll muspekaren över fältet du vill dölja för att visa `-`-knappen. Klicka på den för att dölja fältet.

Du kan också ordna om ordningen på fälten genom att hålla ner fältets namn och dra det dit du vill.

<img src="/images/user-guide/kanban-views/filter.png" style={{width:'100%'}}/>

### Kompakt vy

Du kan också dölja alla fält och få en översikt av alla möjligheter på ett ögonblick. För att göra det, klicka på `Alternativ` uppe till höger och slå på reglaget för `Kompakt vy` efter att ha valt layout i kanban-vy.

<img src="/images/user-guide/kanban-views/compact-view.png" style={{width:'100%'}}/>

## Avancerad pipelinehantering

### Automation with workflows

Use [Workflows](/l/sv/user-guide/workflows/getting-started-workflows) to automate your pipeline:

- **Automatisk stegförflyttning**: Flytta affärer baserat på aktiviteter
- **Notifications**: Alert team members of stage changes
- **Task creation**: Generate follow-up tasks for each stage

### Flera pipelines

Du kan skapa olika pipelines för olika affärslinjer, marknadssegment eller specialiserade försäljningsteam genom att skapa nya vyer. Varje vy kan visa olika möjligheter med specifika filter och steg anpassade till dina behov. Lär dig hur du skapar dessa vyer i vår [Vyhanteringsguide](/l/sv/user-guide/crm-essentials/view-management).
