Hantera dina registerkopplade anteckningar effektivt med den kraftfulla **Anteckningar**-funktionen. Den här guiden går igenom hur du skapar, formaterar, kommenterar och tar bort anteckningar sömlöst på registersidor.

## Vanliga Användningsområden

### Mötesdokumentation

- **Mötesprotokoll**: Anteckna viktiga diskussionspunkter, beslut och åtgärdsposter från kundsamtal.
- **Samtalssammanfattningar**: Registrera viktiga detaljer från försäljningssamtal eller support-samtal.
- **Uppföljningsanteckningar**: Dokumentera nästa steg och åtaganden som görs under möten.

### Kundinteraktioner

- **Supporthistorik**: Spåra kundproblem, tillhandahållna lösningar och status för lösning.
- **Försäljningskontext**: Registrera kundpreferenser, svaga punkter, och köpsignaler.
- **Relationship Building**: Note personal details about contacts to strengthen relationships

### Projektledning

- **Statusuppdateringar**: Dokumentera projektets framsteg och uppnådda milstolpar.
- **Problemspårning**: Logga problem som uppstått och lösningar som implementerats.
- **Teamöverlämningar**: Dela kontext när konto överförs mellan teammedlemmar.

### Automated Note Creation

Använd [arbetsflöden](/l/sv/user-guide/workflows/getting-started-workflows) för att automatiskt skapa anteckningar:

- **Samtalsinspelareintegration**: Auto-generera mötessammanfattningar från inspelade samtal.
- **Överlämningsanteckningar för avtal**: Auto-skapa sammanfattningar av försäljningscykler vid överlämning av nya kunder till implementeringsteam.

## Anteckningsfunktioner

### Relationsfält

Anteckningar innehåller ett **relationsfält** som tillåter dig att bifoga en enda anteckning till flera poster över olika objekt. Till exempel, du kan länka en mötesanteckning till:

- Personen du träffade med
- Företaget de representerar
- Möjligheten som diskuteras
- Alla relevanta uppgifter eller andra poster

Denna flera relationsform säkerställer att viktig information är tillgänglig från alla relevanta post-sidor.

### Användartaggning

**Anteckning**: Användartaggning inom anteckningar är för närvarande inte tillgänglig. Denna funktion är planerad för 2026, vilket kommer tillåta dig att nämna teammedlemmar och trigga notifikationer.

## Skapa Anteckningar

Att skapa anteckningar i systemet är intuitivt och dynamiskt. Du kan antingen:

- Navigera till anteckningsvyn och skapa en ny post.
- Gå till en `Post-sida` och välj fliken Anteckningar och tryck på knappen `Ny anteckning`.

<img src="/images/user-guide/notes/add-note.png"style={{width:'100%'}}/>

### Lägga till innehåll

Börja skriva direkt eller tryck `/` för att lägga till element som rubriker, filer eller bilder direkt.

### Formatera Innehåll

Du kan formatera dina anteckningar direkt från redigeraren. Använd Markdown-syntax, tryck på `/` knappen eller klicka på `+` ikonen på redigeraren för att se de olika blockalternativen, som rubriker, tabeller och listor. Du kan också bifoga bilder till din anteckning.

Markera texten för att se fler formateralternativ som fetstil, kursiv, och justeringsalternativ.

Du kan också ändra bakgrundsfärgen och textfärgen för varje block för att framhäva viktiga saker i din anteckning. För att göra detta, håll musen över blocket du vill formatera och klicka på `⋮` ikonen bredvid `+` ikonen. Klicka på `Färger` för att öppna alla färgalternativ för både text och bakgrund.

## Visa Anteckningar

Systemet visar alla dina anteckningar kopplade till en viss post under avsnittet Anteckningar på motsvarande `Post-sida`.

## Spara och Ta Bort

Alla redigeringar och tillägg till anteckningen sparas automatiskt.

För att radera en anteckning:

1. Öppna anteckningen du vill ta bort genom att klicka på den inom `Post-sidor`.
2. Välj den anteckning du vill radera inom fliken anteckningar.
3. Använd `Option` knappen i nedre högra hörnet för att utföra ytterligare åtgärder inklusive radering.
4. Slutför din radering när du uppmanas av bekräftelsemodellen.

Ett annat sätt att radera en anteckning är genom anteckningsvyn precis som du skulle göra med en vanlig post. Observera att radering av en anteckning är permanent och kan inte ångras.
