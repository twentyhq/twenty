## Aktuella integrationsalternativ

### Inbyggda integrationer

Twenty erbjuder för närvarande inbyggda integrationer med:

- **E-post & Kalender**: Anslut Gmail, Outlook eller SMTP/CalDAV-leverantörer
- **API-åtkomst**: Använd våra REST- och GraphQL-API:er för att bygga egna integrationer

För inställning av e-post och kalender, besök [E-post & Kalenderinställningar](/l/sv/user-guide/settings/email-calendar-setup).

### Workflows (Recommended)

Det primära sättet att ansluta Twenty till andra verktyg är genom vår funktion **Arbetsflöden** inuti appen:

- **HTTP-noder**: Utför API-samtal till externa tjänster
- **Kodnoder**: Skriv anpassad logik för komplexa integrationer
- **Webhook-utlösare**: Ta emot data från externa system
- **Fältuppdateringsutlösare**: Automatisera åtgärder baserat på CRM-förändringar

Läs mer i vår [Arbetsflödensektion](/l/sv/user-guide/workflows/getting-started-workflows).

### Zapier-integration (Äldre)

Vi upprätthåller en Zapier-integration för användare som föredrar kodlös automatisering:

1. Besök [Twenty på Zapier](https://zapier.com/apps/twenty/integrations)
2. Skapa en ny Zap med Twenty som utlösare eller åtgärd
3. Generera en API-nyckel i Inställningar → API & Webhooks
4. Anslut din Twenty-arbetsyta till Zapier

## Framtidsvision: Gemenskapsbyggda kopplingar (2026)

### Utbyggnadsplattform

Vi bygger en utbyggnadsplattform som gör det möjligt för utvecklare att skapa appar som kod. Detta kommer att möjliggöra:

- **Anpassade Kopplingar**: Bygg integrationer med din favoritprogramvara
- **Gemenskapsbidrag**: Dela och upptäck kopplingar som andra har byggt
- **Flexibel Arkitektur**: Utöka Twentys möjligheter bortom kärnfunktioner

### AI-assisterat arbetsflödesbyggande

Kommer 2026, AI-assistans kommer att hjälpa användare:

- **Autogenera arbetsflöden**: Beskriv dina integrationsbehov i enkelt språk
- **Smarta Förslag**: Få rekommendationer för att ansluta dina specifika verktyg
- **Mallbibliotek**: Få tillgång till förbyggda arbetsflöden för vanliga användningsfall

Twentys framtida integrationer är gemenskapsdrivna, AI-assisterade och oändligt utbyggbara.
