## API-nycklar

API-nycklar möjliggör automatiserad åtkomst till din CRM-data, synkronisera data med andra system och skapa anpassade integrationer eller lösningar.

### Skapa en API-nyckel

1. Gå till **Inställningar → API:er & Webhooks**
2. Klicka på **+ Skapa nyckel** uppe i högra hörnet
3. Konfigurera din API-nyckel:
   - **Namn**: Ge din API-nyckel ett beskrivande namn
   - **Utgångsdatum**: Ställ in när nyckeln ska löpa ut
4. Klicka på **Spara** för att generera din API-nyckel
5. **Viktigt**: Kopiera och spara din API-nyckel omedelbart eftersom den bara visas en gång

När den är skapad ger din API-nyckel åtkomst till din anpassade API-dokumentation och testmiljö där du kan testa ändpunkter med din faktiska datamodell.

<Warning>
Eftersom din API-nyckel ger åtkomst till känslig information bör du inte dela den med tjänster du inte helt litar på. Om den läcker, kan någon använda den på ett skadligt sätt. Om säkerheten för din API-nyckel komprometteras, inaktivera den omedelbart och generera en ny.
</Warning>

### Hantera API-nycklar

**Återskapa en API-nyckel:**

1. Gå till **Inställningar → API:er & Webhooks**
2. Klicka på den API-nyckel du vill återskapa
3. Klicka på **Återskapa**-knappen
4. Kopiera och lagra den nya API-nyckeln omedelbart

**Ta bort API-nyckel:**

1. Hitta API-nyckeln i din lista
2. Klicka på nyckeln för att öppna dess detaljer
3. Klicka på **Ta bort** för att ta bort den permanent

## Webkrokar

Webhooks möjliggör omedelbara uppdateringar till din angivna URL om ändringar eller händelser relaterade till dina kunddata.

Till exempel, när en Möjlighet flyttas till "Avslutad Vunnen", kan en webhook automatiskt starta fakturaskapande i ditt bokföringssystem. Observera att denna typ av automation även kan uppnås med Twenty's in-app [Arbetsflödesfunktion](/l/sv/user-guide/workflows/getting-started-workflows), som erbjuder triggers baserade på fältuppdateringar för intern automation.

Webhooks är perfekta för att integrera med externa system, medan Arbetsflöden stödjer både intern automation och externa verktygsanslutningar via webhooktriggers, kodnoder och HTTP-noder.

### Skapa Webhook

1. Gå till **Inställningar → API & Webhooks → Webhooks**
2. Klicka på **+ Skapa webhook**
3. Ange din webhook-URL (där du vill ta emot meddelanden)
4. Klicka **Spara**

Din webhook kommer omedelbart att börja ta emot realtidsmeddelanden om ändringar i dina CRM-data.

### Hantera Webhooks

**Ta bort webhook:**

1. Gå till **Inställningar → API & Webhooks → Webhooks**
2. Hitta webhooken du vill ta bort
3. Klicka på webhooken
4. Klicka **Ta bort** och bekräfta i popup-fönstret

**Redigera en webhook:**

1. Klicka på webhooken du vill ändra
2. Uppdatera URL:en eller andra inställningar
3. Klicka **Spara** för att tillämpa ändringar
