## Latest Releases

Volg Twenty se ontwikkelingsvordering:

- **Release Notes**: See what's new in each version
- **Kenmerkopdaterings**: Ontdek nuwe vermoëns en verbeterings

## Lab Kenmerke

Toegang tot eksperimentele kenmerke voordat hulle amptelik vrygestel word:

- **Beta Toetsing**: Probeer nuwe kenmerke uit terwyl hulle in ontwikkeling is
- **Kenmerkvlae**: Skakel spesifieke eksperimentele funksionaliteit in of uit

## How to Access

1. Gaan na **Instellings → Vrylatings**
2. Kyk na vrylatingaantekeninge en toegang die **Lab** blad vir beta kenmerke
