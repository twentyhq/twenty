## Persoonlike Werksarea Domein

Stel 'n persoonlike webadres op vir jou werksarea:

- Choose your preferred subdomain (e.g., yourcompany.twenty.com)
- Stel persoonlike domeine in vir professionele handelsmerk

## Goedgekeurde Toegangsdomeine

Laat outomatiese werksarea toegang toe vir spesifieke e-pos domeine.

### Hoe Dit Werk

- Voeg jou maatskappy se e-pos domeine by (bv. @joumaatskappy.com)
- Enigiemand met 'n e-pos van hierdie domeine kan outomaties by jou werksarea aansluit
- Geen handmatige uitnodigings nodig vir spanlede nie

### Voordele

- **Vinniger Verwelkoming**: Nuwe spanlede sluit outomaties aan
- **Beter Sekuriteit**: Slegs geverifieerde maatskappy e-posse kan toegang verkry
- **Less Admin Work**: No need to manually invite each team member
