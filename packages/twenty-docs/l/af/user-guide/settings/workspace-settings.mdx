## Werkruimte Naam

- **Naam**: Verander jou werkspasie se vertoonnaam
- Hierdie naam verskyn aan alle werkspasielede

## Gevaar Sone

<Warning>
Die verwydering van jou werkruimte verwyder alle data permanent en kan nie ongedaan gemaak word nie. Alle werkspasie data sal vir altyd verlore wees, alle lede sal toegang onmiddellik verloor, en hierdie aksie kan nie herroep word nie.
</Warning>

Om jou werkspasie uit te vee:

1. Klik op **Verwyder werkspasie** knoppie
2. Bevestig die uitvee wanneer gevra word

**Let wel**: Slegs werkspasie-administrateurs kan werkspasies uitvee.
