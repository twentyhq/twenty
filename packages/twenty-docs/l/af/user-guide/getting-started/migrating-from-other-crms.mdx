## Voordat Jy Begin

### 1. Oudit Jou Huidige Data

- **Kies die paar objekte en velde om te migreer**: hierdie migrasie is die geleentheid vir 'n vars begin
- **Eksporteer hierdie data** vanaf jou huidige CRM
- **Verwyder duplikate** en verouderde rekords
- **List active workflows** and automations

### 2. Skep Jou Nuwe Datamodel

Follow our [data model guide](/l/af/user-guide/data-model/customize-your-data-model) to:

- **Ontwerp die datamodel** wat jy nodig het
- **Karteer bestaande velde** na Twenty se standaard objekte / velde
- **Identifiseer die persoonlike objekte / velde** wat jy nodig sal hê
- **Skep hulle** onder Instellings → Datamodel

## Migrasie Proses

### 1. Voer Jou Data In

**Aanbevole volgorde:**

1. **Maatskappye** eerste (as basisrekords)
2. **Mense** tweede (gekoppel aan maatskappye)
3. **Geleenthede** derde (gekoppel aan mense/maatskappye)

Gebruik die CSV invoer via die Kommando Kieslys `Cmd + K` (Mac) of `Ctrl + K` (Windows). Lees ons [data invoer gids](/l/af/user-guide/getting-started/import-export-data) vir gedetailleerde instruksies.

### 2. Herskep Werkverloe

- **Begin eenvoudig** - herskep jou mees kritieke outomatiserings eerste
- **Use Twenty's workflow builder** to replace existing automations

## Algemene Uitdagings

### Data Formaat Kwessies

- **E-posadresse** - verwyder duplikate (Persoon objek vereiste)
- **Domein** - verwyder duplikate (Maatskappy objek vereiste)

<Warning>
Please note that domain URLs created by the synchronization with your mailbox and calendar have the following format ```https://domain.com```
</Warning>

- **Datumformate** - verseker konsekwente formatering (JJJJ-MM-DD) of wysig hierdie formaat onder Instellings → Ervaring
- **Telefoonnommers** - gebruik internasionale formaat (+1234567890)

### Verhouding Kaartlegging

Om verhoudings tussen rekords te importeer deur die csv invoerfunksie, kan jy die volgende velde gebruik

- **Gebruik Twenty ID's** vir komplekse verhoudings
- **Gebruik e-posadresse** om Mense rekords te koppel
- **Gebruik domeinname** om Maatskappy rekords te koppel
- **Gebruik enige ander veld wat jy as uniek stel**, wat gedoen kan word in die Data Model afdeling.
  Lees ons [invoer-uitvoer data gids](/l/af/user-guide/getting-started/import-export-data) vir gedetailleerde instruksies oor die skep van verhoudings tydens CSV invoer.

As jy vanaf Twenty self-gehosts na Twenty Wolk verskuif:
