## Reporting Bugs

Om 'n fout te rapporteer, skep asseblief [‘n saak op GitHub](https://github.com/twentyhq/twenty/issues/new).

Jy kan ook vra vir hulp op [Discord](https://discord.gg/cx5n4Jzs57).

## Funksie Versoeke

As jy nie seker is of dit 'n fout is nie, en jy voel dis nader aan 'n funksieversoek, moet jy waarskynlik eerder [‘n bespreking oopmaak](https://github.com/twentyhq/twenty/discussions/new).
