<Tabs>
<Tab title="Usage">

```jsx
import { ImageInput } from "@/ui/input/components/ImageInput";

  return <ImageInput/>;

```

</Tab>
<Tab title="Props">

| Eienskappe       | Tipe       | Beskrywing                                                                                                                          |
| ---------------- | ---------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| prent            | string     | Die beeld se bron-URL                                                                                                               |
| onUpload         | funksie    | Die funksie wat geroep word wanneer 'n gebruiker 'n nuwe beeld oplaai. Dit ontvang die `File` objek as 'n parameter |
| onRemove         | funksie    | Die funksie wat geroep word wanneer die gebruiker op die verwyderknoppie klik                                                       |
| onAbort          | funksie    | Die funksie wat geroep word wanneer 'n gebruiker op die staakknoppie tydens beeldoplaai klik                                        |
| isBesigMetOplaai | booleaanse | Dui aan of 'n beeld tans besig is om opgelaai te word                                                                               |
| foutBoodskap     | string     | 'n Opsionele foutboodskap om onder die beeldinvoer te wys                                                                           |
| gedeaktiveer     | booleaanse | If `true`, the entire input is disabled, and the buttons are not clickable                                                          |

</Tab>
