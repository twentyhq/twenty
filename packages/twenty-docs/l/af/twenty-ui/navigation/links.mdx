## Kontak Skakel

'n Gestileerde skakelkomponent vir die vertoon van kontakbesonderhede.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Eienskappe | Tipe              | Beskrywing                                                           |
| ---------- | ----------------- | -------------------------------------------------------------------- |
| klasNaam   | string            | Opsionele naam vir bykomende styl                                    |
| href       | string            | Die teiken URL of pad vir die skakel                                 |
| onClick    | funksie           | Terugroepfunksie wat geaktiveer word wanneer die skakel gekliek word |
| kinders    | `React.ReactNode` | Die inhoud wat binne die skakel vertoon moet word                    |

</Tab>

</Tabs>

## Sosiale Skakel

Gestileerde sosiale skakels, met ondersteuning vir verskeie sosiale skakelsoorte, soos URL's, LinkedIn, en X (of Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Eienskappe | Tipe              | Beskrywing                                                                                                 |
| ---------- | ----------------- | ---------------------------------------------------------------------------------------------------------- |
| href       | string            | Die teiken URL of pad vir die skakel                                                                       |
| kinders    | `React.ReactNode` | Die inhoud wat binne die skakel vertoon moet word                                                          |
| tipe       | string            | Die tipe sosiale skakels. Opsies sluit in: `url`, `LinkedIn`, en `Twitter` |
| onClick    | funksie           | Terugroepfunksie wat geaktiveer word wanneer die skakel gekliek word                                       |

</Tab>
</Tabs>
