## Obecne opcje integracji

### Natywne integracje

Twenty obecnie oferuje natywną integrację z:

- **E-mail i Kalendarz**: Połącz Gmail, Outlook, lub dostawców SMTP/CalDAV
- **Dostęp do API**: Używaj naszych API REST i GraphQL do tworzenia niestandardowych integracji

Aby ustawić konfigurację e-mail i kalendarza, odwiedź [Konfiguracja E-mail i Kalendarza](/l/pl/user-guide/settings/email-calendar-setup).

### Przepływy pracy (Zalecane)

Główną metodą łączenia Twenty z innymi narzędziami jest funkcja **Przepływy pracy** w aplikacji:

- **Węzły HTTP**: Wykonuj wywołania API do zewnętrznych usług
- **Węzły kodu**: Pisz niestandardową logikę dla złożonych integracji
- **Wyzwalacze webhook**: Otrzymuj dane z zewnętrznych systemów
- **Wyzwalacze aktualizacji pól**: Automatyzuj działania na podstawie zmian w CRM

Dowiedz się więcej w naszej [sekcji Przepływy pracy](/l/pl/user-guide/workflows/getting-started-workflows).

### Integracja z Zapier (Legacy)

Utrzymujemy integrację z Zapier dla użytkowników preferujących automatyzację bez kodu:

1. Odwiedź [Twenty na Zapier](https://zapier.com/apps/twenty/integrations)
2. Utwórz nowego Zapa z Twenty jako wyzwalaczem lub akcją
3. Wygeneruj klucz API w Ustawienia → API i webhooks
4. Połącz swoje środowisko pracy Twenty z Zapier

## Wizja przyszłości: Konektory budowane przez społeczność (2026)

### Platforma rozszerzalności

Tworzymy platformę rozszerzalności, która umożliwi deweloperom tworzenie aplikacji jako kod. To umożliwi:

- **Niestandardowe konektory**: Twórz integracje z ulubionym oprogramowaniem
- **Wkład społeczności**: Udostępniaj i odkrywaj konektory stworzone przez innych
- **Elastyczna architektura**: Rozszerzaj możliwości Twenty poza podstawowe funkcje

### Tworzenie przepływów pracy wspomagane sztuczną inteligencją

W 2026 roku pomoc AI będzie wspierać użytkowników:

- **Automatyczne generowanie przepływów pracy**: Opisz swoje potrzeby integracyjne w prostym języku
- **Inteligentne sugestie**: Otrzymuj rekomendacje dotyczące połączenia swoich specyficznych narzędzi
- **Biblioteka szablonów**: Uzyskaj dostęp do zbudowanych już przepływów pracy dla typowych przypadków użycia

Przyszłość integracji Twenty to społecznościowy, wspomagany AI i nieskończenie rozszerzalny.
