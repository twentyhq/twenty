## Zaproś nowych członków

### Za pomocą linku zaproszenia

1. Przejdź do **Ustawienia → Członkowie**
2. Skopiuj link do zaproszenia do obszaru roboczego
3. Udostępnij link nowym członkom zespołu
4. Otrzymają dostęp po zarejestrowaniu się

### Bezpośrednie zaproszenie e-mailowe

1. Przejdź do **Ustawienia → Członkowie**
2. Wprowadź adres e-mail osoby
3. Kliknij **Zaproś**
4. Otrzymają zaproszenie e-mailowe

## Usuń członków

### Usuń członka

1. Przejdź do **Ustawienia → Członkowie**
2. Znajdź członka, którego chcesz usunąć
3. Kliknij przycisk usuwania znajdujący się obok jego nazwiska
4. Potwierdź usunięcie

**Uwaga**: Usunięci członkowie tracą dostęp natychmiast, ale mogą być ponownie zaproszeni później.
