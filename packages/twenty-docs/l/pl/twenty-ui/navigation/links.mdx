## Link Kontaktowy

Stylizowany element łącza do wyświetlania informacji kontaktowych.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Właściwości (Props) | Typ               | Opis                                                |
| -------------------------------------- | ----------------- | --------------------------------------------------- |
| nazwaKlasy                             | ciąg znaków       | Opcjonalna nazwa dla dodatkowego stylowania         |
| href                                   | ciąg znaków       | Docelowy URL lub ścieżka dla łącza                  |
| poKliknięciu                           | funkcja           | Funkcja zwrotna do uruchomienia po kliknięciu linku |
| children                               | `React.ReactNode` | Treść, która ma być wyświetlana wewnątrz łącza      |

</Tab>

</Tabs>

<Tabs>
<Tab title="Usage">

```jsx

  return (

};
```

## Link Społecznościowy

Stylizowane linki społecznościowe z obsługą różnych typów linków społecznych, takich jak URL, LinkedIn i X (lub Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Właściwości (Props) | Typ               | Opis                                                                                                         |
| -------------------------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------ |
| href                                   | ciąg znaków       | Docelowy URL lub ścieżka dla łącza                                                                           |
| children                               | `React.ReactNode` | Treść, która ma być wyświetlana wewnątrz łącza                                                               |
| typ                                    | ciąg znaków       | Typ linków społecznościowych. Opcje obejmują: `url`, `LinkedIn`, i `Twitter` |
| poKliknięciu                           | funkcja           | Funkcja zwrotna do uruchomienia po kliknięciu linku                                                          |

</Tab>
</Tabs>
