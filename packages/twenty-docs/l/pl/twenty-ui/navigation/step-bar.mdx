Wyświetla postęp przez sekwencję ponumerowanych kroków poprzez podświetlanie aktywnego kroku. Renderuje kontener z krokami, z których każdy jest reprezentowany przez komponent `Step`.

<Tabs>
<Tab title="Usage">

```jsx
import { StepBar } from "@/ui/navigation/step-bar/components/StepBar";

export const MyComponent = () => {
  return (
    <StepBar activeStep={2}>
      <StepBar.Step>Step 1</StepBar.Step>
      <StepBar.Step>Step 2</StepBar.Step>
      <StepBar.Step>Step 3</StepBar.Step>
    </StepBar>
  );
};
```

</Tab>

<Tab title="Props">

| Właściwości (Props) | Typ   | Opis                                                                                                         |
| -------------------------------------- | ----- | ------------------------------------------------------------------------------------------------------------ |
| aktywnyKrok                            | numer | Indeks aktualnie aktywnego kroku. To określa, który krok powinien być wizualnie podświetlony |

</Tab>
