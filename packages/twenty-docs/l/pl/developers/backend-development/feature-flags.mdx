Flagi funkcji są używane do ukrywania eksperymentalnych funkcji. W programie Twenty są one ustawiane na poziomie przestrzeni roboczej, a nie użytkownika.

## Dodanie nowej flagi funkcji

W pliku `FeatureFlagKey.ts` dodaj flagę funkcji:

```ts
type FeatureFlagKey =
  | 'IS_FEATURENAME_ENABLED'
  | ...;
```

Dodaj ją również do enuma w `feature-flag.entity.ts`:

```ts
enum FeatureFlagKeys {
    IsFeatureNameEnabled = 'IS_FEATURENAME_ENABLED',
    ...
}
```

Aby zastosować flagę funkcji na funkcji **backend** użyj:

```ts
@Gate({
  featureFlag: 'IS_FEATURENAME_ENABLED',
})
```

Aby zastosować flagę funkcji na funkcji **frontend** użyj:

```ts
const isFeatureNameEnabled = useIsFeatureEnabled('IS_FEATURENAME_ENABLED');
```

## Skonfiguruj flagi funkcji dla wdrożenia

Zmień odpowiedni rekord w tabeli `core.featureFlag`:

| identyfikator | klucz                    | workspaceId | wartość  |
| ------------- | ------------------------ | ----------- | -------- |
| Losowy        | `IS_FEATURENAME_ENABLED` | WorkspaceID | `prawda` |
