Um elemento visual que você pode usar como um contêiner clicável ou não clicável com um rótulo, componentes opcionais à esquerda e à direita, e várias opções de estilo para exibir rótulos e tags.

<Tabs>

<Tab title="Usage">

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Chip
      size="large"
      label="Clickable Chip"
      clickable={true}
      variant="highlighted"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
    />
  );
};

```

</Tab>

<Tab title="Props">

| Props         | Tipo                                     | Descrição                                                                                                                        |
| ------------- | ---------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| linkToEntity  | Nome opcional para estilização adicional | O link para a entidade                                                                                                           |
| entityId      | Nome opcional para estilização adicional | O identificador único para a entidade                                                                                            |
| nome          | Nome opcional para estilização adicional | O nome da entidade                                                                                                               |
| pictureUrl    | Nome opcional para estilização adicional | s picture",                                                                                                                      |
| avatarType    | Avatar Type                              | O tipo de avatar que você deseja exibir. Tem duas opções: `arredondado` e `quadrado`             |
| variante      | enum `EntidadeChipVariante`              | Variante do chip da entidade que você deseja exibir. Tem duas opções: `regular` e `transparente` |
| ÍconeEsquerdo | ComponenteÍcone                          | Um componente React que representa um ícone. Exibido no lado esquerdo do chip                                    |

</Tab>
</Tabs>

## Exemplos

### Chip Transparente Desativado

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Transparent Disabled Chip"
      clickable={false}
      variant="rounded"
      accent="text-secondary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};

```

<br/>

### Chip Desativado com Tooltip

```jsx
import { Chip } from "twenty-ui/components";
  
export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Disabled chip that triggers a tooltip when overflowing."
      clickable={false}
      variant="regular"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};
```

## Chip de Entidade

Um elemento semelhante a um Chip para exibir informações sobre uma entidade.

<Tabs>

<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';
import { IconTwentyStar } from 'twenty-ui/display';
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Router>
      <Chip
        linkToEntity="/entity-link"
        entityId="entityTest"
        name="Entity name"
        pictureUrl=""
        avatarType="rounded"
        variant="regular"
        LeftIcon={IconTwentyStar}
      />
    </Router>
  );
};
```
