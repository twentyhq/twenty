Usa um editor de texto rico baseado em blocos da [BlockNote](https://www.blocknotejs.org/) para permitir que os usuários editem e visualizem blocos de conteúdo.

<Tabs>
<Tab title="Usage">

```jsx
import { useBlockNote } from "@blocknote/react";
import { BlockEditor } from "@/ui/input/editor/components/BlockEditor";

export const MyComponent = () => {
  const BlockNoteEditor = useBlockNote();

  return <BlockEditor editor={BlockNoteEditor} />;
};
```

</Tab>
<Tab title="Props">

| Props  | Tipo              | Descrição                                       |
| ------ | ----------------- | ----------------------------------------------- |
| editor | `EditorBlockNote` | A instância ou configuração do editor de blocos |
