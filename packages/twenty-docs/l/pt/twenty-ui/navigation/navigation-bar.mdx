Renderiza uma barra de navegação que contém múltiplos componentes `NavigationBarItem`.

<Tabs>
<Tab title="Usage">

```jsx
import { IconHome, IconUser, IconSettings } from '@tabler/icons-react';
import { NavigationBar } from "@/ui/navigation/navigation-bar/components/NavigationBar";

export const MyComponent = () => {

   const navigationItems = [
     {
       name: "Home",
       Icon: IconHome,
       onClick: () => console.log("Home clicked"),
     },
     {
       name: "Profile",
       Icon: IconUser,
       onClick: () => console.log("Profile clicked"),
     },
     {
       name: "Settings",
       Icon: IconSettings,
       onClick: () => console.log("Settings clicked"),
     },
   ];

  return <NavigationBar activeItemName="Home" items={navigationItems}/>;
};
```

</Tab>
<Tab title="Props">

| Props          | Tipo                                     | Descrição                                                                                                                                                                                                      |
| -------------- | ---------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| activeItemName | Nome opcional para estilização adicional | O nome do item de navegação que está ativo atualmente                                                                                                                                                          |
| itens          | matriz                                   | Um array de objetos que representa cada item de navegação. Cada objeto contém o `nome` do item, o componente `Ícone` para exibir, e uma função `onClick` a ser chamada quando o item é clicado |

</Tab>
