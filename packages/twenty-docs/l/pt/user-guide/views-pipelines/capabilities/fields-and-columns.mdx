## Selecionar Campos para Exibição

Cada visualização pode mostrar um conjunto diferente de campos. Personalize o que é visível para focar nas informações que importam.

### Mostrar ou Ocultar Campos

1. Clique em **Opções** no canto superior direito
2. Clique em **Campos**
3. Clique no **ícone de olho** ao lado de cada campo para mostrar/ocultar

### Reordenar Campos

Altere a ordem em que os campos aparecem na sua visualização:

1. Clique em **Opções → Campos**
2. Arraste os campos para cima ou para baixo
3. As alterações são salvas automaticamente

## Exibição de Campos por Tipo de Visualização

### Visualizações de Tabela

* Os campos aparecem como colunas
* Redimensione as colunas arrastando as bordas

### Visualizações Kanban

* Os campos aparecem nos cartões
* Reordene em Opções → Campos
* Use a visualização compacta para ocultar todos os campos

### Visualizações de Calendário

* Os campos selecionados aparecem nos eventos do calendário
* Configure em Opções → Campos

## Melhores Práticas

* **Mostre apenas o necessário** — muitos campos poluem a visualização
* **Coloque os campos importantes primeiro** — colunas mais usadas à esquerda
* **Crie várias visualizações** — conjuntos de campos diferentes para finalidades diferentes
* **Use a visibilidade de campos por visualização** — mesmo objeto, foco diferente
