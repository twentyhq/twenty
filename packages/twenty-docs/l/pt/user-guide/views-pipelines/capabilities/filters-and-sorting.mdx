## Filtrando dados

Os filtros ajudam você a focar em registros específicos, mostrando apenas aqueles que correspondem aos seus critérios.

### Adicionando um filtro

1. Clique no botão **Filtrar** na barra de ferramentas
2. Selecione o campo pelo qual filtrar
3. Escolha o operador (igual a, contém, etc.)
4. Insira o valor do filtro
5. Clique em **Aplicar**

### Operadores de filtro

| Tipo de campo    | Operadores disponíveis                               |
| ---------------- | ---------------------------------------------------- |
| Texto            | Igual a, Contém, Começa com, Termina com, Está vazio |
| Número           | Igual a, Maior que, Menor que, Entre, Está vazio     |
| Data             | Igual a, Antes de, Depois de, Entre, Está vazio      |
| Selecionar       | Igual a, É um dos, Está vazio                        |
| Caixa de Seleção | É verdadeiro, É falso                                |
| Relação          | Igual a, Está vazio                                  |

### Vários filtros

Combine vários filtros para restringir os resultados:

* Todos os filtros são aplicados com a lógica AND
* Cada filtro adicional restringe ainda mais os resultados

### Removendo filtros

* Clique no **X** em cada chip de filtro
* Clique em **Limpar tudo** para remover todos os filtros

## Ordenando dados

A ordenação determina a ordem em que os registros aparecem.

### Adicionando uma ordenação

1. Clique no botão **Ordenar** na barra de ferramentas
2. Selecione o campo pelo qual ordenar
3. Escolha crescente (A-Z, 0-9) ou decrescente (Z-A, 9-0)
4. Clique em **Aplicar**

### Várias ordenações

Adicione vários níveis de ordenação:

* A primeira ordenação é a primária
* As ordenações subsequentes se aplicam dentro de grupos de valores iguais

### Ordenação rápida por coluna

Clique no cabeçalho de uma coluna para ordenar:

* Primeiro clique: Crescente
* Segundo clique: Decrescente
* Terceiro clique: Remover ordenação

## Salvando as configurações de filtro e ordenação

Filtros e ordenações são salvos com a visualização:

1. Configure seus filtros e ordenações
2. Clique em **Salvar** para atualizar a visualização atual
3. Ou clique em **Salvar como nova visualização** para criar uma variante
