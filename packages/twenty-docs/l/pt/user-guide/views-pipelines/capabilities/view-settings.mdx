## Visibilidade da visualização

Controle quem pode ver suas visualizações personalizadas.

### Opções de visibilidade

| Configuração           | Quem pode ver                          |
| ---------------------- | -------------------------------------- |
| **Espaço de trabalho** | Todos os membros do espaço de trabalho |
| **Não listado**        | Apenas você                            |

### Alterar a visibilidade

1. Abra a visualização
2. Clique em **Opções → Visibilidade**
3. Selecione **Espaço de trabalho** ou **Não listado**

<Note>
  As visualizações padrão "Todos(as) [Nome do objeto]" não podem ter sua visibilidade alterada.
</Note>

## Renomear uma visualização

1. Abra o menu suspenso de visualizações
2. Clique no menu **⋮** ao lado da visualização
3. Selecione **Editar**
4. Insira o novo nome

## Alterar o ícone da visualização

1. Abra o menu suspenso de visualizações
2. Clique no menu **⋮** ao lado da visualização
3. Selecione **Editar**
4. Clique no ícone para alterá-lo

## Reordenar visualizações

Altere a ordem em que as visualizações aparecem no menu suspenso:

1. Abra o menu suspenso de visualizações
2. Arraste as visualizações pela alça
3. Solte na posição desejada
4. A ordem é salva automaticamente

## Favoritos

Fixe visualizações usadas com frequência para acesso rápido:

1. Abra o menu suspenso de visualizações
2. Clique no menu **⋮** ao lado de uma visualização
3. Selecione **Adicionar aos favoritos**

As visualizações favoritas aparecem em uma seção dedicada para facilitar o acesso.

## Excluir uma visualização

1. Abra o menu suspenso de visualizações
2. Clique no menu **⋮** ao lado da visualização
3. Selecione **Excluir**
4. Confirme a exclusão

<Warning>
  Visualizações excluídas não podem ser recuperadas.
</Warning>
