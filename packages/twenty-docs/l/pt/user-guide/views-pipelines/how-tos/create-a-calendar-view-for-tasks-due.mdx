## Pré-requisitos

Seu objeto Tasks precisa de um campo **Data de vencimento** (tipo Data ou Data e hora).

## Etapas

1. Navegue até **Tasks**
2. Clique no menu suspenso de visualizações → **+ Adicionar visualização**
3. Nomeie sua visualização (por exemplo, "Calendário de tarefas")
4. Clique em **Criar**
5. Clique em **Opções** e selecione **Calendário** como o layout
6. Escolha **Data de vencimento** como o campo de data
7. Clique em **Salvar**

## Configure seu calendário

### Exiba campos nos eventos

1. Clique em **Opções → Campos**
2. Clique no **ícone de olho** para mostrar/ocultar campos
3. Arraste para reordenar

Campos recomendados para exibir:

* **Título** — nome da tarefa
* **Responsável** — quem é o responsável
* **Status** — progresso atual

### Filtre seu calendário

Crie visualizações focadas:

* **Minhas Tarefas**: Filtrar por Responsável = Eu
* **Esta semana**: Filtrar por Data de Vencimento = Esta semana
* **Em atraso**: Filtrar por Data de Vencimento < Hoje, Estado ≠ Concluído

## Outros casos de uso do calendário

| Objeto                 | Campo de Data  | Finalidade                           |
| ---------------------- | -------------- | ------------------------------------ |
| Oportunidades          | Data de Fecho  | Acompanhar fechos esperados          |
| Eventos personalizados | Data do Evento | Planear atividades                   |
| Projetos               | Prazo          | Monitorizar cronogramas dos projetos |

## Dicas

* **Reveja semanalmente**: Comece cada semana por verificar a sua visualização de calendário
* **Combine com a visualização de tabela**: Utilize o calendário para a visão geral e a tabela para os detalhes
* **Defina a visibilidade**: Mantenha os calendários de tarefas pessoais como Não listados
