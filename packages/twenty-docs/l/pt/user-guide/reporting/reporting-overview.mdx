Crie relatórios personalizados para monitorar as métricas que são mais importantes para o seu negócio.

Siga o nosso [repositório do GitHub](https://github.com/twentyhq/twenty) ou confira a seção **Configurações → Lançamentos** no seu workspace para ser notificado quando os recursos de relatórios estiverem disponíveis.
