## Visão Geral

Os créditos alimentam as automações de fluxo de trabalho e os recursos de IA. Eles são consumidos quando ações de fluxo de trabalho são executadas, quando agentes de IA processam tarefas dentro de fluxos de trabalho e no chat de IA.

## Alocação de créditos

Os créditos são baseados no seu ciclo de faturamento, não no seu plano:

| Ciclo de faturamento | Créditos |
| -------------------- | -------- |
| Mensal               | 5/mês    |
| Anual                | 50/ano   |

<Note>
Os 5 créditos mensais foram projetados para permitir que você execute automações sem se preocupar com custos. Para a maioria dos fluxos de trabalho que usam ações padrão, isso é mais do que suficiente. Você só precisará de créditos adicionais ao executar nós de código avançados ou recursos com tecnologia de IA.
</Note>

## Acúmulo de créditos

Créditos não utilizados ao final de um período de faturamento são automaticamente transferidos para o próximo período.

* **Limite**: O valor de transferência é limitado à alocação total de um período, portanto você nunca poderá transferir mais créditos do que o seu plano fornece por período.
* **Visibilidade**: Quando houver créditos transferidos, eles aparecerão como uma linha separada chamada **Créditos transferidos** em **Configurações → Cobrança**, juntamente com um saldo de **Total disponível**.

## Consumo de créditos

Ações diferentes consomem quantidades diferentes de créditos:

| Tipo de ação                                                      | Uso de Créditos          |
| ----------------------------------------------------------------- | ------------------------ |
| **Operações básicas** (pesquisar, atualizar, criar registros)     | Mínimo                   |
| **Operações complexas** (nós de código, chamadas de API externas) | Mais créditos            |
| **Prompt de IA e chats de IA**                                    | Variável com base no uso |

Os créditos são deduzidos em tempo real à medida que os fluxos de trabalho são executados.

## Monitoramento do uso

Acompanhe seu consumo de créditos:

1. Vá para **Configurações → Faturamento**
2. Veja seu uso atual e os créditos restantes
3. Monitore tendências para planejar créditos adicionais, se necessário

## Compra de Créditos Adicionais

Precisa de mais créditos?

1. Vá para **Configurações → Faturamento**
2. Clique na opção para comprar pacotes de créditos adicionais
3. Selecione a quantidade de que você precisa
