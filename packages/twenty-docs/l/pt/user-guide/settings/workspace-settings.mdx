## Imagem do Espaço de Trabalho

- **Carregar Logotipo**: Adicione um logotipo personalizado para o espaço de trabalho
- **Formatos suportados**: Arquivos PNG, JPEG e GIF com até 10MB
- **Remover**: Exclua o logotipo atual do espaço de trabalho

## Nome do Workspace

- **Nome**: Altere o nome de exibição do seu espaço de trabalho
- Este nome aparece para todos os membros do espaço de trabalho

## Zona de Perigo

<Warning>
Excluir seu espaço de trabalho remove permanentemente todos os dados e não pode ser desfeito. Todos os dados do espaço de trabalho serão perdidos para sempre, todos os membros perderão acesso imediatamente e essa ação não pode ser revertida.
</Warning>

Para excluir seu espaço de trabalho:

1. Clique no botão **Excluir espaço de trabalho**
2. Confirme a exclusão quando solicitado

**Nota**: Somente os administradores do espaço de trabalho podem excluir espaços de trabalho.
