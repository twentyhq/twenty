## Convidar Novos Membros

### Usando o Link de Convite

1. Acesse **Configurações → Membros**
2. Copy the workspace invite link
3. Compartilhe o link com novos membros da equipe
4. Eles receberão o acesso assim que se inscreverem

### Convite por Email Direto

1. Acesse **Configurações → Membros**
2. Digite o endereço de email da pessoa
3. Click **Invite**
4. They'll receive an email invitation

## Remover Membros

### Excluir um Membro

1. Acesse **Configurações → Membros**
2. Encontre o membro que deseja remover
3. Clique no botão de excluir/remover ao lado do nome dele
4. Confirme a remoção

**Nota**: Os membros removidos perdem o acesso imediatamente, mas podem ser convidados novamente mais tarde.

## Precisa de Ajuda?

Para gerenciamento de funções e permissões, consulte o [artigo de Permissões](/l/pt/user-guide/settings/permissions).
