## Informações Pessoais

### Nome e E-mail

- **Nome de Exibição**: Atualize como seu nome aparece para outros membros do espaço de trabalho
- **Endereço de E-mail**: Altere seu e-mail de login (requer verificação)
- **Foto de Perfil**: Carregue um avatar personalizado ou use suas iniciais

## Configurações de Segurança

### Autenticação em Dois Fatores (2FA)

Ative o 2FA para adicionar uma camada extra de segurança à sua conta:

1. Vá para **Configurações → Configurações do Perfil**
2. Clique em **Ativar 2FA**
3. Escaneie o código QR com seu aplicativo autenticador
4. Insira o código de verificação para confirmar

### Gerenciamento de Senhas

- **Mudar Senha**: Atualize sua senha atual
- **Requisitos de Senha**: Deve ter pelo menos 8 caracteres

## Gerenciamento de Perfil

### Excluir Conta

<Warning>
Excluir sua conta removerá permanentemente seu acesso a todos os espaços de trabalho. Esta ação não pode ser desfeita, você perderá acesso a todos os espaços de trabalho dos quais é membro e deve considerar sair de espaços de trabalho individuais se desejar apenas sair de equipes específicas.
</Warning>

Para excluir sua conta:

1. Vá para **Configurações → Configurações do Perfil**
2. Role até **Zona de Perigo**
3. Clique em **Excluir Conta**
4. Confirme digitando seu endereço de e-mail
