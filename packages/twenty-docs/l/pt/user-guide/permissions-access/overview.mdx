## Principais funcionalidades

* **Acesso baseado em funções**: Crie funções personalizadas com permissões específicas
* **Permissões de objeto**: Controle quem pode visualizar, editar ou excluir registros
* **Permissões de campo**: Restrinja o acesso a campos sensíveis
* **Permissões de configuração**: Controle o acesso às configurações do espaço de trabalho
* **Integração com SSO**: Configure o Single Sign-On para segurança corporativa (plano Organization)
