O Twenty é o principal CRM de código aberto, desenvolvido por centenas de contribuidores para atender às suas necessidades de negócios únicas.

## Visão

Criar um bom CRM é difícil porque é um ato de equilíbrio.
Para cada empresa, os requisitos parecem simples, mas as necessidades de todos são distintas.
O resultado é um CRM que é ou muito básico ou tenta ser um faz-tudo, mas acaba não sendo mestre em nada.

A princípio, o Twenty parece como a maioria dos CRMs que você já conhece: você pode acompanhar negociações, organizar contatos, gerenciar tarefas e notas.
Mas o que o diferencia é nossa abordagem à extensibilidade. Estamos construindo uma plataforma aberta que oferece os blocos de construção para você resolver seus problemas de negócios únicos.

Prioritizamos princípios universais e padrões comuns em vez de listas de funcionalidades.
Não tentamos ter todas as respostas e, em vez disso, capacitamos os usuários a encontrar o que funciona melhor para eles.
Código aberto é a base de nossa abordagem, garantindo que o Twenty evolua com sua comunidade, para sua comunidade.

## Benefícios

**Customizável:** Projetado para atender às necessidades do seu negócio.

**Dirigido pela comunidade:** Construído e mantido por uma grande comunidade de código aberto.

**Econômico:** Você nunca estará preso a um fornecedor, pois sempre pode hospedar por conta própria.

## Principais Funcionalidades

**Gerenciamento de Contatos:** Armazene e gerencie dados de clientes de forma eficiente. [Saiba mais](/l/pt/user-guide/crm-essentials/contact-and-account-management).

**Objetos Personalizados:** Crie e personalize objetos para atender às necessidades do seu negócio. [Detalhes](/l/pt/user-guide/data-model/objects).

**Campos Personalizados:** Personalize os campos de dados para capturar e organizar informações específicas para suas operações. [Entenda mais](/l/pt/user-guide/data-model/fields).

**Gerenciamento de Negócios:** Rastreie e gerencie suas oportunidades de vendas por meio de [estágios de Pipeline](/l/pt/user-guide/crm-essentials/pipeline) personalizáveis.

**Visualizações Kanban & Tabela:** Torne os dados acionáveis com [visualizações de tabela flexíveis](/l/pt/user-guide/crm-essentials/view-management).

**Workflows:** Automate seus processos de negócios e integre com ferramentas externas usando automação de fluxo de trabalho poderosa. [Comece agora](/l/pt/user-guide/workflows/getting-started-workflows).

**Integração de E-mail:** Veja os e-mails de um cliente ou empresa específico dentro do seu espaço de trabalho. [Sincronize sua caixa de correio](/l/pt/user-guide/collaboration/emails-and-calendars).

**Notas:** Crie notas detalhadas para cada registro para compartilhar conhecimento de forma mais eficaz. [Adicione anotações](/l/pt/user-guide/collaboration/notes).

**Tarefas:** Agende tarefas para rastrear interações com clientes. [Veja como](/l/pt/user-guide/collaboration/tasks).

**Permissões:** Controle o acesso e gerencie funções de usuário com permissões flexíveis de nível de espaço de trabalho e de objeto. [Configure permissões](/l/pt/user-guide/settings/permissions).

**API & Webhooks:** Conecte-se a outros aplicativos e automatize fluxos de trabalho com API e Webhooks. [Comece a integrar](/l/pt/user-guide/integrations-api/api-webhooks).
