## Antes de Começar

### 1. Auditar Seus Dados Atuais

- **Selecione os poucos objetos e campos para migrar**: esta migração é a oportunidade para um novo começo
- **Exporte esses dados** do seu CRM atual
- **Remova duplicatas** e registros desatualizados
- **Liste os fluxos de trabalho ativos** e automações

### 2. Crie Seu Novo Modelo de Dados

Siga nosso [guia de modelo de dados](/l/pt/user-guide/data-model/customize-your-data-model) para:

- **Projete o modelo de dados** que você precisa
- **Mapeie os campos existentes** para os objetos/campos padrão do Twenty
- **Identifique os objetos/campos personalizados** de que você precisará
- **Crie-os** em Configurações → Modelo de dados

## Processo de Migração

### 1. Importe Seus Dados

**Ordem recomendada:**

1. **Empresas** primeiro (como registros base)
2. **Pessoas** em segundo (vinculadas às empresas)
3. **Oportunidades** em terceiro (vinculadas a pessoas/empresas)

Use a importação de CSV através do Menu de Comando `Cmd + K` (Mac) ou `Ctrl + K` (Windows). Veja nosso [guia de importação de dados](/l/pt/user-guide/getting-started/import-export-data) para instruções detalhadas.

### 2. Recrie Fluxos de Trabalho

- **Comece simples** - recrie suas automações mais críticas primeiro
- **Use o construtor de fluxos de trabalho do Twenty** para substituir automações existentes

## Desafios Comuns

### Problemas de Formatação de Dados

- **Endereços de email** - remova duplicatas (requisito do objeto Pessoas)
- **Domínio** - remova duplicatas (requisito do objeto Empresas)

<Warning>
Please note that domain URLs created by the synchronization with your mailbox and calendar have the following format ```https://domain.com```
</Warning>

- **Formatos de data** - garanta formatação consistente (AAAA-MM-DD) ou edite este formato em Configurações → Experiência
- **Números de telefone** - use o formato internacional (+1234567890)

### Mapeamento de Relacionamentos

Para importar relações entre registros usando a função de importação csv, você pode usar os seguintes campos

- **Use IDs do Twenty** para relações complexas
- **Use endereços de email** para vincular registros de Pessoas
- **Use nomes de domínio** para vincular registros de Empresa
- **Use qualquer outro campo que você definir como único**, o que pode ser feito na seção Modelo de Dados.
  Leia nosso [guia de importação-exportação de dados](/l/pt/user-guide/getting-started/import-export-data) para instruções detalhadas sobre como criar relacionamentos durante a importação de CSV.

## Migrando de Autohospedado para a Nuvem

Se você estiver mudando de autohospedagem do Twenty para Twenty Cloud:

1. **Exporte seus dados** da sua instância autohospedada
2. **Siga o processo de migração padrão** acima
3. **Podemos fornecer assistência na migração**, entre em contato com nossa equipe
