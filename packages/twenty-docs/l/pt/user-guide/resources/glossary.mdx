## API

A API (Interface de Programação de Aplicação) permite conectar o Twenty com outros sistemas de software e construir integrações personalizadas.

## Menu de Comando

O Menu de Comando é uma interface de acesso rápido (aberta com `Cmd + K` no Mac e `Ctrl + K` no Windows) que permite realizar ações, criar registros e navegar no seu espaço de trabalho de forma eficiente.

## Empresa & Pessoas

O CRM possui dois tipos fundamentais de registros:

- Uma `Empresa` representa um negócio ou organização.
- `Pessoas` representam os clientes ou potenciais clientes da sua empresa.

## Campos Personalizados

Campos Personalizados são campos de dados que você cria para capturar informações específicas às suas necessidades e processos de negócios.

## Modelo de Dados

Um Modelo de Dados é a estrutura que define como as informações são organizadas no seu CRM, incluindo quais objetos existem, suas propriedades (campos) e como se relacionam entre si.

## Favoritos

Favoritos são registros que você marcou para acesso rápido, aparecendo na sua barra lateral para navegação instantânea a dados importantes.

## Campo

Um campo refere-se a uma área específica onde dados particulares são armazenados para uma entidade.

## Integration

Integrações são ferramentas embutidas que permitem conectar o Twenty a outros softwares ou sistemas.

## Kanban

Um `Kanban` é uma maneira visual de rastrear seus processos de negócios usando cartões e colunas. Cada coluna representa uma etapa em seu processo (por exemplo: novo, em andamento, ganho, perdido), e você move registros por essas etapas conforme eles progridem.

## Objeto

Um Objeto é uma estrutura de dados que representa um tipo específico de entidade no seu CRM (como Pessoas, Empresas ou Oportunidades). Os objetos podem ser padrão (embutidos) ou personalizados (criados por você).

## Oportunidades

Oportunidades no Twenty CRM são potenciais negócios ou vendas com contas ou contatos.

## Gravação

Um Registro indica uma instância de um objeto, como uma conta ou contato específico.

## Campos de Relação

Campos de Relação criam conexões entre diferentes objetos, permitindo que você vincule registros (como conectar uma Pessoa a uma Empresa).

## Campos Padrão

Campos Padrão são campos de dados pré-construídos que vêm com objetos por padrão e fornecem funcionalidade comum em todos os espaços de trabalho.

## Tarefas

Tarefas no Twenty CRM são atividades atribuídas relacionadas a contatos, contas ou oportunidades.

## Visualizações

Você pode personalizar a exibição de seus registros usando visualizações, definindo diferentes filtros, layouts e opções de ordenação para cada visualização.

## Webhooks

Webhooks são mensagens automatizadas enviadas do Twenty para outros aplicativos quando eventos específicos ocorrem, permitindo a sincronização de dados em tempo real.

## Workflows

Workflows são processos automatizados que acionam ações com base em condições específicas, ajudando a automatizar tarefas repetitivas e processos de negócios.

## Workspace

Um `Workspace` geralmente representa uma empresa usando o Twenty. Ele contém todos os registros e dados que você e os membros da sua equipe adicionam ao Twenty.  
Possui um único nome de domínio, que é normalmente o nome de domínio que sua empresa usa para endereços de e-mail dos funcionários.

## Membros do Workspace

Workspace Members are the Twenty users from your team who have access to your workspace. Eles podem ser designados como proprietários ou responsáveis por registros.
