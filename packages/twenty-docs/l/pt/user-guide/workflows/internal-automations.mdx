Below are workflow examples you could roll out to automate repetitive tasks to reduce manual work and keep your data up-to-date.

## Data Management Automations

### Auto-Flag Personal Emails

**Problema**: A sua equipe precisa identificar endereços de e-mail pessoais versus empresariais para melhor direcionamento e conformidade.

**Solution**: Create a workflow that automatically updates an "is personal email" field whenever an email address is added or updated.

**Gatilho**: Registro Atualizado (objeto Pessoas, campo E-mail)

**Ações**:

- Ação de código para verificar se o domínio de e-mail corresponde a provedores pessoais comuns (gmail.com, yahoo.com, etc.)
- Atualizar Registro para definir a marcação "é e-mail pessoal"

### Atribuição de Leads - Round Robin

**Problema**: Novos leads precisam ser distribuídos de forma justa entre sua equipe de vendas para garantir uma carga de trabalho equilibrada.

**Solução**: Atribua automaticamente novos leads aos representantes de vendas usando um sistema round-robin.

**Gatilho**: Registro Criado (objeto Pessoas)

**Ações**:

- Buscar Registros para encontrar o último representante atribuído
- Ação de código para determinar o próximo representante na rotação
- Atualizar Registro para atribuir o lead ao representante selecionado
- Enviar E-mail para notificar o representante atribuído

### Atribuição de Leads - Baseada em Território

**Problema**: Leads devem ser atribuídos com base em territórios geográficos ou características da empresa.

**Solução**: Direcione leads para o representante de vendas apropriado com base na localização, tamanho da empresa ou setor.

**Gatilho**: Registro Criado (objeto Pessoas ou Empresas)

**Ações**:

- Code action to determine territory based on location/industry rules
- Buscar Registros para encontrar o proprietário do território
- Atualizar Registro para atribuir o lead
- Criar Registro para uma tarefa de acompanhamento

## Sales Process Automations

### Gerenciamento de Estágio de Oportunidade - Fechado Ganhado

**Problema**: Quando negócios se fecham, são necessárias várias atualizações manuais em diferentes registros e membros da equipe.

**Solution**: Automatically handle all post-win activities when an opportunity moves to "Closed Won".

**Gatilho**: Registro Atualizado (objeto Oportunidades, campo Estágio = "Fechado Ganhado")

**Ações**:

- Atualizar Registro para mudar o tipo de Empresa de "Prospect" para "Cliente"
- Criar Registro para tarefas de integração atribuídas ao gerente da conta
- Enviar notificação por E-mail para a equipe de sucesso do cliente
- Solicitação HTTP para atualizar o sistema de faturamento externo

### Gerenciamento de Estágio de Oportunidade - Renovação Perdida Fechada

**Problema**: Quando oportunidades de renovação são perdidas, o status do cliente precisa ser atualizado para uma gestão de contas adequada.

**Solução**: Atualize automaticamente o status do cliente quando negócios de renovação forem perdidos.

**Gatilho**: Registro Atualizado (objeto Oportunidades, Estágio = "Fechado Perdido" E Tipo = "Renovação")

**Ações**:

- Atualizar Registro para mudar o tipo de Empresa de "Cliente" para "Cliente Perdido"
- Criar Registro para tarefa de análise de churn
- Enviar alerta por E-mail para o gerente de sucesso do cliente
- Atualizar Registro para adicionar data e motivo de churn

### Stale Opportunity Alerts

**Problema**: Oportunidades permanecem sem atualizações, fazendo com que os negócios esfriem e as previsões se tornem pouco confiáveis.

**Solução**: Envie alertas automáticos quando oportunidades não forem atualizadas recentemente.

**Gatilho**: Em uma programação (diária)

**Ações**:

- Buscar Registros por oportunidades não atualizadas em X dias
- Ação de código para formatar mensagem de alerta com detalhes da oportunidade
- Enviar E-mail para o proprietário e gerente da oportunidade
- Criar Registro para tarefa de acompanhamento se não houver resposta

## Productivity Automations

### Resumo Semanal de Tarefas

**Problema**: Membros da equipe perdem o controle de suas próximas tarefas e prazos.

**Solução**: Envie lembretes automáticos por e-mail semanalmente com resumos das tarefas.

**Gatilho**: Em uma programação (toda segunda-feira às 8h)

**Ações**:

- Buscar Registros por tarefas a vencer nesta semana por responsável
- Ação de código para formatar lista de tarefas por pessoa
- Enviar E-mail para cada membro da equipe com seu resumo de tarefas
- Enviar E-mail para gerentes com a visão geral da equipe

### Meeting Follow-up Automation

**Problema**: Itens de ação importantes de reuniões são esquecidos ou atrasados.

**Solução**: Crie automaticamente tarefas de acompanhamento quando reuniões são agendadas ou concluídas.

**Gatilho**: Registro Criado (objeto Atividades, Tipo = "Reunião")

**Ações**:

- Criar Registro para tarefa de preparação pré-reunião
- Criar Registro para tarefa de acompanhamento pós-reunião
- Enviar lembrete por E-mail para os participantes
- Atualizar Registro para vincular tarefas à reunião

### Sincronização de Campos entre Objetos

**Problema**: Você precisa de informações de registros relacionados facilmente acessíveis (por exemplo, email do contato principal no registro de oportunidade).

**Solução**: Sincronize automaticamente campos entre objetos relacionados até que campos aninhados estejam disponíveis.

**Gatilho**: Registro Atualizado (objeto Oportunidades, campo Ponto de Contato)

**Ações**:

- Buscar Registros para encontrar os detalhes da pessoa vinculada
- Atualizar Registro para copiar o endereço de email para a oportunidade
- Atualizar Registro para copiar o número de telefone para a oportunidade
- Atualizar Registro para copiart informações da empresa

## Validação e Limpeza de Dados

### Padronização de Números de Telefone

**Problema**: Os números de telefone são inseridos em formatos diferentes, dificultando seu uso para chamadas ou mensagens.

**Solução**: Formate automaticamente números de telefone para um formato padrão quando forem inseridos.

**Gatilho**: Registro Atualizado (objeto Pessoas, campo Telefone)

**Ações**:

- Ação de código para analisar e formatar número de telefone
- Atualizar Registro com formato padronizado de telefone
- Atualizar Registro para adicionar código do país se faltar
