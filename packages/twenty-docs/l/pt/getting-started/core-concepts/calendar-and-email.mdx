O Twenty se conecta às suas ferramentas existentes para que o seu CRM permaneça atualizado sem entrada manual de dados.

## Sincronização de email

Conecte a sua caixa de correio do **Google Workspace** ou **Microsoft 365**. Depois de conectado:

* Os emails são vinculados automaticamente aos registros de Empresa e Pessoas correspondentes
* Tópicos completos de email ficam visíveis na linha do tempo de cada registro
* Você pode enviar emails diretamente do Twenty
* Várias caixas de correio por usuário são compatíveis

Você controla o que é importado — filtre por intervalo de datas ou remetente para evitar trazer emails irrelevantes.

## Sincronização de calendário

Os eventos do calendário são sincronizados automaticamente a partir da sua conta conectada. Os eventos aparecem nos registros relevantes do CRM, oferecendo uma visão completa das suas interações com cada contato ou empresa.

## Integrações

Além de email e calendário, o Twenty se conecta a ferramentas externas por meio de:

| Método                              | Caso de uso                                                                                 |
| ----------------------------------- | ------------------------------------------------------------------------------------------- |
| **API**                             | Crie integrações personalizadas com a API GraphQL ou REST                                   |
| **Webhooks**                        | Envie notificações em tempo real para sistemas externos quando os registros forem alterados |
| **Zapier**                          | Conecte-se a mais de 5.000 apps sem código                                                  |
| **Ações HTTP de fluxo de trabalho** | Chame qualquer API externa como parte de um fluxo de trabalho automatizado                  |

## Apps personalizados

Desenvolvedores podem criar apps completos em cima do Twenty — adicionando interface personalizada, lógica no servidor e integrações avançadas. Os apps podem ser publicados para a comunidade ou mantidos privados.
