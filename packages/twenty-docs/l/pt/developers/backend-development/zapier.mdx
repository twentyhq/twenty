Sincronize o Twenty facilmente com mais de 3000 aplicativos usando [Zapier](https://zapier.com/). Automatize tarefas, aumente a produtividade e potencialize seus relacionamentos com clientes!

## Sobre o Zapier

Zapier é uma ferramenta que permite automatizar fluxos de trabalho conectando os aplicativos que sua equipe usa todos os dias. O conceito fundamental do Zapier são os fluxos de trabalho de automação, chamados Zaps, que incluem gatilhos e ações.

Você pode saber mais sobre o funcionamento do Zapier [aqui](https://zapier.com/how-it-works).

## Configuração

### Passo 1: Instale os pacotes do Zapier

```bash
cd packages/twenty-zapier

yarn
```

### Passo 2: Faça login com o CLI

Use suas credenciais do Zapier para fazer login usando o CLI:

```bash
zapier login
```

### Passo 3: Defina variáveis de ambiente

A partir da pasta `packages/twenty-zapier`, execute:

```bash
cp .env.example .env
```

Execute o aplicativo localmente, vá para [http://localhost:3000/settings/api-webhooks](http://localhost:3000/settings/api-webhooks), e gere uma chave de API.

Substitua o valor **YOUR_API_KEY** no arquivo `.env` pela chave de API que você acabou de gerar.

## Desenvolvimento

<Warning>

Certifique-se de executar `yarn build` antes de qualquer comando `zapier`.

</Warning>

### Teste

```bash
yarn test
```

### Lint

```bash
yarn format
```

### Assista e compile enquanto edita o código

```bash
yarn watch
```

### Valide seu aplicativo Zapier

```bash
yarn validate
```

### Implante seu aplicativo Zapier

```bash
yarn deploy
```

### Liste todos os comandos CLI do Zapier

```bash
zapier
```
