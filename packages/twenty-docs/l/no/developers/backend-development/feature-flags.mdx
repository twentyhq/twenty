Funksjonsflagg brukes til å skjule eksperimentelle funksjoner. For Twenty er de satt på arbeidsplassnivå og ikke på brukernivå.

## Legger til et nytt funksjonsflagg

In `FeatureFlagKey.ts` add the feature flag:

```ts
type FeatureFlagKey =
  | 'IS_FEATURENAME_ENABLED'
  | ...;
```

Also add it to the enum in `feature-flag.entity.ts`:

```ts
enum FeatureFlagKeys {
    IsFeatureNameEnabled = 'IS_FEATURENAME_ENABLED',
    ...
}
```

For å bruke et funksjonsflagg på en **backend**-funksjon, bruk:

```ts
@Gate({
  featureFlag: 'IS_FEATURENAME_ENABLED',
})
```

For å bruke et funksjonsflagg på en **frontend**-funksjon, bruk:

```ts
const isFeatureNameEnabled = useIsFeatureEnabled('IS_FEATURENAME_ENABLED');
```

## Konfigurere funksjonsflagg for distribusjonen

Endre den tilsvarende posten i Tabell `core.featureFlag`:

| id        | nøkkel                   | arbeidsplassId | verdi  |
| --------- | ------------------------ | -------------- | ------ |
| Tilfeldig | `IS_FEATURENAME_ENABLED` | ArbeidsplassID | `sant` |
