Gjengir en navigasjonslinje som inneholder flere `Navigasjonslinjeelement` komponenter.

<Tabs>
<Tab title="Usage">

```jsx
import { IconHome, IconUser, IconSettings } from '@tabler/icons-react';
import { NavigationBar } from "@/ui/navigation/navigation-bar/components/NavigationBar";

export const MyComponent = () => {

   const navigationItems = [
     {
       name: "Home",
       Icon: IconHome,
       onClick: () => console.log("Home clicked"),
     },
     {
       name: "Profile",
       Icon: IconUser,
       onClick: () => console.log("Profile clicked"),
     },
     {
       name: "Settings",
       Icon: IconSettings,
       onClick: () => console.log("Settings clicked"),
     },
   ];

  return <NavigationBar activeItemName="Home" items={navigationItems}/>;
};
```

</Tab>
<Tab title="Props">

| <span>Egenskaper</span> | Type                | Beskrivelse                                                                                                                                                                                                                    |
| ----------------------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| activeItemName          | <span>streng</span> | The name of the currently active navigation item                                                                                                                                                                               |
| elementer               | array               | An array of objects representing each navigation item. Hvert objekt inneholder `navnet` på elementet, `Ikon`-komponenten som vises, og en `onClick`-funksjon som kalles når elementet klikkes. |

</Tab>
