Viser fremdriften gjennom en sekvens av nummererte trinn ved å fremheve det aktive trinnet. Den gjengir en beholder med trinn, hver representert av `Step`-komponenten.

<Tabs>
<Tab title="Usage">

```jsx
import { StepBar } from "@/ui/navigation/step-bar/components/StepBar";

export const MyComponent = () => {
  return (
    <StepBar activeStep={2}>
      <StepBar.Step>Step 1</StepBar.Step>
      <StepBar.Step>Step 2</StepBar.Step>
      <StepBar.Step>Step 3</StepBar.Step>
    </StepBar>
  );
};
```

</Tab>

<Tab title="Props">

| <span>Egenskaper</span> | Type   | Beskrivelse                                                                                                         |
| ----------------------- | ------ | ------------------------------------------------------------------------------------------------------------------- |
| activeStep              | nummer | Indeksen for det nåværende aktive trinnet. Dette bestemmer hvilket trinn som skal fremheves visuelt |

</Tab>
