Tillater brukere å laste opp og fjerne et bilde.

<Tabs>
<Tab title="Usage">

```jsx
import { ImageInput } from "@/ui/input/components/ImageInput";

export const MyComponent = () => {
  return <ImageInput/>;
};
```

</Tab>
<Tab title="Props">

| <span>Egenskaper</span> | Type                  | Beskrivelse                                                                                                                            |
| ----------------------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| bilde                   | <span>streng</span>   | Bildets kilde-URL                                                                                                                      |
| onUpload                | <span>funksjon</span> | Funksjonen som kalles når en bruker laster opp et nytt bilde. Den mottar `Fil`-objektet som parameter. |
| onRemove                | <span>funksjon</span> | Funksjonen som kalles når brukeren klikker på fjern-knappen.                                                           |
| onAbort                 | <span>funksjon</span> | Funksjonen som kalles når en bruker klikker på avbryt-knappen under bildeopplasting.                                   |
| lasterOpp               | <span>boolean</span>  | Indikerer om et bilde for øyeblikket lastes opp.                                                                       |
| feilmelding             | <span>streng</span>   | An optional error message to display below the image input                                                                             |
| "deaktivert"            | boolean               | Hvis `true`, er hele innputen deaktivert, og knappene er ikke klikkbare.                                               |

</Tab>
