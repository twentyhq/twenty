## Nåværende integrasjonsalternativer

### Native integrasjoner

Twenty tilbyr for øyeblikket native integrasjon med:

- **E-post og Kalender**: Koble til Gmail, Outlook eller SMTP/CalDAV-leverandører
- **API-tilgang**: Bruk våre REST og GraphQL APIer for å bygge tilpassede integrasjoner

For oppsett av e-post og kalender, besøk [E-post og Kalender Oppsett](/l/no/user-guide/settings/email-calendar-setup).

### Arbeidsflyter (Anbefales)

Den primære måten å koble Twenty til andre verktøy er gjennom vår innebygde **Arbeidsflyter**-funksjon:

- **HTTP-noder**: Gjør API-anrop til eksterne tjenester
- **Kodenoder**: Skriv tilpasset logikk for komplekse integrasjoner
- **Webhook-triggere**: Motta data fra eksterne systemer
- **Feltoppdateringstriggere**: Automatiser handlinger basert på CRM-endringer

Lær mer i vår [Arbeidsflyter-seksjon](/l/no/user-guide/workflows/getting-started-workflows).

### Zapier-integrasjon (Eldre versjon)

Vi vedlikeholder en Zapier-integrasjon for brukere som foretrekker kodefri automatisering:

1. Besøk [Twenty på Zapier](https://zapier.com/apps/twenty/integrations)
2. Opprett en ny Zap med Twenty som utløser eller handling
3. Generer en API-nøkkel i Innstillinger → API & Webhooks
4. Koble Twenty-arbeidsområdet ditt til Zapier

## Fremtidsvisjon: Brukerskapte tilkoblere (2026)

### Utvidelsesplattform

Vi bygger en utvidelsesplattform som vil tillate utviklere å skape apper som kode. Dette vil muliggjøre:

- **Tilpassede tilkoblere**: Bygg integrasjoner med din favorittprogramvare
- **Fellesskapsbidrag**: Del og oppdag tilkoblere bygd av andre
- **Fleksibel arkitektur**: Utvid Twenty's funksjonalitet utover kjernefunksjonene

### AI-assistert oppbygning av arbeidsflyt

Kommer i 2026, AI-assistanse vil hjelpe brukere:

- **Auto-generer arbeidsflyter**: Beskriv dine integrasjonsbehov i enkel tekst
- **Smarte forslag**: Få anbefalinger for å koble spesifikke verktøy
- **Tematbibliotek**: Få tilgang til forhåndsbygde arbeidsflyter for vanlige brukstilfeller

Fremtiden for Twenty-integrasjoner er brukerdrevet, AI-assistert og uendelig utvidbar.
