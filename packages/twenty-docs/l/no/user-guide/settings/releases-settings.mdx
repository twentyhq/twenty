## Nyeste utgivelser

Track Twenty's development progress:

- **Utgivelsesnotater**: Se hva som er nytt i hver versjon
- **Funksjonsoppdateringer**: Oppdag nye egenskaper og forbedringer

## Lab-funksjoner

Tilgang til eksperimentelle funksjoner før de offisielt blir lansert:

- **Beta-testing**: Prøv nye funksjoner mens de er under utvikling
- **Funksjonsflagg**: Aktiver eller deaktiver spesifikk eksperimentell funksjonalitet

## Hvordan få tilgang

1. Gå til **Innstillinger → Utgivelser**
2. Se utgivelsesnotater og få tilgang til **Lab**-fanen for beta-funksjoner

For flere samfunnsressurser, sjekk vår [Ressursseksjon](/l/no/user-guide/resources/glossary).
