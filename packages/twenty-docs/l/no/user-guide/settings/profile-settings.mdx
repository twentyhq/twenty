## Personlig informasjon

### Navn og e-post

- **Visningsnavn**: Oppdater hvordan navnet ditt vises til andre arbeidsområdemedlemmer
- **E-postadresse**: Endre innloggings-e-posten din (krever verifisering)
- **Profilbilde**: Last opp en tilpasset avatar eller bruk initialene dine

## Sikkerhetsinnstillinger

### Tofaktorsautentisering (2FA)

Aktiver 2FA for å gi kontoen din et ekstra sikkerhetslag:

1. Gå til **Innstillinger → Profilinnstillinger**
2. Klikk **Aktiver 2FA**
3. Skann QR-koden med autentiseringsappen din
4. Skriv inn verifiseringskoden for å bekrefte

### Passordhåndtering

- **Endre passord**: Oppdater det nåværende passordet ditt
- **Passordkrav**: Må være minst 8 tegn langt

## Profiladministrasjon

### Slett konto

<Warning>
Å slette kontoen din vil permanent fjerne tilgangen din til alle arbeidsområder. Denne handlingen kan ikke angres, du mister tilgang til alle arbeidsområder der du er medlem, og du bør vurdere å forlate enkeltstående arbeidsområder i stedet hvis du bare vil forlate bestemte lag.
</Warning>

For å slette kontoen din:

1. Gå til **Innstillinger → Profilinnstillinger**
2. Bla til **Farlig sone**
3. Klikk **Slett konto**
4. Bekreft ved å skrive inn e-postadressen din
