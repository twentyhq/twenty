## Arbeidsområdebilde

- **Upload Logo**: Add a custom workspace logo
- **Støttede formater**: PNG, JPEG, og GIF filer under 10MB
- **Remove**: Delete the current workspace logo

## Navn på arbeidsområde

- **Navn**: Endre visningsnavnet for arbeidsområdet ditt
- Dette navnet vises for alle medlemmer av arbeidsområdet

## Fareområde

<Warning>
Å slette arbeidsområdet ditt fjerner alle data permanent og kan ikke angres. Alle data i arbeidsområdet vil gå tapt for alltid, alle medlemmer vil miste tilgang umiddelbart, og denne handlingen kan ikke reverseres.
</Warning>

For å slette arbeidsområdet ditt:

1. Klikk på knappen **Slett arbeidsområde**
2. Bekreft slettingen når du blir bedt om det

**Merk**: Kun administratorer for arbeidsområdet kan slette arbeidsområder.
