## Egendefinert arbeidsområde domene

Angi en personlig tilpasset nettadresse for ditt arbeidsområde:

- Velg ditt foretrukne underdomene (for eksempel, dittfirma.twenty.com)
- Konfigurer egendefinerte domener for profesjonell merkevarestyrking

## Godkjente tilgangsdomener

Tillat automatisk tilgang til arbeidsområdet for bestemte e-postdomener.

### Hvordan det fungerer

- Legg til ditt firmas e-postdomener (for eksempel, @dittfirma.com)
- Alle med en e-post fra disse domenene kan automatisk bli med i ditt arbeidsområde
- Ingen manuell invitasjon nødvendig for teammedlemmer

### Fordeler

- **Raskere ombordstigning**: Nye teammedlemmer blir automatisk med
- **Bedre sikkerhet**: Kun verifiserte firma e-poster får tilgang
- **Mindre administrativt arbeid**: Ikke nødvendig å manuelt invitere hver teammedlem

## Oppsettinstruksjoner

1. Gå til **Innstillinger → Domener**
2. Legg til ditt egendefinerte domene eller godkjente e-postdomener
3. Bekreft domenetilgang hvis nødvendig

**Merk**: Endringer i domenet kan ta tid å tre i kraft og kan kreve bekreftelse.
