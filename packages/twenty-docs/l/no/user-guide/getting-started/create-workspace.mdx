## Trinn 1: Registrering

1. Naviger til [Twenty Sign Up](https://app.twenty.com).
2. Velg din foretrukne registreringsmetode:
   - **Fortsett med Google** for registrering med Google-konto.
   - **Fortsett med Microsoft** for registrering med Microsoft-konto.
   - Eller, **Fortsett med e-post** for registrering med e-post.

<VimeoEmbed videoId="927066829" title="Video demonstration" />
## Step 2: Choosing a Trial Period

Velg mellom to prøveperioder:

### 30 dager

Med kredittkort

### 7 dager

Uten kredittkort

Begge prøveperiodene inkluderer:

- Full tilgang
- Ubegrensede kontakter
- E-postintegrasjon
- Tilpassede objekter
- API og webhooks

Du kan klikke på "Endre plan" for å velge en annen plan eller faktureringsintervall.

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## Trinn 3: Betalingsbekreftelse og kontooppsett

Etter godkjenning av betalingen via Stripe, blir du sendt videre for å opprette arbeidsområdet ditt og brukerprofilen. Husk at du når som helst kan si opp abonnementet ditt.
