## קישור ליצירת קשר

רכיב קישור מעוצב להצגת מידע ליצירת קשר.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| תכונות    | סוג               | תיאור                                           |
| --------- | ----------------- | ----------------------------------------------- |
| className | string            | שם אופציונלי למטרת עיצוב נוסף                   |
| href      | string            | The target URL or path for the link             |
| onClick   | function          | פונקציית קריאה חוזרת שתופעל כאשר נלחץ על הקישור |
| children  | `React.ReactNode` | התוכן שיוצג בתוך הקישור                         |

</Tab>

</Tabs>

## קישור גולמי

רכיב קישור מעוצב להצגת קישורים.

<Tabs>
<Tab title="Usage">

```jsx
import { RawLink } from "/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log("Contact link clicked!", event);
  };

  return (
    <Router>
      <RawLink className href="/contact" onClick={handleLinkClick}>
        Contact Us
      </RawLink>
    </Router>
  );
};

```

</Tab>
<Tab title="Props">

| תכונות    | סוג               | תיאור                                           |
| --------- | ----------------- | ----------------------------------------------- |
| className | string            | שם אופציונלי לעיצוב נוסף                        |
| href      | מחרוזת            | The target URL or path for the link             |
| onClick   | function          | פונקציית קריאה חוזרת שתופעל כאשר נלחץ על הקישור |
| children  | `React.ReactNode` | התוכן שיוצג בתוך הקישור                         |

</Tab>
</Tabs>

## קישור מעוגל

קישור מעוצב בעיגול עם רכיב 'צ'יפ' לקישורים.

<Tabs>
<Tab title="Usage">

```jsx
import { RoundedLink } from "/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log("Contact link clicked!", event);
  };

  return (
    <Router>
      <RoundedLink href="/contact" onClick={handleLinkClick}>
        Contact Us
      </RoundedLink>
    </Router>
  );
};
```

</Tab>

<Tab title="Props">

| תכונות   | סוג               | תיאור                                           |
| -------- | ----------------- | ----------------------------------------------- |
| href     | string            | The target URL or path for the link             |
| children | `React.ReactNode` | התוכן שיוצג בתוך הקישור                         |
| onClick  | function          | פונקציית קריאה חוזרת שתופעל כאשר נלחץ על הקישור |

</Tab>
</Tabs>

## קישור חברתי

קישורים חברתיים מעוצבים, עם תמיכה בסוגי קישורים חברתיים שונים, כגון URLs, LinkedIn, ו-X (או Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| תכונות   | סוג               | תיאור                                                                                                  |
| -------- | ----------------- | ------------------------------------------------------------------------------------------------------ |
| href     | string            | The target URL or path for the link                                                                    |
| children | `React.ReactNode` | התוכן שיוצג בתוך הקישור                                                                                |
| סוג      | string            | סוג הקישורים החברתיים. האופציות כוללות: `url`, `LinkedIn`, ו-`Twitter` |
| onClick  | function          | פונקציית קריאה חוזרת שתופעל כאשר נלחץ על הקישור                                                        |

</Tab>
</Tabs>
