## אודות

מאגר GitHub של Twenty מאחסן מגוון רחב של משאבים כמו קוד מקור, תיעוד, דיונים ומעקב אחרי בעיות. כאן תוכל לגשת לקוד המלא מאחורי Twenty.

[בקר ב-Twenty ב-GitHub](https://github.com/twentyhq/twenty)

## תרומה

השתתפות בפרויקט Twenty ב-GitHub היא דרך מספקת לסייע לשיפור התוכנה שאתה משתמש בה. בין אם אתה מתקן באגים, מציע תכונות או משפר תיעוד, התרומות שלך יתקבלו בברכה.

### דיווח על בעיות

נתקלת בבעיה? [צור בעיה](https://github.com/twentyhq/twenty/issues/new) ב-GitHub, וספק כמה שיותר פרטים.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### הצעת תכונות

אילו שיפורים היית רוצה לראות ב-Twenty? לא משנה מה רמת הידע הטכני שלך, תוכל להצטרף [לשיחה כאן](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### תכנות תכונה

התחל את המסע שלך במציאת משימות ידידותיות למתחילים:

1. עבור ללשונית **[Issues](https://github.com/twentyhq/twenty/issues)** במאגר Twenty.
2. סנן לפי התווית **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** כדי למצוא משימות שמתאימות למתחילים.
3. בחר בעיה, עשה fork למאגר והתחל לתרום.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Ensure you're assigned to the issue to avoid overlapping work with other contributors.

### קוד התנהגות

זכור להיצמד ל-[קוד ההתנהגות](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) של Twenty לאורך כל תהליך התרומה שלך.
