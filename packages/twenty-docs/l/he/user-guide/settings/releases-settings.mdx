## הגרסאות האחרונות

עקוב אחר התקדמות הפיתוח של Twenty:

- **הערות גרסה**: ראה מה חדש בכל גרסה
- **עדכוני תכונות**: גלה יכולות ושיפורים חדשים

## תכונות מעבדה

גש לתכונות ניסיוניות לפני שהן מושקות רשמית:

- **בדיקות בטא**: נסה תכונות חדשות בזמן פיתוחן
- **דגלי תכונה**: אפשר או השבת פונקציונליות ניסיונית מסוימת

## כיצד לגשת

1. Go to **Settings → Releases**
2. View release notes and access the **Lab** tab for beta features

למידע נוסף על משאבי הקהילה שלנו, בדוק את [המדור משאבים](/l/he/user-guide/resources/glossary).
