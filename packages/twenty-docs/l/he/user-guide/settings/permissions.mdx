מערכת ההרשאות של Twenty מאפשרת לך לשלוט בגישה לשלושה תחומים עיקריים:

- **אובייקטים ושדות**: שלוט במי יכול להציג, לערוך או למחוק רשומות ושדות בודדים
- **הגדרות**: נהל גישה להגדרות מרחב העבודה ולתפקודים מנהליים
- **Actions**: Control general workspace actions like importing data or sending emails

## צור תפקיד

To create a new role:

1. גש ל**הגדרות → תפקידים**
2. תחת **כל התפקידים**, לחץ על **+ צור תפקיד**
3. הזן שם לתפקיד
4. בכרטיסיית **הרשאות** המחדלית, קבע הרשאות
5. לחץ על **שמור** לסיום

## מחק תפקיד

כדי למחוק תפקיד:

1. גש ל**הגדרות → תפקידים**
2. לחץ על התפקיד שברצונך להסיר
3. פתח את כרטיסיית **ההגדרות**, ולאחר מכן לחץ על **מחק תפקיד**
4. לחץ על **אשר** במודאל

שים לב: אם תפקיד נמחק, כל חבר במרחב העבודה שמוקצה לו יועבר אוטומטית לתפקיד המחדל. כל התפקידים מלבד **מנהל** ניתנים למחיקה. תמיד חייב להיות לפחות חבר אחד מוקצה לתפקיד **מנהל**.

## הקצה תפקידים לחברים

### צפה בהקצאות הנוכחיות

- גש ל**הגדרות → תפקידים**
- ראה את כל התפקידים וכמה חברים מוקצים לכל אחד מהם
- צפה באילו חברים מחזיקים באילו תפקידים

### הקצה תפקיד לחבר

1. גש ל**הגדרות → תפקידים**
2. לחץ על התפקיד שברצונך להקצות
3. פתח את כרטיסיית **הקצאה**
4. לחץ על **+ הקצה לחבר**
5. בחר חבר ממסך הרשימה
6. אשר את ההקצאה

### הגדר תפקיד מחדל

1. גש ל**הגדרות → תפקידים**
2. In the **Options** section, find **Default Role**
3. בחר לאיזה תפקיד חברים חדשים צריכים להיות מוקצים
4. חברים חדשים למרחב העבודה יוקצו לתפקיד זה כאשר יצטרפו

**הערה**: ניתן להקצות תפקידים רק לחברים קיימים. To invite new members, use [Member Management](/l/he/user-guide/settings/member-management).

## התאם אישית הרשאות

הרשאות קובעות מה כל תפקיד יכול לגשת אליו או לשנות בתוך מרחב העבודה שלך, כולל רשומות אובייקטים של מרחבי העבודה, הגדרות ופעולות.

### הרשאות אובייקטים ושדות

שלוט בגישה לרשומות ושדות בודדים:

#### הרשאות ברמת האובייקט

- תחת **כל האובייקטים**, יש להחיל הרשאות כמו **הצג רשומה**, **ערוך רשומות**, **מחק רשומות**, או **השמד רשומות** על כל האובייקטים.
- תחת **הרשאות ברמת האובייקט**, קבע חריגים לאובייקטים פרטניים. אלו עוקפים את ההגדרות של **כל האובייקטים**.

#### הרשאות ברמת השדה

- קבע הרשאות לשדות בודדים בתוך כל אובייקט
- שלוט במי יכול **לראות שדה**, **לערוך שדה** או שאין להם **גישה** לשדות ספציפיים
- הרשאות שדה פועלות כמו הרשאות אובייקט עם ירושה ועקיפות

#### ניהול עקיפות הרשאות

כדי לעקוף הרשאות של הורה וליישם כללים מחמירים יותר:

- לחץ על **X** כדי להסיר את הכלל היורש
- בחר את ההרשאות הספציפיות לאובייקט או לשדה הנבחר
- לחץ על האייקון **בטל** (חץ מעגלי) כדי לשחזר שינויים

כשמוכן, לחץ על **סיים**, ולאחר מכן **שמור** לאחר שהוחזרת לדף התפקיד.

### הרשאות הגדרות מרחב עבודה

שלוט בגישה להגדרות מרחב העבודה בשתי דרכים:

- שנה את **הרשאה כללית להגדרות** כדי לאפשר גישה מלאה
- Or enable specific permissions (e.g., API key generation, workspace preferences, role assignment, data model configuration, security settings, and workflow management)

### הרשאות פעולה במרחב עבודה

שלוט בגישה לפעולות כלליות במרחב עבודה:

- שנה את **גישה כללית לאפליקציות** כדי לאפשר הרשאות מלאות
- או הפעל פעולות בודדות כמו **שלח אימייל**, **ייבא CSV** ו**ייצא CSV**
