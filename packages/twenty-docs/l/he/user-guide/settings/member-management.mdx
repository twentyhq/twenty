## הזמן חברים חדשים

### באמצעות קישור הזמנה

1. עבור אל **הגדרות → חברים**
2. העתק את קישור ההזמנה למרחב העבודה
3. שתף את הקישור עם חברי צוות חדשים
4. הם יקבלו גישה ברגע שיירשמו

### הזמנה ישירה בדוא"ל

1. עבור אל **הגדרות → חברים**
2. הכנס את כתובת הדוא"ל של האדם
3. לחץ על **הזמן**
4. הם יקבלו הזמנה בדוא"ל

## הסר חברים

### מחק חבר

1. עבור אל **הגדרות → חברים**
2. מצא את החבר שאתה רוצה להסיר
3. לחץ על כפתור מחיקה/הסרה ליד שמם
4. אשר את ההסרה

**הערה**: חברים שהוסרו מאבדים גישה מיידית אך ניתן להזמינם שוב מאוחר יותר.

## צריך עזרה?
