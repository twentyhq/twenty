## אפשרויות האינטגרציה הנוכחיות

### אינטגרציות מקוריות

Twenty מציעה כיום אינטגרציה מקורית עם:

- **דואר אלקטרוני ולוח שנה**: התחבר ל-Gmail, Outlook או ספקי SMTP/CalDAV
- **גישה ל-API**: השתמש ב-API של REST ו-GraphQL כדי לבנות אינטגרציות מותאמות אישית

לצורך התקנה בדוא"ל ולוח שנה, עבור ל-[הגדרת דוא"ל ולוח שנה](/l/he/user-guide/settings/email-calendar-setup).

### Workflows (Recommended)

The primary way to connect Twenty to other tools is through our in-app **Workflows** feature:

- **צמתים HTTP**: בצע קריאות API לשירותים חיצוניים
- **צמתים בקוד**: כתוב היגיון מותאם אישית לאינטגרציות מורכבות
- **טריגרים של Webhook**: קבל נתונים ממערכות חיצוניות
- **טריגרים לעדכון שדות**: יישם פעולות על בסיס שינויים ב-CRM

Learn more in our [Workflows section](/l/he/user-guide/workflows/getting-started-workflows).

### אינטגרציית Zapier (מורשת)

אנו מחזיקים באינטגרציה עם Zapier למשתמשים המעדיפים אוטומציה בלי קוד:

1. עבור ל-[Twenty על Zapier](https://zapier.com/apps/twenty/integrations)
2. צור Zap חדש עם Twenty כטריגר או פעולה
3. צור מפתח API בהגדרות → API & Webhooks
4. חבר את סביבת העבודה שלך ב-Twenty ל-Zapier

## חזון עתידי: חיבורי קהילה העשויים ע"י משתמשים (2026)

### פלטפורמת הרחבות

אנו בונים פלטפורמת הרחבות שתאפשר למפתחים ליצור אפליקציות כקוד. זה יאפשר:

- **מחברים מותאמים אישית**: בנה אינטגרציות עם התוכנות האהובות עליך
- **תרומות קהילתיות**: שתף ומצא מחברים שנוצרו על ידי אחרים
- **ארכיטקטורה גמישה**: הרחב את היכולות של Twenty מעבר לתכונות ליבה

### יצירת תזרימים בעזרת בינה מלאכותית

ב-2026, עזרה מבוססת AI תסייע למשתמשים:

- **Auto-Generate Workflows**: Describe your integration needs in plain language
- **המלצות חכמות**: קבל הצעות כיצד לחבר את הכלים הספציפיים שלך
- **Template Library**: Access pre-built workflows for common use cases

העתיד של אינטגרציות Twenty מונע על ידי קהילה, נתמך על ידי בינה מלאכותית וניתן להרחבה אינסופית.
