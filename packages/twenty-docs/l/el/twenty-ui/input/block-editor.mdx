Χρησιμοποιεί έναν επεξεργαστή κειμένου πλούσιο σε μπλοκ από [BlockNote](https://www.blocknotejs.org/) για να επιτρέψει στους χρήστες να επεξεργάζονται και να βλέπουν μπλοκ περιεχομένου.

<Tabs>
<Tab title="Usage">

```jsx
import { useBlockNote } from "@blocknote/react";
import { BlockEditor } from "@/ui/input/editor/components/BlockEditor";

export const MyComponent = () => {
  const BlockNoteEditor = useBlockNote();

  return <BlockEditor editor={BlockNoteEditor} />;
};
```

</Tab>
<Tab title="Props">

| Ιδιότητες    | Τύπος             | Περιγραφή                                       |
| ------------ | ----------------- | ----------------------------------------------- |
| επεξεργαστής | `BlockNoteEditor` | Η παρουσία ή η διαμόρφωση του επεξεργαστή μπλοκ |
