## Τιμολόγηση

<AccordionGroup>
<Accordion title="Can I use Twenty for free while self-hosting?">
Ναι, μπορείτε να χρησιμοποιήσετε το Twenty δωρεάν κατά τη διάρκεια της αυτοφιλοξενίας. Θα έχετε πρόσβαση σε όλα όσα περιλαμβάνονται στη συνδρομή (cloud) Pro, εκτός από την υποστήριξη από την ομάδα πυρήνα μας. Η υποστήριξη είναι προσβάσιμη μέσω της κοινότητας Discord μας.

Αν θέλετε να φιλοξενήσετε μόνοι σας και να έχετε πρόσβαση στα χαρακτηριστικά που περιλαμβάνονται στη συνδρομή (cloud) Organization, συμπεριλαμβανομένης της υποστήριξης από την ομάδα πυρήνα μας, μπορείτε να το κάνετε επιλέγοντας τις πληρωμένες άδειες Organization, διαθέσιμες στην αυτοφιλοξενία. </Accordion>

<Accordion title="Do you offer free seats for view-only users?">
Δεν προσφέρουμε δωρεάν θέσεις. Η κοστολόγηση είναι ανά χρήστη και κάθε χρήστης χρειάζεται μια άδεια πρόσβασης στο Twenty.
</Accordion>

<Accordion title="Where can I switch my subscription to the Organization plan?">
Μπορείτε να το κάνετε στις `Ρυθμίσεις → Χρέωση`. Στη συνέχεια, κάντε κλικ στο `Μετάβαση στο Organization`.
</Accordion>

<Accordion title="Where can I switch my subscription to the Pro plan?">
Please reach out to our team directly via the Support, there is no easy way to do this using the UI at the moment.
</Accordion>

<Accordion title="Where can I switch my subscription to Yearly?">
Μπορείτε να το κάνετε στις `Ρυθμίσεις → Χρέωση`. Στη συνέχεια, κάντε κλικ στο `Μετάβαση σε Ετήσια`.
</Accordion>

<Accordion title="Where can I switch my subscription to Monthly?">
Please reach out to our team directly via the Support, there is no easy way to do this using the UI at the moment.
</Accordion>

<Accordion title="Where can I see my credit usage?">
You will find this under `Settings → Billing`.
</Accordion>

<Accordion title="How many workflow credits do we have?">
Ο αριθμός των μονάδων ποικίλλει ανάλογα με το πλάνο. Ένας χώρος εργασίας σε **δοκιμή λαμβάνει 5 εκατομμύρια μονάδες**, ένας σε **πλάνο Pro λαμβάνει 10 εκατομμύρια μονάδες ανά μήνα** και ένας σε **πλάνο Organization λαμβάνει 20 εκατομμύρια μονάδες ανά μήνα**.
</Accordion>

<Accordion title="How does workflow credit consumption work?">
Each workflow action consumes credits based on its complexity:
- **Basic internal operations** (such as search, update, create records) consume very few credits
- **More complex operations** like code nodes and requests to external services consume more credits
- **AI prompts** (coming soon!) θα καταναλώνουν επίσης περισσότερες μονάδες ανάλογα με τη χρήση

Οι μονάδες αφαιρούνται σε πραγματικό χρόνο όταν εκτελούνται οι ροές εργασίας. Μπορείτε να παρακολουθείτε τη χρήση σας στις **Ρυθμίσεις → Χρέωση** για να παρακολουθείτε την κατανάλωση και τις υπόλοιπες μονάδες. </Accordion>

<Accordion title="Can I buy more workflow credits?">
You can buy additional credits under `Settings → Billing`.
</Accordion>
</AccordionGroup>

## Χρέωση

<AccordionGroup>
<Accordion title="Where can I manage the Billing details?">
You can do so under `Settings → Billing`.
</Accordion>

<Accordion title="Where can I update my payment method?">
Μπορείτε να το κάνετε στις `Ρυθμίσεις → Χρέωση`. Στη συνέχεια, κάντε κλικ στο `Προβολή λεπτομερειών χρέωσης`. Θα μπορέσετε να προσθέσετε μια νέα μέθοδο πληρωμής εκεί.
</Accordion>

<Accordion title="Where can I update my billing information?">
Μπορείτε να το κάνετε στις `Ρυθμίσεις → Χρέωση`. Στη συνέχεια, κάντε κλικ στο `Προβολή λεπτομερειών χρέωσης`. Θα μπορέσετε να επεξεργαστείτε τις πληροφορίες χρέωσης εκεί.
</Accordion>

<Accordion title="Where can I access my invoices?">
Μπορείτε να το κάνετε στις `Ρυθμίσεις → Χρέωση`. Στη συνέχεια, κάντε κλικ στο `Προβολή λεπτομερειών χρέωσης`. Θα δείτε όλα τα τιμολόγιά σας στο κάτω μέρος της οθόνης.
</Accordion>
</AccordionGroup>
