## Εντοπισμός Σφαλμάτων με Ροές Εργασίας

Χρησιμοποιήστε το περιβάλλον **Ροών Εργασίας** για τον εντοπισμό ζητημάτων:

- Access via the **Runs** panel in the workflow editor
- Κάντε κλικ σε μεμονωμένες εκτελέσεις για να δείτε δεδομένα εισόδου/εξόδου για κάθε βήμα
- Ελέγξτε την κατάσταση εκτέλεσης, τα μηνύματα σφαλμάτων και τη ροή δεδομένων μεταξύ βημάτων

## Συνηθισμένα Θέματα και Λύσεις

### Η Ροή Εργασίας Δεν Εκκινεί

**Πρόβλημα**: Η ροή εργασίας δεν εκτελείται όταν αναμένεται.

**Λύσεις**:

- Βεβαιωθείτε ότι η ροή εργασίας είναι **Ενεργή** (όχι σε λειτουργία πρόχειρου)
- Ελέγξτε τη διαμόρφωση του ενεργοποιητή για να ταιριάζει με τη δομή των δεδομένων σας
- Για ενεργοποιητές εγγραφών, βεβαιωθείτε ότι έχουν επιλεγεί τα σωστά αντικείμενα και πεδία
- Για ενεργοποιητές Webhook, επαληθεύστε τη διεύθυνση URL και τη μορφή δεδομένων που αναμένονται
- Για προγραμματισμένους ενεργοποιητές, ελέγξτε τη διαμόρφωση του χρονοδιαγράμματος

### Οι Φόρμες Δεν Είναι Προσβάσιμες

**Πρόβλημα**: Δυσκολία στην εύρεση ή πρόσβαση στις ενέργειες των φορμών σε αυτοματοποιημένες ροές εργασιών.

<Warning>
Οι φόρμες έχουν σχεδιαστεί επί του παρόντος μόνο για μη αυτόματους ενεργοποιητές. Για ροές εργασιών με άλλους ενεργοποιητές (Δημιουργία Εγγραφής, Ενημέρωση κ.λπ.), οι φόρμες είναι προσβάσιμες μόνο μέσω της διεπαφής εκτέλεσης ροής εργασίας, κάτι που δεν είναι η αναμενόμενη εμπειρία χρήστη. Μια πλατφόρμα ειδοποιήσεων θα κυκλοφορήσει το 2026 για την υποστήριξη φορμών σε αυτοματοποιημένες ροές εργασιών.
</Warning>

**Λύση**: Χρησιμοποιήστε μη αυτόματους ενεργοποιητές όταν απαιτούνται φόρμες ή αναδιαρθρώνετε τις ροές εργασιών για να αποφύγετε τις φόρμες σε αυτοματοποιημένες ροές.

### Υψηλή Κατανάλωση Πιστώσεων

**Πρόβλημα**: Οι ροές εργασίας καταναλώνουν περισσότερες πιστώσεις από τις αναμενόμενες.

**Κοινές Αιτίες & Λύσεις**:

- **Αναποτελεσματικές Κλήσεις API**: Καταρτίστε τις κλήσεις API όποτε είναι δυνατό αντί για μεμονωμένα αιτήματα
- **Εσφαλμένη Διαμόρφωση Μη Αυτόματου Ενεργοποιητή**: Χρησιμοποιήστε `Μαζική` διαθεσιμότητα αντί για `Μεμονωμένη` για την επεξεργασία πολλών εγγραφών σε μια εκτέλεση ροής εργασίας
- **Έλλειψη Φίλτρων**: Προσθέστε συνθήκες για τον τερματισμό ροών εργασιών όταν δεν πληρούνται τα κριτήρια
- **Περιττά Βήματα**: Αφαιρέστε περιττές ενέργειες και βελτιστοποιήστε τη λογική ροής εργασίας
- **Σε Πραγματικό Χρόνο vs. Προγραμματισμένες**: Χρησιμοποιήστε προγραμματισμένες ροές εργασίας για μη επείγουσες διαδικασίες

### Οριακές Ροές Εργασίες

**Πρόβλημα**: Υπέρβαση του ορίου των 100 ταυτόχρονων ροών εργασιών ανά χώρο εργασίας.

<Warning>
You cannot run more than 100 workflows in parallel at any given time per workspace.
</Warning>

**Λύσεις**:

- Χρησιμοποιήστε `Μαζική` διαθεσιμότητα για μη αυτόματους ενεργοποιητές για την επεξεργασία πολλών εγγραφών σε μία εκτέλεση
- Εφαρμόστε καθυστερήσεις μεταξύ των εκτελέσεων ροής εργασιών χρησιμοποιώντας προγραμματισμένους ενεργοποιητές
- Βελτιστοποιήστε τις ροές εργασίας για ταχύτερη εκτέλεση και μείωση του ταυτόχρονου χρόνου εκτέλεσης
- Σκεφτείτε τη μαζική επεξεργασία κατά τις μη αιχμής ώρες

### Περιορισμός Ρυθμού API

**Πρόβλημα**: Αποτυχία εξωτερικών κλήσεων API λόγω περιορισμών ρυθμού.

**Λύσεις**:

- Χρησιμοποιήστε προγραμματισμένες ροές εργασιών αντί για εκκινήσεις σε πραγματικό χρόνο όπου είναι δυνατόν
- Εφαρμόστε καθυστερήσεις μεταξύ των κλήσεων API στις ενέργειες Κώδικα
- Καταρτίστε τα αιτήματα API όταν το υποστηρίζει η εξωτερική υπηρεσία
- Παρακολουθήστε τις εκτελέσεις ροής εργασιών για σφάλματα περιορισμού ρυθμού και προσαρμόστε το χρονοδιάγραμμα

### Ζητήματα με Επαναλήπτη

**Πρόβλημα**: Οι ενέργειες επαναλήπτη δεν λειτουργούν όπως αναμένεται.

**Λύσεις**:

- **Σημείωση**: Ο Διαδοχέας βρίσκεται επί του παρόντος σε beta. Ενεργοποιήστε το μέσω Ρυθμίσεων → Εκδόσεις → Εργαστήριο
- Επιβεβαιώστε ότι η είσοδος είναι ένας πίνακας εγγραφών από προηγούμενο βήμα
- Ελέγξτε ότι οι ενέργειες εντός του επαναλήπτη είναι σωστά διαμορφωμένες
- Χρησιμοποιήστε τον επαναλήπτη με `Μαζική` μη αυτόματους ενεργοποιητές για βέλτιστη απόδοση

### Έλλειψη Δικαιωμάτων

**Πρόβλημα**: Δεν είναι δυνατή η πρόσβαση στην ενότητα ροών εργασιών.

<Warning>
Εάν δεν βλέπετε την ενότητα Ροές Εργασιών, αυτό οφείλεται σε πρόβλημα αδειών. Επικοινωνήστε με τον διαχειριστή της χώρου εργασίας σας για να σας παρέχει πρόσβαση στις ροές εργασιών.
</Warning>

## Συμβουλές Βελτιστοποίησης

### Βέλτιστες Πρακτικές Απόδοσης

- **Ξεκινήστε απλά** και προσθέστε πολυπλοκότητα σταδιακά
- **Χρησιμοποιήστε `Μαζική` διαθεσιμότητα** για μη αυτόματους ενεργοποιητές για επεξεργασία πολλών εγγραφών με αποδοτικότητα
- **Προσθέστε φίλτρα νωρίς** για να προστατέψετε τις ροές εργασίας όταν δεν πληρούνται τα κριτήρια
- **Καταρτίστε κλήσεις API** και χρησιμοποιήστε προγραμματισμένες ροές εργασιών για μη επείγουσες διαδικασίες
- **Αποκρύψτε τις εκτελέσεις ροής εργασίας** από τις σελίδες που δείχνουν όλες τις ροές εργασίας για να βελτιώσετε την απόδοση φόρτωσης
- **Παρακολουθείτε τη χρήση πιστώσεων** τακτικά στις Ρυθμίσεις → Χρέωση

### Πρόληψη Σφαλμάτων

- **Δοκιμάστε σε λειτουργία πρόχειρου** πριν ενεργοποιήσετε τις ροές εργασιών
- **Επαληθεύστε απαντήσεις API** και εφαρμόστε ενέργειες εφεδρικής λειτουργίας
- **Χρησιμοποιήστε περιγραφικά ονόματα βημάτων** για ευκολότερη συντήρηση
- **Τεκμηριώστε τη σύνθετη λογική** για τα μέλη της ομάδας
