Παρακάτω είναι παραδείγματα ροών εργασίας που μπορείτε να υλοποιήσετε για να συνδέσετε το Twenty με το υπόλοιπο σύστημα σας.

## Data Ingestion Use Cases

### Υποβολές Webform

**Πρόβλημα**: Χρειάζεται να καταγράψετε δυνητικούς πελάτες από φόρμες ιστοτόπου, σελίδες προορισμού ή φόρμες επικοινωνίας απευθείας στο Twenty.

**Solution**: Use webhook triggers to automatically create records from form submissions.

**Ρύθμιση**:

- Δημιουργήστε μια ροή εργασίας με ενεργοποιητή Webhook
- Ρυθμίστε το webhook να αναμένει δεδομένα φόρμας (όνομα, email, εταιρεία κλπ)
- Ορίστε τη μέθοδο του webhook σε POST
- Ορίστε τη δομή του αναμενόμενου σώματος στην ενεργοποίηση

**Ενεργοποίηση**: Webhook (αίτηση POST από τη φόρμα σας)

**Ενέργειες**:

- Search Records to check if person/company already exists
- Branch: If exists → Update Record, If not → Create Record
- Δημιουργήστε Εγγραφή για εργασία παρακολούθησης ανατεθειμένη σε αντιπρόσωπο πωλήσεων
- Στείλτε ειδοποίηση Email στην ομάδα πωλήσεων

### Συγχρονισμός Δεδομένων Προϊόντος

**Πρόβλημα**: Η ομάδα πωλήσεων σας χρειάζεται ορατότητα στη χρήση προϊόντος, τιμολόγηση ή δεδομένα υιοθέτησης χαρακτηριστικών αποθηκευμένα στην αποθήκη σας.

**Solution**: Regularly sync product data into Twenty to give sellers context about their accounts.

**Ενεργοποίηση**: Με Χρονοδιάγραμμα (καθημερινά ή εβδομαδιαία)

**Ενέργειες**:

- Αίτηση HTTP προς API της αποθήκης δεδομένων σας
- Ενέργεια κωδικοποίησης για επεξεργασία και μορφοποίηση των δεδομένων
- Χρησιμοποιήστε τη λειτουργία Επαναληπτή για τα επόμενα βήματα
- Αναζητήστε Εγγραφές για να βρείτε εγγραφές εταιρείας που ταιριάζουν
- Ενημερώστε Εγγραφή για να προσθέσετε μετρήσεις χρήσης προϊόντος
- Δημιουργήστε Εγγραφή για εργασίες όταν η χρήση πέσει κάτω από το όριο

### Σημειώσεις από Καταγραφείς Κλήσεων

**Πρόβλημα**: Σημαντικές γνώσεις από κλήσεις πωλήσεων χάνονται ή δεν καταγράφονται σωστά στο CRM.

**Solution**: Automatically create notes and action items from call recording systems.

**Ενεργοποίηση**: Webhook (από πλατφόρμα καταγραφής κλήσεων)

**Ενέργειες**:

- Code action to extract meeting summary and action items
- Αναζητήστε Εγγραφές για να βρείτε την σχετική ευκαιρία ή επαφή
- Create Record for a note with meeting summary
- Δημιουργήστε Εγγραφή για εργασίες παρακολούθησης βάσει στοιχείων ενεργειών
- Send Email to attendees with summary and next steps

### Εμπλουτισμός Δεδομένων

**Problem**: Your contact and company records lack important demographic and firmographic information.

**Λύση**: Εμπλουτίστε αυτόματα εγγραφές χρησιμοποιώντας εξωτερικούς παρόχους δεδομένων.

**Ενεργοποίηση**: Δημιουργείται Εγγραφή (αντικείμενο Άνθρωποι ή Εταιρείες)

**Ενέργειες**:

- Αίτηση HTTP προς API εμπλουτισμού
- Ενέργεια κωδικοποίησης για επεξεργασία της απάντησης εμπλουτισμού
- Χρησιμοποιήστε τη λειτουργία Επαναληπτή για τα επόμενα βήματα
- Ενημερώστε Εγγραφή με πρόσθετες πληροφορίες εταιρείας/επαφής
- Δημιουργήστε Εγγραφή για εργασία πώλησης εάν ταυτοποιηθεί προοπτική υψηλής αξίας
- Στείλτε Email ειδοποίηση εάν ο εμπλουτισμός αποκαλύψει βασικούς αγοραστικούς δείκτες

## Περιπτώσεις Χρήσης Διανομής Δεδομένων

### Διαχείριση Συνδρομητών Ενημερωτικού Δελτίου

**Problem**: You want to send marketing emails to specific segments of your CRM data using specialized email tools.

**Λύση**: Εξάγετε λίστες συνδρομητών στην πλατφόρμα email μάρκετινγκ σας όταν είναι αναγκαίο.

**Ρύθμιση**: Δημιουργήστε μια προβολή στο Twenty με όλους τους παραλήπτες του ενημερωτικού δελτίου

**Ενεργοποίηση**: Εκκίνηση Χειροκίνητα (όταν δεν έχει επιλεγεί εγγραφή)

**Ενέργειες**:

- Search Records using the newsletter view criteria
- Code action to format email addresses for your email platform
- Αίτηση HTTP για να προσθέσετε συνδρομητές στο εργαλείο email μάρκετινγκ σας
- Create Record for campaign tracking
- Send Email confirmation to marketing team

### Email Sequence Triggers

**Πρόβλημα**: Θέλετε να ενεργοποιήσετε περίπλοκες αλληλουχίες email βάσει γεγονότων του CRM χρησιμοποιώντας εξειδικευμένα εργαλεία αυτοματοποίησης email.

**Solution**: Send new leads or customers to your email automation platform when specific events occur.

**Trigger**: Record is Created (People object with specific criteria)

**Ενέργειες**:

- Ενέργεια κωδικοποίησης για τον καθορισμό της κατάλληλης αλληλουχίας email
- Αίτηση HTTP για να προσθέσετε επαφή στην πλατφόρμα αυτοματοποίησης email
- Update Record to track sequence enrollment
- Create Record for follow-up task to monitor engagement

### Lead Scoring Integration

**Problem**: You need sophisticated lead scoring that combines CRM data with external signals.

**Λύση**: Στείλτε δεδομένα leads σε εξωτερικά εργαλεία σκοραρίσματος ή εφαρμόστε λογική σκοραρίσματος μέσα στις ροές εργασιών.

**Επιλογή 1 - Εξωτερικό Εργαλείο**:

**Ενεργοποίηση**: Ενημερώνεται Εγγραφή (αντικείμενο Άνθρωποι)

**Ενέργειες**:

- HTTP Request to send lead data to scoring platform
- Ενέργεια κωδικοποίησης για την επεξεργασία της απάντησης σκορ
- Update Record with lead score
- Create Record for sales task if score exceeds threshold

**Επιλογή 2 - Εσωτερική Λογική**:

**Ενεργοποίηση**: Ενημερώνεται Εγγραφή (αντικείμενο Άνθρωποι)

**Ενέργειες**:

- Code action with scoring algorithm (company size, industry, behavior)
- Ενημερώστε Εγγραφή με το υπολογισμένο σκορ
- Send Email alert to sales rep for high-scoring leads

### Γενιά Τιμολογίων

**Problem**: When deals close, your billing system needs to be updated with customer and deal information.

**Λύση**: Στείλτε αυτόματα δεδομένα συμφωνιών στο σύστημα τιμολόγησής σας όταν κερδίζονται ευκαιρίες.

**Ενεργοποίηση**: Ενημερώνεται Εγγραφή (αντικείμενο Ευκαιρίες, Στάδιο = "Κλειστό Κερδισμένο")

**Ενέργειες**:

- Search Records to get complete customer information
- Ενέργεια κωδικοποίησης για τη μορφοποίηση δεδομένων για το σύστημα τιμολόγησης
- Αίτηση HTTP για δημιουργία πελάτη στην πλατφόρμα τιμολόγησης
- Αίτηση HTTP για δημιουργία τιμολογίου
- Ενημερώστε Εγγραφή για να αποθηκεύσετε την αναφορά τιμολογίου
- Στείλτε Email στην οικονομική ομάδα με λεπτομέρειες τιμολογίου

## Προηγμένα Μοτίβα Ενσωμάτωσης

### Δικτύωση Δεδομένων Δύο Κατευθύνσεων

**Πρόβλημα**: Πρέπει να διατηρείται συγχρονισμός δεδομένων μεταξύ του Twenty και άλλου συστήματος και στις δύο κατευθύνσεις.

**Solution**: Combine scheduled workflows with webhook triggers for real-time sync.

**Από το Twenty στο Εξωτερικό Σύστημα**:

**Ενεργοποίηση**: Ενημερώνεται Εγγραφή (οποιοδήποτε σχετικό αντικείμενο)

**Ενέργειες**:

- Αίτηση HTTP για ενημέρωση εξωτερικού συστήματος
- Ενημερώστε Εγγραφή για την παρακολούθηση κατάστασης συγχρονισμού και χρονικής σήμανσης

**From External System to Twenty**:

**Trigger**: Webhook (from external system)

**Ενέργειες**:

- Search Records to find matching record
- Update Record with new data from external system
- Δημιουργήστε Εγγραφή για εργασία επίλυσης συγκρούσεων αν είναι απαραίτητο

### Επεξεργασία Δεδομένων Πολλαπλών Βημάτων

**Πρόβλημα**: Τα δεδομένα από εξωτερικές πηγές χρειάζονται περίπλοκη επεξεργασία πριν να μπορούν να χρησιμοποιηθούν στο Twenty.

**Λύση**: Χρησιμοποιήστε Ενέργειες Κωδικοποίησης για μετασχηματισμό και επικύρωση δεδομένων.

**Ενεργοποίηση**: Webhook ή Σε Πρόγραμμα

**Ενέργειες**:

- Code action to validate incoming data format
- Code action to transform data structure
- Ενέργεια κωδικοποίησης για εφαρμογή επιχειρηματικών κανόνων και υπολογισμών
- Αναζητήστε Εγγραφές για να ελέγξετε για διπλότυπα
- Δημιουργήστε ή Ενημερώστε Εγγραφή με επεξεργασμένα δεδομένα
- Στείλτε Ειδοποίηση Email αν εντοπιστούν προβλήματα ποιότητας δεδομένων

## Συμβουλές Εφαρμογής

- Αποθηκεύστε τα API keys με ασφάλεια στις Ρυθμίσεις → API & Webhooks
- Χρησιμοποιήστε HTTPS για όλες τις εξωτερικές κλήσεις API
- Λάβετε υπόψη τα όρια ρυθμού API - χρησιμοποιήστε προγραμματισμένες ροές εργασιών όταν είναι δυνατόν
- Consider batch updates "On a Schedule" when real-time processing isn't required
- Θυμηθείτε το όριο 100 ταυτόχρονων ροών εργασίας ανά χώρο εργασίας - χρησιμοποιήστε "Μαζική" διαθεσιμότητα για χειροκίνητες ενεργοποιήσεις όταν επεξεργάζεστε πολλαπλές εγγραφές (δείτε [Δυνατότητες Ροής Εργασιών](/l/el/user-guide/workflows/workflow-features) για λεπτομέρειες)
- Δοκιμάστε με δείγμα δεδομένων πριν ενεργοποιήσετε τις ροές εργασιών

Για αντιμετώπιση προβλημάτων ενσωμάτωσης, δείτε τον οδηγό μας [Αντιμετώπιση Προβλημάτων Ροής Εργασιών](/l/el/user-guide/workflows/workflow-troubleshooting). Για βοήθεια στην υλοποίηση σύνθετων ενσωματώσεων, εξετάστε τις [Επαγγελματικές Υπηρεσίες](/l/el/user-guide/workflows/professional-services) μας.
