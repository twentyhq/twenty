Οι πιστώσεις ροής εργασίας ενεργοποιούν τους αυτοματισμούς σας στο Twenty. Η κατανόηση του τρόπου λειτουργίας τους σας βοηθά να βελτιστοποιήσετε το κόστος και να διαχειριστείτε αποτελεσματικά τον προϋπολογισμό των αυτοματισμών σας.

## Κατανομή Πιστώσεων ανά Σχέδιο

Οι πιστώσεις ροής εργασίας κατανέμονται ανά χώρο εργασίας βάσει του σχεδίου συνδρομής σας:

- **Δοκιμή**: 5 εκατομμύρια πιστώσεις
- **Επαγγελματικό Σχέδιο**: 10 εκατομμύρια πιστώσεις ανά μήνα
- **Σχέδιο Οργάνωσης**: 20 εκατομμύρια πιστώσεις ανά μήνα

## Πώς Λειτουργεί η Κατανάλωση Πιστώσεων

Οι πιστώσεις καταναλώνονται όταν εκτελούνται ροές εργασίας, όχι όταν τις δημιουργείτε. Κάθε ενέργεια ροής εργασίας καταναλώνει μονάδες με βάση τη πολυπλοκότητά της:

### Κατανάλωση Πιστώσεων ανά Τύπο Δράσης

- **Βασικές εσωτερικές λειτουργίες**: Πολύ χαμηλή κατανάλωση πιστώσεων
  - Αναζήτηση Εγγραφών
  - Create Record
  - Ενημέρωση Εγγραφής
  - Διαγραφή Εγγραφής
  - Δράσεις Φορμών

- **Σύνθετες λειτουργίες**: Υψηλότερη κατανάλωση πιστώσεων
  - Δράσεις Κώδικα (εκτέλεση JavaScript)
  - Αιτήσεις HTTP σε εξωτερικές υπηρεσίες

- **Χαρακτηριστικά Τεχνητής Νοημοσύνης**: Σημαντική κατανάλωση πιστώσεων (σύντομα διαθέσιμο)
  - Οι ερωτήσεις και οι επεξεργασίες τεχνητής νοημοσύνης θα καταναλώνουν πιστώσεις βάσει χρήσης

### Real-Time Deduction

Οι πιστώσεις αφαιρούνται σε πραγματικό χρόνο καθώς εκτελούνται οι ροές εργασίας. Αυτό σημαίνει:

- Οι σχέδια εργασίας δεν καταναλώνουν πιστώσεις
- Μόνο οι ενεργές, εκτελούμενες ροές εργασίας χρησιμοποιούν τις κατανεμημένες πιστώσεις σας
- Οι αποτυχημένες ροές εργασίας εξακολουθούν να καταναλώνουν πιστώσεις για τα ολοκληρωμένα βήματα

## Διαχείριση Πιστώσεων

### Έλεγχος Κατανάλωσης Πιστώσεων

1. Μεταβείτε στις **Ρυθμίσεις → Χρέωση**
2. Εμφανίστε την τρέχουσα κατανάλωση πιστώσεων και το υπόλοιπο
3. Παρακολουθήστε τα πρότυπα χρήσης για να βελτιστοποιήσετε τις ροές εργασίας σας

### Αγορά Πρόσθετων Πιστώσεων

Εάν χρειάζεστε περισσότερες πιστώσεις πέραν των κατανεμημένων:

1. Μεταβείτε στις **Ρυθμίσεις → Χρέωση**
2. Κάντε κλικ στην επιλογή αγοράς πρόσθετων πιστώσεων. Διατίθενται πακέτα διαφόρων μεγεθών.
3. Οι πιστώσεις προστίθενται στο τρέχον υπόλοιπό σας

## Καλύτερες Πρακτικές

### Σχεδιασμός Ροής Εργασίας

- **Ξεκινήστε Απλά**: Αρχίστε με βασικές δράσεις και προσθέστε πολυπλοκότητα σταδιακά
- **Δοκιμή σε Σχέδιο**: Ελέγξτε σχολαστικά τις ροές εργασίας πριν από την ενεργοποίηση για να αποφύγετε την σπατάλη πιστώσεων σε λάθη
- **Ελαχιστοποίηση Αιτήσεων HTTP**: Χρησιμοποιήστε αποτελεσματικά κριτήρια αναζήτησης και συνδυάστε λειτουργίες όπου είναι δυνατό
- **Μαζική Επεξεργασία**: Χρησιμοποιήστε αποτελεσματικά τις μαζικές ενέργειες και τις ενέργειες Ιτερωνού
- **Διαχείριση Σφαλμάτων**: Εφαρμόστε σωστή διαχείριση σφαλμάτων για να αποτρέψετε περιττές επαναλαμβανόμενες προσπάθειες
- **Βελτιστοποίηση Χειροκίνητης Ενεργοποίησης**: Για χειροκίνητες ενεργοποιήσεις, επιλέξτε τη διαθεσιμότητα `Μαζική` για να επεξεργαστείτε πολλά αρχεία σε μία ροή εργασίας (δείτε τις [Λειτουργίες Ροής Εργασίας](/l/el/user-guide/workflows/workflow-features) για λεπτομέρειες)

### Βελτιστοποίηση Ενέργειας

- Χρησιμοποιήστε βασικές εσωτερικές λειτουργίες όταν είναι δυνατόν (μειωμένη κατανάλωση πιστώσεων)
- Βελτιστοποιήστε τις δράσεις κώδικα για αποτελεσματικότητα
- Σκεφτείτε την χρήση χειροκίνητων ενεργοποιήσεων για μη επείγοντα δεδομένα
- Εκτελέστε μαζικές λειτουργίες για τη μείωση των κλήσεων ανά ενέργεια

### Διαχείριση Πιστώσεων

- **Τακτικός Έλεγχος**: Ελέγξτε την κατανάλωση πιστώσεων εβδομαδιαίως στις Ρυθμίσεις → Χρέωση
- **Παρακολούθηση Χρήσης**: Κρατήστε στοιχεία για την κατανάλωση για να αποφύγετε απροσδόκητες υπερκαταναλώσεις
- **Σχεδιάστε Ανάλογα**: Σκεφτείτε την αναβάθμιση του σχεδίου σας αν χρειάζεστε περισσότερες πιστώσεις συχνά
- **Ανασκόπηση Ροής Εργασίας**: Ανασκοπήστε τακτικά τις ροές εργασίας με υψηλή κατανάλωση πιστώσεων και απενεργοποιήστε τις αχρησιμοποίητες
- **Στρατηγικός Σχεδιασμός**: Σχεδιάστε την ανάπτυξη ροών εργασίας με βάση τη διαθεσιμότητα πιστώσεων

Για λεπτομερείς πληροφορίες σχετικά με τη χρέωση και τη διαχείριση πιστώσεων, επισκεφθείτε την **Ρυθμίσεις → Χρέωση** στον χώρο εργασίας σας στο Twenty.
