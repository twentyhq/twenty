## Ερεθίσματα Ροής Εργασίας

Οι ροές εργασίας ξεκινούν πάντα με ένα μοναδικό ερέθισμα που καθορίζει πότε θα πρέπει να γίνει η αυτοματοποίηση.

### Record is Created

Ξεκινά τη ροή εργασίας όταν δημιουργείται μία νέα εγγραφή σε ένα επιλεγμένο αντικείμενο (Άνθρωποι, Εταιρείες, Ευκαιρίες ή οποιοδήποτε προσαρμοσμένο αντικείμενο).

**Διαμόρφωση**: Επιλέξτε τον τύπο αντικειμένου που θα παρακολουθείται για νέες εγγραφές.

### Η εγγραφή ενημερώθηκε

Ξεκινά τη ροή εργασίας όταν γίνονται αλλαγές σε μια υπάρχουσα εγγραφή.

**Διαμόρφωση**:

- Επιλέξτε τον τύπο αντικειμένου
- Προαιρετικά καθορίστε ποια πεδία να παρακολουθούνται για αλλαγές

### Η εγγραφή ενημερώθηκε ή δημιουργήθηκε

Ξεκινά τη ροή εργασίας όταν μια εγγραφή είτε δημιουργείται είτε ενημερώνεται σε ένα επιλεγμένο αντικείμενο.

**Γιατί έχει σημασία**: Αυτό το ερέθισμα είναι ιδιαίτερα χρήσιμο γιατί οι εγγραφές που δημιουργούνται με διαφορετικούς τρόπους συμπεριφέρονται διαφορετικά:

- **Εισαγωγές API/CSV**: Οι εγγραφές δημιουργούνται με όλα τα πεδία να έχουν συμπληρωθεί άμεσα
- **Χειροκίνητη δημιουργία**: Οι εγγραφές δημιουργούνται πρώτα, κι έπειτα προστίθενται πεδία σε επόμενες ενημερώσεις

**Διαμόρφωση**:

- Επιλέξτε τον τύπο αντικειμένου που θέλετε να παρακολουθήσετε
- Προαιρετικά καθορίστε ποια πεδία να παρακολουθούνται για αλλαγές
- Η ροή εργασίας θα ενεργοποιηθεί τόσο στην αρχική δημιουργία όσο και σε οποιεσδήποτε επακόλουθες ενημερώσεις

### Η εγγραφή διαγράφηκε

Ξεκινά τη ροή εργασίας όταν μια εγγραφή αφαιρείται από ένα αντικείμενο.

**Διαμόρφωση**: Επιλέξτε τον τύπο αντικειμένου που θα παρακολουθείται για διαγραφές.

### Εκκίνηση Χειροκίνητα

Ξεκινά τη ροή εργασίας όταν ενεργοποιηθεί από μια ενέργεια χρήστη. Αυτό το ερέθισμα μπορεί να προσπελαστεί μέσω του μενού Cmd+K ή μέσω ενός προσαρμοσμένου κουμπιού στην επάνω γραμμή εργαλείων.

**Διαμόρφωση Διαθεσιμότητας**:
Επιλέξτε πώς πρέπει η ροή εργασίας να χειρίζεται την επιλογή εγγραφών:

- **Καθολική**: Δεν απαιτείται εγγραφή για την ενεργοποίηση αυτής της ροής εργασίας. Η ροή εργασίας ενεργοποιείται από οπουδήποτε (από οποιοδήποτε αντικείμενο) και δεν χρησιμοποιεί καμία εγγραφή ως είσοδο.

- **Μοναδιαία**: Η επιλεγμένη εγγραφή/εγγραφές θα περαστεί στη ροή εργασίας σας. Αυτό διαμορφώνεται για ένα δεδομένο αντικείμενο. Πολλές εγγραφές μπορούν να επιλεγούν πριν την ενεργοποίηση της ροής εργασίας. Η ροή εργασίας θα εκτελεστεί τόσες φορές όσες είναι οι επιλεγμένες εγγραφές.

  <Warning>
  You cannot run more than 100 workflows in parallel at any given time.
  </Warning>

- **Μαζικά**: Η επιλεγμένη εγγραφή/εγγραφές θα περαστεί στη ροή εργασίας σας. Αυτό διαμορφώνεται για ένα δεδομένο αντικείμενο. Πολλές εγγραφές μπορούν να επιλεγούν πριν την ενεργοποίηση της ροής εργασίας. Η ροή εργασίας θα εκτελεστεί μια φορά, παρέχοντας όλη τη λίστα εγγραφών ως είσοδο. Αυτό σημαίνει ότι η ροή εργασίας πρέπει να περιέχει μια Ενέργεια Διαδοχέα. Αυτό είναι καλύτερο για άτομα που θέλουν να βελτιστοποιήσουν/περιορίσουν τον αριθμό εκτελέσεων ροής εργασιών.

**Πρόσθετη Διαμόρφωση**:

- Επιλέξτε το αντικείμενο στόχου (για μεμονωμένη και μαζική διαθεσιμότητα)
- Επιλέξτε ένα εικονίδιο εντολής για τον εκκινητή της ροής εργασίας
- Ρυθμίστε την τοποθέτηση του navbar (Καρφιτσωμένο ή Όχι Καρφιτσωμένο)

**Μέθοδοι Πρόσβασης**:

- Μενού Cmd+K για να βρείτε και να εκκινήσετε μη αυτόματες ροές εργασιών
- Προσαρμοσμένο κουμπί στη επάνω γραμμή πλοήγησης (εάν έχει ρυθμιστεί)

### Πρόγραμμα

Εκκινεί τη ροή εργασίας σε περιοδική βάση που ορίζετε.

**Διαμόρφωση**:

- Επιλέξτε μονάδα χρόνου (λεπτά, ώρες, ημέρες)
- Εισάγετε μια τιμή ή χρησιμοποιήστε προσαρμοσμένες εκφράσεις cron για προχωρημένο προγραμματισμό

### Webhook

Εκκινεί τη ροή εργασίας όταν ληφθεί αίτηση GET ή POST από εξωτερική υπηρεσία.

**Διαμόρφωση**:

- Λάβετε μια μοναδική διεύθυνση URL webhook
- Για αιτήσεις POST, ορίστε τη δομή σώματος που αναμένεται
- Ρυθμίστε την ταυτοποίηση, αν είναι απαραίτητο.

## Ενέργειες Ροής Εργασίας

Οι ενέργειες καθορίζουν τι συμβαίνει μετά την ενεργοποίηση ενός εκκινητή. Μπορείτε να συνδέσετε πολλές ενέργειες μαζί.

### Δημιουργία μιας Εγγραφής

Προσθέτει μια νέα εγγραφή σε ένα επιλεγμένο αντικείμενο.

**Διαμόρφωση**:

- Επιλέξτε το αντικείμενο στόχου
- Συμπληρώστε τα απαραίτητα και προαιρετικά πεδία
- Χρησιμοποιήστε δεδομένα από προηγούμενα βήματα για να συμπληρώσετε πεδία

**Έξοδος**: Τα νέα δεδομένα εγγραφής είναι διαθέσιμα για χρήση σε επόμενα βήματα.

### Ενημέρωση Εγγραφής

Τροποποιεί μια υπάρχουσα εγγραφή σε ένα επιλεγμένο αντικείμενο.

**Διαμόρφωση**:

- Επιλέξτε το αντικείμενο στόχου
- Επιλέξτε την συγκεκριμένη εγγραφή για ενημέρωση
- Επιλέξτε πεδία για τροποποίηση και εισάγετε νέες τιμές

**Έξοδος**: Τα ενημερωμένα δεδομένα εγγραφής είναι διαθέσιμα για χρήση σε επόμενα βήματα.

### Διαγραφή Εγγραφής

Αφαιρεί μια εγγραφή από ένα επιλεγμένο αντικείμενο.

**Διαμόρφωση**:

- Επιλέξτε το αντικείμενο στόχου
- Επιλέξτε την συγκεκριμένη εγγραφή για διαγραφή

**Έξοδος**: Τα διαγραμμένα δεδομένα εγγραφής παραμένουν διαθέσιμα για χρήση σε επόμενα βήματα.

### Αναζήτηση Εγγραφών

Βρίσκει εγγραφές μέσα σε ένα επιλεγμένο αντικείμενο χρησιμοποιώντας συνθήκες φίλτρου.

**Διαμόρφωση**:

- Επιλέξτε το αντικείμενο για αναζήτηση
- Ορίστε κριτήρια φίλτρου για να περιορίσετε τα αποτελέσματα
- Ρυθμίστε την ταξινόμηση και τα όρια

**Έξοδος**: Επιστρέφει τις εγγραφές που ταιριάζουν και μπορούν να χρησιμοποιηθούν σε επόμενα βήματα.

**Βέλτιστη Πρακτική**: Χρησιμοποιήστε κλάδους μετά την Αναζήτηση Εγγραφών για να χειριστείτε τα σενάρια "βρέθηκε" και "δεν βρέθηκε".

### Διαδοχέας

Περνάει μέσα από ένα πίνακα εγγραφών που επιστράφηκαν από ένα προηγούμενο βήμα, επιτρέποντάς σας να εκτελέσετε ενέργειες σε κάθε εγγραφή ξεχωριστά.

**Διαμόρφωση**:

- Επιλέξτε τον πίνακα εγγραφών από ένα προηγούμενο βήμα (π.χ., αποτελέσματα από Αναζήτηση Εγγραφών)
- Ορίστε τις ενέργειες που θα εκτελεστούν σε κάθε εγγραφή στον κύκλο
- Ρυθμίστε το όνομα της μεταβλητής για την αναφορά κάθε εγγραφής στην επανάληψη

**Παράδειγμα**: Η Αναζήτηση Εγγραφών επιστρέφει 5 άτομα, στη συνέχεια χρησιμοποιήστε τον Διαδοχέα για να στείλετε email σε κάθε άτομο ή να ενημερώσετε κάθε εγγραφή ξεχωριστά.

**Σημείωση**: Ο Διαδοχέας βρίσκεται επί του παρόντος σε beta. Ενεργοποιήστε το μέσω Ρυθμίσεων → Εκδόσεις → Εργαστήριο.

### Φίλτρο

Φιλτράρει έναν πίνακα εγγραφών βάσει καθορισμένων συνθηκών, επιτρέποντας μόνο στις εγγραφές που πληρούν τα κριτήρια να περάσουν.

**Διαμόρφωση**:

- Επιλέξτε τον πίνακα εγγραφών για φιλτράρισμα
- Ορίστε τις συνθήκες φίλτρου και τα κριτήρια
- Ρυθμίστε ποιες εγγραφές πρέπει να περάσουν στα επόμενα βήματα

**Έξοδος**: Επιστρέφει μόνο τις εγγραφές που ταιριάζουν με τις καθορισμένες συνθήκες φίλτρου.

### Αποστολή Email

Στέλνει ένα email από τη ροή εργασίας σας.

**Προαπαιτούμενα**: Προσθέστε έναν λογαριασμό email στις Ρυθμίσεις > Λογαριασμοί

**Διαμόρφωση**:

- Εισάγετε τη διεύθυνση email του παραλήπτη
- Ορίστε τίτλο θέματος
- Συνθέστε το σώμα του μηνύματος
- Αναφέρετε μεταβλητές από προηγούμενα βήματα για εξατομίκευση

**Σημείωση**: Τα συνημμένα email θα είναι διαθέσιμα το πρώτο τρίμηνο του 2026.

### Κωδικός

Εκτελεί εξατομικευμένη JavaScript στη ροή εργασίας σας.

**Διαμόρφωση**:

- Γράψτε κώδικα JavaScript στον επεξεργαστή
- Πρόσβαση στις μεταβλητές από προηγούμενα βήματα
- Επιστροφή μεταβλητών για χρήση στα επόμενα βήματα
- Δοκιμάστε τον κώδικα άμεσα στο βήμα

**Πρόσβαση**: Διαχειριστείτε τα κλειδιά API στις Ρυθμίσεις → API & Webhooks

### Φόρμα

Παρουσιάζει μια φόρμα κατά την εκτέλεση της ροής εργασίας για τη συλλογή εισόδου από τον χρήστη.

**Διαμόρφωση**:

- Ορίστε πεδία εισόδου με τύπους, ετικέτες και προκαθορισμένα κείμενα
- Ρυθμίστε κανόνες επικύρωσης
- Ορίστε τίτλο και περιγραφή φόρμας

**Έξοδος**: Οι απαντήσεις της φόρμας είναι διαθέσιμες για χρήση στα επόμενα βήματα.

<Warning>
Οι φόρμες έχουν σχεδιαστεί επί του παρόντος μόνο για μη αυτόματους ενεργοποιητές. Για ροές εργασιών με άλλους ενεργοποιητές (Δημιουργία Εγγραφής, Ενημέρωση κ.λπ.), οι φόρμες είναι προσβάσιμες μόνο μέσω της διεπαφής εκτέλεσης ροής εργασίας, κάτι που δεν είναι η αναμενόμενη εμπειρία χρήστη. Μια πλατφόρμα ειδοποιήσεων θα κυκλοφορήσει το 2026 για την υποστήριξη φορμών σε αυτοματοποιημένες ροές εργασιών.
</Warning>

### Αίτηση HTTP

Στέλνει μια αίτηση σε μια εξωτερική API ως μέρος της ροής εργασίας σας.

**Διαμόρφωση**:

- Εισάγετε το URL του τερματικού σημείου API
- Επιλέξτε μέθοδο HTTP (GET, POST, PUT, PATCH, DELETE)
- Add required headers and values
- Συμπεριλάβετε το σώμα αίτησης για αιτήσεις POST/PUT/PATCH
- Παρέχετε παράδειγμα απάντησης για προεπισκόπηση δομής

## Διαχείριση Ροής Εργασίας

### Δημιουργία Ροών Εργασίας

1. Κάντε κλικ στο "+ Προσθήκη Ροής Εργασίας" για να ξεκινήσετε
2. Κάντε κλικ στο "Χωρίς Τίτλο" για να ονομάσετε τη ροή εργασίας σας
3. Επιλέξτε και διαμορφώστε τον εκκινητή της ροής εργασίας σας
4. Προσθέστε ενέργειες στη ροή εργασίας σας
5. Δοκιμάστε και βελτιώστε
6. Ενεργοποιήστε τη ροή εργασίας σας (βρίσκεται σε προσχέδιο) μόλις ολοκληρώσετε την επεξεργασία της

**Σημείωση**: Εάν δεν βλέπετε την ενότητα Ροές Εργασιών, αυτό οφείλεται σε πρόβλημα αδειών. Επικοινωνήστε με τον διαχειριστή της χώρου εργασίας σας για να σας παρέχει πρόσβαση στις ροές εργασιών.

### Κατάσταση Ροής Εργασίας

- **Προσχέδιο**: Υπό επεξεργασία, δεν έχει ακόμη δημοσιευτεί
- **Ενεργή**: Ζωντανή έκδοση που ανταποκρίνεται στους εκκινητές
- **Απενεργοποιημένη**: Προηγουμένως ενεργή αλλά έχει σταματήσει χειροκίνητα
- **Αρχειοθετημένη**: Παρελθούσες εκδόσεις διατηρούνται για ιστορικό

### Ενεργοποίηση Ροών Εργασίας

Κάντε κλικ στο **Ενεργοποίηση** για να δημοσιεύσετε το προσχέδιο σας ως νέα έκδοση. Αυτό κάνει τη ροή εργασίας επιλέξιμη για εκτέλεση όταν ενεργοποιείται, αλλά δεν την εκτελεί αμέσως.

### Δοκιμή Ροών Εργασίας

Δοκιμάστε ροές εργασιών πριν την ενεργοποίηση χρησιμοποιώντας:

- Μη αυτόματοι εκκινητές (όταν δεν έχει επιλεγεί καμία εγγραφή)
- Δοκιμή μεμονωμένων ενεργειών (ειδικά Ενέργειες Κώδικα)
- Δοκιμή λειτουργίας προσχεδίου που δεν ενεργοποιεί τη ροή εργασίας

### Εκτελέσεις Ροής Εργασίας

Μια **Εκτέλεση** είναι μια καταγραφή εκτέλεσης ροής εργασίας που περιέχει:

- Κατάσταση (επιτυχής, απέτυχε, σε εκτέλεση)
- Δεδομένα εξόδου από κάθε βήμα
- Συγγραφέας και χρονοσφραγίδες
- Μηνύματα σφάλματος αν υπάρχουν

**Παρακολούθηση Εκτελέσεων**:

- Ελέγξτε τον **Πίνακα Εκτελέσεων** στον επεξεργαστή ροής εργασίας
- Ανοίξτε την προβολή **Εκτελέσεις Ροής Εργασίας** για παρακολούθηση σε όλες τις ροές εργασιών

**Συμβουλή Απόδοσης**: Κρύψτε τις εκτελέσεις ροής εργασίας από τη σελίδα "Όλες οι ροές εργασίας" και άλλες σελίδες ροής εργασίας για τη βελτίωση της απόδοσης φόρτωσης, καθώς μεγάλος αριθμός εκτελέσεων μπορεί να καθυστερήσει τη φόρτωση της σελίδας.

### Ιστορικό Εκδόσεων

- Ελέγξτε όλες τις εκδόσεις στο πεδίο **Εκδόσεις**
- Κάντε κλικ σε οποιαδήποτε έκδοση για να δείτε λεπτομέρειες
- Χρησιμοποιήστε το **Χρήση ως προσχέδιο** για να επαναφέρετε προηγούμενες εκδόσεις
- Διαχειριστείτε τις συγκρούσεις προσχεδίου με τις επιλογές υπερισχύσεων ή επιστροφής

## Καλύτερες Πρακτικές

### Οργάνωση Ροής Εργασίας

- **Περιγραφικά Ονόματα**: Χρησιμοποιήστε σαφή, συγκεκριμένα ονόματα ροών εργασίας
- **Ονομασία Βημάτων**: Επαναονομάστε βήματα για να περιγράψετε τη λειτουργία τους
- **Τεκμηρίωση**: Προσθέστε σχόλια σε Ενέργειες Κώδικα
- **Κατηγοριοποίηση**: Ομαδοποιήστε συναφείς ροές εργασιών λογικά
- **Προσαρμοσμένα Πεδία**: Προσθέστε πεδία στο αντικείμενο Ροής Εργασίας στο μοντέλο δεδομένων σας (όπως και σε άλλα αντικείμενα) για να οργανώσετε και να κατηγοριοποιήσετε τις ροές εργασιών με προσαρμοσμένες ιδιότητες

### Βελτιστοποίηση Απόδοσης

- **Ελαχιστοποίηση Κλήσεων API**: Συγκεντρώστε λειτουργίες όταν είναι δυνατό
- **Αποδοτικές Αναζητήσεις**: Χρησιμοποιήστε συγκεκριμένα κριτήρια φίλτρου
- **Διαχείριση Σφαλμάτων**: Σχεδιάστε για σενάρια αποτυχίας
- **Περιορισμός Ρυθμών**: Σεβαστείτε τα όρια εξωτερικών API

### Διαχείριση Ροής Δεδομένων

- **Λογική Κλαδιών**: Χρησιμοποιήστε κλάδους μετά την Αναζήτηση Εγγραφών
- **Χρήση Μεταβλητών**: Αξιοποιήστε δεδομένα από προηγούμενα βήματα
- **Επικύρωση Δεδομένων**: Επικυρώστε τις εισόδους σε ενέργειες κώδικα
- **Χαρτογράφηση Πεδίων**: Σχεδιάστε μετασχηματισμούς δεδομένων προσεκτικά

### Παρακολούθηση και Συντήρηση

- **Τακτική Παρακολούθηση**: Ελέγξτε τις εκτελέσεις ροής εργασίας για σφάλματα
- **Ανασκόπηση Απόδοσης**: Αναλύστε τους χρόνους εκτέλεσης και τα ποσοστά επιτυχίας
- **Διαχείριση Ενημέρωσης**: Δοκιμάστε αλλαγές στο προσχέδιο πριν την ενεργοποίηση
- **Συντονισμός Ομάδας**: Τεκμηριώστε τις ροές εργασίας για τα μέλη της ομάδας

Για πρακτικά παραδείγματα αυτών των χαρακτηριστικών σε δράση, δείτε τους οδηγούς μας για [Εσωτερικές Αυτοματοποιήσεις](/l/el/user-guide/workflows/internal-automations) και [Εξωτερική Ενσωμάτωση Εργαλείων](/l/el/user-guide/workflows/external-tool-integration).
