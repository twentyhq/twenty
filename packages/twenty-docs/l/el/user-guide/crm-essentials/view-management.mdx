## Επιλογές Διάταξης

Μπορείτε να εμφανίσετε τα δεδομένα σας σε τρεις διαφορετικές διατάξεις, καθεμία προσαρμοσμένη για διαφορετικούς σκοπούς. Προσαρμοσμένες διατάξεις για να προσαρμόσετε την εμφάνιση της σελίδας για κάθε τύπο καταγραφής θα είναι διαθέσιμες τον Δεκέμβριο του 2025.

### Προεπιλεγμένη Προβολή

Κάθε αντικείμενο έρχεται με μια μη φιλτραρισμένη, μη ταξινομημένη και μη διαγράψιμη προβολή, γνωστή ως Προεπιλεγμένη προβολή. Είναι ονοματισμένο από την πληθυντική ονομασία του αντικειμένου, όπως `Όλες οι Εταιρίες`, `Όλοι οι Άνθρωποι`, `Όλες οι Ευκαιρίες`.
<img src="/images/user-guide/views/default-view.png" style={{width:'100%'}}/>

### Διάταξη Λίστας

Η τυπική μορφή πίνακα που εμφανίζει εγγραφές σε σειρές και στήλες. Αυτό είναι ιδανικό για να βλέπετε λεπτομερείς πληροφορίες με μια ματιά και να συγκρίνετε τις εγγραφές δίπλα-δίπλα.

### Ομαδοποιημένη Διάταξη Λίστας

Οργανώνει τις εγγραφές σας ομαδοποιώντας τις βάσει ενός επιλεγμένου πεδίου. Για παράδειγμα, μπορείτε να ομαδοποιήσετε ευκαιρίες ανά στάδιο, εταιρίες ανά τοποθεσία ή οποιοδήποτε άλλο επιλεγμένο πεδίο. Αυτό σας βοηθά να βλέπετε πρότυπα και να οργανώνετε συναφείς εγγραφές μαζί.

### Διάταξη Kanban

Ένας οπτικός πίνακας όπου κάθε στήλη αντιπροσωπεύει ένα στάδιο και κάθε εγγραφή εμφανίζεται ως κάρτα. Αυτή η διάταξη είναι ιδανική για τη διαχείριση παραγωγών και ροών εργασίας όπου οι εγγραφές περνούν από διαφορετικά στάδια. Για περισσότερες λεπτομέρειες σχετικά με τη χρήση προβολών Kanban για διαχείριση ροής, ανατρέξτε στο άρθρο [Pipeline](/l/el/user-guide/crm-essentials/pipeline).

## Δημιουργία Νέων Προβολών

There are three ways to create a new view:

### Χρησιμοποιώντας το Μενού Εντολών

Χρησιμοποιήστε `Cmd+K` σε Mac (ή `Ctrl+K` σε Windows) για να ανοίξετε το μενού εντολών, στη συνέχεια κάντε κλικ στο `Δημιουργία νέας προβολής`.

### Χρησιμοποιώντας το Μενού Προβολών

Κάντε κλικ στο μενού προβολής (πάνω αριστερά), στη συνέχεια κάντε κλικ στο `Προσθήκη Προβολής` στο κάτω μέρος. Από αυτό το μενού μπορείτε:

- Να επιλέξετε ένα εικονίδιο και όνομα για τη προβολή σας
- Να επιλέξετε τον τύπο διάταξης (Λίστα ή Kanban)

<Warning>
Σημαντικό: Πρώτα πρέπει να επιλέξετε τη διάταξη Λίστα και στη συνέχεια να προσθέσετε μια Ομαδοποίηση Από. Δεν μπορείτε να δημιουργήσετε απευθείας μια Ομαδοποιημένη Διάταξη Λίστας από εκεί.
</Warning>
- For Kanban views, select which select field to use as column headers

<VimeoEmbed videoId="927639721" title="Video demonstration" />

### Δημιουργία Προβολών από Υπάρχοντα Φίλτρα

Όταν τροποιείτε την ταξινόμηση και το φίλτρο μιας υπάρχουσας προβολής, εμφανίζεται ένα κουμπί `Αποθήκευση ως νέα προβολή`. Αυτό σας επιτρέπει να δημιουργήσετε μια νέα προβολή βάσει των τρεχουσών προσαρμογών σας.

## Κάνοντας τις Προβολές Ενεργές

Η καθοδήγηση παρακάτω σας δείχνει πώς να προσαρμόσετε τις στήλες των προβολών σας. Δεν συνιστούμε να κρατήσετε όλες τις στήλες: αυτές οι προβολές μπορούν να απλοποιηθούν και να γίνουν ενεργές προσθέτοντας συνθήκες ταξινόμησης και εμφανίζοντας μόνο τις στήλες που είναι σχετικές με τη συγκεκριμένη χρήση σας.

## Εξατομίκευση των Προβολών σας

Όλες οι διατάξεις υποστηρίζουν τις ίδιες επιλογές εξατομίκευσης: ταξινόμηση, φιλτράρισμα και επιλογή πεδίων. Μπορείτε να κάνετε γρήγορες, μεμονωμένες αλλαγές κάνοντας κλικ απευθείας στο όνομα της στήλης, ή να χρησιμοποιήσετε το `Μενού Επιλογών` για πιο ολοκληρωμένες επεξεργασίες.

### Γρήγορες Ενέργειες vs Μενού Επιλογών

**Για μεμονωμένες αλλαγές**: Κάντε κλικ απευθείας στις επικεφαλίδες των στηλών για να ταξινομήσετε, ή χρησιμοποιήστε τα κουμπιά `Μετακίνηση Αριστερά`, `Μετακίνηση Δεξιά`.

**Για πολλαπλές επεξεργασίες**: Χρησιμοποιήστε το `Μενού Επιλογών` (πάνω δεξιά γωνία) όταν θέλετε να κάνετε πολλαπλές συνεχειές αλλαγές. Αυτό το μενού σας δίνει πρόσβαση σε:

- **Επιλογή διάταξης** (Λίστα, Ομαδοποιημένη Λίστα, Kanban)
- **Επιλογές Ομαδοποίησης** (για διάταξη Ομαδοποιημένης Λίστας)
- **Διαχείριση πεδίων** (εμφάνιση/απόκρυψη και αναδιάταξη στηλών)

### Φιλτράρισμα των Δεδομένων σας

Μπορείτε να εφαρμόσετε φίλτρα για να εμφανίσετε μόνο τις εγγραφές που ταιριάζουν με τα κριτήρια σας. Κάντε κλικ στο `Φίλτρο` στη γραμμή εργαλείων, επιλέξτε ένα πεδίο, επιλέξτε την κατάστασή σας και ορίστε την τιμή. Μπορείτε να προσθέσετε πολλαπλά φίλτρα για προηγμένο φιλτράρισμα βάσει πολλών συνθηκών.

<VimeoEmbed videoId="926282262" title="Video demonstration" />

### Ταξινόμηση των Εγγραφών σας

Ελέγξτε τη σειρά των εγγραφών σας κάνοντας κλικ σε οποιοδήποτε όνομα στήλης για να ταξινομήσετε με βάση αυτό το πεδίο. Κάντε κλικ ξανά για να αντιστρέψετε τη σειρά ταξινόμησης. Μπορείτε να εφαρμόσετε πολλαπλές ταξινομήσεις για σύνθετη οργάνωση.

<VimeoEmbed videoId="927885588" title="Video demonstration" />

### Διαχείριση Πεδίων και Στηλών

Μπορείτε να επιλέξετε ποια πεδία να εμφανίζονται και να τα αναδιατάξετε. Για γρήγορες αλλαγές, κάντε κλικ σε ένα όνομα στήλης απευθείας. Για πολλαπλές επεξεργασίες, χρησιμοποιήστε το `Μενού Επιλογών → Πεδίων`, όπου μπορείτε:

- Να εμφανίσετε ή να αποκρύψετε πεδία χρησιμοποιώντας το εικονίδιο ματιού
- Να αναδιατάξετε πεδία σύροντας και αφήνοντας
- Να κάνετε πολλαπλές αλλαγές αποδοτικά σε ένα μέρος

## Διαχείριση των Προβολών σας

### Χαρακτηριστικά Μενού Προβολών

Το μενού προβολών (πάνω αριστερά) είναι η κεντρική σας γέφυρα για τη διαχείριση των προβολών. Από εδώ μπορείτε:

- **Επεξεργασία ονομάτων και εικονιδίων προβολών**: Κάντε κλικ στις τρεις τελείες δίπλα σε οποιαδήποτε προβολή
- **Αναδιάταξη προβολών**: Σύρετε και αφήστε προβολές για να τις οργανώσετε κατά προτεραιότητα
- **Αποθήκευση προβολών ως αγαπημένα**: Τα αγαπημένα εμφανίζονται ακριβώς κάτω από τις Ρυθμίσεις για γρήγορη πρόσβαση

### Επεξεργασία και Διαγραφή Προβολών

Για να τροποποιήσετε ή να αφαιρέσετε προβολές, ανοίξτε το μενού προβολών και τοποθετήστε το δρομέα πάνω από την προβολή που θέλετε να τροποποιήσετε. Κάντε κλικ στις τρεις τελείες που εμφανίζονται για να αποκτήσετε πρόσβαση στις επιλογές επεξεργασίας και διαγραφής.

### Αγαπημένα και Οργάνωση

Οι προβολές αποθηκευμένες ως αγαπημένα εμφανίζονται ακριβώς κάτω από τις Ρυθμίσεις, παρέχοντας σας άμεση πρόσβαση στις πιο σημαντικές προβολές σας. Χρησιμοποιήστε το μενού προβολών για να οργανώσετε τις προβολές σας, σύροντάς τις στη σειρά που λειτουργεί καλύτερα για τη ροή εργασίας σας.

Για πιο προηγμένη οργάνωση δεδομένων, ανατρέξτε στην ενότητα [Data Model](/l/el/user-guide/data-model/customize-your-data-model) για να μάθετε πώς να προσαρμόζετε πεδία και αντικείμενα.
