## Εισαγωγή δεδομένων στο CRM σας

Όταν ξεκινάτε να χρησιμοποιείτε το Twenty, θα θέλετε να εισαγάγετε τις επαφές και τις εταιρείες σας στο σύστημα. Υπάρχουν πολλοί τρόποι για να ενημερώσετε το CRM σας, ανάλογα με τη ροή εργασίας και τις πηγές δεδομένων σας.

### Χειροκίνητη καταχώρηση

Η πιο απλοϊκή προσέγγιση είναι η προσθήκη αρχείων απευθείας μέσω του περιβάλλοντος του Twenty. Πηγαίνετε στην ενότητα `Πρόσωπα` και κάντε κλικ στο κουμπί `+` για να προσθέσετε μια νέα επαφή. Συμπληρώστε το όνομά τους, το email, το τηλέφωνο και συσχετίστε το με την εταιρεία τους. For companies, head to the `Companies` section and add the organization details: company name, domain, industry, and size.

Το πεδίο τομέα είναι ιδιαίτερα σημαντικό για την αναγνώριση εταιρειών και το πεδίο email είναι ουσιαστικό για την αναγνώριση προσώπων.

### Εισαγωγές CSV

Όταν έχετε υπάρχοντα δεδομένα από φύλλα υπολογισμού ή άλλα συστήματα, η εισαγωγή CSV είναι η πιο γρήγορη επιλογή σας. Μπορείτε να προετοιμάσετε τα δεδομένα σας στο Excel ή το Google Sheets και στη συνέχεια να τα ανεβάσετε όλα μαζί. This is particularly useful when migrating from another CRM or when someone has been tracking contacts in spreadsheets. Ο [Οδηγός Εισαγωγής/Εξαγωγής Δεδομένων](/l/el/user-guide/getting-started/import-export-data) σας καθοδηγεί στη διαδικασία.

### Αυτοματοποιημένη λήψη δεδομένων

Για συνεχή δημιουργία υποψηφίων, μπορείτε να ρυθμίσετε αυτοματοποιημένες εργασιακές ροές που φέρνουν δεδομένα απευθείας στο Twenty:

**Φόρμες ιστότοπου**: Όταν κάποιος συμπληρώνει μια φόρμα στον ιστότοπό σας, μπορείτε να τη ρυθμίσετε να αποστέλλει τις πληροφορίες αυτόματα στο Twenty. Η υποβολή της φόρμας ενεργοποιεί ένα webhook που ενεργοποιεί μια ροή εργασίας στο Twenty, δημιουργώντας νέο αρχείο επαφής χωρίς καμία χειροκίνητη εργασία.

**Ενοποίηση με άλλα συστήματα**: Εάν χρησιμοποιείτε άλλα εργαλεία επιχείρησης, μπορείτε να τα συνδέσετε στο Twenty χρησιμοποιώντας κλήσεις API και ροές εργασίας. Αυτό σας επιτρέπει να συγχρονίζετε αυτόματα δεδομένα μεταξύ των συστημάτων: για παράδειγμα, εισαγωγή νέων πελατών από το σύστημα τιμολόγησης ή υποψηφίων από την πλατφόρμα μάρκετινγκ σας.

Για να μάθετε περισσότερα για τη ρύθμιση αυτών των αυτοματοποιημένων ροών δεδομένων, ελέγξτε το τμήμα μας [Ροές Εργασίας](/l/el/user-guide/workflows/getting-started-workflows).

### Συγχρονισμός email και ημερολογίου

Όταν συνδέετε το γραμματοκιβώτιο και το ημερολόγιό σας στο Twenty, το σύστημα μπορεί να δημιουργήσει αυτόματα αρχεία Προσώπων και Εταιρειών για άτομα που στέλνετε email ή συναντάτε. Εάν στείλετε email σε κάποιον που δεν είναι ήδη στο CRM σας, το Twenty μπορεί να δημιουργήσει ένα νέο αρχείο Προσώπου για αυτόν. Το ίδιο συμβαίνει όταν προγραμματίζετε συναντήσεις με νέες επαφές μέσω του ημερολογίου σας.

Αυτό είναι ιδιαίτερα χρήσιμο για ομάδες πωλήσεων και ανάπτυξης επιχειρήσεων που συνεχώς συναντούν νέους ανθρώπους. Αντί να προσθέτετε κάθε φορά μια νέα επαφή με το χέρι, το Twenty τις καταγράφει αυτόματα καθώς επικοινωνείτε. Μάθετε πώς να το ρυθμίσετε στο οδηγό μας [Emails και Ημερολόγια](/l/el/user-guide/collaboration/emails-and-calendars).

### Μείωση χειροκίνητης εργασίας

Ακόμα και όταν προσθέτετε δεδομένα χειροκίνητα, μπορείτε να χρησιμοποιήσετε ροές εργασίας για να βελτιώσετε τη διαδικασία. Για παράδειγμα, μπορείτε να ρυθμίσετε αυτοματοποίηση που εκχωρεί νέες επαφές στα μέλη της ομάδας βασισμένα στην τοποθεσία τους, ή που δημιουργεί αυτόματα εργασίες παρακολούθησης όταν προστίθενται συγκεκριμένοι τύποι επαφών.

## Οργάνωση των επαφών σας

### Διατήρηση μοναδικών και καθαρών δεδομένων

Το Twenty επιβάλλει αυτόματα την μοναδικότητα για να διατηρεί τα δεδομένα σας οργανωμένα. Η διεύθυνση email κάθε προσώπου λειτουργεί ως μοναδικός αναγνωριστικός: δεν μπορείτε να έχετε δύο άτομα με την ίδια διεύθυνση email. Παρομοίως, οι τομείς εταιρειών είναι μοναδικοί, έτσι δεν θα δημιουργήσετε κατά λάθος διπλότυπες εταιρείες.

Εάν η επιχείρησή σας χρειάζεται άλλα πεδία να είναι μοναδικά (όπως αριθμούς τηλεφώνου, ή κωδικούς αναφοράς), μπορείτε να το ρυθμίσετε στο μοντέλο δεδομένων σας. Μεταβείτε στην ενότητα μας [Μοντέλο Δεδομένων](/l/el/user-guide/data-model/customize-your-data-model) για να μάθετε πώς να ρυθμίσετε πρόσθετους περιορισμούς μοναδικότητας για τις συγκεκριμένες ανάγκες σας.

### Διαχείριση διπλότυπων

Μερικές φορές θα καταλήξετε με διπλότυπα αρχεία. Το Twenty έχει μια λειτουργία συγχώνευσης τόσο για Πρόσωπα όσο και για Εταιρείες: μπορείτε να συνδυάσετε διπλότυπα αρχεία για να διατηρήσετε τη βάση δεδομένων σας καθαρή χωρίς να χάσετε καμία πληροφορία.

Για να συγχωνεύσετε αρχεία, επιλέξτε 2 αρχεία, ανοίξτε το μενού εντολών `Cmd+K` σε Mac, `Ctrl+K` στα Windows και κάντε κλικ στο `Συγχώνευση Αρχείων`.

### Δημιουργία ιεραρχιών εταιρειών

Εάν εργάζεστε με μεγάλους οργανισμούς που έχουν θυγατρικές ή πολλαπλά τμήματα, μπορείτε να δημιουργήσετε σχέσεις μεταξύ εταιρειών. Ρυθμίστε πεδία σχέσεων μεταξύ των αρχείων Εταιρειών για να χαρτογραφήσετε αυτές τις συνδέσεις. Αυτό σας βοηθά να κατανοήσετε τη συνολική οργανωτική δομή με την οποία ασχολείστε.

### Προσαρμογή των προβολών σας

Διαφορετικά μέλη της ομάδας μπορεί να χρειαστεί να δουν διαφορετικές πληροφορίες. Μπορείτε να δημιουργήσετε προσαρμοσμένες προβολές που δείχνουν διαφορετικές στήλες για διαφορετικούς σκοπούς: ίσως η ομάδα πωλήσεων σας χρειάζεται να βλέπει στάδια συμφωνιών, ενώ η ομάδα υποστήριξης σας εστιάζει σε λεπτομέρειες επαφής. Μάθετε περισσότερα για αυτό στο άρθρο μας [Διαχείριση Προβολών](/l/el/user-guide/crm-essentials/view-management).

## Δουλεύοντας με αρχεία

### Τι θα βρείτε σε κάθε αρχείο

Όταν ανοίγετε ένα αρχείο Προσώπου ή Εταιρείας, θα δείτε όλες τις πληροφορίες τους οργανωμένες σε καρτέλες:

- **Πεδία**: Οι βασικές πληροφορίες όπως όνομα, email, τηλέφωνο, και οποιαδήποτε προσαρμοσμένα πεδία έχετε προσθέσει
- **Σχέσεις**: Δείχνει τις συνδέσεις μεταξύ αυτού του αρχείου και αρχείων από άλλα αντικείμενα
- **Χρονολόγιο**: Μία χρονική σειρά όλων των αλληλεπιδράσεων και ενημερώσεων σε αυτό το αρχείο
- **Εργασίες**: Οποιαδήποτε εργασία παρακολούθησης σχετίζεται με αυτή την επαφή
- **Σημειώσεις**: Σημειώσεις ομάδας και παρατηρήσεις σχετικά με αυτό το πρόσωπο ή την εταιρεία
- **Αρχεία**: Έγγραφα και συνημμένα που σχετίζονται με αυτό το αρχείο
- **Emails**: Τα email νήματα με αυτή την επαφή (όταν η ομάδα σας έχει συνδέσει τα γραμματοκιβώτιά της)
- **Ημερολόγιο**: Συναντήσεις και ραντεβού με αυτή την επαφή

The Email and Calendar tabs are particularly powerful: they automatically show all email exchanges and meetings that anyone on your team has had with this contact, as long as they've connected their mailbox to Twenty. Μπορείτε να μάθετε περισσότερα σχετικά με τη ρύθμιση αυτού στο οδηγό μας [Emails και Ημερολόγια](/l/el/user-guide/collaboration/emails-and-calendars).

### Προσθήκη των πεδίων που χρειάζεστε

Τα στάνταρ πεδία μπορεί να μην καταγράφουν όλα τα σημαντικά για την επιχείρησή σας. Αν χρειάζεστε επιπλέον πληροφορίες: όπως τμήματα πελατών, πηγές παραπομπής ή δεδομένα συγκεκριμένα για τη βιομηχανία: μπορείτε να προσθέσετε προσαρμοσμένα πεδία ή να τροποποιήσετε υπάρχοντα. Μεταβείτε στην ενότητα μας [Μοντέλο Δεδομένων](/l/el/user-guide/data-model/customize-your-data-model) για να μάθετε πώς να προσαρμόσετε τις ρυθμίσεις σας.

## Διαχείριση διαγραμμένων εγγραφών

Όταν διαγράφετε μια εγγραφή στο Twenty, δεν χάνεται για πάντα. Οι εγγραφές είναι "μαλακά διαγραμμένες", που σημαίνει ότι είναι κρυφές αλλά μπορούν να επαναφερθούν αν χρειαστεί.

Για να αποκτήσετε πρόσβαση σε διαγραμμένες εγγραφές, ανοίξτε το μενού εντολών `Cmd+K` στο Mac, `Ctrl+K` στα Windows και κάντε κλικ στο `Δείτε τις διαγραμμένες εγγραφές`. Από εκεί, μπορείτε να επαναφέρετε εγγραφές ή να τις διαγράψετε οριστικά αν είστε σίγουροι ότι δεν τις χρειάζεστε.

Αυτό το δίχτυ ασφάλειας σημαίνει ότι μπορείτε να καθαρίσετε τη βάση δεδομένων σας χωρίς να ανησυχείτε για την απώλεια σημαντικών πληροφοριών.
