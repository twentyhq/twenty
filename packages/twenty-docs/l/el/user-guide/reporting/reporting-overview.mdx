## Τι Έρχεται

### Προσαρμοσμένες Αναφορές

Δημιουργήστε προσαρμοσμένες αναφορές για να παρακολουθείτε τους δείκτες που έχουν μεγαλύτερη σημασία για την επιχείρησή σας.

### Πίνακες Πωλήσεων

Οπτικοποιήστε την απόδοση της γραμμής πωλήσεών σας με διαδραστικούς πίνακες.

### Αναλυτικά Στοιχεία Απόδοσης

Παρακολουθήστε την απόδοση της ομάδας και εντοπίστε τάσεις στα δεδομένα των πελατών σας.

## Μείνετε Ενημερωμένοι

Ακολουθήστε το [αποθετήριο GitHub](https://github.com/twentyhq/twenty) μας ή ελέγξτε την ενότητα **Ρυθμίσεις → Εκδόσεις** στον χώρο εργασίας σας για να λάβετε ενημέρωση όταν οι λειτουργίες αναφοράς γίνονται διαθέσιμες.
