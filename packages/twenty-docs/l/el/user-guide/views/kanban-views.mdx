## Σχετικά με τις Προβολές Kanban

Οι προβολές Kanban απεικονίζουν ροές διεργασιών, όπου κάθε στήλη αντιστοιχεί σε μια ξεχωριστή φάση και κάθε κάρτα αντιπροσωπεύει μια εγγραφή.

## Μετακίνηση Καρτών μεταξύ Φάσεων

Μπορείτε να μετακινήσετε κάθε κάρτα μεταξύ των φάσεων καθώς περνάει η ροή εργασίας σας σύροντας και αποθέτοντάς την. Για να προχωρήσετε, κρατήστε το κλικ σας πάνω σε μια κάρτα και μετακινήστε την στην επόμενη φάση.

## Προσθήκη και Διαγραφή Φάσεων

Μπορείτε να προσαρμόσετε τη ροή εργασίας σας για να καλύψετε τις ανάγκες σας χρησιμοποιώντας φάσεις, οι οποίες αντιπροσωπεύουν μια τιμή σε ένα Πεδία Επιλογής:

### Προσθήκη Φάσεων

Για να προσθέσετε μια φάση, αποκτήστε πρόσβαση στις ρυθμίσεις πεδίου Επιλογής πηγαίνοντας στις Ρυθμίσεις > Μοντέλο Δεδομένων, επιλέγοντας το αντικείμενό σας και μετά το πεδίο στο οποίο βασίζεται το Kanban board σας.

<VimeoEmbed videoId="927890428" title="Video demonstration" />

### Διαγραφή Φάσεων

Για να αφαιρέσετε μια φάση, τοποθετήστε το ποντίκι στο όνομα της φάσης ή στο εικονίδιο `⋮`, κάντε κλικ στο `Επεξεργασία από τις ρυθμίσεις` στις ρυθμίσεις πεδίου Επιλογής και κατόπιν κάντε κλικ στο <b>Διαγραφή</b> δίπλα στη σχετική φάση.

## Προβολή πεδίων

Μπορείτε να προσαρμόσετε το Kanban board σας ώστε να εμφανίζονται κάποια πεδία και να αποκρύπτονται άλλα. Για να αποκρύψετε ένα πεδίο, κάντε κλικ στις <b>Επιλογές</b> πάνω δεξιά, στη συνέχεια στις <b>Πεδία</b> για να εμφανιστεί η λίστα επιλογών. Τοποθετήστε το ποντίκι στο πεδίο που θέλετε να αποκρύψετε για να εμφανιστεί το πλήκτρο `-`. Κάντε κλικ στο πλήκτρο αυτό για να αποκρύψετε το πεδίο.

Μπορείτε επίσης να αλλάξετε τη σειρά των πεδίων κρατώντας πατημένο το όνομα του πεδίου και τοποθετώντας το όπου θέλετε.

<img src="/images/user-guide/kanban-views/filter.png" style={{width:'100%'}}/>

## Συμπαγής Προβολή

Μπορείτε επίσης να αποκρύψετε όλα τα πεδία και να έχετε μια συνοπτική εικόνα όλων των ευκαιριών με μια ματιά. Για να το κάνετε, κάντε κλικ στις <b>Επιλογές</b> πάνω δεξιά και ενεργοποιήστε την επιλογή <b>Συμπαγής προβολή</b> αφού επιλέξετε διάταξη στην προβολή Kanban.
