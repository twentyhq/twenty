## Σχετικά με τις Προβολές

Μπορείτε να βλέπετε τις εγγραφές σας με διάφορους τρόπους δημιουργώντας προβολές. Σε μια προβολή, μπορείτε να εφαρμόσετε φίλτρα και ταξινομήσεις για να οργανώσετε το περιεχόμενό σας αποτελεσματικά. Για παράδειγμα, στο αντικείμενο `Άτομα`, μπορείτε να δημιουργήσετε μια προβολή για να απομονώσετε τις επαφές των ΗΠΑ φιλτράροντας εκείνες με αμερικανικό αριθμό τηλεφώνου.

### Προεπιλεγμένη Προβολή

Κάθε αντικείμενο έρχεται με μια μη φιλτραρισμένη, μη ταξινομημένη και μη διαγράψιμη προβολή, γνωστή ως Προεπιλεγμένη προβολή. It's named after the object's plural name, such as "All Companies", "All People", "All Opportunities".

## Δημιουργία, Επεξεργασία και Διαγραφή Προβολών

Μπορείτε να δημιουργήσετε αρκετές προσαρμοσμένες προβολές και να τις μοιραστείτε με την ομάδα σας.

### Δημιουργία Προβολής

Υπάρχουν δύο τρόποι για να δημιουργήσετε μια νέα προβολή. Είτε απευθείας από το `View Switcher`, είτε φιλτράροντας και ταξινομώντας μια υπάρχουσα προβολή.

#### From View Switcher

1. Open the View Switcher
2. Click the `Add View` button at the bottom of the view switcher menu.
3. Choose an Icon and name for your View.
4. Choose a "View type" between Table and Kanban.
5. (Kanban Views only) For your Kanban view, select the Select field you wish to use as the column header. The system prompts you to create one from the Settings before enabling Kanban view creation, if your object lacks a `Select` type field.
6. Click "Create" to generate your new view.

The newly created view opens automatically.

<VimeoEmbed videoId="927639721" title="Video demonstration" />

#### From Sorting and Filtering

When you change the `Sorting` and `Filtering` of an existing view, a `Save as new view` button will appear at the right edge of the `View Bar`. This will open the New View menu mentioned above, allowing you to create a new view out of an existing one.

<VimeoEmbed videoId="927643495" title="Video demonstration" />

### Editing and Deleting a View

To Edit or Delete a view:

1. Open the View Switcher.
2. Hover the View you wish to edit.
3. Click on the three dots that appear upon hover to edit and delete views.

Upon clicking on edit, you can change the icon and the name of the view or delete the view completely.

## Πλοήγηση Μεταξύ Προβολών

Για να αλλάξετε μεταξύ προβολών, απλώς ανοίξτε την εναλλαγή προβολών και κάντε κλικ στην προβολή που επιθυμείτε να ανοίξετε.

## Προσαρμογή Προβολών

Κάθε προβολή μπορεί να προσαρμοστεί για να διευκολύνει τις επιχειρηματικές σας διαδικασίες. Οι προβολές βοηθούν στον διαχωρισμό των δεδομένων σας. Μπορείτε να προσαρμόσετε τις προβολές χρησιμοποιώντας φίλτρα, ταξινομήσεις και εμφάνιση πεδίου.

### Φιλτράρισμα Προβολής

Για να φιλτράρετε μια προβολή:

- Κάντε κλικ στο **Φίλτρο** > επιλέξτε ένα πεδίο (για παράδειγμα: Όνομα, Εταιρεία στο Αντικείμενο Ανθρώπων).
- Επιλέξτε μια συνθήκη, επιλέξτε μια τιμή και εφαρμόστε το φίλτρο.
- Προσθέστε περισσότερα φίλτρα με το `+ Προσθήκη φίλτρου` ή αφαιρέστε τα με το **X**.

<VimeoEmbed videoId="926282262" title="Video demonstration" />

### Ταξινόμηση Προβολής

Ταξινομήστε τα δεδομένα των πεδίων σας σε αύξουσα ή φθίνουσα σειρά:

- Επιλέξτε **Ταξινόμηση**, επιλέξτε ένα πεδίο και ορίστε τη σειρά ταξινόμησης που επιθυμείτε.
- Μπορείτε να εφαρμόσετε και να κανονίσετε αρκετές ταξινομήσεις όπως απαιτείται.
  <VimeoEmbed videoId="927885588" title="Video demonstration" />

### Εμφάνιση πεδίου

Μπορείτε να προσαρμόσετε ποια πεδία θα εμφανίζονται σε μια προβολή. To hide a field, click **Options** > **Fields** and select the hide icon upon hover or click on the field column header and select **hide**.

Μπορείτε να αναδιατάξετε τα πεδία κάνοντας κλικ στην κεφαλίδα της στήλης πεδίου και, στη συνέχεια, πατήστε **Μετακίνηση Δεξιά** ή **Μετακίνηση Αριστερά**.

### Άνοιγμα Καταγραφής

Για να ανοίξετε μια καταγραφή, κάντε κλικ στο όνομα στην πρώτη στήλη. Αυτή η ενέργεια θα ανοίξει την αντίστοιχη Σελίδα Καταγραφής ως πλευρικό πάνελ.
