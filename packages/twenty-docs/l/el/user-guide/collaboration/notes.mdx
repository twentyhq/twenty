Διαχειριστείτε αποτελεσματικά τις σημειώσεις που συνδέονται με τις εγγραφές χρησιμοποιώντας τη δυνατή λειτουργία **Σημειώσεις**. Ο οδηγός αυτός παρουσιάζει πώς να δημιουργείτε, μορφοποιείτε, σχολιάζετε και διαγράφετε σημειώσεις χωρίς προβλήματα μέσα στις σελίδες εγγραφών.

## Κοινές Περιπτώσεις Χρήσης

### Τεκμηρίωση Συνάντησης

- **Πρακτικά Συνάντησης**: Καταγράψτε τα κύρια σημεία συζήτησης, αποφάσεις και ενέργειες από τηλεφωνικές συναντήσεις με πελάτες
- **Σύνοψη Κλήσεων**: Καταγράψτε σημαντικές λεπτομέρειες από συνομιλίες πωλήσεων ή κλήσεις υποστήριξης
- **Σημειώσεις Παρακολούθησης**: Καταγράψτε τα επόμενα βήματα και τις δεσμεύσεις που έγιναν κατά τη διάρκεια των συναντήσεων

### Αλληλεπιδράσεις με Πελάτες

- **Ιστορικό Υποστήριξης**: Παρακολουθήστε τα θέματα πελατών, τις λύσεις που έχουν δοθεί και την κατάσταση επίλυσης
- **Sales Context**: Record customer preferences, pain points, and buying signals
- **Οικοδόμηση Σχέσεων**: Καταγράψτε προσωπικές λεπτομέρειες σχετικά με τους επαφές για να ενισχύσετε τις σχέσεις

### Διαχείριση Έργων

- **Ενημερώσεις Κατάστασης**: Καταγράψτε την πορεία του έργου και τα επιτεύγματα ορόσημων
- **Παρακολούθηση Θεμάτων**: Καταγράψτε τα προβλήματα που προέκυψαν και τις λύσεις που εφαρμόστηκαν
- **Μεταβιβάσεις Ομάδας**: Μοιραστείτε το πλαίσιο όταν μεταβιβάζετε λογαριασμούς μεταξύ μελών της ομάδας

### Αυτόματη Δημιουργία Σημειώσεων

Χρησιμοποιήστε [Workflows](/l/el/user-guide/workflows/getting-started-workflows) για να δημιουργήσετε αυτόματα σημειώσεις:

- **Ενσωμάτωση Καταγραφέα Κλήσεων**: Αυτόματη δημιουργία περιλήψεων συναντήσεων από καταγεγραμμένες κλήσεις
- **Σημειώσεις Μεταβίβασης Συμφωνιών**: Αυτόματη δημιουργία περιλήψεων κύκλου πωλήσεων κατά τη μεταβίβαση νέων πελατών σε ομάδες υλοποίησης

## Χαρακτηριστικά Σημειώσεων

### Πεδίο Σχέσεων

Οι σημειώσεις περιλαμβάνουν ένα **Πεδίο Σχέσεων** που σας επιτρέπει να συνδέσετε μία σημείωση σε πολλαπλά αρχεία σε διάφορα αντικείμενα. Για παράδειγμα, μπορείτε να συνδέσετε μια σημείωση συνάντησης σε:

- Το Πρόσωπο που συναντήσατε
- Η Εταιρεία που εκπροσωπούν
- Την Ευκαιρία που συζητείται
- Οποιαδήποτε σχετική Εργασία ή άλλο αρχείο

Αυτή η σχέση πολλαπλών μορφών εξασφαλίζει ότι οι σημαντικές πληροφορίες είναι προσβάσιμες από όλες τις σχετικές σελίδες αρχείων.

### Επισήμανση Χρηστών

**Note**: User tagging within notes is not currently available. Αυτό το χαρακτηριστικό είναι προγραμματισμένο για το 2026, το οποίο θα σας επιτρέψει να αναφέρετε μέλη της ομάδας και να ενεργοποιήσετε ειδοποιήσεις.

## Δημιουργία Σημειώσεων

Η δημιουργία σημειώσεων στο σύστημα είναι διαισθητική και δυναμική. Μπορείτε είτε:

- Περιηγηθείτε στη σελίδα προβολής σημειώσεων και δημιουργήστε ένα νέο αρχείο.
- Μεταβείτε σε μια `Σελίδα Αρχείου` και επιλέξτε την καρτέλα Σημειώσεων και πατήστε το κουμπί `Νέα σημείωση`.

### Προσθήκη Περιεχομένου

Ξεκινήστε να πληκτρολογείτε άμεσα ή πατήστε `/` για να προσθέσετε στοιχεία όπως επικεφαλίδες, αρχεία ή εικόνες άμεσα.

### Μορφή Περιεχομένου

Μπορείτε να μορφοποιήσετε τις σημειώσεις σας απευθείας από τον επεξεργαστή. Χρησιμοποιήστε το συντακτικό Markdown, πατήστε το πλήκτρο `/` ή κάντε κλικ στο εικονίδιο `+` στον επεξεργαστή για να δείτε τις διαφορετικές επιλογές μπλοκ, όπως επικεφαλίδες, πίνακες και λίστες. Μπορείτε επίσης να επισυνάψετε εικόνες στη σημείωσή σας.

Επισημάνετε το κείμενο για να δείτε περισσότερες επιλογές μορφοποίησης όπως έντονα, πλάγια, και επιλογές ευθυγράμμισης.

Μπορείτε επίσης να αλλάξετε το χρώμα φόντου και το χρώμα κειμένου κάθε μπλοκ για να υπογραμμίσετε σημαντικές πληροφορίες στη σημείωσή σας. Για να το κάνετε αυτό, τοποθετήστε το δείκτη πάνω από το μπλοκ που θέλετε να μορφοποιήσετε και κάντε κλικ στο εικονίδιο `⋮` δίπλα στο εικονίδιο `+`. Κάντε κλικ στο `Χρώματα` για να ανοίξετε όλες τις επιλογές χρωμάτων τόσο για το κείμενο όσο και για το φόντο.

## Προβολή Σημειώσεων

Το σύστημα εμφανίζει όλες τις σημειώσεις σας που συνδέονται με συγκεκριμένο αρχείο κάτω από την ενότητα Σημειώσεων στην αντίστοιχη `Σελίδα Αρχείου`.

## Αποθήκευση και Διαγραφή

Όλες οι επεξεργασίες και οι προσθήκες στη σημείωση αποθηκεύονται αυτόματα.

Για να διαγράψετε μια σημείωση:

1. Ανοίξτε τη σημείωση που θέλετε να αφαιρέσετε κάνοντας κλικ σε αυτήν από την `Σελίδα Αρχείου`.
2. Επιλέξτε τη σημείωση που θέλετε να διαγράψετε στην καρτέλα σημειώσεων.
3. Χρησιμοποιήστε το κουμπί `Επιλογές` στην κάτω δεξιά γωνία για να ζητήσετε επιπλέον ενέργειες που περιλαμβάνουν τη διαγραφή.
4. Ολοκληρώστε τη διαγραφή σας όταν εμφανιστεί το παράθυρο επιβεβαίωσης.

Ένας άλλος τρόπος για να διαγράψετε μια σημείωση είναι μέσω της προβολής σημειώσεων, όπως θα κάνατε ένα κανονικό αρχείο. Σημειώστε ότι η διαγραφή μιας σημείωσης είναι μόνιμη και δεν μπορεί να αναιρεθεί.
