Διαχειριστείτε όλες τις εργασίες στο χώρο εργασίας σας χρησιμοποιώντας τη δυνατότητα **Εργασίες**. Αυτός ο οδηγός θα σας δείξει πώς να δημιουργείτε και να διαχειρίζεστε εργασίες, να μεταβαίνετε ανάμεσα σε επερχόμενες και ολοκληρωμένες εργασίες, να επεξεργάζεστε λεπτομέρειες εργασιών και πολλά άλλα.

## Κοινές Περιπτώσεις Χρήσης

### Παρακολούθηση Πωλήσεων

- **Επόμενα Βήματα Συνάντησης**: Δημιουργήστε εργασίες για θέματα δράσεων που συζητήθηκαν κατά τη διάρκεια των κλήσεων με πελάτες
- **Επόμενα Βήματα Πρότασης**: Ορίστε υπενθυμίσεις για έλεγχο εκκρεμών προτάσεων
- **Αναθεωρήσεις Συμβάσεων**: Προγραμματίστε εργασίες για διαπραγματεύσεις και εγκρίσεις συμβάσεων

### Επιτυχία Πελατών

- **Εργασίες Καθοδήγησης**: Δημιουργήστε αυτόματα λίστες ελέγχου καθοδήγησης όταν κλείνουν συμφωνίες
- **Υπενθυμίσεις Ελέγχου**: Προγραμματίστε τακτικές κλήσεις ελέγχου υγείας πελατών
- **Προετοιμασία Ανανεώσεων**: Ορίστε εργασίες για προετοιμασία ανανεώσεων συμβάσεων 90 ημέρες νωρίτερα

### Εσωτερική Διαχείριση Έργων

Πέρα από τις πωλήσεις και την επιτυχία πελατών, οι Εργασίες υποστηρίζουν ευρύτερες οργανωτικές ανάγκες:

- **Ανάπτυξη Προϊόντος**: Παρακολουθήστε εκδόσεις χαρακτηριστικών, διορθώσεις σφαλμάτων, και ορόσημα ανάπτυξης
- **Καμπάνιες Μάρκετινγκ**: Διαχειριστείτε λανσαρίσματα καμπανιών, δημιουργία περιεχομένου και προωθητικές ενέργειες
- **Λειτουργίες HR**: Διαχειριστείτε διαδικασίες προσλήψεων, ενσωμάτωση εργαζομένων και αξιολογήσεις απόδοσης
- **Οικονομικές Εργασίες**: Προγραμματίστε ελέγχους προϋπολογισμού, επεξεργασία τιμολογίων και οικονομικές αναφορές
- **Λειτουργίες**: Συντονισμός διαχείρισης εγκαταστάσεων, σχέσεων με προμηθευτές και βελτιώσεις διαδικασιών

### Αυτόματη Δημιουργία Εργασιών

Χρησιμοποιήστε [Ροές Εργασίας](/l/el/user-guide/workflows/getting-started-workflows) για να δημιουργείτε αυτόματα εργασίες:

- **Προκλητήρια Νίκης Συμφωνίας**: Δημιουργήστε αυτόματα εργασίες καθοδήγησης που ανατίθενται στην ομάδα CS όταν κλείνουν ευκαιρίες
- **Υπενθυμίσεις Email**: Ρυθμίστε εβδομαδιαίες υπενθυμίσεις εργασιών που λήγουν αυτή την εβδομάδα (στέλνονται κάθε Δευτέρα)
- **Αυτοματισμός Δικτύου**: Δημιουργήστε εργασίες παρακολούθησης όταν οι συμφωνίες σταματήσουν σε συγκεκριμένα στάδια
- **Ενσωμάτωση Συναντήσεων**: Δημιουργία αυτόματων εργασιών από εγγραφές συναντήσεων ή εκδηλώσεις ημερολογίου

## Χαρακτηριστικά Εργασιών

### Πεδίο Σχέσεων

Οι εργασίες περιλαμβάνουν ένα πεδίο `Σχέσεων` που επιτρέπει την επισύναψη μιας εργασίας σε πολλά αρχεία διαφορετικών αντικειμένων. Για παράδειγμα, μπορείτε να συνδέσετε μία εργασία παρακολούθησης με:

- Το Άτομο που πρέπει να επικοινωνήσετε
- Την Εταιρεία που εκπροσωπούν
- Η Ευκαιρία που επιδιώκεται
- Οποιαδήποτε σχετική Σημείωση ή άλλα αρχεία

Αυτή η σχέση με πολλές όψεις διασφαλίζει ότι οι εργασίες είναι προσβάσιμες από όλες τις σχετικές σελίδες αρχείων και παρέχει πλήρες πλαίσιο.

### Επισήμανση Χρηστών

**Σημείωση**: Η επισήμανση χρηστών εντός εργασιών δεν είναι προς το παρόν διαθέσιμη. Αυτή η δυνατότητα προγραμματίζεται για το 2026, η οποία θα σας επιτρέψει να αναφέρετε μέλη της ομάδας και να προκαλέσετε ειδοποιήσεις όταν αναθέτετε ή ενημερώνετε εργασίες.

## Δημιουργία Εργασιών

Η δημιουργία εργασιών στο Twenty είναι απρόσκοπτη. Μπορείτε είτε:

- Πηγαίνετε στην καρτέλα `Εργασίες` και πατήστε το κουμπί `+` στην επάνω δεξιά γωνία της σελίδας.
- Χρησιμοποιήστε τη λειτουργία αναζήτησης πατώντας `cmd/ctrl + k`, και επιλέξτε 'Δημιουργία εργασίας' από τη λίστα με τις γρήγορες ενέργειες.
- Μεταβείτε σε μια σελίδα `Αρχείου` και πατήστε `+` στην επάνω δεξιά γωνία της σελίδας, ή πηγαίνετε στην καρτέλα Εργασιών και πατήστε το κουμπί `Προσθήκη Εργασίας`.

### Προσθήκη Περιεχομένου Εργασίας

Μόλις δημιουργήσετε μια εργασία μπορείτε να την εμπλουτίσετε με πλούσιο περιεχόμενο, όπως τίτλους, κουκκίδες ή ακόμη και εικόνες. Για να το κάνετε, πατήστε `/` και εισάγετε την επιθυμητή εντολή.

## Προβολή Εργασιών

Η σελίδα **Εργασίες** εμφανίζει όλες τις εργασίες σας στο χώρο εργασίας. Εδώ μπορείτε:

- Φιλτράρετε εργασίες που έχουν ανατεθεί σε συγκεκριμένο χρήστη κάνοντας κλικ στο κουμπί με το όνομά σας στην επάνω δεξιά γωνία της οθόνης.
- Toggle between upcoming (`To do`) and completed (`Done`) tasks to see what needs attention and what you have accomplished.

Μπορείτε επίσης να δείτε εργασίες για ένα συγκεκριμένο Αρχείο στη σελίδα του Αρχείου του.

## Επεξεργασία Εργασιών

Για να επεξεργαστείτε μια εργασία, θα πρέπει να κάνετε κλικ στην κάρτα της. Αυτό θα ανοίξει ένα πλευρικό πάνελ που προσφέρει τις ακόλουθες δυνατότητες:

- **Assignee and Due date**: Update the assignee or edit the due date.
- **Σχόλια**: Συνεργαστείτε με τα μέλη της ομάδας σας προσθέτοντας σχόλια σε εργασίες για να δώσετε ενημερώσεις ή ανατροφοδότηση.
- **Αυτοματισμοί**: Χάρη στο API και τους Webhooks, μπορείτε επίσης να αυτοματοποιήσετε τη δημιουργία εργασίων που ενεργοποιούνται από συγκεκριμένες δραστηριότητες στο χώρο εργασίας σας.

## Σήμανση Εργασιών ως Ολοκληρωμένες

Για να σηματοδοτήσετε μια εργασία ως ολοκληρωμένη:

1. Βρείτε την εργασία στη σελίδα `Εργασιών` ή σε μία `Σελίδα Αρχείου`.
2. Κάντε κλικ στον κύκλο αριστερά από την κάρτα της εργασίας, αυτό θα αλλάξει για να προσδιορίσει την ολοκλήρωση.
3. Η κατάσταση της εργασίας θα ενημερωθεί αυτόματα σε `Ολοκληρωμένη`.

This procedure will help keep an updated record of your accomplishments.

## Διαγραφή Εργασίας

Για να καταργήσετε μόνιμα μια εργασία:

1. Ανοίξτε την εργασία που θέλετε να διαγράψετε κάνοντας κλικ στην κάρτα της, είτε από τη σελίδα `Εργασιών` είτε μέσα σε μια `Σελίδα Αρχείου`.
2. Κάντε κλικ στο εικονίδιο του κάδου που βρίσκεται στην επάνω δεξιά γωνία του πάνελ λεπτομερειών της εργασίας.

Παρακαλώ σημειώστε, η διαγραφή μιας εργασίας είναι μόνιμη και δεν μπορεί να αναιρεθεί. Consider marking tasks as `Done` if there is a chance you will need to refer to them again.
