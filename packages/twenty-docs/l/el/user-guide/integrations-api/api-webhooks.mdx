## Κλειδιά API

Τα κλειδιά API επιτρέπουν αυτοματοποιημένη πρόσβαση στα στοιχεία του CRM σας, συγχρονισμό δεδομένων με άλλα συστήματα, και δημιουργία προσαρμοσμένων ενσωματώσεων ή λύσεων.

### Δημιουργήστε ένα Κλειδί API

1. Μεταβείτε στις **Ρυθμίσεις → API & Webhooks**
2. Κάντε κλικ στο **+ Δημιουργία κλειδιού** στην επάνω δεξιά πλευρά
3. Configure your API key:
   - **Όνομα**: Προσθέστε ένα περιγραφικό όνομα στο κλειδί API σας
   - **Ημερομηνία λήξης**: Ορίστε πότε θα λήξει το κλειδί
4. Κάντε κλικ στο **Αποθήκευση** για να δημιουργήσετε το κλειδί API σας
5. **Σημαντικό**: Αντιγράψτε και αποθηκεύστε το κλειδί API σας αμέσως, εμφανίζεται μόνο μια φορά

Όταν δημιουργηθεί, το κλειδί API σας παρέχει πρόσβαση στα προσαρμοσμένα έγγραφα API σας και στον χώρο δοκιμών, όπου μπορείτε να δοκιμάσετε τις καταλήξεις με το πραγματικό σας μοντέλο δεδομένων.

<Warning>
Εφόσον το κλειδί API σας δίνει πρόσβαση σε ευαίσθητες πληροφορίες, δεν πρέπει να το μοιράζεστε με υπηρεσίες που δεν εμπιστεύεστε πλήρως. Εάν διαρρεύσει, κάποιος μπορεί να το χρησιμοποιήσει κακόβουλα. Εάν η ασφάλεια του κλειδιού API σας παραβιαστεί, απενεργοποιήστε το αμέσως και δημιουργήστε ένα νέο.
</Warning>

### Διαχείριση Κλειδιών API

**Δημιουργία εκ νέου ενός Κλειδιού API:**

1. Μεταβείτε στις **Ρυθμίσεις → API & Webhooks**
2. Κάντε κλικ στο κλειδί API που θέλετε να δημιουργήσετε εκ νέου
3. Κάντε κλικ στο κουμπί **Δημιουργία εκ νέου**
4. Αντιγράψτε και αποθηκεύστε το νέο κλειδί API αμέσως.

**Διαγραφή κλειδιού API:**

1. Βρείτε το κλειδί API στη λίστα σας.
2. Κάντε κλικ στο κλειδί για να ανοίξετε τις λεπτομέρειές του.
3. Κάντε κλικ στο **Διαγραφή** για να το αφαιρέσετε μόνιμα.

## Webhooks

Τα webhooks επιτρέπουν την άμεση ενημέρωση της διεύθυνσης URL που έχετε καθορίσει σχετικά με αλλαγές ή γεγονότα που αφορούν τα δεδομένα των πελατών σας.

Για παράδειγμα, όταν μια Ευκαιρία μεταβεί στο "Κλειστό Κερδισμένο", ένα webhook μπορεί να ενεργοποιήσει αυτόματα τη δημιουργία τιμολογίου στο λογιστικό σας σύστημα. Note that this type of automation can also be achieved using Twenty's in-app [Workflows feature](/l/el/user-guide/workflows/getting-started-workflows), which offers triggers based on field updates for internal automation.

Τα webhooks είναι ιδανικά για ενσωμάτωση με εξωτερικά συστήματα, ενώ οι Workflows υποστηρίζουν και εσωτερικό αυτοματισμό αλλά και συνδέσεις με εξωτερικά εργαλεία μέσω ενεργοποιητών webhook, κόμβων κώδικα και κόμβων HTTP.

### Δημιουργήστε Webhook

1. Μεταβείτε στις **Ρυθμίσεις → APIs & Webhooks → Webhooks**
2. Κάντε κλικ στο **+ Δημιουργία webhook**
3. Εισαγάγετε τη διεύθυνση URL του webhook σας (όπου θέλετε να λαμβάνετε ειδοποιήσεις)
4. Κάντε κλικ στο **Αποθήκευση**

Το webhook σας θα αρχίσει αμέσως να λαμβάνει ειδοποιήσεις πραγματικού χρόνου σχετικά με τις αλλαγές στα δεδομένα του CRM σας.

### Διαχείριση Webhooks

**Διαγραφή webhook:**

1. Μεταβείτε στις **Ρυθμίσεις → APIs & Webhooks → Webhooks**
2. Βρείτε το webhook που θέλετε να αφαιρέσετε.
3. Κάντε κλικ στο webhook.
4. Κάντε κλικ στο **Διαγραφή** και επιβεβαιώστε στο αναδυόμενο παράθυρο.

**Επεξεργασία webhook:**

1. Κάντε κλικ στο webhook που θέλετε να τροποποιήσετε.
2. Ενημερώστε τη διεύθυνση URL ή άλλες ρυθμίσεις.
3. Κάντε κλικ στο **Αποθήκευση** για να εφαρμόσετε τις αλλαγές.
