## Περί

Το αποθετήριο Twenty στο GitHub φιλοξενεί μια τεράστια ποικιλία πόρων όπως πηγαίος κώδικας, τεκμηρίωση, συζητήσεις, και παρακολούθηση θεμάτων. Εδώ είναι που θα μπορέσετε να αποκτήσετε πρόσβαση στον πλήρη κώδικα πίσω από το Twenty.

[Επισκεφθείτε το Twenty στο GitHub](https://github.com/twentyhq/twenty)

## Συνεισφέροντας

Contributing to the Twenty project on GitHub is a rewarding way to help improve the software you use. Είτε επιδιορθώνετε σφάλματα, προτείνετε χαρακτηριστικά, είτε βελτιώνετε την τεκμηρίωση, οι συνεισφορές σας είναι ευπρόσδεκτες.

### Reporting Issues

Συναντήσατε κάποιο πρόβλημα; [Δημιουργήστε ένα ζήτημα](https://github.com/twentyhq/twenty/issues/new) στο GitHub, παρέχοντας την καλύτερη δυνατή λεπτομέρεια.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggesting Features

Ποιες βελτιώσεις θα θέλατε να δείτε στο Twenty; No matter your technical know-how, you can join [the conversation here](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Coding a feature

Ξεκινήστε το ταξίδι σας βρίσκοντας καθήκοντα φιλικά για αρχάριους:

1. Πλοηγηθείτε στην καρτέλα **[Issues](https://github.com/twentyhq/twenty/issues)** του αποθετηρίου Twenty.
2. Φιλτράρετε με την ετικέτα **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** για να βρείτε καθήκοντα κατάλληλα για νέους.
3. Επιλέξτε ένα ζήτημα, κάντε fork το αποθετήριο, και ξεκινήστε να συμβάλλετε.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Βεβαιωθείτε ότι είστε ανατεθειμένοι στο ζήτημα για να αποφύγετε την επικάλυψη εργασίας με άλλους συμμετέχοντες.

### Κώδικας Συμπεριφοράς

Θυμηθείτε να τηρείτε τον [Κώδικα Συμπεριφοράς](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) του Twenty καθ' όλη τη διαδικασία της συμβολής σας.
