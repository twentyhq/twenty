## API

API (Application Programming Interface) allows you to connect Twenty with other software systems and build custom integrations.

## Μενού Εντολών

The Command Menu is a quick-access interface (opened with `Cmd + K` on Mac and `Ctrl + K` on Windows) that lets you perform actions, create records, and navigate your workspace efficiently.

## Εταιρεία & Άνθρωποι

Το CRM έχει δύο βασικούς τύπους αρχείων:

- Μια `Εταιρεία` αντιπροσωπεύει μια επιχείρηση ή οργανισμό.
- `Άτομα` εκπροσωπούν τους τρέχοντες και μελλοντικούς πελάτες της εταιρείας σας.

## Προσαρμοσμένα Πεδία

Τα Προσαρμοσμένα Πεδία είναι πεδία δεδομένων που δημιουργείτε για να καταγράφετε πληροφορίες που αφορούν συγκεκριμένα τις ανάγκες και τις διαδικασίες της επιχείρησής σας.

## Μοντέλο Δεδομένων

Ένα Μοντέλο Δεδομένων είναι η δομή που καθορίζει πώς οργανώνονται οι πληροφορίες στο CRM σας, συμπεριλαμβανομένων των αντικειμένων που υπάρχουν, τις ιδιότητές (πεδία) τους, και πώς σχετίζονται μεταξύ τους.

## Αγαπημένα

Τα Αγαπημένα είναι αρχεία που έχετε επισημάνει για γρήγορη πρόσβαση, εμφανιζόμενα στην πλευρική σας μπάρα για άμεση πλοήγηση σε σημαντικά δεδομένα.

## Πεδίο

Ένα πεδίο αναφέρεται σε έναν συγκεκριμένο τομέα όπου αποθηκεύονται δεδομένα για μια οντότητα.

## Integration

Integration are built-in tools that allow to link Twenty with other software or systems.

## Κανμπάν

Το `Kanban` είναι ένας οπτικός τρόπος παρακολούθησης των επιχειρηματικών διαδικασιών σας χρησιμοποιώντας κάρτες και στήλες. Κάθε στήλη αντιπροσωπεύει ένα στάδιο στη διαδικασία σας (για παράδειγμα: νέο, σε εξέλιξη, κερδισμένο, χαμένο), και μετακινείτε αρχεία μέσω αυτών των σταδίων καθώς προχωρούν.

## Αντικείμενο

Ένα Αντικείμενο είναι μια δομή δεδομένων που αντιπροσωπεύει έναν συγκεκριμένο τύπο οντότητας στο CRM σας (όπως Άτομα, Εταιρείες ή Ευκαιρίες). Τα Αντικείμενα μπορεί να είναι στάνταρ (ενσωματωμένα) ή προσαρμοσμένα (δημιουργημένα από εσάς).

## Ευκαιρίες

Οι Ευκαιρίες στο Twenty CRM είναι δυνητικές συμφωνίες ή πωλήσεις με λογαριασμούς ή επαφές.

## Εγγραφή

A Record indicates an instance of an object, like a specific account or contact.

## Πεδία Σχέσεων

Τα Πεδία Σχέσεων δημιουργούν συνδέσεις μεταξύ διαφορετικών αντικειμένων, επιτρέποντάς σας να συνδέετε αρχεία μεταξύ τους (όπως συνδέοντας ένα Άτομο με μια Εταιρεία).

## Πρότυπα Πεδία

Standard Fields are pre-built data fields that come with objects by default and provide common functionality across all workspaces.

## Εργασίες

Οι Εργασίες στο Twenty CRM είναι ανατεθειμένες δραστηριότητες που σχετίζονται με επαφές, λογαριασμούς ή ευκαιρίες.

## Προβολές

You can customize the display of your records using views, setting different filters, layouts and sorting options for each view.

## Webhooks

Τα Webhooks είναι αυτοματοποιημένα μηνύματα που αποστέλλονται από το Twenty σε άλλες εφαρμογές όταν λαμβάνουν χώρα συγκεκριμένα γεγονότα, επιτρέποντας συγχρονισμό δεδομένων σε πραγματικό χρόνο.

## Ροές Εργασίας

Τα Workflows είναι αυτοματοποιημένες διαδικασίες που ενεργοποιούν δράσεις με βάση συγκεκριμένες συνθήκες, βοηθώντας σας να αυτοματοποιείτε επαναλαμβανόμενες εργασίες και επιχειρηματικές διαδικασίες.

## Περιοχή Εργασίας

Ένας `Χώρος Εργασίας` συνήθως αντιπροσωπεύει μια εταιρεία που χρησιμοποιεί το Twenty. Περιλαμβάνει όλα τα αρχεία και τα δεδομένα που εσείς και τα μέλη της ομάδας σας προσθέτετε στο Twenty.  
Έχει ένα μοναδικό όνομα τομέα, το οποίο συνήθως είναι το όνομα τομέα που χρησιμοποιεί η εταιρεία σας για τις διευθύνσεις email των υπαλλήλων.

## Μέλη Χώρου Εργασίας

Τα Μέλη του Χώρου Εργασίας είναι οι χρήστες του Twenty από την ομάδα σας που έχουν πρόσβαση στο χώρο εργασίας σας. Μπορούν να ορίζονται ως ιδιοκτήτες ή αναθέτες για αρχεία.
