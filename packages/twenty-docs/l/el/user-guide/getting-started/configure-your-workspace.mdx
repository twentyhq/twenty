Κάθε επιχείρηση λειτουργεί διαφορετικά. Γι' αυτό το Twenty σας επιτρέπει να διαμορφώσετε το CRM σύμφωνα με τις ανάγκες σας, χωρίς να αναγκάζεστε να προσαρμόσετε τις διαδικασίες σας στο δικό μας μοντέλο.
**Ξεκινήστε με αυτά τα τρία βήματα για να το διαμορφώσετε στα μέτρα σας.**

**Quick Win**: Start with connecting your mailbox and customizing your data model with a few key fields. Αυτό σας προσφέρει άμεσα οφέλη και βοηθά την ομάδα σας να δει το Twenty σε δράση με πραγματικά δεδομένα. Μπορείτε να το κάνετε από το Ρυθμίσεις → Λογαριασμοί.

## 1. Προσαρμόστε το μοντέλο δεδομένων σας

Το Twenty προσφέρει την ευελιξία που χρειάζεστε για να διαμορφώσετε το μοντέλο δεδομένων που θα υποστηρίζει καλύτερα την καθημερινότητά σας.
Δημιουργήστε αντικείμενα και πεδία οποιουδήποτε τύπου, συμπεριλαμβανομένων των σχέσεων μεταξύ των διαφορετικών αντικειμένων σας. Μπορείτε να το κάνετε από το Ρυθμίσεις → Μοντέλο Δεδομένων.
Ορίστε μερικές συμβουλές:

- **Δεν έχετε περιορισμούς στον αριθμό των προσαρμοσμένων πεδίων ούτε στον αριθμό των προσαρμοσμένων αντικειμένων**. Η προσθήκη προσαρμοσμένων αντικειμένων και πεδίων δεν θα απαιτήσει την αναβάθμιση του προγράμματός σας.
- **Άτομα, Εταιρείες και Ευκαιρίες είναι τα τρία αντικείμενα από τα οποία μπορείτε να έχετε πρόσβαση στα email και τις συναντήσεις που συγχρονίζονται από το γραμματοκιβώτιο/ημερολόγιό σας**. Συνιστούμε τη χρήση τους όσο το δυνατόν περισσότερο, προσθέτοντας πεδία για να κατηγοριοποιήσετε τα αρχεία σας αν είναι απαραίτητο. Ορίστε ένα παράδειγμα:
  - Είναι καλύτερο να χρησιμοποιείτε το αντικείμενο Άτομα για τους υποψήφιους και συνεργάτες σας, δημιουργώντας ένα πεδίο στο αντικείμενο Άτομα με όνομα `Τύπος Ατόμου`, αντί να δημιουργείτε ένα προσαρμοσμένο αντικείμενο Συνεργάτης. Επειδή δεν θα μπορείτε να έχετε πρόσβαση στα email που ανταλλάσσονται με αυτό το άτομο από τα αρχεία Συνεργάτη.
  - Δημιουργήστε διαφορετικές προβολές στο Άτομα, μία για να εμφανίζει τους συνεργάτες και μία για να εμφανίζει τους υποψήφιους.
- Δύο Άτομα δεν μπορούν να έχουν την ίδια διεύθυνση email. Δύο Εταιρείες δεν μπορούν να έχουν το ίδιο domain.
- Μπορείτε να απενεργοποιήσετε τα τυπικά πεδία και αντικείμενα που δεν θέλετε να χρησιμοποιήσετε.
- Μπορείτε να κρύψετε πεδία από τις προβολές: μην φοβάστε να δημιουργήσετε πεδία, δεν θα χρειαστεί να τα εμφανίσετε όλα.

Διαβάστε [αυτό το άρθρο](https://docs.twenty.com/l/el/user-guide/data-model/customize-your-data-model) για να μάθετε πώς να σχεδιάσετε το μοντέλο δεδομένων σας.

## 2. Εισαγάγετε τα δεδομένα σας

Η εισαγωγή των υπαρχόντων δεδομένων στο Twenty δίνει στην ομάδα σας πλαίσιο από την αρχή.

### Συνδέστε το γραμματοκιβώτιό σας

Αν δεν το έχετε κάνει ήδη κατά τη δημιουργία του χώρου εργασίας σας, συνδέστε τον **λογαριασμό σας Google ή Microsoft** από το Ρυθμίσεις → Λογαριασμοί. Αυτό επιτρέπει στο Twenty να:

- Εισάγει τα μηνύματά σας και τις συναντήσεις
- Δημιουργεί αυτόματα επαφές με βάση τις αλληλεπιδράσεις σας (προαιρετικά)
- Διατηρεί το ιστορικό επικοινωνίας ορατό για την ομάδα σας

**Χρησιμοποιείτε άλλο πάροχο;**
Μπορείτε να προσθέσετε άλλο γραμματοκιβώτιο μέσω SMTP ή άλλο ημερολόγιο μέσω CalDAV. Θα χρειαστεί να ενεργοποιήσετε το χαρακτηριστικό από το Ρυθμίσεις → Εκδόσεις → Εργαστήριο και μετά να επιστρέψετε στην καρτέλα Ρυθμίσεις → Λογαριασμοί.

### Εισαγωγή δεδομένων μέσω csv

Χρησιμοποιήστε το μενού Εντολής (`Cmd + K` ή `Ctrl + K`) για να εισαγάγετε Άτομα, Εταιρείες, Ευκαιρίες ή οποιοδήποτε προσαρμοσμένο αντικείμενο μέσω CSV.

**Βασικές οδηγίες**:

- Κατεβάστε το αρχείο δείγματος για να κατανοήσετε τη μορφή που απαιτείται
- Περιορίστε κάθε αρχείο σε 10.000 εγγραφές
- Αφαιρέστε τα διπλά email για τα Άτομα ή τα διπλά domain για τις Εταιρείες
- Εξετάστε και διορθώστε τα σφάλματα (σημειωμένα με κίτρινο) πριν από την εισαγωγή

Διαβάστε [αυτό το άρθρο](/l/el/user-guide/getting-started/import-export-data) για να μάθετε περισσότερα σχετικά με την εισαγωγή δεδομένων.

## 3. Δημιουργήστε την πρώτη σας προβολή

Η δημιουργία διαφορετικών προβολών είναι το κλειδί για να καταστεί η πληροφόρηση αξιοποιήσιμη για την ομάδα σας.
Ορίστε πώς να προχωρήσετε:

- **Προσθέστε ή αποκρύψτε στήλες**
  Διαχειριστείτε τα πεδία που είναι ορατά σε μια δεδομένη προβολή κάνοντας κλικ στις Επιλογές → Πεδία (από πάνω δεξιά). Μπορείτε να δείξετε/κρύψετε πεδία από εκεί.

- **Αναδιάταξη πεδίων**
  Αναδιάταξη των πεδίων από μια δεδομένη προβολή κάνοντας κλικ στις Επιλογές → Πεδία (από πάνω δεξιά). Drag and drop the fields to reorder them.

- **Φιλτράρετε την προβολή σας**
  Περιορίστε τα αρχεία που εμφανίζονται χρησιμοποιώντας τα Φίλτρα από πάνω δεξιά.

- **Ταξινόμηση εγγραφών**
  Ταξινόμηση εγγραφών που εμφανίζονται χρησιμοποιώντας τη λειτουργία Ταξινόμησης από πάνω δεξιά ή κάνοντας κλικ στο όνομα της στήλης.

- **Επιλέξτε τη διάταξη**
  Μπορείτε να αλλάξετε σε διάταξη **Kanban** ή σε διάταξη λίστας **Ομάδα Ανά** αρκεί το αντικείμενο να έχει ένα πεδίο τύπου `Στάδιο` ή παρόμοιο τύπο επιλογής.

- **Αποθηκεύστε την προβολή σας ως Αγαπημένα**
  Αυτό μπορεί να γίνει χρησιμοποιώντας το μενού επιλογής που δείχνει τις διάφορες προβολές.
