Όταν συνδεθείτε στο Twenty για πρώτη φορά, η διάταξη θα πρέπει να σας φανεί διαισθητική. Έχει σχεδιαστεί για να σας βοηθάει να κινείστε γρήγορα και να παραμένετε οργανωμένοι, χωρίς να στέκεται εμπόδιο στον δρόμο σας.

## Η Κύρια Διάταξη

Το κέντρο της οθόνης είναι **όπου ζουν οι εγγραφές σας**: άτομα, εταιρείες, ευκαιρίες, εργασίες, σημειώσεις, ροές εργασίας και οποιοδήποτε άλλο αντικείμενο που δημιουργήσατε. Εδώ συμβαίνει η καθημερινή εργασία.
Μπορείτε **να δείτε, να επεξεργαστείτε, να διαγράψετε εγγραφές** από εκεί, καθώς και **να δημιουργήσετε νέες προβολές**.

## Γραμμή Πλοήγησης

Στα αριστερά, από πάνω έως κάτω, θα μπορείτε να:

- Εναλλάξτε ανάμεσα σε **πολλαπλούς χώρους εργασίας** χρησιμοποιώντας το μενού πτώσης ή δημιουργήστε νέο χώρο εργασίας
- Choose between the light and dark modes
- Χρησιμοποιήστε τη **γραμμή αναζήτησης** (πατήστε `/` για άμεση εστίαση)
- Open the **Settings** section

<Warning>
Please note that our API documentation is accessible under the Settings section and not the User Guide.
</Warning>

- Έχετε άμεση πρόσβαση στις **Αγαπημένες προβολές** σας. Οι αγαπημένες είναι μοναδικές για κάθε χρήστη.
- Περιηγηθείτε ανάμεσα σε διαφορετικά αντικείμενα
- **Δημιουργία αυτοματισμών** χρησιμοποιώντας ροές εργασίας
- Επικοινωνήστε με την Υποστήριξη και ανοίξτε τον Οδηγό Χρήσης μας.

## Μενού Εντολών & Γρήγορη Αναζήτηση

Το μενού εντολών σας δίνει **γρήγορη πρόσβαση σε ενέργειες και αναζήτηση** στο Twenty. Μπορείτε να το αποκτήσετε με δύο τρόπους:

- **Συντόμευση πληκτρολογίου**: Πατήστε `Cmd + K` (Mac) ή `Ctrl + K` (Windows)
- **Ποντίκι**: Κάντε κλικ στις τρεις κουκκίδες στη δεξιά γωνία επάνω

Θα δείτε επίσης μια γραμμή αναζήτησης στο πάνω μέρος της πλαϊνής σας μπάρας για γρήγορες αναζητήσεις εγγραφών, ή πατήστε `/` για άμεση εστίαση.

Από εκεί μπορείτε να:

- Δημιουργία νέων εγγραφών
- **Εισαγωγή και εξαγωγή δεδομένων μέσω csv**
- Δημιουργία νέων προβολών
- Πρόσβαση σε διαγραμμένες εγγραφές (Το Twenty υποστηρίζει μαλακές και σκληρές διαγραφές)
- Δείτε τις συντομεύσεις πληκτρολογίου για να αποκτήσετε γρήγορη πρόσβαση σε αντικείμενα στον χώρο εργασίας σας

## Προβολές

Κάθε αντικείμενο (όπως Ευκαιρίες ή Άνθρωποι) υποστηρίζει πολλαπλές προβολές. Δεν έχετε περιορισμούς στον αριθμό των προβολών ανά αντικείμενο.

Χρησιμοποιήστε το μενού πτώσης στο πάνω αριστερό μέρος της κύριας διάταξης για να εναλλάσσετε ανάμεσα στις διαφορετικές προβολές. Για παράδειγμα:

- Χρησιμοποιήστε την προβολή Kanban για να παρακολουθήσετε ευκαιρίες ανά στάδιο
- Χρησιμοποιήστε την προβολή Group By για να δημιουργήσετε ενότητες και να βελτιώσετε την αποδοτικότητα
- Χρησιμοποιήστε φίλτρα για να εστιάσετε σε συγκεκριμένες εγγραφές (π.χ. υποψήφιοι πελάτες που δημιουργήθηκαν την περασμένη εβδομάδα)
- Αποθηκεύστε φιλτραρισμένες προβολές για μελλοντική χρήση
- Κατάταξη προβολών στις αγαπημένες για γρήγορη πρόσβαση

Αν είστε νέοι στις Προβολές, διαβάστε το άρθρο μας [Διαχείριση Προβολών](/l/el/user-guide/crm-essentials/view-management) για να μάθετε πώς να τις δημιουργείτε και να τις προσαρμόζετε.

## Ρυθμίσεις

Ανοίξτε τις Ρυθμίσεις σας από το πάνω αριστερό μέρος για:

- **Συνδέστε τον λογαριασμό σας στο ταχυδρομείο και στο ημερολόγιο** για απρόσκοπτη συγχώνευση email και ημερολογίων
- Προσαρμόστε το **μοντέλο δεδομένων** σας: Δημιουργήστε προσαρμοσμένα αντικείμενα, πεδία και σχέσεις
- **Πρόσβαση στο παρασκήνιο API και ρύθμιση webhooks**
- **Διαχείριση αδειών χρήστη** και έλεγχοι πρόσβασης χώρου εργασίας
- Invite team members and manage user roles
- Επεξεργαστείτε το προφίλ σας και τις προτιμήσεις χώρου εργασίας σας
- Διαμόρφωση χρέωσης και παρακολούθηση χρήσης πιστώσεων ροών εργασίας
- Discover the latest releases and upcoming features (under Releases → Lab tab)

Αν δεν βλέπετε όλες αυτές τις ενότητες κάτω από τις Ρυθμίσεις, επικοινωνήστε με τον διαχειριστή του χώρου εργασίας σας - ορισμένες από αυτές έχουν περιορισμένη πρόσβαση.
