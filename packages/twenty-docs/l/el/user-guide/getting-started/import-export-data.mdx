## Import Data

- Μπορείτε να εισαγάγετε δεδομένα για οποιοδήποτε αντικείμενο χρησιμοποιώντας αρχεία .csv, .xlsx, ή .xls.
- Κάθε αρχείο που ανεβάζετε πρέπει να περιέχει **μόνο έναν τύπο αντικειμένου** (π.χ. μόνο αρχεία Ανθρώπων).
- Μπορείτε να χρησιμοποιήσετε την Εισαγωγή για να **δημιουργείτε ή να ενημερώνετε αρχεία**.

### Download a sample file to match the expected formatting

1. Μεταβείτε σε μια προβολή με το αντικείμενο που πρόκειται να εισαγάγετε.
2. Κάντε κλικ στο εικονίδιο `⋮` πάνω δεξιά και επιλέξτε `Εισαγωγή αρχείων`.
3. Click on `Download sample file`.

### Prepare your csv

Παρακάτω παρατίθενται ορισμένα στοιχεία που πρέπει να ελέγξετε προτού ανεβάσετε το αρχείο σας.

- Limit the number of records to **10,000 per file**.
- **Αφαιρέστε τα διπλότυπα** από το αρχείο σας.
  - Η μοναδικότητα στους `Ανθρώπους` ορίζεται από προεπιλογή από το `id` και το `email`. Έχετε επίσης τη δυνατότητα να καθορίσετε προσαρμοσμένα πεδία από `Ανθρώπους` ως μοναδικά όταν διαμορφώνετε το μοντέλο δεδομένων σας.
  - Η μοναδικότητα στις `Εταιρείες` ορίζεται από προεπιλογή από το `id` και το `domain`. Έχετε επίσης τη δυνατότητα να καθορίσετε προσαρμοσμένα πεδία από `Εταιρείες` ως μοναδικά όταν διαμορφώνετε το μοντέλο δεδομένων σας.
  - Για οποιοδήποτε άλλο αντικείμενο, συμπεριλαμβανομένων των προσαρμοσμένων αντικειμένων, έχετε τη δυνατότητα να ορίσετε ορισμένα πεδία ως μοναδικά. Βεβαιωθείτε ότι δεν έχετε συμπεριλάβει διπλότυπα κατά την προετοιμασία των αρχείων σας.
- Συνιστούμε να χρησιμοποιείτε τη σύνταξη `https://domain.com` όταν ανεβάζετε τα domain σας, καθώς αυτή είναι η σύνταξη που χρησιμοποιεί ο σύνδεσμος με την αλληλογραφία και το ημερολόγιό σας.
- Μπορείτε να **εισάγετε τις σχέσεις μεταξύ αντικειμένων** παρέχοντας ένα από τα μοναδικά πεδία του συσχετιζόμενου εγγραφής.
  - Παράδειγμα: Θέλετε να συσχετίσετε ένα άτομο με μια εταιρεία. Συμπεριλάβετε μια στήλη στο αρχείο που περιέχει όλα τα αρχεία `Ανθρώπων`, η οποία περιέχει το `id` της εταιρείας -- ή το `domain` της. Θα μπορείτε να αντιστοιχίσετε αυτό το πεδίο κατά τη διάρκεια του ανεβάσματος.
    <Warning>

**Σημαντική σημείωση:**

- Οι σχέσεις μεταξύ αντικειμένων στο Twenty είναι "Ένα προς Πολλά". Αυτό σημαίνει ότι κάθε εγγραφή του αντικειμένου Α μπορεί να συνδεθεί με πολλές εγγραφές του αντικειμένου Β. Αλλά κάθε εγγραφή του αντικειμένου Β μπορεί να ανήκει μόνο σε μία εγγραφή του αντικειμένου Α.
  _Για παράδειγμα, μια εταιρεία μπορεί να συνδέεται με πολλά άτομα. Και ένα άτομο μπορεί να ανήκει μόνο σε μία εταιρεία._

- Για να ανεβάσετε σχέσεις μέσω της λειτουργίας Εισαγωγής, πρέπει να παρέχετε το `id` (ή οποιοδήποτε άλλο μοναδικό πεδίο) του συσχετιζόμενου αντικειμένου στο αρχείο που περιέχει τις εγγραφές στο "Πλευρά των Πολλών" της σχέσης.  
  _Για παράδειγμα, παρέχετε το `id` ή το `domain` της εταιρείας όταν ανεβάζετε αρχεία ατόμων. Δεν παρέχετε το `id` των ατόμων (ή το `email`) όταν ανεβάζετε το αρχείο με τις εταιρείες._
  </Warning>

### Ανεβάστε το αρχείο σας

1. Μεταβείτε σε μια προβολή με το αντικείμενο που πρόκειται να εισαγάγετε.
2. Κάντε κλικ στο εικονίδιο `⋮` πάνω δεξιά και επιλέξτε `Εισαγωγή αρχείων`.
3. Κάντε κλικ στο `Επιλογή αρχείου`.
4. Επικυρώστε την αντιστοίχιση των πεδίων.

- You don't have to import all of them, you can choose the "Do not map" option.
- Για τις σχέσεις, συνιστάται να αντιστοιχίσετε μόνο ένα από τα μοναδικά πεδία.
- Μπορεί επίσης να χρειαστεί να αντιστοιχίσετε τις τιμές των πεδίων τύπου επιλογής και πολυεπιλογής.

5. Κάντε κλικ στο `Επόμενα Βήματα` και `Αναθεώρηση των γραμμών με σφάλματα`. Τα κελιά με πρόβλημα επισημαίνονται. **Μπορείτε είτε να αφαιρέσετε τη γραμμή είτε να ενημερώσετε το κελί απευθείας από εκεί**.
6. Αφού τελειώσετε, κάντε κλικ στο `Επιβεβαίωση`

### Import FAQ

<details><summary>Βλέπω προβλήματα με διπλότυπα όταν ανεβάζω το αρχείο μου, τι να κάνω;</summary>

Παρακαλούμε αναφερθείτε στο τμήμα **Προετοιμασία του csv σας** παραπάνω σε αυτό το άρθρο, περιέχει οδηγίες σχετικά με το τι θεωρείται διπλότυπο.

</details>

<details><summary>Μπορώ να εισαγάγω σχέσεις μεταξύ αντικειμένων;</summary>

Ναι, παρακαλούμε αναφερθείτε στο τμήμα **Προετοιμασία του csv σας** παραπάνω σε αυτό το άρθρο, περιέχει μια ενότητα σχετικά με την εισαγωγή σχέσεων.

</details>

<details><summary>Μπορώ να ενημερώσω υπάρχουσες εγγραφές χρησιμοποιώντας τη λειτουργία Εισαγωγής;</summary>

Ναι, μπορείτε να ενημερώσετε υπάρχουσες εγγραφές χρησιμοποιώντας τη λειτουργία Εισαγωγής. Βεβαιωθείτε ότι παρέχετε το id (ή οποιοδήποτε άλλο μοναδικό πεδίο) όταν επανα-ανεβάζετε τις εγγραφές σας.

</details>

<details><summary>Μπορώ να μεταφέρω το `id` από τα υπόλοιπα εργαλεία μου;</summary>

Ναι. Πρέπει να δημιουργήσετε ένα πεδίο που να το ορίσετε ως μοναδικό στο μοντέλο δεδομένων σας, που θα περιέχει το `id` από τα υπόλοιπα εργαλεία σας. Παρακαλώ σημειώστε ότι το όνομα `id` είναι προστατευμένο καθώς χρησιμοποιείται για το id του Twenty.  
Εάν θέλετε να δημιουργήσετε σχέσεις μεταξύ αντικειμένων χρησιμοποιώντας αυτό το πεδίο, ανατρέξτε στο τμήμα **Προετοιμασία του csv σας** παραπάνω σε αυτό το άρθρο. Περιέχει μια ενότητα σχετικά με την εισαγωγή σχέσεων.

</details>

## Export Data

Μπορείτε να κατεβάσετε δεδομένα από τα περισσότερα από τα αντικείμενα σας και έως 20,000 εγγραφές ανά εξαγωγή.
Για να εξάγετε δεδομένα από ένα αντικείμενο:

1. Επισκεφθείτε το ευρετήριο αντικειμένων.
2. Επιλέξτε την προβολή για εξαγωγή δεδομένων. Διαμορφώστε τις στήλες που θέλετε να κατεβάσετε, κρύβοντας ή προσθέτοντας στήλες και βρείτε τις εγγραφές που χρειάζεστε, φιλτράροντας την προβολή σας.
3. Πρόσβαση στο πλευρικό μενού μέσω του εικονιδίου `⋮` πάνω δεξιά.
4. Κάντε κλικ στο `Εξαγωγή προβολής`.
5. Επιλέξτε την τοποθεσία αποθήκευσης για τα δεδομένα CSV. Σημειώστε ότι η εξαγωγή μπορεί να πάρει χρόνο με μεγάλο αριθμό εγγραφών.
