## Βήμα 1: Εγγραφή

1. Μεταβείτε στο [Εγγραφή στο Twenty](https://app.twenty.com).
2. Επιλέξτε την προτιμώμενη μέθοδο εγγραφής σας:
   - **Συνέχεια με Google** για εγγραφή λογαριασμού Google.
   - **Συνέχεια με Microsoft** για εγγραφή λογαριασμού Microsoft.
   - Ή, **Συνέχεια με Email** για εγγραφή μέσω email.

<VimeoEmbed videoId="927066829" title="Video demonstration" />
## Step 2: Choosing a Trial Period

Επιλέξτε μεταξύ δύο δοκιμαστικών περιόδων:

### 30 ημέρες

Με πιστωτική κάρτα

### 7 ημέρες

Χωρίς πιστωτική κάρτα

Και οι δύο δοκιμές περιλαμβάνουν:

- Πλήρης πρόσβαση
- Απεριόριστες επαφές
- Ενοποίηση email
- Προσαρμοσμένα αντικείμενα
- API & Webhooks

Μπορείτε να κάνετε κλικ στο "Αλλαγή πακέτου" για να επιλέξετε ένα διαφορετικό πακέτο ή διάστημα χρεώσεων.

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## Βήμα 3: Επιβεβαίωση Πληρωμής & Ρύθμιση Λογαριασμού

Μετά την έγκριση πληρωμής μέσω Stripe, ανακατευθύνεστε για να δημιουργήσετε το χώρο εργασίας και το προφίλ χρήστη σας. Θυμηθείτε ότι μπορείτε να ακυρώσετε τη συνδρομή σας ανά πάσα στιγμή.
