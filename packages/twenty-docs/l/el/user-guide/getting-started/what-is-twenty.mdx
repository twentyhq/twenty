Το Twenty είναι το κορυφαίο ανοιχτού κώδικα CRM, δημιουργημένο από εκατοντάδες συνεργάτες, για να ταιριάζει στις μοναδικές επιχειρηματικές σας ανάγκες.

## Όραμα

Creating a good CRM is hard because it's a balancing act.
Για κάθε επιχείρηση, οι απαιτήσεις φαίνονται απλές, παρόλα αυτά οι ανάγκες του καθενός είναι ξεχωριστές.
Το αποτέλεσμα είναι ένα CRM που είναι είτε πολύ βασικό, είτε προσπαθεί να είναι παντογνώστης χωρίς να κατακτά κάποιον τομέα.

Στην αρχή, το Twenty μοιάζει με τα περισσότερα CRM που ήδη γνωρίζετε: μπορείτε να παρακολουθείτε συμφωνίες, να οργανώνετε επαφές, να διαχειρίζεστε εργασίες και σημειώσεις.
Αυτό που το κάνει να ξεχωρίζει είναι η προσέγγισή μας στην επεκτασιμότητα. Χτίζουμε μια ανοιχτή πλατφόρμα που παρέχει τα θεμέλια για να λύσετε τα μοναδικά σας επιχειρηματικά προβλήματα.

Προτεραιότητά μας είναι οι καθολικές αρχές και τα κοινά μοτίβα πάνω από τις λίστες χαρακτηριστικών.
Δεν προσπαθούμε να έχουμε όλες τις απαντήσεις, αλλά ενδυναμώνουμε τους χρήστες να βρουν αυτό που λειτουργεί καλύτερα για αυτούς.
Το ανοιχτού κώδικα είναι ο ακρογωνιαίος λίθος της προσέγγισής μας, διασφαλίζοντας ότι το Twenty εξελίσσεται με την κοινότητά του, για την κοινότητά του.

## Οφέλη

**Προσαρμόσιμο:** Σχεδιασμένο για να ταιριάζει στις επιχειρηματικές σας ανάγκες.

**Οδηγούμενο από την κοινότητα:** Δημιουργημένο και συντηρημένο από μια μεγάλη ανοιχτού κώδικα κοινότητα.

**Οικονομικά αποδοτικό:** Δεν θα δεσμευτείτε από κάποιον προμηθευτή, γιατί μπορείτε πάντα να το φιλοξενείτε μόνοι σας.

## Κύρια Χαρακτηριστικά

**Διαχείριση Επαφών:** Αποθηκεύστε και διαχειριστείτε αποτελεσματικά τα δεδομένα των πελατών. [Μάθετε περισσότερα](/l/el/user-guide/crm-essentials/contact-and-account-management).

**Προσαρμοσμένα Αντικείμενα:** Δημιουργήστε και προσαρμόστε αντικείμενα για να ταιριάζουν στις επιχειρηματικές σας ανάγκες. [Λεπτομέρειες](/l/el/user-guide/data-model/objects).

**Προσαρμοσμένα πεδία:** Εξατομικεύστε τα πεδία δεδομένων για να συλλάβετε και να οργανώσετε πληροφορίες που είναι ειδικές για τη λειτουργία σας. [Κατανοήστε περισσότερα](/l/el/user-guide/data-model/fields).

**Διαχείριση Συμφωνιών:** Παρακολουθήστε και διαχειριστείτε τις ευκαιρίες πωλήσεών σας μέσω προσαρμόσιμων [σταδίων Pipeline](/l/el/user-guide/crm-essentials/pipeline).

**Προβολές Kanban & Πίνακα:** Κάντε τα δεδομένα χρήσιμα με [ευέλικτες προβολές πινάκων](/l/el/user-guide/crm-essentials/view-management).

**Workflows:** Automate your business processes and integrate with external tools using powerful workflow automation. [Ξεκινήστε](/l/el/user-guide/workflows/getting-started-workflows).

**Ενσωμάτωση Email:** Προβάλετε τα emails ενός συγκεκριμένου πελάτη ή εταιρείας μέσα στο χώρο εργασίας σας. [Συγχρονίστε το γραμματοκιβώτιό σας](/l/el/user-guide/collaboration/emails-and-calendars).

**Σημειώσεις:** Δημιουργήστε λεπτομερείς σημειώσεις για κάθε εγγραφή, ώστε να μοιράζεστε πιο αποτελεσματικά τη γνώση. [Προσθέστε σημειώσεις](/l/el/user-guide/collaboration/notes).

**Εργασίες:** Προγραμματίστε εργασίες για να παρακολουθείτε τις αλληλεπιδράσεις με τους πελάτες. [Δείτε πώς](/l/el/user-guide/collaboration/tasks).

**Δικαιώματα:** Ελέγξτε την πρόσβαση και διαχειριστείτε τους ρόλους χρηστών με ευέλικτα δικαιώματα επιπέδου χώρου εργασίας και αντικειμένων. [Ρυθμίστε τα δικαιώματα](/l/el/user-guide/settings/permissions).

**API & Webhooks:** Συνδεθείτε με άλλες εφαρμογές και αυτοματοποιήστε ροές εργασίας με API και Webhooks. [Ξεκινήστε την ενσωμάτωση](/l/el/user-guide/integrations-api/api-webhooks).
