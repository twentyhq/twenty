## Πριν ξεκινήσετε

### 1. Ελέγξτε τα Τρέχοντα Δεδομένα σας

- **Επιλέξτε τα λίγα αντικείμενα και πεδία για μετανάστευση**: αυτή η μετανάστευση είναι μια ευκαιρία για μια νέα αρχή
- **Εξάγετε αυτά τα δεδομένα** από το τρέχον CRM σας
- **Αφαιρέστε τα διπλότυπα** και τα παρωχημένα αρχεία
- **Καταγράψτε τις ενεργές ροές εργασίας** και αυτοματισμούς

### 2. Δημιουργήστε το Νέο Μοντέλο Δεδομένων σας

Ακολουθήστε τον [οδηγό μοντέλου δεδομένων](/l/el/user-guide/data-model/customize-your-data-model) μας για να:

- **Σχεδιάστε το μοντέλο δεδομένων** που χρειάζεστε
- **Αντιστοιχίστε τα υπάρχοντα πεδία** στα τυποποιημένα αντικείμενα / πεδία του Twenty
- **Προσδιορίστε τα προσαρμοσμένα αντικείμενα / πεδία** που θα χρειαστείτε
- **Δημιουργήστε τα** στις Ρυθμίσεις → Μοντέλο δεδομένων

## Διαδικασία Μετανάστευσης

### 1. Εισάγετε τα Δεδομένα σας

**Συνιστώμενη σειρά:**

1. **Εταιρείες** πρώτα (ως βασικά αρχεία)
2. **Άτομα** δεύτερον (συνδεδεμένα με εταιρείες)
3. **Ευκαιρίες** τρίτον (συνδεδεμένες με άτομα/εταιρείες)

Χρησιμοποιήστε την εισαγωγή CSV μέσω του Μενού Εντολών `Cmd + K` (Mac) ή `Ctrl + K` (Windows). Δείτε τον οδηγό [εισαγωγής δεδομένων](/l/el/user-guide/getting-started/import-export-data) για λεπτομερείς οδηγίες.

### 2. Αναδημιουργία Ροών Εργασίας

- **Ξεκινήστε με τα απλά** - αναδημιουργήστε πρώτα τους πιο κρίσιμους αυτοματισμούς σας
- **Χρησιμοποιήστε τον δημιουργό ροών εργασίας του Twenty** για να αντικαταστήσετε υπάρχοντες αυτοματισμούς

## Κοινές Προκλήσεις

### Προβλήματα Μορφοποίησης Δεδομένων

- **Διευθύνσεις email** - αφαιρέστε τα διπλότυπα (απαίτηση αντικειμένου Άτομα)
- **Όνομα τομέα** - αφαιρέστε τα διπλότυπα (απαίτηση αντικειμένου Εταιρείες)

<Warning>
Please note that domain URLs created by the synchronization with your mailbox and calendar have the following format ```https://domain.com```
</Warning>

- **Μορφές ημερομηνίας** - διασφαλίστε συνεπή μορφοποίηση (YYYY-MM-DD) ή επεξεργαστείτε αυτή τη μορφή στις Ρυθμίσεις → Εμπειρία
- **Αριθμοί τηλεφώνου** - χρησιμοποιήστε τη διεθνή μορφή (+1234567890)

### Χαρτογράφηση Σχέσεων

Για εισαγωγή σχέσεων μεταξύ των εγγραφών χρησιμοποιώντας τη λειτουργία εισαγωγής csv, μπορείτε να χρησιμοποιήσετε τα εξής πεδία

- **Χρησιμοποιήστε τα αναγνωριστικά του Twenty** για πολύπλοκες σχέσεις
- **Χρησιμοποιήστε διευθύνσεις email** για σύνδεση αντικειμένων Άτομα
- **Χρησιμοποιήστε ονόματα τομέων** για σύνδεση αντικειμένων Εταιρείες
- **Χρησιμοποιήστε οποιοδήποτε άλλο πεδίο που έχετε ορίσει ως μοναδικό**, το οποίο μπορεί να πραγματοποιηθεί στην ενότητα Μοντέλο Δεδομένων.
  Διαβάστε τον οδηγό [εισαγωγής-εξαγωγής δεδομένων](/l/el/user-guide/getting-started/import-export-data) για λεπτομερείς οδηγίες σχετικά με τη δημιουργία σχέσεων κατά την εισαγωγή CSV.

## Επαγγελματική Βοήθεια

### Οι Υπηρεσίες μας

- **Συσκευασίες 4 ωρών εκκίνησης** για καθοδηγούμενη μετανάστευση
- **Συνεργάτες Υλοποίησης** για πιο προχωρημένα έργα

Ανακαλύψτε τις [υπηρεσίες υλοποίησης](/l/el/user-guide/getting-started/implementation-services).

## Μετανάστευση από Αυτο-φιλοξενούμενη σε Νεφός

Εάν μετακινείστε από το Twenty αυτο-φιλοξενούμενο στο Twenty Νεφός:

1. **Εξάγετε τα δεδομένα σας** από την αυτο-φιλοξενούμενη εγκατάσταση σας
2. **Ακολουθήστε τη συνηθισμένη διαδικασία μετανάστευσης** παραπάνω
3. **Μπορούμε να παρέχουμε βοήθεια στη μετανάστευση**, επικοινωνήστε με την ομάδα μας
