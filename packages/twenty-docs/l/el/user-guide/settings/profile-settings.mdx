## Προσωπικές Πληροφορίες

### Όνομα και Ηλεκτρονική Αλληλογραφία

- **Όνομα Προβολής**: Ενημερώστε πώς εμφανίζεται το όνομά σας σε άλλα μέλη του χώρου εργασίας
- **Διεύθυνση Ηλεκτρονικού Ταχυδρομείου**: Αλλάξτε το email σύνδεσής σας (απαιτείται επαλήθευση)
- **Εικόνα Προφίλ**: Ανεβάστε ένα προσαρμοσμένο avatar ή χρησιμοποιήστε τα αρχικά σας

## Ρυθμίσεις Ασφαλείας

### Δύο-Παράγοντας Ταυτοποίηση (2FA)

Ενεργοποιήστε το 2FA για να προσθέσετε επιπλέον επίπεδο ασφάλειας στον λογαριασμό σας:

1. Μεταβείτε στις **Ρυθμίσεις → Ρυθμίσεις Προφίλ**
2. Κάντε κλικ στο **Ενεργοποίηση 2FA**
3. Σαρώστε τον κωδικό QR με την εφαρμογή ταυτοποίησής σας
4. Εισάγετε τον κωδικό επαλήθευσης για επιβεβαίωση

### Διαχείριση Κωδικού

- **Αλλαγή Κωδικού**: Ενημερώστε τον τρέχοντα κωδικό σας
- **Απαιτήσεις Κωδικού**: Πρέπει να είναι τουλάχιστον 8 χαρακτήρες.

## Διαχείριση Προφίλ

### Διαγραφή Λογαριασμού

<Warning>
Η διαγραφή του λογαριασμού σας θα καταργήσει μόνιμα την πρόσβασή σας σε όλους τους χώρους εργασίας. Αυτή η ενέργεια δεν μπορεί να αναιρεθεί, θα χάσετε την πρόσβαση σε όλους τους χώρους εργασίας όπου είστε μέλος, και πρέπει να εξετάσετε το ενδεχόμενο να αποχωρήσετε από μεμονωμένους χώρους εργασίας αν επιθυμείτε να αποχωρήσετε από συγκεκριμένες ομάδες.
</Warning>

Για να διαγράψετε τον λογαριασμό σας:

1. Μεταβείτε στις **Ρυθμίσεις → Ρυθμίσεις Προφίλ**
2. Πηγαίνετε μέχρι το **Επικίνδυνη Ζώνη**
3. Κάντε κλικ στο **Διαγραφή Λογαριασμού**
4. Επιβεβαιώστε πληκτρολογώντας τη διεύθυνση email σας
