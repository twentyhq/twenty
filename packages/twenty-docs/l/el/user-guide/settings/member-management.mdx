## Πρόσκληση Νέων Μελών

### Χρησιμοποιώντας τον Σύνδεσμο Πρόσκλησης

1. Μεταβείτε στις **Ρυθμίσεις → Μέλη**
2. Αντιγράψτε τον σύνδεσμο πρόσκλησης χώρου εργασίας
3. Μοιραστείτε τον σύνδεσμο με νέα μέλη της ομάδας
4. Θα έχουν πρόσβαση μόλις εγγραφούν

### Άμεση Πρόσκληση μέσω Email

1. Μεταβείτε στις **Ρυθμίσεις → Μέλη**
2. Εισάγετε τη διεύθυνση email του ατόμου
3. Κάντε κλικ στην επιλογή **Πρόσκληση**
4. Θα λάβουν πρόσκληση μέσω email

## Αφαίρεση Μελών

### Διαγραφή Μέλους

1. Μεταβείτε στις **Ρυθμίσεις → Μέλη**
2. Βρείτε το μέλος που θέλετε να αφαιρέσετε
3. Κάντε κλικ στο κουμπί διαγραφής/αφαίρεσης δίπλα στο όνομά τους
4. Επιβεβαιώστε την αφαίρεση

**Σημείωση**: Τα αφαιρεθέντα μέλη χάνουν την πρόσβαση άμεσα, αλλά μπορούν να προσκληθούν ξανά αργότερα.

## Χρειάζεστε Βοήθεια;

Για τη διαχείριση ρόλων και δικαιωμάτων, δείτε το [άρθρο για τις Άδειες](/l/el/user-guide/settings/permissions).
