## Πιο Πρόσφατες Κυκλοφορίες

Παρακολουθήστε την εξέλιξη της ανάπτυξης του Twenty:

- **Σημειώσεις Κυκλοφορίας**: Δείτε τι νέο υπάρχει σε κάθε έκδοση
- **Ενημερώσεις Χαρακτηριστικών**: Ανακαλύψτε νέες δυνατότητες και βελτιώσεις

## Χαρακτηριστικά Lab

Αποκτήστε πρόσβαση σε πειραματικά χαρακτηριστικά πριν την επίσημη κυκλοφορία τους:

- **Δοκιμή Beta**: Δοκιμάστε νέα χαρακτηριστικά όσο βρίσκονται σε ανάπτυξη
- **Λειτουργίες Χαρακτηριστικών**: Ενεργοποιήστε ή απενεργοποιήστε συγκεκριμένες πειραματικές λειτουργίες

## Πώς να Αποκτήσετε Πρόσβαση

1. Πηγαίνετε στις **Ρυθμίσεις → Κυκλοφορίες**
2. Δείτε τις σημειώσεις κυκλοφορίας και αποκτήστε πρόσβαση στην καρτέλα **Lab** για δοκιμαστικά χαρακτηριστικά

Για περισσότερους πόρους κοινότητας, ελέγξτε την [ενότητα Πόρων](/l/el/user-guide/resources/glossary).
