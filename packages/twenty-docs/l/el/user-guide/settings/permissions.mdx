Το σύστημα αδειών του Twenty σας επιτρέπει να ελέγχετε την πρόσβαση σε τρεις κύριες περιοχές:

- **Αντικείμενα και Πεδία**: Ελέγξτε ποιος μπορεί να βλέπει, να επεξεργάζεται ή να διαγράφει εγγραφές και επιμέρους πεδία.
- **Ρυθμίσεις**: Διαχειριστείτε την πρόσβαση στη διαμόρφωση και τις διοικητικές λειτουργίες του χώρου εργασίας.
- **Ενέργειες**: Ελέγξτε γενικές ενέργειες χώρου εργασίας, όπως εισαγωγή δεδομένων ή αποστολή email.

## Δημιουργία Ρόλου

Για να δημιουργήσετε νέο ρόλο:

1. Πηγαίνετε στο **Ρυθμίσεις → Ρόλοι**
2. Στο **Όλοι οι Ρόλοι**, κάντε κλικ στο **+ Δημιουργία Ρόλου**
3. Enter a role name
4. Στην προεπιλεγμένη καρτέλα **Δικαιώματα**, διαμορφώστε τα δικαιώματα.
5. Κάντε κλικ στο **Αποθήκευση** για να ολοκληρώσετε.

## Delete a Role

Για να διαγράψετε έναν ρόλο:

1. Πηγαίνετε στο **Ρυθμίσεις → Ρόλοι**
2. Κάντε κλικ στον ρόλο που θέλετε να αφαιρέσετε.
3. Ανοίξτε την καρτέλα **Ρυθμίσεις**, και στη συνέχεια κάντε κλικ στο **Διαγραφή Ρόλου**
4. Πατήστε **Επιβεβαίωση** στο παράθυρο.

Σημείωση: Εάν διαγραφεί ένας ρόλος, οποιοδήποτε μέλος χώρου εργασίας του είχε εκχωρηθεί θα επανακαταλογισθεί αυτόματα στον προεπιλεγμένο ρόλο. Όλοι εκτός από τον ρόλο **Διαχειριστή** μπορούν να διαγραφούν. Πρέπει πάντα να υπάρχει τουλάχιστον ένα μέλος που έχει εκχωρηθεί στον ρόλο **Διαχειριστή**.

## Εκχώρηση Ρόλων σε Μέλη

### Προβολή τρεχόντων αναθέσεων

- Πηγαίνετε στο **Ρυθμίσεις → Ρόλοι**
- Δείτε όλους τους ρόλους και πόσα μέλη έχουν εκχωρηθεί σε καθέναν.
- Δείτε ποια μέλη έχουν ποιους ρόλους.

### Εκχώρηση Ρόλου σε Μέλος

1. Πηγαίνετε στο **Ρυθμίσεις → Ρόλοι**
2. Κάντε κλικ στον ρόλο που θέλετε να εκχωρήσετε.
3. Ανοίξτε την καρτέλα **Ανάθεση**
4. Κάντε κλικ στο **+ Ανάθεση σε μέλος**
5. Επιλέξτε το μέλος του χώρου εργασίας από τη λίστα.
6. Επιβεβαιώστε την ανάθεση.

### Ορίστε Προεπιλεγμένο Ρόλο

1. Πηγαίνετε στο **Ρυθμίσεις → Ρόλοι**
2. Στην ενότητα **Επιλογές**, βρείτε τον **Προεπιλεγμένο Ρόλο**
3. Επιλέξτε ποιον ρόλο θα λαμβάνουν αυτόματα νέα μέλη.
4. Νέα μέλη του χώρου εργασίας θα ανατεθούν σε αυτόν τον ρόλο όταν ενταχθούν.

**Σημείωση**: Μπορείτε να εκχωρήσετε ρόλους μόνο σε μέλη του υπάρχοντος χώρου εργασίας. Για να προσκαλέσετε νέα μέλη, χρησιμοποιήστε τη [Διαχείριση Μελών](/l/el/user-guide/settings/member-management).

## Προσαρμογή Δικαιωμάτων

Τα δικαιώματα καθορίζουν τι μπορεί κάθε ρόλος να έχει πρόσβαση ή να επεξεργαστεί μέσα στο χώρο εργασίας σας, συμπεριλαμβανομένων των εγγραφών αντικειμένων χώρου εργασίας, των ρυθμίσεων και των ενεργειών.

### Δικαιώματα Αντικειμένου και Πεδίου

Ελέγξτε την πρόσβαση σε εγγραφές και επιμέρους πεδία:

#### Δικαιώματα επιπέδου αντικειμένου

- Στην επιλογή **Όλα τα Αντικείμενα**, εφαρμόστε δικαιώματα όπως **Προβολή Εγγραφών**, **Επεξεργασία Εγγραφών**, **Διαγραφή Εγγραφών** ή **Καταστροφή Εγγραφών** σε όλα τα αντικείμενα.
- Στις **Αδειοδοτήσεις Αντικειμένων**, διαμορφώστε εξαιρέσεις για συγκεκριμένα αντικείμενα. Αυτά υπερισχύουν των ρυθμίσεων από **Όλα τα Αντικείμενα**.

#### Δικαιώματα Επιπέδου Πεδίου

- Διαμορφώστε δικαιώματα για επιμέρους πεδία μέσα σε κάθε αντικείμενο.
- Ελέγξτε ποιος μπορεί να **Βλέπει Πεδία**, **Επεξεργάζεται Πεδία** ή να μην έχει **Καμία Πρόσβαση** σε συγκεκριμένα πεδία.
- Τα δικαιώματα πεδίων λειτουργούν όπως τα δικαιώματα αντικειμένων με κληρονομικότητα και υπερισχύσεις.

#### Διαχείριση Υπερβάσεων Δικαιωμάτων

Για να υπερισχύσετε των δικαιωμάτων του γονέα και να εφαρμόσετε αυστηρότερους κανόνες:

- Κάντε κλικ στο **X** για να αφαιρέσετε τον κληρονομημένο κανόνα.
- Επιλέξτε τα συγκεκριμένα δικαιώματα για το επιλεγμένο αντικείμενο ή πεδίο.
- Κάντε κλικ στο πορτοκαλί εικονίδιο **Αναίρεση** (κυκλικό βέλος) για να αναιρέσετε αλλαγές.

Όταν τελειώσετε, κάντε κλικ στο **Τέλος** και μετά στο **Αποθήκευση** όταν επιστραφείτε στη σελίδα ρόλων.

### Δικαιώματα Ρυθμίσεων Χώρου Εργασίας

Ελέγξτε την πρόσβαση στις ρυθμίσεις του χώρου εργασίας με δύο τρόπους:

- Ενεργοποιήστε το **Ρυθμίσεις Πλήρης Πρόσβαση** για να δώσετε πλήρη πρόσβαση.
- Ή ενεργοποιήστε συγκεκριμένα δικαιώματα (π.χ., δημιουργία API key, προτιμήσεις χώρου εργασίας, ανάθεση ρόλων, διαμόρφωση πρότυπου δεδομένων, ρυθμίσεις ασφάλειας και διαχείριση ροών εργασίας).

### Δικαιώματα Ενεργειών Χώρου Εργασίας

Ελέγξτε την πρόσβαση σε γενικές ενέργειες χώρου εργασίας:

- Ενεργοποιήστε το **Εφαρμογή Πλήρης Πρόσβαση** για να δώσετε πλήρη δικαιώματα.
- Ή ενεργοποιήστε μεμονωμένες ενέργειες όπως **Αποστολή Email**, **Εισαγωγή CSV** και **Εξαγωγή CSV**.
