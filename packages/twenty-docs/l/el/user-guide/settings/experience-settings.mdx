## Εμφάνιση

### Επιλογή Θέματος

Επιλέξτε ανάμεσα σε λειτουργίες φωτός και σκοταδιού:

- **Λειτουργία Φωτός**: Καθαρή, φωτεινή διεπαφή ιδανική για καλά φωτισμένα περιβάλλοντα
- **Λειτουργία Σκοταδιού**: Εύκολη στα μάτια σε συνθήκες χαμηλού φωτισμού
- **Σύστημα**: Αντιστοιχεί αυτόματα με τη ρύθμιση θέματος της συσκευής σας.

## Περιφερειακές Ρυθμίσεις

### Γλώσσα

Επιλέξτε τη γλώσσα προτίμησής σας για τη διεπαφή του Twenty:

- Αγγλικά (προεπιλογή)
- Πρόσθετες γλώσσες διαθέσιμες βάσει κοινοτικών μεταφράσεων.

### Ζώνη Ώρας

Ορίστε την τοπική σας ζώνη ώρας για ακριβείς προγραμματισμούς και χρονοσημάνσεις:

- Επηρεάζει τις ώρες συναντήσεων, τις προθεσμίες εργασιών και τα αρχεία δραστηριότητας.
- Ρυθμίζεται αυτόματα για τη χειμερινή/θερινή ώρα.

### Μορφή Ημερομηνίας

Επιλέξτε πώς οι ημερομηνίες εμφανίζονται στο Twenty:

- **MM/DD/YYYY** (Μορφή ΗΠΑ)
- **DD/MM/YYYY** (Ευρωπαϊκή μορφή)
- **YYYY-MM-DD** (Μορφή ISO)

### Μορφή Αριθμών

Ρυθμίστε πώς εμφανίζονται οι αριθμοί και τα νομίσματα:

- **Δεκαδικός Διαχωριστής**: Κόμμα (,) ή τελεία (.)
- **Διαχωριστής Χιλιάδων**: Κενό, κόμμα ή τελεία
- **Σύμβολο Νομίσματος**: Βάσει της περιοχής σας ή προσαρμοσμένο

### Μορφή Ημερολογίου

Ορίστε τη διάταξη ημερολογίου που προτιμάτε:

- **Πρώτη Ημέρα Εβδομάδας**: Κυριακή ή Δευτέρα
- **Αριθμοί Εβδομάδων**: Εμφάνιση ή απόκρυψη ISO αριθμών εβδομάδων
- **Μορφή Ώρας**: 12ωρη (ΠΜ/ΜΜ) ή 24ωρη

## Πώς να Ενημερώσετε τις Ρυθμίσεις

1. Μεταβείτε στις **Ρυθμίσεις → Εμπειρία** από την πλαϊνή μπάρα
2. Προσαρμόστε τις προτιμήσεις σας σε κάθε ενότητα
3. Οι αλλαγές αποθηκεύονται αυτόματα.
4. Ανανεώστε τον περιηγητή σας για να δείτε όλες τις αλλαγές να ισχύουν.
