## Εικόνα Χώρου Εργασίας

- **Μεταφόρτωση Λογότυπου**: Προσθέστε ένα προσαρμοσμένο λογότυπο χώρου εργασίας
- **Υποστηριζόμενες μορφές**: Αρχεία PNG, JPEG, και GIF κάτω από 10MB
- **Αφαίρεση**: Διαγράψτε το τρέχον λογότυπο χώρου εργασίας

## Όνομα Workspace

- **Όνομα**: Αλλάξτε την εμφανιζόμενη ονομασία του χώρου εργασίας σας
- Αυτό το όνομα εμφανίζεται σε όλα τα μέλη του χώρου εργασίας

## Danger Zone

<Warning>
Deleting your workspace permanently removes all data and cannot be undone. Όλα τα δεδομένα του χώρου εργασίας θα χαθούν για πάντα, όλα τα μέλη θα χάσουν άμεσα την πρόσβαση, και αυτή η ενέργεια δεν μπορεί να αντιστραφεί.
</Warning>

Για να διαγράψετε τον χώρο εργασίας σας:

1. Κάντε κλικ στο κουμπί **Διαγραφή Χώρου Εργασίας**
2. Επιβεβαιώστε τη διαγραφή όταν σας ζητηθεί

**Σημείωση**: Μόνο οι διαχειριστές του χώρου εργασίας μπορούν να διαγράψουν χώρους εργασίας.
