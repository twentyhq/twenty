## Σχετικά με τις Προβολές Πινάκων

Οι προβολές πινάκων είναι οπτικές αναπαραστάσεις δεδομένων οργανωμένων σε σειρές και στήλες.

## Δημιουργία εγγραφής

Προσθέστε εγγραφές όπως χρειάζεται, χωρίς περιορισμούς. Για να προσθέσετε μια εγγραφή, μπορείτε είτε να κάνετε κλικ στο κουμπί **+** επάνω δεξιά στην οθόνη είτε επάνω στη στήλη **Όνομα** της εγγραφής.

Εισάγετε το όνομα της εγγραφής και πατήστε `Enter` για να αποθηκεύσετε. Για να τροποποιήσετε το όνομα μιας εγγραφής, κάντε κλικ στο όνομά της στη σελίδα λεπτομερειών.

## Διαγραφή εγγραφής

**Προβολή Ευρετηρίου:** Για να διαγράψετε μια εγγραφή, επιλέξτε το πλαίσιο επιλογής δίπλα στην εγγραφή και κάντε κλικ στο κουμπί διαγραφής επάνω δεξιά.

**Σελίδα Εγγραφής:** Πατήστε το εικονίδιο `⋮` επάνω δεξιά και στη συνέχεια επιλέξτε διαγραφή.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## Προσθήκη Προσαρμοσμένου Πεδίου

Για να δημιουργήσετε ένα προσαρμοσμένο πεδίο, κάντε κλικ στο κουμπί **+** στο δεξιό άκρο των στηλών του πίνακα και επιλέξτε **Προσαρμογή πεδίων**.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

Μπορείτε επίσης να το κάνετε πλοηγώντας στο **Ρυθμίσεις** > **Μοντέλο Δεδομένων** > **Άνθρωποι**. Click on **Add Field**. Επιλέξτε ένα όνομα και τύπο πεδίου. Το νέο πεδίο θα είναι διαθέσιμο στην εφαρμογή.
