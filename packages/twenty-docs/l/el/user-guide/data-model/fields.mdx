## Σχετικά με τα Πεδία

Τα πεδία είναι σαν τις στήλες σε ένα λογιστικό φύλλο. Αποθηκεύουν διάφορους τύπους δεδομένων όπως κείμενο, αριθμούς ή ημερομηνίες. Τα πεδία μπορεί να είναι πρότυπα (ενσωματωμένα) ή προσαρμοσμένα (αυτά που δημιουργείτε εσείς).

### Πρότυπα Πεδία

Τα πρότυπα πεδία έρχονται ενσωματωμένα με το Twenty για την κάλυψη κοινών επιχειρηματικών αναγκών.

Για παράδειγμα, `Όνομα` και `Επώνυμο` είναι πρότυπα πεδία στο αντικείμενο `Άτομα`. Αποθηκεύουν δεδομένα κειμένου για μεμονωμένα ονόματα.

Δεν μπορείτε να διαγράψετε πρότυπα πεδία, αλλά μπορείτε να τα απενεργοποιήσετε αν δεν τα χρειάζεστε.

Μπορείτε επίσης να προσαρμόσετε τις επιλογές των τυπικών πεδίων τύπου `SELECT`, όπως οι επιλογές για το `Στάδιο` στις Ευκαιρίες.

<img src="/images/user-guide/fields/standard-fields.png" style={{width:'100%'}}/>

### Προσαρμοσμένα Πεδία

Μπορείτε να προσθέσετε προσαρμοσμένα πεδία σε οποιοδήποτε αντικείμενο. Μπορείτε να αποθηκεύσετε κείμενο, αριθμούς, ημερομηνίες, επιλογές από αναδυόμενες λίστες και άλλα. Χρησιμοποιήστε προσαρμοσμένα πεδία για να παρακολουθείτε πληροφορίες που είναι συγκεκριμένες για την επιχείρησή σας.

Για παράδειγμα, ένα προσαρμοσμένο πεδίο για το SpaceX θα μπορούσε να είναι `Κατάσταση Ενεργού Πυραύλου`, υποδεικνύοντας εάν ένας πύραυλος είναι σε λειτουργία.

## Δημιουργία Προσαρμοσμένου Πεδίου

To add a custom field to any object, follow these steps:
1.	Μεταβείτε στις `Ρυθμίσεις` στην αριστερή πλευρική μπάρα.
2.	Μεταβείτε στο `Μοντέλο Δεδομένων`, και επιλέξτε το αντικείμενο που θέλετε να προσαρμόσετε.
3.	Συνεχίστε πατώντας στο `Προσθήκη Πεδίου`.
4.	Επιλέξτε ένα όνομα πεδίου και τύπο που ικανοποιούν τις απαιτήσεις σας. Σκεφθείτε να προσθέσετε μια περιγραφή πεδίου για καλύτερη αντίληψη.

Το νέο πεδίο που δημιουργήσατε είναι πλέον διαθέσιμο στα πεδία της εφαρμογής. Για να το εμφανίσετε σε μια συγκεκριμένη προβολή, κάντε κλικ στο μενού επιλογών και επιλέξτε `Πεδία`.

<VimeoEmbed videoId="927628219" title="Video demonstration" />

**Γρήγορος τρόπος:** Κάντε κλικ στο κουμπί **+** στο επάνω δεξιό μέρος οποιουδήποτε πίνακα αντικειμένων, και επιλέξτε `Προσαρμογή πεδίων`. Αυτό σας οδηγεί απευθείας στις ρυθμίσεις του Μοντέλου Δεδομένων.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

## Απενεργοποίηση πεδίου

Μπορείτε να απενεργοποιήσετε ένα πεδίο για να το αποκρύψετε από την εφαρμογή χωρίς να χάσετε τα δεδομένα σας. Σκεφθείτε το σαν να κρύβετε το πεδίο αντί να το διαγράψετε.

Να πως μπορείτε να το κάνετε:

1. Βρείτε το πεδίο που θέλετε να απενεργοποιήσετε στις ρυθμίσεις του αντικειμένου σας.

2. Κάντε κλικ στις τρεις τελείες `⋮` δίπλα στο πεδίο για να ανοίξετε το μενού.

3. Επιλέξτε `Απενεργοποίηση` από το dropdown.

<img src="/images/user-guide/fields/deactivate-field.png" style={{width:'100%'}}/>

Τι συμβαίνει όταν απενεργοποιείτε ένα πεδίο;

1. **Στην εφαρμογή:** Το πεδίο εξαφανίζεται και δεν μπορείτε να προσθέσετε νέες τιμές σε αυτό.

2. **Υπάρχουσες σχέσεις:** Αν είναι πεδίο σχέσεων, οι υπάρχουσες συνδέσεις παραμένουν αλλά δεν μπορείτε να δημιουργήσετε νέες.

3. **Πρόσβαση API:** Μπορείτε ακόμα να έχετε πρόσβαση στο πεδίο και τα δεδομένα του μέσω API.

Μπορείτε να επανενεργοποιήσετε Πρότυπα και Προσαρμοσμένα Πεδία ή να επιλέξετε να τα διαγράψετε μόνιμα.

## Κάνετε τα Πεδία Μοναδικά

Δημιουργήστε ένα πεδίο μοναδικό για να διασφαλίσετε ότι διαφορετικές εγγραφές δεν μπορούν να έχουν την ίδια τιμή. Για παράδειγμα, οι διευθύνσεις ηλεκτρονικού ταχυδρομείου είναι μοναδικές για κάθε άτομο.

Αν εμφανιστεί σφάλμα κατά τον ορισμό της μοναδικότητας, ελέγξτε για διπλές τιμές στα δεδομένα σας (συμπεριλαμβανομένων των διαγραμμένων εγγραφών).

## Καλύτερες Πρακτικές για Διαμόρφωση Πεδίου

### Καλύτερες Πρακτικές για Διαμόρφωση Πεδίου

- **Τα ονόματα πεδίων σχέσεων δεν μπορούν να ενημερωθούν** μετά τη δημιουργία (επηρεάζει τη δομή API)
- **Τα μοναδικά και πολλά ονόματα πρέπει να είναι ξεχωριστά**: Το GraphQL API μας χρειάζεται μοναδικά ονόματα για μεταβολές
- **Προστατευμένα ονόματα πεδίων**: ορισμένα ονόματα είναι κρατημένα για χρήση από το σύστημα (π.χ., `Τύπος`)

### Πεδία Νομισμάτων και Τηλεφώνων

- **Προεπιλεγμένο νόμισμα**: μπορεί να διαμορφωθεί μέσω του μοντέλου δεδομένων
- **Προεπιλεγμένοι κωδικοί χωρών**: μπορούν να διαμορφωθούν για πεδία τηλεφώνων μέσω του μοντέλου δεδομένων

### Επιλεγμένα Πεδία

- **Μπορεί να επιλεγεί μια προεπιλεγμένη επιλογή** για κάθε επιλεγμένο πεδίο

### Πεδία Κειμένου Καταγραφών

- **Κάθε αντικείμενο έχει ένα κύριο πεδίο εμφάνισης**: Αυτό το πεδίο εμφανίζεται στην αριστερή στήλη και αναπαριστά την εγγραφή όταν συνδέεται με άλλα αντικείμενα. Πρέπει να είναι ένα πεδίο κειμένου. Για παράδειγμα, το `Όνομα` σε Άτομα χρησιμοποιείται ως κύριο πεδίο, οπότε όταν συνδέετε ένα άτομο με μια εταιρεία, η εταιρεία βλέπει το όνομά τους.

### Πεδία Σχέσεων

- **Συνδέστε τα αντικείμενα**: Τα πεδία σχέσεων συνδέουν εγγραφές από διάφορα αντικείμενα. Για λεπτομερείς πληροφορίες για τη δημιουργία και διαχείριση σχέσεων, δείτε το άρθρο μας [Πεδία Σχέσεων](/l/el/user-guide/data-model/relation-fields).
