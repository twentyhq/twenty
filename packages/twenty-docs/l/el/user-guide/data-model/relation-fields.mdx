## Τι είναι τα Πεδία Σχέσεων;

Τα πεδία σχέσεων συνδέουν εγγραφές από ένα αντικείμενο σε εγγραφές σε άλλο αντικείμενο. Για παράδειγμα:

- **Άνθρωποι** → **Εταιρείες** (κάθε άτομο εργάζεται για μια εταιρεία)
- **Ευκαιρίες** → **Άνθρωποι** (κάθε συμφωνία έχει ένα άτομο επαφής)
- **Εργασίες** → **Ευκαιρίες** (κάθε εργασία σχετίζεται με μια συγκεκριμένη συμφωνία)

## Δημιουργία Πεδίων Σχέσεων

### 1. Προσθήκη του Πεδίου

Μεταβείτε στις **Ρυθμίσεις → Μοντέλο Δεδομένων → [Το Αντικείμενό Σας]** και κάντε κλικ στην επιλογή **Προσθήκη Πεδίου**.

### 2. Επιλογή Τύπου Σχέσης

- **Τύπος Πεδίου**: Επιλέξτε "Σχέση"
- **Αντικείμενο Στόχος**: Επιλέξτε ποιο αντικείμενο θα συνδεθεί
- **Σχέση**: Υποστηρίζονται μόνο σχέσεις **1-προς-πολλά** αυτή τη στιγμή. Βεβαιωθείτε ότι δημιουργείτε τη σχέση στη σωστή κατεύθυνση.

### 3. Διαμόρφωση Ονομάτων Πεδίων

Θα χρειαστεί να ορίσετε ονόματα και για τις δύο πλευρές της σχέσης:

- **Όνομα πεδίου πηγή**: Πώς εμφανίζεται το πεδίο στο τρέχον αντικείμενό σας
- **Όνομα πεδίου στόχος**: Πώς εμφανίζεται το αντίστροφο πεδίο στο αντικείμενο στόχος

<Warning>
Τα ονόματα των πεδίων δεν μπορούν να επεξεργαστούν αφού αποθηκευτεί η σχέση, καθώς επηρεάζει τη δομή του API. Επιλέξτε προσεκτικά.
</Warning>

## Σχετικά με τα Μέλη της Ομάδας

Μπορείτε να δημιουργήσετε σχέσεις με οποιοδήποτε αντικείμενο, συμπεριλαμβανομένων των **Μελών Εργασίας** (οι χρήστες της ομάδας σας στο Twenty). Αυτό είναι χρήσιμο για τη δημιουργία πεδίων ιδιοκτησίας:

- **Ιδιοκτήτης Λογαριασμού**: Συνδέστε μια εταιρεία με ένα μέλος της ομάδας που τη διαχειρίζεται
- **Ιδιοκτήτης Συμφωνίας**: Αναθέστε μια Ευκαιρία σε έναν συγκεκριμένο πωλητή

Όταν δημιουργείτε μια σχέση με `Μέλη Εργασίας`, θα δείτε τα ονόματα των μελών της ομάδας σας σε αναπτυσσόμενες επιλογές, κάνοντάς το εύκολο να αναθέσετε ιδιοκτησία και αρμοδιότητες.

## Καλύτερες Πρακτικές

- **Σχεδιάστε τις σχέσεις σας** πριν τις δημιουργήσετε
- **Use clear, descriptive names** for both field names
- **Δοκιμάστε τις σχέσεις** με δείγματα δεδομένων πριν από την πλήρη εφαρμογή

## Επερχόμενες Δυνατότητες

- **Σχέσεις Πολλά προς Πολλά** (Έρχεται το Q1 2026)
