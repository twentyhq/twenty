## Τυπικά αντικείμενα

Τα τυπικά αντικείμενα είναι προ-καθορισμένες οντότητες στο χώρο εργασίας σας για να σας βοηθήσουν να ξεκινήσετε. Είναι μέρος ενός κοινόχρηστου μοντέλου δεδομένων προσβάσιμους από όλους τους χρήστες του Twenty. Μπορείτε να τα χρησιμοποιήσετε ως έχουν, να τα προσαρμόσετε ή να τα απενεργοποιήσετε.

<img src="/images/user-guide/objects/standard-objects.png"style={{width:'100%'}}/>

### Άνθρωποι

Το αντικείμενο `Άνθρωποι` αποθηκεύει τις επαφές σας. Περιλαμβάνει λεπτομέρειες επικοινωνίας και ιστορικό αλληλεπίδρασης, ώστε να μπορείτε να δείτε όλες τις αλληλεπιδράσεις πελατών σας σε ένα μέρος.

### Εταιρεία

The `Companies` object stores your business accounts. Περιλαμβάνει λεπτομέρειες όπως τον κλάδο, το μέγεθος και την τοποθεσία. Companies connect to both `People` and `Opportunities` objects.

### Ευκαιρίες

Το αντικείμενο `Ευκαιρίες` αποθηκεύει δεδομένα που σχετίζονται με συμφωνίες. It tracks the progression of potential sales, from prospecting to closure, recording stages, deal sizes, associated account, and expected close date. You can view your sales pipeline in a kanban layout.

## Προσαρμοσμένα αντικείμενα

Τα προσαρμοσμένα αντικείμενα σας επιτρέπουν να αποθηκεύετε πληροφορίες που είναι μοναδικές για την οργάνωσή σας και που τα τυπικά αντικείμενα δεν μπορούν να χειριστούν. Για παράδειγμα, αν είστε SpaceX, μπορεί να θέλετε να δημιουργήσετε ένα προσαρμοσμένο αντικείμενο για Ρουκέτες και Εκτοξεύσεις.

<img src="/images/user-guide/objects/custom-objects.png"style={{width:'100%'}}/>

### Δημιουργία νέου προσαρμοσμένου αντικειμένου

Για να δημιουργήσετε νέο προσαρμοσμένο αντικείμενο:

1. Πηγαίνετε στις Ρυθμίσεις στην πλαϊνή μπάρα στα αριστερά.
2. Κάτω από το Χώρο Εργασίας, πηγαίνετε στο μοντέλο δεδομένων. Εδώ θα μπορείτε να δείτε μια επισκόπηση όλων των υπάρχοντων Τυπικών και Προσαρμοσμένων αντικειμένων (τόσο ενεργών όσο και ανενεργών).

<VimeoEmbed videoId="926288174" title="Video demonstration" />

3. Κάντε κλικ στο `+ Νέο αντικείμενο` στην κορυφή. Εισάγετε το όνομα (τόσο ενικό όσο και πληθυντικό), επιλέξτε ένα εικονίδιο και προσθέστε περιγραφή για το προσαρμοσμένο σας αντικείμενο και κάντε κλικ στο Αποθήκευση (στην κορυφή δεξιά). Χρησιμοποιώντας το καταχώρηση ως παράδειγμα προσαρμοσμένου αντικειμένου, το ενικό θα ήταν "καταχώρηση" και ο πληθυντικός θα ήταν "καταχωρήσεις" μαζί με μια περιγραφή όπως "Καταχωρήσεις που οι οικοδεσπότες δημιούργησαν για να αναδείξουν το περιουσιακό στοιχείο τους."

<Warning>
Τα ονόματα στο ενικό και πληθυντικό πρέπει να είναι διαφορετικά. Αυτό απαιτείται για να λειτουργεί σωστά το API GraphQL μας.
</Warning>

<VimeoEmbed videoId="926293493" title="Video demonstration" />

4. Μόλις δημιουργήσετε το προσαρμοσμένο σας αντικείμενο, θα μπορείτε να το διαχειριστείτε. Μπορείτε να επεξεργαστείτε το όνομα, το εικονίδιο και την περιγραφή, να δείτε τα διάφορα πεδία και να προσθέσετε περισσότερα πεδία.

<img src="/images/user-guide/objects/customize-fields.png"style={{width:'100%'}}/>

**Σημείωση:** Αν δεν είστε σίγουροι αν χρειάζεται νέο αντικείμενο ή πεδίο, ελέγξτε [αυτό το άρθρο](/l/el/user-guide/data-model/customize-your-data-model) για καθοδήγηση σχετικά με το σχεδιασμό του μοντέλου δεδομένων σας.
