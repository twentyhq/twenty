## Διαχείριση Αντικειμένων

<AccordionGroup>
<Accordion title="Can I reorder objects in the left navigation bar?">
Όχι ακόμα. Η ταξινόμηση αντικειμένων στη γραμμή πλοήγησης είναι προς το παρόν σταθερή, αλλά αυτή η δυνατότητα είναι προγραμματισμένη για μελλοντική έκδοση.
</Accordion>

<Accordion title="Can I hide objects from the left navigation bar?">
Όλα τα ενεργά αντικείμενα εμφανίζονται στη γραμμή πλοήγησης. Μπορείτε να απενεργοποιήσετε τα αντικείμενα που δεν χρειάζεστε στο **Ρυθμίσεις → Μοντέλο Δεδομένων**.
</Accordion>

<Accordion title="Can I delete standard objects (People, Companies, etc.)?">
You can deactivate any standard objects but you cannot hard delete them.
</Accordion>
</AccordionGroup>

## Δυνατότητες Πεδίου

<AccordionGroup>
<Accordion title="Can I create formula fields?">
Τα πεδία φόρμουλας θα είναι διαθέσιμα το **Q1 2026**. Εν τω μεταξύ, μπορείτε να χρησιμοποιήσετε ροές εργασίας για να υπολογίσετε και να ενημερώσετε αυτόματα τις τιμές των πεδίων.
</Accordion>

<Accordion title="Can I have nested fields in my objects?">
Τα ενσωματωμένα πεδία θα είναι διαθέσιμα το **Q1 2026**. Προς το παρόν, μπορείτε να χρησιμοποιήσετε ροές εργασίας για να φέρετε τιμές πεδίου από συσχετισμένα αντικείμενα. Για παράδειγμα, για να εμφανίσετε τον κλάδο μιας εταιρείας σε μια καταχώριση «Άνθρωπος», δημιουργήστε ένα προσαρμοσμένο πεδίο στους Ανθρώπους και χρησιμοποιήστε μια ροή εργασίας για να συγχρονίσετε την τιμή.
</Accordion>

<Accordion title="Why can't I update relation field names?">
Τα ονόματα πεδίων σχέσης επηρεάζουν τη δομή του API και δεν μπορούν να αλλάξουν μετά τη δημιουργία. Εάν χρειάζεστε αλλαγή ονόματος σε πεδίο σχέσης, θα πρέπει να δημιουργήσετε ένα νέο και να διαγράψετε το παλιό.
</Accordion>

<Accordion title="Why do I need different singular and plural names?">
Our GraphQL API uses both forms for different operations:
- `createPerson` (singular) for single record actions
- `createPeople` (plural) for bulk operations

Αυτό δημιουργεί περιορισμούς όταν οι μορφές ενικού και πληθυντικού είναι ίδιες, αλλά βελτιώνει την εμπειρία του προγραμματιστή. </Accordion>

<Accordion title="Why are some field names protected?">
Ορισμένα ονόματα πεδίων όπως "Τύπος" ή "Εφαρμογή" είναι κρατημένα για χρήση από το σύστημα. Επιλέξτε εναλλακτικά ονόματα όπως "Κατηγορία" ή "Ταξινόμηση".
</Accordion>
</AccordionGroup>

## Προηγμένες Δυνατότητες

<AccordionGroup>
<Accordion title="Can I create many-to-many relationships?">
Οι σχέσεις πολλών-με-πολλούς θα είναι διαθέσιμες το **Q1 2026**. Προς το παρόν, χρησιμοποιήστε ένα ενδιάμεσο αντικείμενο με δύο σχέσεις 1-to-many ως προσωρινή λύση.
</Accordion>

<Accordion title="What are Morph Many relationships?">
Οι σχέσεις Morph Many επιτρέπουν σε ένα αντικείμενο να σχετίζεται με διαφορετικούς τύπους αντικειμένων. Για παράδειγμα, μια Ευκαιρία θα μπορούσε να σχετίζεται είτε με έναν Άνθρωπο είτε με μια Εταιρεία. Αυτή η λειτουργία είναι τώρα διαθέσιμη.
</Accordion>

<Accordion title="Can I reorder fields in objects?">
Η αλλαγή σειράς πεδίων θα είναι διαθέσιμη με προσαρμοσμένες διατάξεις το **Q4 2025**. Προς το παρόν, τα πεδία εμφανίζονται με αλφαβητική σειρά.
</Accordion>
</AccordionGroup>

## Πρόσβαση και Άδειες

<AccordionGroup>
<Accordion title="Where can I see and edit my data model?">
You can access your Data Model under **Settings → Data Model**.
</Accordion>

<Accordion title="Why can't I see the Data Model under Settings?">
Επικοινωνήστε με τον διαχειριστή του χώρου εργασίας σας. Η πρόσβαση στο μοντέλο δεδομένων είναι συνήθως περιορισμένη μόνο στους διαχειριστές.
</Accordion>

<Accordion title="How many custom objects or fields can I create?">
You can create as many custom objects and fields as you need - the price won't change.
</Accordion>
</AccordionGroup>
