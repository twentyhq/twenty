## Αναφορά Σφαλμάτων

Για να αναφέρετε ένα σφάλμα, παρακαλούμε [δημιουργήστε ένα θέμα στο GitHub](https://github.com/twentyhq/twenty/issues/new).

Μπορείτε επίσης να ζητήσετε βοήθεια στο [Discord](https://discord.gg/cx5n4Jzs57).

## Αιτήματα Λειτουργιών

Αν δεν είστε σίγουροι αν πρόκειται για σφάλμα και αισθάνεστε ότι είναι πιο κοντά σε αίτημα λειτουργίας, τότε πιθανότατα θα πρέπει [να ανοίξετε μια συζήτηση αντ' αυτού](https://github.com/twentyhq/twenty/discussions/new).
