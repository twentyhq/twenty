Feature flags are used to hide experimental features. Για το Twenty, είναι ρυθμισμένες σε επίπεδο χώρου εργασίας και όχι σε επίπεδο χρήστη.

## Adding a new feature flag

In `FeatureFlagKey.ts` add the feature flag:

```ts
type FeatureFlagKey =
  | 'IS_FEATURENAME_ENABLED'
  | ...;
```

Προσθέστε την επίσης στο enum στο `feature-flag.entity.ts`:

```ts
enum FeatureFlagKeys {
    IsFeatureNameEnabled = 'IS_FEATURENAME_ENABLED',
    ...
}
```

To apply a feature flag on a **backend** feature use:

```ts
@Gate({
  featureFlag: 'IS_FEATURENAME_ENABLED',
})
```

To apply a feature flag on a **frontend** feature use:

```ts
const isFeatureNameEnabled = useIsFeatureEnabled('IS_FEATURENAME_ENABLED');
```

## Configure feature flags for the deployment

Αλλαγή της αντίστοιχης εγγραφής στον Πίνακα `core.featureFlag`:

| ταυτότητα | κλειδί                   | workspaceId | αξία     |
| --------- | ------------------------ | ----------- | -------- |
| Τυχαίο    | `IS_FEATURENAME_ENABLED` | WorkspaceID | `αληθής` |
