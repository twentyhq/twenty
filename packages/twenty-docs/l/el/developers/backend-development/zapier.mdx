Συγχρονίστε εύκολα το Twenty με 3000+ εφαρμογές χρησιμοποιώντας το [Zapier](https://zapier.com/). Αυτοματοποιήστε τις εργασίες, αυξήστε την παραγωγικότητα και ενισχύστε τις σχέσεις με τους πελάτες σας!

## Σχετικά με το Zapier

Το Zapier είναι ένα εργαλείο που σας επιτρέπει να αυτοματοποιείτε ροές εργασίας συνδέοντας τις εφαρμογές που χρησιμοποιεί κάθε μέρα η ομάδα σας. Η θεμελιώδης έννοια του Zapier είναι οι ροές εργασίας αυτοματοποίησης, που ονομάζονται Zaps, και περιλαμβάνουν ενεργοποιητές και ενέργειες.

Μπορείτε να μάθετε περισσότερα σχετικά με το πώς λειτουργεί το Zapier [εδώ](https://zapier.com/how-it-works).

## Ρύθμιση

### Βήμα 1: Εγκατάσταση πακέτων Zapier

```bash
cd packages/twenty-zapier

yarn
```

### Βήμα 2: Είσοδος με το CLI

Χρησιμοποιήστε τα διαπιστευτήρια Zapier για να συνδεθείτε χρησιμοποιώντας το CLI:

```bash
zapier login
```

### Βήμα 3: Ορισμός περιβαλλοντικών μεταβλητών

Από το φάκελο `packages/twenty-zapier`, εκτελέστε:

```bash
cp .env.example .env
```

Εκτελέστε την εφαρμογή τοπικά, πηγαίνετε στο [http://localhost:3000/settings/api-webhooks](http://localhost:3000/settings/api-webhooks), και δημιουργήστε ένα API κλειδί.

Αντικαταστήστε την τιμή **YOUR_API_KEY** στο αρχείο `.env` με το API κλειδί που μόλις δημιουργήσατε.

## Ανάπτυξη

<Warning>

Βεβαιωθείτε ότι εκτελέσατε `yarn build` πριν από οποιαδήποτε εντολή `zapier`.

</Warning>

### Δοκιμή

```bash
yarn test
```

### Έλεγχος

```bash
yarn format
```

### Παρακολουθήστε και συντάξτε καθώς επεξεργάζεστε τον κώδικα

```bash
yarn watch
```

### Επικυρώστε την εφαρμογή Zapier σας

```bash
yarn validate
```

### Αναπτύξτε την εφαρμογή Zapier σας

```bash
yarn deploy
```

### Λίστα όλων των εντολών Zapier CLI

```bash
zapier
```
