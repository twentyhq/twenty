Τα αντικείμενα είναι δομές που σας επιτρέπουν να αποθηκεύετε δεδομένα (εγγραφές, χαρακτηριστικά και τιμές) ειδικά για έναν οργανισμό. Twenty provides both standard and custom objects.

Τα στάνταρ αντικείμενα είναι ενσωματωμένα αντικείμενα με ένα σύνολο χαρακτηριστικών διαθέσιμο για όλους τους χρήστες. Παραδείγματα στάνταρ αντικειμένων στο Twenty περιλαμβάνουν την Εταιρεία και το Πρόσωπο. Τα στάνταρ αντικείμενα έχουν στάνταρ πεδία που είναι επίσης διαθέσιμα για όλους τους χρήστες του Twenty, όπως το Company.displayName.

Τα προσαρμοσμένα αντικείμενα είναι αντικείμενα που μπορείτε να δημιουργήσετε για να αποθηκεύετε πληροφορίες που είναι μοναδικές για τον οργανισμό σας. Δεν είναι ενσωματωμένα· τα μέλη της ομάδας σας μπορούν να δημιουργούν και να προσαρμόζουν προσαρμοσμένα αντικείμενα για να αποθηκεύουν πληροφορίες που τα στάνταρ αντικείμενα δεν είναι κατάλληλα για.

## Πώς Λειτουργεί

Τα προσαρμοσμένα αντικείμενα προέρχονται από πίνακες μεταδεδομένων που καθορίζουν το σχήμα, το όνομα και τον τύπο των αντικειμένων. Όλες αυτές οι πληροφορίες υπάρχουν στη βάση σχήματος μεταδεδομένων, που αποτελείται από πίνακες:

- **DataSource**: Λεπτομέρειες πού βρίσκονται τα δεδομένα.
- **Object**: Περιγράφει το αντικείμενο και συνδέεται με ένα DataSource.
- **Field**: Περιγράφει τα πεδία ενός Αντικειμένου και συνδέεται με το Αντικείμενο.

Για να προσθέσετε ένα προσαρμοσμένο αντικείμενο, το μέλος της ομάδας εργασίας σας θα πρέπει να κάνει ένα ερώτημα στο API /metadata. Αυτό ενημερώνει τα μεταδεδομένα ανάλογα και υπολογίζει ένα σχήμα GraphQL βάσει των μεταδεδομένων, αποθηκεύοντάς το σε προσωρινή μνήμη GQL για μελλοντική χρήση.
