╬Ъ╬▒╧Д╬м ╧Д╬╖╬╜ ╬╡╬│╬║╬▒╧Д╬м╧Г╧Д╬▒╧Г╬╖ ╧Д╬┐╧Е Twenty ╬│╬╣╬▒ ╧А╧Б╧О╧Д╬╖ ╧Ж╬┐╧Б╬м, ╬╝╧А╬┐╧Б╬╡╬п ╬╜╬▒ ╬╕╬н╬╗╬╡╧Д╬╡ ╬╜╬▒ ╬▒╬╗╬╗╬м╬╛╬╡╧Д╬╡ ╧Д╬┐╬╜ ╧А╧Б╬┐╬╡╧А╬╣╬╗╬╡╬│╬╝╬н╬╜╬┐ ╬║╧Й╬┤╬╣╬║╧М ╧А╧Б╧М╧Г╬▓╬▒╧Г╬╖╧В ╧Д╬╖╧В ╬▓╬м╧Г╬╖╧В ╬┤╬╡╬┤╬┐╬╝╬н╬╜╧Й╬╜.
╬Я ╬║╧Й╬┤╬╣╬║╧М╧В ╧А╬┐╧Е ╬┐╧Б╬п╬╢╬╡╧Д╬╡ ╬║╬▒╧Д╬м ╧Д╬╖╬╜ ╧А╧Б╧О╧Д╬╖ ╬╡╬│╬║╬▒╧Д╬м╧Г╧Д╬▒╧Г╬╖ ╬▒╧А╬┐╬╕╬╖╬║╬╡╧Н╬╡╧Д╬▒╬╣ ╬╝╧М╬╜╬╣╬╝╬▒ ╧Г╧Д╬┐╬╜ ╧М╬│╬║╬┐ ╧Д╬╖╧В ╬▓╬м╧Г╬╖╧В ╬┤╬╡╬┤╬┐╬╝╬н╬╜╧Й╬╜. ╬С╬╜ ╧А╧Б╬┐╧Г╧А╬▒╬╕╬о╧Г╬╡╧Д╬╡ ╬▒╧Б╬│╧М╧Д╬╡╧Б╬▒ ╬╜╬▒ ╬▒╬╗╬╗╬м╬╛╬╡╧Д╬╡ ╬▒╧Е╧Д╧М╬╜ ╧Д╬┐╬╜ ╬║╧Й╬┤╬╣╬║╧М ╧Г╧Д╬╖╬╜ ╧А╬▒╧Б╬▒╬╝╬╡╧Д╧Б╬┐╧А╬┐╬п╬╖╧Г╬╖ ╧Г╬▒╧В ╧З╧Й╧Б╬п╧В ╬╜╬▒ ╬▒╧Ж╬▒╬╣╧Б╬н╧Г╬╡╧Д╬╡ ╧Д╬┐ ╧А╬▒╬╗╬╣╧М volume, ╬╕╬▒ ╬╗╬м╬▓╬╡╧Д╬╡ ╧Г╧Ж╬м╬╗╬╝╬▒╧Д╬▒ ╧А╬╣╧Г╧Д╬┐╧А╬┐╬п╬╖╧Г╬╖╧В ╬╡╧А╬╡╬╣╬┤╬о ╬╖ ╬▓╬м╧Г╬╖ ╬┤╬╡╬┤╬┐╬╝╬н╬╜╧Й╬╜ ╬╡╬╛╬▒╬║╬┐╬╗╬┐╧Е╬╕╬╡╬п ╬╜╬▒ ╧З╧Б╬╖╧Г╬╣╬╝╬┐╧А╬┐╬╣╬╡╬п ╧Д╬┐╬╜ ╬▒╧Б╧З╬╣╬║╧М ╬║╧Й╬┤╬╣╬║╧М.

тЪая╕П ╬а╬б╬Я╬Х╬Щ╬Ф╬Я╬а╬Я╬Щ╬Ч╬г╬Ч: ╬д╬▒ ╬▒╬║╧М╬╗╬┐╧Е╬╕╬▒ ╬▓╬о╬╝╬▒╧Д╬▒ ╬Ш╬С ╬Ф╬Щ╬С╬У╬б╬С╬и╬Я╬е╬Э ╬Я╬Ы╬С ╧Д╬▒ ╬┤╬╡╬┤╬┐╬╝╬н╬╜╬▒ ╧Д╬╖╧В ╬▓╬м╧Г╬╖╧В! тЪая╕П

#### ╬С╧А╬┐╧Е╧Г╬п╬▒ ╬╝╬╡╧Д╬▒╬┤╬╡╬┤╬┐╬╝╬н╬╜╧Й╬╜ (schema)

╬С╬╜ ╬┤╬╡╬╜ ╧Д╬┐ ╬║╬м╬╜╬╡╧Д╬╡, ╬▓╬╡╬▓╬▒╬╣╧Й╬╕╬╡╬п╧Д╬╡ ╧М╧Д╬╣ ╬┤╬╡╬╜ ╬н╧З╬╡╧Д╬╡ ╧А╬╡╧Б╬╣╧Г╧Г╧М╧Д╬╡╧Б╬╡╧В ╬▒╧А╧М ╬╝╬п╬▒ ╧А╬╡╧Б╬╣╧А╧Д╧О╧Г╬╡╬╣╧В PostgreSQL ╬╜╬▒ ╧Д╧Б╬н╧З╬┐╧Е╬╜ ╧Г╧Д╬┐╬╜ ╧Е╧А╬┐╬╗╬┐╬│╬╣╧Г╧Д╬о ╧Г╬▒╧В.

#### ╬С╧А╬┐╧Е╧Г╬п╬▒ ╧А╬▒╬║╬н╧Д╬┐╧Е twenty-x

#### ╬Ъ╬▒╧Д╬м ╧Д╬╖ ╬╗╬╡╬╣╧Д╬┐╧Е╧Б╬│╬п╬▒ `npx nx start` ╬о `npx nx start twenty-front`, ╬╡╬╝╧Ж╬▒╬╜╬п╬╢╬╡╧Д╬▒╬╣ ╧Г╧Ж╬м╬╗╬╝╬▒ ╬Х╬╛╬▒╬╣╧Д╬п╬▒╧В ╬н╬╗╬╗╬╡╬╣╧И╬╖╧В ╬╝╬╜╬о╬╝╬╖╧В
