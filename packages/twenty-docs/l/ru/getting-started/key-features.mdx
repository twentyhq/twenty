Twenty — это полнофункциональная платформа CRM. Вот что вы можете с ней создать.

<CardGroup cols={2}>
  <Card href="/l/ru/user-guide/data-model/overview" img="/images/user-guide/halftone/data-model.png">
    <CardTitle>Настраиваемая модель данных</CardTitle>
    Определяйте точную структуру данных, которая нужна вашему бизнесу.
    <br />Создавайте пользовательские объекты, добавляйте настраиваемые поля с более чем 20 типами полей и выстраивайте связи между любыми объектами.
  </Card>

  <Card href="/l/ru/user-guide/views-pipelines/overview" img="/images/user-guide/halftone/layout.png">
    <CardTitle>Представления и воронки</CardTitle>
    Переключайтесь между табличными представлениями, досками канбан и календарными представлениями.
    <br />Фильтруйте с логикой AND/OR, сортируйте по нескольким полям, группируйте записи и сохраняйте пользовательские представления.
  </Card>

  <Card href="/l/ru/user-guide/workflows/overview" img="/images/user-guide/halftone/workflows.png">
    <CardTitle>Рабочие процессы и автоматизация</CardTitle>
    Автоматизируйте любые бизнес-процессы без написания кода.
    <br />Запускайте рабочие процессы при изменении записей, по расписанию, вручную или при получении входящих webhooks.
  </Card>

  <Card href="/l/ru/user-guide/calendar-emails/overview" img="/images/user-guide/halftone/calendar-emails.png">
    <CardTitle>Синхронизация календаря и электронной почты</CardTitle>
    Подключите свою учетную запись Google Workspace или Microsoft 365.
    <br />Письма и события календаря автоматически появляются в соответствующих записях CRM.
  </Card>

  <Card href="/l/ru/user-guide/ai/overview" img="/images/user-guide/halftone/ai.png">
    <CardTitle>ИИ</CardTitle>
    Автономные ИИ-агенты, работающие внутри вашей CRM — отвечают на вопросы, обогащают записи и выполняют многошаговые задачи в рамках вашей модели разрешений.
  </Card>

  <Card href="/l/ru/user-guide/dashboards/overview" img="/images/user-guide/halftone/dashboards.png">
    <CardTitle>Панели и отчеты</CardTitle>
    Создавайте настраиваемые панели с виджетами в реальном времени.
    <br />Отслеживайте метрики воронки, эффективность команды и бизнес-KPI с помощью настраиваемых диаграмм и фильтров.
  </Card>

  <Card href="/l/ru/user-guide/permissions-access/overview" img="/images/user-guide/halftone/permissions.png">
    <CardTitle>Права и контроль доступа</CardTitle>
    Ролевое управление доступом на каждом уровне — объекты, поля и отдельные записи.
    <br />Настройте SSO с SAML или OIDC. Журналы аудита отслеживают, кто что сделал.
  </Card>

  <Card href="/l/ru/developers/extend/api" img="/images/user-guide/halftone/dev-api.png">
    <CardTitle>API и расширяемость</CardTitle>
    Ориентированный на разработчиков API, который адаптируется к вашей настраиваемой модели данных.
    <br />И GraphQL, и REST endpoints с автоматически генерируемой документацией для каждого workspace.
  </Card>

  <Card href="/l/ru/user-guide/data-migration/overview" img="/images/user-guide/halftone/data-migration.png">
    <CardTitle>Импорт и экспорт данных</CardTitle>
    Импортируйте данные из файлов CSV или через API.
    <br />Встроены сопоставление полей, обнаружение дубликатов и обработка ошибок. Экспортируйте свои данные в любой момент — без ограничения поставщиком.
  </Card>

  <Card href="/l/ru/developers/self-host/self-host" img="/images/user-guide/halftone/dev-self-host.png">
    <CardTitle>Самостоятельный хостинг</CardTitle>
    Запустите Twenty в своей инфраструктуре одной командой Docker Compose.
    <br />Полный контроль над вашими данными, обновления по вашему графику.
  </Card>
</CardGroup>
