## Основная компоновка

В центре экрана находятся ваши записи — люди, компании, сделки, задачи, заметки, дашборды, рабочие процессы и любые настраиваемые объекты. Здесь вы просматриваете, изменяете и удаляете записи, а также создаете новые представления.

<img src="/images/user-guide/home/main-layout.png" style={{width:'100%'}} />

## Панель навигации

Левая боковая панель содержит:

* **Переключатель рабочих пространств** — переключайтесь между рабочими пространствами или создавайте новое (выпадающий список вверху)
* **Поиск** — нажмите `/`, чтобы сразу перейти к нему; поиск выполняется по всем объектам
* **Настройки** — доступ из левого верхнего угла
* **Избранное** — закрепленные представления, уникальные для каждого пользователя
* **Ярлыки объектов** — быстрый доступ к людям, компаниям, сделкам и т. д.
* **Рабочие процессы** — создавайте автоматизации

Перетаскивайте элементы, чтобы изменить порядок; создавайте папки для группировки связанных объектов; скрывайте то, чем не пользуетесь.

<img src="/images/user-guide/home/navigation-bar.png" style={{width:'100%'}} />

## Меню команд

Нажмите `Cmd+K` (Mac) или `Ctrl+K` (Windows) — либо кликните по трем точкам в правом верхнем углу. Отсюда вы можете:

* Создавать новые записи
* Импортировать и экспортировать данные в формате CSV
* Создавать новые представления
* Получать доступ к удалённым записям (в Twenty поддерживаются мягкое и жёсткое удаление)
* Просматривать сочетания клавиш для навигации по рабочему пространству

<img src="/images/user-guide/home/command-menu.png" style={{width:'100%'}} />

## Поиск

Доступно через меню команд, в верхней части панели навигации или по нажатию `/`. Поиск работает по всем объектам.

<img src="/images/user-guide/home/search-bar.png" style={{width:'100%'}} />

## Боковая панель

Щелкните по записи, чтобы открыть боковую панель справа — краткий обзор ключевой информации о записи без перехода на другую страницу. Нажмите **Открыть**, чтобы перейти на полную страницу записи.

<img src="/images/user-guide/home/side-panel.png" style={{width:'100%'}} />

## Представления

Каждый объект поддерживает несколько представлений — без ограничений для каждого объекта. Используйте выпадающий список в левом верхнем углу, чтобы переключаться между ними.

* **Таблица** — строки и столбцы в формате электронных таблиц с группировкой, встроенным редактированием и настройкой столбцов
* **Канбан** — перетаскиваемые карточки, организованные по полю выбора, идеально для воронок
* **Календарь** — записи, отображаемые по полю даты для планирования во времени

Каждое представление сохраняет свои собственные фильтры, сортировку и видимость полей. Делитесь представлениями со своим рабочим пространством или оставляйте их личными. Добавляйте представления в избранное для быстрого доступа из боковой панели.

<img src="/images/user-guide/home/view-menu.png" style={{width:'100%'}} />

## Страницы записей

Когда вы открываете запись, страница сведений формируется из настраиваемых **вкладок** и **виджетов**. Добавляйте, удаляйте, меняйте порядок и изменяйте размер виджетов в сетке — поля, связанные записи, электронные письма, ленту активности, задачи, заметки, файлы, диаграммы, iframes и многое другое. У каждого типа объекта есть собственная компоновка.
