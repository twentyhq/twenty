Рабочие процессы позволяют автоматизировать повторяющиеся задачи и подключать Twenty к внешним инструментам — без написания кода (хотя при желании вы можете его использовать).

## Как работают рабочие процессы

У каждого рабочего процесса есть три части:

1. **Триггер** — что запускает рабочий процесс
2. **Шаги** — что происходит дальше (одно или несколько действий последовательно)
3. **Переменные** — данные, которые передаются между шагами

## Триггеры

| Триггер            | Когда срабатывает                                                                 |
| ------------------ | --------------------------------------------------------------------------------- |
| **Событие записи** | Запись создается, обновляется, удаляется или вставляется/обновляется              |
| **Вручную**        | Пользователь нажимает кнопку (для одной записи, нескольких записей или глобально) |
| **Расписание**     | Через регулярные интервалы (синтаксис cron)                                       |
| **Вебхук**         | Внешняя система отправляет HTTP POST                                              |

## Действия

Рабочие процессы могут объединять в цепочку любую комбинацию следующих элементов:

* **Операции с записями** — создание, обновление, поиск, удаление или вставка/обновление записей
* **Отправка писем** — отправка или создание черновиков писем из подключенных учетных записей
* **HTTP-запрос** — вызов любого внешнего API
* **Код** — запуск пользовательского JavaScript для сложной логики
* **Ветвления** — условия if/else для разветвления пути рабочего процесса
* **Iterator** — перебор массивов данных
* **Агент ИИ** — позволяет агенту ИИ автономно обрабатывать данные
* **Задержка** — ожидание перед продолжением
* **Форма** — сбор пользовательского ввода в середине рабочего процесса

## Что вы можете создать

* Отправляйте оповещения в Slack, когда сделка достигает определенного этапа
* Автоматически обогащайте новые контакты данными из внешних API
* Выявляйте зависшие сделки и уведомляйте их владельца
* Синхронизируйте данные между Twenty и вашей биллинговой системой
* Генерируйте PDF или счета на основе данных записей
* Автоматически отвечайте на входящие письма, соответствующие определенным критериям
