Twenty подключается к вашим существующим инструментам, поэтому ваш CRM всегда остается актуальным без ручного ввода данных.

## Синхронизация электронной почты

Подключите свой почтовый ящик **Google Workspace** или **Microsoft 365**. После подключения:

* Электронные письма автоматически привязываются к соответствующим записям Компаний и Людей
* Полные цепочки писем отображаются на временной шкале каждой записи
* Вы можете отправлять письма непосредственно из Twenty
* Поддерживается несколько почтовых ящиков на одного пользователя

Вы контролируете, что импортируется — отфильтруйте по диапазону дат или отправителю, чтобы не подтягивать неактуальные письма.

## Синхронизация календаря

События календаря автоматически синхронизируются из подключенной учетной записи. События отображаются в соответствующих записях CRM, давая вам полное представление о ваших взаимодействиях с каждым контактом или компанией.

## Интеграции

Помимо электронной почты и календаря, Twenty подключается к внешним инструментам через:

| Метод                                 | Сценарий использования                                                              |
| ------------------------------------- | ----------------------------------------------------------------------------------- |
| **API**                               | Создавайте собственные интеграции с помощью GraphQL или REST API                    |
| **Вебхуки**                           | Отправляйте уведомления в реальном времени во внешние системы при изменении записей |
| **Zapier**                            | Подключайте более 5 000 приложений без кода                                         |
| **Действия HTTP в рабочих процессах** | Вызывайте любой внешний API как часть автоматизированного рабочего процесса         |

## Пользовательские приложения

Разработчики могут создавать полнофункциональные приложения поверх Twenty — добавляя пользовательский интерфейс, серверную логику и глубокие интеграции. Приложения можно публиковать для сообщества или оставлять приватными.
