## Invite New Members

### Using the Invite Link

1. Go to **Settings → Members**
2. Скопируйте ссылку приглашения в рабочее пространство
3. Поделитесь ссылкой с новыми участниками команды
4. Они получат доступ после регистрации

### Прямое Приглашение по Электронной Почте

1. Go to **Settings → Members**
2. Enter the person's email address
3. Нажмите **Пригласить**
4. Они получат приглашение по электронной почте

## Удалить Участников

### Удалить Участника

1. Go to **Settings → Members**
2. Найдите участника, которого хотите удалить
3. Нажмите кнопку удаления рядом с именем
4. Подтвердите удаление

**Примечание**: Удаленные участники сразу теряют доступ, но их можно пригласить снова позже.
