## Последние выпуски

Следите за развитием Twenty:

- **Примечания к выпускам**: Узнайте, что нового в каждой версии
- **Обновления функций**: Откройте для себя новые возможности и улучшения

## Функции Лаборатории

Получите доступ к экспериментальным функциям до их официального релиза:

- **Бета-тестирование**: Испытайте новые функции в процессе их разработки
- **Флаги функций**: Включайте или отключайте конкретные экспериментальные функции

## Как получить доступ

1. Перейдите в **Настройки → Выпуски**
2. Просмотрите примечания к выпуску и перейдите на вкладку **Лаб** для бета-функций

Для получения дополнительных ресурсов сообщества ознакомьтесь с нашим [Разделом ресурсов](/l/ru/user-guide/resources/glossary).
