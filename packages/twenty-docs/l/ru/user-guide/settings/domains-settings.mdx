## Пользовательский домен рабочей области

Установите персонализированный веб-адрес для вашей рабочей области:

- Выберите предпочитаемый субдомен (например, yourcompany.twenty.com)
- Настройте пользовательские домены для профессионального брендинга

## Одобренные домены доступа

Разрешите автоматический доступ к рабочей области для определённых почтовых доменов.

### Как это работает

- Добавьте почтовые домены вашей компании (например, @yourcompany.com)
- Каждый с адресом электронной почты из этих доменов может автоматически присоединиться к вашей рабочей области
- Нет необходимости в ручных приглашениях для членов команды

### Преимущества

- **Быстрая адаптация**: Новые члены команды присоединяются автоматически
- **Улучшенная безопасность**: Доступ возможен только для проверенных электронных почт компании
- **Меньше административной работы**: Нет необходимости вручную приглашать каждого члена команды

## Инструкции по настройке

1. Перейдите в **Настройки → Домены**
2. Добавьте ваш пользовательский домен или одобренные почтовые домены
3. Подтвердите владение доменом, если это необходимо

**Примечание**: Изменения в домене могут занять время для вступления в силу и могут потребовать подтверждения.
