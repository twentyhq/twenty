Персонализируйте свой опыт использования Twenty в разделе **Настройки → Опыт**.

## Внешний вид

Выберите визуальную тему:

* **Светлая**: Чистый, светлый интерфейс
* **Тёмная**: Менее напрягает глаза в условиях слабого освещения
* **Системные настройки**: Автоматически соответствует теме вашего устройства

## Язык

Выберите предпочитаемый язык для интерфейса Twenty в раскрывающемся списке.

## Форматы

Настройте дату, время, числовой формат, часовой пояс и первый день недели.

| Настройка              | Описание                                                           |
| ---------------------- | ------------------------------------------------------------------ |
| **Часовой пояс**       | Ваш местный часовой пояс для точных отметок времени и планирования |
| **Формат даты**        | Как отображаются даты (например, 12 дек. 2025)                     |
| **Формат времени**     | 12-часовой (7:22 PM) или 24-часовой формат                         |
| **Числовой формат**    | Разделители десятичной части и тысяч (например, 1,234.56)          |
| **Первый день недели** | Первый день недели (воскресенье или понедельник)                   |

Для каждой настройки можно выбрать **Системные настройки**, чтобы автоматически соответствовать предпочтениям вашего устройства, или указать конкретный формат.

## Как обновить

1. Перейдите в **Настройки → Опыт**
2. Настройте свои предпочтения в каждом разделе
3. Изменения сохраняются автоматически
