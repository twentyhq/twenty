Управляйте тем, кто имеет доступ к вашему рабочему пространству, в разделе **Настройки → Участники**.

## Приглашение новых участников

### Использование приглашения по электронной почте

1. Перейдите в **Настройки → Участники**
2. Нажмите **+ Пригласить**
3. Введите адрес электронной почты получателя
4. Выберите роль для нового участника
5. Нажмите **Отправить приглашение**

Приглашённый получит письмо со ссылкой для присоединения к вашему рабочему пространству.

### Использование ссылки-приглашения

1. Перейдите в **Настройки → Участники**
2. Скопируйте ссылку приглашения в рабочее пространство
3. Поделитесь ссылкой с новыми участниками команды
4. Они получат доступ после регистрации

## Просмотр и управление участниками

### Просмотр всех участников

Перейдите в **Настройки → Участники**, чтобы увидеть:

* Все активные участники
* Ожидающие приглашения

### Редактирование профиля участника

Нажмите на участника, чтобы открыть его страницу профиля. Как администратор, вы можете:

* Изменить **имя**
* Обновить **фотографию профиля**
* **Войти от имени** пользователя (полезно для устранения неполадок)
* **Удалить** учётную запись

### Изменение роли участника

На странице профиля участника:

1. Откройте вкладку **Разрешения**
2. Просмотрите текущую назначенную роль
3. Выберите другую роль из выпадающего списка
4. Изменение вступает в силу немедленно

→ [Подробнее о ролях и разрешениях](/l/ru/user-guide/permissions-access/capabilities/permissions)

### Удаление участника

1. Нажмите на участника, чтобы открыть его профиль
2. Нажмите **Удалить**, чтобы удалить участника из рабочего пространства

<Warning>
  Удалённые участники сразу теряют доступ. Их данные (записи, заметки, задачи) остаются в рабочем пространстве.
</Warning>

<Warning>
  **Синхронизация электронной почты также отключается.** Если удалённый пользователь был единственным, кто синхронизировал определённые письма, эти письма будут безвозвратно удалены из рабочего пространства.
</Warning>

## Ожидающие приглашения

Управляйте приглашениями, которые ещё не приняты:

* **Отправить снова**: отправить письмо-приглашение ещё раз
* **Отменить**: отозвать приглашение до его принятия

## Одобренные домены доступа

Разрешите участникам команды присоединяться автоматически на основе домена их электронной почты:

1. Перейдите в **Настройки → Домены**
2. Добавьте домен вашей компании (например, `yourcompany.com`)
3. Любой с таким доменом электронной почты сможет присоединиться без приглашения
