Сгруппируйте табличное представление по полю Select, чтобы организовать записи в сворачиваемые разделы.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

## Шаги

1. Перейдите к объекту (Люди, Компании и т. д.)
2. Откройте выпадающее меню представления → **+ Добавить представление**
3. Назовите представление (например, "Компании по типу")
4. Нажмите **Создать**
5. Нажмите **Параметры → Группировка**
6. Выберите поле Select для группировки
7. Нажмите **Сохранить**

## Настройка порядка групп

В **Параметры → Группировка → Сортировка** выберите порядок групп:

| Вариант                           | Описание                                                              |
| --------------------------------- | --------------------------------------------------------------------- |
| **По алфавиту**                   | От A до Z                                                             |
| **В обратном алфавитном порядке** | От Z до A                                                             |
| **Ручной порядок**                | Перетащите группы, чтобы изменить порядок, в разделе "Видимые группы" |

Нажмите **значок глаза** рядом с группой, чтобы скрыть её в представлении.

<Note>
  **Для лучшей производительности ограничьте количество видимых групп до 10–15.** Если нужно больше, рассмотрите возможность использования вместо этого панели мониторинга.
</Note>

## Пример: Компании по отрасли

1. Перейдите в **Компании**
2. Создайте новое представление с названием "По отрасли"
3. Нажмите **Параметры → Группировка**
4. Выберите поле **Отрасль**
5. Сохранить

Теперь ваши компании организованы по отраслям, что позволяет легко сосредоточиться на одном сегменте.
