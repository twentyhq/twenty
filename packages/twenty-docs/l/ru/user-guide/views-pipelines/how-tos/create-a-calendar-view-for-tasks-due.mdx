## Требования

В объекте **Tasks** должно быть поле **Due Date** (типа Date или Date & Time).

## Шаги

1. Перейдите в **Tasks**
2. Откройте выпадающее меню представления → **+ Добавить представление**
3. Назовите представление (например, "Календарь задач")
4. Нажмите **Создать**
5. Нажмите **Параметры** и выберите **Календарь** в качестве макета
6. Выберите **Due Date** в качестве поля даты
7. Нажмите **Сохранить**

## Настройка календаря

### Отображение полей на событиях

1. Нажмите **Параметры → Поля**
2. Нажмите **значок глаза**, чтобы показать/скрыть поля
3. Перетащите, чтобы изменить порядок

Рекомендуемые поля для отображения:

* **Title** — название задачи
* **Assignee** — ответственный
* **Status** — текущий прогресс

### Фильтрация календаря

Создавайте целевые представления:

* **Мои задачи**: Фильтр по ответственному = Я
* **Эта неделя**: Фильтр по дате выполнения = Эта неделя
* **Просроченные**: Фильтр по дате выполнения < Сегодня, статус ≠ Выполнено

## Другие сценарии использования календаря

| Объект                   | Поле даты     | Назначение                     |
| ------------------------ | ------------- | ------------------------------ |
| Возможности              | Дата закрытия | Отслеживать ожидаемые закрытия |
| Пользовательские события | Дата события  | Планировать активности         |
| Проекты                  | Крайний срок  | Отслеживать сроки проектов     |

## Советы

* **Проверяйте еженедельно**: Начинайте каждую неделю с проверки представления календаря
* **Совмещайте с табличным представлением**: Используйте календарь для обзора, таблицу — для деталей
* **Настройте видимость**: Держите личные календари задач со статусом «Не в списке»
