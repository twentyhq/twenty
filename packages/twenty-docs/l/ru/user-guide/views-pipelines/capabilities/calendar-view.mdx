## О календарном представлении

Календарное представление отображает ваши записи в календаре на основе поля даты. Каждая запись отображается как событие в соответствующую дату.

<img src="/images/user-guide/views/calendar-view-zoom.png" style={{width:'100%'}} />

## Создание календарного представления

1. Перейдите к объекту с полями даты
2. Откройте выпадающее меню представления → **+ Добавить представление**
3. Назовите представление и нажмите  **Создать**
4. Откройте **Параметры** справа
5. Выберите **Календарь** в качестве макета
6. Выберите **поле даты**, которое будет использоваться для размещения записей
7. Нажмите **Обновить представление**

## Настройка календаря

### Выберите поле даты

В разделе **Параметры** выберите, какое поле даты определяет, где записи отображаются в календаре.

### Отображаемые поля

Настройте, какие поля отображаются в каждом событии календаря:

1. Нажмите **Параметры → Поля**
2. Включайте и отключайте поля
3. Перетащите, чтобы изменить порядок

## Сценарии использования

* **Встречи и звонки**: просматривайте предстоящие встречи
* **Сроки**: отслеживайте даты выполнения и закрытия
* **События**: планируйте и визуализируйте запланированные активности
* **Последующие действия**: смотрите, когда наступает срок выполнения задач
