## Фильтрация данных

Фильтры помогают сосредоточиться на конкретных записях, показывая только те, которые соответствуют вашим критериям.

### Добавление фильтра

1. Нажмите кнопку **Фильтр** на панели инструментов
2. Выберите поле для фильтрации
3. Выберите оператор (равно, содержит и т. д.)
4. Введите значение фильтра
5. Нажмите **Применить**

### Операторы фильтра

| Тип поля | Доступные операторы                                    |
| -------- | ------------------------------------------------------ |
| Текст    | Равно, Содержит, Начинается с, Заканчивается на, Пусто |
| Число    | Равно, Больше чем, Меньше чем, Между, Пусто            |
| Дата     | Равно, До, После, Между, Пусто                         |
| Выбрать  | Равно, Любое из, Пусто                                 |
| Флажок   | Истинно, Ложно                                         |
| Связь    | Равно, Пусто                                           |

### Несколько фильтров

Комбинируйте несколько фильтров, чтобы сузить результаты:

* Все фильтры применяются с логикой И
* Каждый дополнительный фильтр ещё больше сужает результаты

### Удаление фильтров

* Нажмите **X** на отдельных чипах фильтров
* Нажмите **Очистить всё**, чтобы удалить все фильтры

## Сортировка данных

Сортировка определяет порядок отображения записей.

### Добавление сортировки

1. Нажмите кнопку **Сортировка** на панели инструментов
2. Выберите поле для сортировки
3. Выберите по возрастанию (A–Z, 0–9) или по убыванию (Z–A, 9–0)
4. Нажмите **Применить**

### Несколько сортировок

Добавьте несколько уровней сортировки:

* Первая сортировка — основная
* Последующие сортировки применяются внутри групп равных значений

### Быстрая сортировка столбцов

Нажмите заголовок любого столбца, чтобы отсортировать:

* Первое нажатие: по возрастанию
* Второе нажатие: по убыванию
* Третье нажатие: убрать сортировку

## Сохранение настроек фильтров и сортировки

Фильтры и сортировки сохраняются вместе с представлением:

1. Настройте фильтры и сортировки
2. Нажмите **Сохранить**, чтобы обновить текущее представление
3. Или нажмите **Сохранить как новое представление**, чтобы создать вариант
