## Текущие варианты интеграции

### Нативные интеграции

По состоянию на сегодняшний день Twenty предлагает нативную интеграцию с:

- **Почта и Календарь**: Подключите Gmail, Outlook или провайдеров SMTP/CalDAV
- **Доступ к API**: Используйте наши REST и GraphQL API для создания пользовательских интеграций

Для настройки почты и календаря посетите [Настройка Почты и Календаря](/l/ru/user-guide/settings/email-calendar-setup).

### Рабочие процессы (рекомендуется)

Основной способ подключения Twenty к другим инструментам — через нашу функцию **Рабочие процессы**:

- **HTTP Узлы**: Выполняйте API-запросы к внешним службам
- **Кодовые Узлы**: Пишите пользовательскую логику для сложных интеграций
- **Webhook Triggers**: Receive data from external systems
- **Активаторы обновлений полей**: Автоматизируйте действия на основе изменений в CRM

Узнайте больше в нашем [разделе о Рабочих процессах](/l/ru/user-guide/workflows/getting-started-workflows).

### Интеграция с Zapier (устаревшее)

Мы поддерживаем интеграцию с Zapier для пользователей, предпочитающих автоматизацию без кода:

1. Посетите [Twenty на Zapier](https://zapier.com/apps/twenty/integrations)
2. Создайте новый Zap, используя Twenty в качестве триггера или действия
3. Generate an API key in Settings → API & Webhooks
4. Подключите ваше рабочее пространство Twenty к Zapier

## Видение будущего: Коннекторы, созданные сообществом (2026)

### Платформа расширяемости

Мы создаем платформу расширяемости, которая позволит разработчикам создавать приложения в виде кода. Это позволит:

- **Пользовательские Коннекторы**: Создавайте интеграции с вашим любимым программным обеспечением
- **Вклад сообщества**: Делитесь и находите коннекторы, созданные другими
- **Гибкая архитектура**: Расширяйте возможности Twenty за пределы основных функций

### Создание рабочих процессов с помощью искусственного интеллекта

В 2026 будет доступна помощь ИИ, которая поможет пользователям:

- **Автоматическое создание рабочих процессов**: Описывайте свои нужды в интеграции простым языком
- **Умные рекомендации**: Получайте предложения по подключению ваших конкретных инструментов
- **Библиотека шаблонов**: Получите доступ к заранее созданным рабочим процессам для общих случаев

Будущее интеграций Twenty — это ориентированное на сообщество, поддерживаемое ИИ, и бесконечно расширяемое.
