## Обзор

Интеллектуальный помощник, который помогает взаимодействовать с данными вашей CRM на естественном языке.

## Возможности

| Функция                           | Описание                                                                                   |
| --------------------------------- | ------------------------------------------------------------------------------------------ |
| **Запросы на естественном языке** | Задавайте вопросы на простом английском вместо создания фильтров                           |
| **Полный доступ к данным**        | Запрашивайте записи, связи и метрики во всём рабочем пространстве                          |
| **Контекст страницы**             | Ссылайтесь на "эту компанию" или "эту сделку" в зависимости от текущего контекста страницы |
| **Диалоговый режим**              | Последующие вопросы сохраняют контекст предыдущих запросов                                 |

## Примеры взаимодействий

### Поиск записей

* "Покажи все сделки свыше $50,000"
* "Найди контакты, которым я не писал за последние 2 недели"
* "Выведи список компаний в сфере здравоохранения"

### Получение аналитики

* "Какова общая стоимость моей воронки продаж?"
* "Сколько сделок было закрыто в прошлом месяце?"
* "На каком этапе больше всего застрявших сделок?"

### Использование контекста страницы

* "Подытожь мои взаимодействия с этим человеком" (на странице контакта)
* "Какие сделки связаны с этой компанией?" (на странице компании)
* "Когда эта сделка в последний раз была обновлена?" (на странице сделки)
