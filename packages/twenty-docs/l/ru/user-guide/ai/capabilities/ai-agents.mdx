## Обзор

Интегрируйте возможности ИИ напрямую в рабочие процессы автоматизации для интеллектуальной обработки данных и принятия решений.

## Возможности

| Функция                      | Описание                                             |
| ---------------------------- | ---------------------------------------------------- |
| **Действия ИИ**              | Добавляйте шаги на базе ИИ в любой рабочий процесс   |
| **Обогащение данных**        | Автоматически обогащайте записи внешними данными     |
| **Классификация**            | Классифицируйте записи на основе анализа содержимого |
| **Резюмирование**            | Создавайте краткие сводки из текстовых полей         |
| **Пользовательские промпты** | Определяйте, как именно ИИ обрабатывает ваши данные  |

## Сценарии использования

* **Lead scoring**: Automatically score and prioritize inbound leads
* **Data cleanup**: Standardize company names and contact information
* **Email drafts**: Generate follow-up emails based on meeting notes
* **Record routing**: Assign records to the right team member based on content
