## Обзор

Кредиты обеспечивают работу ваших автоматизаций рабочих процессов в Twenty. Каждое действие рабочего процесса потребляет кредиты в зависимости от своей сложности.

## Распределение кредитов

Кредиты привязаны к вашему платёжному циклу, а не к плану:

| Платёжный цикл | Кредиты           |
| -------------- | ----------------- |
| Ежемесячно     | 5 миллионов/месяц |
| Ежегодно       | 50 миллионов/год  |

<Note>
  5 миллионов кредитов в месяц позволяют вам запускать автоматизации, не беспокоясь о расходах. Для большинства рабочих процессов, использующих стандартные действия, этого более чем достаточно. Дополнительные кредиты понадобятся только при запуске продвинутых узлов с кодом или функций на базе ИИ.
</Note>

## Потребление кредитов

Разные действия потребляют разное количество кредитов:

| Тип действия                                               | Использование кредита                      |
| ---------------------------------------------------------- | ------------------------------------------ |
| **Базовые операции** (поиск, обновление, создание записей) | Минимальные                                |
| **Сложные операции** (узлы с кодом, внешние вызовы API)    | Больше кредитов                            |
| **Запросы ИИ** (скоро)                                     | Варьируется в зависимости от использования |

Кредиты списываются в режиме реального времени при выполнении рабочих процессов.

## Отслеживание использования

Отслеживайте своё потребление кредитов:

1. Перейдите в **Настройки → Оплата**
2. Просматривайте текущее использование и остаток кредитов
3. Отслеживайте тенденции, чтобы при необходимости планировать дополнительные кредиты

## Приобретение дополнительных кредитов

Нужно больше кредитов?

1. Перейдите в **Настройки → Оплата**
2. Нажмите на опцию, чтобы приобрести дополнительные пакеты кредитов
3. Выберите нужное количество
