## Обзор

Кредиты обеспечивают работу автоматизаций рабочих процессов и возможностей ИИ. Они расходуются при выполнении действий рабочих процессов, когда агенты ИИ обрабатывают задачи внутри рабочих процессов и в чате ИИ.

## Распределение кредитов

Кредиты привязаны к вашему платёжному циклу, а не к плану:

| Платёжный цикл | Кредиты |
| -------------- | ------- |
| Ежемесячно     | 5/месяц |
| Ежегодно       | 50/год  |

<Note>
5 кредитов в месяц позволяют вам запускать автоматизации, не беспокоясь о расходах. Для большинства рабочих процессов, использующих стандартные действия, этого более чем достаточно. Дополнительные кредиты понадобятся только при запуске продвинутых узлов с кодом или функций на базе ИИ.
</Note>

## Перенос кредитов

Неиспользованные кредиты в конце расчетного периода автоматически переносятся на следующий период.

* **Ограничение**: Сумма переноса ограничена полным объемом за один период, поэтому вы не сможете перенести больше кредитов, чем предусмотрено вашим планом за период.
* **Видимость**: Когда доступны перенесенные кредиты, они отображаются отдельной строкой **Перенесенные кредиты** в **Настройки → Биллинг**, рядом с балансом **Всего доступно**.

## Потребление кредитов

Разные действия потребляют разное количество кредитов:

| Тип действия                                               | Использование кредитов                     |
| ---------------------------------------------------------- | ------------------------------------------ |
| **Базовые операции** (поиск, обновление, создание записей) | Минимальное                                |
| **Сложные операции** (узлы с кодом, внешние вызовы API)    | Больше кредитов                            |
| **ИИ-промпт и ИИ-чаты**                                    | Варьируется в зависимости от использования |

Кредиты списываются в режиме реального времени при выполнении рабочих процессов.

## Отслеживание использования

Отслеживайте своё потребление кредитов:

1. Перейдите в **Настройки → Оплата**
2. Просматривайте текущее использование и остаток кредитов
3. Отслеживайте тенденции, чтобы при необходимости планировать дополнительные кредиты

## Приобретение дополнительных кредитов

Нужно больше кредитов?

1. Перейдите в **Настройки → Оплата**
2. Нажмите на опцию, чтобы приобрести дополнительные пакеты кредитов
3. Выберите нужное количество
