## Краткий обзор

| План                                      | Ключевые функции                                                                                                                                                                                                         |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Бесплатно (самостоятельный хостинг)**   | Все функции Pro, поддержка сообщества                                                                                                                                                                                    |
| **Pro (облако)**                          | Всё, кроме функций Premium (SSO и разрешения на уровне строк), стандартная поддержка                                                                                                                                     |
| **Организация (облако)**                  | Всё из Pro + функции Premium (SSO и разрешения на уровне строк), приоритетная поддержка                                                                                                                                  |
| **Организация (самостоятельный хостинг)** | Всё из Pro + функции Premium (SSO, разрешения на уровне строк), поддержка команды Twenty, не требуется публиковать ваш пользовательский код как программное обеспечение с открытым исходным кодом перед распространением |

## Быстрые ответы

**Где управлять выставлением счетов?**
Перейдите в **Настройки → Выставление счетов**, чтобы просмотреть свой план, обновить способы оплаты и получить доступ к счетам.

**Могу ли я использовать Twenty бесплатно?**
Да! Самостоятельно разверните Twenty и получите все функции Pro бесплатно.

**Как выполнить обновление?**
Перейдите в **Настройки → Выставление счетов** и нажмите **Переключиться на «Организацию»** или **Переключиться на годовой план**.

Подробнее см. в [FAQ по выставлению счетов](/l/ru/user-guide/billing/how-tos/billing-faq).
