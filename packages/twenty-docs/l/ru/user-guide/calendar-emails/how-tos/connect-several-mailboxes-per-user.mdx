## Обзор

Twenty поддерживает **неограниченное количество почтовых аккаунтов на пользователя**. Это полезно, если вы управляете несколькими почтовыми ящиками, например:

* Личная рабочая почта + общий командный почтовый ящик
* Несколько адресов электронной почты для работы с клиентами
* Разные почтовые аккаунты для разных ролей

## Как добавить несколько почтовых ящиков

1. Перейдите в **Настройки → Аккаунты**
2. Нажмите **Добавить аккаунт**
3. Подключите дополнительную учетную запись Google или Microsoft
4. Настройте параметры синхронизации для этого почтового ящика
5. Повторите для каждого почтового ящика, который вы хотите подключить

## Управление несколькими почтовыми аккаунтами

У каждого подключенного почтового ящика есть свои настройки:

* **Видимость электронной почты**: выберите, что смогут видеть участники команды
* **Автоматическое создание контактов**: включать/выключать для каждого почтового ящика
* **Выбор папок**: выберите, какие папки синхронизировать (экспериментальная функция)

## Как отображаются электронные письма

Письма из всех ваших подключенных почтовых ящиков синхронизируются с Twenty и отображаются в:

* **Записи людей**: на основе адреса электронной почты контакта
* **Записи компаний**: на основе домена электронной почты
* **Возможности**: на основе связанной компании

<Note>В каждом письме указано, из какого почтового ящика оно отправлено/в какой получено, поэтому вы можете отслеживать, какой аккаунт использовался для каждого общения.</Note>

## Важные замечания

<Warning>Подключать можно только полноценные почтовые ящики. Псевдонимы электронной почты, которые пересылают в другой почтовый ящик, нельзя подключать отдельно — они будут синхронизироваться через основной почтовый ящик.</Warning>
