## Текущий статус

Интеграция электронной почты Twenty предназначена для того, чтобы **синхронизировать и отображать** вашу историю переписки по электронной почте. Письма нельзя составлять или отправлять непосредственно из интерфейса Twenty.

Когда вы просматриваете цепочку писем на странице записи и нажимаете **Ответить**, вы будете перенаправлены к исходной переписке в своём почтовом ящике (Gmail, Outlook и т. п.). Именно там вы создаёте и отправляете свой ответ.

## Что вы можете делать уже сегодня

* **Просматривать историю переписки** в записях Людей, Компаний и Возможностей
* **Читать полные цепочки писем** с контактами в вашей CRM
* **Отслеживать контекст переписки** вместе с событиями календаря
* **Автоматически создавать контакты** из взаимодействий по электронной почте
* **Отвечать через переадресацию** — нажмите «Ответить», чтобы перейти к своему почтовому ящику

## Отправка писем через рабочие процессы

Хотя вы не можете отправлять письма вручную из Twenty, вы **можете автоматически отправлять письма с помощью рабочих процессов**. Это полезно для:

* Автоматические последующие письма
* Уведомления контактам
* Триггерные коммуникации на основе изменений записей

Письма, отправленные через рабочие процессы, проходят через подключённую учётную запись почтового ящика.

→ Узнайте больше о [действии «Отправить письмо»](/l/ru/user-guide/workflows/capabilities/workflow-actions#send-email)

## Последовательности писем и рассылки

Для последовательностей писем и рассылок мы рекомендуем использовать рабочие процессы, чтобы подключить Twenty к специализированному инструменту email-маркетинга.

<Warning>
  Массовые письма не следует отправлять напрямую из вашего почтового ящика, чтобы защитить репутацию домена. Используйте специализированный инструмент для массовых рассылок.
</Warning>

→ См. [Как отправлять письма из рабочих процессов](/l/ru/user-guide/workflows/capabilities/send-emails-from-workflows) для получения инструкций по настройке
