## Предварительная проверка импорта

После загрузки файла и сопоставления полей Twenty проверяет ваши данные **перед** импортом. Это позволяет обнаруживать и исправлять ошибки, не затрагивая ваши существующие данные.

## Как это работает

1. **Загрузите** ваш CSV-файл
2. **Сопоставьте** столбцы с полями Twenty
3. **Просмотрите** потенциальные ошибки, выделенные желтым
4. **Исправляйте ошибки** прямо в интерфейсе
5. **Подтвердите** импорт

## Отображение ошибок

Строки с ошибками выделены **желтым**. Вы можете:

* **Отредактируйте ячейку напрямую**, чтобы исправить ошибку
* **Удалите строку**, чтобы пропустить её целиком

Такое встроенное редактирование экономит время — не нужно возвращаться к таблице, исправлять ошибки и загружать файл заново.

## Распространённые типы ошибок

### Дублирующиеся значения

**Причина**: значение уникального поля (email, домен) уже есть в Twenty или встречается дважды в вашем файле.

**Решение**:

* Отредактируйте дублирующееся значение в интерфейсе импорта
* Удалите одну из строк-дубликатов

См. [Ограничения уникальности](/l/ru/user-guide/data-migration/capabilities/uniqueness-constraints) для подробностей о том, как обеспечивается уникальность.

### Недопустимый формат

**Причина**: данные не соответствуют ожидаемому формату (например, некорректный email, неверный формат даты).

**Решение**: отредактируйте ячейку, указав правильный формат.

См. [Сопоставление полей](/l/ru/user-guide/data-migration/capabilities/field-mapping), чтобы узнать ожидаемый формат для каждого типа поля.

### Отсутствуют обязательные поля

**Причина**: обязательное поле пусто.

**Решение**: введите значение в обязательное поле или удалите строку.

### Связь не найдена

**Причина**: ссылочная запись не существует (например, домен компании, который не был импортирован).

**Решение**:

* Сначала импортируйте родительские записи
* Или исправьте ссылочное значение

См. [Импорт связей](/l/ru/user-guide/data-migration/capabilities/import-relations) для правильного порядка импорта и способов связывания записей.

## Советы для уменьшения количества ошибок

1. **Скачайте шаблон**, чтобы увидеть ожидаемый формат перед импортом файла
2. **Очистите данные** сначала в таблице
3. **Импортируйте файлы в правильном порядке**, чтобы импортировать связи (Компании → Контакты → Сделки)
4. **Протестируйте на небольших партиях** перед полным импортом
5. **Проверьте наличие дублей** перед загрузкой
6. **Ограничьте размер каждого файла 10 000 записями**
