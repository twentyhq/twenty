## Понимание панелей мониторинга

Панели мониторинга в Twenty обеспечивают визуальный способ отслеживать ключевые показатели эффективности и получать инсайты из данных вашей CRM-системы.

<img src="/images/user-guide/dashboard/dashboards_v2.png" alt="Панели управления" />

## Ключевые понятия

### Панели управления

Панель мониторинга — это набор вкладок, которые позволяют быстро просматривать данные вашей CRM-системы. Вы можете создавать несколько панелей мониторинга для разных целей:

* Эффективность продаж
* Активность команды
* Состояние воронки
* Пользовательские метрики

### Вкладки

Вкладки позволяют разделить панель мониторинга на разделы. Каждая вкладка содержит один или несколько виджетов.

### Виджеты

Виджеты — это отдельные визуализации, которые отображают конкретные данные. К типам относятся:

* Гистограммы
* Круговые диаграммы
* Линейные диаграммы
* Числовые метрики
* iFrames

<Note>
  **Текущие ограничения**:

  * Экспорт панелей мониторинга и предоставление доступа внешним пользователям (не являющимся пользователями Twenty) в настоящее время недоступны.
  * Индикаторные диаграммы и таблицы пока недоступны.
</Note>

## Начало работы

### Создание первой панели мониторинга

1. Перейдите в раздел **Панели мониторинга**
2. Нажмите **+ Новая панель мониторинга**
3. Назовите панель мониторинга
4. Добавьте вкладки, чтобы организовать содержимое
5. Добавьте виджеты для отображения данных
6. Сохранить

### Добавление виджетов

1. Откройте вкладку на панели мониторинга
2. Нажмите **+ Добавить виджет**
3. Выберите тип виджета
4. Выберите источник данных (объект)
5. Настройте параметры виджета
6. Сохраните и просмотрите виджет

## Лучшие практики

* **Начните с простого**: выберите несколько ключевых метрик и со временем добавляйте новые
* **Сосредоточьтесь на данных, по которым можно действовать**: отображайте метрики, влияющие на принятие решений
* **Регулярный обзор**: проверяйте панели мониторинга, чтобы выявлять тенденции
* **Делитесь с командой**: сделайте панели мониторинга видимыми для соответствующих участников команды
