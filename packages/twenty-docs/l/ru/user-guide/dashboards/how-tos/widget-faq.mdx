<AccordionGroup>
  <Accordion title="Как отображать проценты вместо количества?">
    Три варианта:

    * **Процент пустых/непустых**: В разделе Отображаемые данные выберите Процент пустых или Процент непустых
    * **Доля**: Для полей типа Выбор или Флажок включите параметр Доля, чтобы показывать конкретный вариант в процентах от всех записей
    * **Суффикс**: Для агрегированных диаграмм добавьте `%` как суффикс в настройках стиля
  </Accordion>

  <Accordion title="В чём разница между сгруппированными столбцами и столбцами с накоплением?">
    При использовании **Группировки по** можно отображать ряды двумя способами:

    * **Сгруппировано**: Столбцы рядом — лучше для сравнения точных значений
    * **С накоплением**: Столбцы один над другим — лучше для демонстрации состава

    Переключите параметр **С накоплением** в настройках стиля, чтобы переключить режим.
  </Accordion>

  <Accordion title="Почему я не вижу параметры Сумма или Среднее?">
    Доступные операции зависят от **типа поля**:

    * **Числовые поля**: Сумма, Среднее, Минимум, Максимум
    * **Поля даты**: Самая ранняя, Самая поздняя (только для агрегированных диаграмм)
    * **Поля-флажки**: Количество true, Количество false

    Если вы не видите нужный параметр, проверьте, что выбран совместимый тип поля.
  </Accordion>

  <Accordion title="Почему некоторые настройки неактивны или отсутствуют?">
    Многие параметры отображаются только при выполнении предварительных условий:

    * **Детализация даты**: сначала выберите поле с датой
    * **Группировка по, Легенда, С накоплением**: сначала задайте Источник
    * **Переключатели «Легенда» и «С накоплением»**: сначала задайте Группировку по

    Если параметр отсутствует, сначала настройте его предварительное условие.
  </Accordion>

  <Accordion title="Что означает 'Нет данных'?">
    Диаграмма не нашла записей, соответствующих вашей конфигурации.

    **Как исправить:** Удалите или скорректируйте фильтры, убедитесь, что выбран правильный объект Источник, или проверьте, что для вашего диапазона дат существуют записи.
  </Accordion>

  <Accordion title="Что означает 'Недопустимая конфигурация'?">
    Отсутствует обязательный параметр или он несовместим.

    **Как исправить:** Убедитесь, что задан Источник, настроены Отображаемые данные и выбраны все обязательные поля группировки.
  </Accordion>

  <Accordion title="Что означает 'Неотображаемые данные'?">
    Ваши данные превышают пределы отображения (100 столбцов, 100 точек данных или 100 секторов).

    **Как исправить:** Добавьте фильтры, чтобы сузить данные, или измените детализацию даты с дня на неделю или месяц.
  </Accordion>

  <Accordion title="Почему на одной оси нет данных?">
    В выбранном поле нет значений для записей, попадающих под ваш фильтр, или тип поля не соответствует ожидаемым данным.

    **Как исправить:** Попробуйте другое поле или проверьте, что у записей есть значения для выбранного поля.
  </Accordion>
</AccordionGroup>
