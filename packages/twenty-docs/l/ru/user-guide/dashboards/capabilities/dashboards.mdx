## Обзор

Панели мониторинга в Twenty организованы иерархически: **Панели мониторинга → Вкладки → Виджеты**. Каждая панель мониторинга может содержать несколько вкладок, и каждая вкладка включает виджеты (диаграммы, числа, iFrames).

## Создание панели мониторинга

1. Перейдите в раздел **Панели мониторинга** в меню навигации
2. Нажмите **+ Новая панель мониторинга**
3. Назовите панель мониторинга
4. Начните добавлять вкладки и виджеты

## Работа со вкладками

Вкладки помогают организовать панель мониторинга на логические разделы.

### Создание вкладок

1. В режиме редактирования нажмите **+ Добавить вкладку**
2. Назовите вкладку (например, "Обзор воронки", "Результативность команды")
3. Добавьте виджеты на вкладку

### Дублирование вкладок

1. Нажмите на вкладку, которую хотите дублировать
2. Нажмите кнопку **Дублировать** на боковой панели

## Макет панели мониторинга

### Расположение виджетов

* Перетаскивайте, чтобы расположить
* Изменяйте размер, чтобы выделить
* Группируйте связанные диаграммы вместе

### Дублирование панели мониторинга

1. Выйдите из режима редактирования (только режим просмотра)
2. Откройте командную палитру с помощью **Cmd + K** (или **Ctrl + K** в Windows)
3. Выберите **Дублировать панель мониторинга**

### Лучшие практики

* **Логическая последовательность**: располагайте от общего к частному
* **Визуальная иерархия**: используйте более крупные диаграммы для ключевых метрик
* **Единый стиль**: используйте согласованные цвета и шрифты

## Видимость и доступ

### Видимость панели мониторинга

Панели мониторинга видны всем, у кого есть доступ к вашему рабочему пространству Twenty. В настоящее время опции приватной панели мониторинга нет.

### Избранное

Вы можете добавлять панели мониторинга в избранное для быстрого доступа. Это персональная настройка — ваше избранное не видно другим пользователям.

Чтобы добавить панель мониторинга в избранное, откройте панель мониторинга и нажмите значок звезды.

### Работа с часовыми поясами

Панели мониторинга сейчас отображают данные в соответствии с часовым поясом пользователя, который их просматривает. Это означает, что одна и та же панель мониторинга может отображать разные метрики для участников команды в разных регионах (например, APAC и США).

<Note>
  **Скоро**: мы добавим возможность задавать конкретный часовой пояс для панели мониторинга, чтобы все пользователи видели согласованные данные независимо от их местоположения.
</Note>

<Note>
  **Скоро**: фильтры на уровне панели мониторинга позволят применять фильтры ко всем виджетам сразу, что ускорит изучение ваших данных.
</Note>
