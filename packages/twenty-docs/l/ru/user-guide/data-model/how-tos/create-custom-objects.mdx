---
title: Создание пользовательских объектов
description: Пошаговое руководство по созданию пользовательских объектов в Twenty.
---

Пользовательские объекты позволяют хранить информацию, уникальную для вашего бизнеса, которую стандартные объекты не охватывают. Например: проекты, товары, заявки или объявления.

<Note>
  **Не уверены, нужен объект или поле?** См. [Понимание вашей модели данных](/l/ru/user-guide/data-model/overview) для получения рекомендаций.
</Note>

## Шаги

1. Перейдите в **Настройки → Модель данных**
2. Нажмите **+ Новый объект**
3. Заполните:
   * **Имя в единственном числе** (например, "Listing")
   * **Имя во множественном числе** (например, "Listings")
   * **Иконка**
   * **Описание** (необязательно)
4. Нажмите **Сохранить**

Ваш объект сразу появится в боковой панели.

## Далее: добавьте поля

Новые объекты изначально содержат базовые поля. Добавьте пользовательские поля, чтобы собирать нужные данные:

1. В разделе **Настройки → Модель данных** выберите свой объект
2. Нажмите **+ Добавить поле**
3. Выберите тип поля, настройте и сохраните

См. [Как создать пользовательские поля](/l/ru/user-guide/data-model/how-tos/create-custom-fields) для подробностей о типах полей и настройке.

## Связывание с другими объектами

Чтобы связать ваш объект с людьми, компаниями или другими объектами, создайте поле связи. См. [Как создать поля связей](/l/ru/user-guide/data-model/how-tos/create-relation-fields).

## Деактивация объекта

Если объект больше не нужен:

1. Перейдите в **Настройки → Модель данных**
2. Отключите объект

Объект будет скрыт, но данные сохранятся. Позже вы сможете повторно активировать или удалить навсегда.
