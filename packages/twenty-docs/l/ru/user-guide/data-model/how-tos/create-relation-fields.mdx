---
title: Создание полей связи
description: Пошаговое руководство по связыванию объектов с помощью полей связи.
---

Поля связи соединяют записи из разных объектов — например, связывают контакты с компаниями.

<Note>
  **Имена связей нельзя изменить после создания** (они влияют на API). Тщательно продумайте имена.
</Note>

## Перед началом

Решите:

* Какие объекты вы соединяете? (например, Контакты → Компании)
* Какая сторона — «один»? (например, Компания)
* Какая сторона — «многие»? (например, Контакты — много контактов работают в одной компании)
* Как должно называться поле на каждой стороне?

См. [Поля связей](/l/ru/user-guide/data-model/capabilities/relation-fields) — там объясняются типы связей.

## Шаги

1. Перейдите в **Настройки → Модель данных**
2. Выберите объект, где нужна связь (обычно это сторона «многие»)
3. Нажмите **+ Добавить поле**
4. Выберите **Связь** как тип поля
5. Выберите **целевой объект**
6. Выберите **Один-ко-многим** или **Многие-к-одному**
7. Введите названия полей для обеих сторон связи
8. Нажмите **Сохранить**

## Пример: Контакты → Компании

* Перейдите в **Настройки → Модель данных → Контакты**
* Добавьте поле связи
* Целевой объект: **Компании**
* Тип: **Многие-к-одному**
* Поле у Контактов: **Компания**
* Поле у Компаний: **Сотрудники**

Теперь каждый контакт может быть связан с компанией, и каждая компания показывает свои контакты.

## Удаление связи

1. Перейдите в **Настройки → Модель данных**
2. Найдите поле связи
3. Нажмите **⋮ → Деактивировать**

Связи сохраняются, но скрыты. Повторно активируйте, чтобы восстановить.

<Note>
  **Удаление связи не удаляет записи.** Удаляется только связь между ними.
</Note>
