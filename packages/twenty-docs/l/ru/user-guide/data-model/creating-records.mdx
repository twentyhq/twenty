## About Table Views

Таблицы представлений - это визуальные представления данных в виде строк и столбцов.

## Создать запись

Добавляйте записи по мере необходимости, без ограничений. Чтобы добавить запись, нажмите кнопку **+** в верхнем правом углу экрана или в верхней части столбца **Имя** записи.

Введите имя записи, затем нажмите `Enter`, чтобы сохранить. Чтобы изменить имя записи, нажмите на ее название на странице с подробностями.

<VimeoEmbed videoId="927071691" title="Video demonstration" />

## Удалить запись

**Просмотр Индекса:** Чтобы удалить запись, выберите флажок рядом с записью и нажмите кнопку удаления в правом верхнем углу.

**Страница Записи:** Нажмите на иконку `⋮` в правом верхнем углу, а затем выберите удалить.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## Добавить пользовательское поле

Чтобы создать пользовательское поле, нажмите кнопку **+** в правом конце столбцов таблицы и выберите **Настроить поля**.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

Вы также можете сделать это, перейдя в **Настройки** > **Модель данных** > **Люди**. Нажмите на **Добавить поле**. Выберите имя и тип поля. Новое поле будет доступно в приложении.
