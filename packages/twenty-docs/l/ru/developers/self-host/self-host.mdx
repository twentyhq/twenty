## Обзор

Twenty можно развернуть в собственной инфраструктуре, что дает вам полный контроль над данными и процессом развертывания.

## Почему самостоятельный хостинг?

* **Владение данными**: Храните все данные CRM на собственных серверах
* **Соответствие требованиям**: Соблюдайте регуляторные требования к локализации данных
* **Настройка**: Полный доступ для изменения и расширения платформы
