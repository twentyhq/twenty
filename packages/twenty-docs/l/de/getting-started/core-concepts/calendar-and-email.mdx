Twenty verbindet sich mit Ihren bestehenden Tools, damit Ihr CRM ohne manuelle Dateneingabe immer auf dem neuesten Stand bleibt.

## E-Mail-Synchronisierung

Verbinden Sie Ihr **Google Workspace**- oder **Microsoft 365**-Postfach. Sobald verbunden:

* E-Mails werden automatisch mit den passenden Unternehmens- und Personendatensätzen verknüpft
* Vollständige E-Mail-Verläufe sind in der Chronik jedes Datensatzes sichtbar
* Sie können E-Mails direkt aus Twenty senden
* Mehrere Postfächer pro Benutzer werden unterstützt

Sie steuern, was importiert wird – filtern Sie nach Datumsbereich oder Absender, um irrelevante E-Mails zu vermeiden.

## Kalender-Synchronisierung

Kalenderereignisse werden automatisch von Ihrem verbundenen Konto synchronisiert. Ereignisse erscheinen in den entsprechenden CRM-Datensätzen und geben Ihnen ein vollständiges Bild Ihrer Interaktionen mit jedem Kontakt oder Unternehmen.

## Integrationen

Über E-Mail und Kalender hinaus verbindet sich Twenty über Folgendes mit externen Tools:

| Methode                    | Anwendungsfall                                                                        |
| -------------------------- | ------------------------------------------------------------------------------------- |
| **API**                    | Erstellen Sie benutzerdefinierte Integrationen mit der GraphQL- oder REST-API         |
| **Webhooks**               | Senden Sie Echtzeitbenachrichtigungen an externe Systeme, wenn sich Datensätze ändern |
| **Zapier**                 | Verbinden Sie sich ohne Code mit über 5.000 Apps                                      |
| **Workflow-HTTP-Aktionen** | Rufen Sie im Rahmen eines automatisierten Workflows jede externe API auf              |

## Benutzerdefinierte Apps

Entwickler können voll funktionsfähige Apps auf Basis von Twenty erstellen – mit benutzerdefinierter UI, serverseitiger Logik und tiefen Integrationen. Apps können für die Community veröffentlicht oder privat gehalten werden.
