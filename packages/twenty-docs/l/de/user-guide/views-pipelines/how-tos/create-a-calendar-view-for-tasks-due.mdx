## Voraussetzungen

Ihr Aufgabenobjekt benötigt ein Feld **Fälligkeitsdatum** (Typ Datum oder Datum & Uhrzeit).

## Schritte

1. Navigieren Sie zu **Aufgaben**
2. Klicken Sie auf das Ansichts-Dropdown → **+ Ansicht hinzufügen**
3. Benennen Sie Ihre Ansicht (z. B. "Aufgabenkalender")
4. Klicken Sie auf **Erstellen**
5. Klicken Sie auf **Optionen** und wählen Sie **Kalender** als Layout
6. Wählen Sie **Fälligkeitsdatum** als Datumsfeld
7. Klicken Sie auf **Speichern**

## Konfigurieren Sie Ihren Kalender

### Felder in Ereignissen anzeigen

1. Klicken Sie auf **Optionen → Felder**
2. Klicken Sie auf das **Augensymbol**, um Felder ein-/auszublenden
3. Ziehen, um neu anzuordnen

Empfohlene Felder zum Anzeigen:

* **Titel** — Name der Aufgabe
* **Zuständige Person** — wer verantwortlich ist
* **Status** — aktueller Fortschritt

### Kalender filtern

Erstellen Sie gezielte Ansichten:

* **Meine Aufgaben**: Filtern nach "Zuständige Person" = Ich
* **Diese Woche**: Filtern nach Fälligkeitsdatum = Diese Woche
* **Überfällig**: Nach Fälligkeitsdatum < Heute filtern, Status ≠ Erledigt

## Weitere Anwendungsfälle für den Kalender

| Objekt                        | Datumsfeld     | Zweck                                     |
| ----------------------------- | -------------- | ----------------------------------------- |
| Opportunities                 | Abschlussdatum | Voraussichtliche Abschlüsse nachverfolgen |
| Benutzerdefinierte Ereignisse | Ereignisdatum  | Aktivitäten planen                        |
| Projekte                      | Frist          | Projektzeitpläne überwachen               |

## Tipps

* **Wöchentlich prüfen**: Beginnen Sie jede Woche damit, Ihre Kalenderansicht zu prüfen
* **Mit Tabellenansicht kombinieren**: Verwenden Sie den Kalender für den Überblick, die Tabellenansicht für Details
* **Sichtbarkeit festlegen**: Persönliche Aufgabenkalender als Unlisted belassen
