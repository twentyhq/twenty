## Daten filtern

Filter helfen Ihnen, sich auf bestimmte Datensätze zu konzentrieren, indem sie nur jene anzeigen, die Ihren Kriterien entsprechen.

### Einen Filter hinzufügen

1. Klicken Sie in der Symbolleiste auf die Schaltfläche **Filter**
2. Wählen Sie das Feld aus, nach dem gefiltert werden soll
3. Wählen Sie den Operator (gleich, enthält usw.)
4. Geben Sie den Filterwert ein
5. Klicken Sie auf **Übernehmen**

### Filteroperatoren

| Feldtyp          | Verfügbare Operatoren                               |
| ---------------- | --------------------------------------------------- |
| Text             | Gleich, Enthält, Beginnt mit, Endet mit, Ist leer   |
| Nummer           | Gleich, Größer als, Kleiner als, Zwischen, Ist leer |
| Datum            | Gleich, Vor, Nach, Zwischen, Ist leer               |
| Auswahl          | Gleich, Ist eines von, Ist leer                     |
| Kontrollkästchen | Ist wahr, Ist falsch                                |
| Beziehung        | Gleich, Ist leer                                    |

### Mehrere Filter

Kombinieren Sie mehrere Filter, um die Ergebnisse einzugrenzen:

* Alle Filter werden mit UND-Logik angewendet
* Jeder zusätzliche Filter schränkt die Ergebnisse weiter ein

### Filter entfernen

* Klicken Sie bei einzelnen Filter-Chips auf das **X**
* Klicken Sie auf **Alle löschen**, um alle Filter zu entfernen

## Daten sortieren

Die Sortierung legt die Reihenfolge fest, in der Datensätze angezeigt werden.

### Eine Sortierung hinzufügen

1. Klicken Sie in der Symbolleiste auf die Schaltfläche **Sortieren**
2. Wählen Sie das Feld aus, nach dem sortiert werden soll
3. Wählen Sie aufsteigend (A-Z, 0-9) oder absteigend (Z-A, 9-0)
4. Klicken Sie auf **Übernehmen**

### Mehrere Sortierungen

Fügen Sie mehrere Sortierstufen hinzu:

* Die erste Sortierung hat Priorität
* Nachfolgende Sortierungen gelten innerhalb von Gruppen gleicher Werte

### Schnelles Spaltensortieren

Klicken Sie auf eine Spaltenüberschrift, um zu sortieren:

* Erster Klick: Aufsteigend
* Zweiter Klick: Absteigend
* Dritter Klick: Sortierung entfernen

## Filter- und Sortiereinstellungen speichern

Filter und Sortierungen werden mit der Ansicht gespeichert:

1. Konfigurieren Sie Ihre Filter und Sortierungen
2. Klicken Sie auf **Speichern**, um die aktuelle Ansicht zu aktualisieren
3. Oder klicken Sie auf **Als neue Ansicht speichern**, um eine Variante zu erstellen
