## Über die Kalenderansicht

Die Kalenderansicht zeigt Ihre Datensätze in einem Kalender basierend auf einem Datumsfeld an. Jeder Datensatz erscheint als Kalendereintrag am entsprechenden Datum.

<img src="/images/user-guide/views/calendar-view-zoom.png" style={{width:'100%'}} />

## Eine Kalenderansicht erstellen

1. Navigieren Sie zu einem Objekt mit Datumsfeldern
2. Klicken Sie auf das Ansichts-Dropdown → **+ Ansicht hinzufügen**
3. Benennen Sie Ihre Ansicht und klicken Sie auf  **Erstellen**
4. Öffnen Sie rechts die **Optionen**
5. Wählen Sie **Kalender** als Layout
6. Wählen Sie das **Datumsfeld**, das für die Positionierung der Datensätze verwendet werden soll
7. Klicken Sie auf **Ansicht aktualisieren**

## Kalender konfigurieren

### Das Datumsfeld auswählen

Wählen Sie unter **Optionen** aus, welches Datumsfeld bestimmt, wo Datensätze im Kalender angezeigt werden.

### Felder anzeigen

Legen Sie fest, welche Felder bei jedem Kalendereintrag angezeigt werden:

1. Klicken Sie auf **Optionen → Felder**
2. Felder ein-/ausschalten
3. Ziehen, um neu anzuordnen

## Anwendungsfälle

* **Besprechungen und Anrufe**: Bevorstehende Termine anzeigen
* **Fristen**: Fälligkeiten und Abschlussdaten nachverfolgen
* **Veranstaltungen**: Terminierte Aktivitäten planen und visualisieren
* **Nachfassaktionen**: Sehen, wann Aufgaben fällig sind
