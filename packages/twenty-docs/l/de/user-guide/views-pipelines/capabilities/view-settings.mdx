## Sichtbarkeit der Ansicht

Steuern Sie, wer Ihre benutzerdefinierten Ansichten sehen kann.

### Sichtbarkeitsoptionen

| Einstellung        | Sichtbar für                        |
| ------------------ | ----------------------------------- |
| **Arbeitsbereich** | Alle Mitglieder des Arbeitsbereichs |
| **Nicht gelistet** | Nur Sie                             |

### Sichtbarkeit ändern

1. Öffnen Sie die Ansicht
2. Klicken Sie auf **Optionen → Sichtbarkeit**
3. Wählen Sie **Arbeitsbereich** oder **Nicht gelistet** aus

<Note>
  Die Standardansichten "All [Object Name]" können nicht in ihrer Sichtbarkeit geändert werden.
</Note>

## Eine Ansicht umbenennen

1. Öffnen Sie das Ansichts-Dropdown-Menü
2. Klicken Sie auf das **⋮**-Menü neben der Ansicht
3. Wählen Sie **Bearbeiten** aus
4. Geben Sie den neuen Namen ein

## Ansichtssymbol ändern

1. Öffnen Sie das Ansichts-Dropdown-Menü
2. Klicken Sie auf das **⋮**-Menü neben der Ansicht
3. Wählen Sie **Bearbeiten** aus
4. Klicken Sie auf das Symbol, um es zu ändern

## Ansichten neu anordnen

Ändern Sie die Reihenfolge, in der Ansichten im Dropdown angezeigt werden:

1. Öffnen Sie das Ansichts-Dropdown-Menü
2. Ziehen Sie die Ansichten am Griffsymbol
3. Lassen Sie sie an der gewünschten Position los
4. Die Reihenfolge wird automatisch gespeichert

## Favoriten

Heften Sie häufig verwendete Ansichten für den schnellen Zugriff an:

1. Öffnen Sie das Ansichts-Dropdown-Menü
2. Klicken Sie auf das **⋮**-Menü neben einer Ansicht
3. Wählen Sie **Zu Favoriten hinzufügen** aus

Zu Favoriten hinzugefügte Ansichten werden in einem eigenen Bereich für den schnellen Zugriff angezeigt.

## Eine Ansicht löschen

1. Öffnen Sie das Ansichts-Dropdown-Menü
2. Klicken Sie auf das **⋮**-Menü neben der Ansicht
3. Wählen Sie **Löschen** aus
4. Bestätigen Sie das Löschen

<Warning>
  Gelöschte Ansichten können nicht wiederhergestellt werden.
</Warning>
