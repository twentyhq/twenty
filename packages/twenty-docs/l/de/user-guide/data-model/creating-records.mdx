## Über Tabellenansichten

Tabellenansichten sind visuelle Darstellungen von Daten, die in Reihen und Spalten strukturiert sind.

## Create record

Fügen Sie bei Bedarf Datensätze hinzu, ohne Begrenzungen. Um einen Datensatz hinzuzufügen, können Sie entweder auf die **+**-Schaltfläche oben rechts auf dem Bildschirm oder oben in der Spalte **Name** klicken.

Geben Sie den Namen des Datensatzes ein und drücken Sie dann die Eingabetaste, um zu speichern. Um einen Datensatznamen zu bearbeiten, klicken Sie auf seinen Namen auf der Detailseite.

<VimeoEmbed videoId="927071691" title="Video demonstration" />

## Datensatz löschen

**Indexansicht:** Um einen Datensatz zu löschen, wählen Sie das Kontrollkästchen neben dem Datensatz und klicken Sie auf die Löschen-Schaltfläche in der oberen rechten Ecke.

**Datensatz-Seite:** Tippen Sie auf das `⋮`-Symbol in der oberen rechten Ecke und wählen Sie Löschen.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## Ein benutzerdefiniertes Feld hinzufügen

Um ein benutzerdefiniertes Feld zu erstellen, klicken Sie am rechten Ende der Tabellenspalten auf die **+**-Schaltfläche und wählen **Felder anpassen**.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

Sie können dies auch tun, indem Sie zu **Einstellungen** > **Datenmodell** > **Personen** navigieren. Klicken Sie auf **Feld hinzufügen**. Wählen Sie einen Feldnamen und -typ. Das neue Feld wird in der App verfügbar sein.
