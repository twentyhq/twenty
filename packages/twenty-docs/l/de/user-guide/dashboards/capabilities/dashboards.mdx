## Übersicht

Dashboards in Twenty sind hierarchisch organisiert: **Dashboards → Registerkarten → Widgets**. Jedes Dashboard kann mehrere Registerkarten enthalten, und jede Registerkarte enthält Widgets (Diagramme, Zahlen, iFrames).

## Ein Dashboard erstellen

1. Gehen Sie in der Navigation zu **Dashboards**
2. Klicken Sie auf **+ Neuer Datensatz**

<img src="/images/user-guide/dashboard/dashboard-create-new.png" alt="Neues Dashboard erstellen" />

3. Geben Sie Ihrem Dashboard einen Namen
4. Fügen Sie Registerkarten und Widgets hinzu

## Mit Registerkarten arbeiten

Registerkarten helfen Ihnen, Ihr Dashboard in logische Abschnitte zu gliedern.

### Registerkarten erstellen

1. Im Bearbeitungsmodus klicken Sie in der Registerkartenleiste auf **+**; der Seitenbereich öffnet sich rechts
2. Benennen Sie Ihre Registerkarte (z. B. "Pipeline-Übersicht", "Teamleistung")
3. Fügen Sie der Registerkarte Widgets hinzu

### Registerkarten neu anordnen

1. Im Bearbeitungsmodus klicken Sie auf die Registerkarte, die Sie verschieben möchten, und halten Sie die Maustaste gedrückt
2. Per Drag-and-Drop an die neue Position verschieben
3. Sie können auch auf die Registerkarte klicken und das Befehlsmenü verwenden, um die Registerkarte nach links oder rechts zu verschieben

### Registerkarten löschen

1. Im Bearbeitungsmodus klicken Sie auf die Registerkarte, die Sie löschen möchten
2. Klicken Sie im Befehlsmenü auf die Schaltfläche **Löschen**

### Registerkarten duplizieren

1. Klicken Sie auf die Registerkarte, die Sie duplizieren möchten
2. Klicken Sie in der Seitenleiste auf die Schaltfläche **Duplizieren**

<img src="/images/user-guide/dashboard/dashboard-tabs.png" alt="Dashboard-Registerkarte" />

## Dashboard-Layout

### Widgets anordnen

* Per Drag-and-Drop positionieren
* Zur Hervorhebung die Größe anpassen
* Verwandte Diagramme gruppieren

### Ein Dashboard duplizieren

1. Wählen Sie das Dashboard aus, das Sie duplizieren möchten
2. Öffnen Sie die Befehlsleiste mit **Cmd + K** (oder **Ctrl + K** unter Windows)
3. Wählen Sie **Dashboard duplizieren**

<img src="/images/user-guide/dashboard/dashboard-duplicate.png" alt="Dashboard duplizieren" />

### Beste Praktiken

* **Logischer Ablauf**: Von der Übersicht zum Detail anordnen
* **Visuelle Hierarchie**: Größere Diagramme für Schlüsselkennzahlen
* **Einheitliches Design**: Verwenden Sie passende Farben und Schriftarten

## Sichtbarkeit & Zugriff

### Sichtbarkeit des Dashboards

Dashboards sind für alle sichtbar, die Zugriff auf Ihren Twenty-Arbeitsbereich haben. Derzeit gibt es keine Option für private Dashboards.

### Favoriten

Sie können Dashboards für einen schnellen Zugriff zu Ihren Favoriten hinzufügen. Dies ist eine persönliche Einstellung—Ihre Favoriten sind für andere Benutzer nicht sichtbar.

Um ein Dashboard zu den Favoriten hinzuzufügen, öffnen Sie das Dashboard und klicken Sie auf das Sternsymbol.

### Zeitzonenverhalten

Dashboards zeigen derzeit Daten basierend auf der Zeitzone des Benutzers, der sie ansieht. Das bedeutet, dass dasselbe Dashboard für Teammitglieder in verschiedenen Regionen unterschiedliche Kennzahlen anzeigen kann (z. B. APAC vs. US).

<Note>
  **Demnächst verfügbar**: Wir werden die Möglichkeit hinzufügen, für ein Dashboard eine bestimmte Zeitzone festzulegen, sodass alle Benutzer unabhängig von ihrem Standort konsistente Daten sehen.
</Note>

<Note>
  **Demnächst verfügbar**: Filter auf Dashboard-Ebene ermöglichen es Ihnen, Filter auf alle Widgets gleichzeitig anzuwenden, sodass Sie Ihre Daten schneller erkunden können.
</Note>
