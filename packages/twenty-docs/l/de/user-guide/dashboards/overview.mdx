## Dashboards verstehen

Dashboards in Twenty bieten eine visuelle Möglichkeit, Ihre wichtigsten Leistungskennzahlen zu verfolgen und Erkenntnisse aus Ihren CRM-Daten zu gewinnen.

<img src="/images/user-guide/dashboard/dashboards_v2.png" alt="Dashboards" />

## Wichtige Konzepte

### Dashboards

Ein Dashboard ist eine Sammlung von Registerkarten, die Ihre CRM-Daten auf einen Blick anzeigen. Sie können mehrere Dashboards für unterschiedliche Zwecke erstellen:

* Vertriebsleistung
* Teamaktivität
* Gesundheit der Pipeline
* Benutzerdefinierte Kennzahlen

### Registerkarten

Registerkarten ermöglichen es Ihnen, Ihr Dashboard in Abschnitte zu gliedern. Jede Registerkarte enthält ein oder mehrere Widgets.

### Widgets

Widgets sind einzelne Visualisierungen, die spezifische Daten anzeigen. Zu den Typen gehören:

* Balkendiagramme
* Tortendiagramme
* Liniendiagramme
* Kennzahlen
* iFrames

<Note>
  **Aktuelle Einschränkungen**:

  * Das Exportieren von Dashboards und das Freigeben für externe Benutzer (Nicht-Twenty-Benutzer) sind derzeit nicht verfügbar.
  * Gauge-Diagramme und Tabellen sind noch nicht verfügbar.
</Note>

## Erste Schritte

### Ihr erstes Dashboard erstellen

1. Navigieren Sie zum Bereich **Dashboards**
2. Klicken Sie auf **+ Neues Dashboard**
3. Geben Sie Ihrem Dashboard einen Namen
4. Fügen Sie Registerkarten hinzu, um Ihre Inhalte zu organisieren
5. Fügen Sie Widgets hinzu, um Ihre Daten anzuzeigen
6. Speichern

### Widgets hinzufügen

1. Öffnen Sie eine Registerkarte in Ihrem Dashboard
2. Klicken Sie auf **+ Widget hinzufügen**
3. Wählen Sie den Widget-Typ aus
4. Wählen Sie die Datenquelle (Objekt)
5. Konfigurieren Sie die Widget-Einstellungen
6. Speichern Sie und zeigen Sie Ihr Widget an

## Beste Praktiken

* **Einfach anfangen**: Beginnen Sie mit einigen wichtigen Kennzahlen und fügen Sie im Laufe der Zeit weitere hinzu
* **Auf umsetzbare Daten fokussieren**: Zeigen Sie Kennzahlen an, die Entscheidungen vorantreiben
* **Regelmäßige Überprüfung**: Prüfen Sie Ihre Dashboards regelmäßig, um Trends zu erkennen
* **Mit dem Team teilen**: Machen Sie Dashboards für relevante Teammitglieder sichtbar
