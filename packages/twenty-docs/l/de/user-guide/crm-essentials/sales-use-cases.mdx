## Erweiterte Verkaufsmöglichkeiten

GTM-Teams benötigen oft erweiterte Verkaufsmöglichkeiten wie Lead-Scoring, Datenanreicherung, Rundlaufprozess, Gebietszuweisung, automatische Erinnerungen und E-Mail-Sequenzen. Obwohl diese keine integrierten Funktionen in Twenty sind, können sie alle mit unserem flexiblen Workflow-System konfiguriert und auf Ihre spezifischen Bedürfnisse zugeschnitten werden.

Besuchen Sie unseren [Workflows-Bereich](/l/de/user-guide/workflows/getting-started-workflows), um zu erfahren, wie Sie diese Automatisierungen Schritt für Schritt erstellen. For detailed examples, see our [Internal Automations](/l/de/user-guide/workflows/internal-automations) and [External Tool Integration](/l/de/user-guide/workflows/external-tool-integration) guides.
