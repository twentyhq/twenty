## Neue Mitglieder einladen

### Verwendung des Einladungslinks

1. Gehen Sie zu **Einstellungen → Mitglieder**
2. Kopieren Sie den Einladungslink für den Arbeitsbereich
3. Teilen Sie den Link mit neuen Teammitgliedern
4. Sie erhalten Zugriff, sobald sie sich anmelden

### Direkte E-Mail-Einladung

1. Gehen Sie zu **Einstellungen → Mitglieder**
2. Geben Sie die E-Mail-Adresse der Person ein
3. Klicken Sie auf **Einladen**
4. Sie erhalten eine E-Mail-Einladung

## Mitglieder entfernen

### Mitglied löschen

1. Gehen Sie zu **Einstellungen → Mitglieder**
2. Finden Sie das Mitglied, das Sie entfernen möchten
3. Klicken Sie auf die Löschen/Entfernen-Schaltfläche neben ihrem Namen
4. Bestätigen Sie die Entfernung

**Hinweis**: Entfernte Mitglieder verlieren sofort den Zugang, können jedoch später erneut eingeladen werden.
