## Neueste Veröffentlichungen

Verfolgen Sie den Entwicklungsfortschritt von Twenty:

- **Versionshinweise**: Sehen Sie, was in jeder Version neu ist
- **Funktionsupdates**: Entdecken Sie neue Fähigkeiten und Verbesserungen

## Laborfunktionen

Zugriff auf experimentelle Funktionen, bevor sie offiziell veröffentlicht werden:

- **Betatest**: Probieren Sie neue Funktionen aus, während sie in der Entwicklung sind
- **Feature Flags**: Aktivieren oder deaktivieren Sie bestimmte experimentelle Funktionen

## How to Access

1. Gehen Sie zu **Einstellungen → Veröffentlichungen**
2. View release notes and access the **Lab** tab for beta features
