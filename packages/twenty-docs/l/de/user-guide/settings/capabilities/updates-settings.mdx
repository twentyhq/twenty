## Über die Update-Einstellungen

Im Bereich Updates können Sie Early-Access-Funktionen aktivieren, bevor sie allgemein verfügbar sind.

## Early-Access-Funktionen

Early-Access-Funktionen sind Funktionen, die vor der allgemeinen Verfügbarkeit noch getestet werden.

**Derzeit im Early Access:**

* **Dashboards** — Visuelles Reporting und Analysen

### So aktivieren Sie Early-Access-Funktionen

1. Gehen Sie zu **Einstellungen → Updates**
2. Navigieren Sie zum Bereich **Early Access**
3. Suchen Sie die Funktion, die Sie aktivieren möchten
4. Aktivieren Sie sie
5. Die Funktion ist sofort verfügbar

<Note>
  Early-Access-Funktionen können sich je nach Nutzerfeedback ändern. Ihr Feedback hilft uns, Twenty zu verbessern!
</Note>

## Feedback zu Funktionen

Ihr Feedback hilft, Twenty zu verbessern:

* Melden Sie Probleme mit Early-Access-Funktionen
* Teilen Sie mit, wie Sie neue Funktionen nutzen
* Schlagen Sie Verbesserungen über den Community-Discord vor
