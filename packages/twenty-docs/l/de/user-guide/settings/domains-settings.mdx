## Benutzerdefinierte Arbeitsbereichsdomäne

Legen Sie eine personalisierte Webadresse für Ihren Arbeitsbereich fest:

- Wählen Sie Ihre bevorzugte Subdomain (z.B. yourcompany.twenty.com)
- Konfigurieren Sie benutzerdefinierte Domänen für professionelles Branding

## Approved Access Domains

Allow automatic workspace access for specific email domains.

### Wie es funktioniert

- Fügen Sie die E-Mail-Domänen Ihres Unternehmens hinzu (z.B. @yourcompany.com)
- Jeder mit einer E-Mail von diesen Domänen kann Ihrem Arbeitsbereich automatisch beitreten
- Keine manuelle Einladung für Teammitglieder erforderlich

### Vorteile

- **Schnelleres Onboarding**: Neue Teammitglieder treten automatisch bei
- **Bessere Sicherheit**: Nur verifizierte Unternehmens-E-Mails können zugreifen
- **Weniger Verwaltungsarbeit**: Es ist nicht erforderlich, jedes Teammitglied manuell einzuladen

## Einrichtungsanweisungen

1. Gehen Sie zu **Einstellungen → Domänen**
2. Fügen Sie Ihre benutzerdefinierte Domäne oder genehmigte E-Mail-Domänen hinzu
3. Verifizieren Sie den Domänenbesitz, falls erforderlich

**Hinweis**: Domänenänderungen können Zeit benötigen, um wirksam zu werden und könnten eine Verifizierung erfordern.
