## Arbeitsbereichseinstellungen

<AccordionGroup>
  <Accordion title="Wie ändere ich den Namen meines Arbeitsbereichs?">
    1. Gehen Sie zu **Einstellungen → Allgemein**
    2. Suchen Sie das Feld Arbeitsbereichsname
    3. Geben Sie Ihren neuen Namen ein
    4. Änderungen werden automatisch gespeichert
  </Accordion>

  <Accordion title="Wie aktualisiere ich das Logo meines Arbeitsbereichs?">
    1. Gehen Sie zu **Einstellungen → Allgemein**
    2. Klicken Sie auf das aktuelle Logo oder den Upload-Bereich
    3. Wählen Sie eine Bilddatei aus (PNG, JPEG oder GIF unter 10 MB)
    4. Das Logo wird sofort aktualisiert
  </Accordion>

  <Accordion title="Kann ich mehrere Arbeitsbereiche haben?">
    Ja, Sie können mehrere Arbeitsbereiche erstellen und ihnen als Mitglied angehören. Jeder Arbeitsbereich hat seine eigenen Daten, Einstellungen und ein eigenes Abonnement.
  </Accordion>

  <Accordion title="Wie lösche ich meinen Arbeitsbereich?">
    1. Gehen Sie zu **Einstellungen → Allgemein**
    2. Scrollen Sie zum Gefahrenbereich
    3. Klicken Sie auf **Arbeitsbereich löschen**
    4. Bestätigen Sie das Löschen

    Hinweis: Dadurch werden alle Daten dauerhaft gelöscht und der Vorgang lässt sich nicht rückgängig machen.
  </Accordion>

  <Accordion title="Ich habe versehentlich mehrere Arbeitsbereiche erstellt. Was sollte ich tun?">
    Löschen Sie die Arbeitsbereiche, die Sie nicht mehr benötigen, unter **Einstellungen → Allgemein → Arbeitsbereich löschen**.

    <Warning>
      Löschen Sie nicht Ihr **Konto** (unter Einstellungen → Profil erreichbar): Ihr Konto wird von all Ihren Arbeitsbereichen gemeinsam genutzt. Wenn Sie Ihr Konto löschen, verlieren Sie den Zugriff auf ALLE Arbeitsbereiche.
    </Warning>
  </Accordion>

  <Accordion title="Wie kann ich meinen Arbeitsbereich deaktivieren, ohne ihn zu löschen?">
    Wenn Sie Ihren Arbeitsbereich vorübergehend deaktivieren möchten (nicht dauerhaft löschen), gehen Sie zu **Einstellungen → Abrechnung** und klicken Sie auf **Plan kündigen**. Ihre Daten werden für einen Kulanzzeitraum aufbewahrt.
  </Accordion>
</AccordionGroup>

## Profileinstellungen

<AccordionGroup>
  <Accordion title="Wie ändere ich mein Passwort?">
    1. Gehen Sie zu **Einstellungen → Profil**
    2. Suchen Sie den Abschnitt Passwort
    3. Geben Sie Ihr aktuelles Passwort ein
    4. Geben Sie Ihr neues Passwort ein
    5. Änderungen speichern
  </Accordion>

  <Accordion title="Wie aktiviere ich die Zwei-Faktor-Authentifizierung?">
    1. Gehen Sie zu **Einstellungen → Profil**
    2. Suchen Sie den Abschnitt 2FA
    3. Klicken Sie auf **2FA aktivieren**
    4. Scannen Sie den QR-Code mit Ihrer Authentifizierungs-App
    5. Geben Sie den Bestätigungscode ein
  </Accordion>

  <Accordion title="Kann ich meine E-Mail-Adresse ändern?">
    Um Ihre E-Mail-Adresse zu ändern, wenden Sie sich bitte an [contact@twenty.com](mailto:contact@twenty.com).
  </Accordion>

  <Accordion title="Wie lösche ich mein Konto?">
    1. Gehen Sie zu **Einstellungen → Profil**
    2. Scrollen Sie zum Gefahrenbereich
    3. Klicken Sie auf **Konto löschen**
    4. Bestätigen Sie, indem Sie Ihre E-Mail-Adresse eingeben

    Hinweis: Dadurch verlieren Sie den Zugriff auf alle Arbeitsbereiche und alle E-Mails, die von Ihren verbundenen Konten synchronisiert wurden, werden gelöscht.
  </Accordion>
</AccordionGroup>

## Erfahrung - Einstellungen

<AccordionGroup>
  <Accordion title="Wie wechsle ich zwischen hellem und dunklem Modus?">
    1. Gehen Sie zu **Einstellungen → Erlebnis**
    2. Suchen Sie den Abschnitt Thema
    3. Wählen Sie Hell, Dunkel oder System
  </Accordion>

  <Accordion title="Wie ändere ich das Datumsformat?">
    1. Gehen Sie zu **Einstellungen → Erlebnis**
    2. Suchen Sie das Datumsformat
    3. Wählen Sie Ihr bevorzugtes Format
    4. Änderungen werden sofort übernommen
  </Accordion>

  <Accordion title="Wie ändere ich meine Zeitzone?">
    1. Gehen Sie zu **Einstellungen → Erlebnis**
    2. Suchen Sie die Zeitzone
    3. Wählen Sie Ihre lokale Zeitzone aus
    4. Alle Zeitstempel werden angepasst
  </Accordion>

  <Accordion title="Kann ich die Sprache ändern?">
    1. Gehen Sie zu **Einstellungen → Erlebnis**
    2. Suchen Sie die Sprache
    3. Wählen Sie aus den verfügbaren Sprachen
    4. Die Oberfläche wird entsprechend Ihrer Auswahl aktualisiert
  </Accordion>
</AccordionGroup>

## Kontoeinstellungen

<AccordionGroup>
  <Accordion title="Wie verbinde ich mein E-Mail-Konto?">
    1. Gehen Sie zu **Einstellungen → Konten**
    2. Klicken Sie auf **Konto hinzufügen**
    3. Wählen Sie Google oder Microsoft
    4. Autorisieren Sie den Zugriff
    5. Konfigurieren Sie die Synchronisierungseinstellungen
  </Accordion>

  <Accordion title="Kann ich mehrere E-Mail-Konten verbinden?">
    Ja, Sie können mehrere E-Mail-Konten verbinden. Gehen Sie zu **Einstellungen → Konten** und fügen Sie nach Bedarf weitere Konten hinzu.
  </Accordion>

  <Accordion title="Wie trenne ich die Verbindung zu einem E-Mail-Konto?">
    1. Gehen Sie zu **Einstellungen → Konten**
    2. Suchen Sie das zu entfernende Konto
    3. Klicken Sie auf **Trennen**
    4. Bestätigen Sie die Aktion
  </Accordion>
</AccordionGroup>

## Domänen

<AccordionGroup>
  <Accordion title="Kann ich meine Arbeitsbereichsdomäne anpassen?">
    Ja! Gehen Sie zu **Einstellungen → Domänen** und klicken Sie auf **Domäne anpassen**. Sie haben zwei Optionen:

    * **Subdomain**: Verwenden Sie eine Twenty-Subdomain wie `yourcompany.twenty.com`
    * **Benutzerdefinierte Domäne**: Verwenden Sie Ihre eigene Domäne wie `crm.yourcompany.com` (erfordert DNS-Konfiguration)

    Eine Subdomain ist schnell eingerichtet, während eine benutzerdefinierte Domäne ein vollständig markenkonformes Erlebnis für Ihr Team bietet.
  </Accordion>

  <Accordion title="Wie funktionieren genehmigte Zugriffsdomänen?">
    Sie können genehmigte Zugriffsdomänen konfigurieren, damit Teammitglieder mit Firmen-E-Mail-Adressen Ihrem Arbeitsbereich automatisch beitreten können. Gehen Sie zu **Einstellungen → Domänen** und fügen Sie Ihre Firmendomäne hinzu (z. B. `yourcompany.com`).
  </Accordion>
</AccordionGroup>

## Early-Access-Funktionen

<AccordionGroup>
  <Accordion title="Was sind Early-Access-Funktionen?">
    Early-Access-Funktionen sind Funktionen, die vor der allgemeinen Verfügbarkeit getestet werden. Derzeit befinden sich nur **Dashboards** im Early Access. Sie können sich basierend auf Benutzerfeedback ändern.
  </Accordion>

  <Accordion title="Sind Early-Access-Funktionen sicher zu verwenden?">
    Early-Access-Funktionen sind funktionsfähig, können sich jedoch basierend auf Benutzerfeedback ändern. Sie sind sicher zu verwenden, können sich jedoch vor der allgemeinen Verfügbarkeit noch weiterentwickeln.
  </Accordion>

  <Accordion title="Wie aktiviere ich Early-Access-Funktionen?">
    1. Gehen Sie zu **Einstellungen → Updates → Early Access**
    2. Suchen Sie die gewünschte Funktion
    3. Aktivieren Sie sie
    4. Die Funktion ist sofort verfügbar
  </Accordion>
</AccordionGroup>
