## Überblick

Integrieren Sie KI-Funktionen direkt in Ihre Automatisierungs-Workflows für intelligente Datenverarbeitung und Entscheidungsfindung.

## Funktionen

| Funktion                       | Beschreibung                                                  |
| ------------------------------ | ------------------------------------------------------------- |
| **KI-Aktionen**                | Fügen Sie jedem Workflow KI-gestützte Schritte hinzu.         |
| **Datenanreicherung**          | Reichern Sie Datensätze automatisch mit externen Daten an.    |
| **Klassifizierung**            | Kategorisieren Sie Datensätze auf Basis einer Inhaltsanalyse. |
| **Zusammenfassung**            | Erstellen Sie Zusammenfassungen aus Textfeldern.              |
| **Benutzerdefinierte Prompts** | Definieren Sie genau, wie die KI Ihre Daten verarbeitet.      |

## Anwendungsfälle

* **Lead-Scoring**: Bewerten und priorisieren Sie eingehende Leads automatisch
* **Datenbereinigung**: Standardisieren Sie Unternehmensnamen und Kontaktdaten
* **E-Mail-Entwürfe**: Generieren Sie Follow-up-E-Mails auf Basis von Besprechungsnotizen
* **Datensatz-Routing**: Weisen Sie Datensätze auf Basis des Inhalts dem richtigen Teammitglied zu
