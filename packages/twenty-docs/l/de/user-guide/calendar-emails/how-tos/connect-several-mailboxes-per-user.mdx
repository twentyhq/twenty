title: Mehrere Postfächer pro Nutzer verbinden
description: Mehrere E-Mail-Konten für einen einzelnen Nutzer verbinden.
---

## Übersicht

Twenty unterstützt **unbegrenzte E-Mail-Konten pro Nutzer**. Dies ist nützlich, wenn Sie mehrere Posteingänge verwalten, zum Beispiel:

* Persönliche Arbeits-E-Mail + gemeinsamer Team-Posteingang
* Mehrere E-Mail-Adressen für die Kundenkommunikation
* Verschiedene E-Mail-Konten für unterschiedliche Rollen

## So fügen Sie mehrere Postfächer hinzu

1. Gehen Sie zu **Einstellungen → Konten**
2. Klicken Sie auf **Konto hinzufügen**
3. Verbinden Sie Ihr zusätzliches Google- oder Microsoft-Konto
4. Synchronisierungseinstellungen für dieses Postfach konfigurieren
5. Wiederholen Sie dies für jedes Postfach, das Sie verbinden möchten

## Mehrere Konten verwalten

Jedes verbundene Postfach hat eigene Einstellungen:

* **E-Mail-Sichtbarkeit**: Wählen Sie aus, was Teammitglieder sehen können
* **Automatische Kontakterstellung**: Pro Postfach aktivieren/deaktivieren
* **Ordnerauswahl**: Wählen Sie, welche Ordner synchronisiert werden (Lab-Funktion)

## So werden E-Mails angezeigt

E-Mails aus all Ihren verbundenen Postfächern werden mit Twenty synchronisiert und erscheinen unter:

* **Personendatensätze**: Basierend auf der E-Mail-Adresse des Kontakts
* **Unternehmensdatensätze**: Basierend auf der E-Mail-Domain
* **Opportunities**: Basierend auf dem verknüpften Unternehmen

<Note>Jede E-Mail zeigt, von welchem Postfach sie gesendet bzw. in welchem sie empfangen wurde, sodass Sie nachverfolgen können, welches Konto für jede Kommunikation verwendet wurde.</Note>

## Wichtige Hinweise

<Warning>Nur echte Postfächer können verbunden werden. E-Mail-Aliase, die an ein anderes Postfach weiterleiten, können nicht separat verbunden werden—sie werden über das Hauptpostfach synchronisiert.</Warning>
