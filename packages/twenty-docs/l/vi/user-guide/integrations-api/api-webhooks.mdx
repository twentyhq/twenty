## Các khóa API

Khóa API cho phép truy cập tự động vào dữ liệu CRM của bạn, đồng bộ hóa dữ liệu với các hệ thống khác, và tạo các tích hợp tùy chỉnh hoặc giải pháp.

### Tạo một Khóa API

1. Go to **Settings → APIs & Webhooks**
2. Nhấp vào **+ Tạo khóa** ở góc trên cùng bên phải
3. Cấu hình khóa API của bạn:
   - **Tên**: Đặt một tên mô tả cho khóa API của bạn
   - **Ngày hết hạn**: Đặt thời điểm khóa nên hết hạn
4. Nhấp vào **Lưu** để tạo khóa API của bạn
5. **Quan trọng**: Sao chép và lưu trữ khóa API của bạn ngay lập tức, nó chỉ được hiển thị một lần

Sau khi được tạo, khóa API của bạn cho phép truy cập vào tài liệu API tùy chỉnh và khu vực tập thử nơi bạn có thể kiểm tra các điểm cuối với mô hình dữ liệu thực tế của bạn.

<Warning>
Vì khóa API của bạn cung cấp quyền truy cập vào thông tin nhạy cảm, bạn không nên chia sẻ nó với các dịch vụ mà bạn không hoàn toàn tin tưởng. Nếu bị rò rỉ, ai đó có thể sử dụng nó một cách ác ý. Nếu tính bảo mật của khóa API của bạn bị xâm phạm, hãy ngay lập tức vô hiệu hóa nó và tạo một cái mới.
</Warning>

### Quản lý Khóa API

**Regenerate an API Key:**

1. Go to **Settings → APIs & Webhooks**
2. Nhấp vào khóa API mà bạn muốn tạo lại
3. Nhấp vào nút **Tạo Lại**
4. Sao chép và lưu trữ khóa API mới ngay lập tức

**Xóa một Khóa API:**

1. Tìm khóa API trong danh sách của bạn
2. Nhấp vào khóa để mở chi tiết của nó
3. Nhấp **Xóa** để loại bỏ nó vĩnh viễn

## Webhooks

Webhooks cho phép cập nhật tức thời đến URL bạn chỉ định về các thay đổi hoặc sự kiện liên quan đến dữ liệu khách hàng của bạn.

Ví dụ: khi một Cơ hội chuyển sang "Đã Đóng - Thắng", một webhook có thể tự động kích hoạt tạo hóa đơn trong hệ thống kế toán của bạn. Lưu ý rằng loại tự động hóa này cũng có thể đạt được bằng cách sử dụng tính năng [Workflows](/l/vi/user-guide/workflows/getting-started-workflows) trong ứng dụng của Twenty, cung cấp các kích hoạt dựa trên cập nhật trường để tự động hóa nội bộ.

Webhooks là lý tưởng để tích hợp với các hệ thống bên ngoài, trong khi Workflow hỗ trợ cả tự động hóa nội bộ và kết nối công cụ bên ngoài thông qua các kích hoạt webhook, nút mã và nút HTTP.

### Tạo Webhook

1. Đi tới **Cài đặt → APIs & Webhooks → Webhooks**
2. Nhấp **+ Tạo Webhook**
3. Nhập URL webhook của bạn (nơi bạn muốn nhận thông báo)
4. Nhấp **Lưu**

Webhook của bạn sẽ ngay lập tức bắt đầu nhận thông báo theo thời gian thực về các thay đổi đối với dữ liệu CRM của bạn.

### Quản lý Webhooks

**Xóa một Webhook:**

1. Đi tới **Cài đặt → APIs & Webhooks → Webhooks**
2. Tìm webhook bạn muốn loại bỏ
3. Nhấp vào webhook
4. Nhấp **Xóa** và xác nhận trong cửa sổ bật lên

**Chỉnh sửa một Webhook:**

1. Nhấp vào webhook bạn muốn chỉnh sửa
2. Cập nhật URL hoặc các cài đặt khác
3. Nhấp **Lưu** để áp dụng thay đổi
