## Liên kết liên hệ

Một thành phần liên kết được thiết kế để hiển thị thông tin liên hệ.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Thuộc Tính | Loại              | Mô tả                                                  |
| ---------- | ----------------- | ------------------------------------------------------ |
| className  | string            | Tên tùy chọn cho kiểu dáng bổ sung                     |
| href       | string            | URL hoặc đường dẫn cho liên kết                        |
| onClick    | hàm               | Hàm callback được thực hiện khi liên kết được nhấp vào |
| children   | `React.ReactNode` | Nội dung được hiển thị bên trong liên kết              |

</Tab>

</Tabs>

## Liên kết bo tròn

Một liên kết có phong cách bo tròn với thành phần Chip cho liên kết.

<Tabs>
<Tab title="Usage">

```jsx
import { RoundedLink } from "/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log("Contact link clicked!", event);
  };

  return (
    <Router>
      <RoundedLink href="/contact" onClick={handleLinkClick}>
        Contact Us
      </RoundedLink>
    </Router>
  );
};
```

</Tab>

<Tab title="Props">

| Thuộc Tính | Loại              | Mô tả                                                  |
| ---------- | ----------------- | ------------------------------------------------------ |
| href       | string            | URL hoặc đường dẫn cho liên kết                        |
| children   | `React.ReactNode` | Nội dung được hiển thị bên trong liên kết              |
| onClick    | hàm               | Hàm callback được thực hiện khi liên kết được nhấp vào |

</Tab>
</Tabs>

## Liên kết mạng xã hội

Liên kết mạng xã hội được thiết kế đẹp mắt, hỗ trợ một số loại liên kết mạng xã hội, như URL, LinkedIn và X (hoặc Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Thuộc Tính | Loại              | Mô tả                                                                                                            |
| ---------- | ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| href       | string            | URL hoặc đường dẫn cho liên kết                                                                                  |
| children   | `React.ReactNode` | Nội dung được hiển thị bên trong liên kết                                                                        |
| loại       | string            | Loại liên kết mạng xã hội. Các tùy chọn bao gồm: `url`, `LinkedIn`, và `Twitter` |
| onClick    | hàm               | Hàm callback được thực hiện khi liên kết được nhấp vào                                                           |

</Tab>
</Tabs>
