Thành phần để phân loại hoặc ghi nhãn nội dung bằng hình ảnh.

<Tabs>

<Tab title="Usage">

```jsx
import { Tag } from "@/ui/display/tag/components/Tag";

export const MyComponent = () => {
  return (
    <Tag
      className
      color="red"
      text="Urgent"
      onClick={() => console.log("click")}
    />
  );
};
```

</Tab>

<Tab title="Props">

| Thuộc Tính | Loại   | Mô tả                                                                                                                                                  |
| ---------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| className  | string | Tên tùy chọn cho kiểu dáng bổ sung                                                                                                                     |
| màu sắc    | string | Màu của thẻ. Options include: `green`, `turquoise`, `sky`, `blue`, `purple`, `pink`, `red`, `orange`, `yellow`, `gray` |
| văn bản    | string | Nội dung của thẻ                                                                                                                                       |
| onClick    | hàm    | Chức năng tùy chọn được gọi khi người dùng nhấn vào thẻ                                                                                                |

</Tab>

</Tabs>
