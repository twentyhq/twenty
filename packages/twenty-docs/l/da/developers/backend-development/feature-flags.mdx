Funktionsflag bruges til at skjule eksperimentelle funktioner. For Twenty er de indstillet på arbejdsrumsniveau og ikke på brugerniveau.

## Tilføjelse af et nyt funktionsflag

I `FeatureFlagKey.ts` tilføj funktionsflaget:

```ts
type FeatureFlagKey =
  | 'IS_FEATURENAME_ENABLED'
  | ...;
```

Tilføj det også til enum i `feature-flag.entity.ts`:

```ts
enum FeatureFlagKeys {
    IsFeatureNameEnabled = 'IS_FEATURENAME_ENABLED',
    ...
}
```

For at anvende et funktionsflag på en **backend** funktion brug:

```ts
@Gate({
  featureFlag: 'IS_FEATURENAME_ENABLED',
})
```

For at anvende et funktionsflag på en **frontend** funktion brug:

```ts
const isFeatureNameEnabled = useIsFeatureEnabled('IS_FEATURENAME_ENABLED');
```

## Konfigurer funktionsflag til implementeringen

Ændr den tilsvarende post i Tablen `core.featureFlag`:

| id        | nøgle                    | workspaceId | værdi  |
| --------- | ------------------------ | ----------- | ------ |
| Tilfældig | `IS_FEATURENAME_ENABLED` | WorkspaceID | `sand` |
