## Nyeste udgivelser

Følg udviklingsfremskridt for Twenty:

- **Udgivelsesnoter**: Se hvad der er nyt i hver version
- **Funktionsopdateringer**: Opdag nye kapabiliteter og forbedringer

## Lab-funktioner

Få adgang til eksperimentelle funktioner før de officielt udgives:

- **Beta-test**: Prøv nye funktioner, mens de er under udvikling
- **Funktionsflag**: Aktiver eller deaktiver bestemte eksperimentelle funktionaliteter

## Sådan får du adgang

1. Gå til **Indstillinger → Udgivelser**
2. Se udgivelsesnoter, og få adgang til **Lab**-fanen for beta-funktioner

For flere fællesskabsressourcer, tjek vores [Ressource-sektion](/l/da/user-guide/resources/glossary).
