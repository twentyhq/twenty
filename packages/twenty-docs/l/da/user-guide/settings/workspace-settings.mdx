## Arbejdsområde Billede

- **Upload Logo**: Tilføj et brugerdefineret arbejdsområde logo
- **Understøttede formater**: PNG, JPEG og GIF filer under 10MB
- **Fjern**: Slet det nuværende arbejdsområde logo

## Arbejdsområdets navn

- **Navn**: Skift dit arbejdsområde visningsnavn
- Dette navn vises for alle arbejdsområde medlemmer

## Farezone

<Warning>
Sletning af dit arbejdsområde fjerner permanent alle data og kan ikke fortrydes. Alle arbejdsområde data vil være tabt for evigt, alle medlemmer vil miste øjeblikkelig adgang, og denne handling kan ikke fortrydes.
</Warning>

For at slette dit arbejdsområde:

1. Klik på **Slet arbejdsområde**-knappen
2. Bekræft sletningen når du bliver spurgt

**Bemærk**: Kun arbejdsområde administratorer kan slette arbejdsområder.
