## Implementeringstjenester

Uanset om du har brug for hjælp til at komme i gang eller skabe avancerede tilpasninger, har vi en løsning.

### Introduktionspakker

Få hjælp fra vores kernegruppe til at opsætte dit Twenty-arbejdsområde med vores 4-timers [Introduktionspakker](https://twenty.com/onboarding-packages):

- **Datamodeldesign**: Design og opret din tilpassede datamodel med objekter, felter og relationer
- **Datamigration**: Migrér dine eksisterende data fra dit nuværende CRM til Twenty
- **Arbejdsprocesseskabelse**: Opret tilpassede arbejdsprocesser for at støtte dine forretningsprocesser

### Implementeringspartnere

Arbejd med certificerede Twenty-partnere for mere avancerede tilpasninger og integrationer. Kontakt vores team via contact@twenty.com for at blive matchet med vores [partnere](https://twenty.com/implementation-services).
