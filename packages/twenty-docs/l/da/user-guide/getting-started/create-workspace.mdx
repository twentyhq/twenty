## Trin 1: Registrering

1. Gå til [Twenty Tilmelding](https://app.twenty.com).
2. Vælg din foretrukne tilmeldingsmetode:
   - **Fortsæt med Google** for registrering via Google-konto.
   - **Fortsæt med Microsoft** for registrering via Microsoft-konto.
   - Eller, **Fortsæt med e-mail** til e-mailregistrering.

## Step 2: Choosing a Trial Period

Vælg mellem to prøveperioder:

### 30 dage

Med kreditkort

### 7 dage

Uden kreditkort

Begge forsøg inkluderer:

- Fuld adgang
- Ubegrænsede kontakter
- E-mailintegration
- Tilpassede objekter
- API & Webhooks

Du kan klikke på "Skift plan" for at vælge en anden plan eller faktureringsinterval.

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## Trin 3: Betalingsbekræftelse & Kontoopsætning

Post payment approval via Stripe, you're directed to create your workspace and user profile. Husk, at du kan opsige dit abonnement når som helst.
