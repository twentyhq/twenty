## Avancerede Salgsfunktioner

GTM-hold har ofte brug for avancerede salgskapabiliteter som lead scoring, dataanrikning, rundesystem, områdefordeling, automatiske påmindelser og e-mail-sekvenser. Selvom disse ikke er indbyggede funktioner i Twenty, kan de alle konfigureres og tilpasses til dine specifikke behov ved hjælp af vores fleksible workflow-system.

Besøg vores [Arbejdsgangssektion](/l/da/user-guide/workflows/getting-started-workflows) for at lære, hvordan du trin for trin opbygger disse automatiseringer. For detaljerede eksempler, se vores [Intern Automatiseringer](/l/da/user-guide/workflows/internal-automations) og [Eksterne Værktøjsintegration](/l/da/user-guide/workflows/external-tool-integration) vejledninger.
