## Kontaktlink

En stiliseret linkkomponent til visning af kontaktinformation.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Egenskaber | Type              | Beskrivelse                                                |
| ---------- | ----------------- | ---------------------------------------------------------- |
| className  | streng            | Valgfrit navn til yderligere styling                       |
| href       | streng            | Mål-URL eller sti for linket                               |
| onClick    | funktion          | Callback function to be triggered when the link is clicked |
| children   | `React.ReactNode` | Indholdet, der skal vises inden i linket                   |

</Tab>

</Tabs>

## Social link

Stiliserede sociale links med støtte for forskellige typer af sociale links som URL'er, LinkedIn og X (eller Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Egenskaber | Type              | Beskrivelse                                                                                                       |
| ---------- | ----------------- | ----------------------------------------------------------------------------------------------------------------- |
| href       | streng            | Mål-URL eller sti for linket                                                                                      |
| children   | `React.ReactNode` | Indholdet, der skal vises inden i linket                                                                          |
| type       | streng            | Typen af sociale links. Valgmuligheder inkluderer: `url`, `LinkedIn` og `Twitter` |
| onClick    | funktion          | Callback function to be triggered when the link is clicked                                                        |

</Tab>
</Tabs>
