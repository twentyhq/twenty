Et visuelt element, du kan bruge som en klikbar eller ikke-klikbar container med en etiket, valgfri venstre og højre komponenter, og forskellige stylingmuligheder til at vise etiketter og tags.

<Tabs>

<Tab title="Usage">

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Chip
      size="large"
      label="Clickable Chip"
      clickable={true}
      variant="highlighted"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
    />
  );
};

```

</Tab>

<Tab title="Props">

| Egenskaber   | Type                     | Beskrivelse                                                                                                                    |
| ------------ | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| linkTilEnhed | streng                   | Linket til enheden                                                                                                             |
| enhedsId     | streng                   | Den unikke identifikator for enheden                                                                                           |
| navn         | string                   | Navnet på enheden                                                                                                              |
| billedeUrl   | string                   | s billede",                                                                                                                    |
| avatarType   | Avatar Type              | Typen af avatar du ønsker at vise. Har to muligheder: `afrundet` og `firkantet`                |
| variant      | `EnhedsChipVariant` enum | Variant af enhedschippen, du ønsker at vise. Har to muligheder: `almindelig` og `gennemsigtig` |
| VenstreIkon  | IkonKomponent            | En React-komponent, der repræsenterer et ikon. Vises på venstre side af chippen                                |

</Tab>
</Tabs>

## Eksempler

### Gennemsigtig Deaktiveret Chip

```jsx
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Transparent Disabled Chip"
      clickable={false}
      variant="rounded"
      accent="text-secondary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};

```

<br/>

### Deaktiveret Chip med Værktøjstips

```jsx
import { Chip } from "twenty-ui/components";
  
export const MyComponent = () => {
  return (
      <Chip 
      size="large"
      label="Disabled chip that triggers a tooltip when overflowing."
      clickable={false}
      variant="regular"
      accent="text-primary"
      leftComponent
      rightComponent
      maxWidth="200px"
      className
      />    
  );
};
```

## Enheds Chip

Et chip-lignende element til at vise information om en enhed.

<Tabs>

<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';
import { IconTwentyStar } from 'twenty-ui/display';
import { Chip } from 'twenty-ui/components';

export const MyComponent = () => {
  return (
    <Router>
      <Chip
        linkToEntity="/entity-link"
        entityId="entityTest"
        name="Entity name"
        pictureUrl=""
        avatarType="rounded"
        variant="regular"
        LeftIcon={IconTwentyStar}
      />
    </Router>
  );
};
```
