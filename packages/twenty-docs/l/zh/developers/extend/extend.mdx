## 概览

Twenty 的设计旨在实现可扩展性。 使用我们的 API、网络钩子和应用框架与您现有的工具集成，并构建自定义功能。

## 你可以做什么

* **API**：使用 REST 或 GraphQL 以编程方式查询和修改您的客户关系管理数据
* **网络钩子**：当 Twenty 中发生事件时接收实时通知
* **应用**：构建扩展 Twenty 功能的自定义应用程序 - 即将推出！
