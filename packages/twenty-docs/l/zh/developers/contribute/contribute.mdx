## 概览

Twenty 是开源的，欢迎社区贡献。 无论您是在修复错误、添加功能，还是改进文档，您的贡献都将帮助 Twenty 变得更好，惠及所有人。

## 贡献方式

* **报告错误**: 帮助识别并记录问题
* **提交功能**: 提出并实现新功能
* **改进文档**: 让我们的文档更清晰、更有帮助
* **前端开发**: 参与基于 React 的 UI 开发
* **后端开发**: 为 NestJS 服务器做出贡献
