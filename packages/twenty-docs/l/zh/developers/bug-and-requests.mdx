## 報告錯誤

要報告錯誤，請[在 GitHub 上創建一個問題](https://github.com/twentyhq/twenty/issues/new)。

您也可以在[Discord](https://discord.gg/cx5n4Jzs57)上尋求幫助。

## 功能請求

如果您不確定這是否為錯誤，而且您覺得更接近功能請求，那麼您應該[改為開啟討論](https://github.com/twentyhq/twenty/discussions/new)。
