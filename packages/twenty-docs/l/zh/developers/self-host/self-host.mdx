## 概览

Twenty 可以在您自己的基础设施上自托管，使您能够完全掌控您的数据和部署。

## 为什么选择自托管？

* **数据所有权**：将所有 CRM 数据保存在您自己的服务器上
* **合规性**：满足有关数据驻留的监管要求
* **自定义**：拥有完全权限来修改和扩展平台
