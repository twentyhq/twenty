仪表板为您的业务指标提供实时可见性——销售管道健康状况、团队绩效、收入趋势，以及您希望跟踪的任何其他内容。

## 小部件

每个仪表板都由小部件组成。 小部件是与您的 CRM 数据关联的单个图表或指标。 您可以配置：

* **图表类型** — 条形图、折线图、饼图、数字等
* **数据源** — 数据模型中的任何对象（标准或自定义）
* **筛选器** — 细化到特定记录、日期范围或细分
* **聚合** — 对任何数值字段进行计数、求和、平均值、最小值、最大值
* **分组** — 按所选字段、日期或关系进行分组

## 可跟踪的内容

* 按阶段划分的销售管道金额
* 随时间变化的成交交易
* 按来源划分的平均交易金额
* 任务完成率
* 任意对象上的自定义指标

## 共享

仪表板位于工作区级别——您团队中的每个人都可以看到它们。 以网格布局排列小部件并调整其大小，以构建适合您团队的视图。
