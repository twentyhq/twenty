Twenty 连接到您现有的工具，让您的 CRM 在无需手动输入数据的情况下保持最新。

## 电子邮件同步

连接您的 **Google Workspace** 或 **Microsoft 365** 邮箱。 连接后：

* 电子邮件会自动关联到匹配的公司和联系人记录
* 完整的电子邮件会话在每条记录的时间轴中可见
* 您可以直接从 Twenty 发送电子邮件
* 支持每位用户使用多个邮箱

您可以控制要导入的内容——按日期范围或发件人筛选，以避免导入不相关的电子邮件。

## 日历同步

日历事件会从您已连接的账户自动同步。 事件会出现在相关的 CRM 记录上，为您提供与每个联系人或公司的互动全景。

## 集成

除了电子邮件和日历之外，Twenty 还通过以下方式连接到外部工具：

| 方法              | 用例                            |
| --------------- | ----------------------------- |
| **API**         | 使用 GraphQL 或 REST API 构建自定义集成 |
| **Webhooks**    | 当记录发生变更时，将实时通知推送到外部系统         |
| **Zapier**      | 无需编码即可连接 5,000+ 应用            |
| **工作流 HTTP 操作** | 在自动化工作流中调用任意外部 API            |

## 自定义应用

开发者可以在 Twenty 之上构建功能完整的应用，添加自定义 UI、服务端逻辑以及深度集成。 应用可以发布给社区，也可以保持私有。
