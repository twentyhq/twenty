## 高级销售功能

GTM团队通常需要高级销售功能，如潜在客户评分、数据增强、循环分配、地区分配、自动提醒和电子邮件序列。 尽管这些不是Twenty系统中的内置功能，但利用我们的灵活工作流系统可以根据您的特定需求进行配置和定制。 雖然這些功能不是 Twenty 的內建功能，但可以透過我們靈活的工作流程系統進行配置和量身訂做，以滿足您的具體需求。

访问我们的[工作流部分](/l/zh/user-guide/workflows/getting-started-workflows)，了解如何逐步构建这些自动化。 有关详细示例，请参阅我们的[内部自动化](/l/zh/user-guide/workflows/internal-automations)和[外部工具集成](/l/zh/user-guide/workflows/external-tool-integration)指南。 For detailed examples, see our [Internal Automations](/l/zh/user-guide/workflows/internal-automations) and [External Tool Integration](/l/zh/user-guide/workflows/external-tool-integration) guides.
