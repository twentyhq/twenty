## 高级销售功能

GTM团队通常需要高级销售功能，如潜在客户评分、数据增强、循环分配、地区分配、自动提醒和电子邮件序列。 尽管这些不是Twenty系统中的内置功能，但利用我们的灵活工作流系统可以根据您的特定需求进行配置和定制。

访问我们的[工作流部分](/l/zh/user-guide/workflows/getting-started-workflows)，了解如何逐步构建这些自动化。 有关详细示例，请参阅我们的[内部自动化](/l/zh/user-guide/workflows/internal-automations)和[外部工具集成](/l/zh/user-guide/workflows/external-tool-integration)指南。
