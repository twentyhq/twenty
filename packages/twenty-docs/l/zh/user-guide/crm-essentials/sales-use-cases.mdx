## 先進的銷售功能

GTM 團隊經常需要如線索評分、數據豐富化、輪轉分配、地域分配、自動提醒和電子郵件序列等先進銷售功能。 雖然這些功能不是 Twenty 的內建功能，但可以透過我們靈活的工作流程系統進行配置和量身訂做，以滿足您的具體需求。

訪問我們的[工作流程部分](/l/zh/user-guide/workflows/getting-started-workflows)以一步一步學習如何構建這些自動化。 For detailed examples, see our [Internal Automations](/l/zh/user-guide/workflows/internal-automations) and [External Tool Integration](/l/zh/user-guide/workflows/external-tool-integration) guides.
