## 关键功能

* **基于角色的访问**：创建具有特定权限的自定义角色
* **对象权限**：控制谁可以查看、编辑或删除记录
* **字段权限**：限制对敏感字段的访问
* **设置权限**：控制对工作区配置的访问
* **SSO 集成**：为企业安全配置单点登录（Organization 计划）
