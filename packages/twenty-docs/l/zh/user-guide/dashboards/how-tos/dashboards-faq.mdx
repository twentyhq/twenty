---
title: 仪表板常见问题
description: 关于 Twenty 中仪表板的常见问题。
---

<AccordionGroup>
  <Accordion title="我可以创建私有仪表板吗？">
    不能。目前，凡能访问你的 Twenty 工作区的所有人都可以查看仪表板。 私有仪表板尚不可用。
  </Accordion>

  <Accordion title="为什么我的指标与同事的不一样？">
    仪表板目前根据查看者的时区显示数据。 如果你们位于不同地区（例如 APAC 与 US），即使是同一个仪表板，看到的数值也可能略有不同。 我们正在开发按仪表板设置时区的功能，以确保跨团队的数据一致。
  </Accordion>

  <Accordion title="我可以导出我的仪表板吗？">
    目前不支持导出仪表板。 此功能已在我们的路线图中。
  </Accordion>

  <Accordion title="我可以与外部用户共享仪表板吗？">
    不能。目前不支持与 Twenty 工作区外的用户（非 Twenty 用户）共享仪表板。
  </Accordion>

  <Accordion title="如何将仪表板添加到我的收藏？">
    打开你想收藏的仪表板，然后点击星形图标。 收藏是个人设置——不会影响其他用户。
  </Accordion>

  <Accordion title="选项卡与小部件有什么区别？">
    * **选项卡**将你的仪表板组织成多个部分（类似于仪表板内的页面）
    * **小部件**是每个选项卡中的独立可视化（图表、数值、iFrames）

    结构：仪表板 → 选项卡 → 小部件
  </Accordion>

  <Accordion title="我可以一次性筛选整个仪表板吗？">
    仪表板级筛选尚不可用，但该功能已在我们的路线图中。 目前需要分别对每个小部件应用筛选器。
  </Accordion>

  <Accordion title="是否提供仪表图或表格？">
    还不可以。 仪表图和表格已在我们的路线图中，将在后续版本中加入。
  </Accordion>

  <Accordion title="如何复制一个仪表板？">
    1. 请确保处于查看模式（而非编辑模式）
    2. 按 **Cmd + K** 打开命令栏（Windows 上为 **Ctrl + K**）
    3. 选择 **复制仪表板**
  </Accordion>

  <Accordion title="仪表板数据多久刷新一次？">
    随着你的客户关系管理（CRM）数据变化，小部件会自动更新：

    * 大多数指标实时更新
    * 如有需要，可使用刷新按钮手动更新
    * 历史数据会被保留以用于趋势分析
  </Accordion>
</AccordionGroup>
