使用強大的**筆記**功能高效管理與記錄相關聯的筆記。 本指南詳述如何在記錄頁面中無縫創建、格式化、評論和刪除筆記。

## 常見用例

### 會議紀錄

- **會議摘要**：記錄客戶通話中的關鍵討論點、決策和行動項目
- **通話摘要**：記錄銷售對話或支持通話中的重要細節
- **後續備註**：記錄會議中的後續步驟和承諾

### 客戶互動

- **支持歷史**：追踪客戶問題、提供的解決方案和解決狀態
- **銷售背景**：記錄客戶偏好、痛點和購買信號
- **關係建立**：記錄聯絡人的個人信息以增強關係

### 項目管理

- **進度更新**：記錄項目進展和里程碑成就
- **問題跟踪**：記錄遇到的問題和實施的解決方案
- **團隊交接**：在團隊成員之間轉移賬戶時共享背景信息

### 自動創建備註

使用 [工作流](/l/zh/user-guide/workflows/getting-started-workflows)自動創建備註：

- **通話記錄集成**：從錄音通話中自動生成會議摘要
- **交易交接備註**：在將新客戶移交給實施團隊時，自動創建銷售周期摘要

## 備註功能

### 關係字段

備註包括一個**關係**字段，允許您將單個備註附加到不同對象的多個記錄中。 例如，您可以將一個會議備註鏈接到：

- 您見面的那個人
- 他們代表的公司
- 所討論的機會
- 任何相關的任務或其他記錄

這種多重關係確保重要信息可從所有相關記錄頁面訪問。

### 用戶標籤

**注意**：目前備註中尚未支持用戶標籤功能。 該功能計劃於2026年推出，屆時您可以提及團隊成員並觸發通知。

## 創建備註

在系統中創建備註簡單且富有動態。 您可以：

- 導航到備註視圖並創建新記錄。
- 進入`記錄頁面`並選擇“備註”選項卡，然後按“新備註”按鈕。

<img src="/images/user-guide/notes/add-note.png"style={{width:'100%'}}/>

### 添加內容

直接開始輸入或按 `/` 鍵快速添加標題、文件或圖片等元素。

### 格式化內容

您可以直接在編輯器中格式化您的備註。 使用 Markdown 語法，按 `/` 鍵或單擊編輯器中的 `+` 圖標以檢視不同的區塊選項，例如標題、表格和清單。 您還可以附加圖片到您的備註中。

突出顯示文本以查看更多格式化選項，如粗體、斜體和對齊選項。

您還可以更改每個區塊的背景顏色和文字顏色以突出您備註中的重要內容。 要進行此操作，將鼠標懸停在您想格式化的區塊上，然後單擊 `⋮` 圖標旁邊的 `+` 圖標。 點擊 `顏色` 以打開所有文字和背景顏色選項。

## 查看備註

系統在相應的`記錄頁面`上的備註部分顯示與特定記錄鏈接的所有備註。

## 保存和刪除

對備註的所有編輯和添加都會自動保存。

要刪除備註：

1. 通過點擊`記錄頁面`中的備註來打開您希望刪除的備註。
2. 在備註選項卡中選擇您要刪除的備註。
3. 使用右下角的 `選項` 按鈕來提示其他操作，包括刪除。
4. 在確認模態中提示時完成刪除。

另一種刪除備註的方式是通過備註視圖，就像您刪除常規記錄一樣。 請注意，刪除的備註是永久性的，無法撤銷。
