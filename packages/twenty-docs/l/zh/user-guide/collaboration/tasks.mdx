使用**任務**功能來管理工作空間中的所有任務。 使用**任務**功能來管理工作空間中的所有任務。 This guide will show you how to create and manage tasks, switch between upcoming and completed tasks, edit task details, and much more.

## 常見用例

### 銷售跟進

- **會議後續步驟**：為客戶電話討論後的行動項目創建任務
- **提案跟進**：設置提醒檢查待定的提案
- **合同審核**：安排合同談判和批准的任務

### 客戶成功

- **新客戶上線任務**：在交易完成時自動創建客戶上線清單
- **檢查提醒**：安排定期的客戶健康檢查電話
- **續約準備**：提早90天設定合同續約的準備任務

### 內部項目管理

除銷售和客戶成功外，任務還支持更廣泛的組織需求：

- **產品開發**：跟踪功能發布、漏洞修復和開發里程碑
- **市場活動**：管理活動的啟動、內容創作和推廣活動
- **人力資源操作**：處理招聘流程、員工入職和績效評估
- **財務任務**：安排預算審核、發票處理和財務報告
- **運營**：協調設施管理、供應商關係和流程改進

### 自動化任務創建

使用[工作流程](/l/zh/user-guide/workflows/getting-started-workflows)自動創建任務：

- **交易達成觸發**：在機會結束時自動創建分配給 CS 團隊的上線任務
- **電子郵件提醒**：設置本週到期任務的每週電子郵件提醒（每週一發送）
- **管道自動化**：當交易在特定階段停滯時創建跟進任務
- **會議集成**：從會議錄音或日曆事件自動生成任務

## 任務功能

### 關係字段

任務包括一個`關聯`字段，允許您將單個任務附加到不同對象的多個記錄中。 例如，您可以將一個跟進任務鏈接到：

- 您需要聯繫的人
- 他們代表的公司
- 正在追求的機會
- 任何相關筆記或其他記錄

這種多對多關係確保任務可從所有相關記錄頁面訪問，並提供完整的上下文。

### 用戶標籤

**注意**：目前任務中不可用用戶標記功能。 此功能計劃於 2026 年推出，屆時您可以在分配或更新任務時提到團隊成員並觸發通知。

## 編輯任務

要編輯任務，請點擊其卡片。 這將打開側面面板，提供以下功能：

- **負責人和截止日期**：更新負責人或編輯截止日期。
- **評論**：透過在任務中添加評論，與團隊成員一起工作以提供更新或反饋。
- **Automations**: Thanks to the API and Webhooks, you can also automate task creation triggered by specific activities in your workspace.

## 標記任務為完成

要標記任務為完成：

1. 在您的`任務`頁面或一個`記錄頁面`上找到該任務。
2. 點擊任務卡左側的圓圈，該圓圈會變化以表示完成。
3. 任務狀態將自動更新為`已完成`。

此程序將幫助您保持成就的更新記錄。

## 刪除任務

要永久刪除任務：

1. 通過單擊其卡片，從`任務`頁面或在一個`記錄頁面`上，打開您想要刪除的任務。
2. 點擊任務詳細信息面板右上角的垃圾箱圖標。

請注意，刪除任務是永久性的，無法撤銷。 如果可能需要再次參考任務，考慮將其標記為 `已完成`。 如果可能需要再次參考任務，考慮將其標記為 `已完成`。
