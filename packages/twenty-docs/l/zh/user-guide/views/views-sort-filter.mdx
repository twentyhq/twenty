## 關於視圖

通過創建視圖，您可以以不同的方式查看您的記錄。 在視圖中，您可以應用篩選和排序以有效地組織您的內容。 例如，在`People`對象上，您可以創建一個視圖來過濾具有美國電話號碼的聯繫人以便隔離美國聯繫人。

### 預設視圖

每個對象都有一個無篩選、無排序和不可刪除的默認視圖。 它以對象的複數名稱命名，例如"所有公司","所有人員","所有機會"。

## 創建、編輯和刪除視圖

您可以創建幾個自定義視圖並與您的團隊共享。

### 創建視圖

創建新視圖有兩種方式。 可以直接從`View Switcher`，或通過篩選和排序現有視圖來創建。

#### 從視圖切換器

1. 打開視圖切換器
2. 單擊視圖切換器菜單底部的 "新增視圖" 按鈕。
3. 為您的視圖選擇一個圖標和名稱。
4. 選擇 "視圖類型"，選擇表格或看板。
5. (僅適用於看板視圖) 為您的看板視圖選擇要用作列標題的選擇字段。 如果您的對象缺少一個`Select`類型字段，系統將提示您先從設置中創建一個才能啟用看板視圖創建功能。
6. 點擊"創建"來生成您的新視圖。

新創建的視圖將自動打開。

<VimeoEmbed videoId="927639721" title="Video demonstration" />

#### 從排序和篩選

當您更改現有視圖的排序和篩選時，"另存為新視圖"按鈕將出現在 "視圖欄"的右邊緣。 這將打開上述的新視圖菜單，允許您從現有視圖創建一個新視圖。

<VimeoEmbed videoId="927643495" title="Video demonstration" />

### 編輯和刪除視圖

要編輯或刪除視圖：

1. 打開視圖切換。
2. 懸停在您想編輯的視圖上。
3. 點擊懸停時顯示的三個點來編輯和刪除視圖。

點擊編輯後，您可以更改視圖的圖標和名稱或完整刪除視圖。

<VimeoEmbed videoId="927645774" title="Video demonstration" />

## 在視圖之間導航

要在視圖之間切換，只需打開視圖切換器並點擊您想打開的視圖。

## 自定義視圖

每個視圖都可以自定義以簡化您的業務流程。 視圖有助於細分您的數據。 您可以使用篩選、排序和字段顯示來自定義視圖。

### 篩選視圖

要篩選視圖：

- 點擊**篩選**>選擇一個字段(如：名稱、公司在People對象中)。
- 選擇一個條件，選擇一個值，並應用篩選。
- 使用`+ 新增篩選`添加更多篩選，或者使用**X**刪除它們。

<VimeoEmbed videoId="926282262" title="Video demonstration" />

### 排序視圖

按升序或降序排列欄位數據：

- 選擇**排序**，選擇一個字段，並定義您希望的排序順序。
- 您可以根據需要應用和安排多個排序。
  <VimeoEmbed videoId="927885588" title="Video demonstration" />

### 欄位顯示

You can customize which fields to display within a view. 要隱藏某個字段，點擊**選項**>**字段**，然後懸停選擇隱藏圖標，或點擊字段列頭選擇**隱藏**。

您可以通過點擊字段列頭，然後按**向右移動**或**向左移動**來重新排列字段。

### 打開記錄

要打開記錄，請點擊第一列中的名稱。 此操作將作為側面板打開相應的記錄頁。
