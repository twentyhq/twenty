## 初始设置

首次创建工作区时，需要配置一些关键设置。

### 工作区名称和徽标

1. 转到 **设置 → 常规**
2. 更新您的工作区名称
3. 上传您的公司徽标
4. 保存更改

### 时区和日期格式

1. 转到 **设置 → 体验**
2. 选择您的时区
3. 选择您首选的日期格式
4. 保存更改

## 关键配置

### 连接电子邮件和日历

设置电子邮件和日历同步:

1. 转到 **设置 → 账户**
2. 点击**添加账户**
3. 连接您的 Google 或 Microsoft 账户
4. 配置同步设置

→ [完整的电子邮件与日历设置指南](/l/zh/user-guide/calendar-emails/overview)

### 邀请您的团队

将团队成员添加到您的工作区:

1. 转到 **设置 → 成员**
2. 点击 **+ 邀请**
3. 输入电子邮件地址
4. 分配合适的角色

<Note>
  在邀请团队之前，请在 **设置 → 角色** 中检查默认角色。 新成员加入时会自动被分配此角色。
</Note>

## 工作区设置清单

* 工作区名称和徽标已配置
* 已设置时区和日期格式
* 已连接电子邮件和日历
* 已邀请团队成员
* 已配置角色与权限
