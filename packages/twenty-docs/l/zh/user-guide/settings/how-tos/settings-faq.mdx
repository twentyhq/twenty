## 工作区设置

<AccordionGroup>
  <Accordion title="如何更改我的工作空间名称？">
    1. 转到 **设置 → 常规**
    2. 找到“工作空间名称”字段
    3. 输入您的新名称
    4. 更改会自动保存
  </Accordion>

  <Accordion title="如何更新我的工作空间徽标？">
    1. 转到 **设置 → 常规**
    2. 点击当前徽标或上传区域
    3. 选择图像文件（PNG、JPEG 或 GIF，大小小于 10MB）
    4. 徽标会立即更新
  </Accordion>

  <Accordion title="我可以拥有多个工作空间吗？">
    可以，您可以创建并成为多个工作空间的成员。 每个工作空间都有自己独立的数据、设置和订阅。
  </Accordion>

  <Accordion title="如何删除我的工作空间？">
    1. 转到 **设置 → 常规**
    2. 滚动到危险区域
    3. 点击 **删除工作空间**
    4. 确认删除

    注意：此操作将永久删除所有数据，且无法撤销。
  </Accordion>

  <Accordion title="我不小心创建了多个工作空间。 我应该怎么办？">
    在 **设置 → 常规 → 删除工作空间** 中删除不再需要的工作空间。

    <Warning>
      不要删除您的 **账户**（可在 设置 → 个人资料 下访问）：您的账户在所有工作空间中共享。 删除您的账户会移除您对所有工作空间的访问权限。
    </Warning>
  </Accordion>

  <Accordion title="如何在不删除的情况下禁用我的工作空间？">
    如果您想暂时禁用您的工作空间（而非永久删除），请前往 **设置 → 账单** 并点击 **取消计划**。 您的数据将在宽限期内保留。
  </Accordion>
</AccordionGroup>

## 个人资料设置

<AccordionGroup>
  <Accordion title="如何更改我的密码？">
    1. 转到 **设置 → 个人资料**
    2. 找到“密码”部分
    3. 输入您的当前密码
    4. 输入您的新密码
    5. 保存更改
  </Accordion>

  <Accordion title="如何启用双重身份验证？">
    1. 转到 **设置 → 个人资料**
    2. 找到“2FA”部分
    3. 单击 **启用 2FA**
    4. 使用您的身份验证应用扫描 QR 码
    5. 输入验证码
  </Accordion>

  <Accordion title="我可以更改我的电子邮件地址吗？">
    若要更改您的电子邮件地址，请联系 [contact@twenty.com](mailto:contact@twenty.com)。
  </Accordion>

  <Accordion title="如何删除我的账户？">
    1. 转到 **设置 → 个人资料**
    2. 滚动到危险区域
    3. 单击 **删除帐户**
    4. 输入您的电子邮件地址以确认

    注意：此操作会移除您对所有工作空间的访问权限，并删除从已连接账户同步的所有电子邮件。
  </Accordion>
</AccordionGroup>

## 体验设置

<AccordionGroup>
  <Accordion title="如何在浅色和深色模式之间切换？">
    1. 转到 **设置 → 体验**
    2. 找到“主题”部分
    3. 选择浅色、深色或系统
  </Accordion>

  <Accordion title="如何更改日期格式？">
    1. 转到 **设置 → 体验**
    2. 找到“日期格式”
    3. 选择您偏好的格式
    4. 更改会立即生效
  </Accordion>

  <Accordion title="如何更改我的时区？">
    1. 转到 **设置 → 体验**
    2. 找到“时区”
    3. 选择您所在的时区
    4. 所有时间戳都会相应调整
  </Accordion>

  <Accordion title="我可以更改语言吗？">
    1. 转到 **设置 → 体验**
    2. 找到“语言”
    3. 从可用语言中进行选择
    4. 界面将根据您的选择进行更新
  </Accordion>
</AccordionGroup>

## 账户设置

<AccordionGroup>
  <Accordion title="如何连接我的电子邮件账户？">
    1. 转到 **设置 → 账户**
    2. 点击**添加账户**
    3. 选择 Google 或 Microsoft
    4. 授权访问
    5. 配置同步设置
  </Accordion>

  <Accordion title="我可以连接多个电子邮件账户吗？">
    可以，您可以连接多个电子邮件账户。 转到 **设置 → 账户**，按需添加其他账户。
  </Accordion>

  <Accordion title="如何断开电子邮件账户的连接？">
    1. 转到 **设置 → 账户**
    2. 找到要移除的账户
    3. 点击 **断开连接**
    4. 确认该操作
  </Accordion>
</AccordionGroup>

## 域名

<AccordionGroup>
  <Accordion title="我可以自定义我的工作空间域吗？">
    是! 转到 **设置 → 域名** 并点击 **自定义域名**。 您有两个选项：

    * **子域名**：使用一个 Twenty 子域名，如 `yourcompany.twenty.com`
    * **自定义域名**：使用您自己的域名，如 `crm.yourcompany.com`（需要 DNS 配置）

    子域名设置快捷，而自定义域名可为您的团队提供完整的品牌化体验。
  </Accordion>

  <Accordion title="批准的访问域名如何工作？">
    您可以配置批准的访问域名，使拥有公司电子邮件地址的团队成员能够自动加入您的工作空间。 转到 **设置 → 域名** 并添加您的公司域名（例如，`yourcompany.com`）。
  </Accordion>
</AccordionGroup>

## 早期访问功能

<AccordionGroup>
  <Accordion title="什么是早期访问功能？">
    早期访问功能是在正式发布前进行测试的功能。 目前，只有**仪表板**处于早期访问阶段。 它们可能会根据用户反馈而改变。
  </Accordion>

  <Accordion title="早期访问功能使用安全吗？">
    早期访问功能可正常使用，但可能会根据用户反馈而改变。 它们可以安全使用，但在正式发布之前可能会有所变化。
  </Accordion>

  <Accordion title="如何启用早期访问功能？">
    1. 前往 **设置 → 更新 → 早期访问**
    2. 找到您想要的功能
    3. 将其切换为开启
    4. 该功能将立即可用
  </Accordion>
</AccordionGroup>
