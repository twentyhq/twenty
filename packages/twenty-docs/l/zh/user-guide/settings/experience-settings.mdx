## 外觀

### 主題選擇

Choose between light and dark modes:

- **亮模式**：明亮清晰的界面，適合光線充足的環境
- **暗模式**：適合低光環境，減少眼睛疲勞
- **系統**：自動匹配您的設備主題設置

## 地區設定

### 語言

選擇您在Twenty界面的首選語言：

- 英語（預設）
- 其他語言基於社區翻譯提供

### 時區

為準確安排和時間戳設置本地時區：

- 影響會議時間、任務期限和活動日誌
- 自動調整以適應夏令時

### 日期格式

選擇Twenty中的日期顯示方式：

- **MM/DD/YYYY**（美國格式）
- **DD/MM/YYYY**（歐洲格式）
- **YYYY-MM-DD**（ISO格式）

### 數字格式

配置數字和貨幣的顯示方式：

- **小數分隔符**：逗號（,）或句號（.）
- **千位分隔符**：空格、逗號或句號
- **貨幣符號**：依據您的地區或自定義

### 日曆格式

設置首選的日曆布局：

- **每周的第一天**：星期日或星期一
- **週數**：顯示或隱藏ISO週數
- **時間格式**：12小時（AM/PM）或24小時格式

## 如何更新設定

1. 從側邊欄轉到**設定 → 體驗**
2. 在每個部分調整您的偏好
3. 更改將自動保存
4. 刷新您的瀏覽器以查看所有變更生效
