## 个人信息

### 姓名和邮箱

- **显示名称**：更新您的姓名对其他工作空间成员的显示方式
- **电子邮件地址**：更改您的登录电子邮件（需要验证）
- **个人头像**：上传自定义头像或使用您的姓名首字母

## 安全设置

### 双因素认证 (2FA)

启用 2FA 为您的帐户增加一个额外的安全层：

1. 转到 **设置 → 个人资料设置**
2. 单击 **启用 2FA**
3. 使用您的身份验证应用扫描 QR 码
4. 输入验证码以确认

### 密码管理

- **更改密码**：更新您当前的密码
- **密码要求**：至少 8 个字符
