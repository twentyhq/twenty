## 個人資訊

### 姓名和電子郵件

- **Display Name**: Update how your name appears to other workspace members
- **電子郵件地址**：更改您的登錄電子郵件（需要驗證）
- **個人資料圖片**：上載自定義頭像或使用您的縮寫

## 安全設定

### Two-Factor Authentication (2FA)

啟用 2FA 以增加一層額外的安全性在您的帳號上：

1. 前往 **設定 → 個人資料設定**
2. 點擊 **啟用 2FA**
3. 使用您的驗證器應用掃描 QR Code
4. 輸入驗證碼進行確認

### 密碼管理

- **更改密碼**：更新您當前的密碼
- **Password Requirements**: Must be at least 8 characters long

## 個人資料管理

### 刪除帳號

<Warning>
刪除您的帳號將永久移除您對所有工作空間的訪問。 此操作無法撤銷，您將失去所有工作空間權限，如您只想退出特定的團隊，應考慮退出個別的工作空間。
</Warning>

要刪除帳號：

1. 前往 **設定 → 個人資料設定**
2. 滾動至 **危險區域**
3. 點擊 **刪除帳號**
4. 通過輸入您的電子郵件地址確認
