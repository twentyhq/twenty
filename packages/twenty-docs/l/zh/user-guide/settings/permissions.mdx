Twenty的權限系統允許您控制訪問三個主要區域：

- **對象和字段**：控制誰能查看、編輯或刪除記錄和單個字段
- **設置**：管理對工作區配置和管理功能的訪問權
- **操作**：控制一般工作區操作，如導入數據或發送電子郵件

## 創建角色

要創建新角色：

1. 進入**設置 → 角色**
2. 在**所有角色**下，點擊\*\*+ 創建角色\*\*
3. 輸入角色名稱
4. 在默認的**權限**選項卡中，配置權限
5. 點擊**保存**完成

## 刪除角色

要刪除角色：

1. 進入**設置 → 角色**
2. 點擊您想要刪除的角色
3. 打開**設置**選項卡，然後點擊**刪除角色**
4. 在模態中點擊**確認**

注意：如果刪除角色，任何分配了該角色的工作區成員將自動重新分配到默認角色。 除了**管理員**角色外，所有角色都可以刪除。 必須始終至少有一名成員分配到**管理員**角色。

## 將角色分配給成員

### 查看當前分配

- 進入**設置 → 角色**
- 查看所有角色以及每個角色分配了多少成員
- 查看哪些成員擁有哪些角色

### 為成員分配角色

1. 進入**設置 → 角色**
2. 點擊您想要分配的角色
3. Open the **Assignment** tab
4. Click **+ Assign to member**
5. 從列表中選擇工作區成員
6. 確認分配

### 設置默認角色

1. 進入**設置 → 角色**
2. 在**選項**部分，找到**默認角色**
3. 選擇新成員應自動接收的角色
4. 當新工作區成員加入時，將被分配此角色

**注意**：您只能將角色分配給現有的工作區成員。 要邀請新成員，請使用[成員管理](/l/zh/user-guide/settings/member-management)。

## 自定義權限

權限決定每個角色能夠訪問或修改在您的工作區中的內容，包括工作區對象的記錄、設置和操作。

### 對象和字段權限

控制訪問記錄和單個字段：

#### 物件級權限

- 在**所有對象**下，應用如**查看記錄**、**編輯記錄**、**刪除記錄**或**銷毀記錄**等權限到所有對象
- 在**對象級別權限**下，為個別對象配置例外情況。 這些設定將覆蓋**所有對象**的設定

#### 字段級別權限

- 為每個對象中的單個字段配置權限
- 控制誰可以**查看字段**、**編輯字段**或對特定字段有**無訪問權限**
- 字段權限的運作方式與對象權限相同，包含繼承和覆蓋

#### 管理權限覆蓋

要覆蓋父權限並應用更嚴格的規則：

- 點擊**X**移除繼承的規則
- 選擇所選對象或字段的具體權限
- 點擊橙色的**撤銷**圖標（圓形箭頭）來恢復更改

完成後，點擊**完成**，然後在重定向到角色頁面後點擊**保存**。

### 工作區設置權限

以兩種方式控制訪問工作區設置：

- 切換**設置全部訪問**以授予完整訪問權限
- 或啟用具體權限（例如，API密鑰生成、工作區偏好、角色分配、數據模型配置、安全設置和工作流管理）

### 工作區操作權限

控制訪問常規工作區行動：

- 切換**應用程序全部訪問**以授予完整權限
- 或啟用個別操作，如**發送電子郵件**、**導入CSV** 和 **導出CSV**
