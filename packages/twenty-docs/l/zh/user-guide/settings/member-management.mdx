## 邀請新成員

### 使用邀請連結

1. 前往 **設置 → 成員**
2. 複製工作區邀請連結
3. 與新團隊成員分享連結
4. 他們註冊後即可獲得訪問權限

### 直接電郵邀請

1. 前往 **設置 → 成員**
2. 輸入該人的電子郵件地址
3. 點擊 **邀請**
4. 他們將收到電郵邀請

## 移除成員

### 刪除成員

1. 前往 **設置 → 成員**
2. 找到要移除的成員
3. 點擊其姓名旁的刪除/移除按鈕
4. 確認移除

**注意**：被移除的成員會立即失去訪問權限，但稍後可以重新邀請。
