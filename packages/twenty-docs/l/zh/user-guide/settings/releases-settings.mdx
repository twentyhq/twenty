## 最新釋出

跟蹤 Twenty 的開發進度：

- **版本說明**：查看每個版本的新特性
- **功能更新**：發現新能力和改進

## 實驗室功能

在正式釋出前訪問實驗功能：

- **Beta 測試**：在開發期間嘗試新功能
- **功能標誌**：啟用或禁用特定實驗性功能

## 如何訪問

1. 前往 **設置 → 釋出**
2. 查看版本說明並訪問 **Lab** 標籤以獲取 beta 功能

如需更多社群資源，請檢查我們的 [資源部分](/l/zh/user-guide/resources/glossary)。
