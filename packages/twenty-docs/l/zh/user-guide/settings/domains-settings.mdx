## 自定義工作空間網域

設置個性化的工作空間網址：

- 選擇您首選的子域名（例如，yourcompany.twenty.com）
- 為專業的品牌塑造設置自定義網域

## 已批准的訪問網域

允許特定電子郵件域的用戶自動訪問工作空間。

### 運作方式

- 添加您公司的電子郵件域名（例如，@yourcompany.com）
- 這些域的任何電子郵件用戶可以自動加入您的工作空間
- 團隊成員不需要手動邀請

### 優勢

- **更快的上手**：新成員自動加入
- **更好的安全性**：只有經驗證的公司電子郵件可以訪問
- **減少管理工作**：不需要手動邀請各成員

## 設置指南

1. 前往 **設置 → 網域**
2. 添加您的自定義網域或已批准的電子郵件網域
3. 如有需要，驗證域名所有權

**注意**：網域更改可能需要時間生效，並可能需要驗證。
