## 當前集成選項

### 本地集成

Twenty目前提供與以下平台的本地集成：

- **電子郵件&日曆**：連接Gmail、Outlook或SMTP/CalDAV提供者
- **API Access**: Use our REST and GraphQL APIs to build custom integrations

有關電子郵件和日曆設置，請訪問[電子郵件&日曆設置](/l/zh/user-guide/settings/email-calendar-setup)。

### 工作流（推薦）

將Twenty連接到其他工具的主要方式是通過我們應用內的**工作流**功能：

- **HTTP節點**：向外部服務發送API調用
- **代碼節點**：編寫複雜集成所需的自定義邏輯
- **Webhook Triggers**: Receive data from external systems
- **字段更新觸發器**：根據CRM變更自動執行操作

在我們的[工作流部分](/l/zh/user-guide/workflows/getting-started-workflows)了解更多。

### Zapier集成（傳統）

我們維護了一個Zapier集成，適合偏好無代碼自動化的用戶：

1. 訪問[Twenty on Zapier](https://zapier.com/apps/twenty/integrations)
2. 以Twenty作為觸發器或操作創建一個新的Zap
3. Generate an API key in Settings → API & Webhooks
4. 將你的Twenty工作區連接到Zapier

## 未來願景：社區製作連接器（2026年）

### 擴展平台

我們正在構建一個擴展平台，允許開發者將應用程序作為代碼創建。 這將實現：

- **自定義連接器**：與您最愛的軟件建立集成
- **社區貢獻**：分享並發掘他人構建的連接器
- **靈活架構**：將Twenty的能力擴展至核心功能之外

### AI支持的工作流構建

2026年推出，AI助手將幫助用戶：

- **自動生成工作流**：用簡單的語言描述您的集成需求
- **智能建議**：獲得連接特定工具的推薦
- **模板庫**：訪問常見用例的預構建工作流

Twenty集成的未來是以社區為驅動，以人工智能為輔助，並具備無限可擴展性。
