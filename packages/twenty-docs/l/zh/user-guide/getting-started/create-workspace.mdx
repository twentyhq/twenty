## 步驟1：註冊

1. 導航到 [Twenty 註冊](https://app.twenty.com)。
2. 選擇您的首選註冊方法：
   - **繼續使用 Google** 以使用 Google 帳戶註冊。
   - **繼續使用 Microsoft** 以使用 Microsoft 帳戶註冊。
   - 或，**繼續使用電子郵件** 進行電子郵件註冊。

<VimeoEmbed videoId="927066829" title="Video demonstration" />
## Step 2: Choosing a Trial Period

在兩個試用期之間選擇：

### 30天

使用信用卡

### 7天

不使用信用卡

兩個試用都包括：

- 完全訪問
- 無限聯繫人
- 電子郵件集成
- 自定義對象
- API 和 Webhooks

您可以點擊 "更改計劃" 選擇不同的計劃或計費週期。

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## 步驟3：付款確認與帳戶設置

經由 Stripe 批准付款後，您將會被引導創建您的工作空間和用戶資料。 請記住，您可以隨時取消訂閱。
