## 1. 自定義您的數據模型

Twenty 提供您需要的靈活性，讓您構建最能支持您日常運作的數據模型。
創建任何類型的對象和字段，包括您不同對象之間的關係。 您可以在設置 → 數據模型下進行。
以下是一些提示：

- **您在自定義字段或自定義對象的數量上不受限制**。 添加自定義對象和字段不會導致計劃升級。
- **人、公司和機會是可從郵箱/日曆同步訪問電子郵件和會議的三個對象**。 我們建議盡量使用這些，並在必要時添加字段以分類您的記錄。 這是一個例子：
  - It is best to use the People object for your prospects and partners, creating a field on the People object named `Person Type`, instead of creating a Partner custom object. 因為您無法從合作夥伴記錄中訪問與此人交換的電子郵件。
  - 在"人物"下創建不同的視圖，一個顯示合作夥伴，一個顯示潛在客戶。
- Two People cannot have the same email address. 兩家公司不能擁有相同的域名。
- 您可以停用不想使用的標準字段和對象。
- 您可以隱藏視圖中的字段：別擔心創建字段，您不必顯示它們的全部。

閱讀[本文](https://docs.twenty.com/l/zh/user-guide/data-model/customize-your-data-model)學習如何設計您的數據模型。

## 2. 輸入數據

將現有數據導入Twenty可以讓您的團隊從一開始就有上下文。

### 連接您的郵箱

如果您在創建工作區時尚未完成此操作，請在設置 → 帳號下連接您的**Google或Microsoft帳號**。 這允許Twenty： 這允許Twenty：

- 導入您的郵件和會議
- 根據互動（可選）自動創建聯繫人
- 保持您的團隊看到的通信歷史記錄可見

**使用其他供應商嗎？**
您可以通過SMTP添加其他郵箱，或通過CalDAV添加其他日曆。 您需要在設置 → 發行版本 → 實驗室中激活該功能，然後回到設置 → 帳號標籤。 您需要在設置 → 發行版本 → 實驗室中激活該功能，然後回到設置 → 帳號標籤。

### 通過csv導入數據

使用命令菜單（`Cmd + K`或`Ctrl + K`）通過CSV導入人物、公司、機會或任何自定義對象。

**關鍵指南**：

- 下載示例文件以了解預期格式
- 將每個文件限制為10k記錄
- 刪除人物的重複電子郵件或公司的重複域
- 在導入之前審核並修復錯誤（以黃色標記）。

閱讀[本文](/l/zh/user-guide/getting-started/import-export-data)以了解更多關於數據導入的內容。

## 3. 創建您的第一個視圖

創建不同的視圖是確保數據對您的團隊有用的關鍵。
執行步驟如下：

- **添加或隱藏列**
  通過在右上角點擊選項 → 欄位管理給定視圖中可見的欄位。 您可以在那裡顯示/隱藏字段。 您可以在那裡顯示/隱藏字段。

- **重新排序字段**
  通過在右上角點擊選項 → 欄位重新排序給定視圖中的欄位。 拖放欄位以重新排序它們。

- **篩選您的視圖**
  使用右上角的篩選器縮小顯示的記錄。

- **排序記錄**
  使用右上角的排序功能重新排序顯示的記錄，或直接點擊列名。

- **選擇佈局**
  只要對象有`Stage`或類似的選擇型字段，您可以切換到**看板佈局**或列表中的**分組按**佈局。

- **Save your view as Favorites**
  This can be done using the dropdown menu showing the different views.
