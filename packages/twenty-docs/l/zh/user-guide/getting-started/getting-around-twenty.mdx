當您第一次登錄Twenty時，界面應該感覺直觀。 它的設計旨在幫助您快速移動並保持有序，而不會妨礙您。

## The Main Layout

屏幕的中心是**您的記錄所在**：人員，公司，機會，任務，筆記，工作流程以及您創建的其他任何對象。 這是日常工作的發生地。

## 指令菜單與快速搜索

命令菜單為您**提供快速操作與搜索的途徑**。 您可以通過兩種方式進入： 您可以通過兩種方式進入：

- **鍵盤快捷鍵**：按`Cmd + K`（Mac）或`Ctrl + K`（Windows）
- **鼠標**：點擊右上角的三個點

在側邊欄頂部還能看到搜索欄進行快速記錄搜索，或按`/`鍵即可快速聚焦。

從這裡，您可以：

- 創建新記錄
- **Import and export data via csv**
- Create new views
- 訪問刪除的記錄（Twenty支持軟刪除和硬刪除）
- 查看快捷鍵以快速訪問您工作空間中的對象

## 視圖

每個對象（例如機會或人員）都支持多種視圖。 您在每個對象的視圖數量上不受限制。

使用版面左上角的下拉菜單在不同視圖之間切換。 例如：

- 使用看板視圖按階段追踪機會
- 使用分組視圖創建部分並提高效率
- 使用篩選器專注於特定記錄（例如：上周創建的潛在客戶）
- 保存篩選後的視圖以便稍後重用
- 收藏視圖以便快速訪問

如果您不熟悉視圖，請閱讀我們的[視圖管理](/l/zh/user-guide/crm-essentials/view-management)文章，了解如何創建和自定義。

## 設置

從左上角打開您的設置：

- **連接您的郵箱和日曆**帳戶，以實現無縫的郵件和日曆同步
- 自定義您的**數據模型**：創建自定義對象、字段和關係
- **訪問API沙盒並配置Webhooks**
- **管理用戶權限**和工作區訪問控制
- 邀請團隊成員並管理用戶角色
- 編輯個人資料和工作區偏好
- 配置賬單並監控工作流點數使用情況
- 發現最新的版本和即將推出的功能（在“發布 → 實驗室”選項卡下）

如果您沒有在設置中看到所有這些部分，請聯繫您的工作區管理員——其中一些具有訪問限制。
