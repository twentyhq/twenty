## 匯入資料

- 您可以使用 .csv、.xlsx 或 .xls 檔案來匯入任何對象的資料。
- 每個您上傳的檔案需只包含 **一種類型的對象** （例如，只包括人員記錄）。
- You can use the Import to **create or update records**.

### 下載樣本檔案以符合預期格式

1. 進入您即將匯入的對象的視圖。
2. 點擊右上角的 `⋮` 圖標，然後選擇 `匯入記錄`。
3. 點擊 `下載樣本檔案`。

### 準備您的 csv

在上傳您的檔案之前，以下是一些需要檢查的項目。

- 將檔案中的記錄數量限制為 **每個檔案 10,000** 條。
- **移除重複項目** 從您的檔案中
  - `People` 的唯一性預設由 `id` 和 `email` 設定。 配置您的資料模型時，您還可以將 `People` 中的自定義字段定義為唯一。 配置您的資料模型時，您還可以將 `People` 中的自定義字段定義為唯一。
  - `Companies` 的唯一性預設由 `id` 和 `domain` 設定。 配置您的資料模型時，您還可以將 `Companies` 中的自定義字段定義為唯一。 配置您的資料模型時，您還可以將 `Companies` 中的自定義字段定義為唯一。
  - 對於其他對象，包括自定義對象，您可以將某些字段定義為唯一。 準備檔案時，請確保不包含重複項目。
- 我們建議在上傳域名時使用語法 `https://domain.com`，因為這是我們的連接器與您的郵箱和日曆一起使用的語法。

### 上傳您的檔案

1. 進入您即將匯入的對象的視圖。
2. 點擊右上角的 `⋮` 圖標，然後選擇 `匯入記錄`。
3. 點擊 `選擇檔案`。
4. 驗證字段的映射。

- 您不必全部匯入，可以選擇"不映射"選項。
- 對於關係，建議只映射一個唯一字段。
- 您可能還需要映射您的選擇類型和多選類型字段的值。

5. 點擊`下一步`並`檢查有問題的行`。 有問題的單元格會被突出顯示。 **您可以直接從那裡刪除行或更新單元格**。
6. 完成後，點擊`確認`

### 匯入常見問題

<details><summary>上傳檔案時我看到重複問題，我該怎麼辦？</summary>

請參照本文上面的 **準備您的 csv** 部分，裡面包含關於什麼會被視為重複的指南。

</details>

<details><summary>我可以匯入對象之間的關係嗎？</summary>

是的，請參閱本文上面的 **準備您的 csv** 部分，其中包含關於關係匯入的部分。

</details>

<details><summary>我可以使用匯入功能更新現有記錄嗎？</summary>

是的，您可以使用匯入功能更新現有的記錄。 再次上傳記錄時，請確保提供 id（或任何其他唯一字段）。

</details>

<details><summary>我可以從其他工具導入 `id` 嗎？</summary>

是. 您需要創建一個字段，您將其定義為唯一，它將包含來自其他工具的 `id`。 請注意，名稱 `id` 是受保護的，因為它用於 Twenty id。  
如果您希望使用此字段創建對象之間的關係，請參閱本文上面的 **準備您的 csv** 部分。 其中包含關於關係匯入的部分。

</details>

## 數據導出

您可以從大多數對象中下載資料，並且每次匯出最多 20,000 條記錄。
要從對象中導出資料：

1. 訪問對象索引。
2. 選擇資料匯出視圖。 配置您要下載的欄位，通過隱藏或添加欄位並通過篩選視圖找到所需的記錄。
3. 點擊右上角的 `⋮` 圖標訪問側邊欄。
4. 點擊 `匯出視圖`。
5. 選擇 CSV 資料的保存位置。 請注意，對於大記錄數量的匯出可能需要時間。 請注意，對於大記錄數量的匯出可能需要時間。
