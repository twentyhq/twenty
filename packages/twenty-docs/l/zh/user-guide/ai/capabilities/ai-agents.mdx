## 概览

将 AI 功能直接集成到您的自动化工作流中，以实现智能数据处理和决策。

## 功能

| 功能        | 描述                  |
| --------- | ------------------- |
| **AI 操作** | 将 AI 驱动的步骤添加到任何工作流中 |
| **数据充实**  | 自动使用外部数据充实记录        |
| **分类**    | 基于内容分析对记录进行分类       |
| **摘要**    | 从文本字段生成摘要           |
| **自定义提示** | 精确定义 AI 如何处理您的数据    |

## 使用场景

* **线索评分**：自动为入站线索打分并设定优先级
* **数据清理**：标准化公司名称和联系信息
* **电子邮件草稿**：基于会议记录生成后续电子邮件
* **记录路由**：根据内容将记录分配给合适的团队成员
