## 了解视图

视图是已保存的配置，用于决定数据的显示方式。 每个视图都可以有：

* **布局**：表格、看板或日历
* **筛选器**：显示哪些记录
* **排序**：记录的排序方式
* **字段**：哪些列可见

## 视图类型

### 表视图

默认的类似电子表格的视图，以行显示记录，并可自定义列。

### 看板视图

记录以卡片形式按阶段组织显示的可视化看板视图。 适用于：

* 销售管道
* 项目跟踪
* 任何具有明确阶段的工作流

### 日历视图

在日历上显示包含日期字段的记录。 非常适合：

* 会议和活动
* 截止日期和到期日
* 基于时间的计划

## 创建视图

有两种方法可以创建新视图。

### 使用视图下拉菜单

1. 导航到任意对象（人员、公司等）
2. 点击左上角的视图名称（显示当前视图并带有下拉箭头）
3. 点击 **+ 添加视图**
4. 为视图命名并点击 **创建**
5. 在 **选项** 下选择布局（表格、看板或日历）
6. 按需添加筛选器和排序
7. 选择要显示的字段并重新排序它们
8. 单击 **保存**

### 从编辑现有视图开始

1. 导航到任意对象（人员、公司等）
2. 在 **选项** 下选择布局（表格、看板或日历），或按需添加筛选器和排序
3. 点击 **另存为新视图**
4. 为视图命名并点击 **创建**
5. 继续编辑新视图
6. 点击 **更新视图** 以保存新增的配置

## 管理视图

### 编辑视图

1. 从下拉菜单中选择视图
2. 进行更改（筛选器、排序、列）
3. 点击 **保存** 以更新视图

### 重命名视图或更改其图标

1. 打开视图下拉菜单
2. 点击视图名称旁的 **⋮** 菜单
3. 选择 **编辑**
4. 更改名称或图标
5. 单击 **保存**

### 重新排序视图

1. 打开视图下拉菜单
2. 点击并通过其手柄拖动视图
3. 将其放到所需位置
4. 新的顺序会自动保存

### 添加到收藏夹

将常用视图固定，便于快速访问：

1. 打开视图下拉菜单
2. 点击视图旁的 **⋮** 菜单
3. 选择 **添加到收藏夹**
4. 该视图会显示在收藏夹区域

### 删除视图

1. 选择要删除的视图
2. 点击视图下拉菜单
3. 点击该视图旁的 **⋮** 菜单
4. 选择 **删除**
5. 确认删除

<Warning>
  已删除的视图无法恢复。 确认前请确保确实要将其删除。
</Warning>

## 视图可见性

每个视图（默认的 "All [Object Name]" 视图除外）都有其各自的可见性设置。

要更改可见性：

1. 打开视图
2. 点击 **选项 → 可见性**
3. 选择：
   * **工作区**：对所有工作区成员可见
   * **未列出**：仅对自己可见

<Note>
  默认的 "All [Object Name]" 视图的可见性无法更改。
</Note>
