使用看板视图将您的项目（或任何具有阶段的对象）可视化为在列间移动的卡片。

<VimeoEmbed videoId="927888627" title="视频演示" />

## 先决条件

您的对象需要一个用作列的 **选择字段**（例如：状态、阶段、环节）。

如果您还没有：

1. 进入 **设置 → 数据模型**
2. 选择你的对象
3. 添加一个包含阶段选项的选择字段

## 步骤

1. 导航到您的对象（例如：项目、任务）
2. 单击视图下拉菜单 → **+ 添加视图**
3. 为视图命名（例如：“项目看板”）
4. 单击 **创建**
5. 单击 **选项**，并选择 **看板** 作为布局
6. 该视图会自动使用您的选择字段作为列
7. 单击 **保存**

## 配置您的看板

### 在卡片上显示关键字段

1. 点击 **选项 → 字段**
2. 在 "隐藏字段" 部分查找字段
3. 单击 **眼睛图标** 以在卡片上显示它们
4. 拖动以重新排序

<img src="/images/user-guide/kanban-views/filter.png" style={{width:'100%'}} />

### 启用紧凑视图

用于高层概览：

1. 单击 **选项**
2. 开启 **紧凑视图**

卡片仅显示记录名称。

<img src="/images/user-guide/kanban-views/compact-view.png" style={{width:'100%'}} />

### 添加聚合

在每列顶部显示计数或总计：

1. 单击列名旁边的数字
2. 选择一种聚合（计数、求和等）
3. 如有需要，选择一个字段

## 移动卡片

在列之间拖放卡片以更新其状态。

<VimeoEmbed videoId="927888627" title="视频演示" />

## 示例：任务看板

| 列（状态）   | 卡片     |
| ------- | ------ |
| **待办**  | 新任务    |
| **进行中** | 进行中的工作 |
| **审核**  | 等待批准   |
| **已完成** | 已完成    |
