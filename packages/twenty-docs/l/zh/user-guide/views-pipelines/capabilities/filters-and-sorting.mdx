## 过滤数据

过滤器通过仅显示符合您条件的记录，帮助您专注于特定记录。

### 添加过滤器

1. 点击工具栏中的 **过滤** 按钮
2. 选择用于过滤的字段
3. 选择运算符（等于、包含等）
4. 输入过滤条件的值
5. 点击 **应用**

### 过滤运算符

| 字段类型 | 可用的运算符             |
| ---- | ------------------ |
| 文本   | 等于、包含、以…开头、以…结尾、为空 |
| 数字   | 等于、大于、小于、介于、为空     |
| 日期   | 等于、早于、晚于、介于、为空     |
| 选择   | 等于、属于以下任一、为空       |
| 复选框  | 为真、为假              |
| 关系   | 等于、为空              |

### 多个过滤器

组合多个过滤器以缩小结果范围：

* 所有过滤器均按 AND 逻辑应用
* 每个额外的过滤器都会进一步限制结果

### 移除过滤器

* 点击单个过滤条件标签上的 **X**
* 点击 **全部清除** 以移除所有过滤器

## 排序数据

排序决定记录的显示顺序。

### 添加排序

1. 点击工具栏中的 **排序** 按钮
2. 选择用于排序的字段
3. 选择升序（A-Z、0-9）或降序（Z-A、9-0）
4. 点击 **应用**

### 多级排序

添加多个排序级别：

* 第一个排序为主排序
* 后续排序在相同值的分组内应用

### 快速列排序

单击任意列标题进行排序：

* 第一次点击：升序
* 第二次点击：降序
* 第三次点击：移除排序

## 保存过滤与排序设置

过滤与排序会随视图保存：

1. 配置您的过滤与排序
2. 点击 **保存** 以更新当前视图
3. 或点击 **另存为新视图** 以创建一个变体
