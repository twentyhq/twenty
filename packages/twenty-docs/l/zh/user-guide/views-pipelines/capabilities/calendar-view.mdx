## 关于日历视图

日历视图会根据某个日期字段在日历上显示您的记录。 每条记录会在相应日期显示为一个事件。

<img src="/images/user-guide/views/calendar-view-zoom.png" style={{width:'100%'}} />

## 创建日历视图

1. 导航到包含日期字段的对象
2. 单击视图下拉菜单 → **+ 添加视图**
3. 为视图命名并单击  **创建**
4. 打开右侧的 **选项**
5. 选择 **日历** 作为布局
6. 选择用于定位记录的 **日期字段**
7. 单击 **更新视图**

## 配置日历

### 选择日期字段

在 **选项** 中，选择决定记录在日历上显示位置的日期字段。

### 显示字段

配置每个日历事件上显示的字段：

1. 单击 **选项 → 字段**
2. 切换字段开/关
3. 拖动以重新排序

## 使用场景

* **会议和通话**：查看即将到来的预约
* **截止日期**：跟踪到期日期和关闭日期
* **事件**：规划并可视化已安排的活动
* **跟进**：查看任务的到期时间
