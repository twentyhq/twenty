## 定价

<AccordionGroup>
<Accordion title="自托管的情况下，我可以免费使用Twenty吗？">
是的，自托管时您可以免费使用Twenty。 您将获得 Pro（云）计划所包含的全部内容，但不包括我们核心团队的支持。 支持可通过我们的Discord社区获得。

如果您希望自行托管并需要高级功能（SSO、行级权限等），可以选择付费的 Organization（自托管）许可证。 这还包括来自 Twenty 团队的支持，并取消在分发前将自定义代码以开源形式发布的要求。
</Accordion>

<Accordion title="高级功能有哪些？">
高级功能仅适用于 Organization 计划（云端或自托管）：
* **SSO 集成**：与您的身份提供商进行单点登录
* **行级权限**：在记录级别进行细粒度的访问控制
* **AI 使用数据**：在整个工作区跟踪 AI 消耗
* **审计日志**：查看每位工作区成员的操作记录
* **电子邮件营销活动**：规划需要发送的电子邮件（即将可用）
* **无限工作区**：创建任意数量的工作区（仅限自托管）
* **私有源代码**：无需将更改发布到公开可用的代码库（仅限自托管）
</Accordion>

<Accordion title="云端的 Organization 计划与自托管的 Organization 计划是否相同？">
它们提供相同的高级功能。 但是，云端订阅不能用于自托管部署。 自托管需要 Enterprise 密钥。
</Accordion>

<Accordion title="你们是否为只读用户提供免费席位？">
我们不提供免费席位。 定价是按用户计费的，每位用户需要许可证才能访问Twenty。
</Accordion>

<Accordion title="我在哪里可以将我的订阅切换到组织计划？">
您可以在 `Settings → Billing → Plans` 下进行操作。 然后点击`切换到组织计划`。
</Accordion>

<Accordion title="我在哪里可以将我的订阅切换到Pro计划？">
您可以在 `Settings → Billing → Plans` 下进行操作。 然后点击 `Switch to Pro`。
</Accordion>

<Accordion title="我在哪里可以将我的订阅切换到包年？">
您可以在`设置 → 账单`下进行操作。 然后点击 `Upgrade to yearly`。
</Accordion>

<Accordion title="我在哪里可以将我的订阅切换到包月？">
您可以在`设置 → 账单`下进行操作。 然后点击 `Downgrade to monthly`。
</Accordion>

<Accordion title="我在哪里可以查看我的信用使用情况？">
您将在`设置 → 账单`下找到此内容。
</Accordion>

<Accordion title="我们有多少工作流积分？">
积分以您的计费周期为准，而非您的计划：
* **按月订阅**：每月5积分
* **按年订阅**：每年50积分
</Accordion>

<Accordion title="工作流积分的消耗是如何运作的？">
每个工作流程操作根据其复杂性消耗积分：
* **基本内部操作**（例如搜索、更新、创建记录）消耗的积分很少
* **更复杂的操作**如代码节点和对外部服务的请求消耗更多积分
* **AI 提示**也会根据使用情况消耗更多积分
* **应用**也会像 Code 节点一样消耗更多积分

当工作流或应用执行时，积分会实时扣除。 您可以在**设置 → 账单**中监控您的使用情况以跟踪消耗和剩余积分。
</Accordion>

<Accordion title="我可以购买更多的工作流积分吗？">
您可以在`设置 → 账单`下购买额外的积分。
</Accordion>

<Accordion title="我可以只为 1 个月购买更多的工作流积分吗？">
可以，前往 `Settings → Billing`，点击 `Increase` 按钮，选择您想要的积分数量并确认，它们将应用于当前计费周期。 然后再次点击 `Increase` 按钮，将数值改回原始值并确认，降级将从下一个计费周期开始生效。
</Accordion>
</AccordionGroup>

## 账单

<AccordionGroup>
<Accordion title="我在哪里可以管理账单细节？">
您可以在`设置 → 账单`下进行操作。
</Accordion>

<Accordion title="我在哪里可以更新我的支付方式？">
您可以在`设置 → 账单`下进行操作。 然后点击`查看账单详情`。 在那里您可以添加新的支付方式。
</Accordion>

<Accordion title="我在哪里可以更新我的账单信息？">
您可以在`设置 → 账单`下进行操作。 然后点击`查看账单详情`。 在那里您可以编辑账单信息。
</Accordion>

<Accordion title="我在哪里可以访问我的发票？">
您可以在`设置 → 账单`下进行操作。 然后点击`查看账单详情`。 您将看到屏幕底部的所有发票。
</Accordion>

<Accordion title="为什么我的席位数与工作区中的用户数不同？">
工作区席位数量每 6 小时与支付系统同步一次；发票会根据差额进行调整。
</Accordion>

<Accordion title="为什么我收到了关于新工作区成员的发票？">
当有新工作区成员被添加到工作区时，会发送一张发票以弥补差额。 当移除某个工作区成员时，更改会在下一张标准的月度/年度发票中生效。
</Accordion>
</AccordionGroup>
