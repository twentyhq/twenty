## 什么是积分？

积分是一种统一的余额，用于支付我们代表你运行且实际产生成本的所有内容：**工作流自动化**、**AI 代理** 和 **AI 聊天**。 你无需再同时处理“自动化运行次数”和“AI 令牌”的单独配额，而只需关注一个简单的余额。

**1 积分 = $1 的用量。** 一个积分就相当于一美元的用量，因此你的余额很好理解——不需要折算任何抽象单位。

## 一个积分能用多少？

成本会随着工作量而变化。 日常自动化和快速 AI 消息只会消耗极小部分积分；大型的多步骤 AI 代理任务则会消耗更多积分。 大致情况如下：

| 你要做什么                                               | 大致花费                | 含义                       |
| --------------------------------------------------- | ------------------- | ------------------------ |
| 运行一个标准工作流步骤（搜索、创建、更新记录）                             | ~$0.0001            | 大约**每个积分可运行 10,000 个步骤** |
| 调用一个外部 API 或运行一个代码节点                                | 一个积分的一小部分           | 每个积分可运行数千次               |
| 发送一条快速 AI 聊天消息或执行一个简单的 AI 步骤                        | 几分之一美分              | 每个积分可运行数百次               |
| 运行一个大型 AI 代理任务（例如 "configure 10 workspace objects"） | 成本可能高达 **约 $1 或更多** | 每个积分可运行几次                |
| 运行一次 AI 网络搜索                                        | 按每次搜索收取固定费用         | 每个积分可进行数百次搜索             |
| 触发应用的逻辑函数                                           | 一个积分的一小部分           | 每个积分可进行数千次               |

<Note>
对大多数团队来说，标准自动化几乎等同于免费——只有在大量使用 AI 功能或运行高负载代码节点时，你的积分余额才会出现明显消耗。 这是有意为之：包含的积分旨在让你在构建时无需时刻盯着计量表。
</Note>

## AI 使用如何计量

当某个操作调用 AI 模型时，我们会精确测量所使用的 token 数量，并按**模型提供商公布的费率**计价，然后将该美元金额直接折算为积分——因此你消耗的积分对应的是实际工作成本，而不是经过加价的内部费率。 一条简短的聊天消息只会花费几分之一美分。 一个大型代理任务——例如配置十个工作区对象——会在不断扩大的上下文中进行多次模型调用，因此总成本可能会累积到一美元或更多。 你的计费完全基于实际使用量，精确到单个 token。

成本会因模型而异：轻量模型每条消息的价格远低于顶级推理模型。 你可以在 **设置 → 账单** 中查看每个操作的详细拆分——包括使用了哪种模型。

## 包含内容

每个订阅都包含积分，数量取决于你的**计费周期**，而不是订阅套餐：

| 计费周期 | 包含的积分 | 等效使用量        |
| ---- | ----- | ------------ |
| 每月   | 5/月   | 每月约 $5 的使用量  |
| 每年   | 50/年  | 每年约 $50 的使用量 |

这些积分旨在从容覆盖日常自动化。 5 个积分可以覆盖数万次标准自动化步骤或数百条快速 AI 消息——不过少量大型、多步骤代理任务也可能更快用完这些积分。 你可以随时充值（见下文）。

## 积分结转

未使用的积分会自动结转到下一个计费周期。

* **上限**：结转额度以一个计费周期的完整配额为上限，因此你结转的积分不会超过你的套餐每个周期提供的积分数量。
* **可见性**：当结转积分可用时，它们会在 **设置 → 账单** 中显示为单独的 **结转积分** 一行，并与 **可用总额** 余额并列。

## 监控使用情况

随时掌握你的积分余额及其去向：

1. 前往**设置 → 结算**
2. 查看你的**剩余积分**和**当前周期的使用量**
3. 深入查看明细，了解哪些工作流和 AI 操作在消耗积分，以及这些 AI 操作使用了哪些模型

定期查看这些信息，可以让你及早发现高负载的工作流，并决定是对其进行优化还是充值。

## 需要更多积分吗？

如果你的积分用完了——或者在进行大规模自动化之前想要多一些余量——可以随时购买额外积分包：

1. 前往**设置 → 结算**
2. 单击 **增加** 按钮
3. 选择你需要的额度并接受更改

购买的积分会立即计入你的可用余额。
