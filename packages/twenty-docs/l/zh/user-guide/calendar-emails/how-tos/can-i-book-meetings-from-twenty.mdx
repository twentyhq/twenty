## 当前状态

Twenty 的日历集成旨在**同步并显示**你现有的日历事件。

## 通过工作流安排会议

虽然无法手动安排会议，但你可以在 Twenty 中使用工作流来安排会议。

1. 创建新的工作流
2. 根据你的需求选择任意触发器
3. 添加 **创建日历事件** 节点
4. 配置所有必要的数据
5. 激活工作流

## 你可以做什么

* **在“人员”、“公司”和“机会”记录中查看会议历史**
* **在你的 CRM 中查看与联系人的即将举行的会议**
* **跟踪会议上下文**，与电子邮件沟通一并查看
* **从会议参与者中自动创建联系人**
