## 当前状态

Twenty 的电子邮件集成旨在**同步并显示**你的电子邮件历史记录，并用于撰写和发送电子邮件。

当你在记录页面查看电子邮件会话并点击**回复**时，你可以直接撰写并发送电子邮件。

## 如何从 Twenty 发送消息

前提条件：已连接可发送电子邮件的邮件账户

1. 使用 **Ctrl+K** 或 **CMD+K** 快捷键打开 Command 菜单
2. 点击 "Compose email" 选项
3. 选择收件人，并为邮件添加主题、正文和附件
4. 点击 "Send" 按钮发送电子邮件

## 你现在可以做什么

* **在“人员”、“公司”和“机会”记录中查看电子邮件历史**
* **在你的 CRM 中阅读与联系人的完整电子邮件会话**
* **与日历事件一并跟踪沟通上下文**
* **从电子邮件互动中自动创建联系人**
* **通过重定向进行回复** — 点击“回复”跳转到你的邮箱

## 通过工作流发送电子邮件

你**可以使用工作流自动发送电子邮件**。 这在以下场景中很有用：

* 自动跟进
* 向联系人发送通知
* 基于记录更改触发的沟通

通过工作流发送的电子邮件将通过你已连接的邮箱账户发送。

→ 了解[“发送电子邮件”操作](/l/zh/user-guide/workflows/capabilities/workflow-actions#send-email)

## 电子邮件序列和新闻通讯

对于电子邮件序列和新闻通讯，我们建议使用工作流将 Twenty 连接到专用的电子邮件营销工具。

<Warning>
为保护你的域名信誉，不应直接从你的邮箱发送群发邮件。 请使用专用工具进行批量沟通。
</Warning>

→ 请参阅[如何通过工作流发送电子邮件](/l/zh/user-guide/workflows/capabilities/send-emails-from-workflows)以获取设置说明
