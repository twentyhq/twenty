## 即將到來的功能

### 自定義報告

創建量身定制的報告以跟踪對您的業務最重要的指標。

### 銷售儀表板

使用互動式儀表板可視化您的銷售渠道績效。

### 性能分析

監控團隊表現並識別您的客戶數據中的趨勢。

## 保持更新

關注我們的 [GitHub 存儲庫](https://github.com/twentyhq/twenty) 或查看您的工作區中的 **設定 → 版本** 區域，以便在報告功能可用時獲得通知。
