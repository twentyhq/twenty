## Крок 1: Реєстрація

1. Перейдіть на [Twenty Sign Up](https://app.twenty.com).
2. Виберіть бажаний спосіб реєстрації:
   - **Продовжити з Google** для реєстрації з обліковим записом Google.
   - **Продовжити з Microsoft** для реєстрації з обліковим записом Microsoft.
   - Або, **Продовжити з електронною поштою** для реєстрації по електронній пошті.

## Step 2: Choosing a Trial Period

Обирайте між двома пробними періодами:

### 30 днів

З кредитною карткою

### 7 днів

Без кредитної картки

Обидва пробні періоди включають:

- Повний доступ
- Необмежена кількість контактів
- Інтеграція електронної пошти
- Користувацькі об'єкти
- API та Webhooks

Ви можете натиснути "Змінити план", щоб вибрати інший план або інтервал оплати.

## Крок 3: Підтвердження платежу та налаштування облікового запису

Після схвалення платежу через Stripe, ви будете перенаправлені для створення свого робочого простору та профілю користувача. Пам'ятайте, що ви можете скасувати підписку в будь-який час.
