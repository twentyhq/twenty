## Поточні варіанти інтеграцій

### Нативні інтеграції

Twenty наразі пропонує нативну інтеграцію з:

- **Email & Calendar**: Підключіть Gmail, Outlook або постачальників SMTP/CalDAV
- **API Доступ**: Використовуйте наші REST та GraphQL API для створення власних інтеграцій

Для налаштування пошти та календаря відвідайте [Налаштування пошти та календаря](/l/uk/user-guide/settings/email-calendar-setup).

### Робочі процеси (рекомендовано)

Основний спосіб підключення Twenty до інших інструментів - через наш внутрішньоапове **Робочі процеси**:

- **HTTP Вузли**: Робіть виклики API для зовнішніх сервісів
- **Код Вузли**: Пишіть власну логіку для складних інтеграцій
- **Тригери Webhook**: Отримуйте дані від зовнішніх систем
- **Тригери Оновлення Поля**: Автоматизуйте дії, засновані на змінах CRM

Дізнайтеся більше в нашому розділі [Робочі процеси](/l/uk/user-guide/workflows/getting-started-workflows).

### Інтеграція з Zapier (Застарілий)

Ми підтримуємо інтеграцію з Zapier для користувачів, які віддають перевагу безкодовій автоматизації:

1. Відвідайте [Twenty на Zapier](https://zapier.com/apps/twenty/integrations)
2. Створіть новий Zap з Twenty як тригер або дії
3. Згенеруйте ключ API в Налаштуваннях → API та Webhooks
4. Підключіть робочий простір Twenty до Zapier

## Майбутнє бачення: Конектори, створені спільнотою (2026)

### Платформа розширюваності

Ми створюємо платформу розширюваності, яка дозволить розробникам створювати додатки у вигляді коду. Це дасть змогу:

- **Власні Конектори**: Створюйте інтеграції з улюбленим програмним забезпеченням
- **Внески Спільноти**: Діліться та знаходьте конектори, створені іншими
- **Гнучка Архітектура**: Розширюйте можливості Twenty за межі основних функцій

### Будування робочих процесів з AI-допомогою

Coming in 2026, AI assistance will help users:

- **Авто-генерація Робочих процесів**: Описуйте потреби інтеграції звичайною мовою
- **Розумні Рекомендації**: Отримуйте рекомендації щодо підключення конкретних інструментів
- **Бібліотека Шаблонів**: Доступ до попередньо створених робочих процесів для загальних випадків використання

The future of Twenty integrations is community-driven, AI-assisted, and infinitely extensible.
