## Що очікується

### Користувацькі звіти

Створюйте індивідуальні звіти, щоб відстежувати найважливіші для вашого бізнесу показники.

### Інформаційні панелі продажів

Візуалізуйте ефективність вашої торгової воронки за допомогою інтерактивних інформаційних панелей.

### Аналітика ефективності

Моніторинг ефективності команди та визначення тенденцій у даних ваших клієнтів.

## Будьте в курсі

Стежте за нашим [репозиторієм на GitHub](https://github.com/twentyhq/twenty) або перевіряйте розділ **Налаштування → Випуски** у вашому робочому просторі, щоб отримати сповіщення, коли функції звітності стануть доступними.
