## Custom Workspace Domain

Встановіть персоналізовану веб-адресу для вашого робочого простору:

- Виберіть свій бажаний піддомен (наприклад, yourcompany.twenty.com)
- Налаштуйте персоналізовані домени для професійного брендування

## Затверджені домени доступу

Дозвольте автоматичний доступ до робочого простору для певних доменів електронної пошти.

### Як це працює

- Додайте домени електронної пошти вашої компанії (наприклад, @yourcompany.com)
- Будь-хто, у кого є електронна адреса з цих доменів, може автоматично долучатись до вашого робочого простору
- Немає потреби в ручних запрошеннях для членів команди

### Переваги

- **Швидше впровадження**: Нові члени команди долучаються автоматично
- **Поліпшена безпека**: Лише підтверджені електронні адреси компанії можуть отримати доступ
- **Менше адміністративної роботи**: Не потрібно вручну запрошувати кожного члена команди

## Інструкції з налаштування

1. Перейдіть до **Налаштування → Доменів**
2. Додайте свій персоналізований домен або затверджені домени електронної пошти
3. Підтвердьте право власності на домен, якщо це потрібно

**Примітка**: Зміни домену можуть потребувати часу, щоб набрати чинності, і можуть вимагати підтвердження.
