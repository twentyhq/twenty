## Опції підключення

### Обліковий запис Google (Gmail та Google Calendar)

1. Перейдіть до **Налаштування → Облікові записи**
2. Натисніть **Додати обліковий запис**
3. Виберіть **Продовжити з Google**
4. Авторизуйте Twenty для доступу до Gmail та Google Calendar
5. Ваша електронна пошта та події календаря почнуть синхронізуватися автоматично

### Обліковий запис Microsoft (Outlook та Microsoft Calendar)

1. Перейдіть до **Налаштування → Облікові записи**
2. Натисніть **Додати обліковий запис**
3. Виберіть **Продовжити з Microsoft**
4. Авторизуйте Twenty для доступу до Outlook та Microsoft Calendar
5. Ваша електронна пошта та події календаря почнуть синхронізуватися автоматично

### Налаштування SMTP/CalDAV (Інші провайдери)

Для інших постачальників електронної пошти та календаря:

1. Перейдіть до **Налаштування → Випуски → Лабораторія** для включення можливості
2. Поверніться до **Налаштування → Облікові записи**
3. Налаштуйте параметри SMTP для електронної пошти
4. Налаштуйте параметри CalDAV для календаря
5. Перевірте з'єднання

### Кілька Пошт

- **Необмежені Аккаунти**: Підключення декількох облікових записів електронної пошти на користувача
- **Управління Обліковими записами**: Перемикання між різними поштовими скриньками
- **Параметри Синхронізації**: Налаштування різних параметрів для кожної скриньки

<Warning>
Можна підключити лише справжні поштові скриньки (наприклад, support@domain.com зі своїм власним інбоксом). Альтернативи електронної пошти, що перенаправляють на іншу скриньку, не можуть бути підключені до Twenty.
</Warning>

## Налаштування Електронної Пошти

### Видимість Повідомлень

Виберіть різні рівні видимості для вашої електронної пошти:

- **Тільки Метадані**: Ділитися тільки основною інформацією (відправник, одержувач, дата, час)
- **Тема та Метадані**: Ділитися заголовком разом з метаданими
- **Весь Вміст Листа**: Ділитися всім вмістом листа включно з вкладеннями

### Автостворення Контактів

- **Деактивовано**: Автоматичного створення контактів немає
- **Для відправлених та отриманих повідомлень**: Створити контакти для всіх зовнішніх електронних взаємодій
- **Тільки для відправлених повідомлень**: Створити контакти тільки для відправлених вами листів
- **Примітка**: Внутрішні листи (однакова доменна зона) ніколи не синхронізуються для забезпечення конфіденційності

### Керуйте тим, які листи синхронізуються через Вибір Папки Повідомлень (Лабораторна функція)

Керувати якими папками електронної пошти синхронізуються з Twenty:

1. Перейдіть до **Налаштування → Випуски → Лабораторія** та увімкніть **Папку Повідомлень**
2. Поверніться до **Налаштування → Облікові записи** та виберіть ваш підключений обліковий запис електронної пошти
3. Виберіть, які папки синхронізувати:
   - **Вхідні**: Основні що надходять листи
   - **Відправлені**: Вихідні листи, які ви відправили
   - **Спеціальні Папки**: Будь-які конкретні папки, які ви хочете включити
   - **Виключити Папки**: Пропустити такі папки, як Спам, Кошик або особисті папки

Це надає вам точний контроль над тим, які листи з'являються у вашій CRM без синхронізації всього.

**Що Синхронізується:**

- **Зовнішні Листи**: Всі листи з зовнішніми контактами з обраних папок
- **Внутрішні Листи**: Не синхронізуються (листи в рамках одного домену залишаються приватними)
- **Вкладення**: Надходять у першому кварталі 2026 року

**Примітка**: Ми не надаємо адресу ел. пошти CC для вибіркового синхронізування. Замість цього використовуйте функцію Папки Повідомлень вище для досягнення такого ж рівня контролю над тим, які листи синхронізуються з Twenty.

## Налаштування Календаря

### Видимість Подій

Виберіть, що буде видно іншим користувачам у вашому робочому просторі:

- **Усе**: Всі деталі події будуть поділені з вашою командою
- **Метадані**: Тільки дата та учасники будуть поділені з вашою командою

### Автостворення Контактів для Зустрічей

- **Так**: Автоматично створити контакти для учасників зустрічі, яких немає у вашій CRM
- **Ні**: Лише зв'язати зустрічі із існуючими контактами

### Керуйте тим, які події синхронізуються

- **Імпорт Зустрічей**: Автоматично імпортувати події календаря
- **Зв'язування Контактів**: Зв'язуйте зустрічі з записами Людей та Компаній

**Що Синхронізується:**

- **Зустрічі**: Календарні події із зовнішніми учасниками
- **Зв'язування Контактів**: Події, автоматично зв'язані з CRM записами
- **Командні Події**: Видимість спільного календаря

## Частота Синхронізації

**Оновлення кожні 5 хвилин**: Дані ел. пошти та календаря синхронізуються автоматично кожні 5 хвилин після первинного імпорту.
