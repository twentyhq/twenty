## Зовнішній вигляд

### Вибір теми

Виберіть між світлим та темним режимами:

- **Світлий режим**: Чистий, яскравий інтерфейс, ідеальний для добре освітлених середовищ
- **Темний режим**: Легше для очей в умовах слабкого освітлення
- **Системний**: Автоматично збігається з налаштуванням теми вашого пристрою

## Регіональні налаштування

### Мова

Виберіть вашу улюблену мову для інтерфейсу Twenty:

- Англійська (за замовчуванням)
- Додаткові мови доступні на основі перекладів спільноти

### Часовий пояс

Встановіть місцевий часовий пояс для точного планування та часових міток:

- Впливає на часи зустрічей, терміни завдань та журнали активності
- Автоматично налаштовується на літній час

### Формат дати

Виберіть, як дати будуть відображатися у Twenty:

- **ММ/ДД/РРРР** (формат США)
- **ДД/ММ/РРРР** (європейський формат)
- **РРРР-ММ-ДД** (формат ISO)

### Формат числа

Налаштуйте, як показуються числа та валюти:

- **Десятковий роздільник**: Кома (,) або крапка (.)
- **Тисячний роздільник**: Пробіл, кома або крапка
- **Символ валюти**: Залежить від вашого регіону або настроюваний

### Формат календаря

Встановіть бажане налаштування календаря:

- **Перший день тижня**: Неділя або понеділок
- **Номери тижнів**: Показувати або приховувати номери тижнів ISO
- **Формат часу**: 12-годинний (АМ/ПМ) або 24-годинний формат

## Як оновити налаштування

1. Перейдіть до **Налаштування → Досвід** з бічної панелі
2. Налаштуйте ваші вподобання в кожному розділі
3. Зміни зберігаються автоматично
4. Оновіть ваш браузер, щоб побачити всі зміни в дії
