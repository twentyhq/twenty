## Запросити нових учасників

### Використання посилання для запрошення

1. Перейдіть до **Налаштування → Учасники**
2. Скопіюйте посилання для запрошення до робочого простору
3. Поділіться посиланням з новими учасниками команди
4. Вони отримають доступ після реєстрації

### Запрошення електронною поштою

1. Перейдіть до **Налаштування → Учасники**
2. Введіть адресу електронної пошти особи
3. Натисніть **Запросити**
4. Вони отримають запрошення на електронну пошту

## Видалити учасників

### Видалити учасника

1. Перейдіть до **Налаштування → Учасники**
2. Знайдіть учасника, якого потрібно видалити
3. Натисніть кнопку видалити/усунути біля їхнього імені
4. Підтвердьте видалення

**Примітка**: Вилучені учасники втрачають доступ негайно, але можуть бути знову запрошені пізніше.
