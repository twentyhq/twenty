## Останні Випуски

Відстежуйте хід розробки Twenty:

- **Примітки до Випуску**: Дивіться, що нового в кожній версії
- **Feature Updates**: Discover new capabilities and improvements

## Особливості Лабораторії

Access experimental features before they're officially released:

- **Beta Testing**: Try new features while they're in development
- **Флаги Функцій**: Увімкніть або вимкніть окремі експериментальні функції

## How to Access

1. Перейдіть до **Налаштування → Випуски**
2. View release notes and access the **Lab** tab for beta features

Для додаткових ресурсів громади зверніться до нашого [Розділ ресурсів](/l/uk/user-guide/resources/glossary).
