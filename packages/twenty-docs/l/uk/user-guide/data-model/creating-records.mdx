## About Table Views

Table views are visual representations of data structured in rows and columns.

## Створити запис

Додавайте записи за потреби, без обмежень. Щоб додати запис, ви можете натиснути кнопку **+** у верхньому правому куті екрана або у верхній частині стовпця **Назва** запису.

Введіть назву запису, після чого натисніть `Enter`, щоб зберегти. Щоб редагувати назву запису, натисніть на його назву на сторінці деталей.

<VimeoEmbed videoId="927071691" title="Video demonstration" />

## Видалити запис

**Індекс перегляду:** Щоб видалити запис, виберіть прапорець поруч із записом і натисніть кнопку видалення у верхньому правому куті.

**Сторінка запису:** Натисніть на іконку `⋮` у верхньому правому куті, потім виберіть видалити.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## Add a Custom Field

To create a custom field, click the **+** button at the right end of the table columns and select **Customize fields**.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

Також ви можете зробити це, перейшовши до **Налаштування** > **Модель даних** > **Люди**. Натисніть на **Додати поле**. Оберіть назву та тип поля. Нове поле буде доступним у застосунку.
