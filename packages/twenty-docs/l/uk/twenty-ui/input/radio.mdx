Використовується, коли користувачі можуть вибрати лише один варіант з кількох можливих.

<Tabs>
<Tab title="Usage">

```jsx
import { Radio } from "twenty-ui/display";

export const MyComponent = () => {

  const handleRadioChange = (event) => {
    console.log("Radio button changed:", event.target.checked);
  };

  const handleCheckedChange = (checked) => {
    console.log("Checked state changed:", checked);
  };


  return (
    <Radio
      checked={true}
      value="Option 1"
      onChange={handleRadioChange}
      onCheckedChange={handleCheckedChange}
      size="large"
      disabled={false}
      labelPosition="right"
    />
  );
};

```

</Tab>

| Властивості     | Тип                     | Опис                                                                                                                |
| --------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| стиль           | властивості `React.CSS` | Додаткові інлайн-стилі для компонента                                                                               |
| className       | рядок                   | Опційний клас CSS для додаткового стилювання                                                                        |
| позначено       | булевий тип             | Вказує, чи є радіокнопка позначеною                                                                                 |
| значення        | рядок                   | The label or text associated with the radio button                                                                  |
| onChange        | функція                 | Функція, що викликається, коли змінюється вибрана радіокнопка                                                       |
| onCheckedChange | функція                 | Функція, що викликається, коли змінюється стан `позначено` радіокнопки                                              |

| позиція ярлика  | рядок                   | Положення тексту ярлика відносно радіокнопки. Має два варіанти: `зліва` та `справа` |

## Група радіо

Groups together related radio buttons.

<Tabs>
<Tab title="Usage">

```jsx
import React, { useState } from "react";
import { Radio, RadioGroup } from "twenty-ui/display";

export const MyComponent = () => {

  const [selectedValue, setSelectedValue] = useState("Option 1");

  const handleChange = (event) => {
    setSelectedValue(event.target.value);
  };
  
  return (
    <RadioGroup value={selectedValue} onChange={handleChange}>
      <Radio value="Option 1" />
      <Radio value="Option 2" />
      <Radio value="Option 3" />
    </RadioGroup>
  );
};

```

</Tab>
<Tab title="Props">

| Властивості   | Тип               | Опис                                                                                             |
| ------------- | ----------------- | ------------------------------------------------------------------------------------------------ |
| значення      | рядок             | Значення вибраної в даний момент радіокнопки                                                     |
| onChange      | функція           | Функція зворотного виклику, що спрацьовує при зміні радіокнопки                                  |
| onValueChange | функція           | Функція зворотного виклику, що спрацьовує при зміні вибраного значення в групі.  |
| children      | `React.ReactNode` | Дозволяє передавати React-компоненти (такі як Радіо) як дочірні в Групу Радіо |
