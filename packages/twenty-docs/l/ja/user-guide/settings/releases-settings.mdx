## 最新のリリース

Twenty の開発進捗を追跡する:

- **リリースノート**: 各バージョンの新機能を確認する
- **機能アップデート**: 新機能や改善を見つける

## ラボ機能

公式リリース前に実験的な機能にアクセスする:

- **ベータテスト**: 開発中の新機能を試す
- **機能フラグ**: 具体的な実験機能を有効または無効にする

## アクセス方法

1. Go to **Settings → Releases**
2. リリースノートを表示し、ベータ機能用の **ラボ** タブにアクセスする

さらなるコミュニティリソースについては、[リソースセクション](/l/ja/user-guide/resources/glossary)をご確認ください。
