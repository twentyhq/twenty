## ワークスペース設定

<AccordionGroup>
  <Accordion title="ワークスペース名を変更するにはどうすればよいですか？">
    1. **設定 → 一般** に移動
    2. ワークスペース名の項目を見つけます
    3. 新しい名前を入力します
    4. 変更は自動的に保存されます
  </Accordion>

  <Accordion title="ワークスペースのロゴを更新するにはどうすればよいですか？">
    1. **設定 → 一般** に移動
    2. 現在のロゴまたはアップロードエリアをクリックします
    3. 画像ファイル（PNG、JPEG、または 10MB 未満の GIF）を選択します
    4. ロゴはすぐに更新されます
  </Accordion>

  <Accordion title="複数のワークスペースを持てますか？">
    はい、複数のワークスペースを作成し、メンバーになることができます。 各ワークスペースは独自のデータ、設定、サブスクリプションを持ちます。
  </Accordion>

  <Accordion title="ワークスペースを削除するにはどうすればよいですか？">
    1. **設定 → 一般** に移動
    2. 危険地帯までスクロールします
    3. **ワークスペースを削除**をクリックします
    4. 削除を確認します

    注意：これによりすべてのデータが永久に削除され、元に戻せません。
  </Accordion>

  <Accordion title="誤って複数のワークスペースを作成してしまいました。 どうすればよいですか？">
    不要なワークスペースは **設定 → 一般 → ワークスペースを削除**から削除してください。

    <Warning>
      **アカウント**を削除しないでください（設定 → プロフィールからアクセス可能）：アカウントはすべてのワークスペースで共有されています。 アカウントを削除すると、すべてのワークスペースへのアクセスを失います。
    </Warning>
  </Accordion>

  <Accordion title="削除せずにワークスペースを無効化するにはどうすればよいですか？">
    ワークスペースを一時的に無効にしたい場合（完全に削除するのではなく）、**設定 → 請求** に移動して**プランをキャンセル**をクリックします。 データは猶予期間中保持されます。
  </Accordion>
</AccordionGroup>

## プロフィール設定

<AccordionGroup>
  <Accordion title="パスワードを変更するにはどうすればよいですか？">
    1. **設定 → プロフィール** に移動
    2. **パスワード** セクションを見つけます
    3. 現在のパスワードを入力します
    4. 新しいパスワードを入力します
    5. 変更を保存します
  </Accordion>

  <Accordion title="二要素認証を有効にするにはどうすればよいですか？">
    1. **設定 → プロフィール** に移動
    2. **2FA** セクションを見つけます
    3. **2FA有効化**をクリック
    4. 認証アプリでQRコードをスキャンします
    5. 認証コードを入力します
  </Accordion>

  <Accordion title="メールアドレスを変更できますか？">
    メールアドレスの変更をご希望の場合は、[contact@twenty.com](mailto:contact@twenty.com) までご連絡ください。
  </Accordion>

  <Accordion title="アカウントを削除するにはどうすればよいですか？">
    1. **設定 → プロフィール** に移動
    2. 危険地帯までスクロールします
    3. **アカウント削除**をクリックします
    4. メールアドレスを入力して確認します

    注意：これにより、すべてのワークスペースへのアクセスが削除され、接続済みアカウントから同期されたすべてのメールが削除されます。
  </Accordion>
</AccordionGroup>

## エクスペリエンス設定

<AccordionGroup>
  <Accordion title="ライトモードとダークモードを切り替えるにはどうすればよいですか？">
    1. **設定 → エクスペリエンス** に移動
    2. **テーマ** セクションを見つけます
    3. ライト、ダーク、またはシステムを選択します
  </Accordion>

  <Accordion title="日付形式を変更するにはどうすればよいですか？">
    1. **設定 → エクスペリエンス** に移動
    2. 日付形式を見つけます
    3. 希望の形式を選択します
    4. 変更はすぐに反映されます
  </Accordion>

  <Accordion title="タイムゾーンを変更するにはどうすればよいですか？">
    1. **設定 → エクスペリエンス** に移動
    2. タイムゾーンを見つけます
    3. 自分の地域のタイムゾーンを選択します
    4. すべてのタイムスタンプが調整されます
  </Accordion>

  <Accordion title="言語を変更できますか？">
    1. **設定 → エクスペリエンス** に移動
    2. 言語を見つけます
    3. 利用可能な言語から選択します
    4. インターフェースは選択に合わせて更新されます
  </Accordion>
</AccordionGroup>

## アカウント設定

<AccordionGroup>
  <Accordion title="メールアカウントを接続するにはどうすればよいですか？">
    1. **設定 → アカウント** に移動します
    2. **アカウントを追加**をクリックします
    3. Google または Microsoft を選択します
    4. アクセスを承認します
    5. 同期設定を構成します
  </Accordion>

  <Accordion title="複数のメールアカウントを接続できますか？">
    はい、複数のメールアカウントを接続できます。 **設定 → アカウント** に移動し、必要に応じてアカウントを追加してください。
  </Accordion>

  <Accordion title="メールアカウントの接続を解除するにはどうすればよいですか？">
    1. **設定 → アカウント** に移動します
    2. 接続を解除するアカウントを見つけます
    3. **接続を解除**をクリックします
    4. 操作を確認します
  </Accordion>
</AccordionGroup>

## ドメイン

<AccordionGroup>
  <Accordion title="ワークスペースドメインをカスタマイズできますか？">
    はい! **設定 → ドメイン** に移動し、**ドメインをカスタマイズ**をクリックします。 次の 2 つの選択肢があります：

    * **サブドメイン**：`yourcompany.twenty.com` のような Twenty のサブドメインを使用します
    * **カスタムドメイン**：`crm.yourcompany.com` のような独自ドメインを使用します（DNS の設定が必要）

    サブドメインはすぐに設定できますが、カスタムドメインはチーム向けに完全なブランド体験を提供します。
  </Accordion>

  <Accordion title="承認されたアクセスドメインはどのように機能しますか？">
    会社のメールアドレスを持つチームメンバーが自動的にあなたのワークスペースに参加できるよう、承認されたアクセスドメインを構成できます。 **設定 → ドメイン** に移動し、会社のドメイン（例：`yourcompany.com`）を追加します。
  </Accordion>
</AccordionGroup>

## 早期アクセス機能

<AccordionGroup>
  <Accordion title="早期アクセス機能とは何ですか？">
    早期アクセス機能は、一般提供に先立って検証中の機能です。 現在、早期アクセスの対象は **ダッシュボード** のみです。 ユーザーからのフィードバックに基づいて変更される場合があります。
  </Accordion>

  <Accordion title="早期アクセス機能は安全に使用できますか？">
    早期アクセス機能は正常に動作しますが、ユーザーからのフィードバックに基づいて変更される場合があります。 安全に使用できますが、一般提供前に変更される可能性があります。
  </Accordion>

  <Accordion title="早期アクセス機能を有効にするにはどうすればよいですか？">
    1. **設定 → 更新 → 早期アクセス**に移動します
    2. 有効にしたい機能を見つけます
    3. オンに切り替えます
    4. 機能はすぐに利用可能になります
  </Accordion>
</AccordionGroup>
