## 新規メンバーを招待

### 招待リンクを使用する

1. Go to **Settings → Members**
2. ワークスペースの招待リンクをコピー
3. 新しいチームメンバーにリンクを共有
4. サインアップするとアクセスできるようになります

### 直接メールでの招待

1. Go to **Settings → Members**
2. その人のメールアドレスを入力
3. **招待** をクリック
4. メール招待を受け取ります

## メンバー削除

### メンバーを削除

1. Go to **Settings → Members**
2. 削除したいメンバーを見つける
3. 名前の横にある削除ボタンをクリック
4. 削除を確認

**注意**: 削除されたメンバーはすぐにアクセスを失いますが、後で再招待できます。
