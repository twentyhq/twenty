## 外観

### テーマの選択

Choose between light and dark modes:

- **ライトモード**：明るい環境に理想的なクリーンで明るいインターフェース
- **ダークモード**：光が少ない条件で目に優しい
- **システム**：デバイスのテーマ設定に自動的に一致

## 地域設定

### 言語

Twenty インターフェースの言語を選択してください：

- 英語 (デフォルト)
- コミュニティ翻訳に基づいた追加言語が利用可能

### タイムゾーン

スケジュールやタイムスタンプを正確に設定するためのローカルタイムゾーンを設定します：

- 会議の時間、タスクの期限、アクティビティログに影響する
- サマータイムに自動調整

### 日付形式

Twenty で日付が表示される方法を選択します：

- **MM/DD/YYYY** (米国形式)
- **DD/MM/YYYY** (ヨーロッパ形式)
- **YYYY-MM-DD** (ISO 形式)

### Number Format

数字や通貨の表示を構成します：

- **小数点の区切り**：カンマ (,) またはピリオド (.)
- **千の区切り**：スペース、カンマ、またはピリオド
- **通貨シンボル**：地域に基づいてカスタマイズ可能

### カレンダー形式

希望のカレンダーレイアウトを設定します：

- **週の初めの日**：日曜日または月曜日
- **週番号**：ISO 週番号を表示または非表示
- **時刻形式**：12時間 (午前/午後) または 24時間形式

## 設定を更新する方法

1. Go to **Settings → Experience** from the sidebar
2. 各セクションで好みを調整します
3. 変更は自動的に保存されます
4. 変更が有効になるようにブラウザを更新してください
