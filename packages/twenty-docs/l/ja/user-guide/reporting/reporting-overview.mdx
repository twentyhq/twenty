## 今後の展開

### カスタムレポート

ビジネスの重要な指標をトラッキングするためのカスタマイズされたレポートを作成します。

### セールスダッシュボード

インタラクティブなダッシュボードで営業パイプラインのパフォーマンスを視覚化します。

### パフォーマンス分析

チームのパフォーマンスを監視し、顧客データのトレンドを特定します。

## 最新情報を入手

[GitHubリポジトリ](https://github.com/twentyhq/twenty)をフォローするか、ワークスペース内の**設定→リリース**セクションをチェックして、レポート機能が利用可能になったときに通知を受け取る。
