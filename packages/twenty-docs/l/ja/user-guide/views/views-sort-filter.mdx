## ビューの作成、編集、削除

複数のカスタムビューを作成し、チームと共有できます。

### ビューの作成

新しいビューを作成する方法は 2 つあります。 新しいビューを作成する方法は 2 つあります。 「ビュー スイッチャー」から直接、または既存のビューをフィルタリングおよびソートすることにより。

#### ビュー スイッチャーから

1. ビュー スイッチャーを開きます。
2. ビュー スイッチャー メニューの下部にある「ビューを追加」ボタンをクリックします。
3. ビューのアイコンと名前を選択します。
4. 「テーブル」および「カンバン」の間で「ビュータイプ」を選択します。
5. （カンバンビューのみ）カンバンビュー用に、列のヘッダーとして使用する選択フィールドを選択します。 カンバンビューの作成を有効化する前に、「Select」タイプのフィールドがオブジェクトにない場合、設定で作成するように求められます。 カンバンビューの作成を有効化する前に、「Select」タイプのフィールドがオブジェクトにない場合、設定で作成するように求められます。
6. 「作成」をクリックして新しいビューを生成します。

新しく作成されたビューが自動的に開きます。

### ビューの編集および削除

ビューを編集または削除するには:

1. ビュー スイッチャーを開きます。
2. 編集したいビューにカーソルを合わせます。
3. カーソル合わせると表示される点のアイコンをクリックして、ビューを編集および削除します。

編集をクリックすると、ビューのアイコンと名前を変更したり、ビューを完全に削除したりできます。

## ビューのカスタマイズ

各ビューは、ビジネス プロセスを合理化するためにカスタマイズできます。 ビューはデータをセグメントするのに役立ちます。 各ビューは、ビジネス プロセスを合理化するためにカスタマイズできます。 ビューはデータをセグメントするのに役立ちます。 フィルター、ソート、フィールド ディスプレイを使用してビューをカスタマイズできます。

### ビューのフィルタ

ビューをフィルタするには:

- **フィルター** をクリックし、フィールド (例: People オブジェクトの名前や会社) を選択します。
- 条件を選択し、値を選択してフィルタを適用します。
- 「+フィルタを追加」ボタンを使用してフィルタを追加するか、「X」をクリックして削除します。

### ビューのソート

フィールド データを昇順または降順に並び替えます:

- **ソート** を選択し、フィールドを選択して、希望するソート順を定義します。
- 必要に応じて、複数のソートを適用および配置できます。

### フィールドの表示

フィールド列のヘッダーをクリックして、**右に移動** または **左に移動** を押してフィールドの位置を変更します。
