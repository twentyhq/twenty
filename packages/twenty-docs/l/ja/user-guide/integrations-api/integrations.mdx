## 現在の統合オプション

### ネイティブ統合

Twenty は現在、次のサービスとネイティブ統合を提供しています:

- **メール & カレンダー**: Gmail、Outlook、または SMTP/CalDAV プロバイダーを接続
- **API アクセス**: カスタム統合を構築するために REST および GraphQL API を使用

メールとカレンダーの設定については、[メール & カレンダーのセットアップ](/l/ja/user-guide/settings/email-calendar-setup)をご覧ください。

### ワークフロー（推奨）

Twenty を他のツールに接続する主な方法は、アプリ内の **Workflows** 機能を通じてです:

- **HTTP ノード**: 外部サービスへの API コール
- **コードノード**: 複雑な統合のためのカスタムロジックを書く
- **Webhook トリガー**: 外部システムからデータを受信
- **フィールド更新トリガー**: CRM の変更に基づいて自動化されたアクション

詳細については、[ワークフローセクション](/l/ja/user-guide/workflows/getting-started-workflows)をご覧ください。

### Zapier 統合（レガシー）

コード不要の自動化を好むユーザー向けに Zapier 統合を維持しています:

1. [Zapier 上の Twenty](https://zapier.com/apps/twenty/integrations) を訪問
2. Twenty をトリガーまたはアクションとして新しい Zap を作成
3. 設定 → API & Webhooks で API キーを生成
4. Twenty ワークスペースを Zapier に接続

## 今後の展望: コミュニティが構築するコネクター（2026年）

### 拡張性プラットフォーム

開発者がコードとしてアプリを作成できるようにする拡張性プラットフォームを構築しています。 これにより可能になること: これにより可能になること:

- **カスタムコネクター**: お気に入りのソフトウェアとの統合を構築
- **コミュニティ貢献**: 他の人が構築したコネクターを共有・発見
- **柔軟なアーキテクチャ**: Twenty の機能をコア機能を超えて拡張

### AI支援によるワークフローの構築

Coming in 2026, AI assistance will help users:

- **ワークフローの自動生成**: 統合のニーズを自然言語で説明
- **スマート提案**: 特定のツールを接続するための推奨案を取得
- **テンプレートライブラリ**: 一般的な使用ケースに対する事前構築されたワークフローへのアクセス

Twenty 統合の未来は、コミュニティ駆動、AI支援、および無限に拡張可能です。
