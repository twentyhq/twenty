### 選択を導く質問

自問してください：

- これはすでに持っているものの単なるプロパティか、それとも独自のプロパティが必要ですか？
- その記録ごとに複数個を追跡する必要があり、事前に何個であるかを知ることができませんか？
- この概念は、1つのオブジェクトだけでなく、いくつかの異なるオブジェクトに関連していますか？
- 固有のライフサイクル（例：ステージ、開始/終了日）を持ちますか？

これらのいずれかに「はい」と答える場合、新しいオブジェクトの準備ができた合図です。
