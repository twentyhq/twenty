## 現在の状況

Twenty のカレンダー連携は、既存のカレンダーイベントを**同期して表示**するように設計されています。

## ワークフローによるミーティングのスケジューリング

ミーティングを手動でスケジュールすることはできませんが、Twenty ではワークフローを使ってスケジュールできます。

1. 新しいワークフローを作成
2. ニーズに応じて任意のトリガーを選択します
3. **Create Calendar Event** ノードを追加します
4. 必要なデータをすべて設定します
5. ワークフローを有効化

## 現在できること

* **ミーティング履歴を表示**（People、Companies、Opportunities のレコード上）
* CRM 内の連絡先との**今後のミーティングを確認**
* メールのやり取りと併せて**ミーティングのコンテキストを追跡**
* ミーティング参加者から**連絡先を自動作成**
