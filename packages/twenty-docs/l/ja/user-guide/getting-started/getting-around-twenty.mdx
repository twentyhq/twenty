## 設定

左上から設定を開いて次のことを行います：

- メールボックスとカレンダーアカウントをシームレスに同期
- **データモデルをカスタマイズ**：カスタムオブジェクト、フィールド、および関係を作成
- APIプレイグラウンドにアクセスし、Webhooksを設定
- ユーザー権限とワークスペースアクセス制御を**管理**
- チームメンバーを招待し、ユーザー役割を管理
- プロフィールとワークスペースの設定を編集
- 請求を設定し、ワークフロークレジットの使用を追跡
- 最新のリリースや今後の機能を発見（リリース → ラブタブ）

設定でこれらのセクションすべてが見られない場合は、ワークスペース管理者にお問い合わせください - いくつかはアクセス制限があります。
