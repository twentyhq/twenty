## 2. データを導入する

既存のデータをTwentyに導入すると、チームが最初からコンテキストを得ることができます。

### メールボックスを接続する

If you have not done so when creating your workspace, connect your **Google or Microsoft account** under Settings → Accounts. これにより、Twentyが次のことを行えるようになります：

- メッセージと会議のインポート
- インタラクションに基づく自動連絡先作成（オプション）
- チームにコミュニケーション履歴を表示し続ける

**他のプロバイダーを使用していますか？**
SMTP経由で別のメールボックスを追加するか、CalDAV経由で別のカレンダーを追加できます。 [設定] → [リリース] → [ラボ]で機能を有効にしてから、[設定] → [アカウント]タブに戻る必要があります。 [設定] → [リリース] → [ラボ]で機能を有効にしてから、[設定] → [アカウント]タブに戻る必要があります。

### csvファイルでデータをインポートする

コマンドメニュー（`Cmd + K`または`Ctrl + K`）を使用して、CSV経由で人、会社、機会、または任意のカスタムオブジェクトをインポートします。

**重要なガイドライン**：

- サンプルファイルをダウンロードして予想される形式を理解します
- 各ファイルを10kレコードまで制限する
- 人の重複メール、会社の重複ドメインを削除する
- インポート前にエラー（黄色でハイライト）をレビューして修正する

データインポートについてさらに学ぶには、[この記事](/l/ja/user-guide/getting-started/import-export-data)を読んでください。
