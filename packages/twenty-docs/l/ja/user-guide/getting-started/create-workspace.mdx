## ステップ 1: 登録

1. [Twenty サインアップ](https://app.twenty.com) に移動します。
2. 希望のサインアップ方法を選択します:
   - **Google で続行** して Google アカウントで登録します。
   - **Microsoft で続行** して Microsoft アカウントで登録します。
   - または、**メールで続行** してメールで登録します。

<VimeoEmbed videoId="927066829" title="Video demonstration" />
## Step 2: Choosing a Trial Period

2 つのトライアル期間から選択:

### 30 日間

クレジットカードで

### 7 日間

クレジットカードなしで

どちらのトライアルも含まれます:

- フルアクセス
- 連絡先無制限
- メール統合
- カスタムオブジェクト
- API & Webhooks

「プランの変更」をクリックして、別のプランや請求間隔を選択できます。
