## カレンダービューについて

カレンダービューは、日付フィールドに基づいてレコードをカレンダー上に表示します。 各レコードは、対応する日付にイベントとして表示されます。

<img src="/images/user-guide/views/calendar-view-zoom.png" style={{width:'100%'}} />

## カレンダービューの作成

1. 日付フィールドのあるオブジェクトに移動します
2. ビューのドロップダウンをクリックし、**+ ビューを追加**を選択します
3. ビューに名前を付け、**作成**をクリックします
4. 右側の**オプション**を開きます
5. レイアウトとして**カレンダー**を選択します
6. レコードの配置に使用する**日付フィールド**を選択します
7. **ビューを更新**をクリックします

## カレンダーの設定

### 日付フィールドを選択

**オプション**で、カレンダー上の表示位置を決定する日付フィールドを選択します。

### 表示するフィールド

各カレンダーイベントに表示するフィールドを設定します:

1. **オプション → フィールド**をクリックします
2. フィールドのオン/オフを切り替えます
3. ドラッグして並べ替えます

## ユースケース

* **会議と通話**: 今後の予定を表示
* **期限**: 期日やクローズ日を追跡
* **イベント**: 予定された活動の計画と可視化
* **フォローアップ**: タスクの期日を確認
