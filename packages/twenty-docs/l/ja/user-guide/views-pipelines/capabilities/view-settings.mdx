## ビューの可視性

カスタムビューを誰が閲覧できるかを管理します。

### 表示範囲のオプション

| 設定          | 誰が閲覧できるか      |
| ----------- | ------------- |
| **ワークスペース** | ワークスペースの全メンバー |
| **非公開**     | 自分のみ          |

### 表示範囲を変更する

1. ビューを開きます
2. **オプション → 表示範囲**をクリックします
3. **ワークスペース** または **非公開** を選択します

<Note>
  デフォルトの "All [Object Name]" ビューは表示範囲を変更できません。
</Note>

## ビューの名前を変更する

1. ビュードロップダウンを開きます
2. ビューの横にある **⋮** メニューをクリックします
3. **編集** を選択します
4. 新しい名前を入力します

## ビューのアイコンを変更する

1. ビュードロップダウンを開きます
2. ビューの横にある **⋮** メニューをクリックします
3. **編集** を選択します
4. アイコンをクリックして変更します

## ビューを並べ替える

ドロップダウンに表示されるビューの順序を変更します:

1. ビュードロップダウンを開きます
2. ハンドルを使ってビューをドラッグします
3. 目的の位置にドロップします
4. 順序は自動的に保存されます

## お気に入り

頻繁に使用するビューをピン留めして、すばやくアクセスできるようにします:

1. ビュードロップダウンを開きます
2. ビューの横にある **⋮** メニューをクリックします
3. **お気に入りに追加** を選択します

お気に入りに追加したビューは、簡単にアクセスできる専用セクションに表示されます。

## ビューを削除する

1. ビュードロップダウンを開きます
2. ビューの横にある **⋮** メニューをクリックします
3. **削除** を選択します
4. 削除を確認します

<Warning>
  削除したビューは復元できません。
</Warning>
