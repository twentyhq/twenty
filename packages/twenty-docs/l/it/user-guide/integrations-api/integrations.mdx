## Opzioni di Integrazione Correnti

### Integrazioni Native

Twenty attualmente offre integrazione nativa con:

- **Email & Calendario**: Connetti Gmail, Outlook, o fornitori SMTP/CalDAV
- **Accesso API**: Usa le nostre API REST e GraphQL per costruire integrazioni personalizzate

Per l'impostazione di email e calendario, visita la sezione [Impostazioni Email & Calendario](/l/it/user-guide/settings/email-calendar-setup).

### Flussi di Lavoro (Consigliato)

The primary way to connect Twenty to other tools is through our in-app **Workflows** feature:

- **Nodi HTTP**: Effettua chiamate API a servizi esterni
- **Nodi Codice**: Scrivi logiche personalizzate per integrazioni complesse
- **Trigger di Webhook**: Ricevi dati da sistemi esterni
- **Trigger di Aggiornamento Campo**: Automatizza azioni basate su cambiamenti CRM

Learn more in our [Workflows section](/l/it/user-guide/workflows/getting-started-workflows).

### Integrazione Zapier (Legacy)

Manteniamo un'integrazione con Zapier per utenti che preferiscono automazione senza codice:

1. Visita [Twenty su Zapier](https://zapier.com/apps/twenty/integrations)
2. Crea un nuovo Zap con Twenty come trigger o azione
3. Genera una chiave API in Impostazioni → API & Webhooks
4. Connetti il tuo workspace Twenty a Zapier

## Visione del Futuro: Connettori Costruiti dalla Comunità (2026)

### Piattaforma di Estensibilità

Stiamo costruendo una piattaforma di estensibilità che permetterà agli sviluppatori di creare app come codice. Questo permetterà:

- **Connettori Personalizzati**: Costruisci integrazioni con il tuo software preferito
- **Contributi della Comunità**: Condividi e scopri connettori costruiti da altri
- **Architettura Flessibile**: Estendi le capacità di Twenty oltre le funzionalità principali

### Creazione di Flussi di Lavoro Assistita da AI

In arrivo nel 2026, l'assistenza AI aiuterà gli utenti:

- **Generazione Automatica di Flussi di Lavoro**: Descrivi le tue esigenze di integrazione in linguaggio naturale
- **Suggerimenti Intelligenti**: Ricevi raccomandazioni per connettere i tuoi strumenti specifici
- **Libreria di Modelli**: Accedi a flussi di lavoro pre-costruiti per casi d'uso comuni

Il futuro delle integrazioni Twenty è guidato dalla comunità, assistito dall'AI e infinitamente estensibile.
