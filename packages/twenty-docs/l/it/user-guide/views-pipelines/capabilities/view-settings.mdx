## Visibilità delle viste

Controlla chi può vedere le tue viste personalizzate.

### Opzioni di visibilità

| Impostazione      | Chi può vedere                        |
| ----------------- | ------------------------------------- |
| **Workspace**     | Tutti i membri dello spazio di lavoro |
| **Non in elenco** | Solo tu                               |

### Modificare la visibilità

1. Apri la vista
2. Fai clic su **Opzioni → Visibilità**
3. Seleziona **Workspace** o **Non in elenco**

<Note>
  Le viste predefinite "All [Object Name]" non possono avere la visibilità modificata.
</Note>

## Rinominare una vista

1. Apri il menu a discesa della vista
2. Fai clic sul menu **⋮** accanto alla vista
3. Seleziona **Modifica**
4. Inserisci il nuovo nome

## Modificare l'icona della vista

1. Apri il menu a discesa della vista
2. Fai clic sul menu **⋮** accanto alla vista
3. Seleziona **Modifica**
4. Fai clic sull'icona per modificarla

## Riordinare le viste

Modifica l'ordine in cui le viste compaiono nel menu a discesa:

1. Apri il menu a discesa della vista
2. Trascina le viste tramite la maniglia
3. Rilascia nella posizione desiderata
4. L'ordine viene salvato automaticamente

## Preferiti

Fissa le viste usate di frequente per un accesso rapido:

1. Apri il menu a discesa della vista
2. Fai clic sul menu **⋮** accanto a una vista
3. Seleziona **Aggiungi ai preferiti**

Le viste aggiunte ai preferiti compaiono in una sezione dedicata per un accesso rapido.

## Eliminare una vista

1. Apri il menu a discesa della vista
2. Fai clic sul menu **⋮** accanto alla vista
3. Seleziona **Elimina**
4. Conferma l'eliminazione

<Warning>
  Le viste eliminate non possono essere ripristinate.
</Warning>
