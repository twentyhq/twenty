## Informazioni sulle viste della tabella

Le viste tabella mostrano i record in righe con colonne personalizzabili—come un foglio di calcolo. Questo è il tipo di vista predefinito per la maggior parte degli oggetti.

<img src="/images/user-guide/table-views/table-view.png" style={{width:'100%'}} />

## Funzionalità

### Configurazione delle colonne

* Mostra o nascondi colonne (campi)
* Ridimensiona la larghezza delle colonne
* Riordina le colonne trascinandole

### Raggruppa per un campo di tipo Select

Organizza i record in gruppi comprimibili in base a un campo di tipo Select.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

1. Fai clic su **Opzioni**
2. Seleziona **Raggruppa**
3. Scegli un campo di tipo Select
4. Configura l'ordine dei gruppi in **Opzioni → Raggruppa → Ordina**:
   * **Alfabetico** o **Alfabetico inverso**
   * **Ordine manuale**: trascina i gruppi sotto "Gruppi visibili" per riordinarli
   * Fai clic sull'**icona a forma di occhio** accanto a un gruppo per nasconderlo

**Casi di utilizzo:**

* Raggruppa Aziende per tipo
* Raggruppa Opportunità per fase
* Raggruppa Attività per stato

<Note>
  **Per prestazioni ottimali, limita a 10-15 gruppi visibili per vista.** Se ti servono più gruppi, prendi in considerazione l'uso di una Dashboard.
</Note>

### Larghezza delle colonne

Ridimensiona le colonne per mostrare più o meno contenuto:

1. Posiziona il puntatore tra due intestazioni di colonna
2. Fai clic e trascina il bordo della colonna
3. Rilascia per impostare la nuova larghezza

## Quando utilizzare le viste tabella

Le viste tabella sono ideali per:

* **Esplorare set di dati di grandi dimensioni** — scorrere rapidamente molti record
* **Inserimento dati** — modificare più record in modo efficiente
* **Analisi dettagliata** — vedere molti campi contemporaneamente
* **Ordinamento e filtraggio** — trovare record specifici
