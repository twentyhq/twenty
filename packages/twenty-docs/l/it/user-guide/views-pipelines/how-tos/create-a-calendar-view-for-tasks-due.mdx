## Prerequisiti

Il tuo oggetto Attività necessita di un campo **Data di scadenza** (tipo Data o Data e Ora).

## Passaggi

1. Vai su **Attività**
2. Clicca sul menu a discesa della vista → **+ Aggiungi vista**
3. Dai un nome alla tua vista (ad es., "Calendario attività")
4. Clicca **Crea**
5. Clicca su **Opzioni** e seleziona **Calendario** come layout
6. Scegli **Data di scadenza** come campo data
7. Clicca su **Salva**

## Configura il tuo calendario

### Visualizza i campi negli eventi

1. Clicca su **Opzioni → Campi**
2. Clicca sull'**icona dell'occhio** per mostrare/nascondere i campi
3. Trascina per riordinare

Campi consigliati da visualizzare:

* **Titolo** — nome dell'attività
* **Assegnatario** — chi è responsabile
* **Stato** — avanzamento attuale

### Filtra il tuo calendario

Crea viste mirate:

* **Le mie attività**: Filtra per Assegnatario = Io
* **Questa settimana**: Filtra per Data di scadenza = Questa settimana
* **Overdue**: Filter by Due Date < Today, Status ≠ Done

## Other Calendar Use Cases

| Oggetto       | Date Field | Purpose                   |
| ------------- | ---------- | ------------------------- |
| Opportunità   | Close Date | Track expected closes     |
| Custom Events | Event Date | Plan activities           |
| Projects      | Deadline   | Monitor project timelines |

## Tips

* **Review weekly**: Start each week by checking your calendar view
* **Combine with table view**: Use calendar for overview, table for details
* **Set visibility**: Keep personal task calendars as Unlisted
