## Informazioni

Il repository GitHub di Twenty ospita una vasta gamma di risorse come codice sorgente, documentazione, discussioni e tracciamento dei problemi. È qui che sarai in grado di accedere al codice completo dietro Twenty.

[Visita Twenty su GitHub](https://github.com/twentyhq/twenty)

## Contribuire

Contribuire al progetto Twenty su GitHub è un modo gratificante per migliorare il software che utilizzi. Che tu stia risolvendo bug, suggerendo funzionalità o migliorando la documentazione, i tuoi contributi sono benvenuti.

### Segnalazione di Problemi

Hai riscontrato un problema? [Create an issue](https://github.com/twentyhq/twenty/issues/new) on GitHub, providing as much detail as possible.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggerire Funzionalità

Quali miglioramenti ti piacerebbe vedere su Twenty? Indipendentemente dalla tua competenza tecnica, puoi unirti [alla conversazione qui](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Codificare una Funzionalità

Inizia il tuo viaggio trovando compiti adatti ai principianti:

1. Navigate to the **[Issues](https://github.com/twentyhq/twenty/issues)** tab on the Twenty repository.
2. Filter by **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** label to find tasks suited for newcomers.
3. Scegli un problema, fai fork del repository e inizia a contribuire.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Assicurati di essere assegnato al problema per evitare sovrapposizioni di lavoro con altri contributori.

### Codice di Condotta

Ricorda di attenerti al [Codice di Condotta](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) di Twenty durante il processo di contributo.
