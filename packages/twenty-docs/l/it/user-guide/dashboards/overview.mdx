## Comprendere le dashboard

Le dashboard in Twenty offrono un modo visivo per monitorare le tue metriche chiave delle prestazioni e ottenere informazioni approfondite dai dati del tuo CRM.

<img src="/images/user-guide/dashboard/dashboards_v2.png" alt="Cruscotti" />

## Concetti chiave

### Cruscotti

Una dashboard è una raccolta di schede che mostra i dati del tuo CRM a colpo d'occhio. Puoi creare più dashboard per scopi diversi:

* Prestazioni di vendita
* Attività del team
* Salute della pipeline
* Metriche personalizzate

### Schede

Le schede ti consentono di organizzare la tua dashboard in sezioni. Ogni scheda contiene uno o più widget.

### Widget

I widget sono visualizzazioni singole che mostrano dati specifici. Tipi disponibili:

* Grafici a barre
* Grafici a torta
* Grafici a linee
* Metriche numeriche
* iFrame

<Note>
  **Limitazioni attuali**:

  * L'esportazione delle dashboard e la condivisione con utenti esterni (utenti non Twenty) non sono disponibili al momento.
  * I grafici a indicatore e le tabelle non sono ancora disponibili.
</Note>

## Per iniziare

### Creare la tua prima dashboard

1. Vai alla sezione **Dashboard**
2. Fai clic su **+ Nuova dashboard**
3. Assegna un nome alla tua dashboard
4. Aggiungi schede per organizzare i tuoi contenuti
5. Aggiungi widget per visualizzare i tuoi dati
6. Salva

### Aggiungere widget

1. Apri una scheda nella tua dashboard
2. Fai clic su **+ Aggiungi widget**
3. Seleziona il tipo di widget
4. Scegli l'origine dati (oggetto)
5. Configura le impostazioni del widget
6. Salva e visualizza il tuo widget

## Migliori Pratiche

* **Inizia in modo semplice**: parti da alcune metriche chiave e aggiungine altre nel tempo
* **Concentrati sui dati azionabili**: mostra metriche che guidano le decisioni
* **Revisione regolare**: controlla regolarmente le tue dashboard per individuare le tendenze
* **Condividi con il team**: rendi le dashboard visibili ai membri del team pertinenti
