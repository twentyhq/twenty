## Widget disponibili

Twenty offre diversi tipi di widget per visualizzare i dati del tuo CRM.

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-bar-popular" />Bar Charts</span>

Mostra i dati come barre orizzontali o verticali.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-bar-chart.png" alt="Bar chart" />
</Frame>

**Ideali per:**

* Confrontare i valori tra categorie
* Mostrare classifiche
* Monitorare le metriche per periodo di tempo

**Esempi di utilizzo:**

* Opportunità per fase
* Ricavi per rappresentante commerciale
* Contatti aggiunti per mese

<Note>
  **Display limits**: Bar charts can show a maximum of 100 bars, and 50 groups per bar when using Group By. Se visualizzi l'avviso "Undisplayed data: max X bars per chart", aggiungi filtri per restringere i dati o modifica il raggruppamento (ad es., raggruppa per settimana invece che per giorni).
</Note>

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-pie" />Pie Charts</span>

Mostra le proporzioni del totale.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-pie-chart.png" alt="Pie chart" />
</Frame>

**Ideali per:**

* Mostrare composizione o distribuzione
* Confrontare le parti con il totale
* Evidenziare i segmenti principali

**Esempi di utilizzo:**

* Distribuzione delle opportunità per fonte
* Ripartizione dei contatti per settore
* Composizione della pipeline per proprietario

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-line" />Line Charts</span>

Mostra le tendenze nel tempo.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-line-chart.png" alt="Line chart" />
</Frame>

**Ideali per:**

* Monitorare i cambiamenti nel tempo
* Individuare le tendenze
* Confrontare più metriche

**Esempi di utilizzo:**

* Andamento mensile del numero di opportunità
* Crescita dei ricavi per trimestre
* Livelli di attività nel tempo

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="sum" />Aggregate Charts</span>

Mostra in evidenza singoli valori chiave.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-aggregate-chart.png" alt="Aggregate chart" />
</Frame>

**Ideali per:**

* Mettere in evidenza i KPI
* Mostrare totali o medie
* Controlli rapidi dello stato

**Esempi di utilizzo:**

* Valore totale della pipeline
* Numero di opportunità aperte
* Tasso di conversione

**Opzioni avanzate:**

* **Rapporto**: Per i campi di tipo Select, calcola i rapporti tra i valori. Vai a **Dati visualizzati** → seleziona il tuo campo → abilita l'opzione **Rapporto**.
* **Prefisso e suffisso**: Aggiungi un testo personalizzato prima o dopo il numero (ad es., prefisso "$" o suffisso "%") per una migliore leggibilità.

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="frame" />iFrames</span>

Incorpora strumenti e contenuti esterni direttamente nella tua dashboard.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-iframe.png" alt="iFrame" />
</Frame>

**Ideali per:**

* Visualizzare report o dashboard esterni
* Integrare strumenti di vendita di terze parti
* Mostrare contenuti in tempo reale da altri sistemi

**Esempi di utilizzo:**

* Metriche dal tuo strumento di assistenza
* Metriche dal tuo dialer
* Contenuti in tempo reale dal tuo strumento per sequenze di vendita

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="align-box-left-top" />Rich Text</span>

Add formatted text and content directly to your dashboard.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-rich-text.png" alt="Rich text" />
</Frame>

**Ideale per:**

* Adding context or instructions
* Dashboard descriptions
* Notes and annotations

**Features:**

* Rich text formatting (bold, italic, lists, etc.)
* Image and file uploads
* Markdown-style editing

### Duplicazione dei widget

1. Fai clic sul widget
2. Apri **Opzioni**
3. Fai clic su **Duplica widget**
