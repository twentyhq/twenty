## Widget disponibili

Twenty offre diversi tipi di widget per visualizzare i dati del tuo CRM.

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-bar-popular" />Grafici a barre</span>

Mostra i dati come barre orizzontali o verticali.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-bar-chart.png" alt="Grafico a barre" />
</Frame>

**Ideali per:**

* Confrontare i valori tra categorie
* Mostrare classifiche
* Monitorare le metriche per periodo di tempo

**Esempi di utilizzo:**

* Opportunità per fase
* Ricavi per rappresentante commerciale
* Contatti aggiunti per mese

<Note>
  **Limiti di visualizzazione**: I grafici a barre possono mostrare al massimo 100 barre e, quando si usa Raggruppa per, 50 gruppi per barra. Se visualizzi l'avviso "Undisplayed data: max X bars per chart", aggiungi filtri per restringere i dati o modifica il raggruppamento (ad es., raggruppa per settimana invece che per giorni).
</Note>

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-pie" />Grafici a torta</span>

Mostra le proporzioni del totale.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-pie-chart.png" alt="Grafico a torta" />
</Frame>

**Ideali per:**

* Mostrare composizione o distribuzione
* Confrontare le parti con il totale
* Evidenziare i segmenti principali

**Esempi di utilizzo:**

* Distribuzione delle opportunità per fonte
* Ripartizione dei contatti per settore
* Composizione della pipeline per proprietario

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="chart-line" />Grafici a linee</span>

Mostra le tendenze nel tempo.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-line-chart.png" alt="Grafico a linee" />
</Frame>

**Ideali per:**

* Monitorare i cambiamenti nel tempo
* Individuare le tendenze
* Confrontare più metriche

**Esempi di utilizzo:**

* Andamento mensile del numero di opportunità
* Crescita dei ricavi per trimestre
* Livelli di attività nel tempo

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="sum" />Grafici aggregati</span>

Mostra in evidenza singoli valori chiave.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-aggregate-chart.png" alt="Grafico aggregato" />
</Frame>

**Ideali per:**

* Mettere in evidenza i KPI
* Mostrare totali o medie
* Controlli rapidi dello stato

**Esempi di utilizzo:**

* Valore totale della pipeline
* Numero di opportunità aperte
* Tasso di conversione

**Opzioni avanzate:**

* **Rapporto**: Per i campi di tipo Select, calcola i rapporti tra i valori. Vai a **Dati visualizzati** → seleziona il tuo campo → abilita l'opzione **Rapporto**.
* **Prefisso e suffisso**: Aggiungi un testo personalizzato prima o dopo il numero (ad es., prefisso "$" o suffisso "%") per una migliore leggibilità.

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="frame" />iFrames</span>

Incorpora strumenti e contenuti esterni direttamente nella tua dashboard.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-iframe.png" alt="iFrame" />
</Frame>

**Ideali per:**

* Visualizzare report o dashboard esterni
* Integrare strumenti di vendita di terze parti
* Mostrare contenuti in tempo reale da altri sistemi

**Esempi di utilizzo:**

* Metriche dal tuo strumento di assistenza
* Metriche dal tuo dialer
* Contenuti in tempo reale dal tuo strumento per sequenze di vendita

### <span style={{ display: 'inline-flex', alignItems: 'center', gap: '8px' }}><ChartIcon icon="align-box-left-top" />Testo formattato</span>

Aggiungi testo formattato e contenuti direttamente alla tua dashboard.

<Frame>
  <img src="/images/user-guide/dashboard/dashboard-widget-rich-text.png" alt="Testo formattato" />
</Frame>

**Ideale per:**

* Aggiunta di contesto o istruzioni
* Descrizioni della dashboard
* Note e annotazioni

**Funzionalità:**

* Formattazione del testo (grassetto, corsivo, elenchi, ecc.)
* Caricamento di immagini e file
* Modifica in stile Markdown

<Note>
  **In arrivo**: I grafici a indicatore e le tabelle non sono ancora disponibili ma sono nella nostra roadmap.

### Duplicazione dei widget

1. Fai clic sul widget
2. Apri **Opzioni**
3. Fai clic su **Duplica widget**
