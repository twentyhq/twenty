## Novità in arrivo

### Report personalizzati

Crea report su misura per monitorare le metriche più importanti per il tuo business.

### Dashboard delle vendite

Visualizza le prestazioni del tuo percorso di vendita con dashboard interattive.

### Analisi delle prestazioni

Monitora le prestazioni del team e identifica le tendenze nei dati dei tuoi clienti.

## Rimani aggiornato

Segui il nostro [repository GitHub](https://github.com/twentyhq/twenty) o controlla la sezione **Impostazioni → Rilasci** nel tuo workspace per essere notificato quando le funzionalità di reportistica saranno disponibili.
