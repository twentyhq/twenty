## Panoramica

I crediti alimentano le automazioni dei tuoi flussi di lavoro in Twenty. Ogni azione del flusso di lavoro consuma crediti in base alla sua complessità.

## Allocazione dei crediti

I crediti si basano sul tuo ciclo di fatturazione, non sul tuo piano:

| Ciclo di fatturazione | Crediti         |
| --------------------- | --------------- |
| Mensile               | 5 milioni/mese  |
| Annuale               | 50 milioni/anno |

<Note>
  I 5 milioni di crediti mensili sono pensati per permetterti di eseguire automazioni senza preoccuparti dei costi. Per la maggior parte dei flussi di lavoro che utilizzano azioni standard, questo è più che sufficiente. Avrai bisogno di crediti aggiuntivi solo quando esegui nodi di codice avanzati o funzionalità basate sull'IA.
</Note>

## Consumo di crediti

Azioni diverse consumano quantità diverse di crediti:

| Tipo di azione                                                       | Utilizzo dei Crediti           |
| -------------------------------------------------------------------- | ------------------------------ |
| **Operazioni di base** (ricerca, aggiornamento, creazione di record) | Minimo                         |
| **Operazioni complesse** (nodi di codice, chiamate alle API esterne) | Più crediti                    |
| **Prompt IA** (in arrivo)                                            | Variabile in base all'utilizzo |

I crediti vengono detratti in tempo reale quando i workflow vengono eseguiti.

## Monitoraggio dell'utilizzo

Monitora il consumo dei tuoi crediti:

1. Vai a **Impostazioni → Fatturazione**
2. Visualizza il tuo utilizzo attuale e i crediti rimanenti
3. Monitora le tendenze per pianificare crediti aggiuntivi se necessario

## Acquisto di Crediti Aggiuntivi

Hai bisogno di più crediti?

1. Vai a **Impostazioni → Fatturazione**
2. Fai clic sull'opzione per acquistare pacchetti di crediti aggiuntivi
3. Seleziona la quantità di cui hai bisogno
