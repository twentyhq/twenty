## Informazioni sulle impostazioni degli aggiornamenti

La sezione Aggiornamenti consente di abilitare le funzionalità di accesso anticipato prima che siano generalmente disponibili.

## Funzionalità di accesso anticipato

Le funzionalità di accesso anticipato sono funzionalità ancora in fase di test prima della disponibilità generale.

**Attualmente in Accesso anticipato:**

* **Dashboard** — Reportistica e analisi visuale

### Come abilitare le funzionalità di accesso anticipato

1. Vai su **Impostazioni → Aggiornamenti**
2. Vai alla sezione **Accesso anticipato**
3. Trova la funzionalità che desideri abilitare
4. Attivala
5. La funzionalità sarà disponibile immediatamente

<Note>
  Le funzionalità di accesso anticipato possono cambiare in base al feedback degli utenti. Il tuo contributo ci aiuta a migliorare Twenty!
</Note>

## Feedback sulle funzionalità

Il tuo feedback aiuta a migliorare Twenty:

* Segnala problemi con le funzionalità di accesso anticipato
* Condividi come stai utilizzando le nuove funzionalità
* Suggerisci miglioramenti tramite il Discord della community
