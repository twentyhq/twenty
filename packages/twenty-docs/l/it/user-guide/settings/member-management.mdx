## Invita Nuovi Membri

### Usando il Link di Invito

1. Vai a **Impostazioni → Membri**
2. Copia il link di invito allo spazio di lavoro
3. Condividi il link con i nuovi membri del team
4. Riceveranno accesso una volta registrati

### Invito Diretto via Email

1. Vai a **Impostazioni → Membri**
2. Inserisci l'indirizzo email della persona
3. Clicca **Invita**
4. Riceveranno un invito per email

## Rimuovi Membri

### Elimina un Membro

1. Vai a **Impostazioni → Membri**
2. Trova il membro che desideri rimuovere
3. Clicca sul pulsante elimina/rimuovi accanto al loro nome
4. Conferma la rimozione

**Nota**: I membri rimossi perdono l'accesso immediatamente ma possono essere reinvitati in seguito.
