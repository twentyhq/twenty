---
title: Conectar varios buzones por usuario
description: Conecta varias cuentas de correo electrónico para un único usuario.
---

## Resumen

Twenty admite **un número ilimitado de cuentas de correo electrónico por usuario**. Esto es útil si gestionas varias bandejas de entrada, como:

* Correo laboral personal + bandeja de entrada compartida del equipo
* Varias direcciones de correo electrónico de cara al cliente
* Cuentas de correo electrónico diferentes para distintos roles

## Cómo añadir varios buzones

1. Ve a **Ajustes → Cuentas**
2. Haz clic en **Añadir cuenta**
3. Conecta tu cuenta adicional de Google o Microsoft
4. Configura los ajustes de sincronización para este buzón
5. Repite para cada buzón que quieras conectar

## Gestión de varias cuentas

Cada buzón conectado tiene sus propios ajustes:

* **Visibilidad del correo**: Elige qué pueden ver los miembros del equipo
* **Creación automática de contactos**: Activar o desactivar por buzón
* **Selección de carpetas**: Elige qué carpetas sincronizar

## Cómo se muestran los correos electrónicos

Los correos de todos tus buzones conectados se sincronizan con Twenty y aparecen en:

* **Registros de personas**: Según la dirección de correo electrónico del contacto
* **Registros de empresas**: Según el dominio del correo electrónico
* **Oportunidades**: Según la empresa vinculada

<Note>Cada correo muestra desde qué buzón se envió/a qué buzón se recibió, para que puedas saber qué cuenta se usó en cada comunicación.</Note>

## Notas importantes

<Warning>Solo se pueden conectar buzones reales. Los alias de correo electrónico que reenvían a otro buzón no se pueden conectar por separado; se sincronizarán a través del buzón principal.</Warning>
