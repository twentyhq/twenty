## Resumen

Integra capacidades de IA directamente en tus flujos de trabajo de automatización para el procesamiento inteligente de datos y la toma de decisiones.

## Capacidades

| Función                      | Descripción                                                |
| ---------------------------- | ---------------------------------------------------------- |
| **Acciones de IA**           | Añade pasos impulsados por IA a cualquier flujo de trabajo |
| **Enriquecimiento de Datos** | Mejora automáticamente los registros con datos externos    |
| **Clasificación**            | Clasifica registros según el análisis de contenido         |
| **Generación de resúmenes**  | Genera resúmenes a partir de campos de texto               |
| **Prompts personalizados**   | Define exactamente cómo la IA procesa tus datos            |

## Casos de uso

* **Puntuación de clientes potenciales**: Puntúa y prioriza automáticamente los clientes potenciales entrantes
* **Limpieza de datos**: Estandariza los nombres de empresas y la información de contacto
* **Borradores de correo electrónico**: Genera correos de seguimiento basados en notas de reuniones
* **Enrutamiento de registros**: Asigna los registros al miembro adecuado del equipo según el contenido
