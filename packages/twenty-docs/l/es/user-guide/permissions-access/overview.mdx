## Características clave

* **Acceso basado en roles**: Crea roles personalizados con permisos específicos
* **Permisos de objetos**: Controla quién puede ver, editar o eliminar registros
* **Permisos de campos**: Restringe el acceso a campos sensibles
* **Permisos de configuración**: Controla el acceso a la configuración del espacio de trabajo
* **Integración con SSO**: Configura el inicio de sesión único para seguridad a nivel empresarial (plan Organization)
