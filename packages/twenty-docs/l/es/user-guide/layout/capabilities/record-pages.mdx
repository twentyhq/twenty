## Resumen

Cuando abres un registro en Twenty, la página de detalle se compone de **pestañas** y **widgets**. Ambos son totalmente personalizables según el tipo de objeto.

## Pestañas

Cada página de registro puede tener varias pestañas — similares a las pestañas de un navegador. Úsalas para organizar distintos aspectos de un registro. Por ejemplo, un registro de Empresa podría tener pestañas para Resumen, Comunicación, Tareas y Archivos.

Puedes:

* Agregar y eliminar pestañas
* Cambiar el nombre de las pestañas
* Reordenar las pestañas arrastrando
* Configurar qué pestaña se muestra de forma predeterminada

## Widgets

Los widgets son los bloques básicos dentro de cada pestaña. Tipos de widgets disponibles:

| Widget                     | Qué muestra                                             |
| -------------------------- | ------------------------------------------------------- |
| **Campos**                 | Campos del registro, agrupados o individualmente        |
| **Registros relacionados** | Tabla de registros vinculados mediante una relación     |
| **Correos electrónicos**   | Historial de correos electrónicos de cuentas conectadas |
| **Calendario**             | Eventos de calendario asociados con el registro         |
| **Cronología**             | Historial de actividades y eventos                      |
| **Tareas**                 | Tareas asociadas                                        |
| **Notas**                  | Notas de texto enriquecido                              |
| **Archivos**               | Archivos adjuntos                                       |
| **Gráficos**               | Datos visuales de registros relacionados                |
| **iFrame**                 | Contenido externo incrustado                            |
| **Texto enriquecido**      | Contenido estático o descripciones                      |

## Personalizar la página del registro

1. Abre cualquier registro
2. Presiona `Cmd+K` y busca "Editar diseño de la página del registro"

o

1. Ve a Configuración > Modelo de datos > objeto de tu elección > Diseño

2. Haz clic en el botón "Personalizar la página del registro" para ese objeto

3. Ahora estás en el modo de personalización:
   * **Agrega widgets** desde el selector de widgets
   * **Arrastra widgets** para reposicionarlos en la cuadrícula
   * **Cambia el tamaño de los widgets** arrastrando sus bordes
   * **Configura los campos** que se muestran dentro de cada widget
   * **Gestiona las pestañas** — agregar, eliminar, cambiar el nombre, reordenar

4. Guarda tus cambios — se aplican a todos los registros de ese tipo de objeto

## Visibilidad de campos

Dentro de un widget de Campos, puedes controlar qué campos son visibles y en qué orden. Esto te permite crear diseños enfocados — por ejemplo, mostrar solo los campos más importantes en la pestaña Resumen y colocar los campos detallados en una pestaña separada.
