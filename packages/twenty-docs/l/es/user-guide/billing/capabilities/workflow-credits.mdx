## Resumen

Los créditos impulsan tus automatizaciones de flujo de trabajo en Twenty. Cada acción de flujo de trabajo consume créditos según su complejidad.

## Asignación de créditos

Los créditos se basan en tu ciclo de facturación, no en tu plan:

| Ciclo de facturación | Créditos        |
| -------------------- | --------------- |
| Mensual              | 5 millones/mes  |
| Anual                | 50 millones/año |

<Note>
  Los 5 millones de créditos mensuales están diseñados para permitirte ejecutar automatizaciones sin preocuparte por los costos. Para la mayoría de los flujos de trabajo que usan acciones estándar, esto es más que suficiente. Solo necesitarás créditos adicionales al ejecutar nodos de código avanzados o funciones impulsadas por IA.
</Note>

## Consumo de créditos

Diferentes acciones consumen distintas cantidades de créditos:

| Tipo de acción                                                       | Uso de Crédito        |
| -------------------------------------------------------------------- | --------------------- |
| **Operaciones básicas** (buscar, actualizar, crear registros)        | Mínimo                |
| **Operaciones complejas** (nodos de código, llamadas a API externas) | Más créditos          |
| **Consultas de IA** (próximamente)                                   | Variable según el uso |

Los créditos se deducen en tiempo real cuando se ejecutan los flujos de trabajo.

## Supervisión del uso

Lleva un seguimiento de tu consumo de créditos:

1. Ir a **Ajustes → Facturación**
2. Consulta tu uso actual y los créditos restantes
3. Supervisa las tendencias para planificar créditos adicionales si es necesario

## Adquirir Créditos Adicionales

¿Necesitas más créditos?

1. Ir a **Ajustes → Facturación**
2. Haz clic en la opción para adquirir paquetes de créditos adicionales
3. Selecciona la cantidad que necesitas
