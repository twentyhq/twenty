## Visibilidad de la vista

Controla quién puede ver tus vistas personalizadas.

### Opciones de visibilidad

| Configuración          | Quién puede ver                           |
| ---------------------- | ----------------------------------------- |
| **Espacio de trabajo** | Todos los miembros del espacio de trabajo |
| **No listada**         | Solo tú                                   |

### Cambiar la visibilidad

1. Abre la vista
2. Haz clic en **Opciones → Visibilidad**
3. Selecciona **Espacio de trabajo** o **No listada**

<Note>
  No se puede cambiar la visibilidad de las vistas predeterminadas "All [Object Name]".
</Note>

## Cambiar el nombre de una vista

1. Abre el menú desplegable de vistas
2. Haz clic en el menú **⋮** junto a la vista
3. Selecciona **Editar**
4. Introduce el nuevo nombre

## Cambiar el icono de la vista

1. Abre el menú desplegable de vistas
2. Haz clic en el menú **⋮** junto a la vista
3. Selecciona **Editar**
4. Haz clic en el icono para cambiarlo

## Reordenar vistas

Cambia el orden en que las vistas aparecen en el menú desplegable:

1. Abre el menú desplegable de vistas
2. Arrastra las vistas con su control de arrastre
3. Suelta en la posición deseada
4. El orden se guarda automáticamente

## Favoritos

Ancla las vistas de uso frecuente para un acceso rápido:

1. Abre el menú desplegable de vistas
2. Haz clic en el menú **⋮** junto a una vista
3. Selecciona **Añadir a favoritos**

Las vistas marcadas como favoritas aparecen en una sección dedicada para facilitar el acceso.

## Eliminar vista

1. Abre el menú desplegable de vistas
2. Haz clic en el menú **⋮** junto a la vista
3. Selecciona **Eliminar**
4. Confirma la eliminación

<Warning>
  Las vistas eliminadas no se pueden recuperar.
</Warning>
