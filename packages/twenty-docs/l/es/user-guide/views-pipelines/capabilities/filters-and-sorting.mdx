## Filtrado de datos

Los filtros te ayudan a centrarte en registros específicos mostrando solo aquellos que cumplen tus criterios.

### Agregar un filtro

1. Haz clic en el botón **Filtrar** en la barra de herramientas
2. Selecciona el campo por el que filtrar
3. Elige el operador (igual a, contiene, etc.)
4. Introduce el valor del filtro
5. Haz clic en **Aplicar**

### Operadores de filtro

| Tipo de campo     | Operadores disponibles                                  |
| ----------------- | ------------------------------------------------------- |
| Texto             | Igual a, Contiene, Empieza con, Termina con, Está vacío |
| Número            | Igual a, Mayor que, Menor que, Entre, Está vacío        |
| Fecha             | Igual a, Antes de, Después de, Entre, Está vacío        |
| Selección         | Igual a, Es cualquiera de, Está vacío                   |
| Caja de selección | Es verdadero, Es falso                                  |
| Relación          | Igual a, Está vacío                                     |

### Varios filtros

Combina varios filtros para acotar los resultados:

* Todos los filtros se aplican con lógica AND
* Cada filtro adicional restringe aún más los resultados

### Eliminar filtros

* Haz clic en la **X** de las etiquetas de filtro individuales
* Haz clic en **Borrar todo** para eliminar todos los filtros

## Ordenar datos

La ordenación determina el orden en que aparecen los registros.

### Agregar un criterio de ordenación

1. Haz clic en el botón **Ordenar** en la barra de herramientas
2. Selecciona el campo por el que ordenar
3. Elige ascendente (A-Z, 0-9) o descendente (Z-A, 9-0)
4. Haz clic en **Aplicar**

### Varios criterios de ordenación

Agrega varios niveles de ordenación:

* La primera ordenación es la principal
* Las ordenaciones posteriores se aplican dentro de los grupos de valores iguales

### Ordenación rápida por columna

Haz clic en el encabezado de una columna para ordenar:

* Primer clic: ascendente
* Segundo clic: descendente
* Tercer clic: quitar la ordenación

## Guardar la configuración de filtros y ordenación

Los filtros y las ordenaciones se guardan con la vista:

1. Configura tus filtros y ordenaciones
2. Haz clic en **Guardar** para actualizar la vista actual
3. O haz clic en **Guardar como nueva vista** para crear una variante
