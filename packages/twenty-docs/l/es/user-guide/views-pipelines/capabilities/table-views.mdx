## Acerca de las Vistas de Tabla

Las vistas de tabla muestran registros en filas con columnas personalizables—como una hoja de cálculo. Este es el tipo de vista predeterminado para la mayoría de los objetos.

<img src="/images/user-guide/table-views/table-view.png" style={{width:'100%'}} />

## Funciones

### Configuración de columnas

* Mostrar u ocultar columnas (campos)
* Ajustar el ancho de las columnas
* Reordenar las columnas arrastrando

### Agrupar por un campo de selección

Organiza los registros en grupos contraíbles según un campo de tipo selección.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

1. Haz clic en **Opciones**
2. Selecciona **Agrupar**
3. Elige un campo de selección
4. Configura el orden de los grupos en **Opciones → Agrupar → Ordenar**:
   * **Alfabético** o **Alfabético inverso**
   * **Orden manual**: Arrastra los grupos en "Grupos visibles" para reordenarlos
   * Haz clic en el **icono de ojo** junto a un grupo para ocultarlo

**Casos de uso:**

* Agrupar Empresas por tipo
* Agrupar Oportunidades por etapa
* Agrupar Tareas por estado

<Note>
  **Para un mejor rendimiento, limita a 10-15 grupos visibles por vista.** Si necesitas más grupos, considera usar un Dashboard en su lugar.
</Note>

### Anchos de columna

Ajusta el ancho de las columnas para mostrar más o menos contenido:

1. Pasa el cursor entre dos encabezados de columna
2. Haz clic y arrastra el borde de la columna
3. Suelta para establecer el nuevo ancho

## Cuándo usar las vistas de tabla

Las vistas de tabla funcionan mejor para:

* **Examinar conjuntos de datos grandes** — revisar muchos registros rápidamente
* **Entrada de datos** — editar varios registros de forma eficiente
* **Análisis detallado** — ver muchos campos a la vez
* **Ordenar y filtrar** — encontrar registros específicos
