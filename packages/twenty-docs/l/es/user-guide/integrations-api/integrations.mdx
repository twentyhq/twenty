## Opciones de Integración Actuales

### Integraciones Nativas

Actualmente, Twenty ofrece integraciones nativas con:

- **Correo Electrónico y Calendario**: Conecte proveedores de Gmail, Outlook o SMTP/CalDAV
- **Acceso a API**: Utilice nuestras APIs REST y GraphQL para crear integraciones personalizadas

Para la configuración de correo electrónico y calendario, visite [Configuración de Correo y Calendario](/l/es/user-guide/settings/email-calendar-setup).

### Flujos de Trabajo (Recomendado)

La forma principal de conectar Twenty a otras herramientas es a través de nuestra función **Flujos de Trabajo** en la aplicación:

- **Nodos HTTP**: Realice llamadas a API de servicios externos
- **Nodos de Código**: Escriba lógica personalizada para integraciones complejas
- **Disparadores de Webhook**: Reciba datos de sistemas externos
- **Disparadores de Actualización de Campo**: Automatice acciones basadas en cambios en el CRM

Aprenda más en nuestra [sección de Flujos de Trabajo](/l/es/user-guide/workflows/getting-started-workflows).

### Integración con Zapier (Legado)

Mantenemos una integración con Zapier para usuarios que prefieren automatización sin código:

1. Visite [Twenty en Zapier](https://zapier.com/apps/twenty/integrations)
2. Cree un nuevo Zap con Twenty como disparador o acción
3. Genere una clave de API en Configuración → API y Webhooks
4. Conecte su espacio de trabajo de Twenty a Zapier

## Visión Futura: Conectores Construidos por la Comunidad (2026)

### Plataforma de Extensibilidad

Estamos construyendo una plataforma de extensibilidad que permitirá a los desarrolladores crear aplicaciones como código. Esto permitirá:

- **Conectores Personalizados**: Cree integraciones con su software favorito
- **Contribuciones de la Comunidad**: Comparta y descubra conectores creados por otros
- **Arquitectura Flexible**: Extienda las capacidades de Twenty más allá de las características principales

### Creación de Flujos de Trabajo Asistida por IA

Llegando en 2026, la asistencia de IA ayudará a los usuarios a:

- **Generación Automática de Flujos de Trabajo**: Describa sus necesidades de integración en lenguaje sencillo
- **Sugerencias Inteligentes**: Reciba recomendaciones para conectar sus herramientas específicas
- **Biblioteca de Plantillas**: Acceda a flujos de trabajo preconstruidos para casos de uso comunes

El futuro de las integraciones de Twenty está impulsado por la comunidad, asistido por IA, y es infinitamente extensible.
