## Capacidades Avanzadas de Ventas

Los equipos de GTM a menudo necesitan capacidades avanzadas de ventas como calificación de clientes potenciales, enriquecimiento de datos, rotación, asignación de territorio, recordatorios automáticos y secuencias de correo electrónico. Aunque estas no son características integradas en Twenty, todas pueden configurarse y adaptarse a sus necesidades específicas utilizando nuestro sistema flexible de flujos de trabajo.

Visite nuestra [sección de Flujos de Trabajo](/l/es/user-guide/workflows/getting-started-workflows) para aprender cómo construir estas automatizaciones paso a paso. Para ejemplos detallados, consulte nuestras guías de [Automatizaciones Internas](/l/es/user-guide/workflows/internal-automations) e [Integración de Herramientas Externas](/l/es/user-guide/workflows/external-tool-integration).
