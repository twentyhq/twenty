---
title: Configuración de Dominios
description: Configura el dominio del espacio de trabajo, los dominios de acceso aprobado y los dominios públicos.
---

Configura los ajustes de dominio en **Ajustes → Dominios**.

## Dominio del espacio de trabajo

Edita el nombre de tu subdominio o establece un dominio personalizado para tu espacio de trabajo.

### Personalizar dominio

1. Haz clic en **Personalizar dominio**
2. Edita tu subdominio (p. ej., `yourcompany.twenty.com`)
3. O configura un dominio personalizado (p. ej., `crm.yourcompany.com`)

Para dominios personalizados, tendrás que configurar los ajustes de DNS con tu proveedor de dominio.

## Dominios Aprobados

Cualquiera con una dirección de correo electrónico en estos dominios puede registrarse automáticamente en este espacio de trabajo.

### Añadir Dominio de Acceso Aprobado

1. Haz clic en **Añadir Dominio de Acceso Aprobado**
2. Introduce el dominio de tu empresa (p. ej., `yourcompany.com`)
3. Guardar

Una vez configurado, cualquiera con una dirección de correo electrónico en ese dominio puede unirse a tu espacio de trabajo sin necesidad de una invitación directa.

<Note>
  Esto es útil para permitir que todo tu equipo se autorregistre y, al mismo tiempo, mantener el espacio de trabajo restringido a tu organización.
</Note>

## Dominios Públicos

Provisiona un entorno de alojamiento completo y seguro en estos dominios.

### Agregar Dominio Público

1. Haz clic en **Agregar Dominio Público**
2. Introduce el dominio que quieres usar
3. Configura los ajustes de DNS según las instrucciones
4. Verifica el dominio

Los certificados SSL se aprovisionan automáticamente para los dominios públicos.
