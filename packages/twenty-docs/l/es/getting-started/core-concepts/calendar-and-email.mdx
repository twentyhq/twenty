Twenty se conecta con tus herramientas existentes para que tu CRM se mantenga actualizado sin necesidad de introducir datos manualmente.

## Sincronización del correo electrónico

Conecta tu buzón de **Google Workspace** o **Microsoft 365**. Una vez conectado:

* Los correos electrónicos se vinculan automáticamente a los registros de Empresa y Personas correspondientes
* Los hilos de correo electrónico completos son visibles en la cronología de cada registro
* Puedes enviar correos electrónicos directamente desde Twenty
* Se admiten múltiples buzones por usuario

Tú controlas lo que se importa: filtra por rango de fechas o remitente para evitar incorporar correos electrónicos irrelevantes.

## Sincronización del calendario

Los eventos del calendario se sincronizan automáticamente desde tu cuenta conectada. Los eventos aparecen en los registros pertinentes del CRM, lo que te ofrece una visión completa de tus interacciones con cada contacto o empresa.

## Integraciones

Más allá del correo electrónico y el calendario, Twenty se conecta a herramientas externas mediante:

| Método                                 | Caso de uso                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------------ |
| **API**                                | Crea integraciones personalizadas con la API GraphQL o REST                          |
| **Webhooks**                           | Envía notificaciones en tiempo real a sistemas externos cuando cambian los registros |
| **Zapier**                             | Conéctate a más de 5.000 aplicaciones sin código                                     |
| **Acciones HTTP de flujos de trabajo** | Llama a cualquier API externa como parte de un flujo de trabajo automatizado         |

## Aplicaciones personalizadas

Los desarrolladores pueden crear aplicaciones completas sobre Twenty, añadiendo interfaz de usuario personalizada, lógica del lado del servidor e integraciones profundas. Las aplicaciones pueden publicarse para la comunidad o mantenerse privadas.
