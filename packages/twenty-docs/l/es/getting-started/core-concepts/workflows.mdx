Los flujos de trabajo te permiten automatizar tareas repetitivas y conectar Twenty con herramientas externas, sin escribir código (aunque puedes hacerlo si quieres).

## Cómo funcionan los flujos de trabajo

Cada flujo de trabajo tiene tres partes:

1. **Disparador** — Lo que inicia el flujo de trabajo
2. **Pasos** — Lo que ocurre a continuación (una o más acciones en secuencia)
3. **Variables** — Datos que fluyen entre los pasos

## Disparadores

| Disparador             | Cuándo se activa                                                                                  |
| ---------------------- | ------------------------------------------------------------------------------------------------- |
| **Evento de registro** | Se crea, actualiza, elimina o se realiza un upsert en un registro                                 |
| **Manual**             | Un usuario hace clic en un botón (en un único registro, en múltiples registros o de forma global) |
| **Programación**       | En un intervalo recurrente (sintaxis cron)                                                        |
| **Webhook**            | Un sistema externo envía un HTTP POST                                                             |

## Acciones

Los flujos de trabajo pueden encadenar cualquier combinación de:

* **Operaciones de registro** — Crear, actualizar, buscar, eliminar o realizar un upsert en registros
* **Enviar correo electrónico** — Enviar o redactar correos electrónicos desde cuentas conectadas
* **Solicitud HTTP** — Llamar a cualquier API externa
* **Código** — Ejecutar JavaScript personalizado para lógica compleja
* **Ramificaciones** — Condiciones if/else para dividir la ruta del flujo de trabajo
* **Iterador** — Recorrer arrays de datos
* **Agente de IA** — Permitir que un agente de IA procese datos de forma autónoma
* **Espera** — Pausar antes de continuar
* **Formulario** — Recoger la información del usuario en mitad del flujo de trabajo

## Lo que puedes crear

* Enviar alertas de Slack cuando una oportunidad alcance una determinada etapa
* Enriquecer automáticamente los nuevos contactos con datos de APIs externas
* Detectar oportunidades estancadas y notificar al propietario
* Sincronizar datos entre Twenty y tu sistema de facturación
* Generar archivos PDF o facturas a partir de los datos de los registros
* Responder automáticamente a los correos electrónicos entrantes que cumplan determinados criterios
