## Aangepast Werkruimtedomein

Stel een gepersonaliseerd webadres voor je werkruimte in:

- Kies je voorkeurs subdomein (bijv. jouwbedrijf.twenty.com)
- Configureer aangepaste domeinen voor professionele branding

## Approved Access Domains

Sta automatische werkruimte toegang voor specifieke e-maildomeinen toe.

### Hoe Het Werkt

- Voeg de e-maildomeinen van je bedrijf toe (bijv. @jouwbedrijf.com)
- Iedereen met een e-mailadres van deze domeinen kan automatisch jouw werkruimte betreden
- Geen handmatige uitnodigingen nodig voor teamleden

### Voordelen

- **Snellere Onboarding**: Nieuwe teamleden treden automatisch toe
- **Betere Beveiliging**: Alleen geverifieerde bedrijfs e-mails hebben toegang
- **Minder Administratief Werk**: Geen noodzaak om elk teamlid handmatig uit te nodigen
