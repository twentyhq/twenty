## Latest Releases

Volg de ontwikkelingsvoortgang van Twenty:

- **Release Notes**: Bekijk wat er nieuw is in elke versie
- **Functie Updates**: Ontdek nieuwe mogelijkheden en verbeteringen

## Lab Features

Toegang tot experimentele functies voordat ze officieel worden vrijgegeven:

- **Beta Testen**: Probeer nieuwe functies terwijl ze in ontwikkeling zijn
- **Feature Flags**: Enable or disable specific experimental functionality
