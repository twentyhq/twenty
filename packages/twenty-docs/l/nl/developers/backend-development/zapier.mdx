Synchroniseer moeiteloos Twenty met meer dan 3000 apps via [Zapier](https://zapier.com/). Automatiseer taken, verhoog de productiviteit en versterk je klantrelaties!

## Over Zapier

Zapier is een hulpmiddel waarmee je workflows kunt automatiseren door de apps die je team dagelijks gebruikt te verbinden. Het fundamentele concept van Zapier is geautomatiseerde workflows, genaamd Zaps, en omvatten triggers en acties.

You can learn more about how Zapier works [here](https://zapier.com/how-it-works).

## Installatie

### Stap 1: Installeer Zapier-pakketten

```bash
cd packages/twenty-zapier

yarn
```

### Stap 2: Login met de CLI

Gebruik je Zapier-inloggegevens om in te loggen via de CLI:

```bash
zapier login
```

### Stap 3: Stel omgevingsvariabelen in

Voer vanuit de map `packages/twenty-zapier` het volgende uit:

```bash
cp .env.example .env
```

Voer de applicatie lokaal uit, ga naar [http://localhost:3000/settings/api-webhooks](http://localhost:3000/settings/api-webhooks), en genereer een API-sleutel.

Vervang de waarde **YOUR_API_KEY** in het `.env`-bestand door de API-sleutel die je zojuist hebt gegenereerd.

## Ontwikkeling

<Warning>

Zorg ervoor dat je `yarn build` uitvoert vóór enig `zapier`-commando.

</Warning>

### Test

```bash
yarn test
```

### Lint

```bash
yarn format
```

### Bekijk en compileer terwijl je code aanpast

```bash
yarn watch
```

### Valideer je Zapier-app

```bash
yarn validate
```

### Implementeer je Zapier-app

```bash
yarn deploy
```

### Toon alle Zapier CLI-commando's

```bash
zapier
```
