## Bugs Rapporteren

Om een bug te melden, [maak een issue op GitHub aan](https://github.com/twentyhq/twenty/issues/new).

Je kunt ook om hulp vragen op [Discord](https://discord.gg/cx5n4Jzs57).

## Functie Verzoeken

Als je niet zeker weet of het een bug is, en je denkt dat het meer op een functieverzoek lijkt, moet je waarschijnlijk [een discussie openen in plaats daarvan](https://github.com/twentyhq/twenty/discussions/new).
