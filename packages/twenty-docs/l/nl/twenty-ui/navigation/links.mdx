## Contact Link

Een gestileerd linkcomponent voor het weergeven van contactinformatie.

<Tabs>
<Tab title="Usage">

```jsx
import { BrowserRouter as Router } from 'react-router-dom';

import { ContactLink } from 'twenty-ui/navigation';

export const MyComponent = () => {
  const handleLinkClick = (event) => {
    console.log('Contact link clicked!', event);
  };

  return (
    <Router>
      <ContactLink
        className
        href="mailto:example@example.com"
        onClick={handleLinkClick}
      >
        example@example.com
      </ContactLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Props     | Soort             | Beschrijving                                                            |
| --------- | ----------------- | ----------------------------------------------------------------------- |
| className | tekenreeks        | Optionele naam voor extra styling                                       |
| href      | tekenreeks        | De doel-URL of pad voor de link                                         |
| onClick   | functie           | Callback-functie die wordt geactiveerd wanneer de link wordt aangeklikt |
| children  | `React.ReactNode` | De inhoud die binnen de link weergegeven moet worden                    |

</Tab>

</Tabs>

## Sociale link

Gestileerde sociale links, met ondersteuning voor verschillende soorten sociale links, zoals URL's, LinkedIn en X (of Twitter).

<Tabs>
<Tab title="Usage">

```jsx
import { SocialLink } from "twenty-ui/navigation";
import { BrowserRouter as Router } from "react-router-dom";

export const MyComponent = () => {
  return (
    <Router>
      <SocialLink
        type="twitter"
        href="https://twitter.com/twentycrm"
      ></SocialLink>
    </Router>
  );
};
```

</Tab>
<Tab title="Props">

| Props    | Soort             | Beschrijving                                                                                            |
| -------- | ----------------- | ------------------------------------------------------------------------------------------------------- |
| href     | tekenreeks        | De doel-URL of pad voor de link                                                                         |
| children | `React.ReactNode` | De inhoud die binnen de link weergegeven moet worden                                                    |
| soort    | tekenreeks        | Het type sociale links. Opties omvatten: `url`, `LinkedIn` en `Twitter` |
| onClick  | functie           | Callback-functie die wordt geactiveerd wanneer de link wordt aangeklikt                                 |

</Tab>
</Tabs>
