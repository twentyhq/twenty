Twenty se propojí s vašimi stávajícími nástroji, takže váš CRM zůstane aktuální bez ručního zadávání dat.

## Synchronizace e-mailu

Připojte svou poštovní schránku **Google Workspace** nebo **Microsoft 365**. Po připojení:

* E-maily se automaticky propojí s odpovídajícími záznamy Společnosti a Osoby
* Celé e-mailové vlákno je viditelné na časové ose každého záznamu
* E-maily můžete odesílat přímo z Twenty
* Je podporováno více poštovních schránek na uživatele

Máte pod kontrolou, co se importuje — filtrujte podle časového období nebo odesílatele, abyste se vyhnuli načítání nerelevantních e-mailů.

## Synchronizace kalendáře

Události v kalendáři se z vašeho připojeného účtu synchronizují automaticky. Události se zobrazují na relevantních záznamech v CRM, což vám poskytne kompletní přehled o vašich interakcích s každým kontaktem nebo společností.

## Integrace

Kromě e-mailu a kalendáře se Twenty propojuje s externími nástroji prostřednictvím:

| Metoda                           | Případ použití                                                                   |
| -------------------------------- | -------------------------------------------------------------------------------- |
| **API**                          | Vytvářejte vlastní integrace pomocí GraphQL nebo REST API                        |
| **Webhooky**                     | Odesílejte notifikace v reálném čase do externích systémů, když se záznamy změní |
| **Zapier**                       | Propojte se s více než 5 000 aplikacemi bez psaní kódu                           |
| **HTTP akce pracovního postupu** | V rámci automatizovaného pracovního postupu volejte libovolné externí API        |

## Vlastní aplikace

Vývojáři mohou na Twenty vytvářet plnohodnotné aplikace — přidávat vlastní uživatelské rozhraní, serverovou logiku a hluboké integrace. Aplikace mohou být publikovány pro komunitu nebo ponechány jako soukromé.
