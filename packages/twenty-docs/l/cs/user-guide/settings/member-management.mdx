## Pozvat nové členy

### Použití odkazu k pozvání

1. Přejděte na **Nastavení → Členové**
2. Copy the workspace invite link
3. Sdílejte odkaz s novými členy týmu
4. Získají přístup, jakmile se zaregistrují

### Přímá emailová pozvánka

1. Přejděte na **Nastavení → Členové**
2. Zadejte emailovou adresu osoby
3. Klikněte **Pozvat**
4. Obdrží emailovou pozvánku

## Odstranit členy

### Smazat člena

1. Přejděte na **Nastavení → Členové**
2. Najděte člena, kterého chcete odstranit
3. Klikněte na tlačítko odstranit/smazat vedle jeho jména
4. Potvrďte odstranění

**Poznámka**: Odstranění členové okamžitě ztrácí přístup, ale mohou být později znovu pozváni.
