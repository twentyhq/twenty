Přizpůsobte si prostředí Twenty v části **Nastavení → Prostředí**.

## Vzhled

Vyberte vizuální motiv:

* **Světlý**: Čisté, jasné rozhraní
* **Tmavý**: Šetrnější na oči v podmínkách slabého osvětlení
* **Systémové nastavení**: Automaticky se přizpůsobí motivu vašeho zařízení

## Jazyk

Vyberte svůj preferovaný jazyk pro rozhraní Twenty z rozbalovací nabídky.

## Formáty

Nastavit datum, čas, číslo, časovou zónu a počáteční den kalendáře.

| Nastavení                   | Popis                                                          |
| --------------------------- | -------------------------------------------------------------- |
| **Časové pásmo**            | Vaše místní časové pásmo pro přesná časová razítka a plánování |
| **Formát data**             | Jak se zobrazují data (např. 12. 12. 2025)                     |
| **Formát času**             | 12hodinový (7:22 PM) nebo 24hodinový formát                    |
| **Formát čísel**            | Oddělovače desetinných míst a tisíců (např. 1 234,56)          |
| **Počáteční den kalendáře** | První den v týdnu (neděle nebo pondělí)                        |

U každého nastavení můžete zvolit **Systémové nastavení**, které se automaticky přizpůsobí předvolbám vašeho zařízení, nebo vybrat konkrétní formát.

## Jak aktualizovat

1. Přejděte do **Nastavení → Prostředí**
2. Upravte své preference v každé sekci
3. Změny se ukládají automaticky
