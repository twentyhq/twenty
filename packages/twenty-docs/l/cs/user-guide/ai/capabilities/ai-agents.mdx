## Přehled

Integrujte možnosti AI přímo do svých automatizačních pracovních postupů pro inteligentní zpracování dat a rozhodování.

## Možnosti

| Funkce              | Popis                                                           |
| ------------------- | --------------------------------------------------------------- |
| **Akce AI**         | Přidejte do libovolného pracovního postupu kroky využívající AI |
| **Obohacování dat** | Automaticky obohacujte záznamy externími daty                   |
| **Klasifikace**     | Kategorizujte záznamy na základě analýzy obsahu                 |
| **Shrnutí**         | Vytvářejte souhrny z textových polí                             |
| **Vlastní prompty** | Přesně určete, jak AI zpracovává vaše data                      |

## Případy použití

* **Skórování leadů**: Automaticky ohodnocujte a upřednostňujte příchozí leady
* **Čištění dat**: Standardizujte názvy společností a kontaktní informace
* **Návrhy e-mailů**: Generujte návrhy následných e-mailů na základě poznámek ze schůzek
* **Směrování záznamů**: Přiřazujte záznamy správnému členovi týmu na základě obsahu
