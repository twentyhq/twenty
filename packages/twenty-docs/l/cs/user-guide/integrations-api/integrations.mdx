## Aktuální možnosti integrace

### Nativní integrace

Twenty v současnosti nabízí nativní integraci s:

- **E-mail & Kalendář**: Připojte Gmail, Outlook nebo poskytovatele SMTP/CalDAV
- **API Přístup**: Použijte naše REST a GraphQL API k vytváření vlastních integrací

Pro nastavení e-mailu a kalendáře navštivte [Nastavení E-mailu & Kalendáře](/l/cs/user-guide/settings/email-calendar-setup).

### Workflows (Recommended)

Hlavní způsob, jak propojit Twenty s jinými nástroji, je skrze naši funkcionalitu **Pracovní postupy** v aplikaci:

- **HTTP Nodes**: Make API calls to external services
- **Kódový Uzel**: Pište vlastní logiku pro komplexní integrace
- **Webhook Triggers**: Receive data from external systems
- **Spouštěč Aktualizací Pole**: Automatizujte akce na základě změn v CRM

Zjistěte více v naší [sekci Pracovních postupů](/l/cs/user-guide/workflows/getting-started-workflows).

### Integrace Zapier (Legacy)

Udržujeme integraci Zapier pro uživatele, kteří preferují automatizaci bez kódu:

1. Navštivte [Twenty na Zapier](https://zapier.com/apps/twenty/integrations)
2. Create a new Zap with Twenty as trigger or action
3. Vygenerujte API klíč v Nastavení → API & Webhooky
4. Propojte své pracovní prostředí Twenty se Zapier

## Budoucí vize: Konektory vytvářené komunitou (2026)

### Platforma rozšiřitelnosti

Vytváříme platformu rozšiřitelnosti, která umožní vývojářům vytvářet aplikace jako kód. To umožní:

- **Vlastní Konektory**: Vytvářejte integrace s vaším oblíbeným softwarem
- **Příspěvky Komunity**: Sdílejte a objevujte konektory vytvořené ostatními
- **Flexibilní Architektura**: Rozšiřujte možnosti Twenty nad rámec základních funkcí

### AI-Asistovaná tvorba workflowů

Coming in 2026, AI assistance will help users:

- **Automaticky generovat workflowy**: Popište své potřeby integrace prostým jazykem
- **Chytré návrhy**: Získejte doporučení pro propojení vašich specifických nástrojů
- **Knihovna šablon**: Přístup k předem vytvořeným workflowům pro běžné případy použití

Budoucnost integrací s Twenty je poháněna komunitou, asistovaná AI a nekonečně rozšiřitelná.
