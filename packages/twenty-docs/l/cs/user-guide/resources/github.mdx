## O aplikaci

Repozitář Twenty na GitHubu obsahuje širokou škálu zdrojů jako zdrojový kód, dokumentaci, diskuse a sledování problémů. Zde budete mít přístup ke kompletnímu kódu, který stojí za Twenty.

[Navštivte Twenty na GitHubu](https://github.com/twentyhq/twenty)

## Přispívání

Přispívání do projektu Twenty na platformě GitHub je odměňující způsob, jak pomoci zlepšit software, který používáte. Ať už opravujete chyby, navrhujete funkce nebo zlepšujete dokumentaci, vaše příspěvky jsou vítány.

### Nahlášení problémů

Narazili jste na problém? [Vytvořte issue](https://github.com/twentyhq/twenty/issues/new) na GitHubu a poskytněte co nejvíce podrobností.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggesting Features

Jaká vylepšení byste chtěli v Twenty vidět? Bez ohledu na to, jaké máte technické znalosti, můžete se zapojit do [diskuze zde](https://github.com/twentyhq/twenty/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Kódování funkce

Začněte svou cestu hledáním úkolů vhodných pro začátečníky:

1. Přejděte na záložku **[Issues](https://github.com/twentyhq/twenty/issues)** v repozitáři Twenty.
2. Filter by **[Good First Issue](https://github.com/twentyhq/twenty/labels/good%20first%20issue)** label to find tasks suited for newcomers.
3. Vyberte si issue, vytvořte fork repozitáře a začněte přispívat.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Ujistěte se, že jste přiřazeni k issue, abyste se vyhnuli překrývání práce s jinými přispěvateli.

### Kodex chování

Při procesu přispívání nezapomeňte dodržovat [Kodex chování](https://github.com/twentyhq/twenty/blob/main/.github/CODE_OF_CONDUCT.md) platformy Twenty.
