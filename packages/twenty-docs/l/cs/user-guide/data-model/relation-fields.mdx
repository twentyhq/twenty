## What are Relation Fields?

Relační pole spojují záznamy z jednoho objektu se záznamy v jiném objektu. Například:

- **Lidé** → **Společnosti** (každá osoba pracuje pro společnost)
- **Příležitosti** → **Lidé** (každý obchod má kontaktní osobu)
- **Úkoly** → **Příležitosti** (každý úkol je spojen s konkrétním obchodem)

## Creating Relation Fields

### 1. Přidat pole

Přejděte na **Nastavení → Datový Model → [Váš Objekt]** a klikněte na **Přidat Pole**.

### 2. Choose Relation Type

- **Typ pole**: Vyberte "Relace"
- **Cílový Objekt**: Zvolte, který objekt chcete připojit
- **Vztah**: Momentálně podporuje pouze vztahy **1-k-mnoha**. Ujistěte se, že vytváříte vztah ve správném směru.

### 3. Konfigurace názvů polí

Budete muset určit názvy pro obě strany vztahu:

- **Název zdrojového pole**: Jak se pole zobrazí na vašem aktuálním objektu
- **Target field name**: How the reverse field appears on the target object

<Warning>
Názvy polí nelze upravovat, jakmile je relace uložena, protože by to ovlivnilo strukturu API. Vyberte opatrně.
</Warning>

## Vztahování se k členům týmu

You can create relations to any object, including **Workspace Members** (your Twenty team users). To je užitečné pro vytváření polí vlastnictví:

- **Vlastník účtu**: Spojte společnost s členem týmu, který ji spravuje
- **Vlastník obchodu**: Přiřaďte příležitost konkrétnímu obchodníkovi

When you create a relation to`Workspace Members`, you'll see your team members' names in dropdown selections, making it easy to assign ownership and responsibilities.

## Osvedčené postupy

- **Plánujte své vztahy** před jejich vytvořením
- **Používejte jasné a popisné názvy** pro obě jména polí
- **Otestujte vztahy** s ukázkovými daty před úplnou implementací
