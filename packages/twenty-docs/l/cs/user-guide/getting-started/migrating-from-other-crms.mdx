## Než začnete

### 1. Proveďte audit vašich aktuálních dat

- **Vyberte několik objektů a polí pro migraci**: tato migrace je příležitostí pro nový začátek
- **Exportujte tato data** z vašeho současného CRM
- **Odstraňte duplicitní** a zastaralé záznamy
- **Seznam aktivních pracovních postupů** a automatizací

### 2. Vytvořte svůj nový datový model

Postupujte podle našeho [průvodce datovým modelem](/l/cs/user-guide/data-model/customize-your-data-model) pro:

- **Navrhněte datový model**, který potřebujete
- **Mapujte existující pole** do standardních objektů / polí Twenty
- **Identifikujte vlastní objekty / pole**, které budete potřebovat
- **Vytvořte je** v Nastavení → Datový model

## Proces migrace

### 1. Importujte svá data

**Recommended order:**

1. **Nejdříve společnosti** (jako základní záznamy)
2. **People** second (linked to companies)
3. **Opportunities** third (linked to people/companies)

Použijte CSV import prostřednictvím menu příkazů  `Cmd + K` (Mac) nebo `Ctrl + K` (Windows). Pro podrobné pokyny si přečtěte náš [průvodce importem dat](/l/cs/user-guide/getting-started/import-export-data).

### 2. Znovuvytvoření pracovních postupů

- **Začněte jednoduše** - nejdříve znovuvytvořte nejdůležitější automatizace
- **Použijte stavitele pracovních postupů Twenty** k nahrazení stávajících automatizací

## Běžné výzvy

### Problémy s formáty dat

- **Emailové adresy** - odstraňte duplicitní (požadavek objektu Lidé)
- **Doména** - odstraňte duplicitní (požadavek objektu Společnosti)

<Warning>
Please note that domain URLs created by the synchronization with your mailbox and calendar have the following format ```https://domain.com```
</Warning>

- **Formáty dat** - zajistěte konzistentní formátování (YYYY-MM-DD) nebo upravte tento formát v Nastavení → Zážitek
- **Telefonní čísla** - používejte mezinárodní formát (+1234567890)

### Parování vztahů

Pro import vztahů mezi záznamy pomocí funkce importu csv můžete použít následující pole

- **Použijte Twenty ID** pro složité vztahy
- **Use email addresses** to link People records
- **Použijte doménové názvy** pro propojení záznamů společnosti
- **Použijte jakékoli jiné pole, které nastavíte jako jedinečné**, které lze provést v sekci Datový model.
  Přečtěte si náš [průvodce importem-exportem dat](/l/cs/user-guide/getting-started/import-export-data) pro detailní pokyny k vytváření vztahů během CSV importu.

## Migrace z vlastních serverů do cloudu

If you're moving from Twenty self-hosted to Twenty Cloud:

1. **Exportujte svá data** z vaší instance hostované na vlastním serveru
2. **Postupujte podle výše uvedeného standardního procesu migrace**
3. **Můžeme poskytnout pomoc při migraci**, kontaktujte náš tým
