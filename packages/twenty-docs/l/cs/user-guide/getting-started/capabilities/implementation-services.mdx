## Úvodní balíčky

Získejte pomoc od našeho hlavního týmu s nastavením vašeho pracovního prostoru Twenty díky našim 4hodinovým úvodním balíčkům:

* **Návrh datového modelu**: Navrhněte a vytvořte vlastní datový model s objekty, poli a vztahy
* **Migrace dat**: Migrujte svá aktuální data z vašeho současného CRM do Twenty
* **Vytváření pracovních toků**: Vytvářejte vlastní pracovní toky na podporu vašich podnikových procesů
