## Filtrování dat

Filtry vám pomohou zaměřit se na konkrétní záznamy tím, že zobrazí pouze ty, které odpovídají vašim kritériím.

### Přidání filtru

1. Klikněte na tlačítko **Filtr** na panelu nástrojů
2. Vyberte pole, podle kterého filtrovat
3. Zvolte operátor (je rovno, obsahuje apod.)
4. Zadejte hodnotu filtru
5. Klikněte na **Použít**

### Operátory filtru

| Typ pole            | Dostupné operátory                                  |
| ------------------- | --------------------------------------------------- |
| Text                | Je rovno, Obsahuje, Začíná na, Končí na, Je prázdné |
| Číslo               | Je rovno, Větší než, Menší než, Mezi, Je prázdné    |
| Datum               | Je rovno, Před, Po, Mezi, Je prázdné                |
| Vybrat              | Je rovno, Je jedním z, Je prázdné                   |
| Zaškrtávací políčko | Je pravdivé, Je nepravdivé                          |
| Vztah               | Je rovno, Je prázdné                                |

### Více filtrů

Kombinujte více filtrů pro zúžení výsledků:

* Všechny filtry se aplikují s logickým operátorem AND
* Každý další filtr dále omezuje výsledky

### Odstranění filtrů

* Klikněte na **X** na jednotlivých štítcích filtrů
* Klikněte na **Vymazat vše** pro odstranění všech filtrů

## Řazení dat

Řazení určuje pořadí, ve kterém se záznamy zobrazují.

### Přidání řazení

1. Klikněte na tlačítko **Řazení** na panelu nástrojů
2. Vyberte pole, podle kterého seřadit
3. Zvolte vzestupné (A-Z, 0-9) nebo sestupné (Z-A, 9-0)
4. Klikněte na **Použít**

### Více řazení

Přidejte více úrovní řazení:

* První řazení je primární
* Další úrovně řazení se uplatní v rámci skupin stejných hodnot

### Rychlé řazení sloupců

Kliknutím na libovolné záhlaví sloupce seřadíte:

* První kliknutí: vzestupně
* Druhé kliknutí: sestupně
* Třetí kliknutí: zruší řazení

## Uložení nastavení filtrů a řazení

Filtry a řazení se ukládají se zobrazením:

1. Nakonfigurujte své filtry a řazení
2. Klikněte na **Uložit** pro aktualizaci aktuálního zobrazení
3. Nebo klikněte na **Uložit jako nové zobrazení** pro vytvoření varianty
