## Předpoklady

Váš objekt Úkoly potřebuje pole **Termín** (typ Datum nebo Datum a čas).

## Postup

1. Přejděte na **Úkoly**
2. Klikněte na rozbalovací nabídku zobrazení → **+ Přidat zobrazení**
3. Pojmenujte své zobrazení (např. "Kalendář úkolů")
4. Klikněte na **Vytvořit**
5. Klikněte na **Možnosti** a jako rozvržení vyberte **Kalendář**
6. Vyberte jako datumové pole **Termín**
7. Klikněte na **Uložit**

## Konfigurace kalendáře

### Zobrazení polí v událostech

1. Klikněte na **Možnosti → Pole**
2. Kliknutím na **ikonu oka** zobrazíte/skryjete pole
3. Přetažením změňte pořadí

Doporučená pole k zobrazení:

* **Název** — název úkolu
* **Přiřazený** — odpovědná osoba
* **Stav** — aktuální průběh

### Filtrování kalendáře

Vytvořte cílená zobrazení:

* **Moje úkoly**: Filtrujte podle Přiřazený = Já
* **Tento týden**: Filtrujte podle Termín = Tento týden
* **Po termínu**: Filtrujte podle termínu < Dnes, Stav ≠ Dokončeno

## Další případy použití kalendáře

| Objekt           | Pole datumu    | Účel                            |
| ---------------- | -------------- | ------------------------------- |
| Příležitosti     | Datum uzavření | Sledování očekávaných uzavření  |
| Vlastní události | Datum události | Plánování aktivit               |
| Projekty         | Termín         | Sledování harmonogramů projektů |

## Tipy

* **Kontrolujte každý týden**: Každý týden začněte kontrolou zobrazení kalendáře
* **Kombinujte s tabulkovým zobrazením**: Kalendář použijte pro přehled, tabulku pro podrobnosti
* **Nastavte viditelnost**: Osobní kalendáře úkolů ponechte jako Unlisted
