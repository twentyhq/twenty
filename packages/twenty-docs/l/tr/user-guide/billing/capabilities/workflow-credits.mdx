## Genel Bakış

Krediler, Twenty'deki iş akışı otomasyonlarınıza güç verir. Her iş akışı eylemi, karmaşıklığına bağlı olarak kredi tüketir.

## Kredi Tahsisi

Krediler planınıza değil, faturalandırma döngünüze göre belirlenir:

| Faturalandırma Döngüsü | Krediler      |
| ---------------------- | ------------- |
| Aylık                  | 5 milyon/ay   |
| Yıllık                 | 50 milyon/yıl |

<Note>
  Aylık 5 milyon kredi, maliyetleri dert etmeden otomasyonları çalıştırabilmenizi sağlamak üzere tasarlanmıştır. Standart eylemler kullanan çoğu iş akışı için bu fazlasıyla yeterlidir. Yalnızca gelişmiş kod düğümlerini veya yapay zeka destekli özellikleri çalıştırırken ek krediye ihtiyaç duyarsınız.
</Note>

## Kredi Tüketimi

Farklı eylemler farklı miktarlarda kredi tüketir:

| Eylem Türü                                                  | Kredi Kullanımı  |
| ----------------------------------------------------------- | ---------------- |
| **Temel işlemler** (arama, güncelleme, kayıt oluşturma)     | Minimum          |
| **Karmaşık işlemler** (kod düğümleri, harici API çağrıları) | Daha fazla kredi |
| **AI istemleri** (yakında geliyor)                          | Kullanıma bağlı  |

Krediler, iş akışları yürütülürken gerçek zamanlı olarak düşer.

## Kullanımı İzleme

Kredi tüketiminizi takip edin:

1. **Ayarlar → Faturalama** bölümüne gidin
2. Mevcut kullanımınızı ve kalan kredilerinizi görüntüleyin
3. Gerekirse ek kredileri planlamak için eğilimleri izleyin

## Ek Kredi Satın Alma

Daha fazla krediye mi ihtiyacınız var?

1. **Ayarlar → Faturalama** bölümüne gidin
2. Ek kredi paketleri satın alma seçeneğine tıklayın
3. İhtiyacınız olan miktarı seçin
