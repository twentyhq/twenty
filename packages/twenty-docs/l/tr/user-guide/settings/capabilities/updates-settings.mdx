## Güncelleme Ayarları Hakkında

Güncellemeler bölümü, genel kullanıma sunulmadan önce Erken Erişim özelliklerini etkinleştirmenizi sağlar.

## Erken Erişim Özellikleri

Erken Erişim özellikleri, genel kullanıma sunulmadan önce test edilen özelliklerdir.

**Şu anda Erken Erişimde:**

* **Gösterge Panelleri** — Görsel raporlama ve analiz

### Erken Erişim Özellikleri Nasıl Etkinleştirilir

1. **Ayarlar → Güncellemeler** bölümüne gidin
2. **Erken Erişim** bölümüne gidin
3. Etkinleştirmek istediğiniz özelliği bulun
4. Açık konuma getirin
5. Özellik hemen kullanılabilir hale gelir

<Note>
  Erken Erişim özellikleri, kullanıcı geri bildirimlerine bağlı olarak değişebilir. Geri bildiriminiz, Twenty'nin geliştirilmesine yardımcı olur!
</Note>

## Özellik Geri Bildirimi

Geri bildiriminiz, Twenty'nin geliştirilmesine yardımcı olur:

* Erken Erişim özellikleriyle ilgili sorunları bildirin
* Yeni özellikleri nasıl kullandığınızı paylaşın
* Topluluk Discord'u aracılığıyla iyileştirmeler önerin
