## Kişisel Bilgiler

### İsim ve E-posta

* **Görünen Ad**: Adınızın diğer üyeler tarafından nasıl görüneceğini güncelleyin
* **E-posta Adresi**: Giriş e-postanızı değiştirin (doğrulama gerektirir)
* **Profil Resmi**: Özel bir avatar yükleyin veya baş harflerinizi kullanın

## Güvenlik Ayarları

### İki Faktörlü Doğrulama (2FA)

2FA etkinleştirerek hesabınıza ekstra güvenlik katmanı ekleyin:

1. **Ayarlar → Profil Ayarları**'na gidin
2. **2FA Etkinleştir**'e tıklayın
3. QR kodunu yetkilendirme uygulamanızla tarayın
4. Doğrulama kodunu girerek onaylayın

### Şifre Yönetimi

* **Şifre Değiştir**: Mevcut şifrenizi güncelleyin
* **Şifre Gereksinimleri**: En az 8 karakter uzunluğunda olmalıdır

## Profil Yönetimi

### Hesabı Sil

<Warning>
  Hesabınızı silmek, tüm çalışma alanlarınıza olan erişiminizi kalıcı olarak kaldıracaktır. Bu işlem geri alınamaz, üyesi olduğunuz tüm çalışma alanlarına erişiminizi kaybedeceksiniz, yalnızca belirli ekiplerden çıkmak istiyorsanız tek tek çalışma alanlarından çıkmayı düşünmelisiniz.
</Warning>

Hesabınızı silmek için:

1. **Ayarlar → Profil Ayarları**'na gidin
2. **Tehlike Bölgesi**'ne kadar kaydırın
3. **Hesabı Sil**'e tıklayın
4. E-posta adresinizi yazarak onaylayın
