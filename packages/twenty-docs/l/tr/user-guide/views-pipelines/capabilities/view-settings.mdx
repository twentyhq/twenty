## Görünüm Gezgini'nden

Özel görünümlerinizi kimin görebileceğini kontrol edin.

### Görünürlük Seçenekleri

| Ayar           | Kimler Görebilir     |
| -------------- | -------------------- |
| **İş Alanı**   | Tüm iş alanı üyeleri |
| **Liste dışı** | Yalnızca siz         |

### Görünürlüğü Değiştirme

1. Görünümü açın
2. **Seçenekler → Görünürlük**'e tıklayın
3. **İş Alanı** veya **Liste dışı** seçeneğini seçin

<Note>
  Varsayılan "All [Object Name]" görünümlerinin görünürlüğü değiştirilemez.
</Note>

## Bir Görünümü Yeniden Adlandırın

1. Görünüm açılır menüsünü açın
2. Görünümün yanındaki **⋮** menüsüne tıklayın
3. **Düzenle**'yi seçin
4. Yeni adı girin

## Görünüm Simgesini Değiştirin

1. Görünüm açılır menüsünü açın
2. Görünümün yanındaki **⋮** menüsüne tıklayın
3. **Düzenle**'yi seçin
4. Değiştirmek için simgeye tıklayın

## Görünümleri Yeniden Sıralayın

Görünümlerin açılır menüde göründüğü sırayı değiştirin:

1. Görünüm açılır menüsünü açın
2. Görünümleri tutamaklarından sürükleyin
3. İstediğiniz konuma bırakın
4. Sıralama otomatik olarak kaydedilir

## Favoriler

Sık kullanılan görünümleri hızlı erişim için sabitleyin:

1. Görünüm açılır menüsünü açın
2. Bir görünümün yanındaki **⋮** menüsüne tıklayın
3. **Sık Kullanılanlara Ekle**'yi seçin

Sık kullanılanlara eklenen görünümler, kolay erişim için ayrılmış bir bölümde görünür.

## Bir Görünümü Silin

1. Görünüm açılır menüsünü açın
2. Görünümün yanındaki **⋮** menüsüne tıklayın
3. **Sil**'i seçin
4. Silmeyi onaylayın

<Warning>
  Silinen görünümler geri getirilemez.
</Warning>
