## Güncel Entegrasyon Seçenekleri

### Yerel Entegrasyonlar

Yirmi şu anda yerel entegrasyon sunuyor:

- **E-posta ve Takvim**: Gmail, Outlook veya SMTP/CalDAV sağlayıcılarıyla bağlantı kurun
- **API Erişimi**: Özel entegrasyonlar oluşturmak için REST ve GraphQL API'lerimizi kullanın

E-posta ve takvim ayarları için [E-posta ve Takvim Kurulumu](/l/tr/user-guide/settings/email-calendar-setup) adresini ziyaret edin.

### İş Akışları (Tavsiye Edilir)

Yirmiyi diğer araçlarla bağlamanın birincil yolu uygulama içi **İş Akışları** özelliğimizdir:

- **HTTP Düğümleri**: Harici hizmetlere API çağrıları yapın
- **Kod Düğümleri**: Karmaşık entegrasyonlar için özel mantık yazın
- **Webhook Tetikleyicileri**: Harici sistemlerden veri alın
- **Alan Güncelleme Tetikleyicileri**: CRM değişikliklerine dayalı eylemleri otomatikleştirin

Daha fazla bilgi için [İş Akışları bölümü](/l/tr/user-guide/workflows/getting-started-workflows) okuyun.

### Zapier Entegrasyonu (Eski)

Kod gerektirmeyen otomasyonu tercih eden kullanıcılar için bir Zapier entegrasyonu sağlıyoruz:

1. Visit [Twenty on Zapier](https://zapier.com/apps/twenty/integrations)
2. Yirmiyi tetikleyici veya eylem olarak yeni bir Zap oluşturun
3. Generate an API key in Settings → API & Webhooks
4. Twenty çalışma alanınızı Zapier'e bağlayın

## Gelecek Vizyonu: Topluluk Tarafından Geliştirilen Bağlayıcılar (2026)

### Genişletilebilirlik Platformu

Geliştiricilerin uygulamaları kod olarak oluşturmalarına olanak tanıyan bir genişletilebilirlik platformu geliştiriyoruz. Bu, şunlara olanak sağlayacaktır:

- **Özel Bağlayıcılar**: Tercih ettiğiniz yazılımla entegrasyonlar oluşturun
- **Topluluk Katkıları**: Başkaları tarafından oluşturulan bağlayıcıları paylaşın ve keşfedin
- **Esnek Mimari**: Twenty'nin özelliklerini temel özelliklerin ötesine taşıyın

### AI Destekli İş Akışı Oluşturma

2026 yılında, AI yardımı kullanıcıların şunları yapmasına olanak tanıyacaktır:

- **Otomatik İş Akışı Üretme**: Entegrasyon ihtiyaçlarınızı basit bir dille anlatın
- **Akıllı Öneriler**: Özel araçlarınızı bağlarken öneriler alın
- **Şablon Kütüphanesi**: Yaygın kullanım durumları için önceden oluşturulmuş iş akışlarına erişin

Yirmi entegrasyonlarının geleceği topluluk odaklı, AI destekli ve sonsuz genişletilebilir olacak.
