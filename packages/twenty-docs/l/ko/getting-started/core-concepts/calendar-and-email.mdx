Twenty는 기존 도구들과 연동되어, 수동으로 데이터를 입력하지 않아도 CRM이 최신 상태로 유지되도록 해 줍니다.

## 이메일 동기화

**Google Workspace** 또는 **Microsoft 365** 메일박스를 연결하세요. 연결이 완료되면:

* 이메일이 자동으로 해당 회사 및 사람 레코드에 연결됩니다.
* 각 레코드의 타임라인에서 전체 이메일 스레드를 확인할 수 있습니다.
* Twenty에서 직접 이메일을 보낼 수 있습니다.
* 사용자당 여러 개의 메일박스를 지원합니다.

가져올 내용을 직접 제어할 수 있으며, 날짜 범위나 보낸 사람으로 필터링해 관련 없는 이메일이 들어오는 것을 막을 수 있습니다.

## 캘린더 동기화

연결된 계정에서 캘린더 이벤트가 자동으로 동기화됩니다. 이벤트가 관련 CRM 레코드에 표시되어, 각 연락처나 회사와의 상호작용을 전체적으로 파악할 수 있습니다.

## 통합

이메일과 캘린더를 넘어, Twenty는 다음을 통해 외부 도구와 연동됩니다.

| 방법                        | 사용 사례                                   |
| ------------------------- | --------------------------------------- |
| **API**                   | GraphQL 또는 REST API를 사용해 맞춤형 통합을 구축하세요. |
| **웹훅**                    | 레코드가 변경될 때 외부 시스템으로 실시간 알림을 전송합니다.      |
| **Zapier**                | 코드 작성 없이 5,000개 이상의 앱과 연동하세요.           |
| **Workflow HTTP actions** | 자동화된 워크플로우의 일부로 모든 외부 API를 호출할 수 있습니다.  |
