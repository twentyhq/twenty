Twenty는 AI를 단순한 보여주기식 기능이 아닌, 귀하의 데이터 모델과 권한 시스템 안에서 작동하는 도구로서 CRM에 직접 통합합니다.

## AI 챗봇

자연어로 데이터에 대해 질문해 보세요. AI 챗봇은 CRM 레코드를 조회하고 정보를 요약하여, 복잡한 필터를 만들지 않고도 원하는 내용을 찾을 수 있도록 도와줍니다.

## AI 에이전트

AI 에이전트는 단순한 채팅을 넘어, 여러 단계를 포함한 작업을 자율적으로 실행할 수 있습니다:

* 외부 소스의 데이터로 레코드를 보강
* 후속 이메일을 작성하고 발송
* 파이프라인 상태를 분석하고 위험 거래를 표시
* 수신 데이터를 처리하고 적절한 팀으로 라우팅

에이전트는 기존 워크플로 내에서 작동하므로, AI를 수동 승인, 조건부 로직, 외부 API 호출과 결합할 수 있습니다.

## 권한 및 보안

Twenty의 AI는 귀하의 권한 모델을 준수합니다. 에이전트는 사용자(또는 역할)에 조회 권한이 있는 오브젝트와 필드에만 접근할 수 있습니다. 민감한 데이터는 AI가 개입하더라도 계속 보호됩니다.
