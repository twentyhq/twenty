Twenty는 완전한 기능을 갖춘 CRM 플랫폼입니다. 이를 통해 무엇을 구축할 수 있는지 살펴보세요.

<CardGroup cols={2}>
  <Card href="/l/ko/user-guide/data-model/overview" img="/images/user-guide/halftone/data-model.png">
    <CardTitle>Custom Data Model</CardTitle>
    비즈니스에 필요한 정확한 데이터 구조를 정의하세요.
    <br />커스텀 오브젝트를 만들고, 20개 이상의 필드 유형으로 커스텀 필드를 추가하며, 어떤 오브젝트끼리든 관계를 구성하세요.
  </Card>

  <Card href="/l/ko/user-guide/views-pipelines/overview" img="/images/user-guide/halftone/layout.png">
    <CardTitle>Views & Pipelines</CardTitle>
    테이블 뷰, 칸반 보드, 캘린더 뷰 간에 전환하세요.
    <br />AND/OR 로직으로 필터링하고, 여러 필드 기준으로 정렬하며, 레코드를 그룹화하고, 커스텀 뷰를 저장하세요.
  </Card>

  <Card href="/l/ko/user-guide/workflows/overview" img="/images/user-guide/halftone/workflows.png">
    <CardTitle>Workflows & Automation</CardTitle>
    코드를 작성하지 않고도 모든 비즈니스 프로세스를 자동화하세요.
    <br />레코드 변경, 일정, 수동 작업 또는 수신 웹훅을 트리거로 워크플로우를 실행하세요.
  </Card>

  <Card href="/l/ko/user-guide/calendar-emails/overview" img="/images/user-guide/halftone/calendar-emails.png">
    <CardTitle>Calendar & Email Sync</CardTitle>
    Google Workspace 또는 Microsoft 365 계정을 연결하세요.
    <br />이메일과 캘린더 이벤트가 관련 CRM 레코드에 자동으로 표시됩니다.
  </Card>

  <Card href="/l/ko/user-guide/ai/overview" img="/images/user-guide/halftone/ai.png">
    <CardTitle>AI</CardTitle>
    CRM 내에서 자율적으로 작동하는 AI 에이전트가 질문에 답변하고, 레코드를 보강하며, 권한 모델 내에서 다단계 작업을 실행합니다.
  </Card>

  <Card href="/l/ko/user-guide/dashboards/overview" img="/images/user-guide/halftone/dashboards.png">
    <CardTitle>Dashboards & Reporting</CardTitle>
    실시간 위젯으로 커스텀 대시보드를 구축하세요.
    <br />구성 가능한 차트와 필터로 파이프라인 지표, 팀 성과, 비즈니스 KPI를 추적하세요.
  </Card>

  <Card href="/l/ko/user-guide/permissions-access/overview" img="/images/user-guide/halftone/permissions.png">
    <CardTitle>Permissions & Access Control</CardTitle>
    오브젝트, 필드, 개별 레코드 등 모든 수준에서 역할 기반 액세스 제어를 구성하세요.
    <br />SAML 또는 OIDC로 SSO를 구성하세요. 감사 로그를 통해 누가 무엇을 했는지 추적할 수 있습니다.
  </Card>

  <Card href="/l/ko/developers/extend/api" img="/images/user-guide/halftone/dev-api.png">
    <CardTitle>API & Extensibility</CardTitle>
    커스텀 데이터 모델에 맞게 확장되는 개발자 우선 API입니다.
    <br />워크스페이스별로 자동 생성되는 문서를 갖춘 GraphQL 및 REST 엔드포인트를 모두 제공합니다.
  </Card>

  <Card href="/l/ko/user-guide/data-migration/overview" img="/images/user-guide/halftone/data-migration.png">
    <CardTitle>Data Import & Export</CardTitle>
    CSV 파일 또는 API를 통해 데이터를 가져오세요.
    <br />필드 매핑, 중복 감지, 오류 처리가 내장되어 있습니다. 언제든지 데이터를 내보낼 수 있어 벤더 종속이 없습니다.
  </Card>

  <Card href="/l/ko/developers/self-host/self-host" img="/images/user-guide/halftone/dev-self-host.png">
    <CardTitle>Self-Hosting</CardTitle>
    단일 Docker Compose 명령으로 자체 인프라에서 Twenty를 실행하세요.
    <br />데이터를 완전히 제어하고, 업데이트도 일정에 맞춰 진행하세요.
  </Card>
</CardGroup>
