## 예정된 기능

### 맞춤형 보고서

비즈니스에 가장 중요한 지표를 추적하도록 맞춤형 보고서를 생성하세요.

### 세일즈 대시보드

대화형 대시보드로 판매 파이프라인 성과를 시각화하세요.

### 성과 분석

팀 성과를 모니터링하고 고객 데이터의 트렌드를 식별하세요.

## 업데이트 유지

리포트 기능이 제공될 때 알림을 받으려면 [GitHub 저장소](https://github.com/twentyhq/twenty)를 팔로우하거나 작업 공간의 **설정 → 릴리스** 섹션을 확인하세요.
