테이블 뷰를 Select 필드로 그룹화하여 레코드를 접을 수 있는 섹션으로 구성하세요.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

## 단계

1. 해당 객체(사람, 회사 등)로 이동합니다.
2. 뷰 드롭다운에서 → **+ 뷰 추가**를 클릭합니다.
3. 뷰의 이름을 지정합니다(예: "유형별 회사").
4. **만들기**를 클릭합니다.
5. **옵션 → 그룹**을 클릭합니다.
6. 그룹 기준이 될 Select 필드를 선택합니다.
7. **저장** 클릭

## 그룹 순서 설정

**옵션 → 그룹 → 정렬**에서 그룹의 정렬 방식을 선택합니다:

| 옵션         | 설명                               |
| ---------- | -------------------------------- |
| **알파벳순**   | A부터 Z                            |
| **알파벳 역순** | Z부터 A                            |
| **수동 정렬**  | "표시되는 그룹" 아래에서 그룹을 끌어 순서를 변경하세요. |

뷰에서 숨기려면 그룹 옆의 **눈 아이콘**을 클릭하세요.

<Note>
  **최상의 성능을 위해 표시되는 그룹을 10~15개로 제한하세요.** 더 필요하다면 대신 대시보드를 사용하는 것을 고려하세요.
</Note>

## 예: 업종별 회사

1. **회사**로 이동합니다.
2. "업종별"이라는 새 뷰를 만듭니다.
3. **옵션 → 그룹**을 클릭합니다.
4. **업종** 필드를 선택합니다.
5. 저장

이제 회사가 업종별로 정리되어 한 번에 한 세그먼트에 집중하기가 쉬워집니다.
