## 보기 이해하기

보기는 데이터가 어떻게 표시될지를 결정하는 저장된 구성입니다. 각 보기는 다음을 고유하게 가질 수 있습니다:

* **레이아웃**: 테이블, 칸반 또는 캘린더
* **필터**: 표시할 레코드
* **정렬**: 레코드의 정렬 방식
* **필드**: 표시할 열

## 보기 유형

### 테이블 보기

스프레드시트와 유사한 기본 보기로, 레코드를 행으로 표시하며 열을 사용자 지정할 수 있습니다.

### 칸반 보기

레코드가 단계별로 구성된 카드로 표시되는 보드형 시각적 보기입니다. 다음에 적합합니다:

* 영업 파이프라인
* 프로젝트 추적
* 정의된 단계가 있는 모든 워크플로

### 캘린더 보기

날짜 필드가 있는 레코드를 달력에 표시합니다. 다음에 적합합니다:

* 회의 및 행사
* 마감일 및 기한
* 시간 기반 계획

## 뷰 생성하기

새 뷰를 생성하는 두 가지 방법이 있습니다.

### 보기 드롭다운 메뉴 사용

1. 임의의 객체(사람, 회사 등)로 이동합니다.
2. 왼쪽 상단의 보기 이름을 클릭합니다(현재 보기가 드롭다운 화살표와 함께 표시됨).
3. **+ 보기 추가**를 클릭합니다.
4. 보기 이름을 입력하고 **만들기**를 클릭합니다.
5. **옵션**에서 레이아웃(테이블, 칸반 또는 캘린더)을 선택합니다.
6. 필요에 따라 필터와 정렬을 추가합니다.
7. 표시할 필드를 선택하고 순서를 재정렬합니다.
8. **저장** 클릭

### 기존 보기를 편집하는 것부터 시작하세요.

1. 임의의 객체(사람, 회사 등)로 이동합니다.
2. **옵션**에서 레이아웃(테이블, 칸반 또는 캘린더)을 선택하거나 필요에 따라 필터와 정렬을 추가합니다.
3. **새 보기로 저장**을 클릭합니다.
4. 보기 이름을 입력하고 **만들기**를 클릭합니다.
5. 새 보기를 계속 편집합니다.
6. 추가 구성 내용을 저장하려면 **보기 업데이트**를 클릭합니다.

## 보기 관리

### 보기 편집

1. 드롭다운에서 보기를 선택합니다.
2. 변경 사항(필터, 정렬, 열)을 적용합니다.
3. 보기를 업데이트하려면 **저장**을 클릭합니다.

### 보기 이름 변경 또는 아이콘 변경

1. 보기 드롭다운을 엽니다.
2. 보기 이름 옆의 **⋮** 메뉴를 클릭합니다.
3. **편집**을 선택합니다.
4. 이름 또는 아이콘을 변경합니다.
5. **저장** 클릭

### 보기 재정렬

1. 보기 드롭다운을 엽니다.
2. 핸들을 클릭한 채로 보기를 드래그합니다.
3. 원하는 위치에 놓습니다.
4. 새 순서는 자동으로 저장됩니다.

### 즐겨찾기에 추가

자주 사용하는 보기를 고정하여 빠르게 액세스하세요:

1. 보기 드롭다운을 엽니다.
2. 보기 옆의 **⋮** 메뉴를 클릭합니다.
3. **즐겨찾기에 추가**를 선택합니다.
4. 해당 보기가 즐겨찾기 섹션에 표시됩니다.

### 보기 삭제

1. 삭제할 보기를 선택합니다.
2. 보기 드롭다운을 클릭합니다.
3. 보기 옆의 **⋮** 메뉴를 클릭합니다.
4. **삭제**를 선택합니다.
5. 삭제를 확인합니다.

<Warning>
  삭제된 보기는 복구할 수 없습니다. 확인하기 전에 제거할지 반드시 확인하세요.
</Warning>

## 뷰 가시성

각 보기(기본 "모든 [객체 이름]" 보기 제외)에는 고유한 가시성 설정이 있습니다.

가시성을 변경하려면:

1. 보기를 엽니다.
2. **옵션 → 가시성**을 클릭합니다.
3. 다음을 선택합니다:
   * **워크스페이스**: 모든 워크스페이스 구성원에게 표시
   * **비공개**: 본인에게만 표시

<Note>
  기본 "모든 [객체 이름]" 보기는 가시성을 변경할 수 없습니다.
</Note>
