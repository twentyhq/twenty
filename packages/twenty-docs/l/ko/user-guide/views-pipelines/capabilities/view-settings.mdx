## 뷰 가시성

사용자 지정 보기를 누가 볼 수 있는지 제어하세요.

### 가시성 옵션

| 설정         | 볼 수 있는 대상    |
| ---------- | ------------ |
| **워크스페이스** | 모든 워크스페이스 멤버 |
| **비공개**    | 본인만          |

### 가시성 변경

1. 보기를 여세요.
2. **Options → Visibility**를 클릭하세요.
3. **워크스페이스** 또는 **비공개**를 선택하세요.

<Note>
  기본 "All [Object Name]" 보기는 가시성을 변경할 수 없습니다.
</Note>

## 보기 이름 변경

1. 보기 드롭다운 메뉴를 여세요.
2. 보기 옆의 **⋮** 메뉴를 클릭하세요.
3. **Edit**을 선택하세요.
4. 새 이름을 입력하세요.

## 보기 아이콘 변경

1. 보기 드롭다운 메뉴를 여세요.
2. 보기 옆의 **⋮** 메뉴를 클릭하세요.
3. **Edit**을 선택하세요.
4. 아이콘을 클릭하여 변경하세요.

## 보기 재정렬

드롭다운에 표시되는 보기의 순서를 변경하려면:

1. 보기 드롭다운 메뉴를 여세요.
2. 핸들을 이용해 보기를 드래그하세요.
3. 원하는 위치에 놓으세요.
4. 순서는 자동으로 저장됩니다.

## 즐겨찾기

빠른 액세스를 위해 자주 사용하는 보기를 고정하세요:

1. 보기 드롭다운 메뉴를 여세요.
2. 보기 옆의 **⋮** 메뉴를 클릭하세요.
3. **Add to favorites**를 선택하세요.

즐겨찾기에 추가된 보기는 손쉽게 액세스할 수 있도록 전용 섹션에 표시됩니다.

## 보기 삭제

1. 보기 드롭다운 메뉴를 여세요.
2. 보기 옆의 **⋮** 메뉴를 클릭하세요.
3. **Delete**를 선택하세요.
4. 삭제를 확인하세요.

<Warning>
  삭제된 보기는 복구할 수 없습니다.
</Warning>
