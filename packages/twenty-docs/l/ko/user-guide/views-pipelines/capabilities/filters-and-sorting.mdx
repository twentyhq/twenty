## 데이터 필터링

필터는 조건과 일치하는 레코드만 표시하여 특정 레코드에 집중할 수 있도록 도와줍니다.

### 필터 추가

1. 도구 모음에서 **필터** 버튼을 클릭합니다
2. 필터링할 필드를 선택합니다
3. 연산자(같음, 포함 등)를 선택합니다
4. 필터 값을 입력합니다
5. **적용**을 클릭합니다

### 필터 연산자

| 필드 유형 | 사용 가능한 연산자             |
| ----- | ---------------------- |
| 텍스트   | 같음, 포함, 시작함, 끝남, 비어 있음 |
| 숫자    | 같음, 초과, 미만, 사이, 비어 있음  |
| 날짜    | 같음, 이전, 이후, 사이, 비어 있음  |
| 선택    | 같음, 다음 중 하나, 비어 있음     |
| 체크박스  | 참, 거짓                  |
| 관계    | 같음, 비어 있음              |

### 다중 필터

결과를 좁히기 위해 여러 필터를 결합합니다:

* 모든 필터는 AND 논리로 적용됩니다
* 필터를 추가할수록 결과가 더 제한됩니다

### 필터 제거

* 개별 필터 칩의 **X**를 클릭합니다
* 모든 필터를 제거하려면 **모두 지우기**를 클릭합니다

## 데이터 정렬

정렬은 레코드가 표시되는 순서를 결정합니다.

### 정렬 추가

1. 도구 모음에서 **정렬** 버튼을 클릭합니다
2. 정렬할 필드를 선택합니다
3. 오름차순(A-Z, 0-9) 또는 내림차순(Z-A, 9-0)을 선택합니다
4. **적용**을 클릭합니다

### 다중 정렬

여러 정렬 수준을 추가합니다:

* 첫 번째 정렬이 기본입니다
* 이후 정렬은 동일한 값 그룹 내에서 적용됩니다

### 빠른 열 정렬

정렬하려면 열 헤더를 클릭하세요:

* 첫 번째 클릭: 오름차순
* 두 번째 클릭: 내림차순
* 세 번째 클릭: 정렬 제거

## 필터 및 정렬 설정 저장

필터와 정렬은 뷰와 함께 저장됩니다:

1. 필터와 정렬을 구성합니다
2. 현재 뷰를 업데이트하려면 **저장**을 클릭합니다
3. 또는 변형을 만들려면 **새 뷰로 저장**을 클릭합니다
