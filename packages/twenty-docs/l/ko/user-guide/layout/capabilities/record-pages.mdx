## 개요

Twenty에서 레코드를 열면 상세 페이지는 **탭**과 **위젯**으로 구성됩니다. 둘 다 오브젝트 유형별로 완전히 사용자 지정할 수 있습니다.

## 탭

각 레코드 페이지에는 여러 개의 탭을 둘 수 있습니다 — 브라우저의 탭과 비슷합니다. 이를 사용해 레코드의 다양한 측면을 구성하세요. 예를 들어 회사 레코드에는 개요, 커뮤니케이션, 작업, 파일용 탭이 있을 수 있습니다.

다음을 수행할 수 있습니다:

* 탭 추가 및 제거
* 탭 이름 변경
* 드래그하여 탭 재정렬
* 기본으로 표시할 탭 설정

## 위젯

위젯은 각 탭 내부의 기본 구성 요소입니다. 사용 가능한 위젯 유형:

| 위젯            | 표시 내용            |
| ------------- | ---------------- |
| **필드**        | 레코드 필드(그룹 또는 개별) |
| **관련 레코드**    | 관계로 연결된 레코드 표    |
| **이메일**       | 연결된 계정의 이메일 기록   |
| **캘린더**       | 레코드와 연관된 캘린더 이벤트 |
| **타임라인**      | 활동 및 이벤트 기록      |
| **작업**        | 연관된 작업           |
| **노트**        | 서식 있는 텍스트 노트     |
| **파일**        | 파일 첨부            |
| **차트**        | 관련 레코드의 시각적 데이터  |
| **iFrame**    | 임베드된 외부 콘텐츠      |
| **서식 있는 텍스트** | 정적 콘텐츠 또는 설명     |

## 레코드 페이지 맞춤 설정

1. 임의의 레코드를 여세요
2. `Cmd+K`를 누른 다음 "Edit record page layout"을 검색하세요

또는

1. 설정 > 데이터 모델 > 원하는 오브젝트 > 레이아웃으로 이동하세요

2. 해당 오브젝트에서 "레코드 페이지 사용자 지정" 버튼을 클릭하세요

3. 이제 맞춤 설정 모드입니다:
   * **위젯 추가** — 위젯 선택기에서
   * **위젯 드래그**하여 그리드에서 재배치하세요
   * **위젯 크기 조절** — 가장자리를 드래그하세요
   * **필드 구성** — 각 위젯에 표시되는 필드를 설정하세요
   * **탭 관리** — 추가, 제거, 이름 변경, 재정렬

4. 변경 내용을 저장하세요 — 해당 오브젝트 유형의 모든 레코드에 적용됩니다

## 필드 가시성

필드 위젯에서 어떤 필드를 표시할지와 표시 순서를 제어할 수 있습니다. 이를 통해 집중된 레이아웃을 만들 수 있습니다 — 예를 들어 개요 탭에는 가장 중요한 필드만 표시하고, 자세한 필드는 별도의 탭에 배치할 수 있습니다.
