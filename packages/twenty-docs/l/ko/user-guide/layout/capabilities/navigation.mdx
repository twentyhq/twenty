왼쪽 사이드바는 Twenty를 탐색하는 기본 방법입니다. 완전히 사용자 지정할 수 있습니다 — 어떤 설정 페이지도 열 필요 없이 워크플로에 맞게 재구성할 수 있습니다.

## 항목 재정렬

사이드바의 어떤 항목이든 끌어서 놓아 위치를 변경하세요. 순서는 사용자별로 저장되므로 팀원마다 자신의 사이드바를 구성할 수 있습니다.

## 폴더

관련 항목을 폴더로 묶으세요. 예를 들어 파이프라인 보기를 담은 "영업" 폴더, 티켓용 "지원" 폴더, 내부 객체용 "운영" 폴더를 만들 수 있습니다.

폴더를 만들려면 사이드바에서 마우스 오른쪽 버튼을 클릭하거나 `+` 버튼을 사용하세요.

## 객체 숨기기

사용하지 않는 객체는 사이드바에서 숨길 수 있습니다. 삭제되는 것이 아닙니다 — 단지 보이지 않게 될 뿐입니다. 설정 > 데이터 모델에서 언제든 다시 표시할 수 있습니다.

## 즐겨찾기

사이드바 상단의 즐겨찾기 섹션에 보기, 레코드 또는 검색을 고정하여 한 번의 클릭으로 접근하세요. 즐겨찾기는 개인별입니다 — 각 사용자가 자신만의 즐겨찾기를 관리합니다.

## 사용자 지정 링크

외부 도구에 대한 링크를 사이드바에 직접 추가하세요. 팀 위키, 다른 도구의 대시보드 또는 팀이 자주 사용하는 모든 URL을 링크하는 데 유용합니다.

## 명령 메뉴

`Cmd+K`(또는 `Ctrl+K`)를 눌러 명령 메뉴를 여세요 — 사이드바를 탐색하지 않고도 어떤 레코드, 보기 또는 작업으로도 빠르게 이동할 수 있는 빠른 액세스 검색창입니다.

## 사이드바 사용자 정의

사이드바를 사용자 정의하려면 사이드바의 "Workspace" 섹션 위에 마우스를 올린 다음 렌치 아이콘을 클릭하세요.
