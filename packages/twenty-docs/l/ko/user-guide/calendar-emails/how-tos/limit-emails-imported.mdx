## 개요

기본적으로 Twenty는 연결된 메일박스의 모든 외부 이메일을 동기화합니다. **폴더 선택**과 **표시 설정**을 사용하여 가져올 항목을 제한할 수 있습니다.

## 방법 1: 폴더 선택(권장)

Twenty와 동기화할 이메일 폴더 제어:

1. **설정 → 계정**으로 이동
2. 연결된 이메일 계정을 선택하세요.
3. 동기화할 폴더를 선택하세요:

| 폴더            | 설명        |
| ------------- | --------- |
| **받은 편지함**    | 주요 수신 이메일 |
| **보낸 편지함**    | 발송한 이메일   |
| **보관함**       | 보관된 메시지   |
| **사용자 지정 폴더** | 원하는 특정 폴더 |

6. 동기화하지 않을 폴더를 제외하세요(스팸, 휴지통, 개인 폴더).

<Note>모든 이메일을 전부 동기화하지 않고도 CRM에 표시되는 이메일을 정확하게 제어할 수 있습니다.</Note>

## 방법 2: 연락처 자동 생성 설정

이메일에서 연락처를 생성할 시점을 제어하세요:

1. **설정 → 계정**으로 이동
2. 연결된 메일박스를 선택하세요.
3. 옵션을 선택하세요:
   * **비활성화**: 연락처는 생성되지 않지만, 이메일은 기존 연락처와 계속 동기화됩니다
   * **발신 및 수신**: 모든 외부 이메일에서 연락처를 생성합니다
   * **발신만**: 사용자가 보낸 이메일에서만 연락처를 생성합니다

## 항상 제외되는 항목

설정과 관계없이 다음 이메일은 동기화되지 않습니다:

* **내부 이메일**: 동료 간(동일 도메인) 메시지
* **그룹 이메일**: 배포 목록 및 그룹 메시지
* **스팸/휴지통**: 시스템 폴더는 일반적으로 제외됩니다

## 중요 참고 사항

<Note>선택적 동기화를 위한 CC 이메일 주소는 제공하지 않습니다. 위의 폴더 선택 기능을 사용하여 동일한 수준의 제어를 구현하세요.</Note>
