## 현재 상태

Twenty의 캘린더 통합은 기존 캘린더 이벤트를 **동기화하고 표시**하도록 설계되었습니다.

## 워크플로우를 통해 미팅 예약하기

미팅을 수동으로 예약할 수는 없지만, Twenty에서는 워크플로우를 사용해 예약할 수 있습니다.

1. 새 워크플로우 만들기
2. 필요에 따라 트리거를 선택하세요.
3. **Create Calendar Event** 노드를 추가하세요.
4. 필요한 모든 데이터를 설정하세요.
5. 워크플로우를 활성화합니다

## 가능한 작업

* 사람, 회사 및 기회 레코드에서 **미팅 기록 보기**
* CRM의 연락처와의 **예정된 미팅 보기**
* 이메일 커뮤니케이션과 함께 **미팅 맥락 추적**
* 미팅 참가자를 기반으로 **연락처 자동 생성**
