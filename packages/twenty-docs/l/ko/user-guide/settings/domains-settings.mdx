## Custom Workspace Domain

작업공간의 개인화된 웹 주소를 설정합니다:

- 선호하는 서브도메인을 선택하세요 (예: yourcompany.twenty.com)
- Configure custom domains for professional branding

## 승인된 접근 도메인

지정된 이메일 도메인에 대해 자동 작업공간 접근을 허용합니다.

### 작동 방식

- 회사 이메일 도메인을 추가하세요 (예: @yourcompany.com)
- 이 도메인에서 이메일을 가진 사용자는 자동으로 작업공간에 가입할 수 있습니다.
- 팀원 초대는 수동으로 필요하지 않습니다.

### 이점

- **빠른 온보딩**: 새로운 팀원은 자동으로 가입합니다.
- **향상된 보안**: 확인된 회사 이메일만 접근할 수 있습니다.
- **관리 작업 감소**: 각 팀원을 수동으로 초대할 필요 없음

## 설정 지침

1. Go to **Settings → Domains**
2. 맞춤 도메인 또는 승인된 이메일 도메인을 추가합니다.
3. 필요한 경우 도메인 소유권을 확인합니다.

**참고**: 도메인 변경사항은 적용에 시간이 걸릴 수 있으며 확인이 필요할 수 있습니다.
