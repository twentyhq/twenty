## 작업 공간 설정

<AccordionGroup>
  <Accordion title="워크스페이스 이름은 어떻게 변경하나요?">
    1. **설정 → 일반**으로 이동하세요.
    2. 워크스페이스 이름 필드를 찾으세요.
    3. 새 이름을 입력하세요.
    4. 변경 사항은 자동으로 저장됩니다.
  </Accordion>

  <Accordion title="워크스페이스 로고는 어떻게 변경하나요?">
    1. **설정 → 일반**으로 이동하세요.
    2. 현재 로고 또는 업로드 영역을 클릭하세요.
    3. 10MB 이하의 PNG, JPEG 또는 GIF 이미지를 선택하세요.
    4. 로고가 즉시 업데이트됩니다.
  </Accordion>

  <Accordion title="여러 개의 워크스페이스를 가질 수 있나요?">
    네, 여러 워크스페이스를 생성하고 멤버가 될 수 있습니다. 각 워크스페이스는 고유한 데이터, 설정 및 구독을 가집니다.
  </Accordion>

  <Accordion title="워크스페이스를 삭제하려면 어떻게 해야 하나요?">
    1. **설정 → 일반**으로 이동하세요.
    2. 위험 구역까지 스크롤하세요.
    3. **워크스페이스 삭제**를 클릭하세요.
    4. 삭제를 확인하세요.

    참고: 이 작업은 모든 데이터를 영구적으로 삭제하며 되돌릴 수 없습니다.
  </Accordion>

  <Accordion title="워크스페이스를 여러 개 실수로 생성했습니다. 어떻게 해야 하나요?">
    필요하지 않은 워크스페이스는 **설정 → 일반 → 워크스페이스 삭제**에서 삭제하세요.

    <Warning>
      **계정**을 삭제하지 마세요(설정 → 프로필에서 접근 가능): 계정은 모든 워크스페이스에서 공유됩니다. 계정을 삭제하면 모든 워크스페이스에 대한 액세스가 제거됩니다.
    </Warning>
  </Accordion>

  <Accordion title="삭제하지 않고 워크스페이스를 비활성화하려면 어떻게 하나요?">
    워크스페이스를 일시적으로 비활성화하려는 경우(영구 삭제가 아님) **설정 → 결제**로 이동하여 **플랜 취소**를 클릭하세요. 데이터는 유예 기간 동안 보존됩니다.
  </Accordion>
</AccordionGroup>

## 프로필 설정

<AccordionGroup>
  <Accordion title="비밀번호는 어떻게 변경하나요?">
    1. **설정 → 프로필**로 이동하세요.
    2. 비밀번호 섹션을 찾으세요.
    3. 현재 비밀번호를 입력하세요.
    4. 새 비밀번호를 입력하세요.
    5. 변경 사항을 저장하세요.
  </Accordion>

  <Accordion title="2단계 인증은 어떻게 활성화하나요?">
    1. **설정 → 프로필**로 이동하세요.
    2. 2단계 인증 섹션을 찾으세요.
    3. **2FA 활성화** 클릭
    4. 인증 앱으로 QR 코드를 스캔합니다
    5. 인증 코드를 입력하세요.
  </Accordion>

  <Accordion title="이메일 주소를 변경할 수 있나요?">
    이메일 주소를 변경하려면 [contact@twenty.com](mailto:contact@twenty.com)으로 문의해 주세요.
  </Accordion>

  <Accordion title="계정을 삭제하려면 어떻게 해야 하나요?">
    1. **설정 → 프로필**로 이동하세요.
    2. 위험 구역까지 스크롤하세요.
    3. **계정 삭제** 클릭
    4. 이메일 주소를 입력하여 확인하세요.

    참고: 이 작업은 모든 워크스페이스에 대한 액세스를 제거하고 연결된 계정에서 동기화된 모든 이메일을 삭제합니다.
  </Accordion>
</AccordionGroup>

## 경험 설정

<AccordionGroup>
  <Accordion title="밝은 모드와 어두운 모드 간 전환은 어떻게 하나요?">
    1. **설정 → 환경**으로 이동하세요.
    2. 테마 섹션을 찾으세요.
    3. 밝은 모드, 어두운 모드 또는 시스템을 선택하세요.
  </Accordion>

  <Accordion title="날짜 형식은 어떻게 변경하나요?">
    1. **설정 → 환경**으로 이동하세요.
    2. 날짜 형식 항목을 찾으세요.
    3. 선호하는 형식을 선택하세요.
    4. 변경 사항이 즉시 적용됩니다.
  </Accordion>

  <Accordion title="시간대는 어떻게 변경하나요?">
    1. **설정 → 환경**으로 이동하세요.
    2. 시간대 항목을 찾으세요.
    3. 사용 중인 지역 시간대를 선택하세요.
    4. 모든 타임스탬프가 조정됩니다.
  </Accordion>

  <Accordion title="언어를 변경할 수 있나요?">
    1. **설정 → 환경**으로 이동하세요.
    2. 언어 항목을 찾으세요.
    3. 사용 가능한 언어 중에서 선택하세요.
    4. 인터페이스가 선택한 언어로 업데이트됩니다.
  </Accordion>
</AccordionGroup>

## 계정 설정

<AccordionGroup>
  <Accordion title="이메일 계정은 어떻게 연결하나요?">
    1. **설정 → 계정**으로 이동
    2. **계정 추가**를 클릭
    3. Google 또는 Microsoft를 선택하세요.
    4. 액세스 권한을 승인하세요.
    5. 동기화 설정을 구성하세요.
  </Accordion>

  <Accordion title="여러 이메일 계정을 연결할 수 있나요?">
    네, 여러 이메일 계정을 연결할 수 있습니다. **설정 → 계정**으로 이동하여 필요한 계정을 추가하세요.
  </Accordion>

  <Accordion title="이메일 계정 연결을 해제하려면 어떻게 하나요?">
    1. **설정 → 계정**으로 이동
    2. 제거할 계정을 찾으세요.
    3. **연결 해제**를 클릭하세요.
    4. 작업을 확인하세요.
  </Accordion>
</AccordionGroup>

## 도메인

<AccordionGroup>
  <Accordion title="워크스페이스 도메인을 맞춤 설정할 수 있나요?">
    예! **설정 → 도메인**으로 이동하여 **도메인 사용자 지정**을 클릭하세요. 다음 두 가지 옵션이 있습니다:

    * **서브도메인**: `yourcompany.twenty.com`과 같은 Twenty 서브도메인을 사용합니다.
    * **사용자 지정 도메인**: `crm.yourcompany.com`과 같은 본인 소유 도메인을 사용합니다(DNS 구성이 필요).

    서브도메인은 빠르게 설정할 수 있고, 사용자 지정 도메인은 팀에 완전한 브랜드 경험을 제공합니다.
  </Accordion>

  <Accordion title="승인된 접근 도메인은 어떻게 작동하나요?">
    승인된 접근 도메인을 구성하면 회사 이메일 주소를 가진 팀원이 워크스페이스에 자동으로 참여할 수 있습니다. **설정 → 도메인**으로 이동하여 회사 도메인(예: `yourcompany.com`)을 추가하세요.
  </Accordion>
</AccordionGroup>

## 얼리 액세스 기능

<AccordionGroup>
  <Accordion title="얼리 액세스 기능이란 무엇인가요?">
    얼리 액세스 기능은 일반 출시 전에 테스트 중인 기능입니다. 현재는 **대시보드**만 얼리 액세스로 제공됩니다. 사용자 피드백에 따라 변경될 수 있습니다.
  </Accordion>

  <Accordion title="얼리 액세스 기능을 사용해도 안전한가요?">
    얼리 액세스 기능은 정상적으로 동작하지만 사용자 피드백에 따라 변경될 수 있습니다. 사용해도 안전하지만 일반 제공 이전에 변경될 수 있습니다.
  </Accordion>

  <Accordion title="얼리 액세스 기능은 어떻게 활성화하나요?">
    1. **설정 → 업데이트 → 얼리 액세스**로 이동하세요
    2. 원하는 기능을 찾으세요.
    3. 토글하여 활성화하세요.
    4. 해당 기능이 즉시 사용 가능해집니다.
  </Accordion>
</AccordionGroup>
