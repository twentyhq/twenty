## 개인 정보

### 이름 및 이메일

* **표시 이름**: 다른 워크스페이스 구성원에게 표시되는 이름을 업데이트하세요
* **이메일 주소**: 로그인 이메일을 변경하세요 (인증 필요)
* **프로필 사진**: 사용자 정의 아바타를 업로드하거나 이니셜을 사용하세요

## 보안 설정

### 이중 인증 (2FA)

계정에 추가 보안을 위한 2FA를 활성화하십시오:

1. **설정 → 프로필 설정**으로 이동합니다
2. **2FA 활성화** 클릭
3. 인증 앱으로 QR 코드를 스캔합니다
4. 확인을 위해 인증 코드를 입력합니다

### 비밀번호 관리

* **비밀번호 변경**: 현재 비밀번호를 업데이트합니다
* **비밀번호 요구사항**: 최소 8자 이상이어야 합니다

## 프로필 관리

### 계정 삭제

<Warning>
  계정을 삭제하면 모든 워크스페이스에 대한 접근이 영구적으로 제거됩니다. 이 작업은 취소할 수 없으며, 특정 팀에서만 나가고자 한다면 개별 워크스페이스를 먼저 떠나는 것을 고려해야 합니다.
</Warning>

계정을 삭제하려면:

1. **설정 → 프로필 설정**으로 이동합니다
2. **위험 구역**까지 스크롤하세요.
3. **계정 삭제** 클릭
4. 이메일 주소를 입력하여 확인합니다
