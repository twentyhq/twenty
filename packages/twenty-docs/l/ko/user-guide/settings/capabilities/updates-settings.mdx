## 업데이트 설정 정보

업데이트 섹션에서는 일반 제공되기 전에 얼리 액세스 기능을 활성화할 수 있습니다.

## 얼리 액세스 기능

얼리 액세스 기능은 일반 제공 전에 아직 테스트 중인 기능입니다.

**현재 얼리 액세스:**

* **대시보드** — 시각적 보고 및 분석

### 얼리 액세스 기능 활성화 방법

1. **설정 → 업데이트**로 이동하세요.
2. **얼리 액세스** 섹션으로 이동하세요
3. 활성화하려는 기능을 찾으세요
4. 토글하여 활성화하세요
5. 해당 기능이 즉시 사용 가능해집니다

<Note>
  얼리 액세스 기능은 사용자 피드백에 따라 변경될 수 있습니다. 여러분의 피드백은 Twenty를 개선하는 데 도움이 됩니다!
</Note>

## 기능 피드백

여러분의 피드백은 Twenty를 개선하는 데 도움이 됩니다:

* 얼리 액세스 기능 관련 문제를 보고하세요
* 새로운 기능을 어떻게 사용하고 있는지 공유하세요
* 커뮤니티 Discord를 통해 개선 사항을 제안하세요
