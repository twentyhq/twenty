title: 경험 설정

---

Twenty 사용 경험을 **설정 → 경험**에서 개인화하세요.

## 외관

화면 테마를 선택하세요:

* **라이트**: 깔끔하고 밝은 인터페이스
* **다크**: 저조도 환경에서 눈에 더 편안합니다
* **시스템 설정**: 기기의 테마에 자동으로 맞춥니다

## 언어

드롭다운 메뉴에서 Twenty 인터페이스에 사용할 선호 언어를 선택하세요.

## 형식

날짜, 시간, 숫자, 시간대 및 캘린더 시작 요일을 구성하세요.

| 설정            | 설명                          |
| ------------- | --------------------------- |
| **시간대**       | 정확한 타임스탬프와 일정 관리를 위한 로컬 시간대 |
| **날짜 형식**     | 날짜 표시 방식(예: 2025년 12월 12일)  |
| **시간 형식**     | 12시간제(오후 7:22) 또는 24시간제     |
| **숫자 형식**     | 소수점 및 천 단위 구분자(예: 1,234.56) |
| **캘린더 시작 요일** | 한 주의 시작 요일(일요일 또는 월요일)      |

각 설정은 **시스템 설정**으로 지정해 기기 설정에 자동으로 맞추거나, 특정 형식을 선택할 수 있습니다.

## 업데이트 방법

1. **설정 → 경험**으로 이동
2. 각 섹션에서 기본 설정을 조정합니다.
3. 변경 사항은 자동으로 저장됩니다.
