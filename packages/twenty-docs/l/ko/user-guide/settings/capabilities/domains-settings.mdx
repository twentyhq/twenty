**설정 → 도메인**에서 도메인 설정을 구성하세요.

## 워크스페이스 도메인

하위 도메인 이름을 편집하거나 작업 공간에 사용자 지정 도메인을 설정하세요.

### 도메인 사용자 지정

1. **도메인 사용자 지정**을 클릭하세요
2. 하위 도메인을 편집하세요(예: `yourcompany.twenty.com`)
3. 또는 사용자 지정 도메인을 설정하세요(예: `crm.yourcompany.com`)

사용자 지정 도메인의 경우 도메인 제공업체에서 DNS 설정을 구성해야 합니다.

## 승인된 도메인

이 도메인의 이메일 주소를 가진 사람은 이 작업 공간에 자동으로 가입할 수 있습니다.

### 승인된 액세스 도메인 추가

1. **승인된 액세스 도메인 추가**를 클릭하세요
2. 회사 도메인을 입력하세요(예: `yourcompany.com`)
3. 저장

구성이 완료되면 해당 도메인의 이메일 주소를 가진 사람은 직접 초대 없이 작업 공간에 가입할 수 있습니다.

<Note>
  작업 공간을 귀 조직으로 제한하면서 팀 전체가 자가 등록할 수 있도록 하는 데 유용합니다.
</Note>

## 공개 도메인

이 도메인에서 완벽하고 안전한 호스팅 환경을 제공하십시오.

### 퍼블릭 도메인 추가

1. **퍼블릭 도메인 추가**를 클릭하세요
2. 사용할 도메인을 입력하세요
3. 안내에 따라 DNS 설정을 구성하세요
4. 도메인을 검증하세요

퍼블릭 도메인에는 SSL 인증서가 자동으로 발급됩니다.
