## 새 멤버 초대

### 초대 링크 사용

1. Go to **Settings → Members**
2. 작업 공간 초대 링크 복사
3. 새로운 팀 구성원에게 링크 공유
4. 가입 후 접근이 가능합니다

### 직접 이메일 초대

1. Go to **Settings → Members**
2. Enter the person's email address
3. **초대** 클릭
4. 이메일 초대장이 발송됩니다.

## 멤버 제거

### 멤버 삭제

1. Go to **Settings → Members**
2. 제거할 멤버를 찾으십시오.
3. 해당 이름 옆의 삭제/제거 버튼 클릭
4. 제거 확인

**참고**: 제거된 멤버는 즉시 접근이 차단되지만 나중에 다시 초대할 수 있습니다.
