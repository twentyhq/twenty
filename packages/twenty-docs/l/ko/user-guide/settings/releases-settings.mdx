## 최신 릴리즈

Twenty의 개발 진행 상황을 추적하십시오:

- **릴리즈 노트**: 각 버전에서 새로운 것이 무엇인지 확인
- **기능 업데이트**: 새로운 기능과 개선 사항 발견

## 랩 기능

정식 출시 전에 실험 기능에 액세스하십시오:

- **베타 테스트**: 개발 중인 새로운 기능을 사용하세요
- **기능 플래그**: 특정 실험적 기능을 활성화하거나 비활성화

## 액세스 방법

1. **설정 → 릴리즈**로 이동하세요.
2. 릴리즈 노트를 보고 베타 기능을 위해 **랩** 탭에 액세스하십시오.

추가 커뮤니티 리소스는 [리소스 섹션](/l/ko/user-guide/resources/glossary)을 확인하세요.
