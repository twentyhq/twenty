## 개요

Twenty는 클라우드 호스팅이든 셀프 호스팅이든, 모든 규모의 팀에 맞는 유연한 요금제를 제공합니다.

## 클라우드 요금제

### Pro(클라우드)

확장을 준비한 팀을 위한:

* 모든 핵심 CRM 기능
* 이메일 및 캘린더 동기화
* 워크플로와 자동화
* 표준 지원

<Note>
프리미엄 기능(SSO, 행 수준 권한, AI 사용 데이터 등)은 Pro 플랜에 포함되지 않습니다.
</Note>

### 조직(클라우드)

고급 기능이 필요한 대규모 팀용:

* Pro의 모든 기능
* **프리미엄 기능**: SSO 통합, 행 수준 권한, AI 사용 데이터 등
* 우선 지원

## 셀프 호스팅 요금제

### 무료(셀프 호스팅)

자체 인프라에 Twenty를 무료로 호스팅하세요:

* Pro의 모든 기능 포함
* Discord를 통한 커뮤니티 지원
* 데이터에 대한 완전한 제어

### 조직(셀프 호스팅)

셀프 호스팅하면서 프리미엄 기능이 필요한 팀용:

* Pro의 모든 기능
* **프리미엄 기능**: SSO 통합, 행 수준 권한, AI 사용 데이터 등
* Twenty 팀의 지원
* 배포 전에 커스텀 코드를 오픈 소스로 공개할 필요 없음

## 프리미엄 기능

프리미엄 기능은 조직 요금제(클라우드 또는 셀프 호스팅)에서만 제공됩니다:

* **SSO 통합**: ID 공급자와의 싱글 사인온
* **행 수준 권한**: 레코드 수준의 세밀한 액세스 제어
* **AI 사용 데이터**: 워크스페이스 전반의 AI 사용량을 추적하세요
* **감사 로그**: 각 워크스페이스 구성원이 무엇을 했는지 확인할 수 있습니다
* **이메일 캠페인**: 어떤 이메일을 발송할지 계획합니다(곧 제공 예정)
* **무제한 워크스페이스**: 원하는 만큼 워크스페이스를 생성할 수 있습니다(셀프 호스팅 전용)
* **비공개 소스 코드**: 변경 사항을 공개 저장소에 게시할 필요가 없습니다(셀프 호스팅 전용)

## 요금제 전환

### 조직으로 업그레이드

1. **설정 → 결제 → 요금제**로 이동
2. **조직으로 전환**을 클릭
3. 업그레이드를 확인하세요

### Pro로 다운그레이드

1. **설정 → 결제 → 요금제**로 이동
2. **Pro로 전환**을 클릭
3. 다운그레이드를 확인하세요

### 연간 결제로 전환

1. **설정 → 결제**로 이동
2. **연간 요금제로 업그레이드**를 클릭
3. 연간 결제로 절약하세요

### 월간 결제로 전환

1. **설정 → 결제**로 이동
2. **월간 요금제로 다운그레이드**를 클릭
3. 월간 결제로 저장

## Organization(셀프 호스팅)용 엔터프라이즈 키 받기

Organization(셀프 호스팅) 플랜을 사용하려면 엔터프라이즈 키를 받아야 합니다:

1. **설정 → 관리자 패널 → 엔터프라이즈**로 이동하세요

<img src="/images/user-guide/billing/enterprise-key.png" alt="엔터프라이즈 키" />

2. **엔터프라이즈 키 받기**를 클릭하세요
3. Stripe로 리디렉션되면 결제 정보를 입력하고 확인하세요
4. 엔터프라이즈 키가 표시되면 이를 엔터프라이즈 설정 페이지에 붙여넣고 Organization 라이선스를 활성화하세요
