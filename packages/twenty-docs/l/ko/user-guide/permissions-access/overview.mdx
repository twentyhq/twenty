## 주요 기능

* **역할 기반 액세스**: 특정 권한을 가진 사용자 지정 역할 생성
* **객체 권한**: 누가 기록을 보고, 편집하거나 삭제할 수 있는지 제어
* **필드 권한**: 민감한 필드에 대한 액세스 제한
* **설정 권한**: 워크스페이스 구성에 대한 액세스 제어
* **SSO 통합**: 엔터프라이즈 보안을 위해 Single Sign-On(SSO) 구성(Organization 요금제)
