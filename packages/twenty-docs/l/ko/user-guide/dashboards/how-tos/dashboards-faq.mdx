---
title: 대시보드 FAQ
description: Twenty의 대시보드에 관한 자주 묻는 질문입니다.
---

<AccordionGroup>
  <Accordion title="비공개 대시보드를 만들 수 있나요?">
    아니요. 현재는 Twenty 워크스페이스에 접근 권한이 있는 모든 사용자에게 대시보드가 표시됩니다. 비공개 대시보드는 아직 제공되지 않습니다.
  </Accordion>

  <Accordion title="내 지표가 동료의 것과 다르게 보이는 이유는 무엇인가요?">
    현재 대시보드는 보는 사람의 시간대를 기준으로 데이터를 표시합니다. 서로 다른 지역(예: APAC과 미국)에 있는 경우, 동일한 대시보드에서도 숫자가 약간 다르게 보일 수 있습니다. 팀 간 일관된 데이터를 보장하기 위해, 대시보드별 시간대 설정을 추가하는 작업을 진행 중입니다.
  </Accordion>

  <Accordion title="대시보드를 내보낼 수 있나요?">
    현재는 대시보드 내보내기를 지원하지 않습니다. 이 기능은 로드맵에 포함되어 있습니다.
  </Accordion>

  <Accordion title="대시보드를 외부 사용자와 공유할 수 있나요?">
    아니요. 현재는 Twenty 워크스페이스 외부 사용자(비 Twenty 사용자)와 대시보드를 공유할 수 없습니다.
  </Accordion>

  <Accordion title="대시보드를 즐겨찾기에 추가하려면 어떻게 하나요?">
    즐겨찾기에 추가할 대시보드를 연 다음 별 아이콘을 클릭하세요. 즐겨찾기는 개인 설정이며 다른 사용자에게는 영향을 주지 않습니다.
  </Accordion>

  <Accordion title="탭과 위젯의 차이점은 무엇인가요?">
    * **탭**은 대시보드를 섹션으로 구성합니다(대시보드 내의 페이지와 비슷함)
    * **위젯**은 각 탭 안에 있는 개별 시각화 요소입니다(차트, 숫자, iFrame 등)

    구성: 대시보드 → 탭 → 위젯
  </Accordion>

  <Accordion title="대시보드 전체에 한 번에 필터를 적용할 수 있나요?">
    대시보드 수준의 필터는 아직 제공되지 않지만, 이 기능은 로드맵에 포함되어 있습니다. 현재는 각 위젯에 개별적으로 필터를 적용해야 합니다.
  </Accordion>

  <Accordion title="게이지 차트나 표를 사용할 수 있나요?">
    아직 안 됩니다. 게이지 차트와 표는 로드맵에 포함되어 있으며 향후 릴리스에 추가될 예정입니다.
  </Accordion>

  <Accordion title="대시보드를 복제하려면 어떻게 하나요?">
    1. 보기 모드인지 확인하세요(편집 중이 아님).
    2. **Cmd + K**(Windows에서는 **Ctrl + K**)로 명령 표시줄을 여세요
    3. **대시보드 복제**를 선택하세요
  </Accordion>

  <Accordion title="대시보드 데이터는 얼마나 자주 새로고침되나요?">
    CRM 데이터가 변경되면 위젯이 자동으로 업데이트됩니다:

    * 대부분의 지표는 실시간으로 업데이트됩니다
    * 필요 시 새로고침 버튼을 눌러 수동으로 업데이트하세요
    * 추세 분석을 위해 과거 데이터가 보존됩니다
  </Accordion>
</AccordionGroup>
