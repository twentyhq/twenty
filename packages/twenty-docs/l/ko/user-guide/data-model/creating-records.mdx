## 테이블 뷰에 대하여

테이블 뷰는 행과 열로 구조화된 데이터의 시각적 표현입니다.

## 레코드 생성

필요에 따라 제한 없이 레코드를 추가하십시오. 레코드를 추가하려면 화면 오른쪽 상단 또는 레코드 **이름** 열 상단에 있는 **+** 버튼을 클릭하세요.

레코드 이름을 입력한 후 `Enter` 키를 눌러 저장합니다. 레코드 이름을 수정하려면 세부 페이지에서 이름을 클릭합니다.

<VimeoEmbed videoId="927071691" title="Video demonstration" />

## 레코드 삭제

**목록 보기:** 레코드를 삭제하려면 레코드 옆의 체크박스를 선택하고 오른쪽 상단의 삭제 버튼을 클릭하세요.

**레코드 페이지:** 오른쪽 상단의 `⋮` 아이콘을 탭한 후 삭제를 선택합니다.

<VimeoEmbed videoId="927073570" title="Video demonstration" />

## 사용자 지정 필드 추가

사용자 지정 필드를 생성하려면 테이블 열의 오른쪽 끝의 **+** 버튼을 클릭한 후 **필드 사용자 지정**을 선택합니다.

<img src="/images/user-guide/fields/quick-new-field.png" style={{width:'100%'}}/>

**설정** > **데이터 모델** > **사람**으로 이동하여 수행할 수 있습니다. **필드 추가**를 클릭합니다. 필드 이름과 유형을 선택하세요. 새 필드는 앱에서 사용할 수 있습니다.
