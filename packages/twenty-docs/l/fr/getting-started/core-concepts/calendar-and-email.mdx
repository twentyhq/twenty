Twenty se connecte à vos outils existants afin que votre CRM reste à jour sans saisie manuelle de données.

## Synchronisation des e-mails

Connectez votre boîte aux lettres **Google Workspace** ou **Microsoft 365**. Une fois la connexion établie :

* Les e-mails sont automatiquement liés aux enregistrements des entreprises et des contacts correspondants
* Les fils de discussion complets sont visibles sur la chronologie de chaque enregistrement
* Vous pouvez envoyer des e-mails directement depuis Twenty
* Plusieurs boîtes aux lettres par utilisateur sont prises en charge

Vous contrôlez ce qui est importé — filtrez par plage de dates ou par expéditeur afin d’éviter de récupérer des e-mails non pertinents.

## Synchronisation du calendrier

Les événements de calendrier sont synchronisés automatiquement depuis votre compte connecté. Les événements apparaissent sur les enregistrements CRM concernés, vous donnant une vue complète de vos interactions avec chaque contact ou entreprise.

## Intégrations

Au-delà de la messagerie et du calendrier, Twenty se connecte à des outils externes via :

| Méthode                             | Cas d'utilisation                                                                                       |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------- |
| **API**                             | Créez des intégrations personnalisées avec l’API GraphQL ou REST                                        |
| **Webhooks**                        | Envoyez des notifications en temps réel vers des systèmes externes lorsque les enregistrements changent |
| **Zapier**                          | Connectez-vous à plus de 5 000 applications sans code                                                   |
| **Actions HTTP de flux de travail** | Appelez n’importe quelle API externe dans le cadre d’un flux de travail automatisé                      |

## Applications personnalisées

Les développeurs peuvent créer des applications complètes sur Twenty — en ajoutant une interface utilisateur personnalisée, une logique côté serveur et des intégrations approfondies. Les applications peuvent être publiées pour la communauté ou rester privées.
