<Tabs>
<Tab title="Utilisation">

```jsx
import { Toggle } from "twenty-ui/input";

export const MyComponent = () => {
  return (
    <Toggle
    value = {true}
    onChange = {()=>console.log('On Change event')}
    color="green"
    toggleSize = "medium"
    />
  );
};
```

</Tab>
<Tab title="Propriétés">


| Propriétés     | Type     | Description                                                                                            | Par défaut           |
| -------------- | -------- | ------------------------------------------------------------------------------------------------------ | -------------------- |
| valeur         | booléen  | L'état actuel de la bascule                                                                            | `faux`               |
| onChange       | fonction | Fonction de rappel déclenchée lorsque le bascule d'état change                                         |                      |
| couleur        | chaîne   | Couleur de l'interrupteur quand il\                                                                    | est de couleur bleue |
| tailleBasculer | chaîne   | Taille du commutateur, affectant à la fois la taille et le poids. A deux options : `small` et `medium` | moyen                |



</Tab>
