## Vue d'ensemble

Intégrez des capacités d'IA directement dans vos flux de travail d'automatisation pour un traitement des données intelligent et une prise de décision intelligente.

## Capacités

| Fonctionnalité                 | Description                                                             |
| ------------------------------ | ----------------------------------------------------------------------- |
| **Actions d'IA**               | Ajoutez des étapes alimentées par l'IA à n'importe quel flux de travail |
| **Enrichissement des données** | Améliorez automatiquement les enregistrements avec des données externes |
| **Classification**             | Catégorisez les enregistrements en fonction de l'analyse du contenu     |
| **Synthèse**                   | Générez des résumés à partir de champs de texte                         |
| **Prompts personnalisés**      | Définissez précisément comment l'IA traite vos données                  |

## Cas d'utilisation

* **Scoring des prospects** : Attribuez automatiquement un score et hiérarchisez les prospects entrants
* **Nettoyage des données** : Normalisez les noms d'entreprise et les coordonnées
* **Brouillons d'e-mails** : Générez des e-mails de suivi à partir de notes de réunion
* **Routage des enregistrements** : Assignez les enregistrements au bon membre de l'équipe en fonction du contenu
