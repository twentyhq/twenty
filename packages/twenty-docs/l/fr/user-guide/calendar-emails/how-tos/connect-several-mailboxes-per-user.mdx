## Vue d'ensemble

Twenty prend en charge **un nombre illimité de comptes e-mail par utilisateur**. C'est utile si vous gérez plusieurs boîtes de réception, par exemple :

* E-mail professionnel individuel + boîte de réception d'équipe partagée
* Plusieurs adresses e-mail destinées aux clients
* Différents comptes e-mail pour différents rôles

## Comment ajouter plusieurs boîtes aux lettres

1. Allez dans **Paramètres → Comptes**
2. Cliquez sur **Ajouter un compte**
3. Connectez votre compte Google ou Microsoft supplémentaire
4. Configurez les paramètres de synchronisation pour cette boîte aux lettres
5. Répétez l'opération pour chaque boîte aux lettres que vous souhaitez connecter

## Gestion de plusieurs comptes

Chaque boîte aux lettres connectée possède ses propres paramètres :

* **Visibilité des e-mails** : Choisissez ce que les membres de l'équipe peuvent voir
* **Création automatique de contacts** : Activer/désactiver par boîte aux lettres
* **Sélection des dossiers** : Choisissez quels dossiers synchroniser

## Comment les e-mails apparaissent

Les e-mails de toutes vos boîtes aux lettres connectées sont synchronisés avec Twenty et apparaissent dans :

* **Fiches Personnes** : Selon l'adresse e-mail du contact
* **Fiches Entreprises** : Selon le domaine de l'e-mail
* **Opportunités** : Selon l'entreprise liée

<Note>Chaque e-mail indique depuis quelle boîte aux lettres il a été envoyé/dans laquelle il a été reçu, afin que vous puissiez suivre quel compte a été utilisé pour chaque communication.</Note>

## Notes importantes

<Warning>Seules de véritables boîtes aux lettres peuvent être connectées. Les alias d'e-mail qui redirigent vers une autre boîte aux lettres ne peuvent pas être connectés séparément - ils se synchroniseront via la boîte aux lettres principale.</Warning>
