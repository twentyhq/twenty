---
title: Créer des objets personnalisés
description: Guide étape par étape pour créer des objets personnalisés dans Twenty.
---

Les objets personnalisés vous permettent de stocker des informations propres à votre entreprise que les objets standard ne couvrent pas. Par exemple : Projets, Produits, Tickets ou Annonces.

<Note>
  **Vous ne savez pas si vous avez besoin d'un objet ou d'un champ ?** Consultez [Comprendre votre modèle de données](/l/fr/user-guide/data-model/overview) pour des conseils.
</Note>

## Étapes

1. Allez dans **Paramètres → Modèle de données**
2. Cliquez sur **+ Nouvel objet**
3. Renseignez :
   * **Nom au singulier** (p. ex. "Annonce")
   * **Nom au pluriel** (p. ex. "Annonces")
   * **Icône**
   * **Description** (facultative)
4. Cliquez sur **Enregistrer**

Votre objet apparaît immédiatement dans la barre latérale.

## Ensuite : ajouter des champs

Les nouveaux objets commencent avec des champs de base. Ajoutez des champs personnalisés pour saisir les données dont vous avez besoin :

1. Dans **Paramètres → Modèle de données**, sélectionnez votre objet
2. Cliquez sur **+ Ajouter un champ**
3. Choisissez un type de champ, configurez et enregistrez

Voir [Comment créer des champs personnalisés](/l/fr/user-guide/data-model/how-tos/create-custom-fields) pour des détails sur les types de champs et la configuration.

## Connexion à d'autres objets

Pour lier votre objet à des Personnes, des Entreprises ou à d'autres objets, créez un champ de relation. Voir [Comment créer des champs de relation](/l/fr/user-guide/data-model/how-tos/create-relation-fields).

## Désactivation d'un objet

Si vous n'avez plus besoin d'un objet :

1. Allez dans **Paramètres → Modèle de données**
2. Désactivez l'objet

L'objet est masqué, mais les données sont conservées. Vous pourrez le réactiver ou le supprimer définitivement plus tard.
