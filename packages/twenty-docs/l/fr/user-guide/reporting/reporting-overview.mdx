## Quoi de neuf

### Rapports personnalisés

Créez des rapports sur mesure pour suivre les indicateurs qui comptent le plus pour votre entreprise.

### Tableaux de bord de vente

Visualisez les performances de votre pipeline de vente avec des tableaux de bord interactifs.

### Analyse des performances

Surveillez la performance de l'équipe et identifiez les tendances dans vos données clients.

## Restez informé

Suivez notre [répertoire GitHub](https://github.com/twentyhq/twenty) ou consultez la section **Paramètres → Versions** dans votre espace de travail pour être averti lorsque les fonctionnalités de reporting deviennent disponibles.
