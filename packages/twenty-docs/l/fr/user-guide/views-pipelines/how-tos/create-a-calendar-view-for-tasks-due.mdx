## Prérequis

L'objet Tasks doit comporter un champ **Due Date** (de type Date ou Date & Time).

## Étapes

1. Accédez à **Tasks**
2. Cliquez sur le menu déroulant de la vue → **+ Ajouter une vue**
3. Nommez votre vue (par exemple, "Tasks Calendar")
4. Cliquez sur **Créer**
5. Cliquez sur **Options** et sélectionnez **Calendar** comme mise en page
6. Choisissez **Due Date** comme champ de date
7. Cliquez sur **Enregistrer**

## Configurez votre calendrier

### Afficher les champs sur les événements

1. Cliquez sur **Options → Champs**
2. Cliquez sur l'**icône en forme d'œil** pour afficher/masquer les champs
3. Faites glisser pour réorganiser

Champs recommandés à afficher :

* **Title** — nom de la tâche
* **Assignee** — personne responsable
* **Status** — avancement actuel

### Filtrez votre calendrier

Créez des vues ciblées :

* **My Tasks** : Filtrez par Assignee = Me
* **This Week** : Filtrez par Due Date = This week
* **En retard** : Filtrer par Date d'échéance < Aujourd'hui, Statut ≠ Terminé

## Autres cas d'utilisation du calendrier

| Objet                    | Champ de date       | Objectif                         |
| ------------------------ | ------------------- | -------------------------------- |
| Opportunités             | Date de clôture     | Suivre les clôtures prévues      |
| Événements personnalisés | Date de l'événement | Planifier les activités          |
| Projets                  | Échéance            | Suivre les échéances des projets |

## Conseils

* **Passez en revue chaque semaine** : Commencez chaque semaine en consultant votre vue Calendrier
* **Combiner avec la vue Tableau** : Utilisez le calendrier pour la vue d'ensemble, le tableau pour les détails
* **Définir la visibilité** : Conservez les calendriers de tâches personnels comme Non répertoriés
