## À propos des Vues de Tableau

Les vues de tableau affichent les enregistrements sous forme de lignes, avec des colonnes personnalisables — comme un tableur. C’est le type de vue par défaut pour la plupart des objets.

<img src="/images/user-guide/table-views/table-view.png" style={{width:'100%'}} />

## Fonctionnalités

### Configuration des colonnes

* Afficher ou masquer des colonnes (champs)
* Redimensionner la largeur des colonnes
* Réorganiser les colonnes en les faisant glisser

### Regrouper par un champ de type Sélection

Organisez les enregistrements en groupes repliables selon un champ de type Sélection.

<img src="/images/user-guide/table-views/table-group-by.png" style={{width:'100%'}} />

1. Cliquez sur **Options**
2. Sélectionnez **Grouper**
3. Choisissez un champ de type Sélection
4. Configurez l’ordre des groupes sous **Options → Grouper → Trier** :
   * **Alphabétique** ou **alphabétique inversé**
   * **Ordre manuel** : Faites glisser les groupes sous "Groupes visibles" pour les réorganiser
   * Cliquez sur l’**icône en forme d’œil** à côté d’un groupe pour le masquer

**Cas d’utilisation :**

* Regrouper les entreprises par type
* Regrouper les opportunités par étape
* Regrouper les tâches par statut

<Note>
  **Pour des performances optimales, limitez-vous à 10-15 groupes visibles par vue.** Si vous avez besoin de plus de groupes, envisagez d’utiliser plutôt un tableau de bord.
</Note>

### Largeur des colonnes

Redimensionnez les colonnes pour afficher plus ou moins de contenu :

1. Placez le curseur entre deux en-têtes de colonne
2. Cliquez et faites glisser la bordure de la colonne
3. Relâchez pour définir la nouvelle largeur

## Quand utiliser les vues de tableau

Les vues de tableau sont idéales pour :

* **Parcourir de grands ensembles de données** — passer rapidement en revue un grand nombre d’enregistrements
* **Saisie de données** — modifier efficacement plusieurs enregistrements
* **Analyse détaillée** — voir de nombreux champs en même temps
* **Tri et filtrage** — trouver des enregistrements spécifiques
