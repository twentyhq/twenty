## Sélection des champs à afficher

Chaque vue peut afficher un ensemble de champs différent. Personnalisez ce qui est visible pour vous concentrer sur les informations importantes.

### Afficher ou masquer des champs

1. Cliquez sur **Options** en haut à droite
2. Cliquez sur **Champs**
3. Cliquez sur l'**icône en forme d'œil** à côté de chaque champ pour l'afficher ou le masquer

### Réorganiser les champs

Modifiez l'ordre dans lequel les champs apparaissent dans votre vue :

1. Cliquez sur **Options → Champs**
2. Faites glisser les champs vers le haut ou vers le bas
3. Les modifications sont enregistrées automatiquement

## Affichage des champs par type de vue

### Vues de Tableau

* Les champs apparaissent sous forme de colonnes
* Redimensionnez les colonnes en faisant glisser les bordures

### Vues Kanban

* Les champs apparaissent sur les cartes
* Réorganisez via Options → Champs
* Utilisez la vue compacte pour masquer tous les champs

### Vues calendrier

* Les champs sélectionnés s'affichent sur les événements du calendrier
* Configurez via Options → Champs

## Meilleures pratiques

* **N'affichez que ce qui est nécessaire** — trop de champs surchargent la vue
* **Placez les champs importants en premier** — les colonnes les plus utilisées à gauche
* **Créez plusieurs vues** — ensembles de champs différents pour différents objectifs
* **Utilisez la visibilité des champs par vue** — même objet, angle différent
