## Visibilité de la vue

Contrôlez qui peut voir vos vues personnalisées.

### Options de visibilité

| Paramètre             | Qui peut voir                           |
| --------------------- | --------------------------------------- |
| **Espace de travail** | Tous les membres de l'espace de travail |
| **Non répertorié**    | Vous uniquement                         |

### Modifier la visibilité

1. Ouvrez la vue
2. Cliquez sur **Options → Visibilité**
3. Sélectionnez **Espace de travail** ou **Non répertorié**

<Note>
  La visibilité des vues par défaut "All [Object Name]" ne peut pas être modifiée.
</Note>

## Renommer une vue

1. Ouvrez le menu déroulant de la vue
2. Cliquez sur le menu **⋮** à côté de la vue
3. Sélectionnez **Modifier**
4. Saisissez le nouveau nom

## Modifier l'icône de la vue

1. Ouvrez le menu déroulant de la vue
2. Cliquez sur le menu **⋮** à côté de la vue
3. Sélectionnez **Modifier**
4. Cliquez sur l'icône pour la modifier

## Réorganiser les vues

Modifiez l'ordre d'affichage des vues dans le menu déroulant:

1. Ouvrez le menu déroulant de la vue
2. Faites glisser les vues à l'aide de leur poignée
3. Déposez à la position souhaitée
4. L'ordre est enregistré automatiquement

## Favoris

Épinglez les vues fréquemment utilisées pour un accès rapide:

1. Ouvrez le menu déroulant de la vue
2. Cliquez sur le menu **⋮** à côté d'une vue
3. Sélectionnez **Ajouter aux favoris**

Les vues ajoutées aux favoris apparaissent dans une section dédiée pour un accès rapide.

## Supprimer une vue

1. Ouvrez le menu déroulant de la vue
2. Cliquez sur le menu **⋮** à côté de la vue
3. Sélectionnez **Supprimer**
4. Confirmez la suppression

<Warning>
  Les vues supprimées ne peuvent pas être restaurées.
</Warning>
