## Filtrage des données

Les filtres vous aident à vous concentrer sur des enregistrements spécifiques en n'affichant que ceux qui correspondent à vos critères.

### Ajout d'un filtre

1. Cliquez sur le bouton **Filtre** dans la barre d'outils
2. Sélectionnez le champ à utiliser pour le filtrage
3. Choisissez l'opérateur (égal à, contient, etc.)
4. Saisissez la valeur du filtre
5. Cliquez sur **Appliquer**

### Opérateurs de filtrage

| Type de champ | Opérateurs disponibles                                   |
| ------------- | -------------------------------------------------------- |
| Texte         | Égal à, Contient, Commence par, Se termine par, Est vide |
| Nombre        | Égal à, Supérieur à, Inférieur à, Entre, Est vide        |
| Date          | Égal à, Avant, Après, Entre, Est vide                    |
| Sélection     | Égal à, Est l'un de, Est vide                            |
| Case à cocher | Est vrai, Est faux                                       |
| Relation      | Égal à, Est vide                                         |

### Filtres multiples

Combinez plusieurs filtres pour affiner les résultats:

* Tous les filtres sont appliqués avec une logique ET
* Chaque filtre supplémentaire restreint davantage les résultats

### Suppression des filtres

* Cliquez sur le **X** de chaque pastille de filtre
* Cliquez sur **Tout effacer** pour supprimer tous les filtres

## Tri des données

Le tri détermine l'ordre dans lequel les enregistrements apparaissent.

### Ajout d'un tri

1. Cliquez sur le bouton **Tri** dans la barre d'outils
2. Sélectionnez le champ à utiliser pour le tri
3. Choisissez l'ordre croissant (A-Z, 0-9) ou décroissant (Z-A, 9-0)
4. Cliquez sur **Appliquer**

### Tris multiples

Ajoutez plusieurs niveaux de tri:

* Le premier tri est prioritaire
* Les tris suivants s'appliquent au sein des groupes de valeurs égales

### Tri rapide par colonne

Cliquez sur l'en-tête d'une colonne pour trier:

* Premier clic: croissant
* Deuxième clic: décroissant
* Troisième clic: supprimer le tri

## Enregistrement des paramètres de filtre et de tri

Les filtres et les tris sont enregistrés avec la vue:

1. Configurez vos filtres et tris
2. Cliquez sur **Enregistrer** pour mettre à jour la vue actuelle
3. Ou cliquez sur **Enregistrer comme nouvelle vue** pour créer une variante
