## Vue d'ensemble

Lorsque vous ouvrez un enregistrement dans Twenty, la page de détails se compose d'**onglets** et de **widgets**. Les deux sont entièrement personnalisables par type d'objet.

## Onglets

Chaque page d'enregistrement peut comporter plusieurs onglets — similaires aux onglets d'un navigateur. Utilisez-les pour organiser différents aspects d'un enregistrement. Par exemple, un enregistrement de type Société peut comporter des onglets pour Aperçu, Communication, Tâches et Fichiers.

Vous pouvez :

* Ajouter et supprimer des onglets
* Renommer des onglets
* Réorganiser les onglets en les faisant glisser
* Définir l'onglet affiché par défaut

## Widgets

Les widgets sont les éléments constitutifs de chaque onglet. Types de widgets disponibles :

| Widget                       | Ce qu'il affiche                                        |
| ---------------------------- | ------------------------------------------------------- |
| **Champs**                   | Champs de l'enregistrement, groupés ou individuellement |
| **Enregistrements associés** | Tableau d'enregistrements liés via une relation         |
| **Emails**                   | Historique des e-mails des comptes connectés            |
| **Calendrier**               | Événements de calendrier associés à l'enregistrement    |
| **Chronologie**              | Historique des activités et des événements              |
| **Tâches**                   | Tâches associées                                        |
| **Notes**                    | Notes en texte enrichi                                  |
| **Fichiers**                 | Fichiers joints                                         |
| **Graphiques**               | Données visuelles provenant d'enregistrements associés  |
| **iFrame**                   | Contenu externe intégré                                 |
| **Texte enrichi**            | Contenu statique ou descriptions                        |

## Personnaliser une page d'enregistrement

1. Ouvrez n'importe quel enregistrement
2. Appuyez sur `Cmd+K` et recherchez "Modifier la mise en page de la page d'enregistrement"

ou

1. Accédez à Paramètres > Modèle de données > objet de votre choix > Mise en page

2. Cliquez sur le bouton "Personnaliser la page d'enregistrement" pour cet objet

3. Vous êtes maintenant en mode de personnalisation :
   * **Ajoutez des widgets** à partir du sélecteur de widgets
   * **Faites glisser les widgets** pour les repositionner sur la grille
   * **Redimensionnez les widgets** en faisant glisser leurs bords
   * **Configurez les champs** affichés dans chaque widget
   * **Gérez les onglets** — ajouter, supprimer, renommer, réorganiser

4. Enregistrez vos modifications — elles s'appliquent à tous les enregistrements de ce type d'objet

## Visibilité des champs

Dans le widget Champs, vous pouvez contrôler quels champs sont visibles et dans quel ordre. Cela vous permet de créer des mises en page ciblées — par exemple, n'afficher que les champs les plus importants dans l'onglet Aperçu et placer les champs détaillés dans un onglet séparé.
