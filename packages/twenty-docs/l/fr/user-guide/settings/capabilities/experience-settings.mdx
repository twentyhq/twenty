Personnalisez votre expérience Twenty dans **Paramètres → Expérience**.

## Apparence

Choisissez votre thème visuel :

* **Clair** : Interface épurée et lumineuse
* **Sombre** : Plus reposant pour les yeux dans des conditions de faible luminosité
* **Paramètres système** : S'adapte automatiquement au thème de votre appareil

## Langue

Sélectionnez votre langue préférée pour l'interface Twenty dans le menu déroulant.

## Formats

Configurer la date, l'heure, le nombre, le fuseau horaire et le jour de début du calendrier.

| Paramètre                       | Description                                                                 |
| ------------------------------- | --------------------------------------------------------------------------- |
| **Fuseau horaire**              | Votre fuseau horaire local pour une planification et des horodatages précis |
| **Format de date**              | Format d'affichage des dates (p. ex. : 12 déc. 2025)                        |
| **Format de l'heure**           | Format 12 heures (7:22 PM) ou 24 heures                                     |
| **Format de nombre**            | Séparateurs pour les décimales et les milliers (p. ex. : 1 234,56)          |
| **Jour de début du calendrier** | Premier jour de la semaine (dimanche ou lundi)                              |

Chaque paramètre peut être défini sur **Paramètres système** pour s'adapter automatiquement aux préférences de votre appareil, ou vous pouvez choisir un format spécifique.

## Comment mettre à jour

1. Allez dans **Paramètres → Expérience**
2. Ajustez vos préférences dans chaque section
3. Les modifications sont enregistrées automatiquement
