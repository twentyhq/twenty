Gérez qui a accès à votre espace de travail dans **Paramètres → Membres**.

## Inviter de nouveaux membres

### Utilisation de l'invitation par e-mail

1. Allez dans **Paramètres → Membres**
2. Cliquez sur **+ Inviter**
3. Entrez l'adresse e-mail de la personne
4. Sélectionnez un rôle pour le nouveau membre
5. Cliquez sur **Envoyer l'invitation**

La personne invitée recevra un e-mail contenant un lien pour rejoindre votre espace de travail.

### Utilisation du lien d'invitation

1. Allez dans **Paramètres → Membres**
2. Copiez le lien d'invitation de l'espace de travail
3. Partagez le lien avec les nouveaux membres de l'équipe
4. Ils auront accès dès leur inscription

## Afficher et gérer les membres

### Afficher tous les membres

Allez dans **Paramètres → Membres** pour voir :

* Tous les membres actifs
* Invitations en attente

### Modifier le profil d'un membre

Cliquez sur un membre pour ouvrir sa page de profil. En tant qu'administrateur, vous pouvez :

* Modifier son **nom**
* Mettre à jour sa **photo de profil**
* **Se connecter à sa place** (utile pour le dépannage)
* **Supprimer** son compte

### Modifier le rôle d'un membre

Sur la page de profil du membre :

1. Ouvrez l'onglet **Autorisations**
2. Affichez le rôle actuellement attribué
3. Sélectionnez un rôle différent dans le menu déroulant
4. La modification prend effet immédiatement

→ [En savoir plus sur les rôles et les autorisations](/l/fr/user-guide/permissions-access/capabilities/permissions)

### Retirer un membre

1. Cliquez sur le membre pour ouvrir son profil
2. Cliquez sur **Supprimer** pour le retirer de l'espace de travail

<Warning>
  Les membres retirés perdent immédiatement l'accès. Leurs données (enregistrements, notes, tâches) restent dans l'espace de travail.
</Warning>

<Warning>
  **La synchronisation des e-mails est également supprimée.** Si l'utilisateur supprimé était le seul à avoir synchronisé certains e-mails, ces e-mails seront définitivement supprimés de l'espace de travail.
</Warning>

## Invitations en attente

Gérez les invitations qui n'ont pas été acceptées :

* **Renvoyer** : envoyer à nouveau l'e-mail d'invitation
* **Annuler** : révoquer l'invitation avant qu'elle ne soit acceptée

## Domaines d'accès approuvés

Autorisez les membres de l'équipe à rejoindre automatiquement l'espace de travail selon leur domaine e-mail :

1. Allez dans **Paramètres → Domaines**
2. Ajoutez le domaine de votre entreprise (par exemple, `yourcompany.com`)
3. Toute personne dont l'adresse e-mail appartient à ce domaine peut rejoindre sans invitation
