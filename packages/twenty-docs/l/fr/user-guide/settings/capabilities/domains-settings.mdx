---
title: Paramètres des domaines
description: Configurez le domaine de l'espace de travail, les domaines d'accès approuvés et les domaines publics.
---

Les paramètres de domaine se trouvent à trois endroits, en fonction de ce que vous configurez.

## Domaine de l'espace de travail

Configurez dans **Paramètres → Général → Domaine de l’espace de travail**.

Modifiez le nom de votre sous-domaine ou définissez un domaine personnalisé pour votre espace de travail.

### Personnaliser le domaine

1. Cliquez sur **Personnaliser le domaine**
2. Modifiez votre sous-domaine (par exemple, `yourcompany.twenty.com`)
3. Ou configurez un domaine personnalisé (par exemple, `crm.yourcompany.com`)

Pour les domaines personnalisés, vous devrez configurer les paramètres DNS auprès de votre fournisseur de domaine.

## Domaines approuvés

Configurez dans **Paramètres → Membres → Invitation**.

Toute personne ayant une adresse e-mail dans l'un de ces domaines est autorisée à s'inscrire automatiquement à cet espace de travail.

### Ajouter un domaine d'accès approuvé

1. Cliquez sur **Ajouter un domaine d'accès approuvé**
2. Saisissez le domaine de votre entreprise (par exemple, `yourcompany.com`)
3. Enregistrer

Une fois configuré, toute personne ayant une adresse e-mail de ce domaine peut rejoindre votre espace de travail sans avoir besoin d'une invitation directe.

<Note>
Cela est utile pour permettre à toute votre équipe de s'auto-inscrire tout en gardant l'espace de travail restreint à votre organisation.
</Note>

## Domaines publics

Configurez dans **Paramètres → Applications → Développeur**.

Provisionner un environnement d'hébergement complet et sécurisé sur ces domaines. Un domaine public peut être lié à une application spécifique — lorsqu’il est lié, seules les fonctions logiques acheminées via HTTP de cette application sont accessibles sur le domaine. Laissez la liaison vide pour exposer toutes les routes HTTP de l’espace de travail.

### Ajouter domaine public

1. Cliquez sur **Ajouter un domaine public**
2. Saisissez le domaine que vous souhaitez utiliser
3. Facultatif : liez-le à une application
4. Configurez les paramètres DNS comme indiqué
5. Vérifiez le domaine

Les certificats SSL sont automatiquement fournis pour les domaines publics.
