## Inviter de nouveaux membres

### Utilisation du lien d'invitation

1. Allez dans **Paramètres → Membres**
2. Copiez le lien d'invitation de l'espace de travail
3. Partagez le lien avec les nouveaux membres de l'équipe
4. Ils recevront l'accès une fois qu'ils se seront inscrits

### Invitation directe par e-mail

1. Allez dans **Paramètres → Membres**
2. Entrez l'adresse e-mail de la personne
3. Cliquez sur **Inviter**
4. Ils recevront une invitation par e-mail

## Supprimer des membres

### Supprimer un membre

1. Allez dans **Paramètres → Membres**
2. Trouvez le membre que vous souhaitez supprimer
3. Cliquez sur le bouton supprimer à côté de leur nom
4. Confirmez la suppression

**Remarque** : Les membres supprimés perdent immédiatement l'accès, mais peuvent être réinvités plus tard.
