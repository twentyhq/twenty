## Vue d'ensemble

Les crédits de flux de travail alimentent vos automatisations dans Twenty. Chaque action du flux de travail consomme des crédits en fonction de sa complexité.

## Allocation des crédits

Les crédits dépendent de votre cycle de facturation, et non de votre forfait :

| Cycle de facturation | Crédits         |
| -------------------- | --------------- |
| Mensuel              | 5 millions/mois |
| Annuel               | 50 millions/an  |

<Note>
  Les 5 millions de crédits mensuels sont conçus pour vous permettre d'exécuter des automatisations sans vous soucier des coûts. Pour la plupart des flux de travail utilisant des actions standard, c'est plus que suffisant. Vous n'aurez besoin de crédits supplémentaires que lors de l'exécution de nœuds de code avancés ou de fonctionnalités alimentées par l'IA.
</Note>

## Consommation de crédits

Les différentes actions consomment des quantités de crédits différentes :

| Type d'action                                                                 | Utilisation des crédits               |
| ----------------------------------------------------------------------------- | ------------------------------------- |
| **Opérations de base** (rechercher, mettre à jour, créer des enregistrements) | Minimale                              |
| **Opérations complexes** (nœuds de code, appels d'API externes)               | Plus de crédits                       |
| **Prompts d'IA** (bientôt disponibles)                                        | Variable en fonction de l'utilisation |

Les crédits sont déduits en temps réel lorsque les flux de travail s'exécutent.

## Suivi de l'utilisation

Suivez votre consommation de crédits :

1. Allez dans **Paramètres → Facturation**
2. Consultez votre utilisation actuelle et vos crédits restants
3. Surveillez les tendances pour planifier des crédits supplémentaires si nécessaire

## Achat de crédits supplémentaires

Besoin de plus de crédits ?

1. Allez dans **Paramètres → Facturation**
2. Cliquez sur l'option pour acheter des packs de crédits supplémentaires
3. Sélectionnez la quantité dont vous avez besoin
