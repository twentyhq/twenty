## En un coup d’œil

| Plan                            | Fonctionnalités clés                                                                                                                                                                                                         |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Gratuit (auto-hébergé)**      | Toutes les fonctionnalités Pro, support communautaire                                                                                                                                                                        |
| **Pro (Cloud)**                 | Tout hormis les fonctionnalités Premium (SSO et autorisations au niveau des lignes), support standard                                                                                                                        |
| **Organisation (Cloud)**        | Tout ce qui est inclus dans Pro + les fonctionnalités Premium (SSO et autorisations au niveau des lignes), support prioritaire                                                                                               |
| **Organisation (auto-hébergé)** | Tout ce qui est inclus dans Pro + les fonctionnalités Premium (SSO, autorisations au niveau des lignes), assistance de l’équipe Twenty, sans obligation de publier votre code personnalisé en open source avant distribution |

## Réponses rapides

**Où puis-je gérer la facturation ?**
Accédez à **Paramètres → Facturation** pour consulter votre offre, mettre à jour les moyens de paiement et accéder aux factures.

**Puis-je utiliser Twenty gratuitement ?**
Oui ! Auto-hébergez Twenty et bénéficiez de toutes les fonctionnalités Pro sans frais.

**Comment effectuer une mise à niveau ?**
Accédez à **Paramètres → Facturation** et cliquez sur **Passer à Organisation** ou **Passer à Annuel**.

Pour d’autres questions, consultez la [FAQ sur la facturation](/l/fr/user-guide/billing/how-tos/billing-faq).
