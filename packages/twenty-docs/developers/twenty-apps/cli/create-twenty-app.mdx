The `create-twenty-app` CLI allow you to create a new Twenty application using the default template.
It is the easiest way to get started with Twenty applications.

Basic usage:

```bash filename="Terminal"
npx create-twenty-app@latest [project-name]
```

## Reference

The following options are available:

| Options                                 | Description                                                           |
| --------------------------------------- | --------------------------------------------------------------------- |
| `-h` or `--help`                        | Show all available options                                            |
| `-v` or `--version`                     | Output the version number                                             |
