## Overview

Environment variables control how your Twenty instance behaves. This guide covers the essential configuration options.

## Required Variables

| Variable | Description |
|----------|-------------|
| `APP_SECRET` | Unique secret for encryption (generate with `openssl rand -base64 32`) |
| `FRONT_BASE_URL` | Public URL of your frontend (e.g., `https://crm.yourcompany.com`) |
| `SERVER_URL` | Public URL of your API server |

## Database Configuration

| Variable | Description |
|----------|-------------|
| `POSTGRES_HOST` | Database hostname |
| `POSTGRES_PORT` | Database port (default: 5432) |
| `POSTGRES_USER` | Database username |
| `POSTGRES_PASSWORD` | Database password |
| `POSTGRES_DB` | Database name |

## Email Configuration

To enable email features:

| Variable | Description |
|----------|-------------|
| `EMAIL_FROM_ADDRESS` | Sender email address |
| `EMAIL_SMTP_HOST` | SMTP server hostname |
| `EMAIL_SMTP_PORT` | SMTP port |
| `EMAIL_SMTP_USER` | SMTP username |
| `EMAIL_SMTP_PASSWORD` | SMTP password |

## Apply Changes

After updating environment variables:

```bash
docker-compose down
docker-compose up -d
```
