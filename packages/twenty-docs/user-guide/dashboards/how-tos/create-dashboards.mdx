## Creating Your First Dashboard

### Steps
1. Navigate to **Dashboards** in the sidebar
2. Click **+ New Dashboard**
3. Enter a name for your dashboard
4. Click **Create**

## Adding Charts

### Add a Chart
1. On your dashboard, click **+ Add Chart**
2. Select a chart type:
   - Bar chart
   - Pie chart
   - Line chart
   - Number metric
3. Configure the chart settings
4. Click **Save**

### Configure Chart Data

**Select Data Source**:
1. Choose the object (Opportunities, People, etc.)
2. Select the metric (count, sum, average)
3. Choose grouping field (stage, owner, date)

**Add Filters**:
1. Click **Add Filter**
2. Select field and condition
3. Apply to narrow the data

## Dashboard Examples

### Sales Overview Dashboard
**Charts to include**:
- Pipeline value by stage (bar chart)
- Deals by owner (pie chart)
- Monthly deal count (line chart)
- Total pipeline value (number)

### Activity Dashboard
**Charts to include**:
- Tasks completed this week (number)
- Activities by type (pie chart)
- Activity trend (line chart)
- Overdue tasks (number)

### Pipeline Health Dashboard
**Charts to include**:
- Deals by stage (bar chart)
- Average deal size (number)
- Win rate (number)
- Close date distribution (line chart)

## Customizing Dashboards

### Rearrange Charts
1. Drag charts to reposition
2. Resize by dragging edges
3. Changes save automatically

### Edit Charts
1. Click the edit icon on a chart
2. Modify settings
3. Click **Save**

### Delete Charts
1. Click the options menu on a chart
2. Select **Delete**
3. Confirm deletion

## Sharing Dashboards

### Share with Team
1. Open dashboard settings
2. Toggle **Share with workspace**
3. Save changes
4. Team members can now view

### Set as Default
1. Open dashboard settings
2. Set as default for your workspace
3. Team members see this on login

## Best Practices

### Design
- Keep dashboards focused
- Use consistent colors
- Group related charts
- Include key metrics prominently

### Maintenance
- Review dashboards regularly
- Update as needs change
- Remove outdated charts
- Keep chart count manageable

### Team Use
- Create role-specific dashboards
- Document what charts show
- Gather feedback from users
