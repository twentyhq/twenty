## Key features

- **Standard objects**: Pre-built People, Companies, and Opportunities objects ready to use
- **Custom objects**: Create objects unique to your business (e.g., Products, Projects, Tickets)
- **Field types**: Text, numbers, dates, selects, relations, and more
- **Relation fields**: Connect records across different objects
- **Deactivation**: Hide objects or fields without losing data
