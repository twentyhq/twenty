## Key features

- **Triggers**: Start workflows on record events, schedules, webhooks, or manual actions
- **Record actions**: Create, update, delete, upsert, and search records
- **Communication**: Send emails, collect input via forms
- **Logic & control**: Filter, branch, iterate through records, add delays
- **Integrations**: Connect to external tools via HTTP requests and webhooks
- **AI**: Run AI agents to process data intelligently
- **Credits**: Track and manage workflow execution credits
