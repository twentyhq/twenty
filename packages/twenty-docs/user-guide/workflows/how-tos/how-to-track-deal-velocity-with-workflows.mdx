description: Use workflows to measure how long opportunities stay in each pipeline stage.

Tracking deal velocity helps you identify bottlenecks in your sales process and forecast more accurately.

This use case requires multiple workflows working together to:
1. Record when opportunities enter each stage
2. Calculate days spent in each stage when moving forward
