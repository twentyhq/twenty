## Welcome to Twenty!

After creating your workspace, your first login will guide you through essential setup steps to get your CRM ready for daily use.

## Initial Login Process

### Accessing Your Workspace
1. **Open Your Browser**: Navigate to your workspace URL (e.g., `yourcompany.twenty.com`)
2. **Enter Credentials**: Use the email and password you created during signup
3. **Two-Factor Authentication**: If enabled, enter your authentication code
4. **Welcome Screen**: You'll see Twenty's onboarding interface

### First-Time Setup Wizard

Twenty's setup wizard helps you configure essential settings:

#### Step 1: Workspace Information
- **Workspace Name**: Confirm or update your workspace name
- **Company Size**: Select your organization size for better recommendations
- **Industry**: Choose your industry to enable relevant features
- **Primary Use Case**: Select how you'll mainly use Twenty (Sales, Support, Projects, etc.)

#### Step 2: Basic Preferences
- **Time Zone**: Set your organization's primary time zone
- **Date Format**: Choose how dates appear throughout the system
- **Currency**: Select your default currency for financial fields
- **Language**: Pick your preferred interface language

#### Step 3: Data Import Options
- **Start Fresh**: Begin with empty objects and add data manually
- **Import Data**: Upload existing data from CSV files or other systems
- **Use Sample Data**: Explore Twenty with pre-loaded example records

## Essential Configuration Tasks

### 1. Review Default Objects

Twenty includes standard objects that work for most businesses:

**People**: Individual contacts and prospects
- Pre-configured with name, email, phone, and company fields
- Ready for immediate use

**Companies**: Organizations you work with
- Includes company name, domain, industry, and size fields
- Automatically links to People records

**Opportunities**: Sales deals and potential revenue
- Tracks deal amount, stage, and probability
- Connected to People and Companies

**Tasks**: Action items and to-dos
- Assignable to team members
- Can be linked to any other record type

**Notes**: Important information and observations
- Rich text formatting supported
- Attachable to any record

### 2. Configure Your Profile

Set up your personal profile information:

1. **Click Your Avatar**: Located in the top-right corner
2. **Select Profile Settings**: Access your personal configuration
3. **Complete Your Information**:
   - Full name and job title
   - Profile photo
   - Contact information
   - Notification preferences

### 3. Customize Your Workspace Appearance

Make Twenty feel like home:

1. **Go to Settings**: Click the gear icon in the sidebar
2. **Select Workspace**: Choose workspace-wide settings
3. **Upload Your Logo**: Add your company logo to the interface
4. **Choose Colors**: Customize the color scheme (Pro feature)
5. **Set Branding**: Configure how your workspace appears to users

### 4. Set Up Email Integration

Connect your email to start tracking communications:

1. **Navigate to Settings**: Go to workspace settings
2. **Select Email & Calendar**: Find integration options
3. **Connect Your Email**: Follow the setup wizard for your email provider
4. **Configure Sync Settings**: Choose what to sync and how often

## Understanding Your Dashboard

### Main Navigation
The sidebar contains your primary navigation:
- **Objects**: Access People, Companies, Opportunities, etc.
- **Views**: Different ways to see your data
- **Tasks**: Your assigned action items
- **Settings**: Configuration and administration

### Quick Actions
The top bar provides quick access to:
- **Search**: Find any record across all objects
- **Create New**: Add records to any object
- **Command Palette**: Access any feature quickly (Cmd+K)
- **Notifications**: Stay updated on important activities

### Dashboard Widgets
Your home dashboard shows:
- Recent activity across all objects
- Upcoming tasks and deadlines
- Key metrics and summaries
- Quick access to frequently used views

## Initial Data Setup

### Option 1: Start with Sample Data
If you chose sample data during setup:
- Explore pre-loaded People, Companies, and Opportunities
- See how objects relate to each other
- Practice with realistic but fake data
- Delete sample data when ready to add real information

### Option 2: Import Your Data
If you have existing data to import:
1. **Prepare Your Files**: Format data as CSV files
2. **Use Import Wizard**: Go to Settings â†’ Import
3. **Map Your Fields**: Match your data to Twenty's fields
4. **Review and Import**: Check the preview and complete the import

### Option 3: Manual Entry
Start adding data manually:
1. **Begin with Companies**: Add the organizations you work with
2. **Add People**: Create contacts and link them to companies
3. **Create Opportunities**: Add any active deals or prospects
4. **Set Up Tasks**: Add immediate action items

## Team Collaboration Setup

### Invite Team Members
1. **Go to Settings**: Navigate to workspace settings
2. **Select Members**: Find team management options
3. **Send Invitations**: Enter email addresses for team members
4. **Set Roles**: Assign appropriate permissions to each user
5. **Welcome Your Team**: Help them complete their own setup

### Create Shared Views
Set up views that your team will use:
1. **Navigate to Any Object**: Go to People, Companies, etc.
2. **Create New View**: Click the view options
3. **Configure Filters**: Set up relevant filters for your team
4. **Share the View**: Make it available to all team members
5. **Set as Default**: Choose which view team members see first

## Troubleshooting First Login Issues

### Can't Access Your Workspace
- Verify the correct workspace URL
- Check your email for login credentials
- Ensure your internet connection is stable
- Try clearing browser cache and cookies

### Forgot Password
- Use the "Forgot Password" link on the login page
- Check your email for reset instructions
- Create a new secure password
- Consider enabling two-factor authentication

### Setup Wizard Problems
- Refresh the page and try again
- Skip optional steps and return later
- Contact support if errors persist
- Check browser compatibility
