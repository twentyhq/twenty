## Key features

- **Workspace settings**: Configure workspace name, logo, and preferences
- **Profile settings**: Manage your personal profile and preferences
- **Experience settings**: Customize date formats, time zones, and currency
- **Domain settings**: Configure custom domains for your workspace
- **Release settings**: Enable beta features and lab experiments
