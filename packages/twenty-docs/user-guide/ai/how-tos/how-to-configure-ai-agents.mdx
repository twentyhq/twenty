## Enabling AI Features

### Prerequisites
- Active Twenty workspace
- Admin or appropriate permissions
- AI features available on your plan

### Steps to Enable
1. Go to **Settings → Releases → Lab**
2. Find the AI features section
3. Toggle on the features you want to use
4. Return to your workspace

## Configuring AI Agents

### Access AI Settings
1. Go to **Settings → AI**
2. View available AI agents
3. Click on an agent to configure

### Configuration Options
- **Enable/Disable**: Turn the agent on or off
- **Data Access**: Configure which data the agent can access
- **Behavior Settings**: Customize how the agent responds

## Using AI Chat

### Starting a Conversation
1. Click the AI icon in the navigation
2. Type your question or request
3. The AI analyzes your data and responds

### Example Prompts
- "Show me opportunities closing this month"
- "Find contacts at companies in healthcare"
- "What's my total pipeline value?"
- "List deals I haven't updated this week"

### Tips for Better Results
- Be specific in your questions
- Include relevant context
- Refine based on responses

## AI in Workflows

### Adding AI Actions
AI can be integrated into workflows for:
- Data enrichment
- Record classification
- Content summarization
- Smart routing decisions

### Configuration
1. Add an AI action to your workflow
2. Configure the input data
3. Set processing rules
4. Define output handling

## Best Practices

### Data Quality
- Clean, organized data produces better results
- Fill in key fields consistently
- Use consistent naming conventions

### Security
- AI respects permission settings
- Review data access configurations
- Monitor AI usage

### Optimization
- Start with simple use cases
- Refine based on results
- Expand usage gradually
