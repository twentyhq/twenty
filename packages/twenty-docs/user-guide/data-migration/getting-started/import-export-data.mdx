## Importing Data

Twenty supports importing data via CSV files into any object (People, Companies, Opportunities, or custom objects).

### Basic Import Steps
1. Navigate to the object where you want to import data
2. Click the **Options** menu (⋮) at the top right
3. Select **Import**
4. Upload your CSV file
5. Map your columns to Twenty fields
6. Review and confirm the import

### Supported Import Objects
- People (contacts)
- Companies
- Opportunities
- Any custom objects you've created

### Import Tips
- **Clean your data first**: Remove duplicates and fix formatting issues before importing
- **Match field formats**: Ensure dates, numbers, and other fields match Twenty's expected formats
- **Start small**: Test with a small batch before importing large datasets
- **Map carefully**: Double-check your column mapping before confirming

## Exporting Data

Export your workspace data for backups, reporting, or migration to other systems.

### Export Steps
1. Navigate to the object you want to export
2. Click the **Options** menu (⋮)
3. Select **Export**
4. Choose your export options
5. Download the CSV file

### What Gets Exported
- All visible fields in the current view
- Record data including custom fields
- Related record IDs for maintaining relationships

## Permissions

Data import and export may require specific permissions:
- **Import**: Requires the "Import CSV" permission
- **Export**: Requires the "Export CSV" permission

Contact your workspace admin if you don't have the necessary permissions.
