Twenty makes it easy to import your existing data and export your workspace data whenever you need it. Whether you're migrating from another CRM or backing up your data, we've got you covered.

## Import Methods

| Method | Best For | Volume |
|--------|----------|--------|
| **CSV Import** | Standard migrations, regular data updates | Up to 10,000 records per file |
| **API Import** | Large-scale migrations, automated processes | Hundreds of thousands of records |

<Note>For large migrations (100,000+ records) or complex data transformations, we recommend using the API. Our [implementation partners](/user-guide/getting-started/implementation-services) can help you run these scripts if needed.</Note>

## Key features

- **CSV import**: Import contacts, companies, and custom objects from CSV files
- **Field mapping**: Map your CSV columns to Twenty fields during import
- **Relation import**: Import relationships between objects via CSV
- **Data export**: Export your entire workspace data
- **API import**: Programmatically import large datasets via the API
