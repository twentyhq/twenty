## Key features

- **Table views**: Spreadsheet-like lists with group by and column configuration
- **Kanban boards**: Visual boards with aggregations for tracking workflows
- **Calendar views**: Date-based display for meetings and deadlines
- **Filters & sorting**: Focus on exactly the records you need
- **View settings**: Control visibility, naming, icons, and favorites
