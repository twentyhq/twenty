## Sales Pipeline Setup

### Configure Your Stages
1. Go to **Settings → Data Model**
2. Select **Opportunities**
3. Edit the **Stage** field
4. Add, remove, or rename stages to match your sales process

### Recommended Stages
- **New**: Fresh opportunities just identified
- **Qualified**: Confirmed as a good fit
- **Meeting**: Engaged in discussions
- **Proposal**: Proposal sent
- **Negotiation**: Working on terms
- **Closed Won**: Deal successful
- **Closed Lost**: Deal unsuccessful

## Pipeline Views

### Create a Pipeline View
1. Go to Opportunities
2. Click **+ Add view**
3. Select **Kanban** layout
4. Choose **Stage** as the column field
5. Name it "Sales Pipeline"

### Optimize Your Pipeline View
- Show key fields: Amount, Close Date, Company
- Use compact view for high-level overview
- Filter by owner for personal pipeline

## Common Sales Views

### "My Pipeline"
- **Type**: Kanban
- **Filter**: Owner = Me
- **Columns**: Stage
- **Fields**: Company, Amount, Close Date

### "Closing This Month"
- **Type**: Table
- **Filter**: Close Date = This month, Stage ≠ Closed
- **Sort**: Close Date ascending
- **Columns**: Name, Company, Amount, Stage, Close Date

### "Stale Deals"
- **Type**: Table
- **Filter**: Last Activity > 14 days ago, Stage = Open
- **Sort**: Last Activity ascending

### "Won This Quarter"
- **Type**: Table
- **Filter**: Stage = Closed Won, Close Date = This quarter
- **Sort**: Amount descending

## Sales Workflows

### Lead Qualification
1. Create workflow trigger: Record Created (People)
2. Actions:
   - Search for existing company
   - Create task for qualification
   - Notify assigned rep

### Deal Stage Updates
1. Create workflow trigger: Record Updated (Stage field)
2. Actions:
   - Update related company status
   - Notify team on major stage changes
   - Create follow-up tasks

### Stale Deal Alerts
1. Create workflow trigger: On Schedule (daily)
2. Actions:
   - Search for deals not updated in X days
   - Send email to owner
   - Create reminder task

## Best Practices

### Pipeline Hygiene
- Update deals daily
- Move or close stale deals
- Keep close dates realistic
- Add notes for context

### Forecasting
- Assign realistic amounts
- Use consistent stage criteria
- Review pipeline weekly
- Track win/loss reasons

### Collaboration
- Tag team members in notes
- Share relevant views
- Document key activities
- Use tasks for follow-ups
