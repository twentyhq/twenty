## Creating a New View

### Steps
1. Navigate to the object (People, Companies, etc.)
2. Click the view dropdown at the top
3. Select **+ Add view**
4. Choose a layout type:
   - **Table**: Spreadsheet-like rows and columns
   - **Kanban**: Visual board with cards in columns
   - **Calendar**: Date-based calendar view
5. Enter a name for your view
6. Click **Create**

## Configuring Your View

### Add Filters
1. Click the **Filter** button
2. Select a field to filter by
3. Choose an operator (equals, contains, etc.)
4. Enter the filter value
5. Click **Apply**
6. Add more filters as needed

### Add Sorting
1. Click the **Sort** button
2. Select a field to sort by
3. Choose ascending or descending
4. Click **Apply**

### Configure Columns (Table View)
1. Click **Options** → **Fields**
2. Toggle fields on/off
3. Drag to reorder columns
4. Click outside to close

### Configure Cards (Kanban View)
1. Click **Options** → **Fields**
2. Toggle fields to show on cards
3. Drag to reorder fields
4. Enable **Compact view** for minimal cards

## View Examples

### "My Open Opportunities"
- **Type**: Kanban
- **Filter**: Owner = Me, Stage ≠ Closed Won, Stage ≠ Closed Lost
- **Sort**: Close Date ascending

### "Contacts Added This Week"
- **Type**: Table
- **Filter**: Created Date = This week
- **Sort**: Created Date descending
- **Columns**: Name, Email, Company, Created Date

### "High-Value Pipeline"
- **Type**: Kanban
- **Filter**: Amount > $10,000
- **Fields**: Company, Amount, Close Date

### "Upcoming Meetings"
- **Type**: Calendar
- **Filter**: Type = Meeting
- **Date Field**: Meeting Date

## Saving and Sharing

### Save View Changes
After making changes:
1. Click **Save** to update the current view
2. Or click **Save as new view** to create a copy

### Share with Team
1. Edit view settings
2. Toggle **Share with workspace**
3. Save the view
4. Team members can now access it

### Set as Default
1. Open the view dropdown
2. Click the star icon next to a view
3. This view opens by default for this object

## Best Practices

### Naming Views
- Use descriptive names
- Include key criteria (e.g., "My Deals", "This Month")
- Keep names concise

### Organizing Views
- Create views for common use cases
- Don't create too many (keep it manageable)
- Archive unused views

### Team Views
- Share views that help the whole team
- Keep personal views private
- Document what shared views are for
