# How-To Tutorials

Welcome to our comprehensive collection of step-by-step tutorials. These guides are organized by user type to help you find exactly what you need.

## For End-Users
Learn the essential day-to-day tasks in Twenty CRM:
- **Working with Data**: Add contacts, companies, and manage records
- **Email & Calendar**: Connect your mailbox and track communications
- **Views & Daily Usage**: Create custom views and organize your workspace
- **Pipelines & Sales**: Manage your sales process and opportunities
- **Notes & Tasks**: Collaborate with your team effectively

## For Admins
Master workspace configuration and management:
- **Workspace Setup**: Configure settings and manage your environment
- **User Management**: Add users, set permissions, and manage roles
- **Data Model**: Create custom objects, fields, and relationships
- **Automations & Workflows**: Build powerful automation processes
- **Dashboards**: Create and customize reporting dashboards

## For Developers
Integrate and extend Twenty:
- **API**: Authenticate and work with our GraphQL API
- **Webhooks**: Set up real-time event notifications
- **Integrations**: Build custom connections and apps

Browse the sections below or use the navigation menu to find specific tutorials for your needs.
