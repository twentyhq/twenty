## Creating Records

You can create records in several ways:

### From the Object View
1. Navigate to any object (People, Companies, Opportunities, etc.)
2. Click the `+` button in the top right
3. Fill in the required fields
4. Click Save

### Using the Command Menu
1. Press `Cmd + K` (Mac) or `Ctrl + K` (Windows)
2. Type "Create" and select the object type
3. Fill in the form and save

### Quick Create from Relationships
When viewing a record, you can create related records directly:
1. Click on a relation field
2. Select "Create new" instead of linking existing
3. The new record will be automatically linked

## Editing Records

### Inline Editing
Most fields can be edited directly in table views:
1. Click on any field in a table view
2. Make your changes
3. Press Enter or click outside to save

### Record Detail View
For more comprehensive editing:
1. Click on a record to open the side panel
2. Edit any field directly
3. Changes are saved automatically

### Bulk Editing
To update multiple records at once:
1. Select multiple records using checkboxes
2. Use the bulk actions menu
3. Choose "Edit selected records"
4. Make changes that apply to all selected records

## Managing Record Relationships

### Linking Records
Connect records to show relationships:
1. Click on a relation field
2. Search for and select the record to link
3. The relationship is created immediately

### Unlinking Records
To remove relationships:
1. Click on the linked record field
2. Click the "X" next to the linked record
3. The relationship is removed (records remain)

## Record States

### Active Records
Normal records that appear in all views and searches.

### Deleted Records
Twenty uses soft deletion by default:
- Deleted records are hidden from normal views
- Data is preserved and can be restored

### Restoring Deleted Records
1. Open Command Menu (`Cmd + K` or `Ctrl + K`)
2. Type "Deleted records"
3. Find the record you want to restore
4. Click the restore button

## Best Practices

### Data Quality
- Fill in required fields completely
- Use consistent formatting for similar data
- Regularly review and clean up duplicate records

### Relationship Management
- Link records to maintain context
- Use the most specific relationship possible
- Regularly audit relationships for accuracy

### Record Organization
- Use views to organize records by purpose
- Apply filters to focus on relevant data
- Create custom fields for business-specific information
