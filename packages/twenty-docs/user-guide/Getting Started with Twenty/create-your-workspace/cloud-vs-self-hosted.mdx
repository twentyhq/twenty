## Choosing Your Deployment Option

Twenty offers two main deployment options to fit different organizational needs, security requirements, and technical capabilities.

## Twenty Cloud (Hosted)

### What It Is
Twenty Cloud is our fully managed service where we handle all the technical infrastructure, maintenance, and updates for you.

### Key Benefits

**Zero Maintenance**: We handle servers, updates, backups, and security patches automatically.

**Instant Setup**: Get started in minutes without any technical configuration.

**Automatic Updates**: Always have the latest features and security improvements.

**Built-in Reliability**: 99.9% uptime with automatic failover and disaster recovery.

**Professional Support**: Direct access to our support team for technical issues.

### Best For
- Small to medium businesses
- Teams without dedicated IT resources
- Organizations that want to focus on using the CRM, not managing it
- Companies that need quick deployment
- Teams that prefer predictable monthly costs

### Pricing
- Free tier available for small teams
- Pro plans start at competitive monthly rates
- Enterprise plans with advanced features and support
- All plans include hosting, maintenance, and support

### Security Features
- SOC 2 Type II compliance
- End-to-end encryption
- Regular security audits
- GDPR and privacy regulation compliance
- Automatic security updates

## Self-Hosted (On-Premises)

### What It Is
Self-hosted Twenty runs on your own servers or cloud infrastructure, giving you complete control over the deployment and data.

### Key Benefits

**Complete Control**: Full ownership of your data and infrastructure.

**Custom Configuration**: Modify and extend Twenty to meet specific requirements.

**Integration Freedom**: Connect to internal systems without external dependencies.

**Compliance Flexibility**: Meet specific regulatory or industry requirements.

**Cost Predictability**: No per-user fees once deployed.

### Best For
- Large enterprises with strict data governance requirements
- Organizations with existing IT infrastructure and expertise
- Companies in highly regulated industries
- Teams that need extensive customizations
- Organizations with specific compliance requirements

### Technical Requirements

**Server Specifications**:
- Minimum: 2 CPU cores, 4GB RAM, 20GB storage
- Recommended: 4 CPU cores, 8GB RAM, 100GB storage
- Database: PostgreSQL 13+ and Redis

**Technical Skills Needed**:
- Server administration experience
- Database management knowledge
- Docker and container orchestration
- SSL certificate management
- Backup and monitoring setup

### Deployment Options

**Docker Compose**: Simplest self-hosted setup using Docker containers.

**Kubernetes**: Scalable deployment for larger organizations with container orchestration.

**Cloud Providers**: Deploy on AWS, Google Cloud, Azure, or other cloud platforms.

**On-Premises**: Run on your own physical servers for maximum control.

## Making Your Decision

### Choose Twenty Cloud If:
- You want to get started quickly
- You don't have dedicated IT resources
- You prefer predictable monthly costs
- Standard security and compliance features meet your needs
- You want automatic updates and maintenance

### Choose Self-Hosted If:
- You have strict data residency requirements
- You need extensive customizations
- You have IT expertise to manage the deployment
- You're in a highly regulated industry
- You want complete control over your infrastructure

## Migration Between Options

### Cloud to Self-Hosted
- Export all your data from Twenty Cloud
- Set up your self-hosted environment
- Import data and reconfigure integrations
- Update API endpoints and webhooks

### Self-Hosted to Cloud
- Export your self-hosted data
- Create a Twenty Cloud workspace
- Import data and team members
- Reconfigure integrations and workflows

## Security Considerations

Both deployment options provide strong security, but with different approaches:

**Twenty Cloud**: We handle security infrastructure, compliance, and updates automatically.

**Self-Hosted**: You control all security aspects but are responsible for implementation and maintenance.
